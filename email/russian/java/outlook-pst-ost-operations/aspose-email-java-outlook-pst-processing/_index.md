## Руководство по внедрению

### Загрузить файл Outlook PST

#### Обзор
Загрузка файла Outlook PST — это ваш первый шаг к программному доступу к его содержимому. Это позволяет вам выполнять такие операции, как чтение писем, извлечение контактов и многое другое.

#### Шаги

**1. Импорт необходимых классов**

Начните с импорта необходимых классов из Aspose.Email:
```java
import com.aspose.email.PersonalStorage;
```

**2. Определить путь к файлу**

Укажите путь, где находится ваш PST-файл Outlook:
```java
String pstFilePath = "YOUR_DOCUMENT_DIRECTORY/Outlook.pst";
```

**3. Загрузите PST-файл**

Загрузите PST с помощью Aspose.Email `PersonalStorage` сорт:
```java
PersonalStorage pst = PersonalStorage.fromFile(pstFilePath);
```

### Доступ к папке «Контакты» в PST

#### Обзор
После загрузки PST-файла откройте его папку «Контакты» для работы с контактными данными.

#### Шаги

**1. Извлечь корневую папку**

Откройте корневую папку загруженного PST:
```java
FolderInfo rootFolder = pst.getRootFolder();
```

**2. Получить подпапку «Контакты»**

Перейдите в подпапку «Контакты»:
```java
FolderInfo contactsFolder = rootFolder.getSubFolder("Contacts");
```

### Итерация по контактам в PST

#### Обзор
Итерация по контактам позволяет обрабатывать каждый контакт по отдельности.

#### Шаги

**1. Извлечение сообщений контакта**

Извлечь все сообщения (контакты) из папки «Контакты»:
```java
MessageInfoCollection messageInfoCollection = contactsFolder.getContents();
```

**2. Сквозные контакты**

Пройдитесь по каждому контакту с помощью цикла for:
```java
for (int i = 0; i < messageInfoCollection.size(); i++) {
    MessageInfo messageInfo = (MessageInfo) messageInfoCollection.get_Item(i);
}
```

### Извлечение и отображение контактной информации

#### Обзор
Извлекайте подробную информацию из каждого контакта для ее использования или отображения.

#### Шаги

**1. Извлечь MapiContact**

Преобразовать `MessageInfo` объект в `MapiContact`:
```java
MapiContact contact = (MapiContact) pst.extractMessage(messageInfo).toMapiMessageItem();
```

**2. Отображение контактных данных**

Извлеките и распечатайте соответствующие контактные данные, такие как отображаемое имя:
```java
String displayName = contact.getNameInfo().getDisplayName();
String entryIdString = messageInfo.getEntryIdString();
System.out.println("Display Name: " + displayName);
```

### Сохраните контактную информацию на диске в формате vCard

#### Обзор
Сохранение извлеченных контактов на диск позволяет легко обмениваться ими и создавать резервные копии.

#### Шаги

**1. Определить выходной путь**

Укажите путь к файлу, в котором вы хотите сохранить контакт:
```java
String outputPath = "YOUR_OUTPUT_DIRECTORY/Contacts" + contact.getNameInfo().getDisplayName() + ".vcf";
```

**2. Сохраните контакт в формате vCard**

Использовать `ContactSaveFormat.VCard` чтобы сохранить ваш контакт:
```java
contact.save(outputPath, ContactSaveFormat.VCard);
```
