## Руководство по внедрению

Мы разделим это руководство на три основные части, которые помогут вам эффективно управлять файлами PST.

### Доступ и изменение структуры файла PST

#### Обзор
Эта функция демонстрирует доступ к файлу PST, поиск определенных подпапок и изменение их класса контейнера с помощью Aspose.Email. `FolderInfo` и `PersonalStorage` классы.

#### Шаги по реализации
##### Загрузите PST-файл
Начните с загрузки вашего PST-файла. Это инициализирует `PersonalStorage` объект.
```java
import com.aspose.email.PersonalStorage;
// Загрузите PST-файл
PersonalStorage pst = PersonalStorage.fromFile("YOUR_DOCUMENT_DIRECTORY/SampleContacts.pst");
```
##### Доступ к корневой папке
Откройте корневую папку, чтобы перемещаться по ее подпапкам.
```java
import com.aspose.email.FolderInfo;
// Получить корневую папку из загруженного PST
FolderInfo rootFolder = pst.getRootFolder();
```
##### Найдите подпапку «Контакты».
Используйте `getSubFolder` метод поиска определенной подпапки по имени.
```java
// Откройте подпапку «Контакты» в корневой папке.
FolderInfo contactsFolder = rootFolder.getSubFolder("Contacts");
```
##### Изменить класс контейнера
Измените класс контейнера вашей целевой подпапки. Здесь мы меняем его на "IPF.Note".
```java
// Изменить класс контейнера папки «Контакты»
contactsFolder.changeContainerClass("IPF.Note");
```
##### Распоряжаться ресурсами
Наконец, убедитесь, что вы избавились от `PersonalStorage` возражают против освобождения ресурсов.
```java
// Очистка путем утилизации объекта PST
pst.dispose();
```

1. **Загрузите PST-файл**

4. **Распоряжаться ресурсами**
   - Всегда звоните `dispose` на `PersonalStorage` объект после операций.
