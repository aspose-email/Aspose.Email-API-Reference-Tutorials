## Руководство по внедрению

Мы разобьем руководство на удобные для освоения разделы на основе конкретных особенностей.

### Создать новый PST-файл

Создание нового PST-файла с помощью Aspose.Email — это просто. Эта функция позволяет вам начать новое хранилище для писем и связанных с ними данных.

#### Шаг 1: Настройте путь к каталогу
Укажите, где будет храниться ваш новый PST-файл:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/outlook/";
```

#### Шаг 2: Создайте PST-файл

Используйте `PersonalStorage.create()` метод инициализации нового файла PST. Первый параметр — это путь, а второй указывает версию формата (0 для Unicode).

```java
import com.aspose.email.PersonalStorage;

// Создайте новый экземпляр PersonalStorage
PersonalStorage pst = PersonalStorage.create(dataDir + "newSample_out.pst", 0);
```

### Добавить папку в корень PST

Добавление папок помогает организовать ваши письма в файле PST. В этом разделе показано, как добавить папку на корневом уровне.

#### Шаг 1: Загрузите PST-файл
Предположим, у вас есть существующий или недавно созданный файл PST:

```java
PersonalStorage pst = PersonalStorage.fromFile(dataDir + "newSample_out.pst");
```

#### Шаг 2: Добавьте новую папку
Создайте и добавьте папку с именем `myInbox` на корневом уровне PST.

```java
pst.getRootFolder().addSubFolder("myInbox");
```

### Добавить сообщение в определенную папку в PST

Управление сообщениями электронной почты имеет решающее значение. Вот как можно добавить существующий файл .msg в ваш PST:

#### Шаг 1: Загрузите файлы PST и MSG
Гарантировать `newSample_out.pst` существует с готовой структурой папок.

```java
PersonalStorage pst = PersonalStorage.fromFile(dataDir + "newSample_out.pst");
MapiMessage message = MapiMessage.fromFile(dataDir + "message.msg");
```

#### Шаг 2: Добавьте сообщение в папку
Вставьте свое сообщение электронной почты в указанную папку.

```java
pst.getRootFolder().getSubFolder("myInbox").addMessage(message);
```
