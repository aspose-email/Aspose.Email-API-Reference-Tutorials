## Руководство по внедрению

### Чтение OST-файла

Первая функция, которую мы рассмотрим, — это чтение OST-файла для отображения имен его подпапок.

#### Обзор

Эта функция позволяет вам загрузить файл OST Microsoft Outlook и перечислить все имена подпапок, содержащиеся в нем. Это может быть особенно полезно для задач аудита или миграции данных.

#### Шаги по реализации

**1. Загрузите OST-файл**

Начните с определения пути к вашему OST-файлу и загрузки его с помощью Aspose.Email:

```java
import com.aspose.email.FolderInfo;
import com.aspose.email.FolderInfoCollection;
import com.aspose.email.PersonalStorage;

public class ReadOSTFeature {
    public static void main(String[] args) {
        // Определите путь к OST-файлу
        String strPSTFile = "YOUR_DOCUMENT_DIRECTORY/Sample.ost";
        
        // Загрузите файл Outlook PST (OST)
        PersonalStorage pst = PersonalStorage.fromFile(strPSTFile);
    }
}
```

**2. Извлечение и отображение подпапок**

После загрузки откройте подпапки корневой папки и просмотрите их, чтобы отобразить каждое имя:

```java
// Извлечь подпапки корневой папки
FolderInfoCollection folderInfoCollection = pst.getRootFolder().getSubFolders();

// Перебрать каждую подпапку и отобразить ее имя
for (int i = 0; i < folderInfoCollection.size(); i++) {
    FolderInfo folderInfo = (FolderInfo) folderInfoCollection.get_Item(i);
    System.out.println(folderInfo.getDisplayName());
}
```

#### Конфигурация ключа
- The `fromFile` метод `PersonalStorage` необходим для загрузки файла OST.
- Доступ к подпапкам через `getSubFolders()` позволяет вам взаимодействовать с каждой папкой по отдельности.

### Конвертация OST в PST

Теперь давайте рассмотрим преобразование файла OST в формат PST.

#### Обзор

Эта функция позволяет преобразовывать файлы OST в более универсальный формат PST для различных почтовых клиентов или целей резервного копирования.

#### Шаги по реализации

**1. Определите входные и выходные пути**

Укажите пути для входного OST-файла и выходного PST-файла:

```java
import com.aspose.email.FileFormat;
import com.aspose.email.PersonalStorage;

public class ConvertOSTToPSTFeature {
    public static void main(String[] args) {
        // Определите путь для входных и выходных файлов
        String inputFilePath = "YOUR_DOCUMENT_DIRECTORY/input.ost";
        String outputFilePath = "YOUR_OUTPUT_DIRECTORY/output.pst";
        
        // Загрузите OST-файл
        PersonalStorage ost = PersonalStorage.fromFile(inputFilePath);
    }
}
```

**2. Выполнить преобразование**

Конвертируйте загруженный файл OST в формат PST с помощью `saveAs` метод:

```java
// Сохраните загруженный OST как PST-файл в указанном каталоге.
ost.saveAs(outputFilePath, FileFormat.Pst);
```

#### Конфигурация ключа
- The `FileFormat.Pst` параметр указывает желаемый формат вывода.
- Убедитесь, что ваши каталоги настроены правильно, чтобы избежать ошибок в путях к файлам.
