## Руководство по внедрению

В этом разделе рассматривается отображение гиперссылок с использованием и без использования `href` атрибуты.

### Пользовательская визуализация гиперссылок с помощью Href

#### Обзор
Повысьте безопасность и удобство использования ссылок, включив `href` атрибут в HTML-теле письма. Такой подход сохраняет целостность гиперссылки.

#### Этапы внедрения

##### Шаг 1: Загрузите электронное письмо
Загрузите свое сообщение электронной почты из файла:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String fileName = dataDir + "LinksSample.eml";
MailMessage msg = MailMessage.load(fileName);
```

##### Шаг 2: Отображение гиперссылок с помощью Href
Реализовать `HyperlinkRenderingCallback` для обработки гиперссылок и включения `href` атрибут:

```java
String htmlTextHref = msg.getHtmlBodyText(new HyperlinkRenderingCallback() {
    @Override
    public String invoke(String source) {
        return renderHyperlinkWithHref(source);
    }
});
```

##### Шаг 3: Извлечение и форматирование гиперссылки
Создайте метод для извлечения `href` атрибут и форматирование:

```java
private static String renderHyperlinkWithHref(String source) {
    int start = source.indexOf("href=\"") + "href=\"".length();
    int end = source.indexOf(\"", start);
    String href = source.substring(start, end);

    start = source.indexOf(">") + 1;
    end = source.indexOf("<", start);
    String text = source.substring(start, end);

    return text + "<" + href + ">";
}
```
**Объяснение**: Этот метод идентифицирует и извлекает `href` атрибут из тега гиперссылки. Он создает форматированную строку с текстом ссылки и ее URL.

### Пользовательская визуализация гиперссылок без Href

#### Обзор
Исключить `href` атрибут для повышения безопасности или когда необходимо только отображение текста ссылки.

#### Этапы внедрения

##### Шаг 1: Загрузите электронное письмо
Загрузите ваше сообщение электронной почты:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String fileName = dataDir + "LinksSample.eml";
MailMessage msg = MailMessage.load(fileName);
```

##### Шаг 2: Отображение гиперссылок без Href
Используйте `HyperlinkRenderingCallback` чтобы исключить `href` атрибут:

```java
String htmlTextHrefLess = msg.getHtmlBodyText(new HyperlinkRenderingCallback() {
    @Override
    public String invoke(String source) {
        return renderHyperlinkWithoutHref(source);
    }
});
```

##### Шаг 3: Извлечение и форматирование гиперссылки
Реализовать метод форматирования гиперссылок без `href`:

```java
private static String renderHyperlinkWithoutHref(String source) {
    int start = source.indexOf(">") + 1;
    int end = source.indexOf("<", start);
    return source.substring(start, end);
}
```
**Объяснение**: Этот метод извлекает только видимый текст гиперссылки, исключая `href` атрибут.
