## Руководство по внедрению

### Функция форматирования текста

**Обзор:**
Настройте способ отображения деталей встречи в виде простого текста. Определите конкретные форматы для различных частей встречи, сделав вывод более структурированным и читабельным.

#### Шаг 1: Загрузите данные о приеме

Загрузить данные о назначении встречи из `.ics` файл:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/email/";
Appointment appointment = Appointment.load(dataDir + "test.ics");
```
На этом этапе данные о вашем событии считываются в `Appointment` объект для дальнейшей обработки.

#### Шаг 2: Настройте пользовательские параметры форматирования

Создать и настроить `AppointmentFormattingOptions` чтобы указать, как должна отображаться каждая часть приема:
```java
AppointmentFormattingOptions formattingOptions = new AppointmentFormattingOptions();
formattingOptions.setLocationFormat("Where: {0}");
formattingOptions.setTitleFormat("Subject: {0}");
formattingOptions.setDescriptionFormat("\r\n*~*~*~*~*~*~*~*~*~*\r\n{0}");
```
Здесь каждая строка формата представляет собой шаблон, где `{0}` будут заменены фактическими данными о назначении приема.

#### Шаг 3: Создание и вывод форматированного текста

Создайте форматированное текстовое представление вашей встречи:
```java
String formattedText = appointment.getAppointmentText(formattingOptions);
system.out.println(formattedText);
```
Теперь этот вывод можно использовать в тексте электронных писем или журналах, где предпочтителен простой текст.

### Функция форматирования HTML

**Обзор:**
Создавайте визуально привлекательные, структурированные HTML-представления встреч для веб-страниц или электронных писем с поддержкой HTML.

#### Шаг 1: Загрузите данные о приеме

Как и в случае с форматированием текста, начните с загрузки вашего `.ics` файл:
```java
Appointment appointment = Appointment.load(dataDir + "test.ics");
```

#### Шаг 2: Создание параметров форматирования HTML

Использовать `createAsHtml()` для инициализации параметров вывода HTML:
```java
AppointmentFormattingOptions formattingOptions = AppointmentFormattingOptions.createAsHtml();
formattingOptions.setLocationFormat("<FONT SIZE=2 FACE=\"Arial\"><b>Where: {0}</b></FONT><BR>");
formattingOptions.setTitleFormat("<FONT SIZE=2 FACE=\"Arial\"><b>Subject: {0}</b></FONT><BR>");
formattingOptions.setDescriptionFormat("<P><FONT SIZE=2 FACE=\"Arial\">-----------<br><i>{0}</i></FONT></P>");
```
Такая настройка позволяет использовать расширенное оформление текста с использованием HTML-тегов для улучшения визуального представления сведений о встрече.

#### Шаг 3: Создание и вывод форматированного HTML-кода

Создайте отформатированную HTML-строку:
```java
String formattedHtml = appointment.getAppointmentText(formattingOptions);
system.out.println(formattedHtml);
```
Его можно встраивать непосредственно в веб-страницы или стилизованные шаблоны электронных писем, поддерживающие HTML-контент.
