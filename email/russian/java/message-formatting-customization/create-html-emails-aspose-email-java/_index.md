## Руководство по внедрению

### Настройка HTML-текста электронного письма
Эта функция позволяет вам задать HTML-текст для ваших электронных писем, сделав их визуально привлекательными и более информативными.

#### Создание экземпляра MailMessage

```java
import com.aspose.email.MailMessage;
import com.aspose.email.SaveOptions;

public class FeatureSetHTMLBody {
    public static void main(String[] args) {
        // Создать новый экземпляр MailMessage
        MailMessage message = new MailMessage();
        
        // Установите HTML-содержимое тела вашего электронного письма
        message.setHtmlBody("<html><body>This is the HTML body</body></html>");
```

#### Сохранение электронного письма

```java
        // Определите выходной путь для сохранения электронной почты (замените фактическим каталогом)
        String outputPath = "YOUR_OUTPUT_DIRECTORY/SetHtmlBody_out.eml";

        // Сохраните MailMessage как файл EML, используя параметры сохранения по умолчанию.
        message.save(outputPath, SaveOptions.getDefaultEml());
    }
}
```

**Пояснение параметров:**
- **`setHtmlBody(String htmlContent)`**: Этот метод устанавливает HTML-контент тела письма. Он позволяет включать форматированный текст, ссылки, изображения и другое форматирование.
  
- **`save(String filePath, SaveOptions options)`**: Сохраняет `MailMessage` объект в файл в формате EML, используя указанные параметры сохранения.

### Советы по устранению неполадок
- Убедитесь, что ваш HTML-контент правильно сформирован, чтобы избежать проблем с отображением.
- Проверьте права доступа к выходному каталогу, если возникли ошибки сохранения.
