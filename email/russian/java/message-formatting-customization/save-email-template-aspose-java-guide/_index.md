### Руководство по внедрению

#### Создание и сохранение шаблона электронного письма

**Обзор:** В этом разделе рассматривается создание `MailMessage` экземпляр с данными отправителя, получателя, темы и текста перед сохранением в формате OFT.

**Шаг 1: Создание почтового сообщения**

Начнем с инициализации `MailMessage` объект:

```java
import com.aspose.email.MailMessage;
import com.aspose.email.MsgSaveOptions;

// Инициализируйте новый экземпляр MailMessage
MailMessage eml = new MailMessage("test@from.to", "test@to.com");
eml.setSubject("Test Email Template");
eml.setBody("This is an example email body.");
```

**Шаг 2: Сохранить как OFT**

Чтобы сохранить это сообщение в формате OFT, используйте `MsgSaveOptions`:

```java
// Определить параметры сохранения для формата OFT
MsgSaveOptions saveOptions = SaveOptions.getDefaultOft();

// Сохраните MailMessage в формате OFT
eml.save("output.oft", saveOptions);
```

**Объяснение:** 
- **ПочтаСообщение**: Этот класс инкапсулирует сообщение электронной почты, включая такие данные, как отправитель, получатель, тема и текст.
- **MsgSaveOptions**: Предоставляет возможности сохранения сообщений в различных форматах; здесь мы используем `getDefaultOft()` для указания формата OFT.
