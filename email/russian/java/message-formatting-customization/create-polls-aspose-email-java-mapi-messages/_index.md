## Руководство по внедрению

### Обзор функций: создание опроса с помощью сообщения MAPI

В этом разделе вы узнаете, как создать и настроить опрос в электронном письме с помощью Aspose.Email. `FollowUpManager` сорт.

#### Шаг 1: Настройка каталогов

Определите пути для ваших документов и выходных каталогов:

```java
String dataDir = YOUR_DOCUMENT_DIRECTORY + "outlook/";
```

Заменять `YOUR_DOCUMENT_DIRECTORY` с фактическим путем к вашему каталогу.

#### Шаг 2: Создайте тестовое MAPI-сообщение

Создайте тестовое сообщение, не устанавливая его как черновик. Это служит нам основой для добавления опций опроса:

```java
MapiMessage msg = createTestMessage(false);
```

#### Шаг 3: Инициализация FollowUpOptions и настройка кнопок голосования

Настройте `FollowUpOptions` чтобы включить желаемые кнопки голосования:

```java
FollowUpOptions options = new FollowUpOptions();
options.setVotingButtons("Yes;No;Maybe;Exactly!");
```

Этот шаг позволяет указать несколько вариантов опроса.

#### Шаг 4: Примените к сообщению дополнительные параметры

Прикрепите настроенные параметры последующих действий к вашему сообщению:

```java
FollowUpManager.setOptions(msg, options);
```

Установив эти параметры, вы включите интерактивное голосование в своей электронной почте.

#### Шаг 5: Сохраните расширенное сообщение электронной почты.

Наконец, сохраните сообщение MAPI с возможностями опроса:

```java
msg.save(YOUR_OUTPUT_DIRECTORY + "MapiMsgWithPoll_out.msg");
```

Этот шаг гарантирует, что ваш опрос будет встроен в файл, готовый к распространению или тестированию.

### Вспомогательный метод для создания тестового MAPI-сообщения

Вот как можно создать простое тестовое сообщение, которое будет дополнено опциями опроса:

```java
private static MapiMessage createTestMessage(boolean draft) {
    return new MapiMessage("from@test.com", "to@test.com",
                           "Polling Feature in Aspose.Email", "This is a test message.");
}
```
