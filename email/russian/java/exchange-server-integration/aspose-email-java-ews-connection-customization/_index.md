### Руководство по внедрению

#### Подключение к веб-службе Exchange (EWS)
**Обзор**: Установка соединения с EWS — это ваш первый шаг в использовании возможностей Aspose.Email for Java. Это позволяет вам выполнять различные операции с вашим почтовым ящиком, такие как чтение сообщений или отправка писем.

##### Шаг 1: Создание экземпляра IEWSClient
```java
import com.aspose.email.EWSClient;
import com.aspose.email.IEWSClient;

public class ConnectToExchange {
    public static void main(String[] args) {
        // Создайте экземпляр для подключения к серверу EWS.
        // Параметры: URL, имя пользователя, пароль.
        IEWSClient client = EWSClient.getEWSClient("exchange.domain.com/exchangeews/Exchange.asmx", "username", "password", "");
    }
}
```

- **Параметры**:
  - `URL`: Конечная точка для вашей службы Exchange.
  - `username` и `password`: Учетные данные для аутентификации.

#### Добавление пользовательских заголовков к запросам EWS
**Обзор**: Пользовательские заголовки могут иметь решающее значение для добавления метаданных или управления аспектами запросов, отправляемых на сервер EWS, расширяя такие функциональные возможности, как привязка сообщений.

##### Шаг 2: Добавьте пользовательский заголовок
```java
import com.aspose.email.EWSClient;
import com.aspose.email.IEWSClient;

public class AddCustomHeaders {
    public static void main(String[] args) {
        // Подключитесь к серверу EWS.
        IEWSClient client = EWSClient.getEWSClient("exchange.domain.com/exchangeews/Exchange.asmx", "username", "password", "");
        
        // Добавьте пользовательский заголовок для улучшенной обработки запросов.
        client.addHeader("X-AnchorMailbox", "username@domain.com");
    }
}
```

- **Конфигурация ключа**: `X-AnchorMailbox` Заголовок помогает в сценариях, когда состояние почтового ящика необходимо сохранять между операциями.

#### Список сообщений в папке «Входящие» EWS
**Обзор**: Чтобы взаимодействовать с электронной почтой, сначала вам нужно получить доступ и просмотреть сообщения в почтовом ящике. Эта операция имеет основополагающее значение для любых последующих задач обработки или управления.

##### Шаг 3: Извлечение и составление списка сообщений
```java
import com.aspose.email.EWSClient;
import com.aspose.email.ExchangeMessageInfoCollection;
import com.aspose.email.IEWSClient;

public class ListInboxMessages {
    public static void main(String[] args) {
        // Установите соединение с сервером EWS.
        IEWSClient client = EWSClient.getEWSClient("exchange.domain.com/exchangeews/Exchange.asmx", "username", "password", "");
        
        // Получите URI почтового ящика из информации о почтовом ящике.
        String inboxUri = client.getMailboxInfo().getInboxUri();
        
        // Список всех сообщений в почтовом ящике.
        ExchangeMessageInfoCollection messageInfoCol = client.listMessages(inboxUri);
    }
}
```

- **Конфигурация ключа**: `listMessages` Метод извлекает коллекцию объектов информации о сообщениях из указанного URI папки.
