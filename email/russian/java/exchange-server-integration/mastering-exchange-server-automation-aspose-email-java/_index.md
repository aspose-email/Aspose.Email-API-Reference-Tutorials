## Руководство по внедрению

### Функция 1: Подключение к серверу Exchange

#### Обзор
Подключение к серверу Exchange — первый шаг в программном управлении электронной почтой. В этом разделе вы узнаете, как установить безопасное соединение с помощью Aspose.Email для Java.

##### Пошаговое руководство

**Определить учетные данные**

Начните с определения URI вашего почтового ящика и учетных данных пользователя:

```java
String mailboxUri = "<HOST>";  // Адрес хоста сервера Exchange
String domain = "";
String username = "<USERNAME>";  // Ваше имя пользователя Exchange
String password = "<PASSWORD>";  // Ваш пароль Exchange
```

**Создать объект NetworkCredential**

Используйте определенные учетные данные для создания `NetworkCredential` объект:

```java
NetworkCredential credentials = new NetworkCredential(username, password, domain);
```

**Установить соединение**

Наконец, установите соединение с вашим сервером Exchange, используя `EWSClient`:

```java
IEWSClient client = EWSClient.getEWSClient(mailboxUri, credentials);
System.out.println("Connected successfully.");
```

#### Советы по устранению неполадок

- **Недействительные учетные данные**: Еще раз проверьте свое имя пользователя и пароль.
- **Проблемы с сетью**: Убедитесь, что сетевое соединение стабильно и адрес сервера указан правильно.

### Функция 2: Список сообщений из папки «Входящие»

#### Обзор
Подключившись к серверу Exchange, вы можете извлекать сообщения, хранящиеся в вашем почтовом ящике. Эта функция позволяет вам получать доступ к данным электронной почты программным путем.

##### Пошаговое руководство

**Извлечь входящие сообщения**

Предполагая, что соединение существует, выведите список всех сообщений в почтовом ящике:

```java
ExchangeMessageInfoCollection msgCollection = client.listMessages(client.getMailboxInfo().getInboxUri());
for (ExchangeMessageInfo info : msgCollection) {
    System.out.println("Subject: " + info.getSubject());
}
```

#### Объяснение

- **`listMessages()`**: Этот метод извлекает электронные письма из указанного URI почтового ящика.
- **`ExchangeMessageInfoCollection`**: Коллекция, содержащая информацию о каждом письме.

### Функция 3: Архивация входящих сообщений

#### Обзор
Архивация сообщений помогает управлять вашим почтовым ящиком, перемещая письма в папку архива. Узнайте, как автоматизировать эту задачу с помощью Aspose.Email для Java.

##### Пошаговое руководство

**Архивировать каждое сообщение**

Просмотрите коллекцию сообщений и заархивируйте каждое из них:

```java
for (ExchangeMessageInfo msgInfo : msgCollection) {
    client.archiveItem(client.getMailboxInfo().getInboxUri(), msgInfo.getUniqueUri());
}
System.out.println("All messages archived.");
```

#### Объяснение

- **`archiveItem()`**Перемещает электронное письмо в папку архива, используя его уникальный URI.
