# Как подключиться и управлять контактами Exchange с помощью Aspose.Email для Java

Это всеобъемлющее руководство проведет вас через использование Aspose.Email для Java для подключения и управления контактами на Microsoft Exchange Server. К концу этого руководства вы освоите:

## Настройка Aspose.Email для Java
Для начала включите следующую зависимость Maven в свой `pom.xml` файл:

```xml

    <groupId>com.aspose</groupId>
    <artifactId>aspose-email</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

## Руководство по внедрению

### Подключиться к серверу Exchange
**Обзор:** Подключение к серверу Exchange необходимо для управления электронной почтой и контактами. С Aspose.Email этот процесс становится простым.

#### Шаг 1: Инициализация клиента EWS
```java
// Импорт необходимых классов
import com.aspose.email.EWSClient;
import com.aspose.email.IEWSClient;

// Подключитесь, используя учетные данные
IEWSClient client = EWSClient.getEWSClient("https://exchange.domain.com/exchangeews/Exchange.asmx", "имя пользователя", "пароль", "domain.com");
```
*Объяснение:* The `getEWSClient` Метод подключается к серверу Exchange, используя URL-адрес вашего домена, имя пользователя, пароль и доменное имя.

### Создать подпапку в папке «Контакты»
**Обзор:** Организуйте контакты, создав подпапки в основной папке «Контакты».

#### Шаг 1: Создайте новую папку
```java
// Импорт необходимых классов
import com.aspose.email.ExchangeFolderInfo;
import com.aspose.email.ExchangeFolderType;

// Создайте «myfolder» в разделе «Контакты»
ExchangeFolderInfo folderInfo = client.createFolder("myfolder", ExchangeFolderType.Contact);
```
*Объяснение:* The `createFolder` Метод создает новую папку с именем «myfolder» в каталоге «Контакты».

### Создать MapiContact в подпапке
**Обзор:** Добавьте отдельные контакты в только что созданную подпапку, используя формат MAPI.

#### Шаг 1: Создайте и сохраните MapiContact
```java
// Импорт необходимых классов
import com.aspose.email.MapiContact;

// Создайте и сохраните новый контакт
client.createContact(folderInfo.getUri(), new MapiContact("MapiContact", "foo@gmail.com"));
```
*Объяснение:* The `createContact` метод сохраняет `MapiContact` объект к указанному URI папки.

### Создать контакт Aspose.Email в подпапке
**Обзор:** Используйте класс Contact из Aspose.Email для более расширенного управления контактами.

#### Шаг 1: Инициализация и сохранение контакта
```java
// Импорт необходимых классов
import com.aspose.email.Contact;

// Инициализируйте новый объект Contact
Contact contact = new Contact();
contact.setDisplayName("Contact");

// Сохраните созданный контакт
client.createContact(folderInfo.getUri(), contact);
```
*Объяснение:* The `createContact` метод сохраняет Aspose.Email `Contact` объект в указанной подпапке.

### Список контактов в подпапке
**Обзор:** Получите список всех контактов в определенной подпапке для эффективного управления ими.

#### Шаг 1: Составьте список контактов
```java
// Импорт необходимых классов
import com.aspose.email.MapiContact[];

// Получить все контакты из 'myfolder'
MapiContact[] myfolderContacts = client.listContacts(folderInfo.getUri());
```
*Объяснение:* The `listContacts` метод извлекает массив `MapiContact` объекты хранятся в указанной папке URI.
