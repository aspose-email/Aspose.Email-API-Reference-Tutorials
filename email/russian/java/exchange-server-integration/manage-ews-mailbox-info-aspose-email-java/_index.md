# Управление информацией почтового ящика EWS с помощью Aspose.Email для Java

Освоив эти методы, вы сможете автоматизировать извлечение размеров почтовых ящиков и различных данных URI, таких как входящие, отправленные элементы и черновики.

В этом уроке мы рассмотрим:
- Подключение к веб-службам Exchange с помощью Aspose.Email
- Получение размера почтового ящика в байтах
- Получение подробной информации об URI почтового ящика

Давайте улучшим ваши возможности управления электронной почтой с помощью Java. Перед погружением убедитесь, что вы готовы с предварительными условиями и настройкой среды.

## Руководство по внедрению

### Функция 1: подключение к веб-службам Exchange

Подключение к EWS осуществляется просто с помощью Aspose.Email for Java. Вот как можно установить соединение:

#### Шаг 1: Создайте экземпляр `EWSClient`

```java
IEWSClient client = EWSClient.getEWSClient("https://exchange.name.com/exchangeews/Exchange.asmx/", "имя пользователя", "пароль", "");
```

- **Параметры**:
  - URL: конечная точка для веб-служб Exchange.
  - Имя пользователя и пароль: учетные данные для аутентификации вашего соединения.

### Функция 2: Получение размера почтового ящика из веб-служб Exchange

После подключения получение размера почтового ящика становится простой задачей:

#### Шаг 1: Получите размер почтового ящика в байтах

```java
long mailboxSize = client.getMailboxSize();
System.out.println("Mailbox size (bytes): " + mailboxSize);
```

- **Возвращаемое значение**: Размер почтового ящика, измеряемый в байтах.

### Функция 3: Получение информации о почтовом ящике из веб-служб Exchange

Получение данных URI для различных разделов почтового ящика необходимо для управления рабочими процессами электронной почты:

#### Шаг 1: Извлечение различных данных URI

```java
ExchangeMailboxInfo mailboxInfo = client.getMailboxInfo();
String mailboxUri = mailboxInfo.getMailboxUri();
String inboxUri = mailboxInfo.getInboxUri();
String sentItemsUri = mailboxInfo.getSentItemsUri();
String draftsUri = mailboxInfo.getDraftsUri();

System.out.println("Mailbox URI: " + mailboxUri);
System.out.println("Inbox URI: " + inboxUri);
System.out.println("Sent Items URI: " + sentItemsUri);
System.out.println("Drafts URI: " + draftsUri);
```

- **Возвращаемые значения**: URI для различных разделов почтового ящика.
