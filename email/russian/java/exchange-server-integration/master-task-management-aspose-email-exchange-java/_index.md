```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-email</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

## Руководство по внедрению

### Создать экземпляр клиента Exchange

#### Обзор
Создание экземпляра `ExchangeClient` класс необходим для подключения и взаимодействия с вашим сервером Microsoft Exchange. Это подключение позволяет вам выполнять различные операции, такие как получение задач или установка часовых поясов.

#### Шаги по реализации

##### Шаг 1: Определите учетные данные
Определите необходимые учетные данные для доступа к вашему серверу Exchange:

```java
String serverUrl = "https://outlook.office365.com/exchangeews/exchange.asmx";
String username = "testUser";
String password = "pwd";
String domain = "domain";
```

##### Шаг 2: Установите соединение
Используйте эти учетные данные для создания экземпляра `IEWSClient` сорт:

```java
IEWSClient client = EWSClient.getEWSClient(serverUrl, username, password, domain);
```

На этом этапе инициализируется ваше соединение с сервером Exchange, что позволяет выполнять дальнейшие операции.

### Установить часовой пояс для задач

#### Обзор
Установка определенного часового пояса гарантирует, что задачи будут управляться точно на основе местного времени пользователей. Эта функция особенно полезна в глобальных командах, работающих в разных часовых поясах.

#### Шаги по реализации

##### Шаг 1: Создание экземпляра IEWSClient
Предполагая, что вы уже создали `IEWSClient` например, приступим к установке часового пояса:

```java
client.setTimezoneId("Central Europe Standard Time");
```

На этом этапе ваши задачи Exchange настраиваются в соответствии с указанным часовым поясом.

### Извлечение задач с определенными статусами

#### Обзор
Извлечение задач на основе их статуса помогает эффективно фильтровать и управлять ими. Эта функция жизненно важна для отслеживания хода выполнения задач в команде.

#### Шаги по реализации

##### Шаг 1: Определите статусы задач
Определите, какие статусы вы хотите фильтровать:

```java
Integer[] statuses = new Integer[]{
    ExchangeTaskStatus.Completed,
    ExchangeTaskStatus.Deferred,
    ExchangeTaskStatus.InProgress,
    ExchangeTaskStatus.NotStarted,
    ExchangeTaskStatus.WaitingOnOthers
};
```

##### Шаг 2: Задачи запроса и фильтрации
Постройте запрос для фильтрации задач на основе определенных статусов:

```java
for (int status : statuses) {
    ExchangeQueryBuilder queryBuilder = new ExchangeQueryBuilder();
    queryBuilder.getTaskStatus().equals(status);
    MailQuery query = queryBuilder.getQuery();
    
    // Получить отфильтрованные задачи
    ExchangeMessageInfoCollection messageInfoCol = client.listMessages(client.getMailboxInfo().getTasksUri(), query);

    if (!messageInfoCol.isEmpty()) {
        ExchangeTask fetchedTask = client.fetchTask(messageInfoCol.get_Item(0).getUniqueUri());
    }
}
```

Такая реализация позволяет эффективно находить задачи, соответствующие определенным критериям.

### Извлечение задач с несколькими критериями

#### Обзор
Объединение нескольких условий в логике поиска задач может дать более точные результаты. Эта возможность имеет важное значение для сложных рабочих процессов, требующих детальной фильтрации.

#### Шаги по реализации

##### Шаг 1: Определите несколько статусов
Установите критерии на основе различных статусов:

```java
Integer[] selectedStatuses = new Integer[]{
    ExchangeTaskStatus.Completed,
    ExchangeTaskStatus.InProgress
};
```

##### Шаг 2: Создание запросов для фильтрации
Используйте эти условия для построения ваших запросов:

```java
ExchangeQueryBuilder queryBuilder = new ExchangeQueryBuilder();
queryBuilder.getTaskStatus().in(Arrays.asList(selectedStatuses));
MailQuery query = queryBuilder.getQuery();

// Извлечение задач, соответствующих указанным статусам
ExchangeMessageInfoCollection messageInfoCol = client.listMessages(client.getMailboxInfo().getTasksUri(), query);
```

Реализация этих запросов позволяет осуществлять комплексное управление задачами на основе сложных условий.
