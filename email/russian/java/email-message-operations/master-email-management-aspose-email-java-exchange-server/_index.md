## Руководство по внедрению

### Подключение к серверу Exchange

#### Обзор
Подключение к серверу Exchange необходимо для программного управления электронной почтой. Эта функция позволяет устанавливать соединение с помощью EWS (Exchange Web Services).

#### Шаги
**Шаг 1**: Импортировать необходимые классы.
```java
import com.aspose.email.EWSClient;
import com.aspose.email.IEWSClient;
```

**Шаг 2**: Создать экземпляр `IEWSClient`.
```java
IEWSClient client = EWSClient.getEWSClient("exchange.domain.com", "username", "password");
```
- **Параметры**: URL-адрес сервера, имя пользователя и пароль.

**Шаг 3**: После завершения работы освободите ресурсы клиента.
```java
if (client != null) {
    client.dispose();
}
```

### Создание и добавление сообщений электронной почты

#### Обзор
Эта функция демонстрирует, как создавать сообщения электронной почты с уникальными темами и добавлять их на сервер Exchange. Она также иллюстрирует сбор URI для будущего использования.

#### Шаги
**Шаг 1**: Установить соединение.
```java
IEWSClient client = EWSClient.getEWSClient("exchange.domain.com", "username", "password");
```

**Шаг 2**: Создание и добавление сообщений в цикле.
```java
List<String> ids = new ArrayList<>();
for (int i = 0; i < 5; i++) {
    MailMessage message = new MailMessage(
        "from@domain.com",
        "to@domain.com",
        "EMAILNET-35033 - " + UUID.randomUUID().toString(),
        "EMAILNET-35033 Messages saved from Sent Items folder doesn't contain 'To' field"
    );
    
    String uri = client.appendMessage(message);
    ids.add(uri);
}
```

**Шаг 3**: Утилизируйте клиента.
```java
if (client != null) {
    client.dispose();
}
```

### Список сообщений с сервера Exchange

#### Обзор
Извлекайте и отображайте сообщения, используя их URI. Эта функция позволяет вам управлять определенными письмами по идентификатору, предоставляя подробную информацию о вашем почтовом ящике.

#### Шаги
**Шаг 1**: Подключитесь к серверу.
```java
IEWSClient client = EWSClient.getEWSClient("exchange.domain.com", "username", "password");
```

**Шаг 2**: Извлечение и отображение сообщений с использованием их идентификаторов.
```java
List<String> ids = new ArrayList<>();
ExchangeMessageInfoCollection messageInfoCol = client.listMessages(ids);

for (var messageInfo : messageInfoCol) {
    System.out.println("Subject: " + messageInfo.getSubject());
}
```

**Шаг 3**: Утилизируйте клиента.
```java
if (client != null) {
    client.dispose();
}
```
