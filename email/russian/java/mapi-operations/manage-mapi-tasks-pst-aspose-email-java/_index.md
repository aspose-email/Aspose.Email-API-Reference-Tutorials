# Управление задачами MAPI в PST с помощью Aspose.Email для Java

- **Управление задачами в PST**: Эффективно организуйте и управляйте своими задачами с помощью файла хранилища.

## Руководство по внедрению

### Создание и настройка задачи MAPI

Создание подробных задач становится простым с Aspose.Email. Выполните следующие шаги:

#### Шаг 1: Инициализация новой задачи MAPI
Создайте задачу, указав ее тему, описание, дату начала и срок выполнения.

```java
MapiTask task = new MapiTask("To Do", "Just click and type to add new task", new Date(), new Date());
```

#### Шаг 2: Установите процент завершения
Укажите, какая часть задачи выполнена, используя процент.

```java
task.setPercentComplete(20);
```

#### Шаг 3: Оцените усилия в минутах
Определите как предполагаемое, так и фактическое время, затраченное на задачу.

```java
task.setEstimatedEffort(2000); // Расчетное усилие в минутах
task.setActualEffort(20);       // Фактические усилия в минутах
```

#### Шаг 4: Определите историю задач
Используйте историю для указания статуса задачи, например, назначена.

```java
task.setHistory(MapiTaskHistory.Assigned);
```

#### Шаг 5: Обновите дату последнего изменения.
Отслеживайте, когда задача была последний раз изменена.

```java
task.setLastUpdate(new Date());
```

#### Шаг 6: Настройте информацию о пользователе
Установите данные пользователя для владения и делегирования задач.

```java
task.getUsers().setOwner("Darius");
task.getUsers().setLastAssigner("Harkness");
task.getUsers().setLastDelegate("Harkness");
task.getUsers().setOwnership(MapiTaskOwnership.AssignersCopy);
```

### Создание и настройка файла персонального хранилища (PST)

#### Шаг 1: Определите выходной путь
Укажите, где будет храниться ваш PST-файл.

```java
String outputPath = "YOUR_OUTPUT_DIRECTORY" + "/TaskPST_out.pst";
```

#### Шаг 2: Создайте PST-файл
Создайте новый PST-файл с поддержкой Unicode для лучшей совместимости.

```java
PersonalStorage pst = PersonalStorage.create(outputPath, FileFormatVersion.Unicode);
```

### Создание и управление папкой задач в PST
Организуйте задачи, создав специальные папки в файле PST.

#### Шаг 1: Повторное использование кода создания PST
Чтобы создать PST-файл, выполните предыдущие шаги.

#### Шаг 2: Создайте папку «Задачи»
Создайте предопределенную папку для управления задачами.

```java
FolderInfo taskFolder = pst.createPredefinedFolder("Tasks", StandardIpmFolder.Tasks);
```

#### Шаг 3: Добавьте задачу MAPI в папку
Вставьте настроенные вами задачи в эту новую папку.

```java
taskFolder.addMapiMessageItem(task);
```
