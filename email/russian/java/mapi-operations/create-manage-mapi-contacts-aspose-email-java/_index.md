## Руководство по внедрению

Мы рассмотрим ключевые особенности управления контактами MAPI:

### Создание и сохранение контакта MAPI

#### Обзор

Создайте новый контакт MAPI, заполните его такими данными, как имена, адреса, адреса электронной почты и фотографии, затем сохраните его как файл MSG.

#### Этапы внедрения

**1. Определите основную информацию о контакте**

Укажите основную информацию о вашем контакте:

```java
MapiContact contact = new MapiContact("Sebastian Wright", "SebastianWright@dayrep.com");
```

**2. Задайте имя и профессиональные свойства**

Настройте дополнительные данные имени и профессиональную информацию:

```java
contact.setNameInfo(new MapiContactNamePropertySet("Bertha", "A.", "Buell"));
contact.setProfessionalInfo(new MapiContactProfessionalPropertySet("Awthentikz", "Social work assistant"));
```

**3. Добавьте личные данные, адрес, адрес электронной почты и телефон.**

Дополнительно персонализируйте контакт с помощью URL-адресов, адресов электронной почты и номеров телефонов:

```java
contact.getPersonalInfo().setPersonalHomePage("B2BTies.com");
contact.getPhysicalAddresses().getWorkAddress().setAddress("Im Astenfeld 59 8580 EDELSCHROTT");
contact.getElectronicAddresses().setEmail1(new MapiContactElectronicAddress("Experwas", "SMTP", "BerthaABuell@armyspy.com"));
contact.setTelephones(new MapiContactTelephonePropertySet("06605045265"));
```

**4. Загрузите и установите фотографию**

Загрузите файл изображения в качестве фотографии контакта:

```java
File fi = new File(dataDir + "Desert.jpg");
byte[] fileContent = Files.readAllBytes(fi.toPath());
MapiContactPhoto photo = new MapiContactPhoto(fileContent, MapiContactPhotoImageFormat.Jpeg);
contact.setPhoto(photo);
```

**5. Сохраните контакт как файл MSG.**

Сохраните свой контакт в указанном каталоге:

```java
contact.save("YOUR_OUTPUT_DIRECTORY/Contact_out.msg", ContactSaveFormat.Msg);
```

### Загрузка контакта из MSG

#### Обзор

Загрузите контакт MAPI из существующего файла MSG.

#### Этапы внедрения

**1. Загрузите MapiMessage**

Загрузите файл сообщения как `MapiMessage` объект:

```java
MapiMessage msg = MapiMessage.fromFile(dataDir + "messageMapi.msg");
```

**2. Преобразовать в объект контакта MAPI**

Преобразовать его в `MapiContact` для дальнейшей обработки или отображения:

```java
MapiContact mapiContact = (MapiContact) msg.toMapiMessageItem();
```

### Загрузка контакта из VCard

#### Обзор

Загрузите контакт непосредственно из файла VCF.

#### Этапы внедрения

**1. Загрузите контакт MAPI**

Загрузите контакт, используя его представление VCard:

```java
MapiContact mapiContact = MapiContact.fromVCard(dataDir + "microsoft.vcf");
```

### Загрузка контакта VCard с указанной кодировкой

#### Обзор

Узнайте, как загрузить контакт из файла VCF, используя указанную кодировку символов.

#### Этапы внедрения

**1. Загрузите контакт, используя кодировку UTF-8**

Укажите кодировку для правильной интерпретации символов:

```java
MapiContact contactReadFromFile = MapiContact.fromVCard(dataDir + "microsoft.vcf", StandardCharsets.UTF_8);
```
