### Базовая инициализация и настройка

Настройте учетные данные сервера Exchange для инициализации Aspose.Email для Java. Настройте `IEWSClient` следующее:

```java
String mailboxUri = "YOUR_EXCHANGE_SERVER_URI"; // URI вашего сервера Exchange
String username = "YOUR_USERNAME";               // Имя пользователя для аутентификации
String password = "YOUR_PASSWORD";               // Пароль

```

Это установит соединение с вашим сервером Exchange с использованием библиотеки Aspose.Email.

## Руководство по внедрению

### Фильтрация назначений по дате

Основная функция этого руководства — фильтрация встреч между определенными датами. Вот как этого можно добиться:

#### Шаг 1: Настройка форматов даты

Начните с настройки `SimpleDateFormat` объект для разбора строк даты в Java `Date` объекты.

```java
import java.text.ParseException;
import java.text.SimpleDateFormat;

SimpleDateFormat sdf = new SimpleDateFormat("dd/MM/yyyy HH:mm:ss");
```

Этот формат будет использоваться для интерпретации дат начала и окончания ваших встреч.

#### Шаг 2: Создание запроса с помощью ExchangeQueryBuilder

Создать экземпляр `ExchangeQueryBuilder` и настройте критерии диапазона дат:

```java
import com.aspose.email.ExchangeQueryBuilder;

ExchangeQueryBuilder builder = new ExchangeQueryBuilder();

// Укажите дату начала фильтрации встреч
builder.getAppointment().getStart().since(sdf.parse("10/05/2016 10:00:00"));

// Определите конечную дату, чтобы включить все встречи, которые были до этого времени или совпадали с ним.
builder.getAppointment().getEnd().beforeOrEqual(sdf.parse("10/15/2016 10:00:00"));
```

#### Шаг 3: Выполнение запроса

Используйте `IEWSClient` экземпляр для выполнения вашего запроса и получения назначений:

```java
import com.aspose.email.MailQuery;

com.aspose.email.MailQuery query = builder.getQuery();
Appointment[] appointments = client.listAppointments(query);
```

Это позволит получить все встречи в указанном диапазоне дат.

### Советы по устранению неполадок
- **Ошибки анализа даты**: Убедитесь, что строки даты соответствуют формату, определенному в `SimpleDateFormat`.
- **Проблемы с аутентификацией**: Еще раз проверьте учетные данные сервера Exchange и сетевое подключение.
- **Результаты запроса пусты**: Убедитесь, что на сервере есть фактические встречи в указанном диапазоне дат.
