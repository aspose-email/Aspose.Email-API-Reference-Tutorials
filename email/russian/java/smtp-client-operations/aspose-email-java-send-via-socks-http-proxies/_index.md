## Руководство по внедрению

### Отправка электронной почты через SOCKS-прокси

#### Обзор
Отправка писем через прокси-сервер SOCKS может повысить безопасность и разрешить доступ из ограниченных сетей. Вот как настроить SMTP-клиент с помощью Aspose.Email с прокси-сервером SOCKS:

##### Шаг 1: Настройка SMTP-клиента

Начните с настройки SMTP-клиента с необходимыми учетными данными и указанием параметров безопасности.

```java
import com.aspose.email.SmtpClient;
import com.aspose.email.SecurityOptions;
import com.aspose.email.SocksProxy;
import com.aspose.email.SocksVersion;
import com.aspose.email.MailMessage;

SmtpClient client = new SmtpClient("smtp.gmail.com", 587, "username", "aspose1234");
client.setSecurityOptions(SecurityOptions.Auto);
```

##### Шаг 2: Настройка прокси-сервера SOCKS

Определите настройки прокси-сервера с помощью протокола SOCKS. Убедитесь, что вы заменили `"proxy.example.com"` с вашим реальным адресом прокси.

```java
String proxyAddress = "proxy.example.com"; // Замените на реальный адрес прокси-сервера.
int proxyPort = 1080; // Стандартный порт для SOCKS-прокси.
SocksProxy proxy = new SocksProxy(proxyAddress, proxyPort, SocksVersion.SocksV5);

client.setProxy(proxy);
```

##### Шаг 3: Отправьте электронное письмо

Настроив SMTP-клиент, вы теперь можете отправлять электронную почту через прокси-сервер SOCKS.

```java
client.send(new MailMessage("sender@domain.com", "receiver@domain.com",
        "Sending Email via SOCKS Proxy",
        "Implement socks proxy protocol for versions 4, 4a, 5 (only Username/Password authentication)"));
```

### Отправка электронной почты через HTTP-прокси

#### Обзор
HTTP-прокси — еще один способ маршрутизации вашего SMTP-трафика. Они особенно полезны, когда вам нужно регистрировать или изменять запросы.

##### Шаг 1: Настройка SMTP-клиента

Как и в случае с SOCKS, начните с настройки SMTP-клиента:

```java
SmtpClient client = new SmtpClient("smtp.gmail.com", 587, "username", "aspose1234");
```

##### Шаг 2: Определите настройки HTTP-прокси

Настройте параметры HTTP-прокси. Заменить `"proxy.example.com"` и `8080` с вашим реальным адресом прокси-сервера и портом.

```java
import com.aspose.email.HttpProxy;

HttpProxy httpProxy = new HttpProxy("proxy.example.com", 8080);
client.setProxy(httpProxy);
```

##### Шаг 3: Отправьте электронное письмо

Наконец, отправьте электронное письмо через настроенный HTTP-прокси:

```java
client.send(new MailMessage(
    "from@domain.com",
    "to@domain.com",
    "Sending Email via HTTP Proxy",
    "Aspose.Email lets you send emails via Http Proxy."));
```
