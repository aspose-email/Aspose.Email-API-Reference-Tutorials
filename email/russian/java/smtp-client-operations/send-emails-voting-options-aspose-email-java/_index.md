## Руководство по внедрению

### Установить соединение с клиентом EWS
Для отправки электронных писем через Microsoft Exchange подключитесь к серверу Exchange Web Services (EWS).

#### Обзор
В этом разделе показано, как установить соединение с помощью Aspose.Email, указав учетные данные и URL-адрес службы.

#### Этапы внедрения
1. **Импортировать необходимые классы**
   ```java
   import com.aspose.email.EWSClient;
   import com.aspose.email.IEWSClient;
   ```
2. **Установить связь**
   ```java
   IEWSClient client = EWSClient.getEWSClient(
       "https://exchange.aspose.com/exchangeews/Exchange.asmx",
       "username",
       "password",
       "aspose.com"
   );
   ```
   - Заменять `"username"` и `"password"` с вашими реальными учетными данными.
   - URL-адрес указывает на конечную точку EWS.

### Создание и настройка MailMessage
Создание почтового сообщения с Aspose.Email просто. Вы можете легко определить отправителя, получателя, тему и текст сообщения.

#### Обзор
В этом разделе рассматривается построение `MailMessage` объект с основными компонентами электронной почты.

#### Этапы внедрения
1. **Импортировать класс**
   ```java
   import com.aspose.email.MailMessage;
   ```
2. **Создать экземпляр MailMessage**
   ```java
   String address = "firstname.lastname@aspose.com";
   MailMessage message = new MailMessage(
       address,  // Отправитель
       address,  // Получатель
       "Flagged Message",  // Предмет
       "Make it concise and descriptive. The description may appear in search engines' search results pages..."
   );
   ```

### Настройте параметры голосования для MailMessage
Улучшите свои электронные письма, добавив опции голосования, чтобы быстро получать обратную связь от получателей.

#### Обзор
Эта функция позволяет добавлять кнопки голосования на `MailMessage`.

#### Этапы внедрения
1. **Импортировать FollowUpOptions**
   ```java
   import com.aspose.email.FollowUpOptions;
   ```
2. **Установить кнопки голосования**
   ```java
   FollowUpOptions options = new FollowUpOptions();
   options.setVotingButtons("Yes;No;Maybe;Exactly!");
   ```

### Отправить почтовое сообщение с вариантами голосования
Объедините все функции для отправки почтового сообщения, снабженного кнопками голосования, через EWS.

#### Обзор
На этом последнем шаге настроенное вами сообщение электронной почты отправляется с использованием установленного соединения EWS.

#### Этапы внедрения
1. **Установить соединение с клиентом EWS** (повторяется для контекста)
   ```java
   IEWSClient client = EWSClient.getEWSClient(
       "https://exchange.aspose.com/exchangeews/Exchange.asmx",
       "username",
       "password",
       "aspose.com"
   );
   ```
2. **Создание и настройка MailMessage** (повторяется для контекста)
   ```java
   String address = "firstname.lastname@aspose.com";
   MailMessage message = new MailMessage(
       address,
       address,
       "Flagged Message",
       "Make it concise and descriptive..."
   );
   ```
3. **Настроить параметры голосования**
   ```java
   FollowUpOptions options = new FollowUpOptions();
   options.setVotingButtons("Yes;No;Maybe;Exactly!");
   ```
4. **Отправить электронное письмо**
   ```java
   client.send(message, options);
   ```
