## Руководство по внедрению
В этом разделе рассматриваются две основные задачи: настройка SMTP-клиента и пересылка сообщений электронной почты.

### Настройка SMTP-клиента с помощью Aspose.Email
#### Обзор
Настройка SMTP-клиента включает в себя настройку данных сервера и параметров безопасности для безопасной отправки электронной почты.

##### 1. Создайте экземпляр SmtpClient
Начните с создания нового экземпляра `SmtpClient`.

```java
import com.aspose.email.SmtpClient;
import com.aspose.email.SecurityOptions;

SmtpClient client = new SmtpClient();
```

##### 2. Установите данные хост-сервера и аутентификации
Определите хост SMTP-сервера, имя пользователя и пароль для аутентификации.

```java
client.setHost("mail.server.com");
client.setUsername("username");
client.setPassword("password");
```

##### 3. Укажите порт и параметры безопасности
Выберите подходящий порт и настройте параметры безопасности, чтобы обеспечить зашифрованную связь.

```java
client.setPort(587); // Общие порты: 587 для TLS/STARTTLS, 465 для SSL
client.setSecurityOptions(SecurityOptions.SSLExplicit);
```

#### Советы по устранению неполадок
- Убедитесь, что данные SMTP-сервера верны.
- Проверьте доступность порта и совместимость настроек безопасности с вашим сервером.

### Загрузка и пересылка сообщения электронной почты
#### Обзор
Узнайте, как загружать существующие сообщения электронной почты из файлов и пересылать их с помощью настроенного SMTP-клиента.

##### 1. Загрузите сообщение электронной почты.
Использовать `MailMessage.load()` чтобы прочитать файл электронной почты в `MailMessage` объект.

```java
import com.aspose.email.MailMessage;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
MailMessage msg = MailMessage.load(dataDir + "test.eml");
```

##### 2. Переслать загруженное электронное письмо
Использовать `SmtpClient.forward()` для отправки электронного письма новым получателям.

```java
client.forward("Recipient1@domain.com", "Recipient2@domain.com", msg);
```

#### Основные параметры конфигурации
- При загрузке писем убедитесь, что пути к файлам указаны правильно.
- Еще раз проверьте правильность адреса получателя.
