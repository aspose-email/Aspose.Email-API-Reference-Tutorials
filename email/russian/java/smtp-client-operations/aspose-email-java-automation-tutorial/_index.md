#### Отправка электронной почты с помощью EWSClient
**Обзор:**
Автоматизируйте отправку электронной почты напрямую через приложение Java с помощью EWSClient от Aspose.Email. Этот метод упрощает подключение к серверу Exchange и отправку электронной почты.

**Шаги:**
1. **Установить соединение:**
   - Использовать `EWSClient.getEWSClient()` для подключения к серверу Exchange с соответствующими учетными данными.
   
   ```java
   IEWSClient client = EWSClient.getEWSClient(
       "https://exchange.aspose.com/exchangeews/Exchange.asmx/", 
       "username", 
       "password"
   );
   ```
2. **Создать сообщение электронной почты:**
   - Построить `MailMessage` объект с отправителем, получателем, темой и телом.
   
   ```java
   MailMessage message = new MailMessage(
       "sender@example.com", 
       "recipient@example.com",
       "Subject Line",
       "Email Body"
   );
   ```
3. **Отправить электронное письмо:**
   - Используйте `client.send()` метод отправки электронного письма.

   ```java
   client.send(message);
   ```

**Советы по устранению неполадок:**
- Убедитесь, что URL-адрес вашего сервера Exchange правильный и доступный.
- Проверьте правильность учетных данных, чтобы избежать ошибок аутентификации.

#### Список сообщений в папке «Входящие»
**Обзор:**
Извлекайте список сообщений из папки «Входящие», что позволяет обрабатывать и управлять электронными письмами программным способом.

**Шаги:**
1. **Подключиться к серверу:**
   - Аналогично этапу отправки электронной почты, установите соединение с помощью `EWSClient.getEWSClient()`.
2. **Список сообщений:**
   - Извлеките коллекцию сообщений из папки «Входящие».
   
   ```java
   ExchangeMessageInfoCollection messageInfoCol = client.listMessages(client.getMailboxInfo().getInboxUri());
   ```

#### Синхронизация элементов папки
**Обзор:**
Синхронизируйте локальные данные с элементами папок на сервере, гарантируя, что у вас всегда будет актуальная информация.

**Шаги:**
1. **Установить соединение и синхронизировать:**
   - Используйте `syncFolder()` метод синхронизации элементов в определенной папке.
   
   ```java
   SyncFolderResult result = client.syncFolder(client.getMailboxInfo().getInboxUri(), null);
   ```
