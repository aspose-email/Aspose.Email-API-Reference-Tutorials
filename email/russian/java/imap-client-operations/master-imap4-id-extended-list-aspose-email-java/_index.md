## Руководство по внедрению
### Поддержка расширения идентификатора IMAP4
Эта функция позволяет идентифицировать вашего клиента на сервере IMAP, обеспечивая индивидуальное взаимодействие на основе возможностей клиента.

#### Обзор
Расширение IMAP4 ID помогает установить двустороннюю линию связи между клиентом и сервером. Представляя идентификатор вашего клиента, серверы могут предоставлять оптимизированные ответы.

#### Этапы внедрения
1. **Инициализировать ImapClient**
   Настройте `ImapClient` с вашими учетными данными и включите параметры безопасности:
   
   ```java
   import com.aspose.email.ImapClient;
   import com.aspose.email.SecurityOptions;

   ImapClient client = new ImapClient("imap.gmail.com", 993, "username", "password");
   client.setSecurityOptions(SecurityOptions.Auto);
   ```

2. **Представьте клиента**
   Получите идентификационную информацию сервера:
   
   ```java
   import com.aspose.email.ImapIdentificationInfo;

   // Получить идентификационные данные сервера с параметрами по умолчанию.
   ImapIdentificationInfo info1 = client.introduceClient();

   // Использовать вводное значение по умолчанию.
   ImapIdentificationInfo info2 = client.introduceClient(ImapIdentificationInfo.getDefaultValue());

   System.out.println("Server Name: " + info1.getName());
   System.out.println("Vendor: " + info1.getVendor());
   System.out.println("Support URL: " + info1.getSupportUrl());
   System.out.println("Version: " + info1.getVersion());
   ```

### Поддержка команд расширенного списка IMAP4
Эта функция проверяет, поддерживается ли команда расширенного списка, и извлекает подробную информацию о папке.

#### Обзор
Команда расширенного списка предоставляет исчерпывающую информацию о папках сервера, включая иерархию и атрибуты, выходящие за рамки основных соглашений об именовании.

#### Этапы внедрения
1. **Проверить расширенный список поддержки**
   Проверьте, поддерживает ли сервер команду расширенного списка:
   
   ```java
   boolean isExtendedListSupported = client.getExtendedListSupported();
   System.out.println("Extended List Supported: " + isExtendedListSupported);
   ```

2. **Получить информацию о папке**
   Используйте `listFolders` Метод получения подробной информации обо всех папках:
   
   ```java
   import com.aspose.email.ImapFolderInfo;
   import com.aspose.email.ImapFolderInfoCollection;

   ImapFolderInfoCollection folderInfoCol = client.listFolders("*");

   for (ImapFolderInfo folderInfo : folderInfoCol) {
       System.out.println("Folder: " + folderInfo.getName() + ", Has Children: " + folderInfo.hasChildren());
   }
   ```
