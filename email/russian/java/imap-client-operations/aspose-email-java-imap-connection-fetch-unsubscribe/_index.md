### Подключиться к IMAP-серверу

#### Обзор
Этот шаг включает настройку и установление безопасного соединения с вашим сервером IMAP с помощью Aspose.Email для Java.

#### Настройте параметры ImapClient
Для начала создайте экземпляр `ImapClient` и задайте необходимые свойства:
```java
// Создать и настроить ImapClient
ImapClient imapClient = new ImapClient();
imapClient.setHost("<HOST>"); // Замените <HOST> на адрес вашего сервера IMAP.
imapClient.setPort(993); // Стандартный порт для IMAP через SSL
imapClient.setUsername("<USERNAME>"); // Замените <USERNAME> на свое имя пользователя.
imapClient.setPassword("<PASSWORD>"); // Замените <ПАРОЛЬ> на свой пароль.
```

#### Установить параметры безопасности
Обеспечьте безопасную связь, настроив протоколы шифрования и параметры безопасности:
```java
// Настройте параметры безопасности
imapClient.setSupportedEncryption(EncryptionProtocols.Tls); // Использовать TLS для шифрования
imapClient.setSecurityOptions(SecurityOptions.SSLImplicit); // Включить неявную безопасность SSL
```

**Почему именно эти конфигурации?**
- **TLS-протокол**: Обеспечивает шифрование данных во время передачи, защищая от перехвата.
- **Опция неявной безопасности SSL**: Автоматически проверяет сертификат сервера перед установлением соединения.

#### Советы по устранению неполадок
- Убедитесь, что адрес и учетные данные вашего сервера IMAP верны.
- Убедитесь, что ваш поставщик услуг электронной почты поддерживает TLS.

### Извлечь и отобразить заголовки списка отмены подписки

#### Обзор
Эта функция позволяет извлекать электронные письма с сервера IMAP и извлекать заголовки «List-Unsubscribe», которые могут быть полезны для управления подписками на электронную почту непосредственно из вашего приложения.

#### Извлечь сообщения
Использовать `ImapClient` для получения информации о сообщении:
```java
// Извлечение сообщений с сервера IMAP
ImapMessageInfoCollection messageInfoCol = imapClient.listMessages();
```

#### Извлечение и отображение заголовков
Просмотрите каждое электронное письмо и выведите заголовок «List-Unsubscribe», если он доступен:
```java
// Распечатать заголовки «Список-Отмена подписки» для каждого письма
for (ImapMessageInfo imapMessageInfo : messageInfoCol) {
    System.out.println("ListUnsubscribe Header: " + imapMessageInfo.getListUnsubscribe());
}
```

**Зачем извлекать эти заголовки?**
- Управление подписками непосредственно из почтового ящика может быть эффективным и автоматизированным, что экономит ваше время.
