## Настройка Aspose.Email для Java

Чтобы начать работу с Aspose.Email, интегрируйте его в свой проект с помощью Maven:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-email</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Базовая инициализация

После установки инициализируйте свой проект, импортировав необходимые классы и настроив свой клиент IMAP. Вот базовая настройка для подключения к серверу IMAP на примере Gmail:

```java
import com.aspose.email.ImapClient;
import com.aspose.email.SecurityOptions;

ImapClient client = new ImapClient();
client.setHost("imap.gmail.com"); // Установите хост сервера IMAP
client.setPort(993);               // Установите номер порта для SSL-соединения
client.setUsername("username");    // Укажите свое имя пользователя для аутентификации
client.setPassword("password");    // Укажите свой пароль для аутентификации
client.setSecurityOptions(SecurityOptions.Auto); // Автоматически выбирать параметр безопасности
```

## Руководство по внедрению

### Установление IMAP-соединения

**Обзор:**
Подключение к серверу IMAP позволяет вам получать доступ и управлять электронными письмами, хранящимися на удаленном сервере. Это важно для приложений, которым необходимо читать, отправлять или организовывать электронные письма.

#### Пошаговое руководство:
1. **Инициализируйте ImapClient:**
   - Создайте новый экземпляр `ImapClient`.
   - Задайте хост, порт, имя пользователя, пароль и параметры безопасности, как показано выше.
2. **Параметры безопасности:**
   - The `SecurityOptions.Auto` настройка автоматически выбирает SSL или TLS в зависимости от поддержки сервера.

### Список папок IMAP

**Обзор:**
Список папок поможет вам понять структуру вашего аккаунта электронной почты и получить доступ к определенным данным в каждой папке.

#### Пошаговое руководство:
1. **Подключитесь к своей учетной записи:**
   - Используйте `ImapClient` настройка, как описано ранее.
2. **Получить информацию о папке:**
   - Получить коллекцию всех папок, используя `listFolders()` метод.
3. **Перебрать папки:**
   ```java
   import com.aspose.email.ImapFolderInfoCollection;
   import com.aspose.email.ImapFolderInfo;

   ImapFolderInfoCollection folderInfoColl = client.listFolders();

   for (ImapFolderInfo folderInfo : folderInfoColl) {
       String folderName = folderInfo.getName();
       int newMessageCount = folderInfo.getNewMessageCount();
       boolean isReadOnly = folderInfo.getReadOnly();
       int totalMessages = folderInfo.getTotalMessageCount();

       // Пример вывода
       System.out.println("Folder: " + folderName);
       System.out.println("Unread Messages: " + newMessageCount);
   }
   ```
4. **Понимание свойств папки:**
   - `getName()`: Возвращает имя папки.
   - `getNewMessageCount()`: Подсчет непрочитанных сообщений в папке.
   - `getReadOnly()`: Проверяет, доступна ли папка только для чтения.
   - `getTotalMessageCount()`: Показывает общее количество сообщений.

### Советы по устранению неполадок

- **Проблемы аутентификации:** Убедитесь, что имя пользователя и пароль верны. Включите доступ для менее защищенных приложений, если используете Gmail.
- **Ошибки подключения:** Проверьте адрес хоста и номер порта. Проверьте настройки брандмауэра, которые могут блокировать соединения IMAP.
