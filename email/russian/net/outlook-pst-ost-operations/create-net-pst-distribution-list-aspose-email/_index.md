## Руководство по внедрению
### Создание списков рассылки из существующих контактов
Автоматизируйте создание списков рассылки в файле PST с использованием существующих контактов. Вот как:

#### Определите отображаемые имена и адреса электронной почты для контактов
Настройте свои контактные данные, указав отображаемые имена и адреса электронной почты:
```csharp
cstring displayName1 = "Sebastian Wright";
cstring email1 = "SebastianWright@dayrep.com";
cstring displayName2 = "Wichert Kroos";
cstring email2 = "WichertKroos@teleworm.us";
```

#### Получите идентификаторы входа для контактов
Вам необходимо получить идентификаторы записи для ваших контактов:
```csharp
cstring strEntryId1;
cstring strEntryId2;
```

#### Определить выходной путь для файла PST
Укажите, где будет сохранен файл PST, и проверьте, существует ли он уже. Если да, удалите его, чтобы избежать конфликтов:
```csharp
cstring path = dataDir + "CreateDistributionListInPST_out.pst";

if (File.Exists(path))
{
    File.Delete(path);
}
```

#### Создать PST и добавить контакты
Вот как создать новый PST-файл и добавить контакты:
```csharp
using (PersonalStorage pst = PersonalStorage.Create(path, FileFormatVersion.Unicode))
{
    FolderInfo contactFolder = pst.CreatePredefinedFolder("Contacts", StandardIpmFolder.Contacts);
    
    // Добавьте контакты в папку здесь...
}
```
