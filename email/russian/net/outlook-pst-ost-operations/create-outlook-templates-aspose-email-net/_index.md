# Создание шаблонов Outlook с помощью Aspose.Email для .NET

## Что вы узнаете:
- Сохраните файл Outlook MSG как шаблон (формат OFT) с помощью Aspose.Email для .NET

Оптимизируйте свой рабочий процесс с помощью этих мощных функций.

## Руководство по внедрению

### Сохранение файла Outlook MSG как шаблона (формат OFT)

**Обзор:**
Эта функция позволяет сохранять файл Outlook MSG непосредственно в качестве шаблона, упрощая повторяющиеся задачи по созданию электронных писем.

#### Пошаговая реализация:
1. **Создайте объект MapiMessage**
   
   Создать новый `MapiMessage` укажите желаемого отправителя, получателя, тему и текст письма.
   
   ```csharp
   using (MapiMessage mapi = new MapiMessage("test@from.to", "test@to.to", "template subject", "Template body"))
   {
       string oftMapiFileName = System.IO.Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "mapiToOft.msg");
       
       mapi.SaveAsTemplate(oftMapiFileName);
   }
   ```

2. **Параметры и конфигурация:**
   - `SaveAsTemplate` используется для сохранения сообщения в формате OFT, необходимом для создания шаблона.
   - Убедитесь, что вы указали правильный путь к каталогу, в котором будет сохранен файл.

### Загрузка файла MSG в MapiMessage

**Обзор:**
Загрузка существующих файлов MSG в ваше приложение позволяет программно манипулировать данными электронной почты или читать их.

#### Этапы реализации:
1. **Загрузите файл MSG**
   
   Использовать `MapiMessage.FromFile` для загрузки файла MSG в `MapiMessage` объект.
   
   ```csharp
   string msgFilePath = System.IO.Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.msg");
   MapiMessage loadedMsg = MapiMessage.FromFile(msgFilePath);
   ```

2. **Доступ к свойствам сообщения**
   
   После загрузки получите доступ к различным свойствам, таким как тема и тело.
   
   ```csharp
   Console.WriteLine("Subject: " + loadedMsg.Subject);
   Console.WriteLine("Body: " + loadedMsg.Body);
   ```
