### Загрузка PST-файла
#### Обзор
Загрузка файла PST — это ваш первый шаг в обработке данных Outlook. Эта операция настраивает среду для чтения и обработки содержимого электронной почты, хранящегося в файле.
#### Этапы внедрения
1. **Инициализировать PersonalStorage**:
   ```csharp
   using Aspose.Email.Storage.Pst;
   
   string dataDir = "YOUR_DOCUMENT_DIRECTORY\PersonalStorage.pst";
   using (PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir))
   {
       // Файл PST теперь загружен и может использоваться для дальнейших операций.
   }
   ```
   - **Параметры**: `dataDir` должен указывать на каталог, содержащий ваш PST-файл.
   - **Цель**: Этот шаг создает экземпляр `PersonalStorage`, представляющий весь PST.

Навигация по подпапкам позволяет организовывать и получать доступ к электронным письмам, хранящимся в разных разделах файла PST. 
#### Этапы внедрения
1. **Доступ к подпапкам корневой папки**:
   ```csharp

   using (PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir))
   {
       foreach (FolderInfo folder in personalStorage.RootFolder.GetSubFolders())
       {
           // Доступ к каждой подпапке осуществляется здесь.
       }
   }
   ```
   - **Цель**: Этот код перечисляет все подпапки в корне, позволяя выполнять целевые операции с определенными категориями электронной почты или папками проектов.
### Перечисление сообщений в папке
#### Обзор
Получив доступ к папке, просмотрите сообщения, выполнив такие задачи, как фильтрация писем по отправителю, теме и т. д. 
#### Этапы внедрения
1. **Итерация сообщений**:
   ```csharp

   string dataDir = "YOUR_DOCUMENT_DIRECTORY\PersonalStorage.pst";
   using (PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir))
   {

       {
           foreach (MessageInfo msg in folder.EnumerateMessages())
           {
               // Здесь можно получить доступ к каждому сообщению в подпапке.
           }
       }
   }
   ```
   - **Цель**: Этот сегмент позволяет вам просматривать каждое электронное письмо, что дает возможность читать или изменять его содержимое.
