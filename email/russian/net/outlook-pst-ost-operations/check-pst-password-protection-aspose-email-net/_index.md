### Проверить функцию защиты паролем
Эта функция помогает определить, установлен ли пароль для PST-файла, обеспечивая безопасную обработку данных.

#### Пошаговая реализация

**1. Откройте PST-файл.**
Откройте ваш PST-файл из его каталога:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (PersonalStorage pst = PersonalStorage.FromFile(dataDir + "/passwordprotectedPST.pst"))
{
    bool isProtected = IsPasswordProtected(pst);
}
```

**2. Определите статус защиты паролем**
The `IsPasswordProtected` Метод проверяет наличие свойства пароля:

```csharp
private static bool IsPasswordProtected(PersonalStorage pst)
{
    // Проверьте, установлен ли для PST-файла параметр пароля.
    if (pst.Store.Properties.ContainsKey(MapiPropertyTag.PR_PST_PASSWORD))
    {
        long passwordHash = pst.Store.Properties[MapiPropertyTag.PR_PST_PASSWORD].GetLong();
        return passwordHash != 0; // Если хэш не равен нулю, он защищен.
    }
    return false;
}
```

- **Параметры:** The `PersonalStorage` экземпляр, представляющий файл PST.
- **Возвращаемое значение:** Логическое значение, указывающее, установлен ли пароль.

**Советы по устранению неполадок:**
- Убедитесь, что путь к PST-файлу указан правильно.
- Убедитесь, что у вас есть права на чтение файла.
