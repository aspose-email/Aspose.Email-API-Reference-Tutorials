## Руководство по внедрению

### Удаление свойства пароля из файла PST

Эта функция позволяет снять защиту паролем, сделав PST доступным без аутентификации.

#### Шаг 1: Загрузите PST-файл
Загрузите ваш PST-файл с помощью Aspose.Email `PersonalStorage` сорт.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/PersonalStorage1.pst";
PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
```

#### Шаг 2: Проверьте и удалите пароль
Проверьте, установлен ли для PST-файла параметр пароля, затем удалите его, установив пустой пароль.

```csharp
if (personalStorage.Store.Properties.ContainsKey(MapiPropertyTag.PR_PST_PASSWORD))
{
    MapiProperty property = new MapiProperty(MapiPropertyTag.PR_PST_PASSWORD, BitConverter.GetBytes((long)0));
    personalStorage.Store.SetProperty(property);
}
```

*Объяснение*: `MapiPropertyTag.PR_PST_PASSWORD` проверяет наличие пароля. Если он присутствует, он заменяется нулевыми байтами для эффективного удаления пароля.

#### Совет по устранению неполадок
- Убедитесь, что у вас есть права на запись в каталог, содержащий PST-файл.
  
### Чтение свойств файла PST

Доступ и отображение основных свойств вашего PST-файла.

#### Шаг 1: Загрузите PST-файл
Начните с загрузки PST-файла, аналогично удалению пароля.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/PersonalStorage1.pst";
PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
```

#### Шаг 2: Доступ и отображение свойств
Получите доступ к таким свойствам, как отображаемое имя, количество элементов и размер, а затем распечатайте их.

```csharp
Console.WriteLine("Display Name: " + personalStorage.DisplayName);
Console.WriteLine("Total Number of Items: " + personalStorage.RootFolder.ContentCount);
Console.WriteLine("Total Size in Bytes: " + personalStorage.RootFolder.SizeInBytes);
```

*Объяснение*: `DisplayName` предоставляет понятное для человека имя, в то время как `ContentCount` и `SizeInBytes` дать представление о содержании файла.
