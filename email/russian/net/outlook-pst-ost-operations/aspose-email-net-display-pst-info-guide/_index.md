## Руководство по внедрению

### Отображение информации о файле PST

В этом разделе вы узнаете, как загрузить PST-файл и отобразить сведения о его папке.

#### Загрузите PST-файл

Укажите путь к вашему PST-файлу и загрузите его с помощью `PersonalStorage.FromFile` метод:
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";
string dst = dataDir + "/PersonalStorage.pst";

// Загрузите PST-файл
PersonalStorage personalStorage = PersonalStorage.FromFile(dst);
```

#### Получить все подпапки

Извлеките все подпапки в корневой папке файла PST:
```csharp
FolderInfoCollection folderInfoCollection = personalStorage.RootFolder.GetSubFolders();
```

#### Итерация и отображение информации о папке

Пройдитесь по каждой папке, чтобы отобразить ее имя, общее количество элементов и количество непрочитанных элементов:
```csharp
foreach (FolderInfo folderInfo in folderInfoCollection)
{
    Console.WriteLine("Folder: " + folderInfo.DisplayName);
    Console.WriteLine("Total items: " + folderInfo.ContentCount);
    Console.WriteLine("Total unread items: " + folderInfo.ContentUnreadCount);
    Console.WriteLine("-----------------------------------");
}
```

**Объяснение:**
- `folderInfo.DisplayName`: Возвращает имя папки.
- `folderInfo.ContentCount`: Показывает общее количество элементов в папке.
- `folderInfo.ContentUnreadCount`: Показывает количество непрочитанных элементов.

### Советы по устранению неполадок

- **Исключение «Файл не найден»**: Убедитесь, что ваш `dataDir` настроен правильно и указывает на существующий файл PST.
