### Создайте файл Outlook PST с помощью Aspose.Email для .NET
#### Обзор
Создание нового файла PST необходимо для начала работы с новыми или архивирования данных. В этом разделе вы узнаете, как создать простой файл Outlook PST с помощью Aspose.Email для .NET.

#### Шаг 1: Определите путь к каталогу
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY"; 
string dst = Path.Combine(dataDir, "PersonalStorage.pst");
```
**Объяснение**: Укажите, где будет сохранен ваш новый файл PST. Убедитесь, что каталог существует, или обработайте создание пути в вашем коде.

#### Шаг 2: Проверьте и удалите существующий файл
```csharp
if (File.Exists(dst))
    File.Delete(dst);
```
**Почему**: Это гарантирует, что вы начнете с нового файла, избегая конфликтов с существующими данными.

#### Шаг 3: Создайте новый PST-файл
```csharp
PersonalStorage pst = PersonalStorage.Create(dst, FileFormatVersion.Unicode);
```
**Параметры**: 
- `dst`: Путь назначения для нового PST.
- `FileFormatVersion.Unicode`: Обеспечивает совместимость и поддерживает символы Unicode.

### Добавить подпапку в существующий файл PST
#### Обзор
Организация вашего PST-файла с подпапками, такими как «Входящие», имеет решающее значение для эффективного управления электронной почтой. В этом разделе показано, как добавить подпапку программным способом.

#### Шаг 1: Откройте существующий файл PST
```csharp
string dst = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "PersonalStorage.pst");
PersonalStorage pst = PersonalStorage.FromFile(dst);
```
**Объяснение**: Получите доступ к файлу PST, который вы создали или уже имеете. Убедитесь, что он доступен и не поврежден.

#### Шаг 2: Добавьте подпапку с именем «Входящие»
```csharp
pst.RootFolder.AddSubFolder("Inbox");
```
**Цель**: создает новую подпапку в корневом каталоге PST, помогая организовать электронные письма по категориям, например «Входящие».
