### Доступ и чтение OLM-файлов
#### Обзор
Доступ к файлам OLM необходим для извлечения данных из устаревших баз данных Outlook. Эта функция демонстрирует, как можно использовать Aspose.Email для эффективного доступа и чтения этих файлов.

**Шаг 1: Настройте путь к файлу**
Сначала укажите путь, где находится ваш OLM-файл. Заменить `@YOUR_DOCUMENT_DIRECTORY` с фактическим каталогом вашего документа:

```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY\\SampleOLM.olm";
```

**Шаг 2: Доступ к OLM-файлу с помощью OlmStorage**
Создать экземпляр `OlmStorage`. Этот объект будет служить нашим шлюзом для доступа и управления файлом OLM.

```csharp
// Создайте экземпляр OlmStorage для доступа к файлу OLM.
OlmStorage storage = new OlmStorage(dataDir);
```

**Шаг 3: Распечатать информацию об иерархии папок**
Теперь, когда у нас есть доступ к файлу OLM, давайте распечатаем иерархию папок и количество сообщений:

```csharp
// Доступ и печать информации об иерархии папок из файла OLM
PrintMessageCount(storage.FolderHierarchy);
```

### Печать количества сообщений в папках OLM
#### Обзор
Эта функция поможет вам понять, сколько сообщений хранится в каждой папке вашего OLM-файла.

**Шаг 1: Пройдитесь по каждой папке**
Для каждого `OlmFolder` объект, выведите его имя и количество содержащихся в нем сообщений:

```csharp
public static void PrintMessageCount(List<OlmFolder> folders)
{
    foreach (OlmFolder folder in folders)
    {
        Console.WriteLine("Message Count [" + folder.Name + "]: " + folder.MessageCount);
    }
}
```

**Объяснение:**
- `folder.Name`: Возвращает имя папки.
- `folder.MessageCount`: Показывает количество сообщений в данной папке.
