### Функция 1: Создание и инициализация файла PST
**Обзор:** В этом разделе вы узнаете, как создать новый PST-файл с помощью библиотеки Aspose.Email, которая необходима для начала управления данными электронной почты с нуля.

#### Пошаговая реализация:
##### **Настройка каталога и проверка существующих файлов**
Сначала определите каталог документов и убедитесь, что если файл PST с нужным именем уже существует, он будет удален во избежание конфликтов.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string pstFilePath = dataDir + "/SetPasswordOnPST_out.pst";

// Убедитесь, что существующие файлы не конфликтуют с новым созданием.
if (File.Exists(pstFilePath))
{
    File.Delete(pstFilePath);
}
```
##### **Создать PST-файл**
Использовать `PersonalStorage.Create` для инициализации нового файла PST. Этот шаг настраивает ваш файл в формате Unicode, который широко совместим.

```csharp
using (PersonalStorage pst = PersonalStorage.Create(pstFilePath, FileFormatVersion.Unicode))
{
    // Теперь файл PST инициализирован и готов к работе.
}
```
### Функция 2: Установка пароля для файла PST
**Обзор:** Узнайте, как защитить вновь созданные или существующие файлы PST, добавив пароль.

#### Пошаговая реализация:
##### **Доступ к существующему файлу PST**
Убедитесь, что у вас есть доступ к файлу PST, затем используйте `FromFile` метод для операций защиты паролем.

```csharp
if (File.Exists(pstFilePath))
{
    using (PersonalStorage pst = PersonalStorage.FromFile(pstFilePath))
    {
        const string password = "Password1";
        
        // Установка пароля для безопасности
        pst.Store.ChangePassword(password);
    }
}
```
### Функция 3: Удаление пароля из файла PST
**Обзор:** Иногда вам может понадобиться удалить пароль из вашего PST-файла. Вот как это сделать.

#### Пошаговая реализация:
##### **Доступ к файлу PST и его изменение**
Чтобы удалить существующий пароль, просто введите `null` в `ChangePassword` метод.

```csharp
if (File.Exists(pstFilePath))
{
    using (PersonalStorage pst = PersonalStorage.FromFile(pstFilePath))
    {
        // Удаление пароля для более легкого доступа
        pst.Store.ChangePassword(null);
    }
}
```
