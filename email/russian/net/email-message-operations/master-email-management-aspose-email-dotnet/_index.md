## Руководство по внедрению

### Импорт файла EML в объект MailMessage

Загрузите файл EML в `MailMessage` объект для дальнейшей обработки.

#### Пошаговое руководство:
**1. Укажите каталог документов**
Определите, где хранятся ваши EML-файлы:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string dstEmail = dataDir + "test.eml";
```
*Почему?*: Это настраивает путь ссылки для поиска файла электронной почты.

**2. Загрузите файл EML**
Использовать `MailMessage.Load` с `EmlLoadOptions`:
```csharp
using Aspose.Email.Mime;

// Загрузите файл EML в объект MailMessage.
MailMessage msg = MailMessage.Load(dstEmail, new EmlLoadOptions());
```
*Почему?*: Преобразует ваш EML-файл в файл, пригодный для обработки `MailMessage` объект.

### Настройка SmtpClient для отправки писем
Настройка SMTP-клиента необходима для отправки писем из вашего приложения.

#### Пошаговое руководство:
**1. Создание и настройка SmtpClient**
Настройте соответствующие параметры сервера:
```csharp
using Aspose.Email.Clients.Smtp;

SmtpClient client = new SmtpClient("smtp.gmail.com");
client.Port = 587;
client.Username = "your-email@gmail.com";
client.Password = "your-password";
client.SecurityOptions = SecurityOptions.Auto;
```
*Почему?*: Гарантирует, что ваше приложение сможет подключаться к SMTP-серверу Gmail и безопасно отправлять электронные письма.
