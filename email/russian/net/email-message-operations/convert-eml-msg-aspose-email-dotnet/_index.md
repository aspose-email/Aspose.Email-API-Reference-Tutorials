## Руководство по внедрению

Давайте разберем процесс на две основные функции: загрузка EML-файла и сохранение его как MSG.

### Функция 1: Загрузка файла EML

#### Обзор
Загрузка существующего файла EML проста с Aspose.Email for .NET. Этот шаг подготавливает данные вашей электронной почты для преобразования или дальнейшей обработки.

#### Этапы внедрения

**Шаг 1**: Определите путь к вашему EML-файлу.
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY/Message.eml";
```

**Шаг 2**: Загрузите EML в `MailMessage` объект.
```csharp
// Загрузите файл EML в объект MailMessage.
MailMessage eml = MailMessage.Load(dataDir);
```
- **Параметры**: Путь к вашему EML-файлу.
- **Цель**: Преобразует файл EML в файл, пригодный для обработки `MailMessage` объект.

### Функция 2: Сохранение сообщения электронной почты в формате MSG

#### Обзор
После загрузки EML вы можете сохранить его как файл MSG, используя функцию сохранения Aspose.Email.

#### Этапы внедрения

**Шаг 1**: Укажите выходной каталог для вашего файла MSG.
```csharp
string outputPath = "@YOUR_OUTPUT_DIRECTORY/AnEmail_out.msg";
```

**Шаг 2**: Сохраните сообщение электронной почты в формате MSG.
```csharp
// Сохраните сообщение электронной почты на диске в формате Unicode (MSG)
eml.Save(outputPath, SaveOptions.DefaultMsgUnicode);
```
- **Параметры**: Выходной путь и параметры сохранения для кодировки Unicode.
- **Цель**: Преобразует и сохраняет вашу электронную почту в виде файла MSG.
