## Руководство по внедрению

Мы разобьем это руководство на две основные части: создание и настройка сообщения электронной почты и его сохранение в различных форматах.

### Создание и настройка сообщения электронной почты

Эта функция демонстрирует, как создать новое электронное письмо, настроить его свойства и сохранить его как файл EML.

#### Обзор
Создание электронных писем программным способом включает настройку темы, содержимого тела, отправителя, получателей и других конфигураций. Мы будем использовать Aspose.Email для .NET, чтобы добиться этого эффективно.

#### Пошаговая реализация

**1. Создайте новое электронное сообщение**

```csharp
using System;
using Aspose.Email.Mime;

// Начните с создания экземпляра класса MailMessage.
MailMessage message = new MailMessage();
```

Этот шаг инициализирует `MailMessage` объект, который служит основой для нашей электронной почты.

**2. Задайте тему и содержимое HTML-текста**

```csharp
// Укажите тему сообщения
message.Subject = "New message created by Aspose.Email for .NET";

// Включить HTML-контент в теле
message.IsBodyHtml = true;
message.HtmlBody = "<b>This line is in bold.</b> <br/> <br/><font color=blue>This line is in blue color</font>";
```

Настройка HTML-текста позволяет вам форматировать свое электронное письмо с помощью расширенного текста и стилей.

**3. Настройте информацию об отправителе**

```csharp
// Определите адрес электронной почты отправителя
message.From = "from@domain.com";
```

The `From` свойство указывает, кто отправляет электронное письмо.

**4. Добавьте получателей (TO и CC)**

```csharp
// Добавить основных получателей
message.To.Add("to1@domain.com");
message.To.Add("to2@domain.com");

// Добавить получателей копий
message.CC.Add("cc1@domain.com");
message.CC.Add("cc2@domain.com");
```

The `To` и `CC` свойства содержат адреса электронной почты получателей.

**5. Сохранить сообщение в формате EML**

```csharp
// Укажите путь для сохранения вашего сообщения электронной почты.
string dstEmail = "YOUR_OUTPUT_DIRECTORY\\Message.eml";
message.Save(dstEmail, SaveOptions.DefaultEml);
```

На этом этапе настроенное электронное письмо сохраняется в виде файла EML, готового к дальнейшему использованию или распространению.

### Сохранение сообщения электронной почты в разных форматах

Aspose.Email поддерживает сохранение писем в различных форматах, таких как EML, MSG и MHTML. Здесь мы сосредоточимся на формате EML.

#### Обзор
После создания сообщения электронной почты вы можете сохранить его в различных форматах в соответствии с конкретными потребностями.

**1. Сохраните объект MailMessage**

```csharp
// Убедитесь, что «сообщение» настроено с необходимыми данными.
string dstEmail = "YOUR_OUTPUT_DIRECTORY\\Message.eml";
message.Save(dstEmail, SaveOptions.DefaultEml);
```

Этот шаг подтверждает, что ваше электронное письмо сохранено в формате EML, который можно открыть стандартными почтовыми клиентами.
