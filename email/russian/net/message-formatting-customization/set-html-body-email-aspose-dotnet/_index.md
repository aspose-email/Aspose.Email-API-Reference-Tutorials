### Создание и отправка электронного письма с текстом HTML

#### Обзор
Эта функция позволяет создавать электронные письма с расширенным текстом и форматированием, встраивая HTML-контент непосредственно в тело письма.

##### Шаг 1: Инициализация объекта MailMessage
Начните с создания `MailMessage` объект, представляющий ваш адрес электронной почты.

```csharp
using Aspose.Email.Mime;
using Aspose.Email.Clients.Smtp;
using Aspose.Email.Clients;

// Создать новый экземпляр MailMessage
double settingHTMLBody()
{
    // Инициализируйте объект MailMessage
    MailMessage msg = new MailMessage();
```

##### Шаг 2: Укажите данные электронной почты
Определите отправителя, получателя и тему. Эти параметры имеют решающее значение для доставки электронной почты.

```csharp
    // Установите адреса электронной почты отправителя и получателя
    msg.From = "newcustomeronnet@gmail.com";
    msg.To = "asposetest123@gmail.com";

    // Определите тему письма
    msg.Subject = "Test Subject";
```

##### Шаг 3: Назначьте HTML-контент
Назначьте желаемый HTML-контент `HtmlBody`На этом этапе используется возможность Aspose.Email обрабатывать форматированный текст.

```csharp
    // Назначьте HTML-контент свойству HtmlBody
    msg.HtmlBody = "<html><body>This is the HTML body</body></html>";
```

##### Шаг 4: Настройка и отправка электронной почты
Настройте свой `SmtpClient` с необходимыми учетными данными и данными сервера, затем отправьте электронное письмо.

```csharp
    // Получить настроенный экземпляр SmtpClient
    SmtpClient client = GetSmtpClient();

    try
    {
        // Отправьте сообщение электронной почты с помощью SmtpClient
        client.Send(msg);
    }
    catch (Exception ex)
    	{
        // Обработка исключений во время отправки электронной почты
        Console.WriteLine(ex.ToString());
    }
}

// Метод настройки и возврата нового экземпляра SmtpClient
private static SmtpClient GetSmtpClient()
{
    // Создайте и настройте объект SmtpClient с данными сервера, учетными данными и параметрами безопасности.
    SmtpClient client = new SmtpClient("smtp.gmail.com", 587, "your.email@gmail.com", "your.password");
    client.SecurityOptions = SecurityOptions.Auto;
    
    return client;
}
```

### Основные параметры конфигурации
- **БезопасностьПараметры**: Автоматически определяет лучший протокол безопасности.
- **Подробности SMTP-сервера**: Убедитесь, что у вас есть точные данные сервера для успешной доставки электронной почты.

#### Советы по устранению неполадок
- Проверьте учетные данные SMTP и настройки сервера, если письма не отправляются.
- Проверьте наличие проблем с сетевым подключением, которые могут блокировать SMTP-запросы.
