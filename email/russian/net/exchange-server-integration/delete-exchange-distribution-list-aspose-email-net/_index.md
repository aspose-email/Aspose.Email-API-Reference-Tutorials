## Руководство по внедрению

### Удаление списков рассылки без включения в список участников

Эта функция демонстрирует, как безопасно удалить список рассылки с сервера Exchange, не перечисляя его участников.

#### Шаг 1: Инициализация клиента EWS
Сначала создайте и инициализируйте свой клиент EWS, используя необходимые учетные данные:
```csharp
IEWSClient client = EWSClient.GetEWSClient(
    "https://outlook.office365.com/ews/exchange.asmx",
    "testUser",
    "pwd",
    "domain"
);
```
- **Параметры**: `GetEWSClient` Для этого метода требуются URL-адрес сервера Exchange, учетные данные пользователя (имя пользователя и пароль) и домен.
- **Цель**: Устанавливает соединение с сервером Exchange для дальнейших операций.

#### Шаг 2: Определите список рассылки
Настройте свой список рассылки, указав его идентификатор:
```csharp
ExchangeDistributionList distributionList = new ExchangeDistributionList();
distributionList.Id = "list's id";
```
- **Параметры**: `Id` Свойство должно соответствовать уникальному идентификатору списка рассылки, который вы хотите удалить.
- **Цель**: Определяет целевой список рассылки для удаления.

#### Шаг 3: Удалить список рассылки
Выполните процесс удаления, убедившись, что в списке нет ни одного участника:
```csharp
client.DeleteDistributionList(distributionList, true);
```
- **Параметры**: `true` флаг принудительно удаляет без подтверждения или перечисления участников.
- **Цель**: Безопасное удаление списка рассылки с сервера Exchange.

#### Советы по устранению неполадок
- Убедитесь, что ваши учетные данные и идентификатор списка верны, чтобы избежать ошибок аутентификации.
- Проверьте сетевое подключение при подключении к серверу Exchange.
