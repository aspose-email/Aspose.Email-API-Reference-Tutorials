## Руководство по внедрению

Мы разделим нашу реализацию на две основные функции: инициализацию клиента Exchange и разрешение контактов по имени.

### Функция 1: Инициализация клиента Exchange

Эта функция фокусируется на создании экземпляра `IEWSClient` класс, используя ваши учетные данные, что имеет решающее значение для безопасного подключения к вашему серверу Exchange.

#### Пошаговая реализация

**Инициализировать экземпляр IEWSClient**

```csharp
public static void InitializeExchangeClient()
{
    // Создать экземпляр IEWSClient с указанными учетными данными и URL-адресом сервера.
    IEWSClient client = EWSClient.GetEWSClient(
        "https://outlook.office365.com/ews/exchange.asmx", 
        "testUser",   // Имя пользователя
        "pwd",        // Пароль
        "domain"      // Домен
    );
}
```
- **Объясняемые параметры:**
  - `"https://outlook.office365.com/ews/exchange.asmx"`URL-адрес сервера для ваших веб-служб Exchange.
  - `"testUser"`: Ваше имя пользователя Exchange.
  - `"pwd"`: Ваш пароль.
  - `"domain"`: Домен, связанный с учетной записью.

### Функция 2: Определение контактов по имени

Узнайте, как разрешать и перечислять контакты с сервера Exchange, используя имя контакта, что полезно для быстрого поиска определенных людей.

#### Пошаговая реализация

**Решить и составить список контактов**

```csharp
public static void ResolveContactsUsingContactName()
{
    try
    {
        // Инициализируйте экземпляр IEWSClient
        IEWSClient client = EWSClient.GetEWSClient(
            "https://outlook.office365.com/ews/exchange.asmx", 
            "testUser",   // Имя пользователя
            "pwd",        // Пароль
            "domain"      // Домен
        );

        // Определите контакты, используя указанное имя, и получите их фотографии
        Contact[] contacts = client.ResolveContacts(
            "Changed Name",  // Имя контакта для поиска
            ExchangeListContactsOptions.FetchPhoto // Возможность также извлечь фотографии контактов
        );

        // Повторить решенные контакты
        foreach (MapiContact contact in contacts)
        {
            // Вывести отображаемое имя контакта и адрес электронной почты
            Console.WriteLine("Name: " + contact.NameInfo.DisplayName + ", Email Address: " + contact.ElectronicAddresses.Email1);
        }
    }
    catch (Exception ex)
    {
        // Обрабатывайте исключения, выводя сообщение об ошибке
        Console.WriteLine(ex.Message);
    }
}
```
- **Объясняемые параметры:**
  - `"Changed Name"`: Имя контакта, проблему которого вы хотите разрешить.
  - `ExchangeListContactsOptions.FetchPhoto`: Возможность включать фотографии в результаты.
