Управление загроможденным почтовым ящиком, особенно при работе с требованиями поиска с учетом регистра, может быть сложной задачей. Если поиск определенных писем из-за разницы в регистре в серверах Outlook или Exchange был проблемой, это руководство для вас. ### Подключение к серверу Exchange

#### Обзор
Подключение к серверу Exchange необходимо для программного управления электронной почтой. Эта функция позволяет вашему приложению взаимодействовать с учетными записями электронной почты, размещенными на сервере Exchange.

#### Пошаговая реализация

**1. Инициализируйте ExchangeClient:**
The `ExchangeClient` класс предоставляет методы для подключения и взаимодействия с сервером Exchange. Предоставьте ему действительные учетные данные, такие как URL сервера, имя пользователя, пароль и домен.
```csharp
using Aspose.Email.Clients.Exchange;

public static void ConnectToExchangeServer()
{
    // Замените заполнители фактическими данными сервера.
    ExchangeClient client = new ExchangeClient("http://ex07sp1/exchange/Администратор", "пользователь", "пароль", "домен");
    
    // Теперь клиент готов выполнять операции на сервере Exchange.
}
```

**Объясняемые параметры:**
- **URL-адрес сервера**: Конечная точка вашего сервера Exchange.
- **Имя пользователя и пароль**: Учетные данные для аутентификации.
- **Домен**: Необязательный домен, если применимо.

### Фильтрация писем с учетом регистра

#### Обзор
Фильтрация писем с учетом регистра обеспечивает выявление точных совпадений, что крайне важно при поиске определенных тем или содержимого писем.

#### Пошаговая реализация

**1. Инициализируйте ExchangeQueryBuilder:**
The `ExchangeQueryBuilder` позволяет создавать запросы для фильтрации писем по различным критериям, таким как тема и дата.
```csharp
using Aspose.Email.Tools.Search;
using System;

public static void FilterEmailsUsingCaseSensitivity()
{
    // Инициализируйте конструктор.
    ExchangeQueryBuilder builder = new ExchangeQueryBuilder();
    
    // Установите условия для поиска с учетом регистра слова «Информационный бюллетень» в темах полученных сегодня электронных писем.
    builder.Subject.Contains("Newsletter", true);
    builder.InternalDate.On(DateTime.Now);

    // Получите построенный запрос.
    MailQuery query = builder.GetQuery();
}
```

**Объясняемые параметры:**
- **Тема содержит**: Поиск писем с определенной темой с учетом регистра.
- **Внутренняя дата включения**: Фильтрует электронные письма, полученные за текущую дату.
