## Руководство по внедрению

### Функция 1: Аутентификация и настройка клиента Exchange

Аутентифицируйтесь и создайте экземпляр `ExchangeClient` в этом разделе.

**Обзор:**
Аутентификация на сервере Exchange необходима для доступа к электронной почте. Вот как можно настроить клиент, используя ваши учетные данные.

#### Шаг 1: Создайте `ExchangeClient` Пример
```csharp
using Aspose.Email.Clients.Exchange;

// Определите URL-адрес вашего сервера, имя пользователя, пароль и домен.
string url = "http://ex07sp1/exchange/Администратор";
string username = "user";
string password = "pwd";
string domain = "domain";

// Инициализируйте ExchangeClient с учетными данными.
ExchangeClient client = new ExchangeClient(url, username, password, domain);
```

**Объяснение:**
- **URL-адрес**: URL-адрес сервера, на котором размещен ваш сервер Exchange.
- **имя пользователя/пароль/домен**: Для аутентификации требуются учетные данные.

### Функция 2: Список сообщений из папки «Входящие»

Используйте аутентифицированный `ExchangeClient` для просмотра списка сообщений в почтовом ящике.

**Обзор:**
Программное перечисление адресов электронной почты экономит время и автоматизирует повторяющиеся задачи. Вот как эффективно извлекать сообщения электронной почты.

#### Шаг 2: Извлечение электронных писем
```csharp
// Предположим, что «клиент» уже создан, как показано ранее.
ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);
```

**Объяснение:**
- `ListMessages`: Извлекает все сообщения из указанного URI почтового ящика (в данном случае — из папки «Входящие»).

### Функция 3: Отображение информации о сообщении

Просмотрите полученные сообщения и отобразите их основную информацию.

#### Шаг 3: Распечатайте данные электронной почты
```csharp
using System;

// Просмотрите каждое сообщение в коллекции.
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Message Size: " + msgInfo.Size);
    Console.WriteLine("==================================");
}
```

**Объяснение:**
- **msgInfo**: Представляет собой индивидуальный адрес электронной почты, предоставляющий доступ к таким свойствам, как `Subject`, `From`, и `Size`.
