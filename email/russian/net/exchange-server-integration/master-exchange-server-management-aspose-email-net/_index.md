## Руководство по внедрению

В этом разделе вы узнаете, как подключиться к серверу Exchange и составить список подпапок с помощью Aspose.Email для .NET.

### Подключитесь к серверу Exchange с помощью EWS

**Обзор:**
Подключение к серверу Exchange через EWS позволяет вашему приложению беспрепятственно взаимодействовать с данными электронной почты.

#### Шаг 1: Установка сетевых учетных данных
Настройте учетные данные, необходимые для аутентификации на сервере Exchange:
```csharp
string mailboxUri = "https://обмен/ews/exchange.asmx";
string domain = @"";
string username = "username@ASE305.onmicrosoft.com";
string password = "password";

NetworkCredential credentials = new NetworkCredential(username, password, domain);
```
*Объяснение:* Эти учетные данные аутентифицируют ваше приложение на сервере.

#### Шаг 2: Инициализация клиента EWS
Создать экземпляр `IEWSClient` для связи с сервером Exchange:
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
```
*Параметры:* 
- `mailboxUri`: Конечная точка для вашего сервера Exchange.
- `credentials`: Данные вашей сетевой аутентификации.

### Список подпапок на сервере Exchange с помощью EWS

**Обзор:**
Перечисление подпапок в почтовом ящике электронной почты помогает эффективно организовывать и управлять электронными письмами.

#### Шаг 1: Извлечение информации о корневой папке
Выберите корневую папку, чтобы просмотреть список ее подпапок:
```csharp
using Aspose.Email.Clients.Exchange;

ExchangeFolderInfoCollection folderInfoCol = client.ListSubFolders(client.MailboxInfo.RootUri);
```
*Возвращаемое значение:* Коллекция `ExchangeFolderInfo` объекты, представляющие каждую подпапку.

#### Шаг 2: Обработка каждой подпапки
Просмотрите коллекцию, чтобы получить доступ к сведениям о каждой папке:
```csharp
foreach (ExchangeFolderInfo folderInfo in folderInfoCol)
{
    // Обрабатывать каждую папку в зависимости от ее типа
}
```
*Объяснение:* Этот цикл позволяет обрабатывать папки по отдельности, адаптируя действия по мере необходимости.

### Определение типов папок с помощью EWS

**Обзор:**
Понимание типов папок помогает адаптировать определенные функции для различных категорий писем.

#### Шаг 1: Определите тип папки
Используйте оператор switch-case для идентификации и обработки каждого типа папки:
```csharp
foreach (ExchangeFolderInfo folderInfo in folderInfoCol)
{
    switch (folderInfo.FolderType)
    {
        case ExchangeFolderType.Appointment:
            // Папка типа «Обработка встреч»
            break;
        case ExchangeFolderType.Contact:
            // Папка типа «Контакт»
            break;
        case ExchangeFolderType.Task:
            // Папка типа «Обработка задач»
            break;
        case ExchangeFolderType.Note:
            // Обработка папки с сообщениями электронной почты
            break;
        case ExchangeFolderType.StickyNote:
            // Обрабатывать папки типа StickyNote
            break;
        case ExchangeFolderType.Journal:
            // Папка-регистратор типа «ручка»
            break;
        default:
            // При необходимости обрабатывайте другие типы.
            break;
    }
}
```
*Параметры:* `folderInfo.FolderType` определяет логику обработки каждой папки.

**Советы по устранению неполадок:**
- Во избежание ошибок аутентификации убедитесь, что ваши учетные данные верны.
- Проверьте URL-адрес и доступность сервера при подключении через EWS.
