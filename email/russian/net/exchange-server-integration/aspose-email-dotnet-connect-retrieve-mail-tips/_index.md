### Руководство по внедрению

Мы рассмотрим две ключевые функции: подключение к серверу Exchange и получение почтовых подсказок.

#### Подключение к серверу Exchange

Сначала установите соединение с сервером Exchange с помощью класса EWSClient из Aspose.Email .NET.

##### Обзор
Подключение к серверу Exchange позволяет автоматизировать задачи управления электронной почтой, такие как отправка писем и управление календарями. Вот как:

##### Пошаговое руководство
**1. Инициализируйте клиент EWS**
Для подключения создайте экземпляр `IEWSClient` с URL-адресом вашего сервера и учетными данными:
```csharp
using System;
using Aspose.Email.Clients.Exchange.WebService;

public static void ConnectToExchangeServer()
{
    // Создайте экземпляр IEWSClient, используя URL-адрес сервера и учетные данные пользователя.
    IEWSClient client = EWSClient.GetEWSClient(
        "https://outlook.office365.com/ews/exchange.asmx", 
        "testUser",   
        "pwd", 
        "domain");
    
    Console.WriteLine("Connected to Exchange Server successfully.");
}
```
**Объясняемые параметры:**
- **URL-адрес сервера**: Конечная точка вашего сервера Exchange.
- **Реквизиты для входа**: Учетные данные пользователя (имя пользователя, пароль) и домен для аутентификации.

#### Извлечение и отображение советов по почте

Теперь, когда вы подключены, давайте получим почтовые подсказки, информирующие о проблемах с доставкой или других уведомлениях, связанных с сообщением электронной почты.

##### Обзор
Советы по почте предоставляют ценную информацию, например, статусы «вне офиса» или оповещения о недействительных получателях перед отправкой писем. Эта функция помогает в упреждающем решении проблем со связью.

##### Пошаговое руководство
**2. Подготовьте адреса электронной почты**
Соберите адреса электронной почты, которые вы хотите проверить на наличие почтовых советов:
```csharp
using System;
using Aspose.Email.Clients.Exchange.WebService;
using Aspose.Email.Mime;

public static void RetrieveAndDisplayMailTips()
{
    // Создать экземпляр EWSClient
    IEWSClient client = EWSClient.GetEWSClient(
        "https://outlook.office365.com/ews/exchange.asmx", 
        "testUser",   
        "pwd", 
        "domain");

    // Подготовьте адреса электронной почты для проверки почтовых советов
    MailAddressCollection addrColl = new MailAddressCollection();
    addrColl.Add(new MailAddress("test.exchange@ex2010.local", true));
    addrColl.Add(new MailAddress("invalid.recipient@ex2010.local", true));

    Console.WriteLine("Mail addresses prepared.");
}
```
**3. Получить почтовые подсказки**
Настройте и извлеките почтовые подсказки с помощью `GetMailTipsOptions`:
```csharp
// Настройте параметры для получения почтовых подсказок, указав отправителя и получателей
GetMailTipsOptions options = new GetMailTipsOptions(
    "administrator@ex2010.local", 
    addrColl, 
    MailTipsType.All);

// Получить почтовые подсказки с сервера
MailTips[] tips = client.GetMailTips(options);
Console.WriteLine("Retrieved mail tips.");
```
**4. Отображение советов по почте**
Просмотрите и отобразите соответствующую информацию:
```csharp
// Просмотрите каждую подсказку по почте, чтобы извлечь подробности
foreach (MailTips tip in tips)
{
    if (tip.OutOfOffice != null)
    {
        Console.WriteLine($"Out of office: {tip.OutOfOffice.ReplyBody.Message}");
    }

    if (tip.InvalidRecipient == true)
    {
        Console.WriteLine($"Invalid recipient: {tip.RecipientAddress}");
    }
}
```
