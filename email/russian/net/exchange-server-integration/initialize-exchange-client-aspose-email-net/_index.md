## Руководство по внедрению

Давайте разберем реализацию на две основные функции: инициализацию клиента Exchange и вывод списка сообщений по идентификатору.

### Функция 1: Инициализация клиента Exchange

#### Обзор
Установите соединение с вашим сервером Microsoft Exchange, создав `ExchangeClient` экземпляр с использованием соответствующих учетных данных.

#### Этапы внедрения

##### Шаг 1: Создание экземпляра ExchangeClient
Укажите URL-адрес сервера, имя пользователя, пароль и домен:
```csharp
using Aspose.Email.Clients.Exchange;
using Aspose.Email.Clients.Exchange.Dav;

public void InitializeExchangeClient()
{
    var client = new ExchangeClient(
        "https://ИмяМашины/Обмен/ИмяПользователя",
        "username",
        "password",
        "domain"
    );
}
```
- **Объяснение параметров**:
  - `server URL`: Конечная точка вашего сервера Exchange.
  - `username`, `password`, и `domain`: Учетные данные для аутентификации.

### Функция 2: Список сообщений по идентификатору

#### Обзор
Эффективно извлекайте входящие сообщения, используя определенные идентификаторы сообщений, после подключения к серверу Exchange.

#### Этапы внедрения

##### Шаг 1: Определите идентификатор сообщения и URI почтового ящика
Определите интересующий вас идентификатор сообщения и получите URI папки «Входящие»:
```csharp
public void ListMessagesById(ExchangeClient client)
{
    string messageId = "23A747F0C7A5DB4BAB299C2BE2383FD556E630DD@machinename.local";
    var inboxUri = client.MailboxInfo.InboxUri;
```

##### Шаг 2: Извлечение сообщений
Используйте `ListMessagesById` метод извлечения сообщений:
```csharp
ExchangeMessageInfoCollection msgCollection = client.ListMessagesById(inboxUri, messageId);
```
- **Цель**: Извлекает информацию о сообщении на основе указанного идентификатора.

##### Шаг 3: Отображение сведений о сообщении
Повторите сбор и распечатайте основные данные каждого электронного письма:
```csharp
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Message ID: " + msgInfo.MessageId);
    Console.WriteLine("Unique URI: " + msgInfo.UniqueUri);
    Console.WriteLine("==================================");
}
```
- **Отображается ключевая информация**: Тема, сведения об отправителе и получателе, идентификатор сообщения и уникальный URI.
