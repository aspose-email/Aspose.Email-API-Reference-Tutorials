## Руководство по внедрению

Мы рассмотрим две основные функции: подключение к серверу Exchange и создание правил для входящих сообщений.

### Подключитесь к Exchange Server с помощью .NET

#### Обзор
Подключение к серверу Exchange позволяет автоматизировать задачи электронной почты, такие как чтение, отправка или организация писем программным способом. Это включает в себя проверку подлинности ваших учетных данных и установку соединения с помощью Aspose.Email for .NET.

#### Этапы внедрения
**Шаг 1:** Импортируйте необходимые пространства имен.
```csharp
using System;
using Aspose.Email.Clients.Exchange;
using Aspose.Email.Clients.Exchange.WebService;
using System.Net;
```

**Шаг 2:** Определите учетные данные и URL-адрес вашего сервера Exchange.
```csharp
string mailboxURI = "https://ex2010/ews/exchange.asmx"; // URL-адрес сервера Exchange
string username = "test.exchange"; // Имя пользователя для аутентификации
string password = "pwd"; // Пароль для аутентификации
string domain = "ex2010.local"; // Информация о домене
```

**Шаг 3:** Создайте объект NetworkCredential и инициализируйте IEWSClient.
```csharp
NetworkCredential credential = new NetworkCredential(username, password, domain);
IEWSClient client = EWSClient.GetEWSClient(mailboxURI, credential);
```
*Объяснение:* The `NetworkCredential` Класс инкапсулирует ваши учетные данные пользователя, необходимые для аутентификации. `GetEWSClient` Метод подключается к серверу Exchange, используя эти учетные данные.

### Создать новое правило на сервере Exchange

#### Обзор
Создание правил для папки «Входящие» помогает автоматизировать такие действия, как перемещение или пометка писем на основе определенных условий, экономя время и обеспечивая организованность.

#### Этапы внедрения
**Шаг 1:** Определите новый объект правила для папки «Входящие».
```csharp
InboxRule rule = new InboxRule();
rule.DisplayName = "Message from client ABC"; // Задайте отображаемое имя для правила.
```

**Шаг 2:** Укажите условия, при которых должно применяться правило.
```csharp
RulePredicates newRules = new RulePredicates();
newRules.ContainsSubjectStrings.Add("ABC"); // Сопоставьте электронные письма с темой, содержащей «ABC».
newRules.FromAddresses.Add(new MailAddress("administrator@ex2010.local", true)); // Сопоставьте электронные письма с определенного адреса.
rule.Conditions = newRules;
```
