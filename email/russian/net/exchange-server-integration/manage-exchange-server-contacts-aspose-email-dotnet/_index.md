## Руководство по внедрению
Мы разделим это руководство на три основных раздела: подключение к серверу Exchange, создание и настройка контактов и их добавление на сервер.

### Подключение к Exchange Server с помощью EWS (Exchange Web Services)

#### Обзор
Подключение к серверу Exchange через EWS позволяет осуществлять программный доступ к функциям почтового ящика. Aspose.Email упрощает этот процесс с помощью своего надежного API.

**Шаг 1: Настройка сетевых учетных данных**
Создать `NetworkCredential` объект, используя ваше имя пользователя, пароль и информацию о домене:
```csharp
using System.Net;

string mailboxUri = "https://ex2010/ews/exchange.asmx";
string username = "test.exchange";
string password = "pwd";
string domain = "ex2010.local";

// Создать сетевые учетные данные
NetworkCredential credentials = new NetworkCredential(username, password, domain);
```

**Шаг 2: Установите соединение с клиентом EWS**
Используйте `EWSClient.GetEWSClient` способ подключения:
```csharp
using Aspose.Email.Clients.Exchange.WebService;

IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
```
На этом этапе устанавливается соединение между вашим приложением и сервером Exchange, что позволяет вам управлять контактами.

### Создание и настройка контакта

#### Обзор
Настройка подробных профилей контактов включает в себя настройку таких атрибутов, как имена, номера телефонов, адреса электронной почты и т. д. Aspose.Email делает этот процесс интуитивно понятным благодаря `Contact` сорт.

**Шаг 1: Создайте новый контакт**
Инициализируйте новый экземпляр `Contact` сорт:
```csharp
using Aspose.Email.PersonalInfo;

// Создать новый контакт
Contact contact = new Contact();
```

**Шаг 2: Укажите общую информацию**
Укажите основные данные для вашего контакта:
```csharp
contact.Gender = Gender.Male;
contact.DisplayName = "Frank Lin";
contact.CompanyName = "ABC Co.";
contact.JobTitle = "Executive Manager";
```

**Шаг 3: Добавьте номера телефонов, связанных лиц и URL-адреса.**
Улучшите профиль контакта, добавив больше информации:
```csharp
// Добавить номера телефонов
contact.PhoneNumbers.Add(new PhoneNumber { Number = "123456789", Category = PhoneNumberCategory.Home });

// Установить связанных лиц
contact.AssociatedPersons.Add(new AssociatedPerson { Name = "Catherine", Category = AssociatedPersonCategory.Spouse });
contact.AssociatedPersons.Add(new AssociatedPerson { Name = "Bob", Category = AssociatedPersonCategory.Child });
contact.AssociatedPersons.Add(new AssociatedPerson { Name = "Merry", Category = AssociatedPersonCategory.Sister });

// Добавить URL-адреса
contact.Urls.Add(new Url { Href = "www.blog.com", Category = UrlCategory.Blog });
contact.Urls.Add(new Url { Href = "www.homepage.com", Category = UrlCategory.HomePage });
```

**Шаг 4: Укажите адреса электронной почты**
Наконец, настройте адреса электронной почты для контакта:
```csharp
// Добавить адреса электронной почты
contact.EmailAddresses.Add(new EmailAddress { Address = "Frank.Lin@Abc.com", DisplayName = "Frank Lin", Category = EmailAddressCategory.Email1 });
```

### Добавление контакта на сервер Exchange

#### Обзор
После настройки контакта добавьте его на сервер Exchange с помощью клиента Aspose.Email.

**Шаг 1: Инициализация клиента EWS**
Убедитесь, что `client` из предыдущего раздела инициализируется:
```csharp
IEWSClient client = null; // Заполнитель, убедитесь, что он настроен правильно
```

**Шаг 2: Добавьте контакт на сервер**
Используйте следующий код для добавления вашего контакта:
```csharp
try
{
    client.CreateContact(contact);
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message); // Обрабатывайте исключения надлежащим образом
}
```
На этом этапе ваш новый контакт интегрируется в сервер Exchange, что делает его доступным для дальнейшего использования.
