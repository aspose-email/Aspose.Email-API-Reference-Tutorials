## Руководство по внедрению

Теперь, когда наша настройка завершена, давайте перейдем к реализации функции вывода списка сообщений с сервера Exchange.

### Подключиться к серверу Exchange

Чтобы составить список адресов электронной почты, сначала подключитесь к серверу Exchange с помощью Aspose.Email. Для этого шага вам понадобится URI сервера и ваши учетные данные.

**Шаг 1: Установите соединение**

Создайте новый экземпляр `ExchangeClient`:

```csharp
string exchangeUri = "https://MachineName/exchange/Username"; // URI вашего сервера Exchange
string username = "username"; // Ваше имя пользователя сервера Exchange
string password = "password"; // Ваш пароль к серверу Exchange

try
{
    var domain = new Domain(); // Заполнитель для класса домена, если необходимо
    ExchangeClient client = new ExchangeClient(exchangeUri, username, password, domain);

    // Перейти к списку сообщений
}
catch (Exception ex)
{
    Console.Write("Error connecting: " + ex.Message);
}
```

Здесь, `ExchangeClient` принимает URI сервера и учетные данные в качестве параметров, обеспечивая безопасное соединение.

### Список сообщений из папки «Входящие»

Теперь, когда соединение установлено, мы можем получать электронные письма:

**Шаг 2: Извлечение сообщений**

Используйте клиент для извлечения сообщений из вашего почтового ящика:

```csharp
try
{
    ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);

    // Отображение информации о сообщении
}
catch (Exception ex)
{
    Console.Write("Error retrieving messages: " + ex.Message);
}
```

`ListMessages` извлекает все сообщения из указанного URI почтового ящика, возвращая их в виде коллекции.

### Отображение информации о сообщении

После получения сообщений вы можете просмотреть их, чтобы отобразить необходимые данные:

**Шаг 3: Итерация и отображение**

```csharp
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Sent Date: " + msgInfo.Date.ToString());
    Console.WriteLine("Read?: " + msgInfo.IsRead.ToString());
    Console.WriteLine("Message ID: " + msgInfo.MessageId);
    Console.WriteLine("Unique URI: " + msgInfo.UniqueUri);
}
```

Этот цикл проходит по каждому сообщению, выводя ключевые атрибуты, такие как тема, отправитель, получатель и статус прочтения.
