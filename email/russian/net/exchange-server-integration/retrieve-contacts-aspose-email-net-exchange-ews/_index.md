## Руководство по внедрению

В этом разделе вы узнаете о каждом шаге, необходимом для извлечения контактов с сервера Exchange.

### Шаг 1: Создание экземпляра IEWSClient

**Обзор**Установите безопасное соединение с сервером Exchange с помощью клиента EWS Aspose.Email.

```csharp
IEWSClient client = EWSClient.GetEWSClient(
    "https://outlook.office365.com/ews/exchange.asmx", 
    "testUser",    
    "pwd",         
    "domain"       
);
```

**Объяснение**: Заменять `"testUser"`, `"pwd"`, и `"domain"` с вашими реальными учетными данными сервера Exchange. Этот шаг аутентифицирует и инициализирует соединение.

### Шаг 2: Перечислите все контакты

**Обзор**: Извлеките контакты из своего почтового ящика с помощью EWS.

```csharp
Contact[] contacts = client.GetContacts(client.MailboxInfo.ContactsUri);
```

**Объяснение**: `client.GetContacts` извлекает все доступные контакты. Метод возвращает массив объектов контактов, которые затем могут быть обработаны.

### Шаг 3: Отображение контактной информации

**Обзор**: Перебрать извлеченные контакты, чтобы отобразить их имена и адреса электронной почты.

```csharp
foreach (MapiContact contact in contacts)
{
    Console.WriteLine(
        "Name: " + contact.NameInfo.DisplayName + ", Email Address: " +
        contact.ElectronicAddresses.Email1
    );
}
```

**Объяснение**: Пройтись по каждому `MapiContact` для доступа и отображения важной информации, такой как имя и адрес электронной почты.

### Советы по устранению неполадок

- Убедитесь, что URL-адрес сервера, имя пользователя, пароль и домен настроены правильно.
- Проверьте сетевое подключение, если не удается подключиться к серверу Exchange.
- Убедитесь, что ваша версия API совместима с версией .NET Framework.
