## Руководство по внедрению
Давайте разберем реализацию на две основные функции: извлечение гиперссылок с атрибутами href и извлечение простого текста из тегов привязки.

### Функция 1: Отображение гиперссылки с помощью Href
Эта функция позволяет извлекать как URL-адрес, так и связанный с ним текст из тега HTML-якоря.

#### Обзор
Вы проанализируете HTML-строку, чтобы получить гиперссылку (`href`) и отобразить текст внутри `<a>` ярлык.

#### Пошаговая реализация

**Шаг 1:** Определите `href` позиция атрибута.

```csharp
string source = "<a href='https://example.com'>Пример</a>";
int startHref = source.IndexOf("href=\"") + "href=\"".Length;
```

*Почему?* На этом этапе определяется место начала гиперссылки в теге для точного извлечения.

**Шаг 2:** Определите конец `href` атрибут.

```csharp
int endHref = source.IndexOf("\"", startHref);
string href = source.Substring(startHref, endHref - startHref);
```

*Почему?* Он помогает изолировать URL-адрес, отмечая его конец внутри тега.

**Шаг 3:** Извлеките текст между `<a>` теги.

```csharp
int startText = source.IndexOf(">") + 1;
int endText = source.IndexOf("<", startText);
string text = source.Substring(startText, endText - startText);
```

*Почему?* Это позволяет захватить видимый текст ссылки для отображения или использования в вашем приложении.

**Шаг 4:** Объедините текст и href для вывода.

```csharp
string link = $"{text} <{href}>";
Console.WriteLine(link); // Вывод: Пример <https://example.com>
```

### Функция 2: Отображение гиперссылки без Href
Эта функция фокусируется на извлечении только видимого текста из тега привязки, игнорируя URL.

#### Обзор
Полезно, когда вам нужен только отображаемый текст для пользовательских интерфейсов или дальнейшей обработки.

#### Пошаговая реализация

**Извлечь только текст**

```csharp
int startNoHref = source.IndexOf(">") + 1;
int endNoHref = source.IndexOf("<", startNoHref);
string plainText = source.Substring(startNoHref, endNoHref - startNoHref);
Console.WriteLine(plainText); // Вывод: Пример
```

*Почему?* Этот метод эффективно извлекает текст без обработки URL.
