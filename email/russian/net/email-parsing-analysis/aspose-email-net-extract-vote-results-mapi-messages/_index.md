### Информация о результатах голосования по чтению

Эта функция демонстрирует, как извлекать информацию о голосовании, такую как ответы и время ответа, из получателей электронной почты. Вот пошаговая разбивка:

#### Шаг 1: Определите каталог документов
Начните с указания пути, где находится файл вашего сообщения.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/AddVotingButtonToExistingMessage.msg";
```

#### Шаг 2: Загрузка сообщения MAPI
Загрузите сообщение MAPI из файла с помощью Aspose.Email `MapiMessage` сорт.
```csharp
MapiMessage msg = MapiMessage.FromFile(dataDir);
```

#### Шаг 3: Перебор получателей
Просмотрите каждого получателя, чтобы получить доступ к его ответам на голосование и времени ответа.
```csharp
foreach (MapiRecipient recipient in msg.Recipients)
{
    // Получить отображаемое имя получателя
    string displayName = recipient.DisplayName;

    // Извлеките ответ на голосование, используя свойство PR_RECIPIENT_AUTORESPONSE_PROP_RESPONSE
    string response = recipient.Properties[MapiPropertyTag.PR_RECIPIENT_AUTORESPONSE_PROP_RESPONSE].GetString();

    // Получите время ответа с помощью свойства PR_RECIPIENT_TRACKSTATUS_TIME
    DateTime responseTime = recipient.Properties[MapiPropertyTag.PR_RECIPIENT_TRACKSTATUS_TIME].GetDateTime();
}
```

#### Пояснение к коду
- **Отображаемое имя**: `recipient.DisplayName` предоставляет читаемый идентификатор для каждого получателя.
- **Свойство ответа**Использует свойство PR_RECIPIENT_AUTORESPONSE_PROP_RESPONSE для доступа к ответам на голосование.
- **Время отклика**: PR_RECIPIENT_TRACKSTATUS_TIME фиксирует время регистрации каждого ответа, что полезно для отслеживания вовлеченности.
