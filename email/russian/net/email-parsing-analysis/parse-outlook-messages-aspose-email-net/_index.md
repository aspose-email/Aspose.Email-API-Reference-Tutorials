## Руководство по внедрению

### Функция 1: Загрузка и анализ файла сообщений Outlook

Первый шаг — загрузить сообщение электронной почты из файла. Эта функция продемонстрирует, как извлечь основные свойства, такие как тема, имя отправителя, содержимое тела и вложения.

#### Обзор
В этом разделе показано, как использовать Aspose.Email для .NET для чтения файла Outlook MSG или EML и доступа к его основным компонентам.

##### Шаг 1: Загрузите электронное письмо
Сначала определите путь, где хранятся файлы вашей электронной почты. Затем загрузите сообщение с помощью `MapiMessage.FromMailMessage`.

```csharp
using System;
using Aspose.Email.Mapi;

public class Feature1
{
    public static void Run()
    {
        string dataDir = @"YOUR_DOCUMENT_DIRECTORY/"; 
        MapiMessage msg = MapiMessage.FromMailMessage(dataDir + "Message.eml");

        // Показать свойства электронной почты
        Console.WriteLine("Subject:" + msg.Subject);
        Console.WriteLine("From:" + msg.SenderName);
        Console.WriteLine("Body:" + msg.Body);
        Console.WriteLine("Attachment Count:" + msg.Attachments.Count);
    }
}
```

**Почему это важно:** Загрузка сообщения обеспечивает доступ ко всем его элементам, что позволяет выполнять детальную обработку и извлечение данных.

##### Шаг 2: Извлечение свойств электронной почты
Использовать свойства `MapiMessage` для извлечения таких деталей, как тема, имя отправителя и содержимое тела. Количество вложений также отображается с помощью `msg.Attachments.Count`.

### Функция 2: Итерация по вложениям

После загрузки сообщения электронной почты просмотр его вложений становится простым.

#### Обзор
В этой части объясняется, как перебрать все вложения в файле сообщения и сохранить их по отдельности.

##### Шаг 1: Сохраните вложения
Вы можете повторить `msg.Attachments` и используйте `Save` метод для каждого из них. Убедитесь, что у вас настроен выходной каталог для сохранения этих файлов.

```csharp
using System;
using Aspose.Email.Mapi;

public class Feature2
{
    public static void Run(MapiMessage msg)
    {
        foreach (MapiAttachment attachment in msg.Attachments)
        {
            Console.WriteLine("Attachment:" + attachment.FileName);
            string outputPath = @"YOUR_OUTPUT_DIRECTORY/" + attachment.LongFileName;
            attachment.Save(outputPath);
        }
    }
}
```

**Почему это важно:** Сохранение вложений по отдельности позволяет управлять ими и хранить их по мере необходимости, что особенно полезно при выполнении задач автоматизации.
