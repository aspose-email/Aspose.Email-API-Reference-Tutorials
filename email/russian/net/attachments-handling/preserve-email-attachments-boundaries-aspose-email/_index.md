## Руководство по внедрению

### Функция: сохранение исходных границ во вложениях электронной почты

#### Обзор
Эта функция гарантирует, что при сохранении письма с вложениями исходная структура и границы MIME остаются нетронутыми. Это особенно полезно для юридических или архивных целей, где целостность данных имеет решающее значение.

#### Шаг 1: Загрузите электронное письмо
Сначала загрузите сообщение электронной почты из файла или потока:

```csharp
using System.IO;
using Aspose.Email.Mime;

// Определите путь к каталогу ваших документов.
string dataDir = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "email.eml");

MailMessage mailMessage = MailMessage.Load(dataDir);
```
**Объяснение**: Здесь мы загружаем сообщение электронной почты из указанного пути. Убедитесь, что `dataDir` указывает на фактическое местоположение вашего файла.

#### Шаг 2: Сохранение с сохранением границ
Чтобы сохранить электронное письмо, сохранив его исходные границы:

```csharp
string outputDir = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "savedEmail.eml");
mailMessage.Save(outputDir, Aspose.Email.SaveOptions.DefaultEml);
```
**Объяснение**: Этот шаг сохраняет загруженное сообщение в новый файл. `SaveOptions.DefaultEml` обеспечивает сохранение исходных границ MIME.
