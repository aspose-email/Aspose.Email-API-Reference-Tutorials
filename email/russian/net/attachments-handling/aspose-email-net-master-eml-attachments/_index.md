## Руководство по внедрению
### Функция 1: Загрузка и отображение вложений электронной почты
Эта функция фокусируется на загрузке файла EML для доступа к его вложениям. Вот как вы можете это реализовать:

#### Пошаговая реализация
**Загрузите файл EML**
Начните с создания `MailMessage` экземпляр и загрузите ваш EML-файл.

```csharp
using System;
using Aspose.Email.Mime;

// Замените фактическим путем к каталогу вашего документа.
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";

// Создайте экземпляр MailMessage и загрузите файл электронной почты.
MailMessage mailMsg = MailMessage.Load(dataDir + "/Test.eml", new EmlLoadOptions());
```

**Перебрать вложения**
Просмотрите каждое вложение в цикле `MailMessage` объект.

```csharp
foreach (Attachment attachment in mailMsg.Attachments)
{
    // Сохраните вложение в указанном вами выходном каталоге.
    attachment.Save("@YOUR_OUTPUT_DIRECTORY/" + attachment.Name);
}
```

**Объяснение**: 
- **Параметры**: Путь к файлу и `EmlLoadOptions` укажите, как загружается электронное письмо.
- **Цель**Этот фрагмент кода сохраняет каждое вложение в указанном каталоге, что обеспечивает легкий доступ к нему в дальнейшем.

#### Советы по устранению неполадок
- Убедитесь, что файл EML существует в указанном месте.
- Проверьте правильность настроек прав доступа к выходному каталогу.

### Функция 2: Извлечение и отображение имен вложений
Далее извлечем и отобразим имена вложений из файла EML:

#### Пошаговая реализация
**Доступ к вложениям**
Используйте `foreach` цикл для доступа к каждому вложению в загруженном сообщении электронной почты.

```csharp
using System;
using Aspose.Email.Mime;

// Замените фактическим путем к каталогу вашего документа.
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";

MailMessage mailMsg = MailMessage.Load(dataDir + "/Test.eml", new EmlLoadOptions());

foreach (Attachment attachment in mailMsg.Attachments)
{
    // Отображение имени каждого вложения
    Console.WriteLine(attachment.Name);
}
```

**Объяснение**: 
- **Параметры**: Как и прежде, здесь используются путь к файлу и параметры загрузки.
- **Цель**: Перебирая вложения, вы можете отображать их имена непосредственно в консоли, что полезно для регистрации или отладки.

#### Советы по устранению неполадок
- Если имена вложений не отображаются, проверьте, содержит ли файл EML какие-либо вложения.
