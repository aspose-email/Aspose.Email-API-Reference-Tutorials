### Использование IGmailClient для создания контакта

#### Обзор
Узнайте, как использовать интерфейс IGmailClient для программного создания контактов в Gmail.

**Пошаговая реализация**

1. **Инициализировать IGmailClient**
   ```csharp

   ```

2. **Создать контакт**
   ```csharp
   string contactUri = client.CreateContact(contact);
   ```
