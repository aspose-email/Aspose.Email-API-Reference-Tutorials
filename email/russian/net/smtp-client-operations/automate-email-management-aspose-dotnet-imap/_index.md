## Руководство по внедрению

### Подключитесь и войдите в систему на сервере IMAP

#### Обзор
Подключение к серверу IMAP — это первый шаг в программном управлении электронной почтой. Мы будем использовать Aspose.Email `ImapClient` класс для этой цели.

**Шаг 1: Определите учетные данные**
Начните с определения учетных данных вашего сервера IMAP:
```csharp
const string host = "your-imap-host";
const int port = 143; // Порт IMAP по умолчанию
const string username = "user@host.com";
const string password = "password";
```

**Шаг 2: Создание и использование ImapClient**
Создайте экземпляр `ImapClient` класс, использующий эти учетные данные:
```csharp
using (ImapClient client = new ImapClient(host, port, username, password))
{
    Console.WriteLine("Connected and logged in to IMAP server.");
}
```

**Совет по устранению неполадок**: Убедитесь, что ваша сеть разрешает соединения на указанном порту IMAP. Если у вас возникли проблемы с аутентификацией, дважды проверьте свои учетные данные.

### Выберите папку IMAP

#### Обзор
После подключения необходимо выбрать папку, например «Входящие», для выполнения операций в ней.

**Шаг 1: Подключитесь к серверу**
Повторное использование наших `ImapClient`подключите как показано ранее:
```csharp
using (ImapClient client = new ImapClient("your-imap-host", 143, "user@host.com", "password"))
{
    // Выберите папку «Входящие»
    client.SelectFolder(ImapFolderInfo.InBox);
    Console.WriteLine("Inbox folder selected.");
}
```

### Создание и выполнение поискового запроса IMAP

#### Обзор
Поиск определенных писем — это обычная задача. Мы покажем, как создать и выполнить поисковый запрос IMAP.

**Шаг 1: Создание ImapQueryBuilder**
Используйте `ImapQueryBuilder` чтобы указать критерии поиска:
```csharp
ImapQueryBuilder builder = new ImapQueryBuilder();
builder.Subject.Contains("Newsletter"); // Фильтр по теме
builder.InternalDate.On(DateTime.Now);  // Письма, полученные сегодня
```

**Шаг 2: Выполнение поискового запроса**
Используйте запрос для получения сообщений:
```csharp
MailQuery query = builder.GetQuery();
ImapMessageInfoCollection messages = client.ListMessages(query);
Console.WriteLine($"Found {messages.Count} message(s) in Inbox.");
```

## Раздел часто задаваемых вопросов
1. **Какой номер порта по умолчанию для IMAP?**
Порт по умолчанию — 143, но для защищенных соединений обычно используется порт 993.
2. **Как работать с SSL/TLS с Aspose.Email?**
Настройте свой `ImapClient` чтобы включить SSL по мере необходимости: `client.SecurityOptions = SecurityOptions.Auto;`
3. **Могу ли я искать электронные письма, отправленные ранее сегодняшнего дня?**
Да, отрегулируйте `InternalDate.On` метод или использовать диапазоны дат в `ImapQueryBuilder`.
4. **Что делать, если мой IMAP-сервер требует аутентификации через OAuth2?**
Aspose.Email поддерживает OAuth2. Реализуйте необходимые шаги для аутентификации с использованием токенов OAuth.
5. **Как эффективно обрабатывать большие объемы электронных писем?**
Используйте пакетную обработку и оптимизируйте запросы для обработки писем управляемыми порциями.
