## Руководство по внедрению

### Создание и настройка почтовых сообщений

#### Обзор
Эта функция демонстрирует, как создать новое электронное письмо, настроить его свойства, такие как отправитель, получатель, тема, приоритет, конфиденциальность и уведомления о доставке, а также сохранить его как файл EML.

##### Шаг 1: Создайте новое электронное письмо
```csharp
using Aspose.Email.Mime;
using System;

// Инициализируйте новый экземпляр MailMessage
MailMessage message = new MailMessage();
message.From = "sender@gmail.com"; // Укажите адрес электронной почты отправителя.
message.To = "receiver@gmail.com"; // Установите адрес электронной почты получателя
message.Subject = "Using MailMessage Features"; // Определите тему

// Установить дополнительные свойства
message.Date = DateTime.Now; // Текущее время как дата сообщения
message.Priority = MailPriority.High; // Приоритет электронной почты установлен на Высокий
message.Sensitivity = MailSensitivity.Normal; // Нормальный уровень чувствительности
message.DeliveryNotificationOptions = DeliveryNotificationOptions.OnSuccess; // Включить уведомление об успешном выполнении
```

##### Шаг 2: Сохраните сообщение.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
message.Save(outputDir + "/UseMailMessageFeatures_out.eml", SaveOptions.DefaultEml);
```
- **SaveOptions.DefaultEml**: эта опция сохраняет электронное письмо в формате EML по умолчанию.

#### Советы по устранению неполадок
Убедитесь, что ваш `outputDir` Путь правильно установлен, чтобы избежать ошибок сохранения файлов. Всегда обрабатывайте исключения во время операций с файлами для надежного управления ошибками.

## Раздел часто задаваемых вопросов

### Что такое Aspose.Email для .NET?
Aspose.Email для .NET — это библиотека, которая упрощает создание, отправку и обработку электронной почты в приложениях .NET.

### Как установить приоритет сообщения электронной почты с помощью Aspose.Email?
Вы можете установить приоритет электронного письма, назначив `MailPriority.High`, `MailPriority.Normal`, или `MailPriority.Low` к `Priority` собственность `MailMessage`.

### Могу ли я настроить уведомления о доставке писем?
Да, вы можете включить такие опции уведомления о доставке, как `OnSuccess` и `OnError` используя `DeliveryNotificationOptions` свойство.
