## Руководство по внедрению

### Функция проверки возвращенных сообщений электронной почты
Эта функция позволяет определить, было ли сообщение электронной почты возвращено, и извлечь соответствующие сведения с помощью Aspose.Email для .NET.

#### Обзор
Загружая файл электронного письма, мы проверим его статус недоставки и извлечем важную информацию, такую как причина и данные получателя.

#### Пошаговая реализация
**1. Определите путь к файлу электронной почты**
```csharp
string fileName = "YOUR_DOCUMENT_DIRECTORY\\failed1.msg";
```
*Почему?*: Указывает местоположение файла образца электронной почты для тестирования функции.

**2. Загрузите сообщение электронной почты.**
```csharp
MailMessage mail = MailMessage.Load(fileName);
```
*Объяснение*: Загружает сообщение электронной почты из указанного файла с помощью Aspose.Email `MailMessage` сорт.

**3. Проверьте статус возврата и получите подробную информацию**
```csharp
BounceResult result = mail.CheckBounced();
```
*Цель*: Анализирует загруженное сообщение, чтобы определить, было ли оно отклонено, возвращая подробную информацию, заключенную в `BounceResult` объект.

**4. Отображение информации о статусе возврата**
```csharp
Console.WriteLine("IsBounced: " + result.IsBounced);
Console.WriteLine("Action: " + result.Action);
Console.WriteLine("Recipient: " + result.Recipient);
```
*Почему?*: Предоставляет немедленную обратную связь о статусе возврата, включая предпринятые действия и соответствующего получателя.

**5. Отображение дополнительных сведений об отказах**
```csharp
Console.WriteLine("Reason: " + result.Reason);
Console.WriteLine("Status: " + result.Status);
```
*Объяснение*: предоставляет больше контекста, показывая причину возврата и ее текущий статус, помогая диагностировать проблемы.

**6. Получите адрес получателя исходного сообщения (если доступен)**
```csharp
if (result.OriginalMessage != null && result.OriginalMessage.To.Count > 0)
{
    Console.WriteLine("OriginalMessage ToAddress 1: " + result.OriginalMessage.To[0].Address);
}
```
*Цель*: Доступ и отображение исходного адреса получателя из возвращенного сообщения, если он доступен.
