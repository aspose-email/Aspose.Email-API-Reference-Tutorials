## Руководство по внедрению

### Отправить электронное письмо с функцией альтернативного текста

#### Обзор
Эта функция позволяет отправлять электронные письма как с содержимым HTML, так и с альтернативным текстом с помощью Aspose.Email для .NET, обеспечивая совместимость со всеми почтовыми клиентами и устройствами.

#### Пошаговая реализация

**1. Инициализируйте MailMessage**

Начните с создания экземпляра `MailMessage` класс и настройте отправителя, получателя и текст сообщения:

```csharp
using System;
using Aspose.Email.Mime;
using Aspose.Email.Clients.Smtp;

class SendEmailWithAlternateTextFeature
{
    public static void Execute()
    {
        // Создать новый экземпляр MailMessage
        MailMessage message = new MailMessage();
        
        // Укажите адрес «От» и адрес электронной почты получателя.
        message.From = "sender@sender.com";
        message.To.Add("receiver@receiver.com");

        // Добавить текст в текстовом формате
        message.Body = "This is Plain Text Body";

        // Добавить альтернативный вид HTML для клиентов, которые его поддерживают
        AlternateView alternateView = AlternateView.CreateAlternateViewFromString(
            "<html><body>This is the <b>HTML</b> version of the email.</body></html>",
            null,
            MediaTypeNames.Text.Html);
        message.AlternateViews.Add(alternateView);
    }
}
```

**2. Настройте SMTP-клиент**

Настройте свой `SmtpClient` с данными сервера для отправки электронного письма:

```csharp
// Создать и настроить экземпляр SmtpClient
SmtpClient client = new SmtpClient();
client.Host = "smtp.server.com";  // Замените на ваш SMTP-сервер
client.Username = "Username";     // Ваше имя пользователя для аутентификации
client.Password = "Password";     // Ваш пароль аутентификации
client.Port = 25;                 // Обычно порт по умолчанию — 25.

try
{
    // Отправьте сообщение электронной почты с помощью метода SmtpClient.Send
    client.Send(message);
}
catch (Exception ex)
{
    // Обработка исключений во время отправки
    System.Diagnostics.Trace.WriteLine(ex.ToString());
}
```

### Настройка почтового клиента для отправки писем

#### Обзор
В этом разделе показано, как настроить SMTP-клиент для отправки писем.

**1. Инициализация и настройка данных сервера**

Создать новый `SmtpClient` экземпляр и настройте необходимые параметры сервера:

```csharp
using Aspose.Email.Clients.Smtp;

class EmailClientConfigurationFeature
{
    public static void Execute()
    {
        SmtpClient client = new SmtpClient();
        client.Host = "smtp.server.com";  // Адрес хост-сервера SMTP
        client.Username = "Username";     // Имя пользователя для аутентификации на сервере
        client.Password = "Password";     // Пароль для аутентификации на сервере
        client.Port = 25;                 // Номер порта, используемый SMTP-сервером (обычно по умолчанию 25)
    }
}
```
