## Руководство по внедрению

### Отправка электронного письма с альтернативным текстом

Эта функция позволяет отправлять электронные письма, содержащие как HTML, так и обычный текстовый контент, используя альтернативные представления. Выполните следующие действия:

#### Шаг 1: Создание экземпляра MailMessage
```csharp
MailMessage message = new MailMessage();
```

#### Шаг 2: Задайте поля «Откуда» и «Куда»
```csharp
message.From = "sender@sender.com";
message.To.Add("receiver@receiver.com");
```
Укажите здесь адреса электронной почты отправителя и получателя.

#### Шаг 3: Создайте AlternateView с альтернативным текстом
```csharp
AlternateView alternate = AlternateView.CreateAlternateViewFromString("This is the alternate text");
```
The `AlternateView` класс определяет текстовую версию содержимого вашего электронного письма, гарантируя его корректное отображение в клиентах, не поддерживающих HTML.

#### Шаг 4: Добавьте альтернативное представление к объекту MailMessage
```csharp
message.AlternateViews.Add(alternate);
```

#### Шаг 5: Настройка и создание экземпляра SmtpClient
```csharp
SmtpClient client = new SmtpClient();
client.Host = "smtp.server.com";
client.Username = "Username";
client.Password = "Password";
client.Port = 25;
```
Замените значения заполнителей фактическими данными вашего SMTP-сервера для аутентификации.

#### Шаг 6: Отправьте электронное сообщение
```csharp
try
{
    client.Send(message);
}
catch (Exception ex)
{
    System.Diagnostics.Trace.WriteLine(ex.ToString());
}
```
На этом этапе предпринимается попытка отправить электронное письмо и регистрируются все исключения, возникшие в ходе процесса.

### Настройка SMTP-клиента Aspose.Email

Для успешной отправки писем настройте `SmtpClient` правильно:

#### Шаг 1: Создание экземпляра SmtpClient
```csharp
SmtpClient client = new SmtpClient();
```

#### Шаг 2: Настройте параметры SMTP-сервера.
- **Хозяин**: Адрес вашего SMTP-сервера.
- **Имя пользователя и пароль**: Учетные данные для аутентификации.
- **Порт**: Обычно устанавливается на уровне 25, но может отличаться в зависимости от вашего провайдера.

Обязательно замените все значения заполнителей фактическими учетными данными и данными сервера.
