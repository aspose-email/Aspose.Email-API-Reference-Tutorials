## Руководство по внедрению

### Создание ежегодно повторяющейся задачи

Эта функция демонстрирует, как настроить ежегодно повторяющуюся задачу, которая повторяется бесконечно в фиксированную дату. Мы будем использовать `MapiCalendarMonthlyRecurrencePattern` чтобы достичь этого.

#### Шаг 1: Настройте часовой пояс и даты

Сначала определите смещение местного часового пояса для точного расчета даты и времени:

```csharp
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan timeSpan = localZone.GetUtcOffset(DateTime.Now);

DateTime StartDate = new DateTime(2015, 7, 1).Add(timeSpan);
DateTime DueDate = new DateTime(2015, 7, 1).Add(timeSpan);
```

#### Шаг 2: Инициализация MapiTask

Создать `MapiTask` с желаемой темой и текстом:

```csharp
MapiTask task = new MapiTask("This is test task", "Sample Body", StartDate, DueDate);
task.State = MapiTaskState.NotAssigned;
```

#### Шаг 3: Настройте шаблон повторения

Настройте шаблон повторения, используя `MapiCalendarMonthlyRecurrencePattern`:

```csharp
var recurrence = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15, // День месяца для повторения.
    Period = 12, // Происходит каждые 12 месяцев (ежегодно).
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.NeverEnd, // Неопределенное количество повторений.
};
task.Recurrence = recurrence;

if (recurrence.OccurrenceCount == 0)
{
    recurrence.OccurrenceCount = 1;
}
```

#### Шаг 4: Сохраните задачу

Наконец, сохраните задачу в желаемом месте:

```csharp
// Раскомментируйте и замените на путь к выходному каталогу.
task.Save("YOUR_OUTPUT_DIRECTORY\SetYearlyNeverEndRecurrence_out.msg", TaskSaveFormat.Msg);
```

### Советы по устранению неполадок

- Убедитесь, что настройки часового пояса правильные, чтобы избежать ошибок даты/времени.
- Проверьте `MapiTask` свойства устанавливаются точно перед сохранением.

## Раздел часто задаваемых вопросов

1. **Могу ли я изменить частоту повторения?**
   Да, отрегулируйте `Period` свойство в шаблоне повторения для установки различных частот.

2. **Что делать, если мой часовой пояс изменится?**
   Обновить `localZone` и пересчитать временной промежуток, чтобы отразить точные настройки даты и времени.

3. **Как остановить повторяющуюся задачу?**
   Изменить `EndType` свойство или удалите задачу из вашей системы хранения.

4. **Можно ли использовать Aspose.Email .NET бесплатно?**
   Пробная версия доступна бесплатно, но для коммерческого использования требуется приобретение лицензии.

5. **Можно ли интегрировать это с другими системами?**
