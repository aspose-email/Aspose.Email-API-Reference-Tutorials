## Руководство по внедрению

### Создание и настройка MapiTask
Создание `MapiTask` включает в себя настройку основных атрибутов, таких как имя, текст, дата начала, дата выполнения и состояние.

#### Создание задачи
```csharp
using Aspose.Email.Mapi;

DateTime StartDate = new DateTime(2015, 7, 16);
DateTime DueDate = new DateTime(2015, 7, 16);

// Создать новый экземпляр MapiTask
task = new MapiTask("This is test task", "Sample Body", StartDate, DueDate);

// Установить состояние задачи на NotAssigned
task.State = MapiTaskState.NotAssigned;
```
**Объяснение**: Здесь мы создаем экземпляр `MapiTask` с именем, телом, датой начала и датой выполнения. Мы также устанавливаем его начальное состояние.

### Установите шаблон ежедневного повторения для MapiTask
Добавьте схему ежедневного повторения, чтобы гарантировать, что задача будет повторяться бесконечно.

#### Установка шаблона повторения
```csharp
var record = new MapiCalendarDailyRecurrencePattern
{
    PatternType = MapiCalendarRecurrencePatternType.Day,
    Period = 1, // Задача повторяется каждый день
    EndType = MapiCalendarRecurrenceEndType.NeverEnd, // Повторение никогда не заканчивается.
};

// Назначьте шаблон повторения задаче
task.Recurrence = record;
```
**Объяснение**: Этот фрагмент определяет ежедневную повторяющуюся схему, которая не закончится. `PatternType` установлен на `Day`, и `Period` указывает интервал в днях между событиями.

### Сохранить MapiTask в файл
Наконец, сохраните настроенную задачу как файл сообщения Outlook.

#### Сохранение задачи
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Замените на путь к каталогу вашего документа.

// Сохраните MapiTask в файле .msg.
task.Save(dataDir + "/SetDailyNeverEndRecurrence_out.msg", TaskSaveFormat.Msg);
```
**Объяснение**Этот код сохраняет вашу задачу в `.msg` файл, который можно открыть в Outlook.
