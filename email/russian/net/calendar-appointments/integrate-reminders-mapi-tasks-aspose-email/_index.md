### Создание задачи MAPI с напоминаниями
Эта функция позволяет добавлять напоминания непосредственно к вашим задачам. Вот как это можно сделать:

#### Шаг 1: Определите каталог данных
Начните с настройки пути к каталогу для хранения ваших документов:
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY"; // Замените фактическим путем к каталогу вашего документа.
```

#### Шаг 2: Создание и настройка задачи MAPI
Создайте новый экземпляр `MapiTask` и настройте его свойства, включая напоминания:
```csharp
// Инициализировать новую задачу MAPI
MapiTask testTask = new MapiTask("Task with Reminder", "This is a sample task.", DateTime.Now, DateTime.Now.AddDays(7));

// Настройте параметры напоминания
testTask.ReminderSet = true;
testTask.ReminderTime = DateTime.Now.AddMinutes(30); // Установите время напоминания
```

#### Объяснение
- `MapiTask`: Представляет объект задачи MAPI.
- `ReminderSet`: Логическое значение, указывающее, включено ли напоминание.
- `ReminderTime`: Указывает, когда должно срабатывать напоминание.

### Советы по устранению неполадок
- **Общие проблемы**: Убедитесь, что путь к каталогу указан правильно, чтобы избежать ошибок «файл не найден».
