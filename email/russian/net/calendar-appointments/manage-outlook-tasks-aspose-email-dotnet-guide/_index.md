## Руководство по внедрению

### Создание и настройка задачи MAPI
В этом разделе показано, как создать задачу Outlook с использованием функциональности MAPI Aspose.Email в .NET.

#### Шаг 1: Определите каталог документов
Укажите путь, где будут храниться ваши документы:
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";
```

#### Шаг 2: Создание и настройка задачи
Использовать `MapiTask` для создания новой задачи с определенными свойствами, такими как название, описание, дата начала, дата выполнения и т. д.

```csharp
using Aspose.Email.Mapi;

// Создать задачу MAPI
class Program
{
    static void Main(string[] args)
    {
        MapiTask task = new MapiTask("To Do", "Just click and type to add a new task", DateTime.Now, DateTime.Now.AddDays(3));

        // Установить различные свойства задачи
        task.PercentComplete = 20;
        task.EstimatedEffort = 2000; // В течение нескольких минут
        task.ActualEffort = 20;
        task.History = MapiTaskHistory.Assigned;
        task.LastUpdate = DateTime.Now;

        // Назначение права собственности и делегирование информации
        task.Users.Owner = "Darius";
        task.Users.LastAssigner = "Harkness";
        task.Users.LastDelegate = "Harkness";
        task.Users.Ownership = MapiTaskOwnership.AssignersCopy;
    }
}
```

### Управление PST-файлами и добавление к ним задач
Узнайте, как управлять PST-файлами и добавлять задачи с помощью Aspose.Email.

#### Шаг 1: Определите путь к выходному файлу PST
Укажите путь для вашего выходного PST-файла. Если он существует, удалите его, чтобы начать заново:
```csharp
string alreadyCreated = dataDir + "AddMapiTaskToPST_out.pst";

if (File.Exists(alreadyCreated))
{
    File.Delete(alreadyCreated); // Удалить, если существует, чтобы начать заново
}
```

#### Шаг 2: Создайте PST-файл и добавьте задачу
Создайте новый PST-файл, настройте папку для задач и добавьте свою задачу MAPI.

```csharp
using System.IO;
using Aspose.Email.Storage.Pst;

class Program
{
    static void Main(string[] args)
    {
        using (PersonalStorage personalStorage = PersonalStorage.Create(dataDir + "AddMapiTaskToPST_out.pst", FileFormatVersion.Unicode))
        {
            FolderInfo taskFolder = personalStorage.CreatePredefinedFolder("Tasks", StandardIpmFolder.Tasks); // Создайте папку «Задачи» в PST
            taskFolder.AddMapiMessageItem(task); // Добавьте настроенную задачу MAPI в эту папку
        }
    }
}
```
