## Руководство по внедрению

В этом разделе описывается, как создать встречу в формате ICS и загрузить ее обратно в приложение. Каждая функция разбирается пошагово.

### Функция 1: Создание встречи в формате ICS

Создание встречи включает в себя настройку различных данных, таких как место проведения, краткое описание и участники, а затем сохранение этой информации в общепринятом формате ICS.

#### Шаг 1: Определите детали встречи
Начните с определения ключевых свойств вашей встречи, таких как ее местоположение, резюме, описание, время начала, время окончания, организатор и участники. Вот как вы можете это сделать:

```csharp
// Создать и инициализировать экземпляр класса Appointment
Appointment appointment = new Appointment(
    "Meeting Room 3 at Office Headquarters", // Расположение
    "Monthly Meeting",                        // Краткое содержание
    "Please confirm your availability.",     // Описание
    new DateTime(2015, 2, 8, 13, 0, 0),       // Дата начала
    new DateTime(2015, 2, 8, 14, 0, 0),       // Дата окончания
    "from@domain.com",                        // Организатор
    "attendees@domain.com");                 // Участники
```

#### Шаг 2: Установите дополнительные свойства

Вы можете задать дополнительные свойства, такие как даты создания и последнего изменения, чтобы отслеживать, когда была назначена или обновлена встреча:

```csharp
// Установить дополнительные свойства назначения
appointment.CreatedDate = new DateTime(2018, 9, 15, 0, 0, 0, DateTimeKind.Utc);
appointment.LastModifiedDate = new DateTime(2018, 9, 16, 0, 0, 0, DateTimeKind.Utc);
```

#### Шаг 3: Сохраните запись на прием

Сохраните назначение в формате ICS в указанном каталоге. Это позволяет легко делиться назначениями или хранить их вне:

```csharp
// Укажите путь для сохранения файла назначения
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string dstEmail = dataDir + "/test.ics";

// Сохранить назначение на диск в формате ICS
appointment.Save(dstEmail, AppointmentSaveFormat.Ics);
```

### Функция 2: Загрузка записи на прием из файла ICS

Загрузка записи на прием включает чтение сохраненного файла ICS и извлечение его данных для отображения или дальнейшей обработки.

#### Шаг 1: Загрузите файл ICS

Используйте `Appointment.Load` Метод чтения сведений о ранее сохраненной встрече:

```csharp
// Укажите путь для загрузки файла назначения
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string dstEmail = dataDir + "/test.ics";

// Загрузить назначение из ранее сохраненного файла ICS
Appointment loadedAppointment = Appointment.Load(dstEmail);
```

#### Шаг 2: Отображение деталей встречи

Извлеките и отобразите различные свойства загруженной встречи, такие как ее краткое содержание, место проведения, дата начала и участники:

```csharp
// Отобразить информацию о назначении на экране (заменить соответствующим выводом в вашем приложении)
Console.WriteLine("Summary: " + loadedAppointment.Summary);
Console.WriteLine("Location: " + loadedAppointment.Location);
Console.WriteLine("Description: " + loadedAppointment.Description);
Console.WriteLine("Start date: " + loadedAppointment.StartDate);
Console.WriteLine("End date: " + loadedAppointment.EndDate);
Console.WriteLine("Organizer: " + loadedAppointment.Organizer);
Console.WriteLine("Attendees: " + loadedAppointment.Attendees);
Console.WriteLine("Created Date: " + loadedAppointment.CreatedDate);
Console.WriteLine("Last Modified Date: " + loadedAppointment.LastModifiedDate);
```
