### Подключиться к веб-службе Exchange (EWS)

**Обзор:** Установление соединения с EWS обеспечивает доступ к данным вашего почтового ящика и календаря, подготавливая почву для задач управления встречами.

1. **Инициализируйте IEWSClient:**
   Создать экземпляр `IEWSClient` используя учетные данные, которые обеспечивают доступ к вашей конечной точке EWS.
   
   ```csharp
   // Создайте и настройте экземпляр IEWSClient с учетными данными.
   using Aspose.Email.Clients.Exchange;
   using Aspose.Email.Clients.Exchange.WebService;

   IEWSClient client = EWSClient.GetEWSClient(
       "https://outlook.office365.com/ews/exchange.asmx",
       "username",
       "password",
       "domain"
   );
   ```

### Фильтрация назначений по диапазону дат с использованием EWS

**Обзор:** Фильтрация встреч по диапазону дат помогает сосредоточиться на определенных периодах, улучшая управление данными и их анализ.

1. **Определите даты начала и окончания:**
   Укажите диапазон дат для фильтрации.
   
   ```csharp
   using System;

   DateTime startTime = new DateTime(2017, 9, 15);
   DateTime endTime = new DateTime(2017, 10, 10);
   ```

2. **Создайте запрос для фильтрации встреч:**
   Использовать `ExchangeQueryBuilder` для построения вашего запроса на основе указанного диапазона дат.
   
   ```csharp
   using Aspose.Email.Tools.Search;

   ExchangeQueryBuilder builder = new ExchangeQueryBuilder();
   builder.Appointment.Start.Since(startTime);
   builder.Appointment.End.BeforeOrEqual(endTime);
   MailQuery query = builder.GetQuery();
   ```

3. **Получить отфильтрованные встречи:**
   Выполните запрос, чтобы получить записи на прием в указанный вами диапазон дат.
   
   ```csharp
   Appointment[] appointmentsByDate = client.ListAppointments(query);
   ```

### Фильтр встреч по повторению с помощью EWS

**Обзор:** Определение неповторяющихся встреч может иметь важное значение для задач, требующих однократного планирования.

1. **Создайте запрос для определения неповторяющихся встреч:**
   Использовать `ExchangeQueryBuilder` для фильтрации повторяющихся встреч.
   
   ```csharp
   ExchangeQueryBuilder builderRecurrence = new ExchangeQueryBuilder();
   builderRecurrence.Appointment.IsRecurring.Equals(false);
   MailQuery queryNonRecurring = builderRecurrence.GetQuery();
   ```

2. **Получить разовые встречи:**
   Выполните запрос, чтобы получить список встреч, которые не повторяются.
   
   ```csharp
   Appointment[] appointmentsByRecurrence = client.ListAppointments(queryNonRecurring);
   ```
