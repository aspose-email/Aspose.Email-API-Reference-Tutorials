### Создание и отправка запросов на встречи по электронной почте

#### Обзор
Создание запроса на встречу включает настройку сообщения электронной почты с подробностями встречи с помощью Aspose.Email. Эта функция автоматизирует процесс прикрепления приглашений календаря к электронным письмам.

#### Пошаговая реализация:

##### 1. Настройте MailMessage

Начните с создания `MailMessage` экземпляр, который будет служить контейнером вашей электронной почты:

```csharp
MailMessage msg = new MailMessage();
msg.From = "newcustomeronnet@gmail.com";
msg.To = "person1@domain.com, person2@domain.com, person3@domain.com, asposetest123@gmail.com";
```

##### 2. Запишитесь на прием

Создайте `Appointment` экземпляр с необходимыми подробностями:

```csharp
Appointment app = new Appointment(
    "Room 112",
    new DateTime(2015, 7, 17, 13, 0, 0),
    new DateTime(2015, 7, 17, 14, 0, 0),
    msg.From,
    msg.To);
```

##### 3. Настройте детали встречи

Составьте резюме и описание встречи:

```csharp
app.Summary = "Release Meeting";
app.Description = "Discuss the next release";
```

##### 4. Прикрепить запись на прием к электронному письму

Добавьте запись на прием в качестве альтернативного варианта в свое электронное письмо:

```csharp
msg.AddAlternateView(app.RequestApointment());
```

### Настройка SMTP-клиента для отправки писем

#### Обзор
Для отправки писем настройте `SmtpClient` с данными вашего SMTP-сервера и учетными данными.

#### Пошаговая реализация:

##### 1. Настройте SmtpClient

Создайте метод для возврата настроенного `SmtpClient`:

```csharp
private static SmtpClient GetSmtpClient()
{
    SmtpClient client = new SmtpClient(
        "smtp.gmail.com", 587,
        "your.email@gmail.com",
        "your.password");
    
    client.SecurityOptions = SecurityOptions.Auto;
    return client;
}
```

##### 2. Отправьте электронное письмо

Использовать `try-catch` блок для обработки потенциальных исключений при отправке:

```csharp
SmtpClient client = GetSmtpClient();
try
{
    client.Send(msg);
}
catch (Exception ex)
{
    Console.WriteLine(ex.ToString());
}
```
