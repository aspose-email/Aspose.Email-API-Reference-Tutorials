## Руководство по внедрению
Давайте шаг за шагом рассмотрим создание и сохранение задачи Outlook.

### Создание объекта MapiTask
А `MapiTask` объект представляет задачу Outlook. Начните с инициализации его с необходимыми свойствами:

#### Шаг 1: Определите задачу
```csharp
MapiTask task = new MapiTask(
    «Что делать», 
    "Just click and type to add new task", 
    DateTime.Now, 
    DateTime.Now.AddDays(3)
);
```
- **"To Do"** это тема.
- Описание содержит дополнительную информацию.
- Даты начала и окончания устанавливаются на сегодняшнюю дату и через три дня.

#### Шаг 2: Определите прогресс и усилия
```csharp
task.PercentComplete = 20;
task.EstimatedEffort = 2000; // В течение нескольких минут
```
- Определите процент выполнения задачи.
- Задайте предполагаемые усилия в минутах, чтобы отслеживать затраченное время.

#### Шаг 3: История задач и обновления
```csharp
task.History = MapiTaskHistory.Assigned;
task.LastUpdate = DateTime.Now;
```
- Запишите, когда задача была назначена или обновлена в последний раз, для лучшего отслеживания.

### Настройка права собственности и компаний
Укажите данные о владельце и связанных компаниях:

```csharp
task.Users.Owner = "Darius";
task.Users.LastAssigner = "Harkness";
task.Users.LastDelegate = "Harkness";
task.Users.Ownership = MapiTaskOwnership.AssignersCopy;

task.Companies = new string[] { "company1", "company2", "company3" };
```

### Категоризация и добавление метаданных
Используйте категории для организации:

```csharp
task.Categories = new string[] { "category1", "category2", "category3" };
task.Mileage = "Some test mileage";
task.Billing = "Test billing information";
task.Users.Delegator = "Test Delegator";
```

### Завершение свойств задачи
Установить чувствительность и статус:

```csharp
task.Sensitivity = MapiSensitivity.Personal;
task.Status = MapiTaskStatus.Complete;

// При необходимости скорректируйте предполагаемые усилия.
task.EstimatedEffort = 5; // В течение нескольких минут
```

### Сохранение задачи как MSG-файла
Наконец, сохраните свою задачу:

```csharp
string outputPath = "@YOUR_OUTPUT_DIRECTORY/MapiTask.msg";
task.Save(outputPath, TaskSaveFormat.Msg);
```

Обязательно замените `@YOUR_OUTPUT_DIRECTORY` на фактический путь к каталогу, в котором вы хотите сохранить файл.
