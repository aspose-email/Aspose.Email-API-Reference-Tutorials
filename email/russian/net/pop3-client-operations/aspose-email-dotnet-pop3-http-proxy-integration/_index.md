### Доступ к почтовому ящику POP3 через HTTP-прокси
Эта функция позволяет вашему приложению подключаться к серверу POP3 с использованием промежуточного HTTP-прокси, что имеет решающее значение в ограниченных сетевых средах.

#### Создать экземпляр HttpProxy
Начните с создания `HttpProxy` экземпляр с необходимыми данными хоста и порта. Это настраивает ваше соединение через указанный прокси:
```csharp
// Определите настройки прокси-сервера
HttpProxy proxy = new HttpProxy("18.222.124.59", 8080); // Заменить на реальный адрес и порт прокси-сервера
```

#### Инициализировать POP3-клиент
Настраивать `Pop3Client` для взаимодействия с почтовым ящиком через HTTP-прокси:
```csharp
// Настройте параметры вашего почтового сервера
Pop3Client client = new Pop3Client("pop.example.com", "username", "password");

// Назначьте экземпляр HttpProxy клиенту
client.Proxy = proxy;
```
- **Параметры**:
  - `"pop.example.com"`: Имя хоста сервера POP3.
  - `"username"` и `"password"`Учетные данные для доступа к вашему почтовому ящику.

#### Подключение и получение электронной почты
После завершения настройки подключитесь к серверу и загрузите электронные письма:
```csharp
try
{
    client.Connect(true); // Используйте SSL, если этого требует сервер.
    int messageCount = client.GetMessageCount();
    
    Console.WriteLine($"Total Messages: {messageCount}");

    for (int i = 1; i <= messageCount; i++)
    {
        MailMessage msg = client.FetchMessage(i);
        Console.WriteLine($"Subject: {msg.Subject}");
    }
}
catch (Exception ex)
{
    Console.WriteLine("Error: " + ex.Message);
}
fine
{
    client.Dispose();
}
```
- **Возвращаемые значения**:
  - `GetMessageCount()`: Возвращает общее количество сообщений в папке «Входящие».
  - `FetchMessage(int)`: Извлекает определенное электронное письмо по его индексу сообщения.

#### Советы по устранению неполадок
Распространенные проблемы включают ошибки сетевого подключения или сбои аутентификации. Убедитесь, что настройки прокси-сервера верны и что у вас есть действительные учетные данные сервера. Также проверьте, требуется ли SSL/TLS сервером POP3 для безопасных подключений.
