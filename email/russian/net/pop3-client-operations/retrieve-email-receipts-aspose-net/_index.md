## Руководство по внедрению

В этом разделе мы рассмотрим, как получить информацию о прочтении и доставке.

### Получение информации о квитанции

#### Обзор:
Эта функция позволяет вам извлекать и анализировать, были ли отправленные вами электронные письма успешно открыты и доставлены.

#### Шаг 1: Загрузите электронное письмо
Начните с загрузки `.msg` файл, содержащий сообщение электронной почты. Здесь мы начинаем наше путешествие по извлечению информации о квитанции.

**Фрагмент кода:**
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
MapiMessage msg = MapiMessage.FromFile(dataDir + "\TestMessage.msg");
```

#### Шаг 2: Перебор получателей
Для каждого получателя проверьте статус прочтения и доставки.

**Доступ к информации о получателе:**
```csharp
foreach (MapiRecipient recipient in msg.Recipients)
{
    Console.WriteLine(string.Format("Recipient: {0}", recipient.DisplayName));
    
    // Проверьте информацию о получении
    if (recipient.MessageStatus == MapiMessageStatus.Read && recipient.ReceiptType == MapiRecipientReceiptType.Read)
    {
        Console.WriteLine("Read Receipt Received.");
    }

    if (recipient.MessageStatus == MapiMessageStatus.Delivered)
    {
        Console.WriteLine("Delivery Receipt Received.");
    }
}
```

**Объяснение:**
- **MapiMessage.FromFile**: Загружает сообщение из указанного файла.
- **msg.Получатели**: Предоставляет доступ к данным каждого получателя.
- **Статус сообщения и тип получения**: Используется для определения статуса получения.

### Советы по устранению неполадок:
- Убедитесь, что ваш `.msg` файлы правильно отформатированы и доступны.
- Убедитесь, что Aspose.Email правильно установлен и указан в вашем проекте.
