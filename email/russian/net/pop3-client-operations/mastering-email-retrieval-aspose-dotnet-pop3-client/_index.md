## Руководство по внедрению

Давайте разобьем процесс на логические этапы на основе ключевых особенностей.

### Создание и настройка Pop3Client

**Обзор:**
Эта функция демонстрирует, как создать экземпляр `Pop3Client` и настроить параметры его подключения.

#### Шаг 1: Создайте новый экземпляр

Начните с создания нового экземпляра `Pop3Client` сорт:

```csharp
Pop3Client client = new Pop3Client();
```

#### Шаг 2: Настройте параметры подключения

Задайте необходимые параметры, такие как хост, имя пользователя, пароль, порт и параметры безопасности:

```csharp
client.Host = "pop.gmail.com"; // Укажите адрес POP3-сервера.
client.Username = "your.username@gmail.com"; // Укажите имя пользователя электронной почты.
client.Password = "your.password"; // Установите пароль электронной почты.
client.Port = 995; // Используйте порт 995 для SSL-соединений.
client.SecurityOptions = SecurityOptions.Auto; // Автоматически определять параметры безопасности.
```

**Объяснение:**
- **Хозяин:** Адрес сервера POP3. Для Gmail используйте `pop.gmail.com`.
- **Имя пользователя и пароль:** Ваши учетные данные электронной почты.
- **Порт:** 995 обычно используется для защищенных соединений с SSL/TLS.
- **Параметры безопасности:** Установить на `Auto` чтобы позволить клиенту автоматически определять протокол безопасности.

**Советы по устранению неполадок:**
- Убедитесь, что ваш брандмауэр или антивирус не блокирует соединение.
- Если вы столкнулись с ошибками аутентификации, дважды проверьте свои учетные данные и настройки сервера.

### Получение размера почтового ящика, информации и количества сообщений

**Обзор:**
Эта функция показывает, как получить размер почтового ящика, информацию и количество сообщений с помощью `Pop3Client` пример.

#### Шаг 1: Получите размер почтового ящика

Используйте `GetMailboxSize()` метод:

```csharp
long nSize = client.GetMailboxSize();
```

#### Шаг 2: Получите подробную информацию

Получите подробную информацию о почтовом ящике, включая количество сообщений и занятый размер:

```csharp
Pop3MailboxInfo info = client.GetMailboxInfo();
int nMessageCount = info.MessageCount;
long nOccupiedSize = info.OccupiedSize;
```

**Объяснение:**
- **nРазмер:** Общий размер почтового ящика в байтах.
- **nMessageCount:** Количество сообщений в почтовом ящике.
- **nЗанятыйРазмер:** Место, занятое электронными письмами.
