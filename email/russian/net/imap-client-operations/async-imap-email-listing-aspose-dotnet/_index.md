## Руководство по внедрению
### Асинхронный список адресов электронной почты IMAP
Функция, которую мы реализуем, позволяет асинхронно выводить список сообщений с сервера IMAP, что идеально подходит для неблокируемых операций в вашем приложении.

#### Пошаговая реализация
**1. Инициализируйте ImapClient**
Начните с настройки `ImapClient` с данными вашего провайдера электронной почты:

```csharp
// Создайте и настройте экземпляр ImapClient
ImapClient client = new ImapClient();
client.Host = "imap.gmail.com";
client.Username = "your.username@gmail.com";
client.Password = "your.password";
client.Port = 993;
client.SecurityOptions = SecurityOptions.Auto;
```

**2. Составьте поисковый запрос**
Использовать `ImapQueryBuilder` чтобы создать запрос, фильтрующий электронные письма по теме:

```csharp
// Создайте поисковый запрос для писем с темой «Тема»
ImapQueryBuilder builder = new ImapQueryBuilder();
builder.Subject.Contains("Subject");
MailQuery query = builder.GetQuery();
```

**3. Асинхронный вывод сообщений**
Выполните асинхронную операцию для вывода списка сообщений, соответствующих вашим критериям:

```csharp
try
{
    // Начать асинхронный вывод сообщений с использованием указанного запроса
    IAsyncResult asyncResult = client.BeginListMessages(query);

    // Завершите операцию и получите результаты
    ImapMessageInfoCollection messages = client.EndListMessages(asyncResult);
    
    foreach (var message in messages)
    {
        Console.WriteLine($"Subject: {message.Subject}");
    }
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    // Очистите ресурсы
    if (client != null) client.Dispose();
}
```
