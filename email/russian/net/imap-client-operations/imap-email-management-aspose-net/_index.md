## Руководство по внедрению
В этом разделе мы расскажем, как подключиться к серверу IMAP и получить информацию о почтовом ящике с помощью Aspose.Email для .NET.

### Подключиться к IMAP-серверу
Подключение к серверу IMAP подразумевает настройку клиента с данными вашего провайдера электронной почты. Вот как это сделать:

#### 1. Настройте параметры клиента
Сначала создайте новый экземпляр `ImapClient` и настройте его свойства:
```csharp
using Aspose.Email.Clients;
using Aspose.Email.Clients.Imap;

// Создать экземпляр ImapClient
ImapClient imapClient = new ImapClient();

// Установить данные сервера
imapClient.Host = "<HOST>"; // Замените <HOST> на адрес хоста вашего IMAP-сервера.
imapClient.Port = 993; // Стандартный порт для IMAP через SSL.
imapClient.Username = "<USERNAME>"; // Замените <USERNAME> на свое имя пользователя.
imapClient.Password = "<PASSWORD>"; // Замените <ПАРОЛЬ> своим паролем.

// Установить параметры безопасности
imapClient.SupportedEncryption = EncryptionProtocols.Tls;
imapClient.SecurityOptions = SecurityOptions.SSLImplicit;
```
**Объяснение:**
- `Host`: Адрес сервера IMAP.
- `Port`: Порт 993 обычно используется для защищенных соединений IMAP через SSL/TLS.
- `Username` и `Password`: Учетные данные, предоставленные вашей службой электронной почты.
- `SupportedEncryption`: Обеспечивает использование шифрования TLS.
- `SecurityOptions`: Настраивает клиент на использование неявной безопасности SSL.

#### Советы по устранению неполадок
Если у вас возникли проблемы с подключением:
- Проверьте данные хоста сервера, имя пользователя и пароль.
- Убедитесь, что порт 993 не заблокирован вашим брандмауэром или сетевой конфигурацией.
- Проверьте, требует ли ваш поставщик услуг электронной почты пароли, специфичные для приложений, для доступа третьих лиц.

### Получить информацию о почтовом ящике
После подключения к серверу IMAP получение информации о почтовом ящике становится простым:

#### Доступ к почтовым ящикам специального назначения
Использовать `ImapMailboxInfo` чтобы получить URI специальных почтовых ящиков, таких как «Входящие» и «Отправленные»:
```csharp
// Получить информацию о почтовом ящике
ImapMailboxInfo mailboxInfo = imapClient.MailboxInfo;

// URI доступа для почтовых ящиков специального назначения
string inboxUri = mailboxInfo.Inbox;
string draftsUri = mailboxInfo.DraftMessages;
string junkUri = mailboxInfo.JunkMessages;
string sentItemsUri = mailboxInfo.SentMessages;
string trashUri = mailboxInfo.Trash;
```
**Объяснение:**
- `ImapMailboxInfo`: Предоставляет информацию о почтовых ящиках, доступных на сервере IMAP.
- Специальные URI, такие как `inbox`, `drafts`и т. д., позволяют вам взаимодействовать с этими конкретными папками программно.
