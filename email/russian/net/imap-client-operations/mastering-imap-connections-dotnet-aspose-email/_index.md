## Руководство по внедрению

### Подключение к IMAP-серверу
#### Обзор
Эта функция позволяет вашему приложению устанавливать соединение с сервером IMAP, обеспечивая взаимодействие с электронными письмами, хранящимися на сервере.

#### Шаг 1: Создание экземпляра ImapClient
Создайте новый экземпляр `ImapClient` указав адрес хоста вместе с именем пользователя и паролем.
```csharp
// Замените «хост», «имя пользователя» и «пароль» фактическими значениями.
ImapClient client = new ImapClient("host", "username", "password");
```
**Объяснение:**
- **Хозяин**: Адрес сервера, на котором хранятся ваши электронные письма.
- **Имя пользователя и пароль**: Учетные данные для аутентификации соединения.

#### Шаг 2: Подключение и аутентификация
Используйте `Connect` метод установления соединения с сервером.
```csharp
client.Connect(true); // Используйте SSL для безопасных соединений.
```
**Объяснение:**
- **Использование SSL**: Установка его на `true` обеспечивает безопасное соединение, часто требуемое современными почтовыми серверами.

3. **Интеграция спам-фильтрации**: Интеграция со службами фильтрации спама для очистки почтовых ящиков.
