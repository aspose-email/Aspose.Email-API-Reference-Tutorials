### Базовая инициализация и настройка
Чтобы начать работу с Aspose.Email, вам сначала нужно инициализировать ImapClient. Вот как настроить его с параметрами безопасного соединения:
```csharp
using Aspose.Email.Clients;
using Aspose.Email.Clients.Imap;

public static void InitializeImapClient()
{
    ImapClient imapClient = new ImapClient();
    imapClient.Host = "<HOST>";
    imapClient.Port = 993; // Общий порт для SSL-соединений
    imapClient.Username = "<USERNAME>";
    imapClient.Password = "<PASSWORD>";
    imapClient.SupportedEncryption = EncryptionProtocols.Tls;
    imapClient.SecurityOptions = SecurityOptions.SSLImplicit;
}
```

## Руководство по внедрению

### Инициализировать ImapClient
Инициализация `ImapClient` необходим для настройки безопасного соединения с вашим сервером IMAP. Вот как вы можете это настроить:

#### Настройка хоста и порта
Укажите хост и номер порта сервера IMAP:
- **Хозяин**: Используйте доменное имя или IP-адрес вашего провайдера электронной почты.
- **Порт**: Обычно для SSL-соединений используется 993.
```csharp
imapClient.Host = "<HOST>";
imapClient.Port = 993;
```

#### Данные аутентификации
Укажите свое имя пользователя и пароль для аутентификации. Это позволит получить доступ к вашему почтовому аккаунту:
```csharp
imapClient.Username = "<USERNAME>";
imapClient.Password = "<PASSWORD>";
```

#### Протокол шифрования
Обеспечьте безопасную связь, установив поддерживаемый протокол шифрования:
```csharp
imapClient.SupportedEncryption = EncryptionProtocols.Tls;
```

### Список сообщений с сервера IMAP
После подключения вы сможете просмотреть все сообщения, доступные в вашем почтовом ящике:

#### Получить коллекцию сообщений
Использовать `ListMessages` чтобы получить коллекцию информации о сообщениях:
```csharp
ImapMessageInfoCollection messageInfoCol = imapClient.ListMessages();
int[] sequenceNumberAr = messageInfoCol.Select(mi => mi.SequenceNumber).ToArray();
string[] uniqueIdAr = messageInfoCol.Select(mi => mi.UniqueId).ToArray();
```

Чтобы получить определенные электронные письма, вы можете использовать их порядковые номера:
