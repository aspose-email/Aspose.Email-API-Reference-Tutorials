## Руководство по внедрению

Мы разобьем эту реализацию на две основные функции: создание `ExchangeClient` экземпляр и получение информации о почтовом ящике.

### Функция 1: Создание экземпляра ExchangeClient

#### Обзор
В этом разделе вы узнаете, как инициализировать `ExchangeClient`, выступающий в качестве шлюза для взаимодействия с функциями сервера Exchange.

#### Пошаговая реализация

**Инициализация учетных данных:**
Начните с настройки учетных данных подключения, включая URL-адрес сервера, имя пользователя, пароль и домен.

```csharp
using Aspose.Email.Clients.Exchange;

// Определить параметры подключения для Exchange Server
string serverUrl = "https://ИмяМашины/Обмен/ИмяПользователя";
string username = "Username";
string password = "password";
string domain = "domain";

// Создать экземпляр класса ExchangeClient
ExchangeClient client = new ExchangeClient(serverUrl, username, password, domain);
```

**Объяснение:**
- `serverUrl`: URL-адрес вашего сервера Exchange. 
- `username`, `password`, и `domain`: Для аутентификации требуются учетные данные.

### Функция 2: Получение информации о почтовом ящике с сервера Exchange

#### Обзор
Узнайте, как использовать `ExchangeClient` экземпляр для получения информации о почтовом ящике.

#### Пошаговая реализация

**Получить размер почтового ящика и подробную информацию:**
Используйте `GetMailboxSize()` и `GetMailboxInfo()` методы получения полной информации о почтовом ящике.

```csharp
try
{
    // Получить размер почтового ящика в байтах
    long mailboxSize = client.GetMailboxSize();

    // Получить подробную информацию о почтовом ящике
    ExchangeMailboxInfo mailboxInfo = client.GetMailboxInfo();
    
    // Примеры URI для демонстрации (замените реальными путями)
    string inboxUri = mailboxInfo.InboxUri;
    string sentItemsUri = mailboxInfo.SentItemsUri;
    string draftsUri = mailboxInfo.DraftsUri;
}
catch (Exception ex)
{
    throw new Exception("An error occurred while retrieving mailbox information: " + ex.Message);
}
```

**Объяснение:**
- `GetMailboxSize()`: Возвращает текущий размер вашего почтового ящика в байтах.
- `GetMailboxInfo()`: Предоставляет подробную информацию, включая URI для различных папок, таких как «Входящие», «Отправленные» и «Черновики».
