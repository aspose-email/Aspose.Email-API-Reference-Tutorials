## Руководство по внедрению
В этом разделе описывается процесс подключения к серверу Exchange и вывод списка входящих сообщений.

### Подключение к веб-службе Exchange
#### Обзор
Подключение к Microsoft Exchange Server позволяет приложениям взаимодействовать с почтовыми службами программным путем. Эта функция использует `IEWSClient` интерфейс, предоставленный Aspose.Email.

**Шаг 1: Создайте экземпляр `ExchangeWebServiceClient`**
```csharp
// Инициализируйте клиент, используя учетные данные сервера Exchange.
IEWSClient client = EWSClient.GetEWSClient("https://outlook.office365.com/ews/exchange.asmx", "Имя пользователя", "Пароль");
```
- **Объяснение параметров**: Заменять `"UserName"` и `"Password"` с реальными учетными данными Exchange. Убедитесь, что URL соответствует конфигурации вашего сервера.

**Шаг 2: Попытка подключения**
```csharp
try
{
    Console.WriteLine("Connected to the Exchange server successfully.");
}
catch (Exception ex)
{
    Console.Write(ex.Message);
}
```
- **Цель**: Этот код пытается установить соединение и выводит сообщение об успешном подключении или любых обнаруженных исключениях, помогая в устранении неполадок.

### Список сообщений из папки «Входящие»
#### Обзор
После подключения вы сможете просматривать сообщения в своем почтовом ящике. `ListMessages` метод извлекает информацию о сообщении.

**Шаг 1: Составьте список сообщений**
```csharp
// Предположим, что «клиент» инициализирован, как показано выше.
ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);
```
- **Объяснение**: Извлекает все сообщения из URI входящих сообщений, используя `ListMessages`.

**Шаг 2: Отображение сведений о сообщении**
```csharp
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Message ID: " + msgInfo.MessageId);
    Console.WriteLine("Unique URI: " + msgInfo.UniqueUri);
}
```
- **Цель**: Просматривает каждое сообщение, отображая основные данные, такие как тема и отправитель.
