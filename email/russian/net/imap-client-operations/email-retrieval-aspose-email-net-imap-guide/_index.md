## Настройка Aspose.Email для .NET
### Установка
Интеграция Aspose.Email в ваш проект проста. Выберите один из следующих методов:

**Использование .NET CLI:**
```bash
dotnet add package Aspose.Email
```

**Через менеджер пакетов в Visual Studio:**

**Пользовательский интерфейс менеджера пакетов NuGet:**

#### Базовая инициализация и настройка
Начните с создания `ImapClient` пример:
```csharp
using Aspose.Email.Clients.Imap;

const string host = "your.imap.host";
const int port = 143; // Стандартный незашифрованный порт IMAP
const string username = "user@host.com";
const string password = "password";

ImapClient client = new ImapClient(host, port, username, password);
```
Обрабатывайте исключения, чтобы гарантировать успешные соединения.

### Функция: получение писем, пришедших сегодня
**Обзор:**
С легкостью извлекайте электронные письма, пришедшие сегодня, используя возможности запросов Aspose.Email.

#### Шаги:
1. **Создайте запрос для сегодняшних писем:**
```csharp
using Aspose.Email.Tools.Search;

MailQueryBuilder builder = new MailQueryBuilder();
builder.InternalDate.On(DateTime.Now);
```
2. **Выполнение и извлечение сообщений:**
```csharp
MailQuery query = builder.GetQuery();
ImapMessageInfoCollection messages = client.ListMessages(query);
Console.WriteLine($"Number of messages found: {messages.Count}");
```
### Функция: получение писем по заданному диапазону дат
**Обзор:**
Получайте доступ к электронным письмам, полученным в течение определенного диапазона дат, расширяя возможности фильтрации электронной почты.

#### Шаги:
1. **Определите запрос диапазона дат:**
```csharp
builder = new MailQueryBuilder();
builder.InternalDate.Before(DateTime.Now);
builder.InternalDate.Since(DateTime.Now.AddDays(-7));
```
2. **Выполнение и извлечение сообщений:**
```csharp
query = builder.GetQuery();
messages = client.ListMessages(query);
Console.WriteLine($"Number of messages found: {messages.Count}");
```
