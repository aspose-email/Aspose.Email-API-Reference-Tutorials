## Руководство по внедрению
Мы рассмотрим три ключевые функции: подключение к серверу IMAP, создание папки и перемещение сообщений.

### Подключение к серверу IMAP (H2)
#### Обзор
Подключение к серверу IMAP имеет основополагающее значение для задач управления электронной почтой. Aspose.Email упрощает это с помощью `ImapClient` сорт.

#### Этапы внедрения
##### Шаг 1: Инициализация ImapClient
Создайте новый экземпляр `ImapClient`, указав данные вашего сервера, номер порта (обычно 993 для SSL), имя пользователя и пароль:
```csharp
using (ImapClient client = new ImapClient("host.domain.com", 993, "username", "password"))
{
    Console.WriteLine("Connected to IMAP server successfully.");
}
```
**Объяснение**: `ImapClient` конструктор берет адрес хоста, порт, имя пользователя и пароль. Мы оборачиваем его в `using` заявление о надлежащем использовании ресурсов.

### Создание папки в учетной записи IMAP (H2)
#### Обзор
Распространена организация писем в папки. Эта функция проверяет наличие папки и создает ее при необходимости.

#### Этапы внедрения
##### Шаг 1: Проверьте существование папки
Используйте `ExistFolder` Метод проверки наличия нужной папки на сервере:
```csharp
string folderName = "YOUR_DOCUMENT_DIRECTORY";

if (!client.ExistFolder(folderName))
{
    client.CreateFolder(folderName);
    Console.WriteLine($"Folder '{folderName}' created successfully.");
}
```
**Объяснение**: Если `ExistFolder` возвращает false, мы переходим к созданию папки с помощью `CreateFolder`.

### Перемещение сообщения в учетной записи IMAP (H2)
#### Обзор
Перемещение сообщений между папками может помочь управлять рабочими процессами электронной почты. Эта функция демонстрирует перемещение электронной почты по ее уникальному идентификатору.

#### Этапы внедрения
##### Шаг 1: Добавить и переместить сообщение
Сначала выберите папку «Входящие» для работы с сообщениями. Затем создайте и добавьте новое сообщение, прежде чем переместить его в другую папку, используя его уникальный идентификатор:
```csharp
string folderName = "YOUR_OUTPUT_DIRECTORY";

if (!client.ExistFolder(folderName))
    client.CreateFolder(folderName);

client.SelectFolder(ImapFolderInfo.InBox);
MailMessage message = new MailMessage(
    "from@domain.com",
    "to@domain.com",
    "Unique Message Subject - " + Guid.NewGuid(),
    "This is the body of the email.");

string uniqueId = client.AppendMessage(ImapFolderInfo.InBox, message);
client.MoveMessage(uniqueId, folderName);
Console.WriteLine($"Moved message with unique ID '{uniqueId}' to '{folderName}'.");
```
**Объяснение**: После добавления нового сообщения в папку «Входящие» мы получаем его уникальный идентификатор. Этот идентификатор используется `MoveMessage` чтобы переместить его в нужную папку.
