## Руководство по внедрению

### Создание и сохранение задачи MAPI
Давайте начнем с создания простой задачи MAPI и сохранения ее в виде файла MSG. Эта функциональность помогает автоматизировать создание задач, обеспечивая согласованность и эффективность.

**Шаг 1: Определите свойства задачи**
Начните с определения дат начала и выполнения вашей задачи:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);
DateTime StartDate = new DateTime(2015, 7, 1).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 1).Add(ts);
```

**Шаг 2: Создание задачи MAPI**
Инициализировать `MapiTask` с вашими определенными свойствами:
```csharp
MapiTask task = new MapiTask("This is test task", "Sample Body", StartDate, DueDate);
task.State = MapiTaskState.NotAssigned;
```
В этом фрагменте:
- «Это тестовое задание» и «Пример текста» — тема и текст задания.
- `StartDate` и `DueDate` укажите время начала и окончания выполнения задачи.

**Шаг 3: Сохраните задачу**
Сохраните вашу задачу в формате MSG:
```csharp
task.Save(dataDir + "Monthly_out.msg", TaskSaveFormat.Msg);
```

### Настройка ежемесячного шаблона повторения для задачи MAPI
Затем настройте шаблон ежемесячного повторения для существующего объекта MAPI-задачи. Это идеально подходит для задач, которые должны повторяться через регулярные интервалы.

**Шаг 1: Определите шаблон повторения**
Создать `MapiCalendarMonthlyRecurrencePattern`:
```csharp
var rec = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15,
    Period = 12,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    OccurrenceCount = 3,
    WeekStartDay = DayOfWeek.Monday
};
```
Эта конфигурация устанавливает повторение задачи 15-го числа каждого месяца три раза в течение 12 месяцев.

**Шаг 2: Применить повторение к задаче**
Назначьте шаблон повторения вашему `MapiTask`:
```csharp
MapiTask taskWithRecurrence = new MapiTask("This is test task", "Sample Body", DateTime.Now, DateTime.Now.AddDays(30));
taskWithRecurrence.Recurrence = rec;
```

**Шаг 3: Сохраните задачу с повторением**
Сохраните повторяющуюся задачу как файл MSG:
```csharp
string dataDirOutput = "YOUR_OUTPUT_DIRECTORY";
taskWithRecurrence.Save(dataDirOutput + "Monthly_out_with_recurrence.msg", TaskSaveFormat.Msg);
```
