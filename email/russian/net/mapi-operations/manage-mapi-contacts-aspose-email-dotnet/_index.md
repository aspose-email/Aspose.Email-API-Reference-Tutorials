## Руководство по внедрению

В этом разделе мы рассмотрим создание и сохранение контактов MAPI с помощью Aspose.Email для .NET.

### Функция: Создание и заполнение контакта MAPI

#### Обзор

Эта функция демонстрирует, как создать экземпляр `MapiContact` и заполните его основными контактными данными, такими как имя, профессия, адреса, адрес электронной почты, номера телефонов, категории и многое другое.

#### Пошаговая реализация

##### Инициализировать выходной каталог

Сначала определите, где вы будете сохранять файлы:

```csharp
string dataDir = "YOUR_OUTPUT_DIRECTORY";
```

##### Создать новый объект MapiContact

Начните с инициализации нового `MapiContact` объект:

```csharp
MapiContact contact = new MapiContact();
```

##### Установите основную информацию

Заполните основные данные, такие как имя и профессия:

```csharp
contact.NameInfo = new MapiContactNamePropertySet("Bertha", "A.", "Buell");
contact.ProfessionalInfo = new MapiContactProfessionalPropertySet("Awthentikz", "Social work assistant");
```

##### Добавить контактную информацию

Включите дополнительную контактную информацию, такую как физические адреса, адреса электронной почты и номера телефонов:

```csharp
// Физический адрес для работы
contact.PhysicalAddresses.WorkAddress.Address = "Im Astenfeld 59 8580 EDELSCHROTT";

// Электронная почта
contact.ElectronicAddresses.Email1 = new MapiContactElectronicAddress("Experwas", "SMTP", "BerthaABuell@armyspy.com");

// Номер телефона
contact.Telephones = new MapiContactTelephonePropertySet("06605045265");
```

##### Добавить дополнительные детали

Вы также можете добавлять различную информацию и пользовательские поля:

```csharp
// Разная информация
contact.Mileage = "Some test mileage";
contact.Billing = "Test billing information";

// Пользовательские поля
contact.OtherFields.Journal = true;
contact.OtherFields.Private = true;
contact.OtherFields.ReminderTime = new DateTime(2014, 1, 1, 0, 0, 55);
contact.OtherFields.UserField1 = "ContactUserField1";
```

##### Загрузить фото

Загрузите изображение в контакт для поля фото:

```csharp
using (FileStream fs = File.OpenRead("YOUR_DOCUMENT_DIRECTORY/Desert.jpg"))
{
    byte[] buffer = new byte[fs.Length];
    fs.Read(buffer, 0, buffer.Length);
    contact.Photo = new MapiContactPhoto(buffer, MapiContactPhotoImageFormat.Jpeg);
}
```

### Функция: сохранение контакта MAPI в различных форматах

#### Обзор

После того, как вы заполнили свой `MapiContact` объект с нужной информацией, пришло время сохранить его в различных форматах, таких как MSG и VCF.

#### Пошаговая реализация

##### Сохранить в формате MSG

```csharp
contact.Save(dataDir + "/MapiContact_out.msg", ContactSaveFormat.Msg);
```

##### Сохранить в формате VCF

```csharp
contact.Save(dataDir + "/MapiContact_out.vcf", ContactSaveFormat.VCard);
```
