### Чтение задачи MAPI из файла

В этом разделе показано, как прочитать задачу MAPI из файла MSG с помощью библиотеки Aspose.Email. Вот как:

#### Загрузить сообщение MAPI

Сначала укажите каталог, в котором находится ваш файл .msg, и загрузите его в свое приложение.

```csharp
// Определите путь к каталогу документов, содержащему файл .msg.
string dataDir = "/path/to/your/documents";

// Загрузите сообщение MAPI из указанного файла. Замените «MapiTask.msg» на фактическое имя файла.
MapiMessage msg = MapiMessage.FromFile(dataDir + "/MapiTask.msg");
```

**Объяснение:**  
- `dataDir` — это путь к каталогу вашего файла MSG.
- `FromFile()` загружает файл .msg в `MapiMessage` объект, позволяющий проводить дальнейшие манипуляции.

#### Преобразовать в задачу MAPI

Затем преобразуйте это загруженное сообщение в задачу MAPI для доступа к его определенным свойствам.

```csharp
// Преобразуйте загруженное сообщение MAPI в объект MapiTask для взаимодействия с атрибутами, специфичными для задачи, такими как тема и сроки выполнения.
MapiTask task = (MapiTask)msg.ToMapiMessageItem();
```

**Объяснение:**  
- `ToMapiMessageItem()` преобразует ваш `MapiMessage` в соответствующий ему тип элемента, здесь `MapiTask`.
- Это позволяет вам взаимодействовать с атрибутами, характерными для задачи, такими как тема и сроки выполнения.
