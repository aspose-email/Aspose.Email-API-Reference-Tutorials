### Чтение именованных свойств из MapiMessage

#### Обзор
Доступ и управление определенными свойствами в пределах `MapiMessage` объект, необходимый для обработки метаданных электронной почты.

#### Пошаговая реализация
1. **Загрузить сообщение**
   Загрузите файл сообщения с помощью `MapiMessage` сорт:
   ```csharp
   // Определите пути к каталогам для входных и выходных файлов.
   string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

   // Загрузить MAPI-сообщение из файла
   MapiMessage message = MapiMessage.FromFile(dataDir + "\\message.msg");
   ```
2. **Доступ к именованным свойствам**
   Получить коллекцию именованных свойств:
   ```csharp
   MapiPropertyCollection properties = message.NamedProperties;
   ```
3. **Итерация и обработка свойств**
   Пройдитесь по каждому свойству, проверяя наличие определенных идентификаторов для соответствующей обработки:
   ```csharp
   foreach (MapiNamedProperty mapiNamedProp in properties.Values)
   {
       // Проверьте наличие определенного идентификатора именованного свойства и выведите его значение.
       switch (mapiNamedProp.NameId)
       {
           case "TEST":
               Console.WriteLine($"{mapiNamedProp.DisplayName} = {mapiNamedProp.Value}");
               break;
           // При необходимости добавьте больше случаев.
       }
   }
   ```
**Объяснение:**
- `MapiMessage.FromFile`: Загружает сообщение из указанного пути к файлу.
- `message.NamedProperties`: Предоставляет доступ ко всем именованным свойствам в MapiMessage.
- `mapiNamedProp.NameId`: Позволяет идентифицировать определенные свойства по уникальным идентификаторам.

### Советы по устранению неполадок
- **Отсутствующие свойства:** Проверьте, содержит ли ваш файл .msg ожидаемые свойства, используя Outlook или другой почтовый клиент.
- **Неправильные пути:** Еще раз проверьте пути к каталогам и убедитесь, что они правильно отформатированы в вашем коде.
