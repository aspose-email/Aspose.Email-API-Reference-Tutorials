## .NET के लिए Aspose.Email का उपयोग करके किसी ईमेल में एकाधिक अनुलग्नक कैसे जोड़ें

क्या आप अपने एप्लिकेशन के भीतर ईमेल में फ़ाइलें संलग्न करने की प्रक्रिया को स्वचालित करना चाहते हैं? यह मार्गदर्शिका दर्शाती है कि इसका उपयोग कैसे करें **.NET के लिए Aspose.Email** कई अनुलग्नकों को सहजता से जोड़ने के लिए। अपनी शक्तिशाली विशेषताओं के साथ, यह लाइब्रेरी जटिल ईमेल प्रबंधन कार्यों को सरल बनाती है।
इस ट्यूटोरियल में आप सीखेंगे:

### कार्यान्वयन मार्गदर्शिका
#### ईमेल में अनुलग्नक जोड़ना
यह सुविधा आपको एक ही ईमेल संदेश में अनेक फ़ाइलों को अनुलग्नक के रूप में जोड़ने में सक्षम बनाती है, जिससे बल्क ईमेल या दस्तावेज़ भेजते समय आपका कार्यप्रवाह सुव्यवस्थित हो जाता है।

##### चरण 1: निर्देशिकाएँ सेट करें और मेल संदेश बनाएँ
सबसे पहले, अनुलग्नक फ़ाइलों को पढ़ने के लिए निर्देशिकाएँ स्थापित करें और निर्दिष्ट करें कि अंतिम ईमेल फ़ाइल को कहाँ सहेजना है। फिर, आरंभ करें `MailMessage` प्रेषक विवरण सहित वस्तु:
```csharp
string dataDir = System.IO.Path.Combine("YOUR_DOCUMENT_DIRECTORY", "EmailAttachments");
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// MailMessage क्लास का एक उदाहरण बनाएँ
MailMessage message = new MailMessage { From = "sender@sender.com" };
message.To.Add("receiver@gmail.com");
```

##### चरण 2: अनुलग्नक लोड करें और जोड़ें
अपनी निर्दिष्ट निर्देशिका से फ़ाइलें लोड करें और उन्हें ईमेल में अनुलग्नक के रूप में जोड़ें:
```csharp
// ईमेल में अनुलग्नक लोड करें और जोड़ें
Attachment attachment1 = new Attachment(dataDir + "/1.txt");
message.Attachments.Add(attachment1);
message.AddAttachment(new Attachment(dataDir + "/1.jpg"));
message.AddAttachment(new Attachment(dataDir + "/1.doc"));
message.AddAttachment(new Attachment(dataDir + "/1.rar"));
message.AddAttachment(new Attachment(dataDir + "/1.pdf"));
```
यहाँ, `AddAttachment` विधि विभिन्न प्रकार की (पाठ, छवि, दस्तावेज़) एकाधिक फ़ाइलों को कुशलतापूर्वक जोड़ती है।

##### चरण 3: ईमेल सहेजें
अंत में, अपने ईमेल को सभी अनुलग्नकों के साथ डिस्क पर सहेजें:
```csharp
string outputFile = System.IO.Path.Combine(outputDir, "outputAttachments_out.msg");
message.Save(outputFile, SaveOptions.DefaultMsgUnicode);
```
