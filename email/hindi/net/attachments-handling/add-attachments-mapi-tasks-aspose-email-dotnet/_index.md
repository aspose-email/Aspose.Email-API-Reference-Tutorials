## कार्यान्वयन मार्गदर्शिका

### MAPI कार्यों में अनुलग्नक जोड़ने का अवलोकन

यह सुविधा MAPI प्रोटोकॉल का उपयोग करके बनाए गए कार्यों में फ़ाइलों को सीधे संलग्न करने की अनुमति देती है, जो कार्य प्रबंधन प्रणालियों के लिए लाभदायक है, जिन्हें दस्तावेज़ या संबंधित फ़ाइलों को सीधे संलग्न करने की आवश्यकता होती है।

#### चरण 1: अपना कार्य बनाएं और कॉन्फ़िगर करें
इसका एक उदाहरण बनाकर शुरू करें `MapiTask`यह ऑब्जेक्ट आपके ईमेल कार्य का प्रतिनिधित्व करता है.

```csharp
using System;
using Aspose.Email.Mapi;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// निर्दिष्ट विवरण के साथ एक नया MAPI कार्य बनाएँ
MapiTask testTask = new MapiTask("Task with attachment", "This is the description of your task.", DateTime.Now, DateTime.Now.AddDays(1));
```
*नोट: प्रतिस्थापित करें `YOUR_DOCUMENT_DIRECTORY` और `YOUR_OUTPUT_DIRECTORY` आपके सिस्टम पर वास्तविक पथों के साथ.*

#### चरण 2: अपने कार्य में अनुलग्नक जोड़ें
अनुलग्नक जोड़ने के लिए, का उपयोग करें `MapiAttachment` class. अनुलग्नक के लिए फ़ाइल पथ और नाम निर्दिष्ट करें.

```csharp
// MAPI अनुलग्नक बनाएँ
string filePath = System.IO.Path.Combine(dataDir, "sample.pdf");
MapiAttachment attachment = new MapiAttachment("sample.pdf", System.IO.File.ReadAllBytes(filePath));

// अपने कार्य में अनुलग्नक जोड़ें
testTask.Attachments.Add(attachment);
```
*स्पष्टीकरण: हम फ़ाइल बाइट्स को पढ़ते हैं `filePath` और एक नया बनाएँ `MapiAttachment`, जिसे फिर कार्य के अनुलग्नकों में जोड़ दिया जाता है।*

#### चरण 3: अपना कार्य सहेजें
अंत में, अपने MAPI कार्य को अनुलग्नक के साथ आउटपुट निर्देशिका में सहेजें।

```csharp
// बचत के लिए मार्ग निर्धारित करें
string outputPath = System.IO.Path.Combine(outputDir, "TaskWithAttachment.msg");

// कार्य को .msg फ़ाइल के रूप में सहेजें
testTask.Save(outputPath);
```

### समस्या निवारण युक्तियों:
- सुनिश्चित करें कि निर्देशिकाएँ `dataDir` और `outputDir` अपना कोड चलाने से पहले मौजूद रहें.
- फ़ाइल पथ या अनुमतियों से संबंधित अपवादों की जाँच करें.

MAPI कार्यों में अनुलग्नक जोड़ने से कार्यप्रवाह को सुव्यवस्थित किया जा सकता है, जैसे:
1. **परियोजना प्रबंधन:** प्रबंधन उपकरण में परियोजना दस्तावेज़ों को कार्य आइटमों से सीधे संलग्न करें।
2. **ग्राहक सहेयता:** समर्थन कार्यों के साथ टिकट, लॉग या स्क्रीनशॉट शामिल करें.
3. **स्वचालित रिपोर्टिंग:** समीक्षा के लिए तैयार की गई रिपोर्ट को निर्धारित कार्यों में संलग्न करें।

Aspose.Email एकीकरण MAPI कार्यों का समर्थन करने वाले विभिन्न प्लेटफार्मों में विस्तार की अनुमति देता है।
