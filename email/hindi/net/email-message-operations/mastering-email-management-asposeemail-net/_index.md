## कार्यान्वयन मार्गदर्शिका

### ईमेल संदेश बनाएँ और जोड़ें
Aspose.Email के साथ ईमेल संदेश बनाना और उसे अपने IMAP मेलबॉक्स में जोड़ना बहुत आसान है। यह सुविधा आपको ईमेल भेजने या व्यवस्थित करने का काम स्वचालित करने देती है।

#### अवलोकन
इस अनुभाग में, हम बताएंगे कि नया खाता कैसे बनाया जाए `MailMessage` ऑब्जेक्ट को चुनें और उसे IMAP सर्वर के इनबॉक्स फ़ोल्डर में जोड़ें।

#### चरण-दर-चरण कार्यान्वयन
**1. ImapClient सेट अप करें**
अपना कॉन्फ़िगरेशन करके प्रारंभ करें `ImapClient` आवश्यक प्रमाण-पत्र के साथ:
```csharp
ImapClient client = new ImapClient();
client.Host = "imap.gmail.com"; // अपने IMAP होस्ट का उपयोग यहां करें
client.Username = "your.username@gmail.com";
client.Password = "your.password";
client.Port = 993; // जीमेल के लिए SSL पोर्ट
client.SecurityOptions = SecurityOptions.Auto;
```
**2. ईमेल बनाएं और जोड़ें**
एक बनाने के `MailMessage` प्रेषक, प्राप्तकर्ता, विषय और मुख्य भाग के साथ उदाहरण:
```csharp
try
{
    MailMessage message = new MailMessage("user@domain1.com", "user@domain2.com", "subject", "message");
    
    // ईमेल को इनबॉक्स फ़ोल्डर में जोड़ें
    string uid = client.AppendMessage(ImapFolderInfo.InBox, message);
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```
### ईमेल संदेश में कस्टम फ़्लैग जोड़ें
कस्टम फ़्लैग आपके एप्लिकेशन के भीतर विशिष्ट कार्यों के लिए ईमेल को वर्गीकृत या चिह्नित करने में आपकी सहायता कर सकते हैं।

#### अवलोकन
IMAP मेलबॉक्स में UID का उपयोग करके किसी ईमेल संदेश में कस्टम फ़्लैग जोड़ने का तरीका जानें।

#### चरण-दर-चरण कार्यान्वयन
**1. इनबॉक्स फ़ोल्डर चुनें**
सुनिश्चित करें कि फ़ोल्डर फ़्लैग ऑपरेशन के लिए तैयार है:
```csharp
client.SelectFolder(ImapFolderInfo.InBox);
```
**2. यूआईडी द्वारा झंडे जोड़ें**
किसी निर्दिष्ट संदेश में उसके विशिष्ट पहचानकर्ता (UID) द्वारा पहचाने जाने वाले कस्टम फ़्लैग जोड़ें:
```csharp
try
{
    string uid = "some-unique-message-id";  // वास्तविक UID से प्रतिस्थापित करें
    
    client.AddMessageFlags(uid, ImapMessageFlags.Keyword("custom1") | ImapMessageFlags.Keyword("custom1_0"));
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```
### ईमेल संदेशों में कस्टम फ़्लैग प्राप्त करें और जाँचें
कस्टम फ्लैग की जांच के लिए संदेशों को पुनः प्राप्त करना, व्यवस्थित ईमेल वर्कफ़्लो को बनाए रखने के लिए महत्वपूर्ण है।

#### अवलोकन
यह अनुभाग दर्शाता है कि आप किसी फ़ोल्डर में सभी संदेशों को कैसे सूचीबद्ध कर सकते हैं और जाँच सकते हैं कि क्या किसी में विशिष्ट कीवर्ड को फ़्लैग के रूप में सेट किया गया है।

#### चरण-दर-चरण कार्यान्वयन
**1. सभी संदेशों की सूची बनाएं**
इनबॉक्स फ़ोल्डर का चयन करें और संदेश जानकारी प्राप्त करें:
```csharp
client.SelectFolder(ImapFolderInfo.InBox);
ImapMessageInfoCollection messageInfos = client.ListMessages();
```
**2. कीवर्ड की जांच करें**
संदेशों को दोहराकर उन संदेशों को खोजें जिनमें विशिष्ट कीवर्ड ध्वज के रूप में हों:
```csharp
try
{
    foreach (var inf in messageInfos)
    {
        if (inf.ContainsKeyword("custom1"))
        {
            Console.WriteLine("Keyword found");
        }
    }
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```
