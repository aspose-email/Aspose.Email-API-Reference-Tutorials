## कार्यान्वयन मार्गदर्शिका

### PST फ़ाइलों को टुकड़ों में विभाजित करना
यह अनुभाग बताता है कि .NET के लिए Aspose.Email का उपयोग करके एक बड़ी PST फ़ाइल को छोटे टुकड़ों में कैसे विभाजित किया जाए।

#### फ़ीचर का अवलोकन
The `SplitInto` विधि एक एकल PST फ़ाइल को छोटे भागों में विभाजित करती है, जिनमें से प्रत्येक का एक निर्दिष्ट आकार होता है। यह तब उपयोगी होता है जब बड़े आकार की PST फ़ाइलों से निपटना होता है जिन्हें प्रबंधित करना मुश्किल होता है।

#### कार्यान्वयन चरण

##### 1. पथ और निर्देशिकाएँ सेटअप करें
अपनी स्रोत PST फ़ाइल के लिए निर्देशिका और विभाजित खंडों के लिए गंतव्य निर्दिष्ट करें।
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY\Sub.pst";
String dstSplit = dataDir + "Chunks\\";
```

##### 2. गंतव्य फ़ोल्डर में मौजूदा फ़ाइलें साफ़ करें
अपने गंतव्य फ़ोल्डर में मौजूद किसी भी फ़ाइल को हटाकर टकराव से बचें:
```csharp
foreach (string file__1 in Directory.GetFiles(dstSplit))
{
    File.Delete(file__1);
}
```

##### 3. PST फ़ाइल लोड करें और विभाजित करें
अपनी PST फ़ाइल लोड करें और उसे निर्दिष्ट आकार के टुकड़ों में विभाजित करें, जैसे, 5MB.
```csharp
using (PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir))
{
    // वैकल्पिक: प्रगति ट्रैकिंग के लिए ईवेंट की सदस्यता लें
    personalStorage.StorageProcessed += PstSplit_OnStorageProcessed;
    personalStorage.ItemMoved += PstSplit_OnItemMoved;

    // PST फ़ाइल को 5MB के टुकड़ों में विभाजित करें
    personalStorage.SplitInto(5000000, dataDir + "\\Chunks\\");
}
```

##### मापदंडों और विधियों का स्पष्टीकरण
- **`FromFile(dataDir)`**: निर्दिष्ट पथ से PST लोड करता है।
- **`SplitInto(5000000, destinationPath)`**: फ़ाइल को भागों में विभाजित करता है, प्रत्येक का आकार 5MB तक होता है। पहला पैरामीटर बाइट्स में चंक का आकार है।

#### समस्या निवारण युक्तियों
- सुनिश्चित करें कि आपके पास फ़ाइलें पढ़ने और लिखने के लिए पर्याप्त अनुमतियाँ हैं।
- सत्यापित करें कि निर्दिष्ट पथ मौजूद हैं और पहुँच योग्य हैं।
- यदि समस्या उत्पन्न होती है तो अपडेट या संगतता नोट के लिए Aspose के दस्तावेज़ पृष्ठ की जाँच करें `SplitInto` तरीका।
