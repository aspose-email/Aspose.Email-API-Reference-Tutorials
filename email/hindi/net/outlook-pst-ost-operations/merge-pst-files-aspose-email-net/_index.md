## कार्यान्वयन मार्गदर्शिका

### एकाधिक PST फ़ाइलों को एक फ़ाइल में मर्ज करना
यह सुविधा आपको निर्दिष्ट निर्देशिका से एकाधिक PST फ़ाइलों को एक एकल फ़ाइल में एकीकृत करने की अनुमति देती है।

#### अवलोकन
विशिष्ट ईवेंट की सदस्यता लेकर, हम मर्जिंग प्रक्रिया को ट्रैक कर सकते हैं और यहां तक कि प्रति फ़ोल्डर स्थानांतरित किए गए संदेशों की संख्या की निगरानी भी कर सकते हैं। यह ऑपरेशन के दौरान पारदर्शिता और नियंत्रण प्रदान करता है।

#### कार्यान्वयन चरण

##### चरण 1: पथ परिभाषित करें और स्टोरेज आरंभ करें
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // इसे अपने निर्देशिका पथ के साथ अद्यतन करें
string dst = Path.Combine(dataDir, "Sub.pst");
int totalAdded = 0;

try
{
    using (PersonalStorage personalStorage = PersonalStorage.FromFile(dst))
    {
        // प्रक्रिया पर नज़र रखने के लिए ईवेंट की सदस्यता लें
        personalStorage.StorageProcessed += PstMerge_OnStorageProcessed;
        personalStorage.ItemMoved += PstMerge_OnItemMoved;

        // निर्दिष्ट निर्देशिका में स्थित सभी PST फ़ाइलों को मर्ज करें
        personalStorage.MergeWith(Directory.GetFiles(Path.Combine(dataDir, "MergePST")));
    }
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message + "\nThis example will only work if you apply a valid Aspose Email License.");
}
```
- **पैरामीटर्स की व्याख्या:**
  - `dataDir`: वह निर्देशिका जहाँ आपकी PST फ़ाइलें संग्रहीत हैं.
  - `dst`: मर्ज किए गए PST के लिए गंतव्य फ़ाइल पथ.

##### चरण 2: ईवेंट संभालें

**भंडारण प्रसंस्करण के लिए इवेंट हैंडलर:**
यह इवेंट तब लॉग होता है जब प्रत्येक संग्रहण संसाधित किया जा रहा होता है.
```csharp
static void PstMerge_OnStorageProcessed(object sender, StorageProcessedEventArgs e)
{
    Console.WriteLine("*** The storage is merging: {0}", e.FileName);
}
```

**आइटम मूवमेंट के लिए इवेंट हैंडलर:**
प्रति फ़ोल्डर स्थानांतरित किए गए संदेशों की संख्या को ट्रैक करता है और तदनुसार अद्यतन करता है।
```csharp
static void PstMerge_OnItemMoved(object sender, ItemMovedEventArgs e)
{
    static string currentFolder = null;
    static int messageCount = 0;

    if (currentFolder == null)
    {
        currentFolder = e.DestinationFolder.RetrieveFullPath();
    }

    string folderPath = e.DestinationFolder.RetrieveFullPath();

    if (currentFolder != folderPath)
    {
        Console.WriteLine("    Added {0} messages to \"{1}\"", messageCount, currentFolder);
        messageCount = 0;
        currentFolder = folderPath;
    }

    messageCount++;
    totalAdded++;
}
```

1. **बैकअप समेकन:** आसान प्रबंधन के लिए विभिन्न बैकअप सत्रों से एकाधिक PST फ़ाइलों को एक में मर्ज करें।
2. **डेटा माइग्रेशन:** ईमेल डेटा को नए सिस्टम में स्थानांतरित करते समय, प्रक्रिया को सुव्यवस्थित करने के लिए PST को समेकित करें।
3. **ईमेल संग्रहण:** विभिन्न उपयोगकर्ताओं या विभागों से प्राप्त संग्रहीत ईमेल को एकल संग्रह फ़ाइल में केंद्रीकृत करें।
