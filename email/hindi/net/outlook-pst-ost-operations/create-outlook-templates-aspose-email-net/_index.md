## कार्यान्वयन मार्गदर्शिका

### Outlook MSG फ़ाइल को टेम्पलेट के रूप में सहेजना (OFT प्रारूप)

**अवलोकन:**
यह सुविधा आपको Outlook MSG फ़ाइल को सीधे टेम्पलेट के रूप में सहेजने की अनुमति देती है, जिससे दोहराए जाने वाले ईमेल निर्माण कार्य सरल हो जाते हैं।

#### चरण-दर-चरण कार्यान्वयन:
1. **MapiMessage ऑब्जेक्ट बनाएँ**
   
   एक नया बनाएँ `MapiMessage` अपने इच्छित प्रेषक, प्राप्तकर्ता, विषय और मुख्य भाग के साथ एक उदाहरण दर्ज करें।
   
   ```csharp
   using (MapiMessage mapi = new MapiMessage("test@from.to", "test@to.to", "template subject", "Template body"))
   {
       string oftMapiFileName = System.IO.Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "mapiToOft.msg");
       
       mapi.SaveAsTemplate(oftMapiFileName);
   }
   ```

2. **पैरामीटर और कॉन्फ़िगरेशन:**
   - `SaveAsTemplate` संदेश को OFT प्रारूप में सहेजने के लिए उपयोग किया जाता है, जो टेम्पलेट निर्माण के लिए आवश्यक है।
   - सुनिश्चित करें कि आप एक वैध निर्देशिका पथ निर्दिष्ट करें जहां फ़ाइल सहेजी जाएगी।

### MapiMessage में MSG फ़ाइल लोड हो रही है

**अवलोकन:**
अपने अनुप्रयोग में मौजूदा MSG फ़ाइलों को लोड करने से प्रोग्रामेटिक हेरफेर या ईमेल डेटा को पढ़ने में सुविधा होती है।

#### कार्यान्वयन चरण:
1. **MSG फ़ाइल लोड करें**
   
   उपयोग `MapiMessage.FromFile` MSG फ़ाइल को लोड करने के लिए `MapiMessage` वस्तु।
   
   ```csharp
   string msgFilePath = System.IO.Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.msg");
   MapiMessage loadedMsg = MapiMessage.FromFile(msgFilePath);
   ```

2. **संदेश गुण तक पहुँचना**
   
   एक बार लोड हो जाने पर, विषय और मुख्य भाग जैसे विभिन्न गुणों तक पहुंचें।
   
   ```csharp
   Console.WriteLine("Subject: " + loadedMsg.Subject);
   Console.WriteLine("Body: " + loadedMsg.Body);
   ```

यहां कुछ वास्तविक दुनिया परिदृश्य दिए गए हैं जहां ये विशेषताएं चमकती हैं:
1. **स्वचालित ईमेल अभियान**: विपणन अभियानों के लिए ईमेल टेम्पलेट्स को शीघ्रता से तैयार और अनुकूलित करें।
2. **ग्राहक सेवा स्वचालन**ग्राहक संपर्क बढ़ाने के लिए मानकीकृत प्रतिक्रियाएँ या अनुरोध बनाएँ।
3. **आंतरिक संचार टेम्पलेट्स**पूर्वनिर्धारित टेम्पलेट्स का उपयोग करके आंतरिक सूचनाओं को सुव्यवस्थित करें।
