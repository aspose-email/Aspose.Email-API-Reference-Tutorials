## कार्यान्वयन मार्गदर्शिका

### फ़ीचर 1: OST फ़ाइल पार्स करें और फ़ोल्डर नाम पुनर्प्राप्त करें

यह सुविधा दर्शाती है कि .NET के लिए Aspose.Email का उपयोग करके OST फ़ाइल को कैसे खोला जाए और सभी फ़ोल्डर नामों को उनके मूल नामों के साथ कैसे पुनर्प्राप्त किया जाए।

#### अवलोकन
OST फ़ाइल को पार्स करने से आप इसकी संरचना के माध्यम से नेविगेट कर सकते हैं, प्रत्येक फ़ोल्डर के नाम और पदानुक्रम की पहचान कर सकते हैं। ईमेल डेटा को प्रभावी ढंग से व्यवस्थित करने और उस तक पहुँचने के लिए यह महत्वपूर्ण है।

##### चरण 1: निर्देशिका पथ परिभाषित करें
सबसे पहले उस निर्देशिका को निर्दिष्ट करें जहां आपकी OST फ़ाइलें संग्रहीत हैं:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string path = Path.Combine(dataDir, "PersonalStorage.pst");
```

##### चरण 2: OST फ़ाइल पढ़ें और खोलें
OST फ़ाइल को पढ़ने और उसे स्ट्रीम के रूप में खोलने के लिए बाइट ऐरे का उपयोग करें:
```csharp
byte[] buffer = File.ReadAllBytes(path);
using (Stream s = File.OpenRead(path))
{
    PersonalStorage pst = PersonalStorage.FromStream(s);
    
    // यदि आवश्यक हो तो किसी विशिष्ट फ़ोल्डर को उसकी प्रविष्टि आईडी द्वारा पुनर्प्राप्त करें
    FolderInfo target = pst.GetFolderById("AAAAAB9of1CGOidPhTb686WQY68igAAA");
    IList<string> folderData = new List<string>();
    
    // सभी फ़ोल्डरों में जाकर उनके प्रदर्शन नाम और मूल नाम एकत्रित करें
    WalkFolders(pst.RootFolder, "N/A", folderData);
}
```

##### चरण 3: फ़ोल्डरों में पुनरावर्ती रूप से जाएँ
फ़ोल्डर संरचना को पुनरावर्ती रूप से नेविगेट करने के लिए एक विधि परिभाषित करें:
```csharp
private static void WalkFolders(FolderInfo folder, string parentFolderName, IList<string> folderData)
{
    // प्रदर्शन नाम निर्धारित करें या यदि यह शून्य या रिक्त है तो 'रूट' का उपयोग करें
    string displayName = (string.IsNullOrEmpty(folder.DisplayName)) ? "ROOT" : folder.DisplayName;
    
    // फ़ोल्डर जानकारी को स्ट्रिंग के रूप में फ़ॉर्मेट और संग्रहीत करें
    string folderNames = string.Format("DisplayName = {0}; Parent.DisplayName = {1}", displayName, parentFolderName);
    folderData.Add(folderNames);
    
    // यदि सबफ़ोल्डर मौजूद हों तो उन्हें प्रोसेस करें
    if (!folder.HasSubFolders) return;
    
    FolderInfoCollection coll = folder.GetSubFolders(FolderKind.Search | FolderKind.Normal);
    foreach (FolderInfo subfolder in coll)
    {
        WalkFolders(subfolder, displayName, folderData);
    }
}
```

### फ़ीचर 2: OST और प्रक्रिया विशिष्ट फ़ोल्डर खोलें

यह सुविधा OST फ़ाइल को खोलने और उनके प्रदर्शन नामों के आधार पर विशिष्ट फ़ोल्डरों को संसाधित करने पर केंद्रित है।

#### अवलोकन
किसी OST फ़ाइल के भीतर विशिष्ट फ़ोल्डरों को फ़िल्टर करने और संसाधित करने से डेटा प्रबंधन कार्य सरल हो सकता है, जिससे आप प्रासंगिक ईमेल डेटा पर ध्यान केंद्रित कर सकते हैं।

##### चरण 1: निर्देशिका पथ परिभाषित करें
पिछली सुविधा के समान, अपने निर्देशिका पथ परिभाषित करें:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string path = Path.Combine(dataDir, "PersonalStorage.pst");
```

##### चरण 2: विशिष्ट फ़ोल्डर खोलें और प्रक्रिया करें
OST फ़ाइल को स्ट्रीम के रूप में खोलें और विशिष्ट मानदंडों के साथ फ़ोल्डर्स को प्रोसेस करें:
```csharp

{
    PersonalStorage pst = PersonalStorage.FromStream(s);

    foreach (FolderInfo folder in folders)
    {
        // उदाहरण: 'फाइंडर' नामक प्रक्रिया फ़ोल्डर
        if (folder.DisplayName == "Finder")
        {
            // फाइंडर फ़ोल्डर को संभालने के लिए तर्क जोड़ें
        }
    }
}
```

4. **बैकअप समाधान**: आपदा पुनर्प्राप्ति के लिए OST फ़ाइल के भीतर महत्वपूर्ण फ़ोल्डरों का बैकअप बनाएँ।
