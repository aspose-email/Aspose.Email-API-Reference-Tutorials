## कार्यान्वयन मार्गदर्शिका

### PST फ़ाइल जानकारी लोड और प्रदर्शित करें

यह सुविधा दर्शाती है कि आप .NET के लिए Aspose.Email का उपयोग करके PST फ़ाइल को कैसे लोड कर सकते हैं और उसका प्रारूप कैसे प्रदर्शित कर सकते हैं।

#### अवलोकन
PST फ़ाइल लोड करने से आप फ़ोल्डर और संदेशों सहित इसकी संरचना तक पहुँच सकते हैं। यह किसी भी डेटा निष्कर्षण प्रक्रिया में पहला कदम है। इस अनुभाग में, हम Outlook PST फ़ाइल लोड करने और उसके प्रारूप को प्रिंट करने के लिए कोड लिखेंगे।

#### कार्यान्वयन चरण

##### चरण 1: अपनी PST फ़ाइल का पथ निर्धारित करें

सबसे पहले, वह निर्देशिका निर्दिष्ट करें जहां आपकी PST फ़ाइल स्थित है:

```csharp
string path = @"YOUR_DOCUMENT_DIRECTORY\PersonalStorage.pst";
```

##### चरण 2: PST फ़ाइल लोड करें

Aspose.Email का उपयोग करें `PersonalStorage` क्लास का प्रयोग करें और अपनी PST फ़ाइल लोड करें।

```csharp
try
{
    PersonalStorage pst = PersonalStorage.FromFile(path);
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message);
}
```

##### चरण 3: PST प्रारूप प्रदर्शित करें

एक बार लोड होने के बाद, आप PST फ़ाइल के विभिन्न गुणों तक पहुँच सकते हैं। यहाँ, हम इसका प्रारूप प्रदर्शित करेंगे:

```csharp
Console.WriteLine("Display Format: " + pst.Format);
```

### PST फ़ाइल से संदेश निकालें

यह सुविधा PST फ़ाइल के भीतर प्रत्येक फ़ोल्डर से संदेशों के पुनरावर्ती निष्कर्षण की अनुमति देती है।

#### अवलोकन
डेटा विश्लेषण और माइग्रेशन कार्यों के लिए ईमेल संदेशों को निकालना महत्वपूर्ण है। यह अनुभाग आपको आपकी PST फ़ाइल के प्रत्येक फ़ोल्डर से सभी संदेशों को निकालने की प्रक्रिया के माध्यम से मार्गदर्शन करता है।

#### कार्यान्वयन चरण

##### चरण 1: PST फ़ाइल लोड करें

प्रदर्शन के लिए लोड करने के समान, रूट फ़ोल्डर तक पहुंच कर प्रारंभ करें:

```csharp
string path = dataDir + @"\PersonalStorage.pst";
try
{
    PersonalStorage pst = PersonalStorage.FromFile(path);
    FolderInfo folderInfo = pst.RootFolder;
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message);
}
```

##### चरण 2: संदेशों को पुनरावर्ती रूप से निकालें

वर्तमान फ़ोल्डर से संदेशों को निकालने और उपफ़ोल्डरों के लिए इसे पुनरावर्ती रूप से कॉल करने के लिए एक विधि परिभाषित करें:

```csharp
private static void ExtractMsgFiles(FolderInfo folderInfo, PersonalStorage pst)
{
    Console.WriteLine("Folder: " + folderInfo.DisplayName);

    MessageInfoCollection messageInfoCollection = folderInfo.GetContents();
    
    foreach (MessageInfo messageInfo in messageInfoCollection)
    {
        MapiMessage message = pst.ExtractMessage(messageInfo);
        string subject = message.Subject.Replace(":", " ");
        message.Save(subject + @"_" + folderInfo.DisplayName + ".msg");
    }

    if (folderInfo.HasSubFolders)
    {
        foreach (FolderInfo subfolderInfo in folderInfo.GetSubFolders())
        {
            ExtractMsgFiles(subfolderInfo, pst);
        }
    }
}
```
