## कार्यान्वयन मार्गदर्शिका
### मौजूदा संपर्कों से वितरण सूचियाँ बनाना
मौजूदा संपर्कों का उपयोग करके PST फ़ाइल के भीतर वितरण सूचियों के निर्माण को स्वचालित करें। यहाँ बताया गया है कि कैसे:

#### संपर्कों के लिए प्रदर्शन नाम और ईमेल परिभाषित करें
प्रदर्शन नाम और ईमेल पते निर्दिष्ट करके अपना संपर्क विवरण सेट करें:
```csharp
cstring displayName1 = "Sebastian Wright";
cstring email1 = "SebastianWright@dayrep.com";
cstring displayName2 = "Wichert Kroos";
cstring email2 = "WichertKroos@teleworm.us";
```

#### संपर्कों के लिए प्रविष्टि आईडी प्राप्त करें
आपको अपने संपर्कों के लिए प्रविष्टि आईडी प्राप्त करनी होगी:
```csharp
cstring strEntryId1;
cstring strEntryId2;
```

#### PST फ़ाइल के लिए आउटपुट पथ परिभाषित करें
निर्दिष्ट करें कि PST फ़ाइल कहाँ सहेजी जाएगी और जाँचें कि क्या यह पहले से मौजूद है। यदि ऐसा है, तो टकराव से बचने के लिए इसे हटा दें:
```csharp
cstring path = dataDir + "CreateDistributionListInPST_out.pst";

if (File.Exists(path))
{
    File.Delete(path);
}
```

#### PST बनाएं और संपर्क जोड़ें
यहां बताया गया है कि आप नई PST फ़ाइल कैसे बना सकते हैं और संपर्क कैसे जोड़ सकते हैं:
```csharp
using (PersonalStorage pst = PersonalStorage.Create(path, FileFormatVersion.Unicode))
{
    FolderInfo contactFolder = pst.CreatePredefinedFolder("Contacts", StandardIpmFolder.Contacts);
    
    // यहां फ़ोल्डर में संपर्क जोड़ें...
}
```
