## कार्यान्वयन मार्गदर्शिका

### PST फ़ाइल से पासवर्ड प्रॉपर्टी हटाना

यह सुविधा आपको पासवर्ड सुरक्षा हटाने की अनुमति देती है, जिससे PST को प्रमाणीकरण के बिना भी एक्सेस किया जा सकता है।

#### चरण 1: PST फ़ाइल लोड करें
Aspose.Email का उपयोग करके अपनी PST फ़ाइल लोड करें `PersonalStorage` कक्षा।

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/PersonalStorage1.pst";
PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
```

#### चरण 2: पासवर्ड जांचें और हटाएं
जाँच करें कि क्या PST फ़ाइल में पासवर्ड गुण सेट है, फिर खाली पासवर्ड सेट करके उसे हटा दें।

```csharp
if (personalStorage.Store.Properties.ContainsKey(MapiPropertyTag.PR_PST_PASSWORD))
{
    MapiProperty property = new MapiProperty(MapiPropertyTag.PR_PST_PASSWORD, BitConverter.GetBytes((long)0));
    personalStorage.Store.SetProperty(property);
}
```

*स्पष्टीकरण*: द `MapiPropertyTag.PR_PST_PASSWORD` पासवर्ड की जाँच करता है। यदि मौजूद है, तो पासवर्ड को प्रभावी ढंग से हटाने के लिए इसे शून्य बाइट्स से बदल दिया जाता है।

#### समस्या निवारण सुझाव
- सुनिश्चित करें कि आपके पास PST फ़ाइल वाली निर्देशिका में लिखने की अनुमति है।
  
### PST फ़ाइल गुण पढ़ना

अपनी PST फ़ाइल के आवश्यक गुणों तक पहुँचें और उन्हें प्रदर्शित करें।

#### चरण 1: PST फ़ाइल लोड करें
पासवर्ड हटाने के समान, PST फ़ाइल लोड करके प्रारंभ करें।

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/PersonalStorage1.pst";
PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
```

#### चरण 2: गुणों तक पहुंचें और प्रदर्शित करें
प्रदर्शन नाम, आइटम संख्या और आकार जैसे गुणों तक पहुंचें, फिर उन्हें प्रिंट करें।

```csharp
Console.WriteLine("Display Name: " + personalStorage.DisplayName);
Console.WriteLine("Total Number of Items: " + personalStorage.RootFolder.ContentCount);
Console.WriteLine("Total Size in Bytes: " + personalStorage.RootFolder.SizeInBytes);
```

*स्पष्टीकरण*: `DisplayName` एक मानव-पठनीय नाम प्रदान करता है, जबकि `ContentCount` और `SizeInBytes` फ़ाइल की सामग्री के बारे में जानकारी दें.
