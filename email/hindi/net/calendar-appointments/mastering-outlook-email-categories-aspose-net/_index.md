**आप क्या सीखेंगे:**
- .NET के लिए Aspose.Email का उपयोग करके Outlook में ईमेल श्रेणियाँ कैसे सेट करें

## कार्यान्वयन मार्गदर्शिका

### ईमेल श्रेणियों के प्रबंधन का अवलोकन

इस अनुभाग में, हम Aspose.Email का उपयोग करके ईमेल श्रेणियों को प्रभावी ढंग से प्रबंधित करने का तरीका जानेंगे। हम Outlook संदेशों से श्रेणियाँ जोड़ना, पुनर्प्राप्त करना और निकालना कवर करेंगे।

#### ईमेल में श्रेणियाँ जोड़ना

Aspose.Email का उपयोग करके Outlook में किसी ईमेल संदेश के लिए रंग श्रेणियाँ निर्धारित करने के लिए:

**चरण 1: संदेश लोड करें**

सबसे पहले, अपनी आउटलुक संदेश फ़ाइल को एक में लोड करें `MapiMessage` वस्तु।

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // अपने निर्देशिका पथ से प्रतिस्थापित करें
MapiMessage msg = MapiMessage.FromFile(dataDir + "/message1.msg");
```

**चरण 2: श्रेणियाँ जोड़ें**

उपयोग `FollowUpManager.AddCategory()` श्रेणियाँ निर्दिष्ट करने की विधि। यहाँ बताया गया है कि आप बैंगनी और लाल श्रेणियाँ कैसे जोड़ते हैं:

```csharp
// बैंगनी और लाल श्रेणियाँ जोड़ना
FollowUpManager.AddCategory(msg, "Purple Category");
FollowUpManager.AddCategory(msg, "Red Category");
```

#### निर्दिष्ट श्रेणियाँ पुनः प्राप्त करना

यह देखने के लिए कि आपके संदेश को कौन सी श्रेणियाँ सौंपी गई हैं, उन्हें निम्नलिखित विधि का उपयोग करके प्राप्त करें:

```csharp
IList<string> categories = FollowUpManager.GetCategories(msg);
// श्रेणियों की सूची आउटपुट करें
foreach (var category in categories)
{
    Console.WriteLine(category);
}
```

#### विशिष्ट और सभी श्रेणियों को हटाना

किसी विशिष्ट श्रेणी को हटाना या सभी श्रेणियों को साफ़ करना सरल है:

**श्रेणी हटाएँ:**

```csharp
FollowUpManager.RemoveCategory(msg, "Red Category");
```

**सभी श्रेणियाँ साफ़ करें:**

```csharp
FollowUpManager.ClearCategories(msg);
```
