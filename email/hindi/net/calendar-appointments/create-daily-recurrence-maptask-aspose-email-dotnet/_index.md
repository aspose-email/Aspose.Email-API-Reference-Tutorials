**आप क्या सीखेंगे:**
- .NET के लिए Aspose.Email के साथ अपना परिवेश सेट अप करना
- बनाना और कॉन्फ़िगर करना `MapiTask` नाम, मुख्य भाग, आरंभ तिथि, नियत तिथि और स्थिति जैसी विशेषताओं के साथ

## कार्यान्वयन मार्गदर्शिका

### MapiTask बनाएं और कॉन्फ़िगर करें
बनाना एक `MapiTask` इसमें नाम, मुख्य भाग, आरंभ तिथि, नियत तिथि और स्थिति जैसी आवश्यक विशेषताएं निर्धारित करना शामिल है।

#### कार्य बनाना
```csharp
using Aspose.Email.Mapi;

DateTime StartDate = new DateTime(2015, 7, 16);
DateTime DueDate = new DateTime(2015, 7, 16);

// नया MapiTask इंस्टेंस बनाएं
task = new MapiTask("This is test task", "Sample Body", StartDate, DueDate);

// कार्य की स्थिति को NotAssigned पर सेट करें
task.State = MapiTaskState.NotAssigned;
```
**स्पष्टीकरण**: यहाँ, हम एक उदाहरण देते हैं `MapiTask` नाम, बॉडी, आरंभ तिथि और नियत तिथि के साथ। हमने इसकी आरंभिक स्थिति भी निर्धारित की है।

### MapiTask के लिए दैनिक पुनरावृत्ति पैटर्न सेट करें
यह सुनिश्चित करने के लिए कि कार्य अनिश्चित काल तक दोहराया जाता रहे, एक दैनिक पुनरावृत्ति पैटर्न जोड़ें।

#### पुनरावृत्ति पैटर्न निर्धारित करना
```csharp
var record = new MapiCalendarDailyRecurrencePattern
{
    PatternType = MapiCalendarRecurrencePatternType.Day,
    Period = 1, // कार्य प्रतिदिन दोहराया जाता है
    EndType = MapiCalendarRecurrenceEndType.NeverEnd, // पुनरावृत्ति कभी ख़त्म नहीं होती
};

// कार्य को पुनरावृत्ति पैटर्न निर्दिष्ट करें
task.Recurrence = record;
```
**स्पष्टीकरण**: यह स्निपेट एक दैनिक पुनरावृत्ति पैटर्न को परिभाषित करता है जो समाप्त नहीं होगा। `PatternType` इसके लिए सेट है `Day`, और `Period` घटनाओं के बीच दिनों के अंतराल को निर्दिष्ट करता है।

### MapiTask को फ़ाइल में सहेजें
अंत में, अपने कॉन्फ़िगर किए गए कार्य को Outlook संदेश फ़ाइल के रूप में सहेजें।

#### कार्य को सहेजना
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // अपने दस्तावेज़ निर्देशिका पथ से बदलें

// MapiTask को .msg फ़ाइल में सहेजें
task.Save(dataDir + "/SetDailyNeverEndRecurrence_out.msg", TaskSaveFormat.Msg);
```
**स्पष्टीकरण**यह कोड आपके कार्य को सहेजता है `.msg` फ़ाइल, जिसे आउटलुक में खोला जा सकता है।
