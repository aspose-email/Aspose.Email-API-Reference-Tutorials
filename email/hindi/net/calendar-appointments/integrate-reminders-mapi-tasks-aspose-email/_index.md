### अनुस्मारक के साथ MAPI कार्य बनाना
यह सुविधा आपको अपने कार्यों में सीधे अनुस्मारक सूचनाएँ जोड़ने की अनुमति देती है। यहाँ बताया गया है कि आप इसे कैसे प्राप्त कर सकते हैं:

#### चरण 1: डेटा निर्देशिका निर्धारित करें
अपने दस्तावेज़ों को संग्रहीत करने के लिए निर्देशिका पथ सेट करके आरंभ करें:
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY"; // अपने वास्तविक दस्तावेज़ निर्देशिका पथ से बदलें
```

#### चरण 2: MAPI कार्य बनाएँ और कॉन्फ़िगर करें
इसका एक नया उदाहरण बनाएं `MapiTask` और अनुस्मारक सहित इसके गुण सेट करें:
```csharp
// नया MAPI कार्य आरंभ करें
MapiTask testTask = new MapiTask("Task with Reminder", "This is a sample task.", DateTime.Now, DateTime.Now.AddDays(7));

// अनुस्मारक विकल्प कॉन्फ़िगर करें
testTask.ReminderSet = true;
testTask.ReminderTime = DateTime.Now.AddMinutes(30); // अनुस्मारक समय सेट करें
```

#### स्पष्टीकरण
- `MapiTask`: एक MAPI कार्य ऑब्जेक्ट का प्रतिनिधित्व करता है.
- `ReminderSet`: एक बूलियन जो यह इंगित करता है कि अनुस्मारक सक्षम है या नहीं।
- `ReminderTime`: निर्दिष्ट करता है कि अनुस्मारक कब सक्रिय होना चाहिए.
