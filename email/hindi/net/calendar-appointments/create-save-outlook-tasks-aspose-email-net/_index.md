## कार्यान्वयन मार्गदर्शिका
आइए, Outlook कार्य को चरण-दर-चरण बनाने और सहेजने का तरीका जानें।

### MapiTask ऑब्जेक्ट बनाना
ए `MapiTask` ऑब्जेक्ट एक Outlook कार्य को दर्शाता है। इसे आवश्यक गुणों के साथ आरंभ करके शुरू करें:

#### चरण 1: कार्य को परिभाषित करें
```csharp
MapiTask task = new MapiTask(
    "करने के लिए", 
    "Just click and type to add new task", 
    DateTime.Now, 
    DateTime.Now.AddDays(3)
);
```
- **"To Do"** विषय है.
- विवरण अतिरिक्त जानकारी प्रदान करता है.
- आरंभ तिथि और नियत तिथि आज की तिथि तथा अब से तीन दिन बाद निर्धारित की गई है।

#### चरण 2: प्रगति और प्रयास निर्धारित करें
```csharp
task.PercentComplete = 20;
task.EstimatedEffort = 2000; // कुछ ही मिनटों में
```
- कार्य पूर्णता का प्रतिशत निर्धारित करें।
- खर्च किए गए समय को ट्रैक करने के लिए अनुमानित प्रयास को मिनटों में सेट करें।

#### चरण 3: कार्य इतिहास और अपडेट
```csharp
task.History = MapiTaskHistory.Assigned;
task.LastUpdate = DateTime.Now;
```
- बेहतर ट्रैकिंग के लिए रिकॉर्ड करें कि कार्य पिछली बार कब सौंपा गया था या कब अपडेट किया गया था।

### स्वामित्व और कंपनियों का विन्यास
स्वामित्व विवरण और संबद्ध कंपनियां निर्दिष्ट करें:

```csharp
task.Users.Owner = "Darius";
task.Users.LastAssigner = "Harkness";
task.Users.LastDelegate = "Harkness";
task.Users.Ownership = MapiTaskOwnership.AssignersCopy;

task.Companies = new string[] { "company1", "company2", "company3" };
```

### मेटाडेटा को वर्गीकृत करना और जोड़ना
संगठन के लिए श्रेणियों का उपयोग करें:

```csharp
task.Categories = new string[] { "category1", "category2", "category3" };
task.Mileage = "Some test mileage";
task.Billing = "Test billing information";
task.Users.Delegator = "Test Delegator";
```

### कार्य गुणधर्मों को अंतिम रूप देना
संवेदनशीलता और स्थिति सेट करें:

```csharp
task.Sensitivity = MapiSensitivity.Personal;
task.Status = MapiTaskStatus.Complete;

// यदि आवश्यक हो तो अनुमानित प्रयास समायोजित करें
task.EstimatedEffort = 5; // कुछ ही मिनटों में
```

### कार्य को MSG फ़ाइल के रूप में सहेजना
अंत में, अपना कार्य सहेजें:

```csharp
string outputPath = "@YOUR_OUTPUT_DIRECTORY/MapiTask.msg";
task.Save(outputPath, TaskSaveFormat.Msg);
```

प्रतिस्थापन सुनिश्चित करें `@YOUR_OUTPUT_DIRECTORY` उस वास्तविक निर्देशिका पथ के साथ जहाँ आप फ़ाइल को सहेजना चाहते हैं.
