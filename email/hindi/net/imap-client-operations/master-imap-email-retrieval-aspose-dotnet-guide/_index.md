**आप क्या सीखेंगे:**

## कार्यान्वयन मार्गदर्शिका

### ImapClient आरंभ करें
का आरंभीकरण `ImapClient` आपके IMAP सर्वर से सुरक्षित कनेक्शन स्थापित करने के लिए यह आवश्यक है। आप इसे इस प्रकार कॉन्फ़िगर कर सकते हैं:

#### होस्ट और पोर्ट सेट करना
IMAP सर्वर होस्ट और पोर्ट संख्या निर्दिष्ट करें:
- **मेज़बान**: अपने ईमेल प्रदाता के डोमेन नाम या आईपी पते का उपयोग करें।
- **पत्तन**सामान्यतः 993 का प्रयोग SSL कनेक्शन के लिए किया जाता है।
```csharp
imapClient.Host = "<HOST>";
imapClient.Port = 993;
```

#### प्रमाणीकरण विवरण
प्रमाणीकरण के लिए अपना उपयोगकर्ता नाम और पासवर्ड प्रदान करें। इससे आपके ईमेल खाते तक पहुँच मिलती है:
```csharp
imapClient.Username = "<USERNAME>";
imapClient.Password = "<PASSWORD>";
```

#### एन्क्रिप्शन प्रोटोकॉल
समर्थित एन्क्रिप्शन प्रोटोकॉल सेट करके सुरक्षित संचार सुनिश्चित करें:
```csharp
imapClient.SupportedEncryption = EncryptionProtocols.Tls;
```

### IMAP सर्वर से संदेशों की सूची बनाएं
एक बार कनेक्ट होने के बाद, आप अपने इनबॉक्स में उपलब्ध सभी संदेशों को सूचीबद्ध कर सकते हैं:

#### संदेश संग्रह पुनः प्राप्त करें
उपयोग `ListMessages` संदेश जानकारी का संग्रह प्राप्त करने के लिए:
```csharp
ImapMessageInfoCollection messageInfoCol = imapClient.ListMessages();
int[] sequenceNumberAr = messageInfoCol.Select(mi => mi.SequenceNumber).ToArray();
string[] uniqueIdAr = messageInfoCol.Select(mi => mi.UniqueId).ToArray();
```

### अनुक्रम संख्या द्वारा संदेश प्राप्त करें
विशिष्ट ईमेल प्राप्त करने के लिए, आप उनकी अनुक्रम संख्या का उपयोग कर सकते हैं:

#### अनुक्रम संख्या का उपयोग करके प्राप्त करें
वांछित अनुक्रम संख्याएँ पास करें `FetchMessages`:
```csharp
IList<MailMessage> fetchedMessages = imapClient.FetchMessages(sequenceNumbers);
```

### विशिष्ट आईडी द्वारा संदेश प्राप्त करें
वैकल्पिक रूप से, विशिष्ट आईडी का उपयोग करके संदेश प्राप्त करें:

#### विशिष्ट आईडी द्वारा ईमेल प्राप्त करें
ईमेल प्राप्त करने के लिए पहले प्राप्त किए गए विशिष्ट पहचानकर्ताओं का उपयोग करें:
```csharp
code
IList<MailMessage> fetchedMessages = imapClient.FetchMessages(uniqueIds);
```
