## कार्यान्वयन मार्गदर्शिका

### IMAP सर्वर से कनेक्ट करना और लॉग इन करना

Aspose.Email के साथ IMAP सर्वर से कनेक्शन स्थापित करना सरल है:

**अवलोकन:**
यह सुविधा आपके ईमेल प्रदाता के IMAP सर्वर से सुरक्षित कनेक्शन सक्षम करती है, जिससे आप अपने क्रेडेंशियल्स का उपयोग करके प्रमाणीकरण कर सकते हैं।

**कार्यान्वयन चरण:**

#### 1. कनेक्शन विवरण सेट करें

अपना होस्ट, पोर्ट, उपयोगकर्ता नाम और पासवर्ड निम्नानुसार कॉन्फ़िगर करें:

```csharp
const string host = "your-host.com"; // वास्तविक होस्ट से प्रतिस्थापित करें
const int port = 993; // सुरक्षित IMAP पोर्ट (IMAPS)
const string username = "user@host.com"; // आपका ईमेल पता
const string password = "password"; // आपका ईमेल पासवर्ड
```

#### 2. ImapClient का एक इंस्टेंस बनाएं

```csharp
ImapClient client = new ImapClient(host, port, username, password);
```
**स्पष्टीकरण:**
The `ImapClient` सर्वर के साथ संचार को सुविधाजनक बनाने के लिए कनेक्शन विवरण के साथ इसे इंस्टैन्शिएट किया जाता है।

#### 3. IMAP सर्वर से कनेक्ट करें

त्रुटि प्रबंधन के लिए try-catch ब्लॉक का उपयोग करें:

```csharp
try
{
    client.Connect(true);
}
catch (Exception ex)
{
    throw new Exception("Failed to connect and log into IMAP server: " + ex.Message);
}
```
**यह दृष्टिकोण क्यों?**
ट्राई-कैच ब्लॉक कनेक्शन त्रुटियों का सुचारू संचालन सुनिश्चित करता है, तथा गलत क्रेडेंशियल या नेटवर्क समस्याओं जैसे मुद्दों को डीबग करने में सहायता करता है।

### AND शर्तों के साथ जटिल क्वेरीज़ का निर्माण

क्वेरी बनाने से परिष्कृत ईमेल खोज की अनुमति मिलती है। आइए तार्किक AND शर्त का उपयोग करके एक क्वेरी बनाएँ:

#### अवलोकन

यह सुविधा कई शर्तों को संयोजित करके खोज परिणामों को सीमित करने में मदद करती है, जिनका पूरा होना आवश्यक है।

**कार्यान्वयन चरण:**

#### 1. MailQueryBuilder को प्रारंभ करें

```csharp
MailQueryBuilder builder = new MailQueryBuilder();
```

#### 2. क्वेरी शर्तें परिभाषित करें

अधिक विशिष्ट खोजों के लिए मानदंड संयोजित करें:

```csharp
builder.From.Contains("SpecificHost.com");
builder.InternalDate.Before(DateTime.Now);
builder.InternalDate.Since(DateTime.Now.AddDays(-7));
```
**स्पष्टीकरण:**
यह क्वेरी पिछले सप्ताह में निर्दिष्ट डोमेन से प्राप्त ईमेल को फ़िल्टर करती है।

#### 3. अंतिम क्वेरी ऑब्जेक्ट पुनः प्राप्त करें

```csharp
MailQuery query = builder.GetQuery();
```

### OR शर्तों के साथ प्रश्नों का संयोजन

व्यापक खोजों के लिए तार्किक OR का उपयोग करके खोज शर्तों को संयोजित करें:

**अवलोकन:**
यह सुविधा किसी भी निर्दिष्ट मानदंड से मेल खाने वाले ईमेल को पुनः प्राप्त करने में लचीलापन प्रदान करती है।

#### कार्यान्वयन चरण:

#### 1. MailQueryBuilder को पुनः प्रारंभ करें

```csharp
builder = new MailQueryBuilder(); // बिल्डर रीसेट करें
```

#### 2. OR शर्तें परिभाषित करें

```csharp
builder.Or(
    builder.Subject.Contains("test"),
    builder.From.Contains("noreply@host.com")
);
```
**स्पष्टीकरण:**
यह क्वेरी या तो विषय में "test" वाले ईमेल या किसी विशिष्ट प्रेषक से ईमेल प्राप्त करती है।

#### 3. अंतिम क्वेरी ऑब्जेक्ट पुनः प्राप्त करें

```csharp
query = builder.GetQuery();
```
