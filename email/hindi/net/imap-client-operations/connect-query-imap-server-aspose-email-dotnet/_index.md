# .NET के लिए Aspose.Email के साथ IMAP सर्वर को कनेक्ट करना और क्वेरी करना

## परिचय

आज के डिजिटल युग में, ईमेल व्यक्तिगत और व्यावसायिक दोनों ही स्थितियों में एक महत्वपूर्ण संचार उपकरण बना हुआ है। ईमेल को प्रोग्रामेटिक रूप से एक्सेस करना और प्रबंधित करना चुनौतीपूर्ण हो सकता है। यह व्यापक गाइड आपको .NET के लिए शक्तिशाली Aspose.Email लाइब्रेरी का उपयोग करके IMAP सर्वर से कनेक्ट करने में मदद करेगी। इस सुविधा-समृद्ध API का लाभ उठाकर, आप विशिष्ट मानदंडों के आधार पर ईमेल डेटा को कुशलतापूर्वक प्राप्त और क्वेरी कर सकेंगे।

- विषय पंक्ति पैटर्न के आधार पर ईमेल को फ़िल्टर करने के लिए जटिल क्वेरीज़ बनाने की तकनीकें।

## कार्यान्वयन मार्गदर्शिका

इस अनुभाग में, हम दो प्रमुख विशेषताओं का पता लगाएंगे: IMAP सर्वर से कनेक्ट करना और Aspose.Email के क्वेरी बिल्डर का उपयोग करके संदेशों को क्वेरी करना।

### विशेषता 1: IMAP सर्वर से कनेक्ट करना

यह सुविधा दर्शाती है कि Aspose.Email लाइब्रेरी का उपयोग करके IMAP सर्वर के साथ कनेक्शन कैसे स्थापित किया जाए। यह किसी भी ईमेल प्रबंधन कार्य में पहला कदम है।

#### अवलोकन
एक सुरक्षित कनेक्शन स्थापित करने से आप प्रोग्रामेटिक रूप से ईमेल तक पहुंच और प्रबंधन कर सकते हैं। `ImapClient` क्लास इस प्रक्रिया को कुशलतापूर्वक संभालता है.

#### कार्यान्वयन चरण

##### चरण 1: ImapClient का एक इंस्टेंस बनाएँ

एक उदाहरण आरंभ करके प्रारंभ करें `ImapClient` अपने सर्वर के होस्ट, उपयोगकर्ता नाम और पासवर्ड के साथ:

```csharp
using System;
using Aspose.Email.Clients.Imap;

public class ImapConnectionFeature
{
    public static void Run()
    {
        // होस्ट, उपयोगकर्ता और पासवर्ड के साथ ImapClient का एक उदाहरण बनाएँ
        ImapClient client = new ImapClient("imap.gmail.com", 993, "your-email@gmail.com", "your-password");
        
        // सुरक्षित कनेक्शन के लिए SSL का उपयोग करें
        client.SecurityOptions = SecurityOptions.Auto;
        
        try
        {
            // जाँचें कि कनेक्शन सफल है या नहीं
            if (client.IsConnected)
            {
                Console.WriteLine("Connection established successfully.");
            }
        }
        catch (Exception ex)
        {
            Console.WriteLine($"Error connecting to IMAP server: {ex.Message}");
        }
    }
}
```

##### चरण 2: कनेक्शन सत्यापित करें

सुनिश्चित करें कि आपके क्रेडेंशियल सही हैं और सर्वर पहुंच योग्य है `IsConnected`यह कदम कॉन्फ़िगरेशन समस्याओं को जल्दी पहचानने में मदद करता है।

### फ़ीचर 2: IMAP क्वेरी बिल्डर का उपयोग करके संदेशों को क्वेरी करना

यह सुविधा दिखाती है कि Aspose.Email के अंतर्निहित क्वेरी बिल्डर का उपयोग करके विशिष्ट विषय मानदंडों के आधार पर ईमेल फ़िल्टर करने के लिए जटिल खोज क्वेरी का निर्माण कैसे किया जाए।

#### अवलोकन
परिष्कृत ईमेल फ़िल्टर बनाने की क्षमता के साथ, आप अपनी खोज प्रक्रिया को सुव्यवस्थित कर सकते हैं और केवल प्रासंगिक संदेश ही प्राप्त कर सकते हैं।

#### कार्यान्वयन चरण

##### चरण 1: ImapClient आरंभ करें

सुनिश्चित करें कि आपका IMAP क्लाइंट मान्य क्रेडेंशियल्स के साथ आरंभ किया गया है:

```csharp
using System;
using Aspose.Email.Clients.Imap;
using Aspose.Email.Tools.Search;

public class ImapQueryFeature
{
    public static void Run()
    {
        using (ImapClient client = new ImapClient("imap.gmail.com", 993, "your-email@gmail.com", "your-password"))
        {
            // सुरक्षित कनेक्शन के लिए SSL का उपयोग करें
            client.SecurityOptions = SecurityOptions.Auto;
```

##### चरण 2: एक क्वेरी बनाएँ

उपयोग `ImapQueryBuilder` ईमेल विषयों में विशिष्ट पैटर्न की खोज करने वाली क्वेरीज़ का निर्माण करना:

```csharp
// ImapQueryBuilder का एक उदाहरण बनाएँ
ImapQueryBuilder builder = new ImapQueryBuilder();

// तार्किक OR शर्तों का उपयोग करके क्वेरी का निर्माण करें
MailQuery query = builder.Or(
    builder.Subject.Contains(" (1) "),
    builder.Subject.Contains(" (2) "),
    builder.Subject.Contains(" (3) "),
    builder.Subject.Contains(" (4) "),
    builder.Subject.Contains(" (5) "));
```

##### चरण 3: क्वेरी निष्पादित करें

अपने क्वेरी मानदंड के आधार पर संदेश प्राप्त करें और सफल पुनर्प्राप्ति सत्यापित करें:

```csharp
// इनबॉक्स फ़ोल्डर का चयन करें
client.SelectFolder(ImapFolderInfo.InBox);

try
{
    // संदेश जानकारी प्राप्त करने के लिए क्वेरी निष्पादित करें
    ImapMessageInfoCollection messageInfoCol = client.ListMessages(query, 4);
    Console.WriteLine((messageInfoCol.Count == 4) ? "Success" : "Failure");
}
catch (Exception ex)
{
    Console.WriteLine($"Error querying IMAP server: {ex.Message}");
}
    }
}
```
