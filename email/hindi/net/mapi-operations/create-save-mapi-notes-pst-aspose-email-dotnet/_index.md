## कार्यान्वयन मार्गदर्शिका
यह अनुभाग चरण-दर-चरण MAPI नोट्स के निर्माण और PST फ़ाइल में सहेजने को कवर करता है।

### मौजूदा PST फ़ाइल बनाएँ और हटाएँ
अपनी दस्तावेज़ निर्देशिका सेट अप करके और मौजूदा फ़ाइलों को संभालकर आरंभ करें:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // इसे अपने वास्तविक पथ पर सेट करें
if (File.Exists(dataDir + "/SampleNote_out.pst"))
{
    File.Delete(dataDir + "/SampleNote_out.pst"); // यदि मौजूद हो तो उसे हटा दें ताकि नई शुरुआत हो सके
}
```

### एक नई PST फ़ाइल और पूर्वनिर्धारित फ़ोल्डर बनाएँ
पूर्वनिर्धारित "नोट्स" फ़ोल्डर के साथ यूनिकोड प्रारूप में एक नई PST फ़ाइल बनाएँ:
```csharp
using (PersonalStorage pst = PersonalStorage.Create(dataDir + "/SampleNote_out.pst", FileFormatVersion.Unicode))
{
    FolderInfo notesFolder = pst.CreatePredefinedFolder("Notes", StandardIpmFolder.Notes);
```

### MSG को MAPI नोट में लोड करें और परिवर्तित करें
एक मौजूदा MSG फ़ाइल लोड करें और इसे में परिवर्तित करें `MapiMessage`:
```csharp
MapiMessage message = MapiMessage.FromFile(dataDir + "/Note.msg"); // सुनिश्चित करें कि आपके पास यह MSG फ़ाइल उपलब्ध है
```

### नोट्स बनाएं और अनुकूलित करें
#### नोट #1: पीले रंग का नोट
पहले नोट के लिए विषय, मुख्य पाठ और रंग जैसे गुण सेट करें.
```csharp
// पीले रंग से नोट #1 बनाएं
MapiNote note1 = (MapiNote)message.ToMapiMessageItem();
note1.Subject = "Yellow color note";
note1.Body = "This is a yellow color note";
```

#### नोट #2: गुलाबी रंग का नोट
दूसरे नोट को विभिन्न गुणों के साथ अनुकूलित करें।
```csharp
// गुलाबी रंग से नोट #2 बनाएं
MapiNote note2 = (MapiNote)message.ToMapiMessageItem();
note2.Subject = "Pink color note";
note2.Body = "This is a pink color note";
note2.Color = NoteColor.Pink;
```

#### नोट #3: आयामों के साथ नीले रंग का नोट
अधिक अनुकूलन के लिए तीसरे नोट में आयाम जोड़ें।
```csharp
// नीले रंग और विशिष्ट आयामों के साथ नोट #3 बनाएं
MapiNote note3 = (MapiNote)message.ToMapiMessageItem();
note3.Subject = "Blue color note";
note3.Body = "This is a blue color note";
note3.Color = NoteColor.Blue;
note3.Height = 500; // कस्टम ऊंचाई
note3.Width = 500; // कस्टम चौड़ाई
```

### नोट्स को PST फ़ाइल में सहेजें
सभी बनाए गए नोट्स को अपनी नई PST फ़ाइल के 'नोट्स' फ़ोल्डर में जोड़ें:
```csharp
// फ़ोल्डर में नोट्स जोड़ें
notesFolder.AddMapiMessageItem(note1);
notesFolder.AddMapiMessageItem(note2);
notesFolder.AddMapiMessageItem(note3);
}
```
