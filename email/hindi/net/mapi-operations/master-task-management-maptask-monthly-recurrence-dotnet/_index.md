## कार्यान्वयन मार्गदर्शिका

इस अनुभाग में दिनांक समायोजन के साथ MapiTask बनाना और मासिक पुनरावृत्ति सेट करना शामिल है।

### दिनांक समायोजन के साथ MapiTask बनाना

निम्नलिखित चरणों का उपयोग करके स्थानीय समय क्षेत्र सेटिंग का सम्मान करने वाले कार्य बनाएँ:

#### चरण 1: अपने कार्य का विवरण निर्धारित करें

निर्देशिकाओं के लिए प्लेसहोल्डर्स को परिभाषित करके, वर्तमान समय क्षेत्र को पुनः प्राप्त करके, और स्थानीय समय ऑफसेट की गणना करके प्रारंभ करें:

```csharp
using Aspose.Email.Mapi;
using System;

public class Feature1
{
    public static void Run()
    {
        string documentDirectory = "@YOUR_DOCUMENT_DIRECTORY";
        string outputDirectory = "@YOUR_OUTPUT_DIRECTORY";

        TimeZone localZone = TimeZone.CurrentTimeZone;
        TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);

        DateTime startDate = new DateTime(2015, 7, 1).Add(ts);
        DateTime dueDate = new DateTime(2015, 7, 1).Add(ts);

        MapiTask task = new MapiTask("This is a test task", "Sample Body", startDate, dueDate);
        
        task.State = MapiTaskState.NotAssigned;
    }
}
```

**स्पष्टीकरण:**
- The `TimeZone.CurrentTimeZone.GetUtcOffset` विधि आपके कार्य की शुरुआत और नियत तारीखों को तदनुसार समायोजित करने के लिए स्थानीय समय ऑफसेट की गणना करती है।
- सेटिंग `MapiTask.State` संपत्ति आपके कार्य की वर्तमान स्थिति को परिभाषित करती है.

### किसी कार्य के लिए मासिक पुनरावृत्ति कॉन्फ़िगर करना

कार्यों के लिए अक्सर पुनरावृत्ति पैटर्न की आवश्यकता होती है। इन चरणों के साथ एक मासिक पुनरावृत्ति सेट करें जो कभी समाप्त न हो:

#### चरण 2: पुनरावृत्ति पैटर्न परिभाषित करें

इसका एक उदाहरण बनाएं `MapiCalendarMonthlyRecurrencePattern` यह सुनिश्चित करने के लिए कि यह अनिश्चित काल तक हर महीने दोहराया जाता रहे:

```csharp
using Aspose.Email.Mapi;

public class Feature2
{
    public static void Run()
    {
        var recurrence = new MapiCalendarMonthlyRecurrencePattern
        {
            Day = 15,                  // प्रत्येक माह की 15 तारीख को पुनरावृत्ति
            Period = 1,                // प्रत्येक माह
            PatternType = MapiCalendarRecurrencePatternType.Month,
            EndType = MapiCalendarRecurrenceEndType.NeverEnd,
            WeekStartDay = DayOfWeek.Monday
        };
    
        // उदाहरण उपयोग:
        // MapiTask कार्य = नया MapiTask ("नमूना कार्य", "बॉडी", प्रारंभ दिनांक, देय दिनांक);
        // कार्य.पुनरावृत्ति = पुनरावृत्ति;
    }
}
```

**स्पष्टीकरण:**
- The `Day` संपत्ति महीने में वह दिन निर्दिष्ट करती है जब कार्य दोहराया जाता है।
- सेटिंग `EndType` को `NeverEnd` यह सुनिश्चित करता है कि यह पैटर्न अनिश्चित काल तक जारी रहे।

### समस्या निवारण युक्तियों

सामान्य मुद्दों में शामिल हैं:
- **समय क्षेत्र बेमेल:** सुनिश्चित करें कि सटीक तिथि समायोजन के लिए आपका सिस्टम समय क्षेत्र सही ढंग से कॉन्फ़िगर किया गया है।
- **पुनरावर्ती त्रुटियाँ:** यदि कार्य अपेक्षित रूप से नहीं दोहराया जाता है तो पुनरावृत्ति पैरामीटर की दोबारा जांच करें।
