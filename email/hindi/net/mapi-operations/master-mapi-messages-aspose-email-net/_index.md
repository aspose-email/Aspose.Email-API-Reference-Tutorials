## कार्यान्वयन मार्गदर्शिका

### MAPI संदेश बनाना और कॉन्फ़िगर करना

#### अवलोकन
नया MAPI संदेश बनाने में इसे प्रेषक, प्राप्तकर्ता विवरण, विषय और मुख्य भाग के साथ आरंभ करना शामिल है। हम यह भी पता लगाएंगे कि विशिष्ट फ़्लैग और गुण कैसे सेट करें।

#### चरण 1: नया MAPI संदेश बनाएँ
इसका एक उदाहरण बनाएं `MapiMessage`:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // अपने दस्तावेज़ निर्देशिका पथ से बदलें

// संदेश आरंभ करें
double time = DateTime.Now.TimeOfDay.TotalSeconds;
string uniqueSubject = $"Unique Subject {time}";

MapiMessage msg = new MapiMessage(
    "from@test.com",
    "to@test.com",
    uniqueSubject,
    "Make it nice and short, but descriptive. The description may appear in search engines' search results pages..."
);
```

#### चरण 2: संदेश फ़्लैग कॉन्फ़िगर करें
वैकल्पिक रूप से, आप विशिष्ट फ़्लैग को टॉगल करके ईमेल को भेजे गए ईमेल के रूप में अनुकरण कर सकते हैं:

```csharp
bool draft = false; // यदि आप ड्राफ्ट चाहते हैं तो इसे true पर सेट करें
if (!draft) {
    msg.SetMessageFlags(msg.Flags ^ MapiMessageFlags.MSGFLAG_UNSENT);
}
```

#### चरण 3: संदेश सहेजें
अपना संदेश निर्दिष्ट निर्देशिका में सहेजें:

```csharp
msg.Save(dataDir + "/MapiMsgExample.msg");
```

### MAPI संदेशों से वोटिंग बटन सेट करना और हटाना

#### अवलोकन
वोटिंग बटन जोड़ने से इंटरैक्टिव ईमेल बेहतर हो सकते हैं। हम इन विकल्पों को जोड़ने और हटाने के बारे में बात करेंगे।

#### चरण 1: मौजूदा संदेश बनाएँ या लोड करें
अनुवर्ती विकल्पों के साथ एक नया संदेश बनाएँ:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // अपने दस्तावेज़ निर्देशिका पथ से बदलें

MapiMessage msgWithPoll = new MapiMessage(
    "from@test.com",
    "to@test.com",
    "Voting Message",
    "Select your option."
);
```

#### चरण 2: वोटिंग बटन सेट करें
मतदान विकल्पों को कॉन्फ़िगर करें `FollowUpOptions`:

```csharp
FollowUpOptions options = new FollowUpOptions();
options.VotingButtons = "Yes;No;Maybe;Exactly!";
FollowUpManager.SetOptions(msgWithPoll, options);

msgWithPoll.Save(dataDir + "/MapiMsgWithPoll.msg");
```

#### चरण 3: वोटिंग बटन हटाएँ
आप विशिष्ट या सभी मतदान बटन हटा सकते हैं:

```csharp
// किसी विशिष्ट बटन को हटाने के लिए
FollowUpManager.RemoveVotingButton(msgWithPoll, "Exactly!");

// या सभी मतदान बटन साफ़ करें
FollowUpManager.ClearVotingButtons(msgWithPoll);
```

#### चरण 4: अपडेट किया गया संदेश सहेजें
सुनिश्चित करें कि आपने अपने परिवर्तन सहेज लिए हैं:

```csharp
msgWithPoll.Save(dataDir + "/MapiMsgUpdated.msg");
```

## अक्सर पूछे जाने वाले प्रश्न अनुभाग
1. **मैपीमैसेज क्या है?**
   - MAPI संदेश एक ईमेल का प्रतिनिधित्व करने वाला ऑब्जेक्ट है, जो झंडे और वोटिंग विकल्प जैसे विभिन्न गुणों को सेट करने की अनुमति देता है।
2. **क्या मैं तुरंत लाइसेंस खरीदे बिना Aspose.Email का उपयोग कर सकता हूँ?**
   - हां, पहले इसकी विशेषताओं को जानने के लिए निःशुल्क परीक्षण या अस्थायी लाइसेंस से शुरुआत करें।
3. **मैं किसी संदेश से विशिष्ट मतदान बटन कैसे हटाऊं?**
   - उपयोग `FollowUpManager.RemoveVotingButton()` विधि, संदेश ऑब्जेक्ट और बटन पाठ को पास करना।

   - हाँ, टॉगल करके `MSGFLAG_UNSENT` उचित ढंग से ध्वज लगाएं।
5. **Aspose.Email का उपयोग करते समय कुछ प्रदर्शन संबंधी विचार क्या हैं?**
   - कुशल मेमोरी प्रबंधन महत्वपूर्ण है; उन वस्तुओं को हटा दें जिनकी अब आवश्यकता नहीं है और बेहतर अनुप्रयोग प्रत्युत्तरशीलता के लिए अतुल्यकालिक परिचालनों पर विचार करें।
