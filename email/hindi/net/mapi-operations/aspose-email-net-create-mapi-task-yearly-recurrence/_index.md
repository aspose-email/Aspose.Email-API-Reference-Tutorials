## कार्यान्वयन मार्गदर्शिका
यह अनुभाग विशिष्ट गुणों के साथ MAPI कार्य बनाने और वार्षिक पुनरावृत्ति सेट अप करने को कवर करता है।
### MapiTask बनाएं और सहेजें
#### अवलोकन
टास्क बनाने में इसके गुण जैसे विषय, मुख्य भाग, आरंभ तिथि, नियत तिथि और स्थिति को परिभाषित करना शामिल है। हम इसे एक के रूप में सहेजेंगे `.msg` फ़ाइल, आउटलुक कार्यों के लिए मानक है।
#### कार्यान्वयन चरण
**1. निर्देशिकाएँ सेट करें**
अपने दस्तावेज़ और आउटपुट निर्देशिकाओं के लिए पथ परिभाषित करें:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```
**2. समय क्षेत्र कॉन्फ़िगर करें**
स्थानीय समय क्षेत्र के आधार पर तिथियाँ समायोजित करें:
```csharp
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan timeSpan = localZone.GetUtcOffset(DateTime.Now);
DateTime StartDate = new DateTime(2023, 1, 1).Add(timeSpan);
DateTime DueDate = new DateTime(2023, 12, 31).Add(timeSpan);
```
**3. मैपीटास्क बनाएं**
एक उदाहरण बनाना `MapiTask` निर्दिष्ट गुणों के साथ:
```csharp
MapiTask task = new MapiTask("Yearly Review Task", "Annual review of project milestones.", StartDate, DueDate);
task.State = MapiTaskState.NotStarted;
```
**4. कार्य को .msg फ़ाइल के रूप में सहेजें**
निर्मित कार्य को आउटपुट निर्देशिका में सहेजें:
```csharp
string filePath = outputDir + "/YearlyReviewTask_out.msg";
task.Save(filePath, TaskSaveFormat.Msg);
```
### MapiTask के लिए वार्षिक पुनरावृत्ति सेट करें
#### अवलोकन
पुनरावृत्ति पैटर्न उन कार्यों के लिए महत्वपूर्ण हैं जो समय के साथ दोहराए जाते हैं। हम यहां एक वार्षिक पुनरावृत्ति पैटर्न सेट करेंगे।
#### कार्यान्वयन चरण
**1. पुनरावृत्ति पैटर्न को परिभाषित करें**
एक बनाने के `MapiCalendarYearlyRecurrencePattern`:
```csharp
MapiCalendarYearlyRecurrencePattern rec = new MapiCalendarYearlyRecurrencePattern
{
    DayOfMonth = 15,
    MonthOfYear = MapiMonth.January, // जनवरी में शुरू करें
    Type = MapiCalendarRecurrenceType.Month,
    EndType = MapiCalendarRecurrenceEndType.NeverEnding,
};
```
**2. कार्य को पुनरावृत्ति असाइन करें**
कार्य को पुनरावृत्ति पैटर्न निर्दिष्ट करें:
```csharp
MapiTask task = new MapiTask("Yearly Review Task", "Annual review of project milestones.", DateTime.Now, DateTime.Now.AddDays(1));
task.Recurrence = rec;
```
**3. आवर्ती कार्य सहेजें**
आवर्ती कार्य को गैर-आवर्ती कार्य के समान ही सहेजें:
```csharp
string filePath = outputDir + "/YearlyReviewTask_out.msg";
task.Save(filePath, TaskSaveFormat.Msg);
```
### समस्या निवारण युक्तियों
- सुनिश्चित करें कि रास्ते `dataDir` और `outputDir` सही हैं.
- सत्यापित करें कि Aspose.Email सीमाओं से बचने के लिए सही ढंग से लाइसेंस प्राप्त है।
- यदि कार्य गलत तिथियों के साथ दिखाई देते हैं तो समय क्षेत्र सेटिंग जांचें.
