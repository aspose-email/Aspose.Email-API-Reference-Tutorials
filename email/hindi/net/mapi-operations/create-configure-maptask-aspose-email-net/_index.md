## कार्यान्वयन मार्गदर्शिका

### MapiTask बनाना और कॉन्फ़िगर करना

**अवलोकन:** यह सुविधा आपको विस्तृत गुणों वाले कार्य बनाने और आवधिक अनुस्मारकों के लिए पुनरावृत्ति पैटर्न सेट करने की अनुमति देती है।

#### चरण 1: एक नया MapiTask बनाएँ
इसका एक उदाहरण बनाकर शुरू करें `MapiTask`:
```csharp
using Aspose.Email.Mapi;

// शीर्षक, मुख्य भाग, प्रारंभ और नियत तिथियों के साथ एक नया कार्य आरंभ करें
MapiTask task = new MapiTask("This is test task", "Sample Body", new DateTime(2015, 7, 1), new DateTime(2015, 7, 1));
task.State = MapiTaskState.NotAssigned;
```
**स्पष्टीकरण:** यहाँ, `MapiTask` शीर्षक और मुख्य भाग के साथ आरंभ किया गया है। आरंभ और नियत तिथियाँ प्रारंभ में 1 जुलाई, 2015 निर्धारित की गई हैं।

#### चरण 2: वार्षिक पुनरावृत्ति पैटर्न निर्धारित करें
इसके बाद, कार्य को वार्षिक रूप से दोहराने के लिए कॉन्फ़िगर करें:
```csharp
// 15वें दिन से शुरू होने वाला वार्षिक पुनरावृत्ति पैटर्न निर्धारित करें, जो 3 बार प्रत्येक 12 महीने में दोहराया जाए
var rec = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15,
    Period = 12,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    OccurrenceCount = 3,
};

// अमान्य कॉन्फ़िगरेशन से बचने के लिए सुनिश्चित करें कि घटना की संख्या कम से कम 1 हो
if (rec.OccurrenceCount == 0)
{
    rec.OccurrenceCount = 1;
}
task.Recurrence = rec;
```
**स्पष्टीकरण:** यह ब्लॉक 15 जुलाई से शुरू होने वाली वार्षिक पुनरावृत्ति की स्थापना करता है, जो हर साल तीन पुनरावृत्तियों के लिए होती है।

### समय क्षेत्र समायोजन

**अवलोकन:** विभिन्न क्षेत्रों में सटीक शेड्यूलिंग सुनिश्चित करने के लिए स्थानीय समय क्षेत्र ऑफसेट के अनुसार कार्य तिथियों को समायोजित करें।

#### चरण 3: स्थानीय समय क्षेत्र ऑफसेट प्राप्त करें
समायोजित करना `DateTime` वर्तमान स्थानीय समय क्षेत्र का उपयोग करने वाली वस्तुएँ:
```csharp
using System;

// वर्तमान समयक्षेत्र और उसका UTC ऑफ़सेट प्राप्त करें
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);

// स्थानीय समय क्षेत्र ऑफ़सेट जोड़कर तिथियाँ समायोजित करें
DateTime StartDate = new DateTime(2015, 7, 1).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 1).Add(ts);
DateTime endByDate = new DateTime(2020, 12, 31).Add(ts);
```
**स्पष्टीकरण:** यह कोड स्थानीय समय क्षेत्र को प्रतिबिंबित करने के लिए कार्य प्रारंभ और नियत तिथियों को समायोजित करता है, जो विभिन्न भौगोलिक स्थानों पर उपयोग किए जाने वाले अनुप्रयोगों के लिए आवश्यक है।
