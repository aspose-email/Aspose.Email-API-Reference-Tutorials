## कार्यान्वयन मार्गदर्शिका

### फ़ाइल पथ सेट करना
**अवलोकन**फ़ाइल पथ को परिभाषित करने से यह सुनिश्चित होता है कि आपका अनुप्रयोग जानता है कि इनपुट फ़ाइलों को कहाँ पढ़ना है और आउटपुट को कहाँ सहेजना है।
```csharp
using System;

string documentDirectory = @"C:\\Your\\Document\\Directory"; // वास्तविक पथ से प्रतिस्थापित करें
string outputDirectory = @"C:\\Your\\Output\\Directory";      // वास्तविक पथ से प्रतिस्थापित करें
```
### MapiJournal ऑब्जेक्ट बनाना
**अवलोकन**MapiJournal ऑब्जेक्ट संरचित जर्नल प्रविष्टियाँ बनाने के लिए केंद्रीय है।
```csharp
using Aspose.Email.Mapi;

MapiJournal journal = new MapiJournal(
    subject: "testJournal",
    body: "This is a test journal",
    categories: "Phone call",
    importance: MapiMessageImportance.Normal);

// जर्नल प्रविष्टि के लिए आरंभ और समाप्ति समय निर्धारित करें
journal.StartTime = DateTime.Now;
journal.EndTime = journal.StartTime.AddHours(1);
// इस जर्नल प्रविष्टि में शामिल कंपनियों को जोड़ें
timeJournal.Companies = new string[] { "company 1", "company 2", "company 3" };
```
### MapiJournal में अनुलग्नक जोड़ना
**अवलोकन**प्रासंगिक फ़ाइलें संलग्न करके अपनी जर्नल प्रविष्टियों को बेहतर बनाएँ।
```csharp
using System.IO;

string[] attachFileNames = new string[] {
    documentDirectory + "\\Desert.jpg",
    documentDirectory + "\\download.png"
};

foreach (string attach in attachFileNames)
{
    // फ़ाइल बाइट्स का उपयोग करके MapiJournal में फ़ाइलें संलग्न करें
    journal.Attachments.Add(attach, File.ReadAllBytes(attach));
}
```
### MapiJournal ऑब्जेक्ट सहेजा जा रहा है
**अवलोकन**अंत में, अपने कॉन्फ़िगर किए गए MapiJournal ऑब्जेक्ट को MSG फ़ाइल के रूप में सहेजें।
```csharp
journal.Save(outputDirectory + "\\AddAttachmentsToMapiJournal_out.msg");
```
