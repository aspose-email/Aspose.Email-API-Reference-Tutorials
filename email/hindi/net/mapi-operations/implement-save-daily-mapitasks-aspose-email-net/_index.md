# .NET में Aspose.Email के साथ दैनिक आवर्ती MapiTasks को क्रियान्वित करें और सहेजें

## परिचय

उत्पादकता बनाए रखने के लिए कुशल कार्य प्रबंधन आवश्यक है, खासकर जब आवर्ती घटनाओं से निपटना हो। चाहे आप व्यक्तिगत रूप से या किसी बड़े संगठन के भीतर कार्यों का प्रबंधन कर रहे हों, स्वचालित अनुस्मारक सेट अप करने से समय की बचत हो सकती है और त्रुटियों को कम किया जा सकता है। यह ट्यूटोरियल आपको Aspose.Email .NET लाइब्रेरी का उपयोग करके दैनिक आवर्ती MapiTasks बनाने और प्रबंधित करने के बारे में मार्गदर्शन करेगा।

.NET के लिए Aspose.Email का लाभ उठाकर, आपके एप्लिकेशन में ईमेल कार्यक्षमताओं को एकीकृत करना सहज हो जाता है, जिससे कार्य प्रबंधन कुशल हो जाता है। इस गाइड में, आप सीखेंगे:
- .NET के लिए Aspose.Email कैसे सेट करें

- दैनिक पुनरावृत्ति पैटर्न को लागू करना

आइये, पूर्वापेक्षाओं से शुरुआत करें!

## कार्यान्वयन मार्गदर्शिका

### MapiTask बनाना

#### अवलोकन

मैपीटास्क बनाने में शीर्षक, विवरण, आरंभ तिथि और नियत तिथि जैसे गुणों को परिभाषित करना शामिल है।

#### चरण-दर-चरण कार्यान्वयन

**कार्य विवरण परिभाषित करें**
```csharp
using Aspose.Email.Mapi;
using System;

public static void CreateMapiTask()
{
    // आरंभ तिथि और नियत तिथि के साथ कार्य विवरण परिभाषित करें
    DateTime StartDate = new DateTime(2023, 10, 1);
    DateTime DueDate = new DateTime(2023, 10, 2);

    // शीर्षक, मुख्य भाग, प्रारंभ और नियत तिथियों के साथ MapiTask को तत्काल बनाएं
    MapiTask task = new MapiTask("Daily Report", "Prepare the daily report", StartDate, DueDate);

    // कार्य की प्रारंभिक स्थिति को NotAssigned के रूप में सेट करें
    task.State = MapiTaskState.NotAssigned;
}
```
**स्पष्टीकरण**: द `MapiTask` कंस्ट्रक्टर शीर्षक और विवरण के साथ-साथ प्रारंभ और नियत तिथियों के लिए पैरामीटर लेता है। `State` को `NotAssigned` यह दर्शाता है कि कार्य अभी तक सौंपा नहीं गया है.

### किसी कार्य के लिए दैनिक पुनरावृत्ति सेट करना

#### अवलोकन

ऐसे कार्यों के लिए जिनमें दोहराव की आवश्यकता होती है, जैसे दैनिक अनुस्मारक, पुनरावृत्ति पैटर्न स्थापित करना आवश्यक है।

#### चरण-दर-चरण कार्यान्वयन

**पुनरावृत्ति पैटर्न को परिभाषित और निर्दिष्ट करें**
```csharp
public static void SetDailyRecurrence()
{
    // कार्य प्रारंभ और नियत तिथियां निर्धारित करें
    DateTime StartDate = new DateTime(2023, 10, 1);
    DateTime DueDate = new DateTime(2023, 10, 2);

    // MapiTask इंस्टैंस बनाएँ
    MapiTask task = new MapiTask("Daily Report", "Prepare the daily report", StartDate, DueDate);

    // दैनिक पुनरावृत्ति पैटर्न कॉन्फ़िगर करें
    var record = new MapiCalendarDailyRecurrencePattern
    {
        PatternType = MapiCalendarRecurrencePatternType.Day,
        Period = 1,
        WeekStartDay = DayOfWeek.Sunday,
        EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
        OccurrenceCount = 5 // कार्य पांच बार होगा
    };

    // कार्य को पुनरावृत्ति पैटर्न निर्दिष्ट करें
    task.Recurrence = record;
}
```
**स्पष्टीकरण**: द `MapiCalendarDailyRecurrencePattern` क्लास आपको यह निर्दिष्ट करने की अनुमति देता है कि कोई कार्य कितनी बार दोहराया जाएगा। यहाँ, इसे प्रतिदिन दोहराने के लिए सेट किया गया है (`Period = 1`) पांच बार दोहराया गया।

### किसी कार्य को MSG फ़ाइल के रूप में सहेजना

#### अवलोकन

MapiTask को .msg फ़ाइल के रूप में सहेजने से कार्यों का आसान वितरण और संग्रहण संभव हो जाता है।

#### चरण-दर-चरण कार्यान्वयन

**MapiTask सहेजें**
```csharp
public static void SaveTaskAsMsg()
{
    // पुनरावृत्ति पैटर्न के साथ कार्य विवरण परिभाषित करें
    DateTime StartDate = new DateTime(2023, 10, 1);
    DateTime DueDate = new DateTime(2023, 10, 2);

    MapiTask task = new MapiTask("Daily Report", "Prepare the daily report", StartDate, DueDate);
    
    var record = new MapiCalendarDailyRecurrencePattern
    {
        PatternType = MapiCalendarRecurrencePatternType.Day,
        Period = 1,
        WeekStartDay = DayOfWeek.Sunday,
        EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
        OccurrenceCount = 5
    };

    task.Recurrence = record;

    // सहेजने के लिए फ़ाइल पथ निर्धारित करें
    string outputDir = "YOUR_OUTPUT_DIRECTORY";

    // MapiTask को MSG फ़ाइल के रूप में सहेजें
    task.Save(outputDir + "/DailyReport_out.msg", TaskSaveFormat.Msg);
}
```
**स्पष्टीकरण**: द `Save` विधि MapiTask को MSG प्रारूप में निर्दिष्ट पथ पर लिखती है, जो आउटलुक जैसे ईमेल क्लाइंट के साथ संगत है।

- **परियोजना प्रबंधन**: दैनिक स्थिति अद्यतन या स्टैंड-अप अनुस्मारक को स्वचालित करें।

- **टीम समन्वय**: स्वचालित रूप से नियमित चेक-इन या प्रगति बैठकें सेट करें।
- **व्यक्तिगत उत्पादकता**: एक दैनिक कार्य सूची बनाए रखें जो सभी डिवाइस पर एक जैसी हो।
- **कैलेंडर के साथ एकीकरण**कार्य अनुस्मारकों को सीधे कैलेंडर अनुप्रयोगों में सिंक करें।
