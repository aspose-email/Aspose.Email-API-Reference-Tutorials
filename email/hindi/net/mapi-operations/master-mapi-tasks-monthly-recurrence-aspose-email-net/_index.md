## कार्यान्वयन मार्गदर्शिका

### MAPI कार्य बनाना और सहेजना
आइए एक सरल MAPI कार्य बनाकर और उसे MSG फ़ाइल के रूप में सहेजकर शुरू करें। यह कार्यक्षमता कार्य निर्माण को स्वचालित करने में मदद करती है, जिससे स्थिरता और दक्षता सुनिश्चित होती है।

**चरण 1: कार्य गुण परिभाषित करें**
अपने कार्य के लिए आरंभ और नियत तिथि निर्धारित करके आरंभ करें:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);
DateTime StartDate = new DateTime(2015, 7, 1).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 1).Add(ts);
```

**चरण 2: MAPI कार्य बनाएँ**
आरंभ करें `MapiTask` आपके परिभाषित गुणों के साथ:
```csharp
MapiTask task = new MapiTask("This is test task", "Sample Body", StartDate, DueDate);
task.State = MapiTaskState.NotAssigned;
```
इस स्निपेट में:
- "यह परीक्षण कार्य है" और "नमूना निकाय" कार्य का विषय और निकाय हैं।
- `StartDate` और `DueDate` कार्य कब शुरू और कब समाप्त होगा यह निर्दिष्ट करें.

**चरण 3: कार्य को सहेजें**
अपना कार्य MSG प्रारूप में सहेजें:
```csharp
task.Save(dataDir + "Monthly_out.msg", TaskSaveFormat.Msg);
```

### MAPI कार्य के लिए मासिक पुनरावृत्ति पैटर्न कॉन्फ़िगर करना
इसके बाद, किसी मौजूदा MAPI कार्य ऑब्जेक्ट पर मासिक पुनरावृत्ति पैटर्न सेट करें। यह उन कार्यों के लिए आदर्श है जिन्हें नियमित अंतराल पर दोहराए जाने की आवश्यकता होती है।

**चरण 1: पुनरावृत्ति पैटर्न को परिभाषित करें**
एक बनाने के `MapiCalendarMonthlyRecurrencePattern`:
```csharp
var rec = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15,
    Period = 12,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    OccurrenceCount = 3,
    WeekStartDay = DayOfWeek.Monday
};
```
यह कॉन्फ़िगरेशन कार्य को हर महीने की 15 तारीख को दोहराने के लिए सेट करता है, जो 12 महीने की अवधि में तीन बार दोहराया जाता है।

**चरण 2: कार्य पर पुनरावृत्ति लागू करें**
अपने पुनरावृत्ति पैटर्न को निर्दिष्ट करें `MapiTask`:
```csharp
MapiTask taskWithRecurrence = new MapiTask("This is test task", "Sample Body", DateTime.Now, DateTime.Now.AddDays(30));
taskWithRecurrence.Recurrence = rec;
```

**चरण 3: पुनरावृत्ति के साथ कार्य को सहेजें**
अपने आवर्ती कार्य को MSG फ़ाइल के रूप में सहेजें:
```csharp
string dataDirOutput = "YOUR_OUTPUT_DIRECTORY";
taskWithRecurrence.Save(dataDirOutput + "Monthly_out_with_recurrence.msg", TaskSaveFormat.Msg);
```
