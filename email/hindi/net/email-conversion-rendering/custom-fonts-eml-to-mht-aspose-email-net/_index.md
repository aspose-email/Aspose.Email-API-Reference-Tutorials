### कार्यान्वयन मार्गदर्शिका
यह अनुभाग आपको EML से MHT रूपांतरण के दौरान फ़ॉन्ट बदलने के बारे में मार्गदर्शन करता है।

#### चरण 1: निर्देशिका पथ सेट करें
अपनी इनपुट और आउटपुट फ़ाइलों के लिए पथ परिभाषित करें:

```csharp
string dataDir = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "Data");
string inputFile = Path.Combine(dataDir, "input.eml");
string outputFile = Path.Combine(dataDir, "output.mht");
```

#### चरण 2: EML फ़ाइल लोड करें
अपनी EML फ़ाइल को एक में लोड करें `MailMessage` वस्तु:

```csharp
var message = MailMessage.Load(inputFile);
```

ईमेल लोड करने से आप रूपांतरण से पहले इसकी सामग्री में बदलाव कर सकते हैं।

#### चरण 3: फ़ॉन्ट शैली अनुकूलित करें
फ़ॉन्ट शैली बदलकर ईमेल के HTML मुख्य भाग को अनुकूलित करें:

```csharp
message.HtmlBody = message.HtmlBody.Replace("font-family", "font-family: 'Your Custom Font', sans-serif;");
```

यह कोड स्निपेट निम्नलिखित के उदाहरणों को प्रतिस्थापित करता है `font-family` अपनी इच्छित शैली के साथ.

#### चरण 4: MHT में कनवर्ट करें
संशोधित ईमेल को MHT फ़ाइल के रूप में सहेजें:

```csharp
var mhtSaveOptions = new MhtmlSaveOptions();
message.Save(outputFile, mhtSaveOptions);
```

The `MhtmlSaveOptions` यदि आवश्यक हो तो क्लास अतिरिक्त कॉन्फ़िगरेशन की अनुमति देता है।

2. **कानूनी दस्तावेज़ीकरण**: MHT फ़ाइलों के रूप में संग्रहीत कानूनी संचार में सुसंगत फ़ॉन्ट उपयोग सुनिश्चित करें।
3. **विपणन अभियान**प्रचार सामग्री की पठनीयता और आकर्षण को बढ़ाना।
