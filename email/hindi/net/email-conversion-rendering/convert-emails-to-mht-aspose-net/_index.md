## कार्यान्वयन मार्गदर्शिका

हम इस प्रक्रिया को दो मुख्य विशेषताओं में विभाजित करेंगे: वैकल्पिक सेटिंग्स के साथ ईमेल सहेजना और इनलाइन छवियों को बाहर करना।

### वैकल्पिक सेटिंग्स के साथ MHTML सहेजें

यह सुविधा आपको स्वरूपण विकल्प निर्दिष्ट करते हुए ईमेल संदेशों को MHT फ़ाइलों के रूप में सहेजने की अनुमति देती है।

#### अवलोकन

आप हेडर जानकारी शामिल करके, सामग्री एन्कोडिंग को मान्य करके, और मूल हेडर प्रदर्शित करके अपने ईमेल को सहेजने के तरीके को अनुकूलित कर सकते हैं।

#### कार्यान्वयन चरण

1. **फ़ाइल पथ सेट करें**
   
   इनपुट ईमेल पढ़ने और आउटपुट MHT फ़ाइलों को सहेजने के लिए निर्देशिका पथ परिभाषित करें।
   ```csharp
   string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Email");
   ```

2. **ईमेल संदेश लोड करें**

   उपयोग `MailMessage.Load` किसी फ़ाइल से ईमेल पढ़ने के लिए.
   ```csharp
   MailMessage eml = MailMessage.Load(Path.Combine(dataDir, "Message.eml"));
   ```

3. **MHT सहेजें विकल्प कॉन्फ़िगर करें**

   स्थापित करना `MhtSaveOptions` वांछित स्वरूपण विकल्पों के साथ.
   ```csharp
   MhtSaveOptions mhtSaveOptions = new MhtSaveOptions
   {
       MhtFormatOptions = MhtFormatOptions.WriteHeader | MhtFormatOptions.HideExtraPrintHeader | MhtFormatOptions.DisplayAsOutlook,
       CheckBodyContentEncoding = true
   };
   ```

4. **ईमेल को MHT फ़ाइल के रूप में सहेजें**

   उपयोग `Save` निर्दिष्ट विकल्पों के साथ ईमेल सामग्री लिखने की विधि।
   ```csharp
   eml.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "outMessage_out.mht"), mhtSaveOptions);
   ```

### इनलाइन छवियों के बिना MHTML में कनवर्ट करें

यह सुविधा इनलाइन छवियों को छोड़ते हुए ईमेल को MHT फ़ाइल के रूप में सहेजने का प्रदर्शन करती है।

#### अवलोकन

सेटिंग करके `SkipInlineImages` यदि आप इसे true पर सेट करते हैं, तो आप फ़ाइल में सीधे कोई छवि एम्बेड किए बिना ईमेल सामग्री को सहेज सकते हैं।

#### कार्यान्वयन चरण

1. **फ़ाइल पथ सेट करें**
   
   पहले की तरह, अपनी इनपुट और आउटपुट निर्देशिकाएँ परिभाषित करें।
   ```csharp
   string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Email");
   ```

2. **ईमेल संदेश लोड करें**

   वह ईमेल लोड करें जिसे आप परिवर्तित करना चाहते हैं.
   ```csharp
   MailMessage eml = MailMessage.Load(Path.Combine(dataDir, "Message.eml"));
   ```

3. **MHT सहेजें विकल्प कॉन्फ़िगर करें**

   तय करना `SkipInlineImages` अपने विकल्प कॉन्फ़िगरेशन में true को सेट करें.
   ```csharp
   MhtSaveOptions mhtSaveOptions = new MhtSaveOptions
   {
       SkipInlineImages = true
   };
   ```

4. **ईमेल को MHT फ़ाइल के रूप में सहेजें**

   अंत में, ईमेल को इनलाइन छवियों के बिना सेव करें।
   ```csharp
   eml.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "EmlToMhtmlWithoutInlineImages_out.mht"), mhtSaveOptions);
   ```
