## कार्यान्वयन मार्गदर्शिका
आइए कार्यान्वयन को दो मुख्य विशेषताओं में विभाजित करें: href विशेषताओं के साथ हाइपरलिंक निकालना और एंकर टैग से सादा पाठ पुनर्प्राप्त करना।

### फ़ीचर 1: Href के साथ हाइपरलिंक प्रस्तुत करें
यह सुविधा आपको HTML एंकर टैग से URL और संबंधित पाठ दोनों को निकालने की अनुमति देती है।

#### अवलोकन
आप हाइपरलिंक संदर्भ प्राप्त करने के लिए एक HTML स्ट्रिंग को पार्स करेंगे (`href`) और पाठ प्रदर्शित करें `<a>` टैग।

#### चरण-दर-चरण कार्यान्वयन

**स्टेप 1:** पहचान करें `href` विशेषता की स्थिति.

```csharp
string source = "<a href='https://example.com'>उदाहरण</a>";
int startHref = source.IndexOf("href=\"") + "href=\"".Length;
```

*क्यों?* यह चरण सटीक निष्कर्षण के लिए टैग के भीतर हाइपरलिंक के प्रारंभ स्थान का पता लगाता है।

**चरण दो:** का अंत निर्धारित करें `href` गुण।

```csharp
int endHref = source.IndexOf("\"", startHref);
string href = source.Substring(startHref, endHref - startHref);
```

*क्यों?* यह टैग के भीतर URL के अंत को चिह्नित करके उसे अलग करने में मदद करता है।

**चरण 3:** के बीच का पाठ निकालें `<a>` टैग.

```csharp
int startText = source.IndexOf(">") + 1;
int endText = source.IndexOf("<", startText);
string text = source.Substring(startText, endText - startText);
```

*क्यों?* यह आपके अनुप्रयोग में रेंडरिंग या उपयोग के लिए दृश्यमान लिंक टेक्स्ट को कैप्चर करता है।

**चरण 4:** आउटपुट के लिए टेक्स्ट और href को संयोजित करें।

```csharp
string link = $"{text} <{href}>";
Console.WriteLine(link); // आउटपुट: उदाहरण <https://example.com>
```

### फ़ीचर 2: Href के बिना हाइपरलिंक प्रस्तुत करें
यह सुविधा URL को अनदेखा करते हुए एंकर टैग से केवल दृश्यमान पाठ को निकालने पर केंद्रित है।

#### अवलोकन
यह तब उपयोगी होता है जब आपको उपयोगकर्ता इंटरफेस या आगे की प्रक्रिया के लिए केवल प्रदर्शन पाठ की आवश्यकता होती है।

#### चरण-दर-चरण कार्यान्वयन

**केवल पाठ निकालें**

```csharp
int startNoHref = source.IndexOf(">") + 1;
int endNoHref = source.IndexOf("<", startNoHref);
string plainText = source.Substring(startNoHref, endNoHref - startNoHref);
Console.WriteLine(plainText); // आउटपुट: उदाहरण
```

*क्यों?* यह विधि URL को संसाधित किए बिना कुशलतापूर्वक पाठ निकालती है।

## व्यावहारिक अनुप्रयोगों
यहां कुछ वास्तविक परिदृश्य दिए गए हैं जहां इन सुविधाओं को लागू किया जा सकता है:
