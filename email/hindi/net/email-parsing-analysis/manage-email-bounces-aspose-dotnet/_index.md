## कार्यान्वयन मार्गदर्शिका

यह अनुभाग आपको किसी फ़ाइल से ईमेल संदेश लोड करने और उसकी बाउंस स्थिति की जांच करने में मार्गदर्शन करेगा।

### ईमेल संदेश लोड करना

#### चरण 1: फ़ाइल पथ सेट करें

अपनी ईमेल फ़ाइलों का पथ निर्धारित करें:

```csharp
string fileName = "YOUR_DOCUMENT_DIRECTORY\\failed1.msg";
```

#### चरण 2: ईमेल लोड करें

किसी फ़ाइल से संदेश लोड करने के लिए Aspose.Email का उपयोग करें:

```csharp
MailMessage mail = MailMessage.Load(fileName);
```
यह चरण आपकी ईमेल सामग्री को पढ़ता है `MailMessage` आगे की प्रक्रिया के लिए वस्तु।

### बाउंस स्थिति की जाँच करना

#### चरण 3: जाँचें कि ईमेल बाउंस हुआ है या नहीं

निर्धारित करें कि ईमेल संदेश बाउंस हुआ है या नहीं:

```csharp
BounceResult result = mail.CheckBounced();
```
The `CheckBounced()` विधि एक लौटाती है `BounceResult` उछाल विवरण के साथ वस्तु.

### बाउंस विवरण को समझना

#### चरण 4: बाउंस जानकारी तक पहुंचें

के विभिन्न गुणों तक पहुँचें `BounceResult` यह समझने के लिए कि ईमेल बाउंस क्यों हुआ:

```csharp
bool isBounced = result.IsBounced;
string action = result.Action; // सुझाई गई कार्रवाइयां (जैसे, पुनः प्रयास करें)
MailAddress recipient = result.Recipient;
string reason = result.Reason; // बाउंस का कारण
string status = result.Status; // बाउंस स्थिति (जैसे, सफलता, विफलता)
// यदि उपलब्ध हो तो मूल संदेश विवरण तक पहुँचना
string originalMessageToAddress1 = result.OriginalMessage.To[0].Address;
```
प्रत्येक प्रॉपर्टी बाउंस इवेंट के बारे में जानकारी प्रदान करती है, जिससे आपको बाउंस हुए ईमेल से निपटने के बारे में सही निर्णय लेने में मदद मिलती है।
