## कार्यान्वयन मार्गदर्शिका
Aspose.Email के POP3 क्लाइंट का उपयोग करके ईमेल सारांश प्राप्त करें।

### विशिष्ट आईडी का उपयोग करके संदेश सारांश जानकारी प्राप्त करें
#### अवलोकन
संपूर्ण संदेश को डाउनलोड किए बिना विषय और दिनांक जैसी आवश्यक जानकारी प्राप्त करें, त्वरित ईमेल स्कैन के लिए आदर्श।

#### कदम
**चरण 1: सर्वर विवरण परिभाषित करें**
अपना POP3 सर्वर विवरण निर्दिष्ट करें:
```csharp
string host = "host.domain.com"; // वास्तविक होस्ट डोमेन से प्रतिस्थापित करें
int port = 456; // सही पोर्ट संख्या
string username = "username"; // वास्तविक उपयोगकर्ता नाम
string password = "password"; // वास्तविक पासवर्ड
```

**चरण 2: Pop3Client इंस्टेंस बनाएँ**
प्रारंभ `Pop3Client` और सुरक्षा विकल्प कॉन्फ़िगर करें:
```csharp
Pop3Client client = new Pop3Client(host, port, username, password);
client.SecurityOptions = SecurityOptions.Auto;
```

**चरण 3: अद्वितीय संदेश आईडी निर्धारित करें**
संदेश को उसकी विशिष्ट आईडी का उपयोग करके पहचानें:
```csharp
string uniqueId = "unique id of a message from server"; // वास्तविक विशिष्ट आईडी
```

**चरण 4: सारांश जानकारी प्राप्त करें**
सारांश विवरण प्राप्त करें `GetMessageInfo` तरीका:
```csharp
Pop3MessageInfo messageInfo = client.GetMessageInfo(uniqueId);
```

**चरण 5: संदेश विवरण आउटपुट करें**
प्राप्त जानकारी की जांच करें और प्रिंट करें:
```csharp
if (messageInfo != null)
{
    Console.WriteLine(messageInfo.Subject); // संदेश का विषय
    Console.WriteLine(messageInfo.Date);    // संदेश दिनांक
}
```
#### समस्या निवारण युक्तियों
- POP3 सर्वर क्रेडेंशियल सत्यापित करें.
- सुनिश्चित करें कि संदेश की विशिष्ट आईडी मेलबॉक्स में मौजूद है।
