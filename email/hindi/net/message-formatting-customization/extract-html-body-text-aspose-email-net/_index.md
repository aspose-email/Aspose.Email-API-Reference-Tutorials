## कार्यान्वयन मार्गदर्शिका

### HTML बॉडी टेक्स्ट निकालना: URL शामिल करें/बहिष्कृत करें

यह सुविधा आपको ईमेल की HTML सामग्री से सादा पाठ निकालने की अनुमति देती है, चाहे वह एम्बेडेड URL के साथ हो या उसके बिना।

#### चरण 1: ईमेल फ़ाइल लोड करें

सबसे पहले अपनी ईमेल फ़ाइल लोड करें:
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY"; // अपना दस्तावेज़ निर्देशिका पथ यहाँ सेट करें
MailMessage mail = MailMessage.Load(dataDir + "/HtmlWithUrlSample.eml");
```

**स्पष्टीकरण:** यह चरण आरंभ करता है `MailMessage` यह EML फ़ाइल लोड करके ऑब्जेक्ट तक पहुँच प्रदान करता है, जो इसकी HTML सामग्री तक पहुँचने के लिए महत्वपूर्ण है।

#### चरण 2: URL के साथ HTML बॉडी टेक्स्ट निकालें

अपने निकाले गए पाठ में URL शामिल करने के लिए:
```csharp
string body_with_url = mail.GetHtmlBodyText(true); // URL शामिल करने के लिए 'सत्य'
```

**स्पष्टीकरण:** The `GetHtmlBodyText` विधि ईमेल के मुख्य भाग को सादे पाठ के रूप में निकालती है, जिसमें यदि सत्य पर सेट किया गया हो तो हाइपरलिंक भी शामिल होता है।

#### चरण 3: URL के बिना HTML बॉडी टेक्स्ट निकालें

URL बहिष्कृत करने के लिए:
```csharp
string body_without_url = mail.GetHtmlBodyText(false); // URL को बाहर करने के लिए 'गलत'
```

**स्पष्टीकरण:** पैरामीटर को गलत पर सेट करने से निकाले गए पाठ से URL हट जाते हैं, जिससे विशिष्ट उपयोग मामलों के लिए साफ़ आउटपुट मिलता है।

- **लाइब्रेरी संस्करण संघर्ष:** सत्यापित करें कि आप संगत लाइब्रेरी संस्करण का उपयोग कर रहे हैं।
