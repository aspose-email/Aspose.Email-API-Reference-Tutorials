# .NET के लिए Aspose.Email के साथ ईमेल कैसे कनेक्ट करें और जोड़ें

## परिचय

ईमेल को प्रोग्रामेटिक रूप से प्रबंधित करने से आपका कार्यप्रवाह काफी हद तक सुव्यवस्थित हो सकता है। **.NET के लिए Aspose.Email** IMAP सर्वर से कनेक्ट करने और ईमेल को कुशलतापूर्वक जोड़ने के लिए एक शक्तिशाली समाधान प्रदान करता है। यह ट्यूटोरियल आपको इसका उपयोग करने में मार्गदर्शन करेगा `ImapClient` .NET में क्लास, जिससे आप आसानी से ईमेल प्रबंधन को स्वचालित कर सकते हैं।

- ImapClient का उपयोग करके IMAP सर्वर से कनेक्ट करना
- नए ईमेल संदेश बनाना और उन्हें अपने इनबॉक्स में जोड़ना

## कार्यान्वयन मार्गदर्शिका

### IMAP सर्वर से कनेक्ट करना

#### अवलोकन
यह अनुभाग आपके IMAP सर्वर से कनेक्शन स्थापित करने को कवर करता है `ImapClient`.

#### चरण-दर-चरण मार्गदर्शिका

**1. ImapClient कॉन्फ़िगर करें**
बनाएँ और कॉन्फ़िगर करें `ImapClient` अपने सर्वर विवरण के साथ इंस्टेंस:

```csharp
using Aspose.Email.Clients;

ImapClient client = new ImapClient();
client.Host = "imap.gmail.com"; // IMAP सर्वर होस्ट निर्दिष्ट करें
client.Username = "your.username@gmail.com"; // अपना ईमेल उपयोगकर्ता नाम सेट करें
client.Password = "your.password"; // अपना ईमेल पासवर्ड सेट करें
client.Port = 993; // SSL कनेक्शन के लिए मानक पोर्ट
client.SecurityOptions = SecurityOptions.Auto; // सुरक्षा विकल्पों का स्वचालित रूप से चयन करें
```

**स्पष्टीकरण:**
- `Host` IMAP सर्वर पता निर्दिष्ट करता है.
- `Username` और `Password` प्रमाणीकरण के लिए आवश्यक हैं।
- पत्तन `993` सुरक्षित कनेक्शन (SSL/TLS) के लिए उपयोग किया जाता है।
- `SecurityOptions.Auto` इष्टतम सुरक्षा सेटिंग्स सुनिश्चित करता है.

#### समस्या निवारण युक्तियों
- सुनिश्चित करें कि आपका नेटवर्क पोर्ट 993 से कनेक्शन की अनुमति देता है।
- सत्यापित करें कि आपके ईमेल क्रेडेंशियल सही हैं।

### IMAP फ़ोल्डर में नया संदेश बनाना और जोड़ना

#### अवलोकन
जानें कि नया ईमेल संदेश कैसे बनाएं और उसे इनबॉक्स फ़ोल्डर में कैसे जोड़ें।

#### चरण-दर-चरण मार्गदर्शिका

**1. मेल संदेश बनाएं**
इसका एक नया उदाहरण बनाएं `MailMessage`:

```csharp
using Aspose.Email.Clients.Imap;
using Aspose.Email.Mime;

// नए ईमेल संदेश के लिए MailMessage इंस्टेंस बनाएँ
MailMessage msg = new MailMessage("user@domain1.com", "user@domain2.com", "subject", "message");
```

**स्पष्टीकरण:**
- `MailMessage` यह प्रेषक, प्राप्तकर्ता, विषय और मुख्य भाग के विवरण के साथ एक ईमेल का प्रतिनिधित्व करता है।

**2. फ़ोल्डर चुनें**
इनबॉक्स फ़ोल्डर का चयन करें:

```csharp
// IMAP सर्वर पर इनबॉक्स फ़ोल्डर का चयन करें
client.SelectFolder(ImapFolderInfo.InBox);
```

**3. संदेश जोड़ें**
संदेश को वर्तमान फ़ोल्डर में जोड़ें:

```csharp
try
{
    // वर्तमान फ़ोल्डर में परिवर्तनों की सदस्यता लें (वैकल्पिक)
    client.SubscribeFolder(client.CurrentFolder.Name);

    // नव निर्मित संदेश को चयनित फ़ोल्डर में जोड़ें
    client.AppendMessage(client.CurrentFolder.Name, msg);
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```

**स्पष्टीकरण:**
- `SelectFolder` सक्रिय फ़ोल्डर सेट करता है.
- `AppendMessage` आपके ईमेल को चुने गए फ़ोल्डर में जोड़ देता है.
