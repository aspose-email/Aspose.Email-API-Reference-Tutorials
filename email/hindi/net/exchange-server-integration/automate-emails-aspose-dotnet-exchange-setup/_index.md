## कार्यान्वयन मार्गदर्शिका

### सुविधा 1: एक्सचेंज क्लाइंट प्रमाणीकरण और सेटअप

प्रमाणीकृत करें और इसका एक उदाहरण बनाएं `ExchangeClient` इस खंड में।

**अवलोकन:**
ईमेल एक्सेस के लिए एक्सचेंज सर्वर से प्रमाणीकरण करना ज़रूरी है। यहां बताया गया है कि आप अपने क्रेडेंशियल का इस्तेमाल करके क्लाइंट कैसे सेट कर सकते हैं।

#### चरण 1: बनाएँ `ExchangeClient` उदाहरण
```csharp
using Aspose.Email.Clients.Exchange;

// अपना सर्वर URL, उपयोगकर्ता नाम, पासवर्ड और डोमेन निर्धारित करें.
string url = "http://ex07sp1/एक्सचेंज/एडमिनिस्ट्रेटर";
string username = "user";
string password = "pwd";
string domain = "domain";

// ExchangeClient को क्रेडेंशियल के साथ आरंभ करें.
ExchangeClient client = new ExchangeClient(url, username, password, domain);
```

**स्पष्टीकरण:**
- **यूआरएल**: वह सर्वर URL जहाँ आपका Exchange सर्वर होस्ट किया गया है.
- **उपयोगकर्ता नाम/पासवर्ड/डोमेन**प्रमाणीकरण के लिए आवश्यक क्रेडेंशियल.

### फ़ीचर 2: इनबॉक्स से संदेशों को सूचीबद्ध करना

प्रमाणीकृत का उपयोग करें `ExchangeClient` इनबॉक्स में संदेशों को सूचीबद्ध करने के लिए.

**अवलोकन:**
ईमेल को प्रोग्रामेटिक रूप से सूचीबद्ध करने से समय की बचत होती है और दोहराए जाने वाले कार्य स्वचालित हो जाते हैं। ईमेल संदेशों को कुशलतापूर्वक प्राप्त करने का तरीका यहां बताया गया है।

#### चरण 2: ईमेल पुनः प्राप्त करें
```csharp
// मान लें कि 'क्लाइंट' पहले से ही बनाया गया है जैसा कि पहले दिखाया गया है।
ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);
```

**स्पष्टीकरण:**
- `ListMessages`: निर्दिष्ट मेलबॉक्स URI (इस मामले में, इनबॉक्स) से सभी संदेशों को पुनर्प्राप्त करता है।

### सुविधा 3: संदेश जानकारी प्रदर्शित करना

प्राप्त संदेशों को लूप करें और उनकी मूल जानकारी प्रदर्शित करें।

#### चरण 3: ईमेल विवरण प्रिंट करें
```csharp
using System;

// संग्रह में प्रत्येक संदेश को लूप में देखें।
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Message Size: " + msgInfo.Size);
    Console.WriteLine("==================================");
}
```

**स्पष्टीकरण:**
- **msgसूचना**: एक व्यक्तिगत ईमेल का प्रतिनिधित्व करता है, जैसे गुणों तक पहुंच प्रदान करता है `Subject`, `From`, और `Size`.
