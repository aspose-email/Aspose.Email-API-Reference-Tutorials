## कार्यान्वयन मार्गदर्शिका

सेटअप पूरा होने के बाद, आइए एक ईमेल वितरण सूची बनाएं और उसे सेव करें।

### चरण 1: MapiDistributionListMemberCollection ऑब्जेक्ट बनाएँ

अपनी वितरण सूची के सदस्यों को रखने के लिए एक संग्रह स्थापित करके शुरुआत करें, जो आपकी सूची की नींव का काम करता है।

#### कोड स्निपेट:
```csharp
using Aspose.Email.Mapi;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // वितरण सूची सहेजने का पथ

// MapiDistributionListMemberCollection ऑब्जेक्ट बनाएँ
MapiDistributionListMemberCollection oneOffMembers = new MapiDistributionListMemberCollection();
```

### चरण 2: संग्रह में सदस्य जोड़ें

इसके बाद, अपनी वितरण सूची में सदस्यों को शामिल करें। प्रत्येक सदस्य को एक सदस्य द्वारा दर्शाया जाता है। `MapiDistributionListMember` वस्तु।

#### कोड स्निपेट:
```csharp
// संग्रह में सदस्य जोड़ें
oneOffMembers.Add(new MapiDistributionListMember("John R. Patrick", "john@example.com"));
```

### चरण 3: गुण सेट करें और सहेजें

सभी आवश्यक सदस्यों को जोड़ने के बाद, अपनी सूची को सहेजने से पहले उसके लिए कोई भी अतिरिक्त गुण सेट करें।

#### कोड स्निपेट:
```csharp
// वितरण सूची बनाएं
MapiDistributionList distributionList = new MapiDistributionList("My Distribution List", oneOffMembers);

// फाइल को बचाएं
distributionList.Save(dataDir + "MyDistributionList.mlst");
```
