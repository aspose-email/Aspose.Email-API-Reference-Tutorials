## परिचय

ईमेल को प्रोग्रामेटिक रूप से प्रबंधित करना जटिल हो सकता है, खासकर बड़े पैमाने के एंटरप्राइज़ वातावरण में। यह ट्यूटोरियल आपको एक्सचेंज वेब सर्विसेज (EWS) क्लाइंट को आरंभ करने और संदेश हटाने, नए संदेश जोड़ने और ईमेल सूची में उपयोगकर्ताओं का प्रतिरूपण करने जैसे ऑपरेशन करने के लिए Aspose.Email लाइब्रेरी का उपयोग करने के बारे में मार्गदर्शन करता है। चाहे ईमेल प्रबंधन को स्वचालित करना हो या मौजूदा सिस्टम के साथ एकीकृत करना हो, यह गाइड एक व्यापक दृष्टिकोण प्रदान करता है।

**आप क्या सीखेंगे:**
- अपने प्रोजेक्ट में .NET के लिए Aspose.Email सेट अप करें
- विभिन्न उपयोगकर्ता क्रेडेंशियल का उपयोग करके EWSClient आरंभ करें
- विशिष्ट फ़ोल्डरों में संदेशों को हटाना और जोड़ना
- किसी अन्य उपयोगकर्ता के दृष्टिकोण से ईमेल सूचीबद्ध करने के लिए प्रतिरूपण लागू करें

यह सुनिश्चित करना कि आप पूर्वापेक्षाओं को पूरा करते हैं, आपको सेटअप प्रक्रिया में उतरने के लिए तैयार करेगा।

## कार्यान्वयन मार्गदर्शिका

### एक्सचेंज क्लाइंट आरंभीकरण

के उदाहरण बनाएँ `EWSClient` उपयोगकर्ता क्रेडेंशियल का उपयोग कर वर्ग:

**ग्राहकों को आरंभ करें:**

```csharp
using Aspose.Email.Clients.Exchange.WebService;

// दो अलग-अलग उपयोगकर्ताओं के लिए EWS क्लाइंट बनाना
IEWSClient client1 = EWSClient.GetEWSClient("https://outlook.office365.com/ews/exchange.asmx", "testUser1", "pwd", "डोमेन");
IEWSClient client2 = EWSClient.GetEWSClient("https://outlook.office365.com/ews/exchange.asmx", "testUser2", "pwd", "डोमेन");
```

### संदेश हटाना और जोड़ना

किसी विशिष्ट फ़ोल्डर से संदेश हटाएं और नए संदेश जोड़ें.

**संदेश हटाएं:**

```csharp
using Aspose.Email.Clients.Exchange;
using Aspose.Email.Mime;

string folder = "Drafts";

// क्लाइंट1 के लिए निर्दिष्ट फ़ोल्डर में सभी संदेशों को हटाना
foreach (ExchangeMessageInfo messageInfo in client1.ListMessages(folder))
{
    client1.DeleteItem(messageInfo.UniqueUri, DeletionOptions.DeletePermanently);
}
```

**संदेश जोड़ें:**

```csharp
string subj1 = string.Format("NETWORKNET_33354 {0} {1}", "User", "User1");
client1.AppendMessage(folder, new MailMessage("User1@exchange.conholdate.local", "To@aspsoe.com", subj1, ""));

// भिन्न विषय और प्राप्तकर्ताओं के साथ क्लाइंट 2 के लिए दोहराएँ
```

### प्रतिरूपण और संदेश सूचीकरण

संदेशों को सूचीबद्ध करने के लिए किसी उपयोगकर्ता का प्रतिरूपण करें.

**उपयोगकर्ता का प्रतिरूपण करना:**

```csharp
client1.ImpersonateUser(ItemChoice.PrimarySmtpAddress, "User2@exchange.conholdate.local");
ExchangeMessageInfoCollection messInfoColl1 = client1.ListMessages(folder);

// प्रतिरूपण रीसेट करें
client1.ResetImpersonation();
```
