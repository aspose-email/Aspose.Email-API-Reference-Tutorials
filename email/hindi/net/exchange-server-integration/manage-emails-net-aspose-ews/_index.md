## कार्यान्वयन मार्गदर्शिका

### EWS का उपयोग करके Exchange सर्वर से कनेक्ट करना
ईमेल को प्रोग्रामेटिक रूप से प्रबंधित करने के लिए सर्वर से कनेक्ट होना महत्वपूर्ण है।

#### चरण:
**चरण 1: EWS क्लाइंट का इंस्टेंस बनाएं**
```csharp
using System;
using Aspose.Email.Clients.Exchange.WebService;

public static void ConnectToExchangeServer()
{
    // सर्वर URL, उपयोगकर्ता नाम, पासवर्ड और डोमेन के साथ EWS क्लाइंट का एक इंस्टेंस बनाएं
    IEWSClient client = EWSClient.GetEWSClient(
        "https://आउटलुक.ऑफिस365.com/ews/एक्सचेंज.asmx", 
        "testUser", 
        "pwd", 
        "domain");
}
```
**स्पष्टीकरण:**
- `GetEWSClient` प्रदान किए गए क्रेडेंशियल्स का उपयोग करके कनेक्शन आरंभ करता है।

### नया ईमेल संदेश बनाना और जोड़ना
ईमेल संदेश तैयार करना और उसे अपने सर्वर से जोड़ना सीखें।

#### चरण:
**चरण 1: मेलमैसेज ऑब्जेक्ट बनाएँ**
```csharp
using System;
using Aspose.Email.Mime;
using Aspose.Email.Clients.Exchange.WebService;

public static void CreateAndAppendEmail(IEWSClient client)
{
    // एक नया MailMessage ऑब्जेक्ट बनाएँ
    MailMessage message = new MailMessage(
        "from@domain.com", 
        "to@domain.com", 
        "EMAILNET-34997 - " + Guid.NewGuid().ToString(), 
        "EMAILNET-34997 Exchange: Copy a message and get reference to the new Copy item");

    // बनाए गए ईमेल संदेश को सर्वर पर जोड़ें
    string messageUri = client.AppendMessage(message);
}
```
**स्पष्टीकरण:**
- `MailMessage` यह प्रेषक, प्राप्तकर्ता, विषय और मुख्य भाग सहित एक ईमेल को दर्शाता है।
- `AppendMessage` संदेश को सर्वर पर संग्रहीत करता है.

### ईमेल संदेश को दूसरे फ़ोल्डर में कॉपी करना
अपने ईमेल को उनके URI का उपयोग करके फ़ोल्डरों के बीच कॉपी करके कुशलतापूर्वक व्यवस्थित करें।

#### चरण:
**चरण 1: ईमेल कॉपी करने के लिए IEWSClient का उपयोग करें**
```csharp
using System;
using Aspose.Email.Clients.Exchange.WebService;

public static void CopyEmailMessage(IEWSClient client, string messageUri)
{
    // ईमेल संदेश को DeletedItems फ़ोल्डर में कॉपी करें
    string newMessageUri = client.CopyItem(
        messageUri, 
        client.MailboxInfo.DeletedItemsUri);
}
```
**स्पष्टीकरण:**
- `CopyItem` किसी संदेश को उसके वर्तमान स्थान से दूसरे फ़ोल्डर में ले जाता है.
