## कार्यान्वयन मार्गदर्शिका

हम कार्यान्वयन को दो प्राथमिक विशेषताओं में विभाजित करेंगे: एक्सचेंज सर्वर से कनेक्ट करना और वार्तालापों का प्रबंधन करना।

### एक्सचेंज सर्वर से कनेक्ट करें

**अवलोकन**
एक्सचेंज सर्वर से कनेक्ट करने से आप मेलबॉक्स आइटमों तक प्रोग्रामेटिक रूप से पहुंच और प्रबंधन कर सकते हैं।

#### चरण 1: क्रेडेंशियल सेट अप करें
```csharp
using System.Net;
using Aspose.Email.Clients.Exchange.WebService;

var mailboxUri = "https://exchange/ews/exchange.asmx"; // अपने Exchange सर्वर URL से बदलें
var domain = "\"; // यदि लागू न हो तो खाली छोड़ दें
var username = "username@ASE305.onmicrosoft.com"; // आपका ईमेल पता
var password = "password"; // आपका सुरक्षित पासवर्ड

NetworkCredential credentials = new NetworkCredential(username, password, domain);
```
**स्पष्टीकरण**: 
The `NetworkCredential` ऑब्जेक्ट में आपके लॉगिन विवरण होते हैं, जो प्रमाणीकरण के लिए आवश्यक हैं। सुनिश्चित करें कि URI आपके Exchange सर्वर के EWS एंडपॉइंट की ओर इंगित करता है।

#### चरण 2: कनेक्शन स्थापित करें
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials); // Exchange सर्वर से कनेक्ट करें
```
**स्पष्टीकरण**: 
कोड की यह पंक्ति प्रदान किए गए URI और क्रेडेंशियल का उपयोग करके Exchange सर्वर से कनेक्शन आरंभ करती है। यह एक रिटर्न देता है `IEWSClient` ऑब्जेक्ट जिसका उपयोग आप आगे के कार्यों के लिए कर सकते हैं.

### वार्तालाप ढूंढें और स्थानांतरित करें

**अवलोकन**
एक बार कनेक्ट होने के बाद, आप अपने इनबॉक्स में वार्तालाप ढूंढ सकते हैं और विशिष्ट मानदंडों के आधार पर उन्हें स्थानांतरित कर सकते हैं।

#### चरण 1: इनबॉक्स वार्तालाप पुनः प्राप्त करें
```csharp
var inboxUri = client.MailboxInfo.InboxUri;
ExchangeMessageInfoCollection messages = client.ListMessages(inboxUri);
```
**स्पष्टीकरण**: 
The `ListMessages` विधि आपके इनबॉक्स में सभी वार्तालापों को लाती है। इस संग्रह का उपयोग विशिष्ट वार्तालापों को फ़िल्टर करने और स्थानांतरित करने के लिए किया जाएगा।

#### चरण 2: किसी शर्त के आधार पर बातचीत को स्थानांतरित करें
```csharp
foreach (var messageInfo in messages)
{
    if (messageInfo.ConversationTopic.Contains("Specific Keyword"))
    {
        client.MoveItem(messageInfo.UniqueUri, "DestinationFolderId");
    }
}
```
**स्पष्टीकरण**: 
प्रत्येक वार्तालाप को दोहराएँ और अपनी शर्त लागू करें। यदि किसी वार्तालाप के विषय में निर्दिष्ट कीवर्ड शामिल है, तो उसे निर्दिष्ट फ़ोल्डर में ले जाएँ।
