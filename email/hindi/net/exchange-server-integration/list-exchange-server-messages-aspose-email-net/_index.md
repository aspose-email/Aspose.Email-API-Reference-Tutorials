## कार्यान्वयन मार्गदर्शिका

अब जबकि हमारा सेटअप पूरा हो गया है, आइए एक्सचेंज सर्वर से संदेशों को सूचीबद्ध करने की सुविधा को क्रियान्वित करने के लिए आगे बढ़ें।

### Exchange सर्वर से कनेक्ट करें

ईमेल सूचीबद्ध करने के लिए, सबसे पहले Aspose.Email का उपयोग करके अपने Exchange सर्वर से कनेक्ट करें। इस चरण के लिए आपको सर्वर URI और अपने क्रेडेंशियल की आवश्यकता होगी।

**चरण 1: कनेक्शन स्थापित करें**

इसका एक नया उदाहरण बनाएं `ExchangeClient`:

```csharp
string exchangeUri = "https://MachineName/exchange/Username"; // आपका Exchange सर्वर URI
string username = "username"; // आपका एक्सचेंज सर्वर उपयोगकर्ता नाम
string password = "password"; // आपका एक्सचेंज सर्वर पासवर्ड

try
{
    var domain = new Domain(); // यदि आवश्यक हो तो डोमेन वर्ग के लिए प्लेसहोल्डर
    ExchangeClient client = new ExchangeClient(exchangeUri, username, password, domain);

    // संदेशों की सूची पर आगे बढ़ें
}
catch (Exception ex)
{
    Console.Write("Error connecting: " + ex.Message);
}
```

यहाँ, `ExchangeClient` सर्वर URI और क्रेडेंशियल्स को पैरामीटर के रूप में लेता है, जिससे सुरक्षित कनेक्शन की सुविधा मिलती है।

### इनबॉक्स से संदेश सूचीबद्ध करें

एक स्थापित कनेक्शन के साथ, अब हम ईमेल प्राप्त कर सकते हैं:

**चरण 2: संदेश पुनः प्राप्त करें**

अपने इनबॉक्स से संदेश लाने के लिए क्लाइंट का उपयोग करें:

```csharp
try
{
    ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);

    // संदेश जानकारी प्रदर्शित करें
}
catch (Exception ex)
{
    Console.Write("Error retrieving messages: " + ex.Message);
}
```

`ListMessages` निर्दिष्ट मेलबॉक्स URI से सभी संदेशों को प्राप्त करता है, तथा उन्हें एक संग्रह के रूप में लौटाता है।

### संदेश जानकारी प्रदर्शित करें

संदेशों को पुनः प्राप्त करने के बाद, आप आवश्यक विवरण प्रदर्शित करने के लिए उन्हें पुनरावृत्त कर सकते हैं:

**चरण 3: पुनरावृति और प्रदर्शन**

```csharp
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Sent Date: " + msgInfo.Date.ToString());
    Console.WriteLine("Read?: " + msgInfo.IsRead.ToString());
    Console.WriteLine("Message ID: " + msgInfo.MessageId);
    Console.WriteLine("Unique URI: " + msgInfo.UniqueUri);
}
```

यह लूप प्रत्येक संदेश को दोहराता है, तथा विषय, प्रेषक, प्राप्तकर्ता, तथा पठन स्थिति जैसी प्रमुख विशेषताओं को प्रिंट करता है।
