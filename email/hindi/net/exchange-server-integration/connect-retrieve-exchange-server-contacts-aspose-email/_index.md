## कार्यान्वयन मार्गदर्शिका
यह अनुभाग एक्सचेंज सर्वर से कनेक्ट करने और संपर्कों को पुनः प्राप्त करने के बारे में बताता है।

### एक्सचेंज सर्वर से कनेक्ट करना
अपने Exchange सर्वर से सुरक्षित रूप से कनेक्ट होना ज़रूरी है। यहाँ बताया गया है कि कैसे:

#### चरण 1: क्रेडेंशियल परिभाषित करें
अपने WebDAV क्रेडेंशियल्स को सेट अप करें `NetworkCredential` सुरक्षित प्रमाणीकरण के लिए.

```csharp
using System.Net;

string mailboxURI = "http://ex2003/exchange/administrator"; // एक्सचेंज सर्वर URI
string username = "administrator";
string password = "pwd";
string domain = "domain.local";

// सर्वर से कनेक्ट करने के लिए नेटवर्क क्रेडेंशियल बनाएं
NetworkCredential credential = new NetworkCredential(username, password, domain);
```

#### चरण 2: आरंभ करें `ExchangeClient`
उपयोग `ExchangeClient` Aspose.Email से कनेक्शन स्थापित करने के लिए।

```csharp
using Aspose.Email.Clients.Exchange.Dav;

// URI और क्रेडेंशियल के साथ ExchangeClient का एक इंस्टेंस बनाएं
tExchangeClient client = new ExchangeClient(mailboxURI, credential);
```

### एक्सचेंज सर्वर से संपर्क पुनर्प्राप्त करना
एक बार कनेक्ट हो जाने पर, अपने सर्वर पर संग्रहीत संपर्क प्राप्त करें।

#### चरण 1: संपर्क प्राप्त करें
उपयोग `GetContacts` सभी संपर्कों को पुनः प्राप्त करने की विधि.

```csharp
using Aspose.Email.Mapi;

// निर्देशिका से संपर्क पुनः प्राप्त करें
Contact[] contacts = client.GetContacts(client.MailboxInfo.ContactsUri);
```

#### चरण 2: संपर्क जानकारी प्रदर्शित करें
प्रत्येक संपर्क पर नजर रखें और नाम तथा ईमेल पते जैसे प्रासंगिक विवरण प्रदर्शित करें।

```csharp
foreach (MapiContact contact in contacts)
{
    Console.WriteLine("Name: " + contact.NameInfo.DisplayName + ", Email Address: " + contact.ElectronicAddresses.Email1);
}
```
