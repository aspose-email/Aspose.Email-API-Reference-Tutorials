## कार्यान्वयन मार्गदर्शिका

हम कार्यान्वयन को दो मुख्य विशेषताओं में विभाजित करेंगे: OAuth प्रमाणीकरण और IGmailClient का उपयोग करके संपर्क बनाना और प्रबंधित करना।

### सुविधा 1: OAuth प्रमाणीकरण

Gmail संपर्कों तक सुरक्षित रूप से पहुँचने के लिए OAuth प्रमाणीकरण बहुत ज़रूरी है। इसे आप इस तरह सेट कर सकते हैं:

#### अवलोकन
यह सुविधा Aspose.Email के माध्यम से Gmail के साथ बातचीत करने के लिए आवश्यक एक्सेस टोकन और रिफ्रेश टोकन प्राप्त करना प्रदर्शित करती है।

**चरण-दर-चरण कार्यान्वयन**

1. **उपयोगकर्ता क्रेडेंशियल परिभाषित करें**
   ```csharp
   GoogleTestUser user = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
   ```

2. **एक्सेस और रिफ्रेश टोकन प्राप्त करें**
   ```csharp
   string accessToken;
   string refreshToken;
   GoogleOAuthHelper.GetAccessToken(user, out accessToken, out refreshToken);
   ```

यह कदम टोकन के लिए क्लाइंट क्रेडेंशियल्स का आदान-प्रदान करके सुरक्षित पहुंच सुनिश्चित करता है।

### फ़ीचर 2: जीमेल संपर्क बनाना

Gmail में विस्तृत संपर्क विवरण बनाना Aspose.Email के साथ स्वचालित किया जा सकता है।

#### अवलोकन
नया Gmail संपर्क बनाते समय पते, फ़ोन नंबर और ईवेंट जैसे विभिन्न फ़ील्ड भरने का तरीका जानें.

**चरण-दर-चरण कार्यान्वयन**

1. **नया संपर्क आरंभ करें**
   ```csharp
   Contact contact = new Contact();
   ```

2. **मूलभूत जानकारी भरें**
   ```csharp
   contact.GivenName = "GivenName";
   contact.Surname = "Surname";
   ```

3. **पते और फ़ोन नंबर जोड़ें**
   ```csharp
   PostalAddress address = new PostalAddress {
       Address = "Address",
       City = "City"
   };
   contact.PhysicalAddresses.Add(address);

   PhoneNumber pnWork = new PhoneNumber { Number = "1234567890", Category = PhoneNumberCategory.Work };
   contact.PhoneNumbers.Add(pnWork);
   ```

4. **अतिरिक्त विवरण जोड़ें**
   ```csharp
   contact.Events.Birthday = DateTime.Now.AddYears(-30);
   contact.EmailAddresses.Add(new EmailAddress { Address = "email@gmail.com" });
   ```

### संपर्क बनाने के लिए IGmailClient का उपयोग करना

#### अवलोकन
Gmail में प्रोग्रामेटिक रूप से संपर्क बनाने के लिए IGmailClient इंटरफ़ेस का उपयोग करना सीखें।

**चरण-दर-चरण कार्यान्वयन**

1. **IGmailClient आरंभ करें**
   ```csharp
   IGmailClient client = GmailClient.GetInstance(accessToken, user.EMail);
   ```

2. **संपर्क बनाएं**
   ```csharp
   string contactUri = client.CreateContact(contact);
   ```

यह प्रक्रिया स्वचालित और बड़ी संख्या में संपर्कों के निर्माण की अनुमति देती है, जिससे कार्यकुशलता बढ़ती है।
