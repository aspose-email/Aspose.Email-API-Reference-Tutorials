## कार्यान्वयन मार्गदर्शिका

हम ट्यूटोरियल को विशिष्ट विशेषताओं के आधार पर प्रबंधनीय खंडों में विभाजित करेंगे।

### एक नई PST फ़ाइल बनाएँ

Aspose.Email के साथ एक नई PST फ़ाइल बनाना बहुत आसान है। यह सुविधा आपको ईमेल और संबंधित डेटा के लिए नया स्टोरेज शुरू करने की अनुमति देती है।

#### चरण 1: निर्देशिका पथ सेट करें
निर्दिष्ट करें कि आपकी नई PST फ़ाइल कहाँ संग्रहीत की जाएगी:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/outlook/";
```

#### चरण 2: PST फ़ाइल बनाएँ

उपयोग `PersonalStorage.create()` एक नई PST फ़ाइल आरंभ करने की विधि। पहला पैरामीटर पथ है, और दूसरा प्रारूप संस्करण (यूनिकोड के लिए 0) निर्दिष्ट करता है।

```java
import com.aspose.email.PersonalStorage;

// PersonalStorage का एक नया उदाहरण बनाएँ
PersonalStorage pst = PersonalStorage.create(dataDir + "newSample_out.pst", 0);
```

### PST के रूट में एक फ़ोल्डर जोड़ें

फ़ोल्डर जोड़ने से आपके ईमेल को PST फ़ाइल में व्यवस्थित करने में मदद मिलती है। यह अनुभाग दर्शाता है कि रूट स्तर पर फ़ोल्डर कैसे जोड़ा जाता है।

#### चरण 1: PST फ़ाइल लोड करें
मान लें कि आपके पास एक मौजूदा या नई बनाई गई PST फ़ाइल है:

```java
PersonalStorage pst = PersonalStorage.fromFile(dataDir + "newSample_out.pst");
```

#### चरण 2: नया फ़ोल्डर जोड़ें
नाम का एक फ़ोल्डर बनाएं और जोड़ें `myInbox` पीएसटी के मूल स्तर पर।

```java
pst.getRootFolder().addSubFolder("myInbox");
```

### PST में किसी विशिष्ट फ़ोल्डर में संदेश जोड़ें

ईमेल संदेशों का प्रबंधन करना महत्वपूर्ण है। यहां बताया गया है कि आप अपने PST में मौजूदा .msg फ़ाइल कैसे जोड़ सकते हैं:

#### चरण 1: PST और MSG फ़ाइल लोड करें
सुनिश्चित करना `newSample_out.pst` फ़ोल्डर संरचना तैयार के साथ मौजूद है.

```java
PersonalStorage pst = PersonalStorage.fromFile(dataDir + "newSample_out.pst");
MapiMessage message = MapiMessage.fromFile(dataDir + "message.msg");
```

#### चरण 2: संदेश को फ़ोल्डर में जोड़ें
अपना ईमेल संदेश निर्दिष्ट फ़ोल्डर में डालें।

```java
pst.getRootFolder().getSubFolder("myInbox").addMessage(message);
```
