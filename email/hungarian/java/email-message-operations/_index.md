## Hogyan konvertáljunk e‑mail formátumokat Java-val?
`MailMessage.load` betölti az e‑mail fájlt (EML, MSG stb.) egy MailMessage objektumba. `MailMessage.save` az objektumot egy új formátumba írja. A konvertáláshoz hívja meg a `MailMessage.load`-ot a forrás útvonallal, majd a `save`-ot a kívánt kimeneti formátummal és célhellyel. Az API automatikusan kezeli a kódolást, mellékleteket és metaadatokat.

## Milyen fájlformátumokat támogat az Aspose.Email for Java?
Az Aspose.Email for Java több mint 30 e‑mail és archívum formátumot képes olvasni és írni, többek között EML, MSG, MHTML, OFT, PST, TNEF, EMLX és EMLZ. A könyvtár formátum‑specifikus `MailMessage.save` túlterheléseket biztosít, amelyek egyetlen metódushívással zökkenőmentes konvertálást tesznek lehetővé bármely támogatott típus között, miközben megőrzik az üzenet pontosságát.
