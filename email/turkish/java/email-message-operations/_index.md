## Java kullanarak e‑posta formatlarını nasıl dönüştürürüm?
`MailMessage.load` bir e‑posta dosyasını (EML, MSG vb.) `MailMessage` nesnesine yükler. `MailMessage.save` nesneyi yeni bir formata yazar. Dönüştürmek için, kaynak yolu ile `MailMessage.load` çağırın, ardından hedef format ve konumu belirterek `save` metodunu çalıştırın. API kodlama, ekler ve meta verileri otomatik olarak yönetir.

Kütüphane, `MailMessage.save` için format‑özel aşırı yüklemeler sunar; böylece tek bir metod çağrısıyla desteklenen herhangi iki tip arasında sorunsuz dönüşüm yapılırken mesaj bütünlüğü korunur.
