## Uygulama Kılavuzu

### Aspose.Email for Java Kullanarak OFT'yi MSG'ye Nasıl Dönüştürülür?

Bu bölüm, bir Outlook Şablonunu tam yapılandırılmış bir Outlook Mesajına dönüştürme sürecini uçtan uca açıklar. Önce OFT dosyasını yüklersiniz, ardından gönderen, alıcı ve gövde içeriği gibi alanları kişiselleştirirsiniz ve sonunda sonucu MSG dosyası olarak kaydedersiniz. Yaklaşım hafiftir, sadece birkaç satır kod gerektirir ve yüksek hacimli işleme için toplu işler veya web hizmetlerine entegre edilebilir.

#### Outlook Şablon Dosyasını Yükle ve Güncelle

##### Genel Bakış

Bir OFT (Outlook Şablonu) dosyasının içeriğini nasıl manipüle edeceğinizi ve tam yapılandırılmış bir MSG e‑posta mesajına nasıl dönüştüreceğinizi öğrenin.

##### Uygulama Adımları

**1. Outlook Şablonunu Yükle**  
`MailMessage` Aspose.Email'in bellekte bir e‑posta mesajını temsil eden temel sınıfıdır. Konu, gövde, alıcılar ve ekler için özellikler sağlar.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MailMessage message = MailMessage.load(dataDir + "sample.oft");
```

**2. Gönderici ve Alıcı Detaylarını Ayarla**  
`MailMessage` `from`, `to`, `cc` ve `bcc` alanlarını doğrudan ayarlamanıza olanak tanır, böylece son MSG doğru yönlendirme bilgilerini yansıtır.

```java
message.setSender(new MailAddress("john@abc.com", "John"));
message.getTo().addMailAddress(new MailAddress("william@xzy.com", "William"));
```

**3. HTML Gövde İçeriğini Güncelle**  
`mailMessage.setHtmlBody()` metoduna bir HTML dizesi atayarak şablonu isimler, tarihler veya toplantı linkleri gibi dinamik verilerle kişiselleştirebilirsiniz.

```java
String htmlBody = message.getHtmlBody();
htmlBody = htmlBody.replace("DisplayName", "<b>William</b>");
htmlBody = htmlBody.replace("MeetingPlace", "<u>Hall 1, Convention Center, New York, USA</u>");
htmlBody = htmlBody.replace("MeetingTime", "<u>Monday, June 28, 2010</u>");
message.setHtmlBody(htmlBody);
```

**4. MSG Dosyası Olarak Kaydet**  
`mailMessage.save("output.msg", SaveOptions.getDefaultMsg())` çağrısı, tam hazırlanmış mesajı MSG formatında diske yazar ve **convert oft to msg** işlemini tamamlar.

```java
MapiMessage mapimessage = MapiMessage.fromMailMessage(message);
mapimessage.setMessageFlags(MapiMessageFlags.MSGFLAG_UNSENT);
mapimessage.save(dataDir + "Invitation.msg");
```

### Aspose.Email ile Yeni Outlook Şablonu (OFT) Nasıl Oluşturulur?

Sıfırdan yeni bir Outlook Şablonu oluşturmak, kampanyalar veya bildirimler arasında yeniden kullanılabilecek standart bir düzen tanımlamanızı sağlar. Önce bir `MapiMessage` oluşturur, özelliklerini (konu, gövde, ekler) yapılandırır ve ardından bir OFT dosyası olarak kalıcı hale getirirsiniz. Bu şablon daha sonra gerektiğinde yüklenebilir, özelleştirilebilir ve MSG'ye dönüştürülebilir.

**1. Yeni Bir E‑posta Mesajı Oluştur**  
`MapiMessage` Aspose.Email'in Outlook mesajının düşük seviyeli temsilidir ve OFT dosyaları için gerekli MAPI özellikleri üzerinde tam kontrol sağlar.

```java
MapiMessage mapi = new MapiMessage("test@from.to", "test@to.to", "template subject", "Template body");
```

**2. Şablon Dosyası Olarak Kaydet**  
`MapiMessage` örneğini gelecekte yeniden kullanım için bir OFT dosyası olarak kalıcı hale getirin.

```java
try {
    mapi.saveAsTemplate(dataDir + "mapiToOft.oft");
} finally {
    if (mapi != null) ((IDisposable)mapi).dispose();
}
```
