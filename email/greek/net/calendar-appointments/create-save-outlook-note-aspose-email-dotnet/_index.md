## Οδηγός Εφαρμογής

Ας εμβαθύνουμε στη δημιουργία και αποθήκευση μιας σημείωσης του Outlook χρησιμοποιώντας το Aspose.Email για .NET.

### Δημιουργία σημείωσης MAPI
Αρχικά, θα δημιουργήσετε μια παρουσία του `MapiNote`Αυτό το αντικείμενο χρησιμεύει ως βάση για τη σημείωσή σας:

```csharp
// Δημιουργήστε μια παρουσία του MapiNote
MapiNote note3 = new MapiNote();
```

#### Ρύθμιση ιδιοτήτων
Τώρα, ας ορίσουμε διάφορες ιδιότητες όπως θέμα, σώμα, χρώμα και διαστάσεις.

**Θέμα**: Ο τίτλος ή η επικεφαλίδα της σημείωσης.
```csharp
note3.Subject = "Blue Color Note"; // Ορίστε το θέμα
```

**Σώμα**: Κύριο κείμενο περιεχομένου της σημείωσης.
```csharp
note3.Body = "This is a blue color note"; // Ορισμός κειμένου σώματος
```

**Χρώμα**: Οπτική προσαρμογή για εύκολη αναγνώριση.
```csharp
note3.Color = NoteColor.Blue; // Ορισμός χρώματος σε Μπλε
```

**Διαστάσεις**: Ορίστε το μέγεθος σε pixel.
```csharp
note3.Height = 500; // Ορισμός ύψους
note3.Width = 500; // Ορισμός πλάτους
```

### Αποθήκευση της σημείωσης
Τέλος, αποθηκεύστε τη σημείωσή σας ως `.msg` αρχείο για εύκολη πρόσβαση και κοινή χρήση:

```csharp
// Αποθήκευση της σημείωσης σε έναν καθορισμένο κατάλογο εξόδου
note3.Save(outputDir + "MapiNote_out.msg");
```
