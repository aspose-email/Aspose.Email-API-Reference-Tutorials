### Δημιουργία εργασίας με ετήσια επανάληψη

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να δημιουργήσετε ένα MapiTask που επαναλαμβάνεται ετησίως, χρήσιμο για τον προγραμματισμό επαναλαμβανόμενων συμβάντων ή υπενθυμίσεων στην εφαρμογή σας.

#### Βήματα Υλοποίησης
##### 1. Ορίστε τις ημερομηνίες έναρξης και λήξης
Ορίστε την ημερομηνία έναρξης της εργασίας λαμβάνοντας υπόψη τις τοπικές μετατοπίσεις ζώνης ώρας:
```csharp
DateTime startDate = new DateTime(2023, 7, 1);
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan timeSpan = localZone.GetUtcOffset(DateTime.Now);
startDate = startDate.Add(timeSpan);

DateTime dueDate = startDate; // Αρχικά ορίστηκε για την ίδια ημέρα.
```
##### 2. Ορίστε το μοτίβο επανάληψης
Διαμορφώστε ένα ετήσιο μοτίβο επανάληψης χρησιμοποιώντας `MapiCalendarMonthlyRecurrencePattern`:
```csharp
DateTime endByDate = new DateTime(2030, 12, 31).Add(timeSpan);
var rec = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15,
    Period = 1,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterDate,
    EndDate = endByDate,
    OccurrenceCount = GetOccurrenceCount(startDate, endByDate, "FREQ=YEARLY;BYMONTHDAY=15;INTERVAL=1")
};
```
##### 3. Δημιουργία και διαμόρφωση της εργασίας
Αρχικοποίηση ενός `MapiTask` με συγκεκριμένες λεπτομέρειες:
```csharp
MapiTask task = new MapiTask("This is test task", "Sample Body", startDate, dueDate)
{
    State = MapiTaskState.NotAssigned
};
task.Recurrence = rec;
```
##### 4. Υπολογισμός εμφανίσεων
Χρήση `GetOccurrenceCount` για τον προσδιορισμό των επαναλαμβανόμενων περιστατικών:
```csharp
private static uint GetOccurrenceCount(DateTime start, DateTime endBy, string rrule)
{
    CalendarRecurrence pattern = new CalendarRecurrence(string.Format("DTSTART:{0}\r\nRRULE:{1}", 
        start.ToString("yyyyMMdd"), rrule));
    DateCollection dates = pattern.GenerateOccurrences(start, endBy);
    return (uint)dates.Count;
}
```
