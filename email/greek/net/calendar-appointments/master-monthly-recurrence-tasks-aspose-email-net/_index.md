## Οδηγός Εφαρμογής

### Ρύθμιση μηνιαίας εργασίας επανάληψης με λήξη μετά από N εμφανίσεις

Μάθετε πώς να δημιουργείτε εργασίες που επαναλαμβάνονται μηνιαίως και σταματούν μετά από έναν συγκεκριμένο αριθμό επαναλήψεων.

#### Επισκόπηση:
Θα χρησιμοποιήσουμε `MapiTask` από το Aspose.Email.Mapi, ρυθμίστε το για μηνιαία επανάληψη και ορίστε μια συνθήκη λήξης.

##### Βήμα 1: Ορισμός ημερομηνιών εργασιών
Ορίστε την ημερομηνία έναρξης, την ημερομηνία λήξης και την ημερομηνία λήξης χρησιμοποιώντας την τοπική ζώνη ώρας σας, ώστε να ευθυγραμμιστείτε με τις προσδοκίες των χρηστών.

```csharp
using System;
using Aspose.Email.Mapi;
using Aspose.Email.Calendar.Recurrences;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);
DateTime StartDate = new DateTime(2015, 7, 16).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 16).Add(ts);
DateTime endByDate = new DateTime(2015, 12, 31).Add(ts);
```

##### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων της εργασίας
Αρχικοποίηση ενός `MapiTask` παράδειγμα με την περιγραφή της εργασίας σας και τις ημερομηνίες.

```csharp
// Δημιουργήστε ένα MapiTask με ημερομηνίες έναρξης και λήξης.
MapiTask task = new MapiTask("This is a test task", "Test Description", StartDate, DueDate);
```

##### Βήμα 3: Ορισμός μηνιαίου μοτίβου επανάληψης
Ρυθμίστε τις παραμέτρους του μοτίβου επανάληψης ώστε να επαναλαμβάνεται μηνιαίως και καθορίστε τον αριθμό των εμφανίσεων.

```csharp
// Δημιουργήστε έναν μηνιαίο κανόνα επανάληψης που λήγει μετά από 10 εμφανίσεις.
MapiCalendarEventRecurrence recurrence = new MapiCalendarEventRecurrence();
recurrence.Pattern = new MonthlyPattern(1); // Επαναλαμβάνεται κάθε μήνα
recurrence.EndType = MapiCalendarEventRecurrenceEndType.NoEndDate;
recurrence.Range = new OccurrenceRange(StartDate, endByDate, 10);

// Αντιστοιχίστε τον κανόνα επανάληψης στην εργασία.
task.Recurrence = recurrence;
```

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι όλοι οι υπολογισμοί ημερομηνίας και ώρας λαμβάνουν υπόψη τις διαφορές στις τοπικές ζώνες ώρας.
- Επαληθεύστε την εγκατάσταση του Aspose.Email ελέγχοντας την έκδοση του πακέτου στο έργο σας.
