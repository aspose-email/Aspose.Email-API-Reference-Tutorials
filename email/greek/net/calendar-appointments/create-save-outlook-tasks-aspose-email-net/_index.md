## Οδηγός Εφαρμογής
Ας δούμε βήμα προς βήμα τη διαδικασία δημιουργίας και αποθήκευσης μιας εργασίας του Outlook.

### Δημιουργία αντικειμένου MapiTask
ΕΝΑ `MapiTask` Το αντικείμενο αντιπροσωπεύει μια εργασία του Outlook. Ξεκινήστε αρχικοποιώντας την με τις βασικές ιδιότητες:

#### Βήμα 1: Ορισμός της εργασίας
```csharp
MapiTask task = new MapiTask(
    "Να κάνω", 
    "Just click and type to add new task", 
    DateTime.Now, 
    DateTime.Now.AddDays(3)
);
```
- **"To Do"** είναι το θέμα.
- Η περιγραφή παρέχει πρόσθετες πληροφορίες.
- Η ημερομηνία έναρξης και η ημερομηνία λήξης έχουν οριστεί στη σημερινή ημερομηνία και σε τρεις ημέρες από τώρα.

#### Βήμα 2: Ορισμός προόδου και προσπάθειας
```csharp
task.PercentComplete = 20;
task.EstimatedEffort = 2000; // Σε λίγα λεπτά
```
- Ορίστε το ποσοστό ολοκλήρωσης της εργασίας.
- Ορίστε την εκτιμώμενη προσπάθεια σε λεπτά για να παρακολουθείτε τον χρόνο που αφιερώνεται.

#### Βήμα 3: Ιστορικό εργασιών και ενημερώσεις
```csharp
task.History = MapiTaskHistory.Assigned;
task.LastUpdate = DateTime.Now;
```
- Καταγράψτε πότε ανατέθηκε ή ενημερώθηκε τελευταία φορά η εργασία για καλύτερη παρακολούθηση.

### Διαμόρφωση Ιδιοκτησίας και Εταιρειών
Ανάθεση στοιχείων ιδιοκτησίας και συνδεδεμένων εταιρειών:

```csharp
task.Users.Owner = "Darius";
task.Users.LastAssigner = "Harkness";
task.Users.LastDelegate = "Harkness";
task.Users.Ownership = MapiTaskOwnership.AssignersCopy;

task.Companies = new string[] { "company1", "company2", "company3" };
```

### Κατηγοριοποίηση και προσθήκη μεταδεδομένων
Χρησιμοποιήστε κατηγορίες για την οργάνωση:

```csharp
task.Categories = new string[] { "category1", "category2", "category3" };
task.Mileage = "Some test mileage";
task.Billing = "Test billing information";
task.Users.Delegator = "Test Delegator";
```

### Οριστικοποίηση ιδιοτήτων εργασίας
Ορισμός ευαισθησίας και κατάστασης:

```csharp
task.Sensitivity = MapiSensitivity.Personal;
task.Status = MapiTaskStatus.Complete;

// Προσαρμόστε την εκτιμώμενη προσπάθεια εάν χρειάζεται
task.EstimatedEffort = 5; // Σε λίγα λεπτά
```

### Αποθήκευση της εργασίας ως αρχείο MSG
Τέλος, αποθηκεύστε την εργασία σας:

```csharp
string outputPath = "@YOUR_OUTPUT_DIRECTORY/MapiTask.msg";
task.Save(outputPath, TaskSaveFormat.Msg);
```

Βεβαιωθείτε ότι θα αντικαταστήσετε `@YOUR_OUTPUT_DIRECTORY` με την πραγματική διαδρομή καταλόγου όπου θέλετε να αποθηκεύσετε το αρχείο.
