## Οδηγός Εφαρμογής

### Επισκόπηση λειτουργιών: Ανάκτηση εκτεταμένων χαρακτηριστικών για στοιχεία ημερολογίου
Αυτή η λειτουργία σάς επιτρέπει να ανακτάτε προσαρμοσμένες ιδιότητες από στοιχεία ημερολογίου που είναι αποθηκευμένα σε έναν διακομιστή Exchange, παρέχοντας βελτιωμένες δυνατότητες διαχείρισης και ανάκτησης δεδομένων.

#### Δημιουργία σύνδεσης με το EWS
**Βήμα 1:** Δημιουργήστε μια σύνδεση με το πρόγραμμα-πελάτη EWS χρησιμοποιώντας τα διαπιστευτήριά σας. Αυτό το βήμα είναι κρίσιμο, καθώς επιτρέπει την πρόσβαση στα δεδομένα του γραμματοκιβωτίου του Exchange.
```csharp
IEWSClient client = EWSClient.GetEWSClient("https://exchange.office365.com/Exchange.asmx", "όνομα χρήστη", "κωδικός πρόσβασης");
```

#### Ανάκτηση στοιχείων ημερολογίου
**Βήμα 2:** Ανάκτηση όλων των στοιχείων ημερολογίου από τον διακομιστή. Αυτό σας δίνει μια λίστα με URI που αντιπροσωπεύουν κάθε στοιχείο.
```csharp
string[] uriList = client.ListItems(client.MailboxInfo.CalendarUri);
```

#### Ορισμός Περιγραφέων Ιδιοτήτων
**Βήμα 3:** Καθορίστε ποια εκτεταμένα χαρακτηριστικά θα αναζητηθούν δημιουργώντας ένα `PidNamePropertyDescriptor`Αυτή η περιγραφή ορίζει το όνομα, τον τύπο δεδομένων και το σχετικό GUID της προσαρμοσμένης ιδιότητας.
```csharp
PropertyDescriptor propertyDescriptor = new PidNamePropertyDescriptor(
    "K1", // Όνομα της προσαρμοσμένης ιδιότητας
    PropertyDataType.Integer32, // Τύπος δεδομένων
    new Guid("00020329-0000-0000-C000-000000000046") // GUID για το εκτεταμένο σύνολο χαρακτηριστικών
);
```

#### Ανάκτηση και εμφάνιση χαρακτηριστικών
**Βήμα 4:** Χρησιμοποιήστε την περιγραφή για να ανακτήσετε στοιχεία ημερολογίου με την καθορισμένη προσαρμοσμένη ιδιότητα. Επαναλάβετε κάθε στοιχείο και εκτυπώστε τις ιδιότητές του.
```csharp
IList<MapiCalendar> mapiCalendarList = client.FetchMapiCalendar(uriList, new PropertyDescriptor[] { propertyDescriptor });

foreach (MapiCalendar cal in mapiCalendarList)
{
    foreach (MapiNamedProperty namedProperty in cal.NamedProperties.Values)
    {
        Console.WriteLine(namedProperty.NameId + " = " + namedProperty.GetInt32());
    }
}
```
