## Οδηγός Εφαρμογής

Αυτός ο οδηγός χωρίζεται σε λογικές ενότητες ανά χαρακτηριστικό. Κάθε ενότητα παρέχει μια επισκόπηση και λεπτομερή βήματα με αποσπάσματα κώδικα.

### Σύνδεση με την υπηρεσία Exchange Web Service (EWS)

**Επισκόπηση:** Η δημιουργία σύνδεσης με τον EWS επιτρέπει την πρόσβαση στα δεδομένα του γραμματοκιβωτίου και του ημερολογίου σας, προετοιμάζοντας το έδαφος για εργασίες διαχείρισης συναντήσεων.

1. **Αρχικοποίηση του IEWSClient:**
   Δημιουργήστε μια παρουσία του `IEWSClient` χρησιμοποιώντας διαπιστευτήρια που παρέχουν πρόσβαση στο τελικό σημείο του EWS σας.
   
   ```csharp
   // Δημιουργήστε και ρυθμίστε μια παρουσία IEWSClient με διαπιστευτήρια.
   using Aspose.Email.Clients.Exchange;
   using Aspose.Email.Clients.Exchange.WebService;

   IEWSClient client = EWSClient.GetEWSClient(
       "https://outlook.office365.com/ews/exchange.asmx",
       "username",
       "password",
       "domain"
   );
   ```

### Φιλτράρισμα ραντεβού κατά εύρος ημερομηνιών χρησιμοποιώντας το EWS

**Επισκόπηση:** Το φιλτράρισμα των ραντεβού ανά εύρος ημερομηνιών σάς βοηθά να εστιάσετε σε συγκεκριμένες περιόδους, βελτιώνοντας τη διαχείριση και την ανάλυση δεδομένων.

1. **Ορίστε ημερομηνίες έναρξης και λήξης:**
   Καθορίστε το εύρος ημερομηνιών για το φιλτράρισμα.
   
   ```csharp
   using System;

   DateTime startTime = new DateTime(2017, 9, 15);
   DateTime endTime = new DateTime(2017, 10, 10);
   ```

2. **Δημιουργήστε ένα ερώτημα για να φιλτράρετε τα ραντεβού:**
   Χρήση `ExchangeQueryBuilder` για να δημιουργήσετε το ερώτημά σας με βάση το καθορισμένο εύρος ημερομηνιών.
   
   ```csharp
   using Aspose.Email.Tools.Search;

   ExchangeQueryBuilder builder = new ExchangeQueryBuilder();
   builder.Appointment.Start.Since(startTime);
   builder.Appointment.End.BeforeOrEqual(endTime);
   MailQuery query = builder.GetQuery();
   ```

3. **Ανάκτηση φιλτραρισμένων ραντεβού:**
   Εκτελέστε το ερώτημα για να λάβετε ραντεβού εντός του καθορισμένου εύρους ημερομηνιών.
   
   ```csharp
   Appointment[] appointmentsByDate = client.ListAppointments(query);
   ```

### Φιλτράρισμα ραντεβού κατά επανάληψη χρησιμοποιώντας το EWS

**Επισκόπηση:** Ο εντοπισμός μη επαναλαμβανόμενων ραντεβού μπορεί να είναι απαραίτητος για εργασίες που απαιτούν εφάπαξ προγραμματισμό.

1. **Δημιουργήστε ένα ερώτημα για τον εντοπισμό μη επαναλαμβανόμενων ραντεβού:**
   Χρήση `ExchangeQueryBuilder` για να φιλτράρετε τα επαναλαμβανόμενα ραντεβού.
   
   ```csharp
   ExchangeQueryBuilder builderRecurrence = new ExchangeQueryBuilder();
   builderRecurrence.Appointment.IsRecurring.Equals(false);
   MailQuery queryNonRecurring = builderRecurrence.GetQuery();
   ```

2. **Ανάκτηση μη επαναλαμβανόμενων ραντεβού:**
   Εκτελέστε το ερώτημα για να λάβετε μια λίστα με ραντεβού που δεν είναι επαναλαμβανόμενα.
   
   ```csharp
   Appointment[] appointmentsByRecurrence = client.ListAppointments(queryNonRecurring);
   ```
