## Οδηγός Εφαρμογής (H2)
Αυτή η ενότητα παρουσιάζει τον τρόπο δημιουργίας ενός μηνύματος ηλεκτρονικού ταχυδρομείου με συνημμένο ραντεβού χρησιμοποιώντας το Aspose.Email για .NET.

### Δημιουργία μηνύματος ηλεκτρονικού ταχυδρομείου (H3)
Ξεκινήστε ρυθμίζοντας το `MailMessage` τάξη:

```csharp
using System;
using Aspose.Email.Mime;

// Αρχικοποίηση μιας νέας παρουσίας της κλάσης MailMessage
dynamic msg1 = new MailMessage();
msg1.To.Add("to@domain.com");
msg1.From = "from@gmail.com";
```

**Εξήγηση:**
- `msg1.To.Add(...)`: Προσθέτει έναν παραλήπτη στο email.
- `msg1.From`: Ορίζει τη διεύθυνση του αποστολέα.

### Δημιουργία αντικειμένου ραντεβού (H3)
Κλείστε ραντεβού με τις απαραίτητες λεπτομέρειες:

```csharp
using System;
using Aspose.Email.Calendar;

DateTime StartDate = new DateTime(2023, 12, 1, 17, 0, 0);
DateTime EndDate = new DateTime(2023, 12, 31, 17, 30, 0);

// Δημιουργήστε ένα ραντεβού
Appointment agendaAppointment = new Appointment("same place", StartDate, EndDate, msg1.From, msg1.To.ToArray());
agendaAppointment.UniqueId = Guid.NewGuid().ToString();
agendaAppointment.Description = "Meeting Details";
```

**Εξήγηση:**
- `DateTime`: Καθορίζει τις ημερομηνίες έναρξης και λήξης.
- Ο `Appointment` Ο κατασκευαστής ορίζει βασικές ιδιότητες όπως η τοποθεσία και οι συμμετέχοντες.

### Ορισμός μοτίβου επανάληψης για ένα ραντεβού (H3)
Ορίστε ένα εβδομαδιαίο μοτίβο επανάληψης:

```csharp
using Aspose.Email.Calendar.Recurrences;

WeeklyRecurrencePattern pattern1 = new WeeklyRecurrencePattern(14);
pattern1.StartDays = new[] { CalendarDay.Monday, CalendarDay.Tuesday, CalendarDay.Thursday };
pattern1.Interval = 1;
agendaAppointment.Recurrence = pattern1;
```

**Εξήγηση:**
- `WeeklyRecurrencePattern`: Ρυθμίζει την εβδομαδιαία επανάληψη σε συγκεκριμένες ημέρες.

### Επισύναψη ραντεβού σε μήνυμα ηλεκτρονικού ταχυδρομείου και αποστολή μέσω SMTP (H3)
Επισυνάψτε το ραντεβού ως εναλλακτική προβολή στο μήνυμα ηλεκτρονικού ταχυδρομείου σας και στείλτε το:

```csharp
using Aspose.Email.Clients.Smtp;
using System.Net.Security;

// Προσθήκη του ραντεβού ως εναλλακτική προβολή
dynamic alternateView = agendaAppointment.RequestApointment();
msg1.AlternateViews.Add(alternateView);

SmtpClient client = new SmtpClient("smtp.gmail.com", 587, "your.email@gmail.com", "your.password");
client.SecurityOptions = SecurityOptions.Auto;

// Στείλτε το email με το συνημμένο αίτημα ραντεβού
client.Send(msg1);
```

**Εξήγηση:**
- `msg1.AlternateViews.Add(...)`Επισυνάπτει το ραντεβού ως εναλλακτική προβολή.
- `SmtpClient`: Ρυθμίζει και στέλνει το email μέσω SMTP.
