## Οδηγός Εφαρμογής

Τώρα θα αναλύσουμε την υλοποίηση σε ξεχωριστά χαρακτηριστικά για καλύτερη κατανόηση.

### Χαρακτηριστικό 1: Ρύθμιση μηνιαίου μοτίβου επανάληψης

#### Επισκόπηση
Αυτή η δυνατότητα επιδεικνύει τη ρύθμιση ενός μηνιαίου μοτίβου επανάληψης για μια εργασία του Outlook, επιτρέποντας στις εργασίες να επαναλαμβάνονται σε συγκεκριμένες ημέρες κάθε μήνα.

#### Βήμα προς βήμα εφαρμογή

##### Ορίστε ημερομηνίες έναρξης και λήξης
Αρχικά, προσδιορίστε την ημερομηνία έναρξης της εργασίας σας και πότε θα πρέπει να λήξει. Προσαρμόστε αυτές τις ημερομηνίες σύμφωνα με την τοπική ζώνη ώρας:

```csharp
using Aspose.Email.Mapi;
using System;

// Ορίστε ημερομηνίες έναρξης και λήξης με προσαρμογές ζώνης ώρας
DateTime StartDate = new DateTime(2015, 7, 1);
DateTime endByDate = new DateTime(2015, 12, 31);

TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);
StartDate = StartDate.Add(ts);
endByDate = endByDate.Add(ts);
```

##### Δημιουργία νέας εργασίας του Outlook
Δημιουργήστε και διαμορφώστε την εργασία σας:

```csharp
// Δημιουργήστε ένα νέο MapiTask
MapiTask task = new MapiTask("This is test task", "Sample Body", StartDate, StartDate);
task.State = MapiTaskState.NotAssigned;
```

##### Ορισμός του μηνιαίου μοτίβου επανάληψης
Ρυθμίστε τις λεπτομέρειες του μοτίβου επανάληψης:

```csharp
var recurrence = new MapiCalendarMonthlyRecurrencePattern {
    Day = 15,
    Period = 1,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterDate,
    OccurrenceCount = GetOccurrenceCount(StartDate, endByDate, "FREQ=MONTHLY;BYMONTHDAY=15;INTERVAL=1"),
    WeekStartDay = DayOfWeek.Monday,
    EndDate = endByDate
};
task.Recurrence = recurrence;
```

##### Βοηθητική μέθοδος για τον υπολογισμό συμβάντων

```csharp
private static uint GetOccurrenceCount(DateTime start, DateTime endBy, string rrule) {
    CalendarRecurrence pattern = new CalendarRecurrence(string.Format("DTSTART:{0}\r\nRRULE:{1}", start.ToString("yyyyMMdd"), rrule));
    DateCollection dates = pattern.GenerateOccurrences(start, endBy);
    return (uint)dates.Count;
}
```

### Χαρακτηριστικό 2: Υπολογισμός αριθμού επανάληψης

#### Επισκόπηση
Υπολογίστε τον αριθμό των εμφανίσεων για έναν δεδομένο κανόνα επανάληψης μεταξύ δύο καθορισμένων ημερομηνιών.

#### Βήμα προς βήμα εφαρμογή

##### Υπολογισμός εμφανίσεων
Δημιουργήστε και διαμορφώστε τη λογική υπολογισμού επανάληψης:

```csharp
using Aspose.Email.Calendar.Recurrences;
using System;

public static uint CalculateOccurrences(DateTime start, DateTime endBy, string rrule) {
    CalendarRecurrence pattern = new CalendarRecurrence(string.Format("DTSTART:{0}\r\nRRULE:{1}", start.ToString("yyyyMMdd"), rrule));
    DateCollection dates = pattern.GenerateOccurrences(start, endBy);
    return (uint)dates.Count;
}
```
