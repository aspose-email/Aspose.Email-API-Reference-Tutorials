## Οδηγός Εφαρμογής

### Δημιουργία και ρύθμιση παραμέτρων ενός MapiTask
Δημιουργώντας ένα `MapiTask` περιλαμβάνει τον ορισμό βασικών χαρακτηριστικών όπως όνομα, σώμα, ημερομηνία έναρξης, ημερομηνία λήξης και κατάσταση.

#### Δημιουργία της εργασίας
```csharp
using Aspose.Email.Mapi;

DateTime StartDate = new DateTime(2015, 7, 16);
DateTime DueDate = new DateTime(2015, 7, 16);

// Δημιουργήστε μια νέα παρουσία MapiTask
task = new MapiTask("This is test task", "Sample Body", StartDate, DueDate);

// Ορίστε την κατάσταση της εργασίας σε "Δεν έχει ανατεθεί"
task.State = MapiTaskState.NotAssigned;
```
**Εξήγηση**: Εδώ, δημιουργούμε ένα παράδειγμα `MapiTask` με όνομα, σώμα, ημερομηνία έναρξης και ημερομηνία λήξης. Ορίζουμε επίσης την αρχική του κατάσταση.

### Ορισμός μοτίβου ημερήσιας επανάληψης για ένα MapiTask
Προσθέστε ένα μοτίβο ημερήσιας επανάληψης για να διασφαλίσετε ότι η εργασία επαναλαμβάνεται επ' αόριστον.

#### Ρύθμιση του μοτίβου επανάληψης
```csharp
var record = new MapiCalendarDailyRecurrencePattern
{
    PatternType = MapiCalendarRecurrencePatternType.Day,
    Period = 1, // Η εργασία επαναλαμβάνεται κάθε μέρα
    EndType = MapiCalendarRecurrenceEndType.NeverEnd, // Η επανάληψη δεν τελειώνει ποτέ
};

// Αντιστοίχιση του μοτίβου επανάληψης στην εργασία
task.Recurrence = record;
```
**Εξήγηση**Αυτό το τμήμα κώδικα ορίζει ένα μοτίβο ημερήσιας επανάληψης που δεν θα τερματιστεί. `PatternType` έχει οριστεί σε `Day`, και `Period` καθορίζει το χρονικό διάστημα των ημερών μεταξύ των εμφανίσεων.

### Αποθήκευση ενός MapiTask σε αρχείο
Τέλος, αποθηκεύστε την διαμορφωμένη εργασία σας ως αρχείο μηνύματος του Outlook.

#### Αποθήκευση της εργασίας
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή καταλόγου εγγράφου σας

// Αποθήκευση του MapiTask σε αρχείο .msg
task.Save(dataDir + "/SetDailyNeverEndRecurrence_out.msg", TaskSaveFormat.Msg);
```
**Εξήγηση**Αυτός ο κώδικας αποθηκεύει την εργασία σας σε ένα `.msg` αρχείο, το οποίο μπορεί να ανοιχτεί στο Outlook.
