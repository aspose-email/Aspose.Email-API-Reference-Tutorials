## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει, ας προχωρήσουμε στη δημιουργία μιας επαναλαμβανόμενης εργασίας.

### Δημιουργία και αποθήκευση εργασίας με επανάληψη

Αυτή η ενότητα εστιάζει στον τρόπο δημιουργίας μιας εργασίας του Outlook χρησιμοποιώντας το Aspose.Email για .NET και στη διαμόρφωση της ώστε να επαναλαμβάνεται εβδομαδιαίως.

#### Επισκόπηση
Θα μάθετε να ορίζετε την ημερομηνία έναρξης, την ημερομηνία λήξης και το μοτίβο επανάληψης μιας εργασίας, διασφαλίζοντας ότι οι εργασίες σας προγραμματίζονται αυτόματα σύμφωνα με τις ανάγκες σας.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός Τοπικής Ζώνης Ώρας**

Για να διασφαλίσετε την ακρίβεια στον προγραμματισμό, καταγράψτε πρώτα την τοπική ζώνη ώρας από την UTC:

```csharp
using System;

TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);
```

Εδώ, `ts` διατηρεί τη διαφορά ώρας μεταξύ της τοπικής σας ώρας και της UTC. Αυτό διασφαλίζει ότι οι εργασίες δημιουργούνται στην τοπική σας ώρα.

**2. Ορίστε ημερομηνίες έναρξης και λήξης**

Στη συνέχεια, ορίστε πότε πρέπει να ξεκινήσει και να τελειώσει η εργασία:

```csharp
DateTime StartDate = new DateTime(2015, 7, 16).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 16).Add(ts);
DateTime endByDate = new DateTime(2015, 9, 1).Add(ts);
```

Αυτές οι ημερομηνίες προσαρμόζονται στην τοπική σας ζώνη ώρας, διασφαλίζοντας ότι είναι ακριβείς σε διαφορετικές περιοχές.

**3. Δημιουργήστε ένα MapiTask**

Δημιουργήστε την εργασία χρησιμοποιώντας `MapiTask`, προσδιορίζοντας το αντικείμενό του και άλλες λεπτομέρειες:

```csharp
MapiTask task = new MapiTask("This is a test task", "Description of the task", StartDate, DueDate);
```

**4. Ορισμός μοτίβου επανάληψης**

Για να επαναλαμβάνεται αυτή η εργασία εβδομαδιαίως σε συγκεκριμένες ημέρες, διαμορφώστε το μοτίβο επανάληψής της:

```csharp
RecurrencePattern recurrence = new WeeklyRecurrencePattern(StartDate)
{
    OccursEveryWeek = true,
    DayOfWeekMask = MapiWeeklyRecurrencePattern.WeekDays.Monday | 
                     MapiWeeklyRecurrencePattern.WeekDays.Wednesday |
                     MapiWeeklyRecurrencePattern.WeekDays.Friday
};

task.RecurrencePattern = recurrence;
```

Αυτό το μοτίβο κάνει την εργασία να πραγματοποιείται κάθε Δευτέρα, Τετάρτη και Παρασκευή ξεκινώντας από `StartDate`.

**5. Αποθήκευση της εργασίας**

Τέλος, αποθηκεύστε την εργασία σας σε έναν καθορισμένο κατάλογο:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
task.Save(dataDir + "\TaskWithRecurrence.msg", TaskSaveFormat.Msg);
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Ζητήματα Ζώνης Ώρας**: Βεβαιωθείτε `ts` αντικατοπτρίζει με ακρίβεια την τοπική σας ζώνη ώρας. Οι λανθασμένες μετατοπίσεις μπορούν να οδηγήσουν σε προγραμματισμό εργασιών σε λάθος ώρες.
- **Σφάλματα διαδρομής αρχείου**: Επαληθεύστε ότι `dataDir` έχει ρυθμιστεί σωστά και είναι προσβάσιμο από την εφαρμογή σας.
