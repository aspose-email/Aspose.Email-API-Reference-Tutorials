# Πώς να δημιουργήσετε εβδομαδιαίες επαναλαμβανόμενες εργασίες MapiTasks στο .NET χρησιμοποιώντας το Aspose.Email

## Εισαγωγή

Η αποτελεσματική διαχείριση επαναλαμβανόμενων εργασιών είναι ζωτικής σημασίας για τους προγραμματιστές που εργάζονται σε εφαρμογές που περιλαμβάνουν λειτουργίες προγραμματισμού ή ημερολογίου. Είτε αναπτύσσετε ένα εσωτερικό εργαλείο για τη διαχείριση εργασιών είτε ενσωματώνετε λειτουργίες ημερολογίου σε μια επιχειρηματική εφαρμογή, η δημιουργία και η διαχείριση εβδομαδιαίων επαναλαμβανόμενων εργασιών μπορεί να βελτιώσει σημαντικά την παραγωγικότητα.

Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να χρησιμοποιήσουμε **Aspose.Email για .NET** για να δημιουργείτε εβδομαδιαίες επαναλαμβανόμενες εργασίες MapiTasks που λήγουν μετά από μια συγκεκριμένη ημερομηνία. Αυτή η λειτουργία είναι ανεκτίμητη για προγραμματιστές που θέλουν να αυτοματοποιήσουν τον προγραμματισμό στις εφαρμογές τους χρησιμοποιώντας τις ισχυρές λειτουργίες του Aspose.Email.

### Τι θα μάθετε:
- Ρύθμιση και ρύθμιση παραμέτρων του Aspose.Email για .NET
- Δημιουργία ενός εβδομαδιαίου επαναλαμβανόμενου MapiTask με καθορισμένη ημερομηνία λήξης
- Εφαρμογή μοτίβων επανάληψης πολλαπλών ημερών
- Υπολογισμός αριθμού εμφανίσεων με βάση προσαρμοσμένους κανόνες επανάληψης

Είστε έτοιμοι να βυθιστείτε στη δημιουργία ισχυρών λύσεων προγραμματισμού; Ας ξεκινήσουμε!

## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: τη δημιουργία μιας εβδομαδιαίας επανάληψης μίας ημέρας και τη ρύθμιση επαναλήψεων πολλαπλών ημερών.

### Δημιουργία εβδομαδιαίου επαναλαμβανόμενου MapiTask που λήγει μετά από μια συγκεκριμένη ημερομηνία

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε εργασίες που επαναλαμβάνονται μια συγκεκριμένη ημέρα κάθε εβδομάδα μέχρι να λήξουν μετά από μια δεδομένη ημερομηνία. Είναι ιδανική για τον προγραμματισμό επαναλαμβανόμενων συσκέψεων ή προθεσμιών.

#### Βήματα Υλοποίησης
**Βήμα 1: Διαμόρφωση του μοτίβου επανάληψης**
Εδώ, θα ρυθμίσουμε το μοτίβο επανάληψης χρησιμοποιώντας `MapiCalendarWeeklyRecurrencePattern`.
```csharp
var rec = new MapiCalendarWeeklyRecurrencePattern
{
    EndType = MapiCalendarRecurrenceEndType.EndAfterDate,
    PatternType = MapiCalendarRecurrencePatternType.Week,
    Period = 1, // Εβδομαδιαία επανάληψη
    WeekStartDay = DayOfWeek.Sunday,
    DayOfWeek = MapiCalendarDayOfWeek.Friday, // Επαναλαμβάνεται τις Παρασκευές
    EndDate = new DateTime(2015, 9, 1).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now)),
    OccurrenceCount = GetOccurrenceCount(
        new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now)),
        new DateTime(2015, 9, 1).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now)), 
        "FREQ=WEEKLY;BYDAY=FR;INTERVAL=1")
};
```
**Βήμα 2: Δημιουργήστε το MapiTask**
Τώρα που έχουμε διαμορφώσει το μοτίβο επανάληψης, ας δημιουργήσουμε μια εργασία και ας της αντιστοιχίσουμε αυτό το μοτίβο.
```csharp
MapiTask task = new MapiTask(
    "This is test task",
    "Sample Body",
    new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now)),
    new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now))
);
task.State = MapiTaskState.NotAssigned;

if (rec.OccurrenceCount == 0)
{
    rec.OccurrenceCount = 1; // Βεβαιωθείτε ότι υπάρχει τουλάχιστον μία εμφάνιση
}

task.Recurrence = rec;
```
**Βήμα 3: Αποθήκευση της εργασίας**
Τέλος, αποθηκεύστε την εργασία σας σε ένα αρχείο .msg για διατήρηση.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
task.Save(dataDir + "/SetWeeklyEndAfterDateEveryDayRecurrence_out.msg", TaskSaveFormat.Msg);
```

### Δημιουργία εβδομαδιαίου επαναλαμβανόμενου MapiTask σε πολλές ημέρες που λήγει μετά από μια συγκεκριμένη ημερομηνία

#### Επισκόπηση
Αυτή η δυνατότητα επεκτείνει την προηγούμενη ρύθμιση ώστε να επιτρέπει εργασίες που επαναλαμβάνονται πολλές ημέρες κάθε εβδομάδα, παρέχοντας ευελιξία για πιο σύνθετες ανάγκες προγραμματισμού.

#### Βήματα Υλοποίησης
**Βήμα 1: Διαμόρφωση του μοτίβου επανάληψης πολλαπλών ημερών**
Ορίστε ένα μοτίβο που περιλαμβάνει πολλαπλές ημέρες επανάληψης ανά εβδομάδα.
```csharp
var record = new MapiCalendarWeeklyRecurrencePattern
{
    EndType = MapiCalendarRecurrenceEndType.EndAfterDate,
    PatternType = MapiCalendarRecurrencePatternType.Week,
    Period = 2, // Συμβαίνει κάθε δύο εβδομάδες
    WeekStartDay = DayOfWeek.Sunday,
    EndDate = new DateTime(2015, 9, 1).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now)),
    DayOfWeek = MapiCalendarDayOfWeek.Friday | MapiCalendarDayOfWeek.Monday, // Επαναλαμβάνεται τις Παρασκευές και τις Δευτέρες
    OccurrenceCount = GetOccurrenceCount(
        new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now)), 
        new DateTime(2015, 9, 1).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now)), 
        "FREQ=WEEKLY;BYDAY=FR,MO;INTERVAL=2")
};
```
**Βήμα 2: Δημιουργία και ανάθεση του MapiTask**
Όπως και πριν, δημιουργήστε μια εργασία και αντιστοιχίστε αυτό το μοτίβο πολλαπλών ημερών.
```csharp
MapiTask task = new MapiTask(
    "This is test task",
    "Sample Body",
    new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now)),
    new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now))
);
task.State = MapiTaskState.NotAssigned;
task.Recurrence = record;
```
**Βήμα 3: Αποθήκευση της πολυήμερης εργασίας**
Αποθηκεύστε την εργασία σας με παρόμοιο τρόπο για να βεβαιωθείτε ότι έχει αποθηκευτεί σωστά.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
task.Save(dataDir + "/SetWeeklyEndAfterDateMultipleDaysRecurrence_out.msg", TaskSaveFormat.Msg);
```
