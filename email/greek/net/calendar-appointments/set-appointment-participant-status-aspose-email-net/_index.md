### Ορισμός κατάστασης συμμετέχοντα για τους συμμετέχοντες σε ραντεβού
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε τον τρόπο με τον οποίο κάθε συμμετέχων θα συμμετάσχει στο ραντεβού σας, ορίζοντας την κατάστασή του ως "Αποδεχόμενος" ή "Απορριφθέν". Αυτό είναι κρίσιμο για την αποτελεσματική διαχείριση των συσκέψεων.

##### Βήμα 1: Ορισμός Οργανωτή και Συμμετεχόντων
Ξεκινήστε ορίζοντας τον διοργανωτή και τους συμμετέχοντες με τις αντίστοιχες διευθύνσεις ηλεκτρονικού ταχυδρομείου τους:

```csharp
string location = "Room 5";
DateTime startDate = new DateTime(2023, 10, 12, 10, 0, 0);
DateTime endDate = new DateTime(2023, 10, 12, 11, 0, 0);

MailAddress organizer = new MailAddress("organizer@example.com", "Organizer");
MailAddressCollection attendees = new MailAddressCollection();
```

##### Βήμα 2: Ορισμός Κατάστασης Συμμετοχής
Αντιστοίχιση καταστάσεων σε κάθε συμμετέχοντα:

```csharp
// Συμμετέχων 1: Κατάσταση αποδοχής.
MailAddress attendee1 = new MailAddress("attendee1@example.com", "First attendee");
attendee1.ParticipationStatus = ParticipationStatus.Accepted;
attendees.Add(attendee1);

// Συμμετέχων 2: Κατάσταση απόρριψης.
MailAddress attendee2 = new MailAddress("attendee2@example.com", "Second attendee");
attendee2.ParticipationStatus = ParticipationStatus.Declined;
attendees.Add(attendee2);
```

##### Βήμα 3: Δημιουργήστε το Ραντεβού
Χρησιμοποιήστε τις καθορισμένες λεπτομέρειες για να δημιουργήσετε ένα ραντεβού:

```csharp
Appointment target = new Appointment(location, startDate, endDate, organizer, attendees);
```
