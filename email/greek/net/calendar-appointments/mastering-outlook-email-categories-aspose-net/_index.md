## Οδηγός Εφαρμογής

### Επισκόπηση της Διαχείρισης Κατηγοριών Ηλεκτρονικού Ταχυδρομείου

Σε αυτήν την ενότητα, θα εξερευνήσουμε τον τρόπο αποτελεσματικής διαχείρισης κατηγοριών email χρησιμοποιώντας το Aspose.Email. Θα καλύψουμε την προσθήκη, την ανάκτηση και την αφαίρεση κατηγοριών από μηνύματα του Outlook.

#### Προσθήκη κατηγοριών σε ένα email

Για να ορίσετε κατηγορίες χρωμάτων για ένα μήνυμα ηλεκτρονικού ταχυδρομείου στο Outlook χρησιμοποιώντας το Aspose.Email:

**Βήμα 1: Φόρτωση του μηνύματος**

Αρχικά, φορτώστε το αρχείο μηνύματος του Outlook σε ένα `MapiMessage` αντικείμενο.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή καταλόγου σας
MapiMessage msg = MapiMessage.FromFile(dataDir + "/message1.msg");
```

**Βήμα 2: Προσθήκη κατηγοριών**

Χρησιμοποιήστε το `FollowUpManager.AddCategory()` μέθοδος για την ανάθεση κατηγοριών. Δείτε πώς προσθέτετε κατηγορίες Μωβ και Κόκκινο:

```csharp
// Προσθήκη κατηγοριών Μωβ και Κόκκινο
FollowUpManager.AddCategory(msg, "Purple Category");
FollowUpManager.AddCategory(msg, "Red Category");
```

#### Ανάκτηση αντιστοιχισμένων κατηγοριών

Για να δείτε ποιες κατηγορίες έχουν αντιστοιχιστεί στο μήνυμά σας, ανακτήστε τες χρησιμοποιώντας την ακόλουθη μέθοδο:

```csharp
IList<string> categories = FollowUpManager.GetCategories(msg);
// Έξοδος της λίστας κατηγοριών
foreach (var category in categories)
{
    Console.WriteLine(category);
}
```

#### Αφαίρεση συγκεκριμένων και όλων των κατηγοριών

Η κατάργηση μιας συγκεκριμένης κατηγορίας ή η διαγραφή όλων των κατηγοριών είναι απλή:

**Αφαίρεση κατηγορίας:**

```csharp
FollowUpManager.RemoveCategory(msg, "Red Category");
```

**Καθαρισμός όλων των κατηγοριών:**

```csharp
FollowUpManager.ClearCategories(msg);
```
