## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει δύο κύρια χαρακτηριστικά: τη δημιουργία μιας εβδομαδιαίας επαναλαμβανόμενης εργασίας και τον υπολογισμό των εμφανίσεων.

### Χαρακτηριστικό 1: Εβδομαδιαία δημιουργία εργασιών με επανάληψη

**Επισκόπηση:**
Μάθετε πώς να δημιουργείτε ένα MapiTask που επαναλαμβάνεται εβδομαδιαίως χρησιμοποιώντας το Aspose.Email `MapiCalendarWeeklyRecurrencePattern`, αυτοματοποιώντας τη δημιουργία εργασιών χωρίς χειροκίνητη παρέμβαση για κάθε εμφάνιση.

#### Βήμα 1: Ορισμός ημερομηνιών και προσαρμογή για ζώνη ώρας
```csharp
// Ορίστε τις ημερομηνίες έναρξης, λήξης και λήξης για την εργασία
DateTime StartDate = new DateTime(2015, 7, 16);
DateTime DueDate = new DateTime(2015, 7, 16);
DateTime EndByDate = new DateTime(2015, 9, 1);

// Προσαρμόστε τις ημερομηνίες με βάση την τοπική ζώνη ώρας
timeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);
StartDate = StartDate.Add(ts);
DueDate = DueDate.Add(ts);
EndByDate = EndByDate.Add(ts);
```
**Εξήγηση:**
Οι ημερομηνίες έναρξης, λήξης και λήξης της εργασίας προσαρμόζονται για την τρέχουσα μετατόπιση ζώνης ώρας, ώστε να διασφαλίζεται η ακρίβεια σε διαφορετικές περιοχές.

#### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων του MapiTask
```csharp
// Δημιουργήστε ένα νέο MapiTask με συγκεκριμένες λεπτομέρειες
MapiTask task = new MapiTask("This is a test task", "Sample Body", StartDate, DueDate);
task.State = MapiTaskState.NotAssigned;
```
**Εξήγηση:**
Αρχικοποίηση του `MapiTask` αντικείμενο με τίτλο, κύριο σώμα, ημερομηνία έναρξης και ημερομηνία λήξης. Ορίστε την κατάσταση της εργασίας σε `NotAssigned`, επισημαίνοντάς το ως εκκρεμές.

#### Βήμα 3: Ορισμός μοτίβου εβδομαδιαίας επανάληψης
```csharp
// Ρύθμιση παραμέτρων του εβδομαδιαίου μοτίβου επανάληψης για την εργασία
var rec = new MapiCalendarWeeklyRecurrencePattern
{
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    PatternType = MapiCalendarRecurrencePatternType.Week,
    Period = 1,
    WeekStartDay = DayOfWeek.Sunday,
    DayOfWeek = MapiCalendarDayOfWeek.Friday,
    OccurrenceCount = GetOccurrenceCount(StartDate, EndByDate, "FREQ=WEEKLY;BYDAY=FR"),
};

// Βεβαιωθείτε ότι υπάρχει τουλάχιστον μία εμφάνιση
if (rec.OccurrenceCount == 0)
{
    rec.OccurrenceCount = 1;
}
task.Recurrence = rec;
```
**Εξήγηση:**
Αυτό το τμήμα κώδικα διαμορφώνει την εργασία ώστε να επαναλαμβάνεται εβδομαδιαίως τις Παρασκευές. `GetOccurrenceCount` Η συνάρτηση υπολογίζει πόσες εμφανίσεις εμπίπτουν μεταξύ των ημερομηνιών έναρξης και λήξης.

#### Βήμα 4: Αποθήκευση εργασίας
```csharp
// Αποθήκευση της εργασίας σε ένα αρχείο στον καθορισμένο κατάλογο εξόδου
string outputPath = "@YOUR_OUTPUT_DIRECTORY/Weekly_out.msg";
task.Save(outputPath, TaskSaveFormat.Msg);
```
**Εξήγηση:**
Η ολοκληρωμένη εργασία αποθηκεύεται ως αρχείο MSG. Βεβαιωθείτε ότι έχετε αντικαταστήσει `@YOUR_OUTPUT_DIRECTORY` με την πραγματική σου πορεία.

### Χαρακτηριστικό 2: Υπολογισμός εμφανίσεων μεταξύ δύο ημερομηνιών με κανόνα επανάληψης

**Επισκόπηση:**
Προσδιορίστε τον αριθμό των φορών που εμφανίζεται ένα επαναλαμβανόμενο συμβάν μεταξύ δύο ημερομηνιών χρησιμοποιώντας το Aspose.Email `CalendarRecurrence` τάξη.

#### Βήμα 1: Ορισμός ημερομηνιών και κανόνα επανάληψης
```csharp
// Ορίστε ημερομηνίες έναρξης και λήξης για τον υπολογισμό των εμφανίσεων
DateTime Start = new DateTime(2015, 7, 16);
DateTime EndBy = new DateTime(2015, 9, 1);
string RRule = "FREQ=WEEKLY;BYDAY=FR";
```
**Εξήγηση:**
Αυτές οι μεταβλητές ορίζουν το εύρος ημερομηνιών και έναν κανόνα για εβδομαδιαίες εμφανίσεις τις Παρασκευές.

#### Βήμα 2: Υπολογισμός εμφανίσεων
```csharp
// Λήψη του αριθμού των εμφανίσεων μεταξύ των δύο ημερομηνιών με βάση τον κανόνα επανάληψης
uint occurrenceCount = GetOccurrenceCount(Start, EndBy, RRule);
```
**Εξήγηση:**
Η συνάρτηση υπολογίζει πόσες φορές επαναλαμβάνεται η εργασία εντός της καθορισμένης περιόδου.

#### Βήμα 3: Υλοποίηση `GetOccurrenceCount` Μέθοδος
```csharp
private static uint GetOccurrenceCount(DateTime start, DateTime endBy, string rrule)
{
    // Δημιουργήστε ένα αντικείμενο CalendarRecurrence με μορφή DTSTART και RRULE
    CalendarRecurrence pattern = new CalendarRecurrence(string.Format("DTSTART:{0}\r\nRRULE:{1}", start.ToString("yyyyMMdd"), rrule));

    // Δημιουργία εμφανίσεων εντός του καθορισμένου εύρους ημερομηνιών
    DateCollection dates = pattern.GenerateOccurrences(start, endBy);

    // Επιστρέψτε τον αριθμό των δημιουργημένων εμφανίσεων
    return (uint)dates.Count;
}
```
**Εξήγηση:**
Ο `CalendarRecurrence` Το αντικείμενο αρχικοποιείται με μια ημερομηνία έναρξης και έναν κανόνα επανάληψης, δημιουργώντας εμφανίσεις που εμπίπτουν εντός του δεδομένου εύρους.
