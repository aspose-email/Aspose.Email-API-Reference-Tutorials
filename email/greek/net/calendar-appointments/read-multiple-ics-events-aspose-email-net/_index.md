## Οδηγός Εφαρμογής

### Ανάγνωση πολλαπλών συμβάντων από ένα αρχείο ICS

Θα επικεντρωθούμε στην υλοποίηση μιας λειτουργίας για την ανάγνωση πολλαπλών συμβάντων από ένα `.ics` αρχείο.

#### Βήμα 1: Αρχικοποίηση του CalendarReader και της λίστας για ραντεβού

Αρχικοποίηση του `CalendarReader` με το δικό σου `.ics` διαδρομή αρχείου και, στη συνέχεια, δημιουργήστε μια λίστα για ραντεβού:

```csharp
// Αρχικοποίηση λίστας για την κράτηση ραντεβού
dateList<Appointment> appointments = new dateList<Appointment>();

// Δημιουργήστε μια παρουσία του CalendarReader με τη διαδρομή αρχείου ICS
CalendarReader reader = new CalendarReader(dataDir);
```

#### Βήμα 2: Επανάληψη συμβάντων και προσθήκη τους στη λίστα

Επαναλάβετε κάθε συμβάν στο `.ics` αρχείο χρησιμοποιώντας έναν βρόχο, προσθέτοντάς τα στη λίστα σας:

```csharp
// Επαναλάβετε κάθε συμβάν στο αρχείο ICS και προσθέστε το στη λίστα
do {
    var currentEvent = reader.NextEvent();
    if (currentEvent != null)
        appointments.Add(currentEvent);
} while (reader.NextEvent() != null);
```

**Εξήγηση**: Το `NextEvent()` Η μέθοδος επαναλαμβάνεται σε συμβάντα και ο βρόχος διασφαλίζει ότι όλα τα ραντεβού καταγράφονται αποτελεσματικά.
