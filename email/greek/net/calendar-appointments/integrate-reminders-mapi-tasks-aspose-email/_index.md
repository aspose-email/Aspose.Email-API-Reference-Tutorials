#### Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων μιας εργασίας MAPI
Δημιουργήστε μια νέα παρουσία του `MapiTask` και ορίστε τις ιδιότητές του, συμπεριλαμβανομένων των υπενθυμίσεων:
```csharp
// Αρχικοποίηση μιας νέας εργασίας MAPI
MapiTask testTask = new MapiTask("Task with Reminder", "This is a sample task.", DateTime.Now, DateTime.Now.AddDays(7));

// Ρύθμιση παραμέτρων επιλογών υπενθύμισης
testTask.ReminderSet = true;
testTask.ReminderTime = DateTime.Now.AddMinutes(30); // Ορίστε την ώρα υπενθύμισης
```

#### Εξήγηση
- `MapiTask`: Αντιπροσωπεύει ένα αντικείμενο εργασίας MAPI.
- `ReminderSet`: Μια λογική τιμή που υποδεικνύει εάν μια υπενθύμιση είναι ενεργοποιημένη.
- `ReminderTime`: Καθορίζει πότε θα πρέπει να ενεργοποιηθεί η υπενθύμιση.
