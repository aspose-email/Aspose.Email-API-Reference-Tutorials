### Ρύθμιση εβδομαδιαίας επανάληψης με πολλαπλές ημέρες
#### Επισκόπηση
Μάθετε πώς να διαμορφώνετε μια εργασία που επαναλαμβάνεται σε συγκεκριμένες ημέρες της εβδομάδας σε καθορισμένα χρονικά διαστήματα. Αυτό είναι ιδανικό για τη δημιουργία ημερολογίων ή υπενθυμίσεων για εργασίες όπως οι δεκαπενθήμερες συσκέψεις που πραγματοποιούνται Δευτέρες και Παρασκευές.

#### Βήμα 1: Αρχικοποίηση λεπτομερειών εργασίας
Ξεκινήστε ορίζοντας την ημερομηνία έναρξης, την ημερομηνία λήξης και την ημερομηνία λήξης με εφαρμογή της μετατόπισης ζώνης ώρας:
```csharp
DateTime StartDate = new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now));
DateTime DueDate = new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now));
DateTime endByDate = new DateTime(2015, 9, 1).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now));

MapiTask task = new MapiTask("This is a test task", "Sample Body", StartDate, DueDate);
task.State = MapiTaskState.NotAssigned;
```
#### Βήμα 2: Ρύθμιση παραμέτρων μοτίβου επανάληψης
Στη συνέχεια, ορίστε το μοτίβο επανάληψης. Εδώ, καθορίζετε ότι η εργασία θα πρέπει να επαναλαμβάνεται ανά δύο εβδομάδες, Δευτέρα και Παρασκευή:
```csharp
var rec = new MapiCalendarWeeklyRecurrencePattern
{
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    PatternType = MapiCalendarRecurrencePatternType.Week,
    Period = 2, // Διάστημα δύο εβδομάδων
    WeekStartDay = DayOfWeek.Sunday,
    DayOfWeek = MapiCalendarDayOfWeek.Friday | MapiCalendarDayOfWeek.Monday,
};

// Υπολογισμός του αριθμού εμφανίσεων μεταξύ των ημερομηνιών έναρξης και λήξης
rec.OccurrenceCount = GetOccurrenceCount(StartDate, endByDate, "FREQ=WEEKLY;BYDAY=FR,MO;INTERVAL=2");
if (rec.OccurrenceCount == 0)
{
    rec.OccurrenceCount = 1;
}
task.Recurrence = rec;
```
#### Βήμα 3: Αποθήκευση της εργασίας
Τέλος, αποθηκεύστε την εργασία σε ένα αρχείο. Αυτό το βήμα διασφαλίζει ότι η ρύθμιση της επανάληψης διατηρείται και ότι θα είναι δυνατή η πρόσβαση σε αυτήν αργότερα.
```csharp
task.Save("YOUR_OUTPUT_DIRECTORY\SetWeeklyRecurrenceMultipleDaysInWeekWithInterval_out.msg", TaskSaveFormat.Msg);
```
### Υπολογισμός εμφανίσεων από έναν κανόνα επανάληψης
Αυτή η λειτουργία υπολογίζει τον αριθμό των εμφανίσεων για έναν δεδομένο κανόνα μεταξύ δύο ημερομηνιών, διασφαλίζοντας ότι το χρονοδιάγραμμα εργασιών σας είναι ακριβές και αξιόπιστο.
#### Επισκόπηση μεθόδου
Η μέθοδος `GetOccurrenceCount` λαμβάνει μια ημερομηνία έναρξης, μια ημερομηνία λήξης και έναν κανόνα επανάληψης (RRULE) για να υπολογίσει πόσες φορές θα συμβεί το συμβάν εντός της καθορισμένης περιόδου:
```csharp
private static uint GetOccurrenceCount(DateTime start, DateTime endBy, string rrule)
{
    var pattern = new CalendarRecurrence(string.Format("DTSTART:{0}\r\nRRULE:{1}", start.ToString("yyyyMMdd"), rrule));
    DateCollection dateOccurrences = pattern.GenerateOccurrences(start, endBy);
    return (uint)dateOccurrences.Count;
}
```
### Συμβουλές αντιμετώπισης προβλημάτων
- **Προβλήματα Ζώνης Ώρας:** Βεβαιωθείτε ότι οι ρυθμίσεις ζώνης ώρας είναι συνεπείς σε όλα τα αντικείμενα DateTime.
- **Σφάλματα κανόνα επανάληψης:** Ελέγξτε ξανά τη σύνταξη του RRULE για τυπογραφικά λάθη ή λανθασμένες παραμέτρους.
