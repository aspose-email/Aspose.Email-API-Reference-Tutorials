## Οδηγός Εφαρμογής

### Δημιουργία ραντεβού με συγκεκριμένο τύπο μεθόδου

Η δημιουργία ραντεβού μέσω προγραμματισμού είναι απλή. Δείτε πώς μπορείτε να το κάνετε βήμα προς βήμα.

#### Βήμα 1: Αρχικοποίηση των λεπτομερειών ραντεβού

Ξεκινήστε ορίζοντας τις διευθύνσεις email του αποστολέα και του παραλήπτη:
```csharp
string sender = "test@gmail.com";
string recipient = "test@email.com";
```
Στη συνέχεια, δημιουργήστε ένα `Appointment` αντικείμενο με τις απαραίτητες λεπτομέρειες όπως τοποθεσία, ώρα έναρξης, ώρα λήξης, θέμα και συμμετέχοντες.
```csharp
// Ορίστε τον κατάλογο για την αποθήκευση αρχείων ραντεβού (προσαρμόστε τη διαδρομή όπως απαιτείται)
string directory = @"YOUR_DOCUMENT_DIRECTORY";

// Δημιουργία παρουσίας ραντεβού
Appointment app = new Appointment(
    "Room 112", // Τοποθεσία
    DateTime.Now.AddHours(1), // Ώρα έναρξης
    DateTime.Now.AddHours(2), // Ώρα λήξης
    sender,                    // Διοργανωτής
    new[] { recipient },       // Συμμετέχοντες
    "Discussion on Aspose.Email Features"); // Θέμα
```
#### Βήμα 2: Ρύθμιση παραμέτρων τύπου μεθόδου ραντεβού

Καθορίστε τον τύπο μεθόδου του ραντεβού (π.χ., CreateOrUpdate) για να ορίσετε τη συμπεριφορά του:
```csharp
app.MethodType = AppointmentMethodType.CreateOrUpdate;
```
Αυτή η ρύθμιση καθορίζει εάν η συνάντηση θα δημιουργηθεί ή θα ενημερωθεί, εάν υπάρχει ήδη.

#### Βήμα 3: Αποθήκευση του ραντεβού

Αποθηκεύστε το ραντεβού σας σε ένα αρχείο σε μορφή ICS, το οποίο μπορεί να χρησιμοποιηθεί από εφαρμογές ημερολογίου όπως το Outlook:
```csharp
app.Save(directory + "Appointment.ics", AppointmentSaveFormat.Ics);
```
### Βασικές επιλογές διαμόρφωσης και συμβουλές αντιμετώπισης προβλημάτων

- **Τύπος Μεθόδου:** Επιλέγω `Create` ή `CreateOrUpdate` με βάση τις ανάγκες σας.
- **Ρυθμίσεις ζώνης ώρας:** Βεβαιωθείτε ότι η ώρα του ραντεβού αντικατοπτρίζει τη σωστή ζώνη ώρας για να αποφύγετε σύγχυση.

**Συνήθη προβλήματα:**
- **Λανθασμένες Ζώνες Ώρας:** Ελέγξτε ξανά τις ρυθμίσεις ζώνης ώρας στο περιβάλλον της εφαρμογής σας.
- **Σφάλματα διαδρομής αρχείου:** Βεβαιωθείτε ότι η διαδρομή καταλόγου έχει καθοριστεί σωστά και είναι προσβάσιμη.
