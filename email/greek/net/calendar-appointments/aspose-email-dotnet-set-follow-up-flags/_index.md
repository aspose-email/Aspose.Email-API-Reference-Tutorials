## Οδηγός Εφαρμογής

### Ορισμός παρακολούθησης για παραλήπτες

Αυτή η ενότητα παρουσιάζει τη δημιουργία ενός προσχεδίου μηνύματος με επιλογές παρακολούθησης χρησιμοποιώντας το MapiMessage.

#### Επισκόπηση
Ο ορισμός σημαιών παρακολούθησης σάς επιτρέπει να προσθέτετε υπενθυμίσεις και σημειώσεις απευθείας στα email, βοηθώντας στην αποτελεσματική παρακολούθηση σημαντικών επικοινωνιών.

#### Οδηγός βήμα προς βήμα

**1. Δημιουργήστε το μήνυμα ηλεκτρονικού ταχυδρομείου**
Ξεκινήστε δημιουργώντας μια παρουσία του `MailMessage`:
```csharp
using System;
using Aspose.Email;
using Aspose.Email.Mapi;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του καταλόγου σας.

// Δημιουργήστε μια νέα παρουσία MailMessage.
MailMessage mailMsg = new MailMessage();
mailMsg.Sender = "AETest12@gmail.com";
mailMsg.To = "receiver@gmail.com";
mailMsg.Body = "This message will test if follow up options can be added to a new Mapi message.";
```

**2. Μετατροπή σε MapiMessage και σήμανση ως προσχέδιο**
Μετατρέψτε το `MailMessage` να `MapiMessage`, επισημαίνοντάς το ως μη απεσταλμένο:
```csharp
// Μετατρέψτε το MailMessage σε MapiMessage, επισημαίνοντάς το ως πρόχειρο.
MapiMessage mapi = MapiMessage.FromMailMessage(mailMsg);
mapi.SetMessageFlags(MapiMessageFlags.MSGFLAG_UNSENT); // Επισήμανση μηνύματος ως προχείρου
```

**3. Ορίστε ημερομηνία και ώρα παρακολούθησης**
Ορίστε την ημερομηνία υπενθύμισης για την παρακολούθηση:
```csharp
// Ορίστε την ημερομηνία και την ώρα της υπενθύμισης.
DateTime dtReminderDate = new DateTime(2013, 5, 23, 16, 40, 0);

// Ορίστε τη σημαία παρακολούθησης με μια καθορισμένη ημερομηνία υπενθύμισης.
FollowUpManager.SetFlagForRecipients(mapi, "Follow up", dtReminderDate);
```

**4. Αποθήκευση του μηνύματος**
Τέλος, αποθηκεύστε το προσχέδιο του μηνύματός σας:
```csharp
// Αποθηκεύστε το μήνυμα σε ένα αρχείο εξόδου.
mapi.Save($"{dataDir}\SetFollowUpForRecipients_out.msg");
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Βεβαιωθείτε ότι οι διαδρομές είναι σωστές:** Επαληθεύστε ότι `dataDir` και οι διαδρομές του καταλόγου εξόδου υπάρχουν.
- **Μορφή ημερομηνίας ελέγχου:** Βεβαιωθείτε ότι η μορφή ημερομηνίας υπενθύμισης ταιριάζει με τις τοπικές ρυθμίσεις σας.
