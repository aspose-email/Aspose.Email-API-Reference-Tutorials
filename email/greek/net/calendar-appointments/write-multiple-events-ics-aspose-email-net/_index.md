### Εγγραφή πολλαπλών συμβάντων σε ένα αρχείο ICS

#### Επισκόπηση
Δημιουργήστε αποτελεσματικά μια σειρά από συμβάντα ημερολογίου σε ένα `.ics` αρχείο.

#### Βήματα για την Υλοποίηση

**Βήμα 1: Ορισμός καταλόγου εξόδου**
```csharp
// Καθορίστε τον κατάλογο εξόδου για την αποθήκευση του αρχείου ICS.
string dataDir = "YOUR_OUTPUT_DIRECTORY" + "/WriteMultipleEventsToICS_out.ics";
```
Εδώ, `dataDir` είναι εκεί που είσαι εσύ `.ics` το αρχείο θα αποθηκευτεί.

**Βήμα 2: Αρχικοποίηση επιλογών αποθήκευσης**
```csharp
// Ρυθμίστε τις επιλογές αποθήκευσης για να δημιουργήσετε νέα συμβάντα.
IcsSaveOptions saveOptions = new IcsSaveOptions();
saveOptions.Action = AppointmentAction.Create;
```
Αυτή η ρύθμιση παραμέτρων καθορίζει ότι η ενέργεια για αυτά τα ραντεβού είναι η δημιουργία νέων καταχωρήσεων στο αρχείο ημερολογίου σας.

**Βήμα 3: Δημιουργία στιγμιότυπου CalendarWriter**
```csharp
using (CalendarWriter writer = new CalendarWriter(dataDir, saveOptions))
{
    // Επαναλάβετε και δημιουργήστε πολλά συμβάντα.
    for (int i = 0; i < 10; i++)
    {
        Appointment app = new Appointment(string.Empty, DateTime.Now, DateTime.Now.AddHours(1), "sender@domain.com", "receiver@domain.com");

        // Ορίστε μοναδικές ιδιότητες για κάθε συμβάν.
        app.Description = "Test body " + i;
        app.Summary = "Test summary: " + i;

        // Γράψτε το ραντεβού στο αρχείο .ics χρησιμοποιώντας την παρουσία του writer.
        writer.Write(app);
    }
}
```
Σε αυτόν τον βρόχο, δημιουργούμε δέκα συμβάντα με διάρκεια μίας ώρας. Κάθε ένα `Appointment` έχει μοναδικές περιγραφές και συνόψεις, που δείχνουν πώς μπορείτε να προσαρμόσετε κάθε συμβάν.
