## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να φορτώσετε τα email και να τα αποθηκεύσετε με προσαρμοσμένες επιλογές MHT.

### Φόρτωση μηνύματος ηλεκτρονικού ταχυδρομείου από αρχείο

#### Επισκόπηση
Η φόρτωση μηνυμάτων email είναι απλή με το Aspose.Email. Ξεκινήστε διαβάζοντας ένα `.msg` αρχείο και την προετοιμασία του για μετατροπή.

#### Βήμα 1: Ορισμός διαδρομών και φόρτωση του μηνύματος
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string fileName = "Meeting with Recurring Occurrences.msg";

// Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου από την καθορισμένη διαδρομή αρχείου
MailMessage msg = MailMessage.Load(dataDir + fileName);
```

### Αποθήκευση μηνυμάτων ηλεκτρονικού ταχυδρομείου ως MHTML

#### Επισκόπηση
Η αποθήκευση μηνυμάτων ηλεκτρονικού ταχυδρομείου ως αρχεία MHT διατηρεί το περιεχόμενό τους, συμπεριλαμβανομένων των συνημμένων και της εμπλουτισμένης μορφοποίησης.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης MHT
```csharp
MhtSaveOptions options = new MhtSaveOptions();

// Προσαρμόστε τις επιλογές απόδοσης για κεφαλίδες και συμβάντα ημερολογίου
options.MhtFormatOptions = MhtFormatOptions.WriteHeader | MhtFormatOptions.RenderCalendarEvent;

// Ορίστε προσαρμοσμένα πρότυπα για διάφορες ιδιότητες
options.FormatTemplates[MhtTemplateName.Start] = "<span class='headerLineTitle'>Start:</span><span class='headerLineText'>{0}</span><br/>";
options.FormatTemplates[MhtTemplateName.End] = "<span class='headerLineTitle'>End:</span><span class='headerLineText'>{0}</span><br/>";
options.FormatTemplates[MhtTemplateName.Recurrence] = "<span class='headerLineTitle'>Recurrence:</span><span class='headerLineText'>{0}</span><br/>";
options.FormatTemplates[MhtTemplateName.RecurrencePattern] = "<span class='headerLineTitle'>RecurrencePattern:</span><span class='headerLineText'>{0}</span><br/>";
options.FormatTemplates[MhtTemplateName.Organizer] = "<span class='headerLineTitle'>Organizer:</span><span class='headerLineText'>{0}</span><br/>";
options.FormatTemplates[MhtTemplateName.RequiredAttendees] = "<span class='headerLineTitle'>RequiredAttendees:</span><span class='headerLineText'>{0}</span><br/>";
```

#### Βήμα 3: Αποθηκεύστε το email ως MHTML
```csharp
msg.Save(dataDir + "Meeting with Recurring Occurrences.mhtml", options);
```

Εξερευνήστε περαιτέρω την προσαρμογή για στοιχεία email:

- **Λεπτομέρειες επανάληψης**: Προσαρμόστε την απόδοση των πληροφοριών επανάληψης για σαφήνεια.
- **Διοργανωτής και συμμετέχοντες**: Επισημάνετε τους βασικούς συμμετέχοντες στην έξοδο MHTML για εύκολη αναφορά.

- Βεβαιωθείτε ότι οι διαδρομές αρχείων έχουν καθοριστεί σωστά για να αποφύγετε `FileNotFoundException`.
