## Οδηγός Εφαρμογής
### Μετατροπή EML σε MSG με HTML Body
Αυτή η ενότητα θα σας καθοδηγήσει στη διαδικασία μετατροπής ενός email από μορφή EML σε MSG, διατηρώντας παράλληλα το κυρίως μέρος του ως HTML. Αυτή η λειτουργία είναι ιδιαίτερα χρήσιμη για τη διατήρηση της μορφοποίησης όταν τα email μεταφέρονται μεταξύ διαφορετικών συστημάτων.

#### Βήμα 1: Φόρτωση του αρχείου EML
Ξεκινήστε φορτώνοντας το αρχείο EML σας σε ένα `MailMessage` αντικείμενο. Θα χρειαστεί να καθορίσετε τον κατάλογο που περιέχει το έγγραφό σας:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
MailMessage mailMessage = MailMessage.Load(dataDir + "TestAppointment.eml");
```

#### Βήμα 2: Ορισμός επιλογών μετατροπής
Στη συνέχεια, διαμορφώστε τις επιλογές μετατροπής χρησιμοποιώντας `MapiConversionOptions`Αυτό το βήμα είναι κρίσιμο για να διασφαλίσετε ότι το σώμα του email σας θα παραμείνει σε μορφή HTML:
```csharp
MapiConversionOptions conversionOptions = new MapiConversionOptions();
conversionOptions.Format = OutlookMessageFormat.Unicode;
conversionOptions.ForcedRtfBodyForAppointment = false;  // Χρησιμοποιήστε HTML αντί για RTF
```

#### Βήμα 3: Εκτελέστε τη μετατροπή
Έχοντας ορίσει τις επιλογές σας, μετατρέψτε το `MailMessage` σε ένα `MapiMessage`, εφαρμόζοντας τις καθορισμένες ρυθμίσεις μετατροπής:
```csharp
MapiMessage mapiMessage = MapiMessage.FromMailMessage(mailMessage, conversionOptions);
```

#### Βήμα 4: Αποθήκευση του μετατρεπόμενου αρχείου
Τέλος, αποθηκεύστε το μήνυμα ηλεκτρονικού ταχυδρομείου που έχει μετατραπεί ως αρχείο MSG στην επιθυμητή τοποθεσία:
```csharp
mapiMessage.Save(dataDir + "TestAppointment_out.msg");
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Προβλήματα διαδρομής αρχείου:** Βεβαιωθείτε ότι `dataDir` δείχνει σε έναν έγκυρο κατάλογο.
