## Οδηγός Εφαρμογής

Ας αναλύσουμε τις εργασίες μας: φόρτωση ενός email, εξαγωγή συνημμένων και ανάκτηση ενσωματωμένων εικόνων.

### Φόρτωση αρχείου email

**Επισκόπηση**

Η φόρτωση ενός αρχείου email με το Aspose.Email για .NET είναι απλή. Μπορείτε να φορτώσετε διάφορες μορφές όπως EML απευθείας σε ένα `MailMessage` αντικείμενο.

#### Βήματα για τη φόρτωση ενός email

1. **Ορισμός της διαδρομής καταλόγου**: Καθορίστε πού αποθηκεύονται τα αρχεία email σας.
2. **Φόρτωση του email**: Χρησιμοποιήστε το `MailMessage.Load()` μέθοδος.

```csharp
using Aspose.Email.Mime;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
MailMessage mailMsg = MailMessage.Load(dataDir + "EmailWithAttandEmbedded.eml");
```

Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική διαδρομή προς τα email σας.

### Εξαγωγή συνημμένων από email

**Επισκόπηση**

Μόλις φορτωθεί ένα email, η εξαγωγή συνημμένων γίνεται απλή. Αυτή η λειτουργία σάς επιτρέπει να αποθηκεύσετε κάθε συνημμένο στο δίσκο ή να το επεξεργαστείτε περαιτέρω στη μνήμη.

#### Βήματα για την εξαγωγή συνημμένων

1. **Επανάληψη μέσω συνημμένων**: Επαναλαμβάνω το `mailMsg.Attachments` συλλογή.
2. **Αποθήκευση κάθε συνημμένου**: Χρησιμοποιήστε το `Attachment.Save()` μέθοδος.

```csharp
using System.IO;

foreach (Attachment attachment in mailMsg.Attachments)
{
    string outputPath = "YOUR_OUTPUT_DIRECTORY" + "/" + attachment.Name;
    attachment.Save(outputPath);

    // Προαιρετικά: Αποθηκεύστε τα συνημμένα σε ένα MemoryStream για περαιτέρω επεξεργασία.
    using (MemoryStream ms = new MemoryStream())
    {
        attachment.Save(ms);
    }
}
```

Αντικαθιστώ `'YOUR_OUTPUT_DIRECTORY'` με την επιθυμητή τοποθεσία αποθήκευσης.

### Εξαγωγή ενσωματωμένων εικόνων από email

**Επισκόπηση**

Οι ενσωματωμένες εικόνες, που χρησιμοποιούνται συχνά σε υπογραφές email ή email μάρκετινγκ, μπορούν να εξαχθούν και να αποθηκευτούν ξεχωριστά χρησιμοποιώντας το Aspose.Email.

#### Βήματα για την εξαγωγή ενσωματωμένων εικόνων

1. **Πρόσβαση σε συνδεδεμένους πόρους**: Πλοηγηθείτε στο `mailMsg.LinkedResources` συλλογή.
2. **Αποθήκευση κάθε πόρου**: Χρησιμοποιήστε το `LinkedResource.Save()` μέθοδος.

```csharp
using System.IO;

foreach (LinkedResource lr in mailMsg.LinkedResources)
{
    string outputPath = "YOUR_OUTPUT_DIRECTORY" + "/" + lr.ContentType.Name;
    lr.Save(outputPath);
}
```

Εξασφαλίζω `'YOUR_OUTPUT_DIRECTORY'` έχει οριστεί στο σημείο όπου θέλετε να αποθηκευτούν οι εικόνες.
