## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα περιηγηθούμε στην εφαρμογή δύο βασικών λειτουργιών: τη φόρτωση/αποθήκευση αρχείων EML και την παρακολούθηση της προόδου κατά τη μετατροπή.

### Χαρακτηριστικό 1: Φόρτωση και αποθήκευση αρχείου EML

#### Επισκόπηση

Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο EML σε ένα `MailMessage` αντικείμενο και αποθηκεύστε το ως `MemoryStream`Αυτό είναι κρίσιμο για την επεξεργασία δεδομένων ηλεκτρονικού ταχυδρομείου στις εφαρμογές σας.

#### Βήματα για την εφαρμογή

##### Βήμα 1: Φόρτωση του αρχείου EML

```csharp
using System;
using System.IO;
using Aspose.Email;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
var fileName = Path.Combine(dataDir, "test.eml");

// Φόρτωση του αρχείου EML σε ένα αντικείμενο MailMessage
MailMessage msg = MailMessage.Load(fileName);
```

*Εξήγηση:* Ο `Load` Η μέθοδος διαβάζει το αρχείο EML και το μετατρέπει σε ένα αρχείο που μπορεί να χειριστεί. `MailMessage` αντικείμενο. Αυτό είναι απαραίτητο για τυχόν επόμενες λειτουργίες στα δεδομένα email.

##### Βήμα 2: Αποθήκευση στο MemoryStream

```csharp
using Aspose.Email.Mime;

MemoryStream ms = new MemoryStream();
EmlSaveOptions opt = new EmlSaveOptions(MailMessageSaveType.EmlFormat);

// Αποθήκευση του MailMessage σε ένα MemoryStream με συγκεκριμένες επιλογές
msg.Save(ms, opt);
```

*Εξήγηση:* Εδώ, δημιουργούμε ένα `MemoryStream` και χρήση `EmlSaveOptions` για να καθορίσετε την αποθήκευση σε μορφή EML. Αυτή η μέθοδος σάς επιτρέπει να χειρίζεστε δεδομένα email μέσω προγραμματισμού χωρίς να γράφετε απευθείας στον δίσκο.

### Χαρακτηριστικό 2: Προσαρμοσμένος χειριστής προόδου για μετατροπή EML

#### Επισκόπηση

Η εφαρμογή ενός προσαρμοσμένου χειριστή προόδου βοηθά στην παρακολούθηση της διαδικασίας μετατροπής, παρέχοντας πληροφορίες για κάθε βήμα χειρισμού ενός αρχείου EML.

#### Βήματα για την εφαρμογή

##### Βήμα 1: Ορισμός του χειριστή προόδου

```csharp
using Aspose.Email;

private static void ShowEmlConversionProgress(ProgressEventHandlerInfo info)
{
    int total;
    int saved;

    switch (info.EventType)
    {
        case ProgressEventType.MimeStructureCreated:
            total = info.TotalMimePartCount;
            saved = info.SavedMimePartCount;
            Console.WriteLine("MimeStructureCreated - Total: {0}, Saved: {1}", total, saved);
            break;

        case ProgressEventType.MimePartSaved:
            total = info.TotalMimePartCount;
            saved = info.SavedMimePartCount;
            Console.WriteLine("MimePartSaved - Total: {0}, Saved: {1}", total, saved);
            break;

        case ProgressEventType.SavedToStream:
            total = info.TotalMimePartCount;
            saved = info.SavedMimePartCount;
            Console.WriteLine("SavedToStream - Total: {0}, Saved: {1}", total, saved);
            break;
    }
}
```

*Εξήγηση:* Αυτός ο χειριστής ακούει για συγκεκριμένα συμβάντα (`MimeStructureCreated`, `MimePartSaved`, και `SavedToStream`) κατά τη διάρκεια της διαδικασίας μετατροπής EML. Εξάγει λεπτομέρειες προόδου στην κονσόλα, προσφέροντας παρακολούθηση σε πραγματικό χρόνο.
