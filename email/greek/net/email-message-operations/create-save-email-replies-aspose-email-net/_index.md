## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία δημιουργίας και αποθήκευσης ενός μηνύματος απάντησης.

### Φόρτωση υπάρχοντος μηνύματος MAPI

Ξεκινήστε φορτώνοντας το αρχικό μήνυμα ηλεκτρονικού ταχυδρομείου στο οποίο θέλετε να απαντήσετε χρησιμοποιώντας το `MapiMessage` τάξη:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
MapiMessage originalMsg = MapiMessage.FromFile(dataDir + "/message1.msg");
```

**Εξήγηση:**
Αυτό το βήμα φορτώνει ένα μήνυμα από ένα αρχείο, επιτρέποντάς σας να αποκτήσετε πρόσβαση στο περιεχόμενο και τις ιδιότητές του.

### Αρχικοποίηση του ReplyMessageBuilder

Χρησιμοποιήστε το `ReplyMessageBuilder` τάξη για να κατασκευάσετε την απάντησή σας:

```csharp
ReplyMessageBuilder builder = new ReplyMessageBuilder();
builder.ReplyAll = true; // Ρύθμιση για απάντηση σε όλους τους παραλήπτες.
```

**Εξήγηση:**
Ο `ReplyMessageBuilder` βοηθά στη διαμόρφωση του τρόπου με τον οποίο θέλετε να συντάξετε την απάντησή σας. Εδώ, ο καθορισμός `ReplyAll` να `true` διασφαλίζει ότι η απάντηση αποστέλλεται σε όλους τους παραλήπτες του αρχικού email.

### Ρύθμιση παραμέτρων ιδιοτήτων απάντησης

Ορίστε πρόσθετες ιδιότητες και περιεχόμενο για την απάντησή σας:

```csharp
builder.AdditionMode = OriginalMessageAdditionMode.Textpart;
builder.ResponseText = "<p><b>Dear Friend,</b></p> I want to do is introduce my co-author and co-teacher. <p><a href=\"www.google.com\">This is a first link</a></p><p><a href=\"www.google.com\">This is a second link</a></p>";
```

**Εξήγηση:**
Εδώ, καθορίζετε πώς θα πρέπει να προστεθεί το αρχικό περιεχόμενο του μηνύματος (`Textpart`) και να παρέχετε μια απάντηση σε μορφή HTML.

### Δημιουργήστε το μήνυμα απάντησης

Κατασκευάστε την πραγματική απάντηση χρησιμοποιώντας το εργαλείο δημιουργίας:

```csharp
MapiMessage replyMsg = builder.BuildResponse(originalMsg);
```

**Εξήγηση:**
Αυτή η μέθοδος χρησιμοποιεί το διαμορφωμένο `ReplyMessageBuilder` για να δημιουργήσετε ένα νέο μήνυμα MAPI που θα χρησιμεύσει ως απάντησή σας.

### Αποθήκευση του μηνύματος απάντησης

Τέλος, αποθηκεύστε το κατασκευασμένο μήνυμα σε ένα αρχείο εξόδου:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
replyMsg.Save(outputDir + "/reply_out.msg");
```

**Εξήγηση:**
Αυτό το βήμα γράφει το νεοδημιουργημένο μήνυμα απάντησης σε ένα αρχείο στον καθορισμένο κατάλογο.
