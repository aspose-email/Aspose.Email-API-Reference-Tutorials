## Οδηγός Εφαρμογής

Θα καλύψουμε δύο βασικές λειτουργίες: τον έλεγχο των email που έχουν αποσταλεί και τη διαμόρφωση ενός προγράμματος-πελάτη POP3.

### Λειτουργία 1: Φόρτωση και έλεγχος μηνυμάτων ηλεκτρονικού ταχυδρομείου που έχουν ανακληθεί

#### Επισκόπηση
Προσδιορίστε εάν ένα email έχει απορριφθεί (ανακληθεί) από τον διακομιστή του παραλήπτη, για να διατηρήσετε αποτελεσματικά κανάλια επικοινωνίας.

**Βήμα 1: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου**
Φόρτωση του email από ένα αρχείο:
```csharp
using Aspose.Email;

// Ορίστε εδώ τη διαδρομή του καταλόγου εγγράφων σας
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string dstEmail = dataDir + "test.eml";

// Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου από ένα αρχείο
MailMessage mail = MailMessage.Load(dstEmail);
```

**Βήμα 2: Έλεγχος κατάστασης εγκατάλειψης**
Αξιολογήστε την κατάσταση εγκατάλειψης χρησιμοποιώντας `CheckBounced()`:
```csharp
// Ελέγξτε αν το email έχει ανακληθεί
BounceResult result = mail.CheckBounced();

// Λεπτομέρειες εξόδου σχετικά με την κατάσταση εγκατάλειψης
Console.WriteLine("FileName: " + dstEmail);
Console.WriteLine("Is Bounced : " + result.IsBounced);
Console.WriteLine("Action : " + result.Action);
Console.WriteLine("Recipient : " + result.Recipient);
Console.WriteLine(Environment.NewLine + "Bounce information displayed successfully.");
```

**Εξήγηση:** Ο `CheckBounced()` η μέθοδος επιστρέφει ένα `BounceResult` αντικείμενο με λεπτομέρειες σχετικά με την αναπήδηση, όπως εάν συνέβη και τυχόν ενέργειες που έγιναν.

### Λειτουργία 2: Ρύθμιση παραμέτρων προγράμματος-πελάτη POP3 για ανάκτηση email

#### Επισκόπηση
Ρυθμίστε ένα πρόγραμμα-πελάτη POP3 για την ασφαλή ανάκτηση μηνυμάτων ηλεκτρονικού ταχυδρομείου από τον διακομιστή σας.

**Βήμα 1: Ρύθμιση του προγράμματος-πελάτη POP3**
Ορίστε τις λεπτομέρειες του διακομιστή email και δημιουργήστε ένα `Pop3Client` παράδειγμα:
```csharp
using Aspose.Email.Clients.Pop3;

// Ορίστε εδώ τα στοιχεία του διακομιστή email σας
string host = "your.pop3.host";
int port = 995; // Προεπιλεγμένη θύρα SSL για POP3
bool useSsl = true;
string username = "your_username";
string password = "your_password";

// Δημιουργία και ρύθμιση παραμέτρων του προγράμματος-πελάτη POP3
Pop3Client client = new Pop3Client(host, port, username, password);
client.SecurityOptions = useSsl ? SecurityOptions.Auto : SecurityOptions.None;
```

**Βήμα 2: Σύνδεση με τον διακομιστή**
Δημιουργήστε μια σύνδεση:
```csharp
// Σύνδεση με τον διακομιστή
client.Connect(true);
Console.WriteLine("Connected to POP3 server successfully.");
```

**Βήμα 3: Αποσύνδεση από τον διακομιστή**
Αποσυνδέστε με ασφάλεια μόλις ολοκληρωθεί η διαδικασία:
```csharp
// Αποσύνδεση από τον διακομιστή
client.Disconnect();
Console.WriteLine("Disconnected from POP3 server.");
```

**Εξήγηση:** Ο `Pop3Client` η τάξη διευκολύνει την ασφαλή σύνδεση και την ανάκτηση email. Προσαρμόστε το `SecurityOptions` με βάση τις απαιτήσεις του διακομιστή σας.
