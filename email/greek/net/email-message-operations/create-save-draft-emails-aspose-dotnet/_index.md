## Οδηγός Εφαρμογής

Ας χωρίσουμε τη διαδικασία σε διαχειρίσιμα μέρη για λόγους σαφήνειας.

### Δημιουργία μηνύματος ηλεκτρονικού ταχυδρομείου

Ξεκινήστε δημιουργώντας ένα `MailMessage` παράδειγμα, το οποίο αντιπροσωπεύει το μήνυμα ηλεκτρονικού ταχυδρομείου σας:
```csharp
// Αρχικοποίηση ενός νέου αντικειμένου MailMessage
MailMessage message = new MailMessage();
message.From = "from@domain.com";
message.To.Add("to1@domain.com");
message.To.Add("to2@domain.com");
message.Subject = "New message created by Aspose.Email";
```

#### Ορισμός ιδιοτήτων μηνύματος
Μπορείτε να προσαρμόσετε περαιτέρω το email ορίζοντας ιδιότητες όπως:
- **Σώμα HTML:** Επιτρέπει τη μορφοποίηση εμπλουτισμένου κειμένου.
  ```csharp
  message.IsBodyHtml = true;
  message.HtmlBody = "<b>This line is in bold.</b> <br/> <br/><font color=blue>This line is in blue color</font>";
  ```

### Μετατροπή σε μορφή προσχεδίου
Για να αποθηκεύσετε το email ως μη απεσταλμένο προσχέδιο, μετατρέψτε το χρησιμοποιώντας `MapiMessage`:
```csharp
// Μετατροπή MailMessage σε MapiMessage
MapiMessage mapiMsg = MapiMessage.FromMailMessage(message);

// Ορισμός σημαιών μηνυμάτων για την κατάσταση προσχεδίου
mapiMsg.SetMessageFlags(MapiMessageFlags.MSGFLAG_UNSENT | MapiMessageFlags.MSGFLAG_FROMME);
```

### Αποθήκευση του προσχεδίου email
Τέλος, αποθηκεύστε το email σας ως `.msg` αρχείο για να καθορίσετε ότι είναι προσχέδιο:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string dstEmail = dataDir + "New-Draft.msg";

// Αποθήκευση μηνύματος σε μορφή MSG
mapiMsg.Save(dstEmail);
```

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι οι διαδρομές έχουν καθοριστεί σωστά.
- Επαληθεύστε ότι η βιβλιοθήκη Aspose.Email είναι σωστά εγκατεστημένη και έχει άδεια χρήσης.
