#### Επισκόπηση
Η φόρτωση ενός υπάρχοντος αρχείου EML είναι απλή με το Aspose.Email για .NET. Αυτό το βήμα προετοιμάζει τα δεδομένα του email σας για μετατροπή ή περαιτέρω χειρισμό.

#### Βήματα Υλοποίησης

**Βήμα 1**: Ορίστε τη διαδρομή προς το αρχείο EML σας.
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY/Message.eml";
```

**Βήμα 2**: Φορτώστε το EML σε ένα `MailMessage` αντικείμενο.
```csharp
// Φόρτωση του αρχείου EML σε ένα αντικείμενο MailMessage

```
- **Παράμετροι**: Η διαδρομή προς το αρχείο EML σας.
- **Σκοπός**Μετατρέπει ένα αρχείο EML σε ένα χειρίσιμο αρχείο `MailMessage` αντικείμενο.

### Λειτουργία 2: Αποθήκευση μηνύματος ηλεκτρονικού ταχυδρομείου σε μορφή MSG

#### Επισκόπηση
Μόλις φορτωθεί το EML, μπορείτε να το αποθηκεύσετε ως αρχείο MSG χρησιμοποιώντας τη λειτουργία αποθήκευσης του Aspose.Email.

#### Βήματα Υλοποίησης

**Βήμα 1**Καθορίστε τον κατάλογο εξόδου για το αρχείο MSG σας.
```csharp
string outputPath = "@YOUR_OUTPUT_DIRECTORY/AnEmail_out.msg";
```

**Βήμα 2**: Αποθηκεύστε το μήνυμα email σε μορφή MSG.
```csharp
// Αποθήκευση του μηνύματος ηλεκτρονικού ταχυδρομείου στο δίσκο σε μορφή Unicode (MSG)
eml.Save(outputPath, SaveOptions.DefaultMsgUnicode);
```
- **Παράμετροι**: Η διαδρομή εξόδου και οι επιλογές αποθήκευσης για την κωδικοποίηση Unicode.
- **Σκοπός**: Μετατρέπει και αποθηκεύει το email σας ως αρχείο MSG.
