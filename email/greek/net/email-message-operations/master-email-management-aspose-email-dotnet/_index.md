## Οδηγός Εφαρμογής

### Εισαγωγή αρχείου EML σε αντικείμενο MailMessage

Φορτώστε ένα αρχείο EML σε ένα `MailMessage` αντικείμενο για περαιτέρω επεξεργασία.

#### Οδηγός βήμα προς βήμα:
**1. Καθορίστε τον Κατάλογο Εγγράφων**
Προσδιορίστε πού αποθηκεύονται τα αρχεία EML σας:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string dstEmail = dataDir + "test.eml";
```
*Γιατί;*: Αυτό ορίζει μια διαδρομή αναφοράς για τον εντοπισμό του αρχείου email σας.

**2. Φορτώστε το αρχείο EML**
Χρήση `MailMessage.Load` με `EmlLoadOptions`:
```csharp
using Aspose.Email.Mime;

// Φόρτωση του αρχείου EML σε ένα αντικείμενο MailMessage
MailMessage msg = MailMessage.Load(dstEmail, new EmlLoadOptions());
```
*Γιατί;*Μετατρέπει το αρχείο EML σας σε ένα χειρίσιμο αρχείο `MailMessage` αντικείμενο.

### Ρύθμιση παραμέτρων του SmtpClient για την αποστολή μηνυμάτων ηλεκτρονικού ταχυδρομείου
Η ρύθμιση ενός προγράμματος-πελάτη SMTP είναι απαραίτητη για την αποστολή email από την εφαρμογή σας.

#### Οδηγός βήμα προς βήμα:
**1. Δημιουργία και ρύθμιση παραμέτρων του SmtpClient**
Ρύθμιση με τα κατάλληλα στοιχεία διακομιστή:
```csharp
using Aspose.Email.Clients.Smtp;

SmtpClient client = new SmtpClient("smtp.gmail.com");
client.Port = 587;
client.Username = "your-email@gmail.com";
client.Password = "your-password";
client.SecurityOptions = SecurityOptions.Auto;
```
*Γιατί;*: Διασφαλίζει ότι η εφαρμογή σας μπορεί να συνδεθεί στον διακομιστή SMTP του Gmail και να στείλει email με ασφάλεια.
