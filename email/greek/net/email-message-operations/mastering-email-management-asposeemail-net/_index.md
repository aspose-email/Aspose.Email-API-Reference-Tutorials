## Οδηγός Εφαρμογής

### Δημιουργία και προσθήκη μηνύματος ηλεκτρονικού ταχυδρομείου
Η δημιουργία ενός μηνύματος email και η προσάρτησή του στο γραμματοκιβώτιό σας IMAP είναι απλή με το Aspose.Email. Αυτή η λειτουργία σάς επιτρέπει να αυτοματοποιήσετε την αποστολή ή την οργάνωση email.

#### Επισκόπηση
Σε αυτήν την ενότητα, θα περιγράψουμε πώς να δημιουργήσετε ένα νέο `MailMessage` αντικείμενο και να το προσαρτήσετε στον φάκελο Εισερχόμενα ενός διακομιστή IMAP.

#### Βήμα προς βήμα εφαρμογή
**1. Ρύθμιση του ImapClient**
Ξεκινήστε διαμορφώνοντας το δικό σας `ImapClient` με τα απαραίτητα δικαιολογητικά:
```csharp
ImapClient client = new ImapClient();
client.Host = "imap.gmail.com"; // Χρησιμοποιήστε τον κεντρικό υπολογιστή IMAP εδώ.
client.Username = "your.username@gmail.com";
client.Password = "your.password";
client.Port = 993; // Θύρα SSL για το Gmail
client.SecurityOptions = SecurityOptions.Auto;
```
**2. Δημιουργία και προσθήκη email**
Δημιουργήστε ένα `MailMessage` παράδειγμα με αποστολέα, παραλήπτη, θέμα και σώμα:
```csharp
try
{
    MailMessage message = new MailMessage("user@domain1.com", "user@domain2.com", "subject", "message");
    
    // Προσθήκη του email στον φάκελο Εισερχόμενα
    string uid = client.AppendMessage(ImapFolderInfo.InBox, message);
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```
### Προσθήκη προσαρμοσμένων σημαιών σε μήνυμα ηλεκτρονικού ταχυδρομείου
Οι προσαρμοσμένες σημαίες μπορούν να σας βοηθήσουν να κατηγοριοποιήσετε ή να επισημάνετε τα email για συγκεκριμένες ενέργειες εντός της εφαρμογής σας.

#### Επισκόπηση
Μάθετε πώς να προσθέτετε προσαρμοσμένες σημαίες σε ένα μήνυμα email χρησιμοποιώντας το UID του στο γραμματοκιβώτιο IMAP.

#### Βήμα προς βήμα εφαρμογή
**1. Επιλέξτε Φάκελο Εισερχομένων**
Βεβαιωθείτε ότι ο φάκελος είναι έτοιμος για λειτουργίες σηματοδότησης:
```csharp
client.SelectFolder(ImapFolderInfo.InBox);
```
**2. Προσθήκη σημαιών κατά UID**
Προσθήκη προσαρμοσμένων σημαιών σε ένα συγκεκριμένο μήνυμα που αναγνωρίζεται από το μοναδικό αναγνωριστικό του (UID):
```csharp
try
{
    string uid = "some-unique-message-id";  // Αντικατάσταση με το πραγματικό UID
    
    client.AddMessageFlags(uid, ImapMessageFlags.Keyword("custom1") | ImapMessageFlags.Keyword("custom1_0"));
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```
### Ανάκτηση και έλεγχος προσαρμοσμένων σημαιών σε μηνύματα ηλεκτρονικού ταχυδρομείου
Η ανάκτηση μηνυμάτων για έλεγχο για προσαρμοσμένες σημαίες είναι ζωτικής σημασίας για τη διατήρηση οργανωμένων ροών εργασίας email.

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς μπορείτε να παραθέσετε όλα τα μηνύματα σε έναν φάκελο και να ελέγξετε αν κάποια έχουν συγκεκριμένες λέξεις-κλειδιά που έχουν οριστεί ως σημαίες.

#### Βήμα προς βήμα εφαρμογή
**1. Λίστα όλων των μηνυμάτων**
Επιλέξτε τον φάκελο Εισερχόμενα και ανακτήστε τις πληροφορίες του μηνύματος:
```csharp
client.SelectFolder(ImapFolderInfo.InBox);
ImapMessageInfoCollection messageInfos = client.ListMessages();
```
**2. Ελέγξτε για λέξεις-κλειδιά**
Επαναλάβετε τη διαδικασία μέσα από τα μηνύματα για να βρείτε εκείνα με συγκεκριμένες λέξεις-κλειδιά ως σημαίες:
```csharp
try
{
    foreach (var inf in messageInfos)
    {
        if (inf.ContainsKeyword("custom1"))
        {
            Console.WriteLine("Keyword found");
        }
    }
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```
