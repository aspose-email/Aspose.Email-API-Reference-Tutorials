## Οδηγός Εφαρμογής

### Φόρτωση και αποθήκευση μηνύματος ηλεκτρονικού ταχυδρομείου ως MHTML

Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου από ένα αρχείο και να το αποθηκεύσετε σε μορφή MHTML, διατηρώντας τη δομή και το περιεχόμενό του σε όλες τις πλατφόρμες.

#### Βήμα 1: Ρύθμιση καταλόγων

Αρχικά, ορίστε τους καταλόγους εγγράφων και εξόδου:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου

Φόρτωση μηνύματος ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας `MailMessage.Load()` μέθοδος:
```csharp
MailMessage eml = MailMessage.Load(Path.Combine(documentDirectory, "Attachments.eml"));
```
*Ο παραπάνω κώδικας φορτώνει ένα αρχείο email με το όνομα "Attachments.eml" από τον κατάλογο εγγράφων σας.*

#### Βήμα 3: Αποθήκευση ως MHTML

Ορίστε τις προεπιλεγμένες επιλογές αποθήκευσης MHT και αποθηκεύστε το μήνυμα:
```csharp
MhtSaveOptions opt = SaveOptions.DefaultMhtml;
eml.Save(Path.Combine(outputDirectory, "CustomOrderOfInformationInMHTML_1.mhtml"), opt);
```
*Αυτό αποθηκεύει το email σας σε μορφή MHTML στον καθορισμένο κατάλογο εξόδου.*

### Προσαρμόστε τη σειρά των κεφαλίδων στην έξοδο MHT

Μπορείτε να προσαρμόσετε τον τρόπο εμφάνισης των κεφαλίδων κατά τη μετατροπή των email σε MHT.

#### Βήμα 4: Προσθήκη προσαρμοσμένων κεφαλίδων

Καθορίστε ποιες κεφαλίδες θέλετε και τη σειρά τους:
```csharp
opt.RenderingHeaders.Add(MhtTemplateName.From);
opt.RenderingHeaders.Add(MhtTemplateName.Subject);
opt.RenderingHeaders.Add(MhtTemplateName.To);
opt.RenderingHeaders.Add(MhtTemplateName.Sent);
```
*Η προσθήκη αυτών των κεφαλίδων σάς επιτρέπει να ελέγχετε τη σειρά παρουσίασης στην έξοδο MHT.*

#### Βήμα 5: Αποθήκευση με προσαρμοσμένες κεφαλίδες

Αποθηκεύστε ξανά το email για να εμφανιστούν οι αλλαγές σας:
```csharp
eml.Save(Path.Combine(outputDirectory, "CustomOrderOfInformationInMHTML_2.mhtml"), opt);
```

#### Βήμα 6: Αλλαγή συνόλου κεφαλίδων

Μπορείτε επίσης να αλλάξετε και να επαναφέρετε τις κεφαλίδες για διαφορετικές προβολές:
```csharp
opt.RenderingHeaders.Clear();
opt.RenderingHeaders.Add(MhtTemplateName.Attachments);
opt.RenderingHeaders.Add(MhtTemplateName.Cc);
opt.RenderingHeaders.Add(MhtTemplateName.Subject);

eml.Save(Path.Combine(outputDirectory, "CustomOrderOfInformationInMHTML_3.mhtml"), opt);
```
