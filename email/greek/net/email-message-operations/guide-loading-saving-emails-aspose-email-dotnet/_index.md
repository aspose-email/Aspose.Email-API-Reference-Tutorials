## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία σε δύο κύρια χαρακτηριστικά: τη φόρτωση ενός αρχείου EML και την αποθήκευσή του ως MSG με διατηρημένες ημερομηνίες.

### Χαρακτηριστικό 1: Φόρτωση αρχείου EML

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα υπάρχον αρχείο EML χρησιμοποιώντας το Aspose.Email, επιτρέποντας τον χειρισμό ή την ανάλυση του περιεχομένου του.

**Βήμα προς βήμα εφαρμογή**

##### Βήμα 1: Ορίστε τη διαδρομή καταλόγου
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/";
```

##### Βήμα 2: Φόρτωση του αρχείου EML
Χρησιμοποιώντας το `MailMessage.Load` με τη μέθοδο, καθορίστε τη διαδρομή και τη μορφή του αρχείου email σας:
```csharp
// Αρχικοποιήστε και φορτώστε ένα υπάρχον αρχείο EML καθορίζοντας το MessageFormat
MailMessage eml = MailMessage.Load(dataDir + "Message.eml");
```

- **Παράμετροι:**
  - `dataDir`: Ο κατάλογος που περιέχει τα αρχεία EML σας.
  - `MailMessage.Load()`: Μια μέθοδος που διαβάζει το αρχείο email και επιστρέφει ένα `MailMessage` αντικείμενο.

### Χαρακτηριστικό 2: Αποθήκευση MSG με διατηρημένες ημερομηνίες

#### Επισκόπηση
Αυτή η ενότητα καλύπτει την αποθήκευση ενός αρχείου EML ως αρχείου MSG διατηρώντας παράλληλα τις αρχικές ημερομηνίες αποστολής/λήψης.

**Βήμα προς βήμα εφαρμογή**

##### Βήμα 1: Ορισμός διαδρομής καταλόγου εξόδου
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY/";
```

##### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης
Διαμορφώστε το `MsgSaveOptions` για να διασφαλίσετε ότι οι ημερομηνίες διατηρούνται κατά τη λειτουργία αποθήκευσης:
```csharp
// Ρύθμιση παραμέτρων του MsgSaveOptions για διατήρηση των αρχικών ημερομηνιών κατά τη λειτουργία αποθήκευσης
MsgSaveOptions msgSaveOptions = new MsgSaveOptions(MailMessageSaveType.OutlookMessageFormatUnicode)
{
    PreserveOriginalDates = true // Διασφαλίζει ότι διατηρούνται οι αρχικές ημερομηνίες αποστολής/παραλαβής
};
```

- **Βασικές επιλογές διαμόρφωσης:**
  - `PreserveOriginalDates`: Μια δυαδική σημαία για τη διατήρηση των αρχικών χρονικών σημάνσεων των email.

##### Βήμα 3: Αποθηκεύστε το email ως MSG
Αποθηκεύστε το φορτωμένο αρχείο EML χρησιμοποιώντας τις καθορισμένες επιλογές:
```csharp
// Αποθηκεύστε το MailMessage ως αρχείο MSG με επιλογές διατήρησης ημερομηνιών
eml.Save(Path.Combine(outputDir, "outTest_out.msg"), msgSaveOptions);
```

- **Εξήγηση:** 
  - `Path.Combine()`Συνδυάζει διαδρομές καταλόγων και ονόματα αρχείων.
  - `MailMessage.Save()`Αποθηκεύει το αντικείμενο email στην καθορισμένη μορφή.
