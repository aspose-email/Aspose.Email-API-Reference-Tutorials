## Οδηγός Εφαρμογής

### Δημιουργία και διαμόρφωση ενός MapiTask

**Επισκόπηση:** Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε εργασίες με λεπτομερείς ιδιότητες και να ρυθμίζετε μοτίβα επανάληψης για περιοδικές υπενθυμίσεις.

#### Βήμα 1: Δημιουργήστε ένα νέο MapiTask
Ξεκινήστε δημιουργώντας μια παρουσία του `MapiTask`:
```csharp
using Aspose.Email.Mapi;

// Αρχικοποίηση μιας νέας εργασίας με τίτλο, σώμα, ημερομηνία έναρξης και ημερομηνία λήξης
MapiTask task = new MapiTask("This is test task", "Sample Body", new DateTime(2015, 7, 1), new DateTime(2015, 7, 1));
task.State = MapiTaskState.NotAssigned;
```
**Εξήγηση:** Εδώ, `MapiTask` αρχικοποιείται με τίτλο και κύριο σώμα. Οι ημερομηνίες έναρξης και λήξης έχουν οριστεί αρχικά στην 1η Ιουλίου 2015.

#### Βήμα 2: Ορισμός ετήσιου μοτίβου επανάληψης
Στη συνέχεια, ρυθμίστε την εργασία ώστε να επαναλαμβάνεται ετησίως:
```csharp
// Ορίστε ένα ετήσιο μοτίβο επανάληψης που ξεκινά την 15η ημέρα, επαναλαμβανόμενο κάθε 12 μήνες για 3 εμφανίσεις
var rec = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15,
    Period = 12,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    OccurrenceCount = 3,
};

// Βεβαιωθείτε ότι ο αριθμός εμφανίσεων είναι τουλάχιστον 1 για να αποφύγετε μη έγκυρες ρυθμίσεις παραμέτρων
if (rec.OccurrenceCount == 0)
{
    rec.OccurrenceCount = 1;
}
task.Recurrence = rec;
```
**Εξήγηση:** Αυτό το μπλοκ ορίζει μια ετήσια επανάληψη που ξεκινά στις 15 Ιουλίου, η οποία συμβαίνει κάθε χρόνο για τρεις επαναλήψεις.

### Προσαρμογή ζώνης ώρας

**Επισκόπηση:** Προσαρμόστε τις ημερομηνίες εργασιών σύμφωνα με την τοπική ζώνη ώρας για να διασφαλίσετε τον ακριβή προγραμματισμό σε διαφορετικές περιοχές.

#### Βήμα 3: Λήψη μετατόπισης τοπικής ζώνης ώρας
Προσαρμόζω `DateTime` αντικείμενα που χρησιμοποιούν την τρέχουσα τοπική ζώνη ώρας:
```csharp
using System;

// Ανάκτηση της τρέχουσας ζώνης ώρας και της μετατόπισης UTC
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);

// Προσαρμόστε τις ημερομηνίες προσθέτοντας την τοπική ζώνη ώρας
DateTime StartDate = new DateTime(2015, 7, 1).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 1).Add(ts);
DateTime endByDate = new DateTime(2020, 12, 31).Add(ts);
```
**Εξήγηση:** Αυτός ο κώδικας προσαρμόζει τις ημερομηνίες έναρξης και λήξης των εργασιών ώστε να αντικατοπτρίζουν την τοπική ζώνη ώρας, κάτι που είναι απαραίτητο για εφαρμογές που χρησιμοποιούνται σε διαφορετικές γεωγραφικές τοποθεσίες.
