## Οδηγός Εφαρμογής

### Ορισμός διαδρομών αρχείων
**Επισκόπηση**Ο ορισμός διαδρομών αρχείων διασφαλίζει ότι η εφαρμογή σας γνωρίζει πού να διαβάσει τα αρχεία εισόδου και να αποθηκεύσει τα αποτελέσματα.
```csharp
using System;

string documentDirectory = @"C:\\Your\\Document\\Directory"; // Αντικατάσταση με την πραγματική διαδρομή
string outputDirectory = @"C:\\Your\\Output\\Directory";      // Αντικατάσταση με την πραγματική διαδρομή
```
### Δημιουργία αντικειμένου MapiJournal
**Επισκόπηση**Το αντικείμενο MapiJournal είναι κεντρικό για τη δημιουργία δομημένων καταχωρίσεων ημερολογίου.
```csharp
using Aspose.Email.Mapi;

MapiJournal journal = new MapiJournal(
    subject: "testJournal",
    body: "This is a test journal",
    categories: "Phone call",
    importance: MapiMessageImportance.Normal);

// Ορισμός ωρών έναρξης και λήξης για την καταχώρηση ημερολογίου
journal.StartTime = DateTime.Now;
journal.EndTime = journal.StartTime.AddHours(1);
// Προσθήκη εταιρειών που εμπλέκονται σε αυτήν την καταχώρηση ημερολογίου
timeJournal.Companies = new string[] { "company 1", "company 2", "company 3" };
```
### Προσθήκη συνημμένων στο MapiJournal
**Επισκόπηση**Βελτιώστε τις καταχωρίσεις του ημερολογίου σας επισυνάπτοντας σχετικά αρχεία.
```csharp
using System.IO;

string[] attachFileNames = new string[] {
    documentDirectory + "\\Desert.jpg",
    documentDirectory + "\\download.png"
};

foreach (string attach in attachFileNames)
{
    // Επισύναψη αρχείων στο MapiJournal χρησιμοποιώντας bytes αρχείων
    journal.Attachments.Add(attach, File.ReadAllBytes(attach));
}
```
### Αποθήκευση αντικειμένου MapiJournal
**Επισκόπηση**Τέλος, αποθηκεύστε το διαμορφωμένο αντικείμενο MapiJournal ως αρχείο MSG.
```csharp
journal.Save(outputDirectory + "\\AddAttachmentsToMapiJournal_out.msg");
```
