# Πρόσβαση σε ιδιότητες MAPI σε .NET με το Aspose.Email: Ένας πλήρης οδηγός

## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει την ανάγνωση ιδιοτήτων MAPI με όνομα από ένα συνημμένο ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας το Aspose.Email για .NET.

### Ανάγνωση ονομασμένων ιδιοτήτων MAPI από συνημμένο

Θα δείξουμε πώς να αποκτήσετε πρόσβαση σε συγκεκριμένες ιδιότητες μέσα σε ένα `MapiMessage` αντικείμενο. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Φόρτωση του MapiMessage από ένα αρχείο
Ξεκινήστε φορτώνοντας το αρχείο μηνύματος ηλεκτρονικού ταχυδρομείου σας σε ένα `MapiMessage` αντικείμενο.
```csharp
using System;
using Aspose.Email.Mapi;

namespace EmailFeatures
{
    public class ReadNamedMAPIPropertyFromAttachment
    {
        public static void Run()
        {
            string dataDir = "@YOUR_DOCUMENT_DIRECTORY/message.msg"; // Αντικαταστήστε με τη διαδρομή του αρχείου σας
            MapiMessage msg = MapiMessage.FromFile(dataDir);
```
Ο `FromFile` Η μέθοδος φορτώνει το μήνυμα ηλεκτρονικού ταχυδρομείου στη μνήμη για πρόσβαση στις ιδιότητες.

#### Βήμα 2: Πρόσβαση σε συγκεκριμένες ιδιότητες του μηνύματος
Ανάκτηση ιδιοτήτων όπως το θέμα παρακάτω:
```csharp
            string subject;

            // Προσπάθεια λήψης της ιδιότητας PR_SUBJECT (ANSI)
            MapiProperty prop = msg.Properties[MapiPropertyTag.PR_SUBJECT];

            // Εάν δεν βρεθεί, δοκιμάστε να λάβετε την έκδοση Unicode της ιδιότητας PR_SUBJECT
            if (prop == null)
            {
                prop = msg.Properties[MapiPropertyTag.PR_SUBJECT_W];
            }

            // Ελέγξτε εάν η ιδιότητα του θέματος ανακτήθηκε με επιτυχία
            if (prop != null)
            {
                subject = prop.GetString();
                Console.WriteLine("Subject: " + subject);
            }
            else
            {
                Console.WriteLine("No subject property found!");
                return;
            }
```
Αυτό το τμήμα κώδικα χειρίζεται εκδόσεις ANSI και Unicode μιας ιδιότητας.

#### Βήμα 3: Πρόσβαση σε πρόσθετες ιδιότητες
Ανάκτηση άλλων ιδιοτήτων, όπως το αναγνωριστικό κωδικοσελίδας:
```csharp
            prop = msg.Properties[MapiPropertyTag.PR_INTERNET_CPID];
            if (prop != null)
            {
                int codePage = prop.GetLong();
                Console.WriteLine("Code Page ID: " + codePage);
            }
        }
    }
}
```
Αυτή η ενότητα δείχνει την πρόσβαση στο `PR_INTERNET_CPID` ιδιοκτησίας και ανάκτησης της αξίας της.
