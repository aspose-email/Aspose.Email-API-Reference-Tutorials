## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να δημιουργούμε επαφές MAPI χρησιμοποιώντας το Aspose.Email για .NET. 

### Δημιουργία επαφών MAPI
**Επισκόπηση**Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε μέσω προγραμματισμού πολλές επαφές MAPI, διευκολύνοντας τη διαχείρισή τους μέσα στην εφαρμογή σας.

#### Βήμα 1: Αρχικοποίηση του περιβάλλοντος
Ορίστε τη διαδρομή του καταλόγου σας και αρχικοποιήστε ένα αντικείμενο επαφής:

```csharp
using Aspose.Email.Mapi;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

MapiContact contact1 = new MapiContact("Sebastian Wright");
```

**Εξήγηση**: Το `dataDir` Η μεταβλητή περιέχει την τοποθεσία όπου θα αποθηκεύσετε ή θα ανακτήσετε αρχεία επαφών. Η `MapiContact` Το αντικείμενο αντιπροσωπεύει μία μόνο επαφή.

#### Βήμα 2: Ρύθμιση παραμέτρων ιδιοτήτων επαφής
Προσθέστε λεπτομερείς πληροφορίες στις επαφές σας:

```csharp
contact1.NameInfo = new MapiContactNamePropertySet("Sebastian", "Wright");
contact1.PersonalInfo = new MapiContactPersonalInfoPropertySet();
contact1.PersonalInfo.Title = "Software Engineer";
```

**Εξήγηση**: Το `MapiContactNamePropertySet` και `MapiContactPersonalInfoPropertySet` Οι κλάσεις σάς επιτρέπουν να ορίσετε διάφορες ιδιότητες όπως ονόματα, τίτλους και άλλα.

#### Βήμα 3: Αποθήκευση της επαφής
Τέλος, αποθηκεύστε την επαφή σας στην επιθυμητή μορφή:

```csharp
contact1.Save(dataDir + "SebastianWright.vcf", ContactSaveFormat.VCard);
```

**Εξήγηση**: Το `Save` Η μέθοδος γράφει τα δεδομένα επαφής σε ένα αρχείο. Εδώ, τα αποθηκεύουμε ως αρχείο VCF (vCard).
