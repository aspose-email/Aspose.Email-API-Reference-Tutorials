## Οδηγός Εφαρμογής

Έχοντας έτοιμο το περιβάλλον μας, ας εφαρμόσουμε διάφορα μοτίβα επανάληψης για εργασίες MAPI.

### Δημιουργία και αποθήκευση μιας βασικής εργασίας MAPI

#### Επισκόπηση
Η δημιουργία μιας βασικής εργασίας είναι απλή με το Aspose.Email. Αυτή η ενότητα σας καθοδηγεί στη δημιουργία μιας απλής εργασίας χωρίς κανένα μοτίβο επανάληψης.

#### Βήμα προς βήμα εφαρμογή
**1. Αρχικοποίηση της εργασίας**
Ξεκινήστε δημιουργώντας μια παρουσία του `MapiTask` χρησιμοποιώντας τον κατασκευαστή, ο οποίος απαιτεί λεπτομέρειες όπως θέμα, περιγραφή, ημερομηνία έναρξης και ημερομηνία λήξης:

```csharp
using Aspose.Email.Mapi;

DateTime startDate = new DateTime(2015, 04, 30, 10, 00, 00);
MapiTask task = new MapiTask("abc", "def", startDate, startDate.AddHours(1));
task.State = MapiTaskState.NotAssigned;
```

**2. Αποθήκευση της εργασίας**
Στη συνέχεια, αποθηκεύστε αυτήν την εργασία σε ένα αρχείο σε μορφή MSG χρησιμοποιώντας το `Save` μέθοδος:

```csharp	ask.Save("YOUR_OUTPUT_DIRECTORY\AsposeBasic_out.msg", TaskSaveFormat.Msg);
```

### Προσθήκη ημερήσιας επανάληψης σε μια εργασία MAPI

#### Επισκόπηση
Ορίστε ένα μοτίβο ημερήσιας επανάληψης για την εργασία σας χρησιμοποιώντας `MapiCalendarDailyRecurrencePattern`.

#### Βήμα προς βήμα εφαρμογή
**1. Ορίστε μοτίβο ημερήσιας επανάληψης**
Ρυθμίστε την επανάληψη αρχικοποιώντας `MapiCalendarDailyRecurrencePattern`:

```csharp
var dailyRecurrence = new MapiCalendarDailyRecurrencePattern
{
    PatternType = MapiCalendarRecurrencePatternType.Day,
    Period = 1, // Κάθε μέρα
    WeekStartDay = DayOfWeek.Sunday,
    EndType = MapiCalendarRecurrenceEndType.NeverEnd,
    OccurrenceCount = 0
};
task.Recurrence = dailyRecurrence;
```

**2. Αποθήκευση της εργασίας με επανάληψη**
Αποθηκεύστε το όπως μια βασική εργασία:

```csharp	ask.Save("YOUR_OUTPUT_DIRECTORY\AsposeDaily_out.msg", TaskSaveFormat.Msg);
```

### Προσθήκη εβδομαδιαίας επανάληψης σε μια εργασία MAPI

#### Επισκόπηση
Οι εβδομαδιαίες εργασίες είναι συνηθισμένες για συσκέψεις ή επαναλαμβανόμενες εκδηλώσεις και το Aspose.Email απλοποιεί αυτήν τη διαδικασία.

#### Βήμα προς βήμα εφαρμογή
**1. Ορισμός εβδομαδιαίου μοτίβου επανάληψης**
Ρυθμίστε την επανάληψη χρησιμοποιώντας `MapiCalendarWeeklyRecurrencePattern`:

```csharp
var weeklyRecurrence = new MapiCalendarWeeklyRecurrencePattern
{
    PatternType = MapiCalendarRecurrencePatternType.Week,
    Period = 1, // Κάθε εβδομάδα
    DayOfWeek = MapiCalendarDayOfWeek.Wednesday,
    EndType = MapiCalendarRecurrenceEndType.NeverEnd,
    OccurrenceCount = 0
};
task.Recurrence = weeklyRecurrence;
```

**2. Αποθήκευση της εργασίας**
```csharp	ask.Save("YOUR_OUTPUT_DIRECTORY\AsposeWeekly_out.msg", TaskSaveFormat.Msg);
```

### Προσθήκη μηνιαίας επανάληψης σε μια εργασία MAPI

#### Επισκόπηση
Οι μηνιαίες εργασίες μπορούν να ρυθμιστούν ώστε να επαναλαμβάνονται σε συγκεκριμένες ημέρες κάθε μήνα.

#### Βήμα προς βήμα εφαρμογή
**1. Ρύθμιση μηνιαίας επανάληψης**
Χρήση `MapiCalendarMonthlyRecurrencePattern`:

```csharp
var monthlyRecurrence = new MapiCalendarMonthlyRecurrencePattern
{
    PatternType = MapiCalendarRecurrencePatternType.Month,
    Period = 1, // Κάθε μήνα
    EndType = MapiCalendarRecurrenceEndType.NeverEnd,
    Day = 30, // Επανάληψη στις 30
    OccurrenceCount = 0,
    WeekStartDay = DayOfWeek.Sunday
};
task.Recurrence = monthlyRecurrence;
```

**2. Αποθήκευση της εργασίας**
```csharp	ask.Save("YOUR_OUTPUT_DIRECTORY\AsposeMonthly_out.msg", TaskSaveFormat.Msg);
```

### Προσθήκη ετήσιας επανάληψης σε μια εργασία MAPI

#### Επισκόπηση
Η ετήσια επανάληψη είναι ιδανική για ετήσιες εκδηλώσεις ή υπενθυμίσεις.

#### Βήμα προς βήμα εφαρμογή
**1. Ρύθμιση ετήσιας επανάληψης**
Προσαρμόστε το μοτίβο επανάληψης χρησιμοποιώντας `MapiCalendarMonthlyRecurrencePattern`:

```csharp
var yearlyRecurrence = new MapiCalendarMonthlyRecurrencePattern
{
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.NeverEnd,
    OccurrenceCount = 10, // Επαναλαμβάνεται για δέκα χρόνια
    Period = 12 // Κάθε χρόνο
};
task.Recurrence = yearlyRecurrence;
```

**2. Αποθήκευση της εργασίας**
```csharp	ask.Save("YOUR_OUTPUT_DIRECTORY\AsposeYearly_out.msg", TaskSaveFormat.Msg);
```
