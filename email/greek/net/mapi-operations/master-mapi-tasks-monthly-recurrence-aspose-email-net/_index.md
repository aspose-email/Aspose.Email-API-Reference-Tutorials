## Οδηγός Εφαρμογής

### Δημιουργία και αποθήκευση μιας εργασίας MAPI
Ας ξεκινήσουμε δημιουργώντας μια απλή εργασία MAPI και αποθηκεύοντάς την ως αρχείο MSG. Αυτή η λειτουργικότητα βοηθά στην αυτοματοποίηση της δημιουργίας εργασιών, διασφαλίζοντας συνέπεια και αποτελεσματικότητα.

**Βήμα 1: Ορισμός ιδιοτήτων εργασίας**
Ξεκινήστε ορίζοντας τις ημερομηνίες έναρξης και λήξης για την εργασία σας:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);
DateTime StartDate = new DateTime(2015, 7, 1).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 1).Add(ts);
```

**Βήμα 2: Δημιουργήστε την εργασία MAPI**
Αρχικοποίηση ενός `MapiTask` με τις καθορισμένες ιδιότητές σας:
```csharp
MapiTask task = new MapiTask("This is test task", "Sample Body", StartDate, DueDate);
task.State = MapiTaskState.NotAssigned;
```
Σε αυτό το απόσπασμα:
- Τα "Αυτή είναι δοκιμαστική εργασία" και "Δείγμα σώματος" είναι το θέμα και το σώμα της εργασίας.
- `StartDate` και `DueDate` καθορίστε πότε ξεκινά και πότε τελειώνει η εργασία.

**Βήμα 3: Αποθήκευση της εργασίας**
Αποθηκεύστε την εργασία σας σε μορφή MSG:
```csharp
task.Save(dataDir + "Monthly_out.msg", TaskSaveFormat.Msg);
```

### Ρύθμιση παραμέτρων μηνιαίου μοτίβου επανάληψης για μια εργασία MAPI
Στη συνέχεια, ορίστε ένα μηνιαίο μοτίβο επανάληψης σε ένα υπάρχον αντικείμενο εργασίας MAPI. Αυτό είναι ιδανικό για εργασίες που πρέπει να επαναλαμβάνονται σε τακτά χρονικά διαστήματα.

**Βήμα 1: Ορίστε το μοτίβο επανάληψης**
Δημιουργήστε ένα `MapiCalendarMonthlyRecurrencePattern`:
```csharp
var rec = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15,
    Period = 12,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    OccurrenceCount = 3,
    WeekStartDay = DayOfWeek.Monday
};
```
Αυτή η ρύθμιση παραμέτρων ορίζει την εργασία να επαναλαμβάνεται στις 15 κάθε μήνα, τρεις φορές σε μια περίοδο 12 μηνών.

**Βήμα 2: Εφαρμογή επανάληψης στην εργασία**
Αντιστοιχίστε το μοτίβο επανάληψης στο δικό σας `MapiTask`:
```csharp
MapiTask taskWithRecurrence = new MapiTask("This is test task", "Sample Body", DateTime.Now, DateTime.Now.AddDays(30));
taskWithRecurrence.Recurrence = rec;
```

**Βήμα 3: Αποθήκευση της εργασίας με επανάληψη**
Αποθηκεύστε την επαναλαμβανόμενη εργασία σας ως αρχείο MSG:
```csharp
string dataDirOutput = "YOUR_OUTPUT_DIRECTORY";
taskWithRecurrence.Save(dataDirOutput + "Monthly_out_with_recurrence.msg", TaskSaveFormat.Msg);
```
