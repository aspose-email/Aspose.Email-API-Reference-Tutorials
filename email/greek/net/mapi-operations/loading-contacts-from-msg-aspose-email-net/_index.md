### Φόρτωση επαφών από αρχείο MSG

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να εξαγάγετε στοιχεία επικοινωνίας που είναι αποθηκευμένα σε ένα αρχείο MSG του Outlook, μετατρέποντάς το σε αντικείμενο MapiContact.

#### Βήματα Υλοποίησης
**Βήμα 1: Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας**
Πριν από τη φόρτωση οποιωνδήποτε αρχείων, καθορίστε πού βρίσκονται τα αρχεία MSG:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική διαδρομή προς τα αρχεία σας.

**Βήμα 2: Φόρτωση του αρχείου MSG σε ένα αντικείμενο MapiMessage**
Για να διαβάσετε ένα αρχείο MSG, χρησιμοποιήστε το `MapiMessage.FromFile` μέθοδος. Αυτό το βήμα περιλαμβάνει τον καθορισμό της διαδρομής του αρχείου και τη φόρτωσή της:
```csharp
// Φορτώστε ένα αρχείο MSG στο αντικείμενο MapiMessage.
MapiMessage msg = MapiMessage.FromFile(dataDir + "/Contact.msg");
```
**Βήμα 3: Μετατροπή MapiMessage σε MapiContact**
Μόλις φορτωθεί το αρχείο MSG, μετατρέψτε το σε `MapiContact` αντικείμενο για ευκολότερη πρόσβαση στα στοιχεία επικοινωνίας:
```csharp
// Μετατρέψτε το αντικείμενο MapiMessage σε MapiContact.
MapiContact mapiContact = (MapiContact)msg.ToMapiMessageItem();
```
