## Οδηγός Εφαρμογής

### Δημιουργία ενός MapiTask

#### Επισκόπηση

Η δημιουργία ενός MapiTask περιλαμβάνει τον ορισμό ιδιοτήτων όπως τίτλος, περιγραφή, ημερομηνία έναρξης και ημερομηνία λήξης.

#### Βήμα προς βήμα εφαρμογή

**Ορισμός λεπτομερειών εργασίας**
```csharp
using Aspose.Email.Mapi;
using System;

public static void CreateMapiTask()
{
    // Ορίστε λεπτομέρειες εργασίας με την ημερομηνία έναρξης και την ημερομηνία λήξης
    DateTime StartDate = new DateTime(2023, 10, 1);
    DateTime DueDate = new DateTime(2023, 10, 2);

    // Δημιουργήστε ένα MapiTask με τίτλο, σώμα, ημερομηνία έναρξης και ημερομηνία λήξης
    MapiTask task = new MapiTask("Daily Report", "Prepare the daily report", StartDate, DueDate);

    // Ορισμός της αρχικής κατάστασης της εργασίας ως Μη Αντιστοιχισμένης
    task.State = MapiTaskState.NotAssigned;
}
```
**Εξήγηση**: Το `MapiTask` Ο κατασκευαστής δέχεται παραμέτρους για τον τίτλο και την περιγραφή μαζί με τις ημερομηνίες έναρξης και λήξης. Ορίζοντας το `State` να `NotAssigned` υποδεικνύει ότι η εργασία δεν έχει ανατεθεί ακόμη.

### Ρύθμιση ημερήσιας επανάληψης για μια εργασία

#### Επισκόπηση

Για εργασίες που απαιτούν επανάληψη, όπως οι καθημερινές υπενθυμίσεις, η δημιουργία ενός μοτίβου επανάληψης είναι απαραίτητη.

#### Βήμα προς βήμα εφαρμογή

**Ορισμός και ανάθεση μοτίβου επανάληψης**
```csharp
public static void SetDailyRecurrence()
{
    // Ορίστε ημερομηνίες έναρξης και λήξης εργασιών
    DateTime StartDate = new DateTime(2023, 10, 1);
    DateTime DueDate = new DateTime(2023, 10, 2);

    // Δημιουργήστε μια παρουσία MapiTask
    MapiTask task = new MapiTask("Daily Report", "Prepare the daily report", StartDate, DueDate);

    // Ρύθμιση παραμέτρων ημερήσιας επανάληψης
    var record = new MapiCalendarDailyRecurrencePattern
    {
        PatternType = MapiCalendarRecurrencePatternType.Day,
        Period = 1,
        WeekStartDay = DayOfWeek.Sunday,
        EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
        OccurrenceCount = 5 // Η εργασία θα εκτελεστεί πέντε φορές
    };

    // Αντιστοίχιση του μοτίβου επανάληψης στην εργασία
    task.Recurrence = record;
}
```
**Εξήγηση**: Το `MapiCalendarDailyRecurrencePattern` Η κλάση σάς επιτρέπει να καθορίσετε πόσο συχνά επαναλαμβάνεται μια εργασία. Εδώ, έχει οριστεί να επαναλαμβάνεται καθημερινά (`Period = 1`) για πέντε εμφανίσεις.

### Αποθήκευση εργασίας ως αρχείο MSG

#### Επισκόπηση

Η αποθήκευση του MapiTask ως αρχείο .msg επιτρέπει την εύκολη διανομή και αρχειοθέτηση εργασιών.

#### Βήμα προς βήμα εφαρμογή

**Αποθήκευση MapiTask**
```csharp
public static void SaveTaskAsMsg()
{
    // Ορισμός λεπτομερειών εργασίας με μοτίβο επανάληψης
    DateTime StartDate = new DateTime(2023, 10, 1);
    DateTime DueDate = new DateTime(2023, 10, 2);

    MapiTask task = new MapiTask("Daily Report", "Prepare the daily report", StartDate, DueDate);
    
    var record = new MapiCalendarDailyRecurrencePattern
    {
        PatternType = MapiCalendarRecurrencePatternType.Day,
        Period = 1,
        WeekStartDay = DayOfWeek.Sunday,
        EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
        OccurrenceCount = 5
    };

    task.Recurrence = record;

    // Ορίστε τη διαδρομή αρχείου για αποθήκευση
    string outputDir = "YOUR_OUTPUT_DIRECTORY";

    // Αποθηκεύστε το MapiTask ως αρχείο MSG
    task.Save(outputDir + "/DailyReport_out.msg", TaskSaveFormat.Msg);
}
```
**Εξήγηση**: Το `Save` Η μέθοδος γράφει το MapiTask σε μια καθορισμένη διαδρομή σε μορφή MSG, η οποία είναι συμβατή με προγράμματα-πελάτες ηλεκτρονικού ταχυδρομείου όπως το Outlook.
