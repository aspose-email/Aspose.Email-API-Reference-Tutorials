## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα περιηγηθούμε στη διαδικασία ανάκτησης ιδιοτήτων MAPI από ένα αρχείο MSG.

### Φόρτωση και πρόσβαση σε ιδιότητες
#### Επισκόπηση
Θα φορτώσουμε ένα αρχείο MSG και θα έχουμε πρόσβαση σε αυτό. `PR_SUBJECT` ιδιότητα. Εάν δεν είναι διαθέσιμη σε μορφή ANSI, επιστρέφουμε στην έκδοση Unicode (`PR_SUBJECT_W`).

**Φόρτωση του αρχείου MSG**

```csharp
// Φόρτωση ενός αρχείου MSG από την καθορισμένη τοποθεσία
MapiMessage msg = MapiMessage.FromFile(dataDir + "/message.msg");
```

**Πρόσβαση στην ιδιότητα PR_SUBJECT**

```csharp
// Αποκτήστε πρόσβαση στην ιδιότητα PR_SUBJECT, η οποία περιέχει το θέμα του email
MapiProperty prop = msg.Properties[MapiPropertyTag.PR_SUBJECT];
```

**Εφεδρική ιδιότητα Unicode**

Αν `PR_SUBJECT` είναι null, ανακτήστε τον ομότιμο Unicode:

```csharp
if (prop == null)
{
    prop = msg.Properties[MapiPropertyTag.PR_SUBJECT_W];
}
```

### Εξήγηση
- **MapiMessage.FromFile**Αυτή η μέθοδος φορτώνει το αρχείο MSG από τον καθορισμένο κατάλογο. Βεβαιωθείτε ότι η διαδρομή είναι σωστή για να αποφύγετε εξαιρέσεις.
  
- **Λεξικό Ιδιοτήτων**Πρόσβαση στις ιδιότητες MAPI χρησιμοποιώντας `MapiPropertyTag`Εάν δεν βρεθεί κάποια ιδιότητα, ελέγξτε το αντίστοιχο Unicode για ευρύτερη συμβατότητα.

**Συμβουλές αντιμετώπισης προβλημάτων**
- **Προβλήματα διαδρομής αρχείου**Ελέγξτε ξανά τις διαδρομές των αρχείων σας και βεβαιωθείτε ότι έχουν μορφοποιηθεί σωστά.
- **Μηδενικές τιμές ιδιοτήτων**Πάντα να επαληθεύετε εάν η ιδιότητα που επιστρέφεται είναι null πριν αποκτήσετε πρόσβαση στην τιμή της, για να αποτρέψετε σφάλματα χρόνου εκτέλεσης.
