## Οδηγός Εφαρμογής

### Δυνατότητα 1: Φόρτωση και αποθήκευση μηνυμάτων MAPI με συνημμένα

Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα μήνυμα MAPI από ένα αρχείο και να το αποθηκεύσετε με συνημμένα χρησιμοποιώντας το Aspose.Email για .NET.

#### Επισκόπηση
Ο σκοπός αυτής της λειτουργίας είναι η διαχείριση μηνυμάτων MAPI φορτώνοντάς τα στην εφαρμογή σας, αποθηκεύοντάς τα όπως απαιτείται και διασφαλίζοντας ότι όλα τα συνημμένα είναι άθικτα.

#### Βήμα 1: Φόρτωση μηνύματος MAPI από αρχείο
```csharp
using Aspose.Email.Mapi;
using System;

class FeatureLoadAndSaveMAPI
{
    public static void Run()
    {
        // Ορίστε τη διαδρομή καταλόγου όπου βρίσκεται το αρχείο εισόδου
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // Ενημερώστε αυτό με τον πραγματικό κατάλογο εγγράφων σας

        // Φόρτωση μηνύματος MAPI από αρχείο.
        MapiMessage msg = MapiMessage.FromFile(dataDir + "/MsgWithAtt.msg");
        
        Console.WriteLine("MAPI message loaded successfully.");
    }
}
```
**Εξήγηση:** Αυτό το τμήμα κώδικα φορτώνει το μήνυμα MAPI από έναν καθορισμένο κατάλογο. Βεβαιωθείτε ότι `dataDir` έχει ρυθμιστεί σωστά στο περιβάλλον σας.

#### Βήμα 2: Αποθήκευση του φορτωμένου μηνύματος MAPI με συνημμένα
```csharp
public static void Run()
{
    // Ορίστε τη διαδρομή καταλόγου όπου βρίσκεται το αρχείο εισόδου
    string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // Ενημερώστε αυτό με τον πραγματικό κατάλογο εγγράφων σας

    // Φόρτωση μηνύματος MAPI από αρχείο.
    MapiMessage msg = MapiMessage.FromFile(dataDir + "/MsgWithAtt.msg");

    // Αποθηκεύστε το φορτωμένο μήνυμα MAPI σε ένα άλλο αρχείο με συνημμένα.
    string outputFilePath = dataDir + "/AttachmentsToDestroy_out.msg";
    msg.Save(outputFilePath);

    Console.WriteLine("MAPI message saved successfully.");
}
```
**Εξήγηση:** Εδώ, αποθηκεύουμε το φορτωμένο μήνυμα σε ένα νέο αρχείο. Αυτό διασφαλίζει ότι όλα τα συνημμένα θα διατηρηθούν κατά τη διάρκεια της διαδικασίας αποθήκευσης.

### Δυνατότητα 2: Καταστροφή συνημμένων σε μήνυμα MAPI

Αυτή η λειτουργία δείχνει πώς να καταργήσετε αποτελεσματικά όλα τα συνημμένα από ένα καθορισμένο αρχείο μηνύματος MAPI.

#### Επισκόπηση
Η αφαίρεση περιττών συνημμένων μπορεί να σας βοηθήσει να βελτιστοποιήσετε τη διαχείριση των email σας και να μειώσετε τις απαιτήσεις αποθήκευσης.

#### Βήμα 1: Καθορίστε το αρχείο με τα συνημμένα
```csharp
using Aspose.Email.Mapi;
using System;

class FeatureDestroyAttachments
{
    public static void Run()
    {
        // Ορίστε τη διαδρομή καταλόγου όπου βρίσκεται το αρχείο εξόδου
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // Ενημερώστε αυτό με τον πραγματικό κατάλογο εγγράφων σας

        // Καθορίστε το αρχείο μηνύματος MAPI από το οποίο θα καταστραφούν τα συνημμένα.
        string filePath = dataDir + "/AttachmentsToDestroy_out.msg";
        
        Console.WriteLine("File specified for attachment removal.");
    }
}
```
**Εξήγηση:** Αυτό το βήμα ορίζει τη διαδρομή προς το αρχείο προορισμού σας, διασφαλίζοντας ότι εργάζεστε με το σωστό αρχείο.

#### Βήμα 2: Αφαίρεση όλων των συνημμένων από το αρχείο
```csharp
public static void Run()
{
    // Ορίστε τη διαδρομή καταλόγου όπου βρίσκεται το αρχείο εξόδου
    string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // Ενημερώστε αυτό με τον πραγματικό κατάλογο εγγράφων σας

    // Καθορίστε το αρχείο μηνύματος MAPI από το οποίο θα καταστραφούν τα συνημμένα.
    string filePath = dataDir + "/AttachmentsToDestroy_out.msg";
    
    // Καλέστε τη στατική μέθοδο για να αφαιρέσετε όλα τα συνημμένα από το καθορισμένο αρχείο.
    MapiMessage.DestroyAttachments(filePath);

    Console.WriteLine("All attachments removed successfully.");
}
```
**Εξήγηση:** Ο `MapiMessage.DestroyAttachments` Η μέθοδος διαγράφει αποτελεσματικά όλα τα συνημμένα, διασφαλίζοντας ότι το μήνυμά σας είναι καθαρό και βελτιστοποιημένο.
