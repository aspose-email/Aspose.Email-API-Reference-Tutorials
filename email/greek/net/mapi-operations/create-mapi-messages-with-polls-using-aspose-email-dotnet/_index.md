## Οδηγός Εφαρμογής

### Δυνατότητα: Δημιουργία και αποθήκευση μηνύματος MAPI με δημοσκόπηση

Αυτή η λειτουργία σάς επιτρέπει να δημιουργήσετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας το Aspose.Email για .NET, να το διαμορφώσετε με επιλογές δημοσκόπησης και να το αποθηκεύσετε ως αρχείο.

#### Επισκόπηση
Θα μάθετε πώς να:
- Δημιουργήστε ένα βασικό μήνυμα MAPI.
- Ρυθμίστε κουμπιά ψηφοφορίας μέσα στο email.
- Αποθηκεύστε το διαμορφωμένο μήνυμα στην επιθυμητή θέση.

#### Βήματα Υλοποίησης

##### Βήμα 1: Ορισμός καταλόγου εξόδου
Ξεκινήστε καθορίζοντας πού θέλετε να αποθηκεύσετε το αρχείο εξόδου. Αντικαταστήστε `"YOUR_OUTPUT_DIRECTORY"` με μια πραγματική διαδρομή στο μηχάνημά σας.
```csharp
string dataDir = "YOUR_OUTPUT_DIRECTORY";
```

##### Βήμα 2: Δημιουργήστε ένα δοκιμαστικό μήνυμα MAPI
Δημιουργήστε την αρχική δομή του μηνύματος χρησιμοποιώντας προκαθορισμένες λεπτομέρειες αποστολέα, παραλήπτη, θέματος και σώματος μηνύματος.
```csharp
private static MapiMessage CreateTestMessage(bool draft)
{
    MapiMessage msg = new MapiMessage(
        "from@test.com",
        "to@test.com",
        "Flagged Message",
        "Make it nice and short, but descriptive. The description may appear in search engines' search results pages..."
    );

    if (!draft)
    {
        msg.SetMessageFlags(msg.Flags ^ MapiMessageFlags.MSGFLAG_UNSENT);
    }

    return msg;
}
```
*Εξήγηση*Αυτή η μέθοδος κατασκευάζει ένα `MapiMessage` αντικείμενο με λεπτομέρειες email και προαιρετικά ορίζει το μήνυμα ως απεσταλμένο.

##### Βήμα 3: Ρύθμιση επιλογών δημοσκόπησης
Διαμορφώστε τη δημοσκόπηση ορίζοντας κουμπιά ψηφοφορίας. Εδώ, χρησιμοποιούμε ως επιλογές τα "Ναι", "Όχι", "Ίσως" και "Ακριβώς!".
```csharp
FollowUpOptions options = new FollowUpOptions();
options.VotingButtons = "Yes;No;Maybe;Exactly!";
```

##### Βήμα 4: Εφαρμογή επιλογών παρακολούθησης στο μήνυμα
Συνδέστε τη διαμόρφωση της δημοσκόπησής σας με το μήνυμα χρησιμοποιώντας `FollowUpManager`.
```csharp
FollowUpManager.SetOptions(msg, options);
```

##### Βήμα 5: Αποθήκευση του μηνύματος MAPI σε αρχείο
Τέλος, αποθηκεύστε το διαμορφωμένο μήνυμα σε ένα αρχείο στον καθορισμένο κατάλογο.
```csharp
msg.Save(dataDir + "/MapiMsgWithPoll.msg");
```

**Συμβουλές αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι όλες οι διαδρομές έχουν οριστεί σωστά και έχουν τα κατάλληλα δικαιώματα. Εάν αντιμετωπίσετε προβλήματα με την αποθήκευση αρχείων, επαληθεύστε ότι ο κατάλογος υπάρχει ή δημιουργήστε τον μέσω προγραμματισμού.
