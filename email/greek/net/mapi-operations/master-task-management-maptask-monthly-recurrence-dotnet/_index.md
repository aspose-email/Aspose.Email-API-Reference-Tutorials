## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει τη δημιουργία ενός MapiTask με προσαρμογές ημερομηνίας και τη ρύθμιση μηνιαίας επανάληψης.

### Δημιουργία MapiTask με προσαρμογές ημερομηνίας

Δημιουργήστε εργασίες που σέβονται τις ρυθμίσεις τοπικής ζώνης ώρας ακολουθώντας τα παρακάτω βήματα:

#### Βήμα 1: Ορίστε τις λεπτομέρειες της εργασίας σας

Ξεκινήστε ορίζοντας placeholders για καταλόγους, ανακτώντας την τρέχουσα ζώνη ώρας και υπολογίζοντας την τοπική χρονική μετατόπιση:

```csharp
using Aspose.Email.Mapi;
using System;

public class Feature1
{
    public static void Run()
    {
        string documentDirectory = "@YOUR_DOCUMENT_DIRECTORY";
        string outputDirectory = "@YOUR_OUTPUT_DIRECTORY";

        TimeZone localZone = TimeZone.CurrentTimeZone;
        TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);

        DateTime startDate = new DateTime(2015, 7, 1).Add(ts);
        DateTime dueDate = new DateTime(2015, 7, 1).Add(ts);

        MapiTask task = new MapiTask("This is a test task", "Sample Body", startDate, dueDate);
        
        task.State = MapiTaskState.NotAssigned;
    }
}
```

**Εξήγηση:**
- Ο `TimeZone.CurrentTimeZone.GetUtcOffset` Η μέθοδος υπολογίζει την τοπική χρονική μετατόπιση για να προσαρμόσει ανάλογα τις ημερομηνίες έναρξης και λήξης της εργασίας σας.
- Ρύθμιση του `MapiTask.State` Η ιδιότητα ορίζει την τρέχουσα κατάσταση της εργασίας σας.

### Ρύθμιση παραμέτρων μηνιαίας επανάληψης για μια εργασία

Οι εργασίες συχνά απαιτούν μοτίβα επανάληψης. Ρυθμίστε μια μηνιαία επανάληψη που δεν τελειώνει ποτέ με αυτά τα βήματα:

#### Βήμα 2: Ορισμός μοτίβου επανάληψης

Δημιουργήστε μια παρουσία του `MapiCalendarMonthlyRecurrencePattern` για να διασφαλιστεί ότι επαναλαμβάνεται κάθε μήνα επ' αόριστον:

```csharp
using Aspose.Email.Mapi;

public class Feature2
{
    public static void Run()
    {
        var recurrence = new MapiCalendarMonthlyRecurrencePattern
        {
            Day = 15,                  // Επαναλαμβάνεται στις 15 κάθε μήνα
            Period = 1,                // Κάθε μήνα
            PatternType = MapiCalendarRecurrencePatternType.Month,
            EndType = MapiCalendarRecurrenceEndType.NeverEnd,
            WeekStartDay = DayOfWeek.Monday
        };
    
        // Παράδειγμα χρήσης:
        // MapiTask task = new MapiTask("Δείγμα Εργασίας", "Σώμα", Ημερομηνία Έναρξης, Ημερομηνία Παράδοσης);
        // task.Recurrence = επανάληψη;
    }
}
```

**Εξήγηση:**
- Ο `Day` Η ιδιότητα καθορίζει την ημέρα του μήνα κατά την οποία επαναλαμβάνεται η εργασία.
- Σύνθεση `EndType` να `NeverEnd` διασφαλίζει ότι αυτό το μοτίβο θα συνεχιστεί επ' αόριστον.
