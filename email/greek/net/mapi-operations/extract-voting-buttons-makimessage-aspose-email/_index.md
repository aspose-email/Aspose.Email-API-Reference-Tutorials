## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Κουμπιά Ψηφοφορίας Μόνο για Ανάγνωση

Αυτή η λειτουργία εστιάζει στην εξαγωγή κουμπιών ψηφοφορίας από αντικείμενα MapiMessage. Ας αναλύσουμε κάθε βήμα.

#### Βήμα 1: Προετοιμάστε το περιβάλλον σας

Αρχικά, βεβαιωθείτε ότι το έργο σας περιλαμβάνει τους απαραίτητους χώρους ονομάτων:

```csharp
using Aspose.Email.Mapi;
using System.Collections.Generic;
using System.IO;
```

#### Βήμα 2: Φόρτωση του MapiMessage από μια ροή αρχείων

Ξεκινήστε φορτώνοντας ένα αρχείο μηνύματος που περιέχει κουμπιά ψηφοφορίας σε μια ροή μνήμης.

```csharp
string fileName = @"YOUR_DOCUMENT_DIRECTORY\MessageWithVotingButtons.msg";

try
{
    using (MemoryStream ms = new MemoryStream(File.ReadAllBytes(fileName)))
    {
        // Φόρτωση του MapiMessage από τη ροή
        MapiMessage testMsg = MapiMessage.FromStream(ms);
        
        // Η μεταβλητή 'testMsg' περιέχει πλέον το αντικείμενο μηνύματός σας
    }
}
catch (IOException e)
{
    Console.WriteLine("An error occurred while reading the message file: " + e.Message);
}
```

**Γιατί αυτό το βήμα;**Η φόρτωση μηνυμάτων στη μνήμη επιτρέπει τον χειρισμό και την εξαγωγή δεδομένων χωρίς άμεση αλληλεπίδραση με το σύστημα αρχείων, βελτιώνοντας τόσο την απόδοση όσο και την ασφάλεια.

#### Βήμα 3: Εξαγωγή κουμπιών ψηφοφορίας

Χρησιμοποιήστε το Aspose.Email `FollowUpManager.GetVotingButtons` μέθοδος για την ανάκτηση επιλογών ψηφοφορίας ως συλλογή συμβολοσειρών:

```csharp
try
{
    using (MemoryStream ms = new MemoryStream(File.ReadAllBytes(fileName)))
    {
        MapiMessage testMsg = MapiMessage.FromStream(ms);
        
        // Εξαγωγή κουμπιών ψηφοφορίας σε λίστα
        IList<string> buttons = FollowUpManager.GetVotingButtons(testMsg);

        foreach (var button in buttons)
        {
            Console.WriteLine(button);
        }
    }
}
catch (IOException e)
{
    Console.WriteLine("An error occurred while reading the message file: " + e.Message);
}
```

**Γιατί αυτό το βήμα;**Αυτή η συνάρτηση αναλύει το MapiMessage για να ανακτήσει ενσωματωμένες επιλογές ψηφοφορίας, επιτρέποντας περαιτέρω ανάλυση ή ενσωμάτωση συστήματος.

### Συμβουλές αντιμετώπισης προβλημάτων

- **Το αρχείο δεν βρέθηκε**Ελέγξτε ξανά τη διαδρομή του αρχείου σας για τυπογραφικά λάθη.
- **IOException**Βεβαιωθείτε ότι έχετε δικαιώματα ανάγνωσης στον κατάλογο που περιέχει τα αρχεία μηνυμάτων σας.
