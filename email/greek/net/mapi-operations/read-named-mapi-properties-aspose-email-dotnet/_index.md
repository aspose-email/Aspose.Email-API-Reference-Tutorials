## Οδηγός Εφαρμογής

### Ανάγνωση ονομασμένων ιδιοτήτων από ένα MapiMessage

#### Επισκόπηση
Πρόσβαση και χειρισμός συγκεκριμένων ιδιοτήτων μέσα σε ένα `MapiMessage` αντικείμενο, απαραίτητο για τη διαχείριση μεταδεδομένων ηλεκτρονικού ταχυδρομείου.

#### Βήμα προς βήμα εφαρμογή
1. **Φόρτωση του μηνύματος**
   Φορτώστε το αρχείο μηνύματός σας χρησιμοποιώντας το `MapiMessage` τάξη:
   ```csharp
   // Ορίστε διαδρομές καταλόγων για αρχεία εισόδου και εξόδου
   string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

   // Φόρτωση μηνύματος MAPI από ένα αρχείο
   MapiMessage message = MapiMessage.FromFile(dataDir + "\\message.msg");
   ```
2. **Πρόσβαση σε επώνυμες ιδιότητες**
   Ανάκτηση της συλλογής ιδιοτήτων με όνομα:
   ```csharp
   MapiPropertyCollection properties = message.NamedProperties;
   ```
3. **Επανάληψη και Επεξεργασία Ιδιοτήτων**
   Πραγματοποιήστε επανάληψη σε κάθε ιδιότητα, ελέγχοντας για συγκεκριμένα αναγνωριστικά για να τα επεξεργαστείτε ανάλογα:
   ```csharp
   foreach (MapiNamedProperty mapiNamedProp in properties.Values)
   {
       // Ελέγξτε για ένα συγκεκριμένο αναγνωριστικό ιδιότητας με όνομα και εκτυπώστε την τιμή του
       switch (mapiNamedProp.NameId)
       {
           case "TEST":
               Console.WriteLine($"{mapiNamedProp.DisplayName} = {mapiNamedProp.Value}");
               break;
           // Προσθέστε περισσότερες περιπτώσεις όπως απαιτείται
       }
   }
   ```
**Εξήγηση:**
- `MapiMessage.FromFile`: Φορτώνει το μήνυμα από μια καθορισμένη διαδρομή αρχείου.
- `message.NamedProperties`Παρέχει πρόσβαση σε όλες τις ιδιότητες με όνομα εντός του MapiMessage.
- `mapiNamedProp.NameId`Επιτρέπει την αναγνώριση συγκεκριμένων ιδιοτήτων με μοναδικά αναγνωριστικά.
