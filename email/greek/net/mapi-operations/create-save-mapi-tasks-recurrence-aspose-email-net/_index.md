## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Δημιουργία και αποθήκευση MapiTask με επανάληψη

**Επισκόπηση:**
Η δημιουργία μιας εργασίας MAPI περιλαμβάνει τον ορισμό ωρών έναρξης, ημερομηνιών λήξης, μοτίβων επανάληψης και την αποθήκευσή τους. Αυτή η ενότητα καλύπτει τη ρύθμιση μιας ημερήσιας επαναλαμβανόμενης εργασίας που λήγει μετά από έναν συγκεκριμένο αριθμό εμφανίσεων.

#### Βήμα 1: Ορισμός ημερομηνιών με μετατόπιση ζώνης ώρας

Ξεκινήστε ορίζοντας τις ημερομηνίες έναρξης και λήξης, ενσωματώνοντας μετατοπίσεις ζώνης ώρας:
```csharp
DateTime StartDate = new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now));
DateTime DueDate = new DateTime(2015, 7, 16).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now));
DateTime endByDate = new DateTime(2015, 8, 1).Add(TimeZone.CurrentTimeZone.GetUtcOffset(DateTime.Now));
```

Αυτό διασφαλίζει ότι οι ημερομηνίες των εργασιών σας είναι ακριβείς σε διαφορετικές ζώνες ώρας.

#### Βήμα 2: Δημιουργήστε το MapiTask

Αρχικοποίηση ενός `MapiTask` με συγκεκριμένες λεπτομέρειες όπως το θέμα και το σώμα:
```csharp
MapiTask task = new MapiTask("Automate Task", "Sample Body", StartDate, DueDate);
task.State = MapiTaskState.NotAssigned;
```

#### Βήμα 3: Ορισμός μοτίβου ημερήσιας επανάληψης

Διαμορφώστε το μοτίβο επανάληψης χρησιμοποιώντας `MapiCalendarDailyRecurrencePattern`:
```csharp
var rec = new MapiCalendarDailyRecurrencePattern
{
    PatternType = MapiCalendarRecurrencePatternType.Day,
    Period = 1, // Συχνότητα σε ημέρες
    WeekStartDay = DayOfWeek.Sunday,
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    OccurrenceCount = GetOccurrenceCount(StartDate, endByDate, "FREQ=DAILY"),
};

if (rec.OccurrenceCount == 0)
{
    rec.OccurrenceCount = 1; // Βεβαιωθείτε ότι υπάρχει τουλάχιστον μία εμφάνιση
}
task.Recurrence = rec;
```

#### Βήμα 4: Αποθήκευση της εργασίας

Τέλος, αποθηκεύστε την εργασία σας σε ένα αρχείο:
```csharp
string outputPath = System.IO.Path.Combine("YOUR_OUTPUT_DIRECTORY", "Daily_out.msg");
task.Save(outputPath, TaskSaveFormat.Msg);
```

### Χαρακτηριστικό 2: Υπολογισμός αριθμού εμφανίσεων για μοτίβο επανάληψης

**Επισκόπηση:**
Ο υπολογισμός του αριθμού των εμφανίσεων για ένα μοτίβο επανάληψης είναι απαραίτητος για τον ορισμό των συνθηκών λήξης. Αυτή η λειτουργία δείχνει πώς να μετράτε τις εμφανίσεις μεταξύ δύο ημερομηνιών.

#### Βήμα 1: Μορφοποίηση συμβολοσειράς κανόνα επανάληψης

Δημιουργήστε και μορφοποιήστε τη συμβολοσειρά κανόνα για την ημερήσια συχνότητα:
```csharp
string rrule = string.Format("DTSTART:{0}\r\nRRULE:FREQ=DAILY", start.ToString("yyyyMMdd"));
```

#### Βήμα 2: Δημιουργία εμφανίσεων

Χρήση `CalendarRecurrence` για να δημιουργήσετε ημερομηνίες μεταξύ καθορισμένων ορίων:
```csharp
CalendarRecurrence pattern = new CalendarRecurrence(rrule);
DateCollection dates = pattern.GenerateOccurrences(start, endBy);
uint occurrenceCount = (uint)dates.Count;
return occurrenceCount;
```

Αυτό σας δίνει τον συνολικό αριθμό εμφανίσεων εντός της καθορισμένης περιόδου.
