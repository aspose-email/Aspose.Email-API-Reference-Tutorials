## Οδηγός Εφαρμογής

Αυτός ο οδηγός χωρίζεται σε τρία βασικά μέρη: φόρτωση μηνυμάτων, κρυπτογράφηση email και αποκρυπτογράφησή τους.

### Φόρτωση και έλεγχος κρυπτογράφησης μηνυμάτων

#### Επισκόπηση
Πριν μπορέσετε να κρυπτογραφήσετε ή να αποκρυπτογραφήσετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου, είναι απαραίτητο να φορτώσετε το μήνυμα και να ελέγξετε την κατάσταση κρυπτογράφησής του. Αυτή η ενότητα θα σας δείξει πώς να το κάνετε αυτό.

**Βήμα 1: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
MailMessage mailMessageOrig = MailMessage.Load(Path.Combine(dataDir, "Message.msg"), new MsgLoadOptions());
bool isEncryptedOriginal = mailMessageOrig.IsEncrypted;
```
- **Παράμετροι**: Το `dataDir` Η μεταβλητή θα πρέπει να δείχνει στον κατάλογο εγγράφων σας. `MailMessage.Load` Η μέθοδος διαβάζει το μήνυμα ηλεκτρονικού ταχυδρομείου από μια καθορισμένη διαδρομή αρχείου.
- **Σκοπός**: Αυτό το βήμα φορτώνει ένα email και ελέγχει αν είναι ήδη κρυπτογραφημένο.

**Συμβουλή αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι η διαδρομή του αρχείου είναι σωστή και προσβάσιμη, διαφορετικά ενδέχεται να αντιμετωπίσετε ένα FileNotFoundException.

### Κρυπτογράφηση μηνυμάτων ηλεκτρονικού ταχυδρομείου

#### Επισκόπηση
Η κρυπτογράφηση των email σας διασφαλίζει ότι μόνο εξουσιοδοτημένα άτομα μπορούν να τα διαβάσουν. Ας κρυπτογραφήσουμε ένα μήνυμα χρησιμοποιώντας ένα δημόσιο πιστοποιητικό.

**Βήμα 2: Κρυπτογράφηση του μηνύματος**
```csharp
string publicCertFile = Path.Combine(dataDir, "MartinCertificate.cer");
X509Certificate2 publicCert = new X509Certificate2(publicCertFile);
MailMessage mailMessage = mailMessageOrig.Encrypt(publicCert);
bool isEncryptedAfterEncryption = mailMessage.IsEncrypted;
```
- **Παράμετροι**: `publicCert` αντιπροσωπεύει το πιστοποιητικό που χρησιμοποιείται για την κρυπτογράφηση.
- **Σκοπός**Κρυπτογραφεί το μήνυμα, διασφαλίζοντας την εμπιστευτικότητά του.

**Βασικές επιλογές διαμόρφωσης**Επιλέξτε ένα ισχυρό πιστοποιητικό και διαχειριστείτε τα κλειδιά σας με ασφάλεια για να αποφύγετε μη εξουσιοδοτημένη πρόσβαση.

### Αποκρυπτογράφηση μηνυμάτων ηλεκτρονικού ταχυδρομείου

#### Επισκόπηση
Για να διαβάσετε ένα κρυπτογραφημένο email, πρέπει να το αποκρυπτογραφήσετε χρησιμοποιώντας το αντίστοιχο ιδιωτικό πιστοποιητικό. Δείτε πώς μπορείτε να το κάνετε:

**Βήμα 3: Αποκρυπτογράφηση του μηνύματος**
```csharp
string privateCertFile = Path.Combine(dataDir, "MartinCertificate.pfx");
X509Certificate2 privateCert = new X509Certificate2(privateCertFile, "anothertestaccount");
MailMessage decryptedMailMessage = mailMessage.Decrypt(privateCert);
bool isEncryptedAfterDecryption = decryptedMailMessage.IsEncrypted;
```
- **Παράμετροι**: `privateCert` διατηρεί το ιδιωτικό σας κλειδί για αποκρυπτογράφηση.
- **Σκοπός**: Αυτό το βήμα αποκρυπτογραφεί το email ώστε να είναι δυνατή η ανάγνωσή του.

**Συμβουλή αντιμετώπισης προβλημάτων**Ελέγξτε ξανά τον κωδικό πρόσβασης του πιστοποιητικού σας και βεβαιωθείτε ότι ταιριάζει με αυτόν που χρησιμοποιήθηκε κατά την κρυπτογράφηση.
