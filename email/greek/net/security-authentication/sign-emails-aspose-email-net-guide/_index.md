## Οδηγός Εφαρμογής
Αυτή η ενότητα αναλύει τη διαδικασία σε διαχειρίσιμα βήματα.

### Φόρτωση και αρχικοποίηση πιστοποιητικών
#### Επισκόπηση
Η φόρτωση πιστοποιητικών X.509 είναι ζωτικής σημασίας για την ψηφιακή υπογραφή email. Θα χρησιμοποιήσουμε `Aspose.Email` για να φορτώσετε δημόσια και ιδιωτικά πιστοποιητικά από αρχεία.

##### Βήμα 1: Φόρτωση του δημόσιου πιστοποιητικού
Το δημόσιο πιστοποιητικό, συνήθως σε `.cer` μορφή, μπορεί να φορτωθεί ως εξής:
```csharp
using System.Security.Cryptography.X509Certificates;

string publicCertFile = @"YOUR_DOCUMENT_DIRECTORY\MartinCertificate.cer";
X509Certificate2 publicCert = new X509Certificate2(publicCertFile);
```
*Εξήγηση*: Αυτό το τμήμα κώδικα φορτώνει το πιστοποιητικό από μια καθορισμένη διαδρομή αρχείου. Το `X509Certificate2` Η κλάση χρησιμοποιείται για τη διαχείριση του πιστοποιητικού.

##### Βήμα 2: Φόρτωση του Ιδιωτικού Πιστοποιητικού με Κωδικό Πρόσβασης
Η φόρτωση του ιδιωτικού πιστοποιητικού απαιτεί τον καθορισμό του κωδικού πρόσβασής του:
```csharp
string privateCertFile = @"YOUR_DOCUMENT_DIRECTORY\MartinCertificate.pfx";
X509Certificate2 privateCert = new X509Certificate2(privateCertFile, "password");
```
*Εξήγηση*Το αρχείο PFX που περιέχει το ιδιωτικό κλειδί πρέπει να φορτωθεί με κωδικό πρόσβασης για λόγους ασφαλείας.

### Δημιουργία και υπογραφή μηνύματος ηλεκτρονικού ταχυδρομείου
#### Επισκόπηση
Έχοντας έτοιμα τα πιστοποιητικά σας, ας δημιουργήσουμε και ας υπογράψουμε ένα μήνυμα ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας το Aspose.Email.

##### Βήμα 1: Δημιουργήστε ένα `MailMessage`
Αρχικά, κατασκευάστε ένα `MailMessage` αντικείμενο:
```csharp
using Aspose.Email.Mime;

MailMessage msg = new MailMessage("userfrom@gmail.com", "userto@domain.com");
msg.Subject = "Secure Communication";
msg.Body = "This is a digitally signed email.";
```
*Εξήγηση*Εδώ, ορίζουμε τον αποστολέα, τον παραλήπτη, το θέμα και το σώμα του email μας.

##### Βήμα 2: Επισύναψη ψηφιακής υπογραφής
Για να επισυνάψετε την ψηφιακή υπογραφή:
```csharp
msg.Attachments.Add(new Attachment(privateCert));
```
*Εξήγηση*Χρησιμοποιούμε το ιδιωτικό πιστοποιητικό για να υπογράψουμε το μήνυμα. Αυτό το βήμα διασφαλίζει ότι η ακεραιότητα και η προέλευση του μηνύματος επαληθεύονται από τους παραλήπτες.
