## Οδηγός Εφαρμογής

### Αποστολή email με αποσπασμένο πιστοποιητικό
Αυτή η λειτουργία δείχνει πώς να στείλετε ένα email υπογεγραμμένο με ένα αποσπασμένο πιστοποιητικό, διασφαλίζοντας ότι οι παραλήπτες μπορούν να επαληθεύσουν ανεξάρτητα την ταυτότητά σας.

#### Βήμα 1: Φόρτωση του ιδιωτικού σας πιστοποιητικού
Φορτώστε το ιδιωτικό πιστοποιητικό που χρησιμοποιείται για την υπογραφή email:
```csharp
// Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Φόρτωση του ιδιωτικού πιστοποιητικού από ένα αρχείο
string privateCertFile = dataDir + "/MartinCertificate.pfx";
X509Certificate2 privateCert = new X509Certificate2(privateCertFile, "anothertestaccount");
```
**Γιατί;** Η αποσπασμένη υπογραφή χρησιμοποιεί το ιδιωτικό σας κλειδί.

#### Βήμα 2: Δημιουργία και υπογραφή του μηνύματος ηλεκτρονικού ταχυδρομείου
Δημιουργήστε ένα `MailMessage` αντιταχθείτε και υπογράψτε το με το φορτωμένο πιστοποιητικό:
```csharp
// Δημιουργήστε ένα μήνυμα ηλεκτρονικού ταχυδρομείου για αποστολή
MailMessage msg = new MailMessage("user@domain.com", "receiver@domain.com", 
    "subject:Signed message only by AE", "body:Test Body of signed message by AE");

// Επισυνάψτε την υπογραφή χρησιμοποιώντας το ιδιωτικό πιστοποιητικό και ορίστε την ως αποσπασμένη
MailMessage signed = msg.AttachSignature(privateCert, true);
```
**Γιατί;** Η επισύναψη μιας αποσπασμένης υπογραφής την διαχωρίζει από το περιεχόμενο του email για ανεξάρτητη επαλήθευση.

#### Βήμα 3: Ρύθμιση παραμέτρων προγράμματος-πελάτη SMTP
Διαμορφώστε το `SmtpClient` για να στείλετε το υπογεγραμμένο μήνυμα με ασφάλεια:
```csharp
// Λήψη ρυθμίσεων προγράμματος-πελάτη SMTP
SmtpClient smtp = new SmtpClient("smtp.domain.com", "user@domain.com", "password") 
{ 
    Port = 25,
    SecurityOptions = SecurityOptions.SSLAuto 
};
```
**Γιατί;** Το SSL/TLS διασφαλίζει την ασφαλή μετάδοση email μέσω του διαδικτύου.

#### Βήμα 4: Στείλτε το email
Τέλος, προσπαθήστε να στείλετε το υπογεγραμμένο μήνυμα:
```csharp
// Προσπάθεια αποστολής του υπογεγραμμένου μηνύματος
try 
{
    smtp.Send(signed);
} 
catch (Exception ex) 
{
    // Χειρισμός τυχόν εξαιρέσεων που προκύπτουν κατά την αποστολή
    Console.WriteLine(ex.Message);
}
```
**Γιατί;** Ο χειρισμός εξαιρέσεων είναι ζωτικής σημασίας για τον εντοπισμό και την επίλυση προβλημάτων κατά τη μετάδοση email.

### Ρύθμιση παραμέτρων προγράμματος-πελάτη SMTP
Ακολουθεί μια μέθοδος που δείχνει πώς μπορείτε να δημιουργήσετε και να διαμορφώσετε τον υπολογιστή-πελάτη SMTP:
```csharp
private static SmtpClient GetSmtpClient()
{
    // Δημιουργήστε μια νέα παρουσία της κλάσης SmtpClient με διεύθυνση διακομιστή και διαπιστευτήρια χρήστη
    SmtpClient client = new SmtpClient("smtp.domain.com", "user@domain.com", "password"); 
    
    // Ορισμός θύρας SMTP και επιλογών ασφαλείας για SSL/TLS
    client.Port = 25;
    client.SecurityOptions = SecurityOptions.SSLAuto;
    
    return client;
}
```
**Γιατί;** Η προσαρμογή των ρυθμίσεων SMTP διασφαλίζει ότι τα email αποστέλλονται μέσω ασφαλούς καναλιού.
