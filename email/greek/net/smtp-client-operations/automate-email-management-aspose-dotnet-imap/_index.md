## Οδηγός Εφαρμογής

### Σύνδεση και σύνδεση στον διακομιστή IMAP

#### Επισκόπηση
Η σύνδεση σε έναν διακομιστή IMAP είναι το πρώτο βήμα στη διαχείριση των email μέσω προγραμματισμού. Θα χρησιμοποιήσουμε το Aspose.Email. `ImapClient` τάξη για τον σκοπό αυτό.

**Βήμα 1: Ορισμός διαπιστευτηρίων**
Ξεκινήστε ορίζοντας τα διαπιστευτήρια του διακομιστή IMAP:
```csharp
const string host = "your-imap-host";
const int port = 143; // Προεπιλεγμένη θύρα IMAP
const string username = "user@host.com";
const string password = "password";
```

**Βήμα 2: Δημιουργία και χρήση του ImapClient**
Δημιουργήστε μια παρουσία του `ImapClient` κλάση χρησιμοποιώντας αυτά τα διαπιστευτήρια:
```csharp
using (ImapClient client = new ImapClient(host, port, username, password))
{
    Console.WriteLine("Connected and logged in to IMAP server.");
}
```

**Συμβουλή αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι το δίκτυό σας επιτρέπει συνδέσεις στην καθορισμένη θύρα IMAP. Εάν αντιμετωπίζετε προβλήματα ελέγχου ταυτότητας, ελέγξτε ξανά τα διαπιστευτήριά σας.

### Επιλέξτε έναν φάκελο IMAP

#### Επισκόπηση
Μόλις συνδεθείτε, η επιλογή ενός φακέλου όπως το Inbox είναι απαραίτητη για την εκτέλεση λειτουργιών μέσα σε αυτόν.

**Βήμα 1: Σύνδεση με διακομιστή**
Επαναχρησιμοποίηση του `ImapClient`συνδέστε όπως φαίνεται προηγουμένως:
```csharp
using (ImapClient client = new ImapClient("your-imap-host", 143, "user@host.com", "password"))
{
    // Επιλέξτε τον φάκελο Εισερχόμενα
    client.SelectFolder(ImapFolderInfo.InBox);
    Console.WriteLine("Inbox folder selected.");
}
```

### Δημιουργία και εκτέλεση ερωτήματος αναζήτησης IMAP

#### Επισκόπηση
Η αναζήτηση συγκεκριμένων μηνυμάτων ηλεκτρονικού ταχυδρομείου είναι μια συνηθισμένη εργασία. Θα δείξουμε πώς να δημιουργήσετε και να εκτελέσετε ένα ερώτημα αναζήτησης IMAP.

**Βήμα 1: Δημιουργία ImapQueryBuilder**
Χρησιμοποιήστε το `ImapQueryBuilder` για να καθορίσετε τα κριτήρια αναζήτησής σας:
```csharp
ImapQueryBuilder builder = new ImapQueryBuilder();
builder.Subject.Contains("Newsletter"); // Φιλτράρισμα κατά γραμμή θέματος
builder.InternalDate.On(DateTime.Now);  // Ηλεκτρονικά μηνύματα που ελήφθησαν σήμερα
```

**Βήμα 2: Εκτέλεση ερωτήματος αναζήτησης**
Χρησιμοποιήστε το ερώτημα για να ανακτήσετε μηνύματα:
```csharp
MailQuery query = builder.GetQuery();
ImapMessageInfoCollection messages = client.ListMessages(query);
Console.WriteLine($"Found {messages.Count} message(s) in Inbox.");
```

## Ενότητα Συχνών Ερωτήσεων
1. **Ποιος είναι ο προεπιλεγμένος αριθμός θύρας για το IMAP;**
Η προεπιλεγμένη θύρα είναι 143, αλλά οι ασφαλείς συνδέσεις συνήθως χρησιμοποιούν τη θύρα 993.
2. **Πώς μπορώ να χειριστώ SSL/TLS με το Aspose.Email;**
Διαμορφώστε το `ImapClient` για να ενεργοποιήσετε το SSL όπως απαιτείται: `client.SecurityOptions = SecurityOptions.Auto;`
3. **Μπορώ να αναζητήσω email παλαιότερα από σήμερα;**
Ναι, προσαρμόστε το `InternalDate.On` μέθοδος ή χρήση εύρους ημερομηνιών σε `ImapQueryBuilder`.
4. **Τι γίνεται αν ο διακομιστής IMAP μου απαιτεί έλεγχο ταυτότητας μέσω OAuth2;**
Το Aspose.Email υποστηρίζει OAuth2. Υλοποιήστε τα απαραίτητα βήματα για τον έλεγχο ταυτότητας χρησιμοποιώντας διακριτικά OAuth.
5. **Πώς μπορώ να χειρίζομαι αποτελεσματικά μεγάλους όγκους email;**
Χρησιμοποιήστε την ομαδοποίηση και βελτιστοποιήστε τα ερωτήματά σας για να επεξεργάζεστε τα email σε διαχειρίσιμα τμήματα.
