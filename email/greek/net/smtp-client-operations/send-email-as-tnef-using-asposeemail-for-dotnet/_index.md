## Οδηγός Εφαρμογής

Ας δούμε τώρα πώς να στείλετε ένα email ως TNEF χρησιμοποιώντας το Aspose.Email για .NET.

### Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου με διατηρημένες τις επιλογές TNEF

#### Βήμα 1: Φόρτωση του αρχείου email σας
Ξεκινήστε φορτώνοντας το μήνυμα ηλεκτρονικού ταχυδρομείου σας από ένα `.eml` αρχείο για να διατηρήσετε τυχόν συνημμένα TNEF:
```csharp
using System;
using Aspose.Email.Mime;
using Aspose.Email.Clients.Smtp;

var emlFileName = @"YOUR_DOCUMENT_DIRECTORY\Message.eml"; // Διαδρομή προς το αρχείο EML

// Φορτώστε το μήνυμα ηλεκτρονικού ταχυδρομείου με επιλογές για τη διατήρηση των συνημμένων TNEF.
MailMessage eml1 = MailMessage.Load(emlFileName, new EmlLoadOptions() { PreserveTnefAttachments = true });
```

**Εξήγηση**: Το `EmlLoadOptions` Η κλάση έχει ρυθμιστεί ώστε να διατηρεί τα δεδομένα που αφορούν το TNEF ανέπαφα κατά τη φόρτωση.

### Ρύθμιση πληροφοριών ηλεκτρονικού ταχυδρομείου

#### Βήμα 2: Ρύθμιση παραμέτρων αποστολέα και παραλήπτη
Ορίστε τα πεδία αποστολέα και παραλήπτη του email:
```csharp
eml1.From = "somename@gmail.com";
eml1.To.Clear();
eml1.To.Add(new MailAddress("first.last@test.com"));
```

**Εξήγηση**Αντικατάσταση των email placeholder με πραγματικές διευθύνσεις.

#### Βήμα 3: Ορισμός θέματος και ημερομηνίας
Προσθέστε μια γραμμή θέματος και ορίστε την τρέχουσα ημερομηνία για το email σας:
```csharp
eml1.Subject = "With PreserveTnef flag during loading";
eml1.Date = DateTime.Now;
```

### Ρύθμιση παραμέτρων προγράμματος-πελάτη SMTP για TNEF

#### Βήμα 4: Ρύθμιση ρυθμίσεων SMTP
Ρυθμίστε τις παραμέτρους του προγράμματος-πελάτη SMTP για αποστολή μηνυμάτων ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας τη μορφή TNEF:
```csharp
SmtpClient client = new SmtpClient("smtp.gmail.com", 587, "somename", "password");
client.SecurityOptions = SecurityOptions.Auto; // Αυτόματη προσαρμογή των ρυθμίσεων ασφαλείας
client.UseTnef = true;
```

**Εξήγηση**: Το `UseTnef` Η ιδιότητα είναι κρίσιμη για την αποστολή email σε μορφή TNEF. Βεβαιωθείτε ότι ο διακομιστής SMTP σας υποστηρίζει αυτήν τη λειτουργία.

### Αποστολή του ηλεκτρονικού ταχυδρομείου

#### Βήμα 5: Στείλτε το email σας
Τέλος, στείλτε το email και χειριστείτε τυχόν πιθανές εξαιρέσεις:
```csharp
try
{
    client.Send(eml1);
}
catch (Exception ex)
{
    Console.Write(ex.Message); // Καταγράψτε ή διαχειριστείτε τις εξαιρέσεις ανάλογα
}
```

**Εξήγηση**Η ενσωμάτωση της διαδικασίας αποστολής σε ένα μπλοκ try-catch βοηθά στη διαχείριση των σφαλμάτων με ομαλό τρόπο.
