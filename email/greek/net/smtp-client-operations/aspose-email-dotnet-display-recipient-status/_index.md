## Οδηγός Εφαρμογής

### Ανάγνωση καταστάσεων παραληπτών
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να έχετε πρόσβαση και να εκτυπώνετε την κατάσταση παρακολούθησης κάθε παραλήπτη σε ένα μήνυμα MAPI. Είναι χρήσιμη για την αποτελεσματική διαχείριση των απαντήσεων σε αιτήματα σε σύσκεψη.
##### Βήμα 1: Φόρτωση του μηνύματος MAPI
Ξεκινήστε φορτώνοντας το αρχείο αιτήματος σύσκεψης σε ένα `MapiMessage` αντικείμενο:
```csharp
// Βεβαιωθείτε ότι αυτή η διαδρομή δείχνει στο πραγματικό αρχείο .msg.
string filePath = @"YOUR_DOCUMENT_DIRECTORY\Test Meeting.msg";
MapiMessage message = MapiMessage.FromFile(filePath);
```
##### Βήμα 2: Επανάληψη σε παραλήπτες
Επαναλάβετε τη διαδικασία σε κάθε παραλήπτη στο φορτωμένο `MapiMessage` και να εκτυπώσουν την κατάσταση παρακολούθησής τους:
```csharp
foreach (MapiRecipient recipient in message.Recipients)
{
    // Εκτυπώστε την κατάσταση παρακολούθησης κάθε παραλήπτη.
    Console.WriteLine(recipient.RecipientTrackStatus);
}
```
**Εξήγηση**: Το `RecipientTrackStatus` Η ιδιότητα παρέχει πληροφορίες σχετικά με το εάν ένας παραλήπτης έχει αποδεχτεί, απορρίψει ή δεν έχει απαντήσει στο αίτημά σας για σύσκεψη.
