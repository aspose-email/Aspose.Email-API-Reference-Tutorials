## Οδηγός Εφαρμογής

### Δημιουργία και διαμόρφωση μηνυμάτων αλληλογραφίας

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα νέο μήνυμα ηλεκτρονικού ταχυδρομείου, να προσαρμόσετε τις ιδιότητές του όπως αποστολέα, παραλήπτη, θέμα, προτεραιότητα, ευαισθησία και ειδοποιήσεις παράδοσης και να το αποθηκεύσετε ως αρχείο EML.

##### Βήμα 1: Δημιουργήστε ένα νέο μήνυμα ηλεκτρονικού ταχυδρομείου
```csharp
using Aspose.Email.Mime;
using System;

// Αρχικοποίηση μιας νέας παρουσίας MailMessage
MailMessage message = new MailMessage();
message.From = "sender@gmail.com"; // Ορίστε τη διεύθυνση email του αποστολέα
message.To = "receiver@gmail.com"; // Ορίστε τη διεύθυνση email του παραλήπτη
message.Subject = "Using MailMessage Features"; // Ορίστε το θέμα

// Ορισμός πρόσθετων ιδιοτήτων
message.Date = DateTime.Now; // Τρέχουσα ώρα ως ημερομηνία μηνύματος
message.Priority = MailPriority.High; // Η προτεραιότητα ηλεκτρονικού ταχυδρομείου έχει οριστεί σε Υψηλή
message.Sensitivity = MailSensitivity.Normal; // Κανονικό επίπεδο ευαισθησίας
message.DeliveryNotificationOptions = DeliveryNotificationOptions.OnSuccess; // Ενεργοποίηση ειδοποίησης επιτυχίας
```

##### Βήμα 2: Αποθήκευση του μηνύματος
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
message.Save(outputDir + "/UseMailMessageFeatures_out.eml", SaveOptions.DefaultEml);
```
- **SaveOptions.DefaultEml**: Αυτή η επιλογή αποθηκεύει το email σε μια προεπιλεγμένη μορφή EML.

#### Συμβουλές αντιμετώπισης προβλημάτων
Βεβαιωθείτε ότι το δικό σας `outputDir` Η διαδρομή έχει οριστεί σωστά για την αποφυγή σφαλμάτων αποθήκευσης αρχείων. Να χειρίζεστε πάντα τις εξαιρέσεις κατά τη διάρκεια των λειτουργιών αρχείων για ισχυρή διαχείριση σφαλμάτων.
