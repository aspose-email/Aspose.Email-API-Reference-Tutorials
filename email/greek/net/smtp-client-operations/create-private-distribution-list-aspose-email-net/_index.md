## Οδηγός Εφαρμογής

### Δημιουργία ιδιωτικής λίστας διανομής
Αυτή η λειτουργία σάς επιτρέπει να δημιουργήσετε μια ιδιωτική λίστα διανομής στο Microsoft Exchange χρησιμοποιώντας το Aspose.Email.

#### Βήμα 1: Αρχικοποίηση του προγράμματος-πελάτη EWS
Ξεκινήστε ρυθμίζοντας τη σύνδεσή σας με τον διακομιστή. Βεβαιωθείτε ότι έχετε τη σωστή διεύθυνση URL, όνομα χρήστη, κωδικό πρόσβασης και τομέα για έλεγχο ταυτότητας.

```csharp
IEWSClient client = EWSClient.GetEWSClient("https://outlook.office365.com/ews/exchange.asmx", "testUser", "προεπιλογή", "τομέας");
```

#### Βήμα 2: Ρύθμιση λεπτομερειών λίστας διανομής
Δημιουργήστε ένα νέο `ExchangeDistributionList` αντικείμενο και ορίστε το εμφανιζόμενο όνομά του.

```csharp
ExchangeDistributionList distributionList = new ExchangeDistributionList();
distributionList.DisplayName = "Test Private List";
```

#### Βήμα 3: Προσθήκη μελών στη λίστα
Χρήση `MailAddressCollection` για να προσθέσετε διευθύνσεις ηλεκτρονικού ταχυδρομείου στη λίστα σας. Αυτή η συλλογή σάς επιτρέπει να διαχειρίζεστε αποτελεσματικά πολλά μέλη.

```csharp
MailAddressCollection members = new MailAddressCollection();
members.Add("address1@host.com");
members.Add("address2@host.com");
members.Add("address3@host.com");
```

#### Βήμα 4: Δημιουργήστε τη λίστα διανομής στον Exchange Server
Τέλος, χρησιμοποιήστε το `CreateDistributionList` μέθοδος για να δημιουργήσετε τη λίστα σας στον διακομιστή.

```csharp
client.CreateDistributionList(distributionList, members);
```

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι όλες οι διευθύνσεις ηλεκτρονικού ταχυδρομείου έχουν σωστή μορφοποίηση.
- Επαληθεύστε τη συνδεσιμότητα δικτύου και τα δικαιώματα πρόσβασης στο τελικό σημείο του EWS.
