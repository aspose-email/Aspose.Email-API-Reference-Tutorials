## Οδηγός Εφαρμογής
### Δυνατότητα προώθησης email SMTP
Αυτή η λειτουργία σάς επιτρέπει να προωθείτε email απευθείας χρησιμοποιώντας `SmtpClient` χωρίς να δημιουργήσετε χειροκίνητα ένα `MailMessage`Ας αναλύσουμε τη διαδικασία υλοποίησης.

#### Ορισμός λεπτομερειών διακομιστή και διαπιστευτηρίων
Ξεκινήστε καθορίζοντας τα στοιχεία του διακομιστή email σας:

```csharp
string host = "mail.server.com";
int smtpPort = 587;
string username = "username"; // Το όνομα χρήστη SMTP σας
string password = "password"; // Ο κωδικός πρόσβασής σας SMTP
```

Αυτές οι παράμετροι είναι κρίσιμες για τη δημιουργία σύνδεσης με τον διακομιστή SMTP.

#### Καθορισμός Αποστολέα και Παραληπτών
Προσδιορίστε ποιος θα στείλει το email και σε ποιον πρέπει να προωθηθεί:

```csharp
// Καθορίστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του αποστολέα
cstring sender = "Sender@domain.com";

// Ορισμός παραληπτών ως συλλογή
MailAddressCollection recipients = new MailAddressCollection();
recipients.Add("recepient1@domain.com, recepient2@domain.com");
```

#### Προώθηση του ηλεκτρονικού ταχυδρομείου
Η βασική λειτουργικότητα είναι η προώθηση ενός υπάρχοντος αρχείου email:

```csharp
using (SmtpClient client = new SmtpClient(host, smtpPort, username, password, SecurityOptions.Auto))
{
    // Διαδρομή προς το αρχείο email σας (μορφή .eml)
    string fileName = @"YOUR_DOCUMENT_DIRECTORY\test.eml";

    // Άνοιγμα του αρχείου email για ανάγνωση
    using (FileStream fs = File.OpenRead(fileName))
    {
        // Προώθηση του email από τον αποστολέα στους παραλήπτες
        client.Forward(sender, recipients, fs);
    }
}
```

**Βασικές επιλογές διαμόρφωσης:**
- `SecurityOptions.Auto`: Επιλέγει αυτόματα το πρωτόκολλο ασφαλείας με βάση τις δυνατότητες του διακομιστή.
- Χρησιμοποιήστε μπλοκ try-catch γύρω από τις λειτουργίες δικτύου για τον χειρισμό σφαλμάτων.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι τα στοιχεία του διακομιστή SMTP σας είναι σωστά και προσβάσιμα από το περιβάλλον ανάπτυξής σας.
- Επαληθεύστε ότι η διαδρομή του αρχείου email είναι ακριβής και η μορφή του αρχείου είναι `.eml`.
- Ελέγξτε τις ρυθμίσεις του τείχους προστασίας εάν προκύψουν προβλήματα σύνδεσης.
