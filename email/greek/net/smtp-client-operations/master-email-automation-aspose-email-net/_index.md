## Οδηγός Εφαρμογής
### Σύνδεση με την υπηρεσία Exchange Web
#### Επισκόπηση
Η σύνδεση με τον Ενσωματωμένο Διακομιστή Ιστού (EWS) είναι ζωτικής σημασίας για την πρόσβαση στα δεδομένα του γραμματοκιβωτίου. Αυτή η ενότητα παρουσιάζει τη δημιουργία σύνδεσης χρησιμοποιώντας το `Aspose.Email.Clients.Exchange.WebService` ονοματοχώρος.

#### Σύνδεση βήμα προς βήμα
1. **Ρύθμιση διαπιστευτηρίων**
   ```csharp
   string mailboxUri = "https://ex2010/ews/exchange.asmx";
   string username = "test.exchange";
   string password = "pwd";
   string domain = "ex2010.local";

   NetworkCredential credentials = new NetworkCredential(username, password, domain);
   ```
   *Εξήγηση*: Διαμορφώστε τα διαπιστευτήρια δικτύου που απαιτούνται για τον έλεγχο ταυτότητας. `NetworkCredential` Η κλάση διατηρεί με ασφάλεια τα στοιχεία σύνδεσής σας.

2. **Αρχικοποίηση προγράμματος-πελάτη EWS**
   ```csharp
   IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
   ```
   *Εξήγηση*: Αυτή η γραμμή δημιουργεί μια παρουσία του `IEWSClient`, το οποίο παρέχει μεθόδους για την αλληλεπίδραση με τον διακομιστή Exchange χρησιμοποιώντας το παρεχόμενο URI και τα διαπιστευτήρια.

### Επέκταση μιας Δημόσιας Λίστας Διανομής
#### Επισκόπηση
Η επέκταση των λιστών διανομής σάς επιτρέπει να ανακτήσετε όλες τις διευθύνσεις ηλεκτρονικού ταχυδρομείου των μελών. Αυτό είναι χρήσιμο για εργασίες μαζικής επικοινωνίας ή διαχείρισης δεδομένων.

#### Βήμα προς βήμα επέκταση
1. **Προσδιορίστε τη λίστα διανομής**
   ```csharp
   MailAddress distributionList = new MailAddress("public.distribution.list@host.com");
   ```
   *Εξήγηση*Καθορίστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου της δημόσιας λίστας διανομής που θα αναπτυχθεί.

2. **Ανάκτηση Μελών**
   ```csharp
   MailAddressCollection members = client.ExpandDistributionList(distributionList);
   foreach (MailAddress member in members)
   {
       // Αποκτήστε πρόσβαση στη διεύθυνση ηλεκτρονικού ταχυδρομείου κάθε μέλους εδώ.
   }
   ```
   *Εξήγηση*: Το `ExpandDistributionList` Η μέθοδος ανακτά όλα τα μέλη της καθορισμένης λίστας διανομής, επιστρέφοντας μια συλλογή που μπορεί να επαναληφθεί για πρόσβαση σε μεμονωμένα μηνύματα ηλεκτρονικού ταχυδρομείου.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι το URI του γραμματοκιβωτίου σας και τα διαπιστευτήρια είναι σωστά.
- Επαληθεύστε τη σύνδεση δικτύου με τον διακομιστή Exchange.
- Ελέγξτε για τυχόν ρυθμίσεις τείχους προστασίας που ενδέχεται να αποκλείουν αιτήματα EWS.
