## Οδηγός Εφαρμογής

### Αποστολή email με δυνατότητα εναλλακτικού κειμένου

#### Επισκόπηση
Αυτή η λειτουργία επιτρέπει την αποστολή email με περιεχόμενο HTML και εναλλακτικές απλού κειμένου χρησιμοποιώντας το Aspose.Email για .NET, διασφαλίζοντας τη συμβατότητα σε όλα τα προγράμματα-πελάτες και τις συσκευές email.

#### Βήμα προς βήμα εφαρμογή

**1. Αρχικοποίηση του MailMessage**

Ξεκινήστε δημιουργώντας μια παρουσία του `MailMessage` κλάση και ρυθμίστε τον αποστολέα, τον παραλήπτη και το σώμα του μηνύματός σας:

```csharp
using System;
using Aspose.Email.Mime;
using Aspose.Email.Clients.Smtp;

class SendEmailWithAlternateTextFeature
{
    public static void Execute()
    {
        // Δημιουργήστε μια νέα παρουσία MailMessage
        MailMessage message = new MailMessage();
        
        // Ορίστε τη διεύθυνση «Από» και τη διεύθυνση ηλεκτρονικού ταχυδρομείου του παραλήπτη
        message.From = "sender@sender.com";
        message.To.Add("receiver@receiver.com");

        // Προσθήκη απλού κειμένου
        message.Body = "This is Plain Text Body";

        // Προσθήκη εναλλακτικής προβολής HTML για υπολογιστές-πελάτες που την υποστηρίζουν
        AlternateView alternateView = AlternateView.CreateAlternateViewFromString(
            "<html><body>This is the <b>HTML</b> version of the email.</body></html>",
            null,
            MediaTypeNames.Text.Html);
        message.AlternateViews.Add(alternateView);
    }
}
```

**2. Ρύθμιση παραμέτρων του προγράμματος-πελάτη SMTP**

Ρυθμίστε το δικό σας `SmtpClient` με τα στοιχεία του διακομιστή για την αποστολή του email:

```csharp
// Δημιουργία και ρύθμιση παραμέτρων μιας παρουσίας του SmtpClient
SmtpClient client = new SmtpClient();
client.Host = "smtp.server.com";  // Αντικαταστήστε με τον κεντρικό διακομιστή SMTP
client.Username = "Username";     // Το όνομα χρήστη ελέγχου ταυτότητας σας
client.Password = "Password";     // Ο κωδικός πρόσβασης ελέγχου ταυτότητας
client.Port = 25;                 // Συνήθως, η προεπιλεγμένη θύρα είναι 25

try
{
    // Στείλτε το μήνυμα ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας τη μέθοδο SmtpClient.Send
    client.Send(message);
}
catch (Exception ex)
{
    // Χειρισμός εξαιρέσεων κατά την αποστολή
    System.Diagnostics.Trace.WriteLine(ex.ToString());
}
```

### Ρύθμιση παραμέτρων προγράμματος-πελάτη ηλεκτρονικού ταχυδρομείου για την αποστολή μηνυμάτων ηλεκτρονικού ταχυδρομείου

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να ρυθμίσετε ένα πρόγραμμα-πελάτη SMTP για την αποστολή email.

**1. Αρχικοποίηση και ορισμός λεπτομερειών διακομιστή**

Δημιουργήστε ένα νέο `SmtpClient` instance και ρυθμίστε τις απαραίτητες λεπτομέρειες του διακομιστή:

```csharp
using Aspose.Email.Clients.Smtp;

class EmailClientConfigurationFeature
{
    public static void Execute()
    {
        SmtpClient client = new SmtpClient();
        client.Host = "smtp.server.com";  // Διεύθυνση διακομιστή SMTP
        client.Username = "Username";     // Όνομα χρήστη για έλεγχο ταυτότητας με τον διακομιστή
        client.Password = "Password";     // Κωδικός πρόσβασης για έλεγχο ταυτότητας με τον διακομιστή
        client.Port = 25;                 // Αριθμός θύρας που χρησιμοποιείται από τον διακομιστή SMTP (η προεπιλογή είναι συνήθως 25)
    }
}
```
