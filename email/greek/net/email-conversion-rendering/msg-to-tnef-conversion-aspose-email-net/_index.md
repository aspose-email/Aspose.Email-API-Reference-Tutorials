## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στην υλοποίηση της μετατροπής MSG σε TNEF και στον χειρισμό διαφορετικών μορφών.

### Μετατροπή MSG σε TNEF

#### Επισκόπηση
Η μετατροπή ενός αρχείου MSG του Outlook σε TNEF ενσωματώνει όλο το πλούσιο περιεχόμενο, διασφαλίζοντας την πιστότητα κατά τη μεταφορά μεταξύ συστημάτων.

#### Βήματα για τη μετατροπή
**1. Φορτώστε το αρχείο MSG:**
```csharp
using System.IO;
using Aspose.Email.Mapi;

string documentDir = "YOUR_DOCUMENT_DIRECTORY";
MapiMessage msg = MapiMessage.FromFile(documentDir + "/Message.msg");
```

**2. Ορίστε τις επιλογές μετατροπής:**
```csharp
MailConversionOptions options = new MailConversionOptions { ConvertAsTnef = true };
```

**3. Μετατροπή και επεξεργασία:**
```csharp
using Aspose.Email;

MailMessage mail = msg.ToMailMessage(options);
// Περαιτέρω επεξεργασία ή αποθήκευση της «αλληλογραφίας» μπορεί να γίνει εδώ.
```

### Φόρτωση και αποθήκευση αρχείου email
#### Επισκόπηση
Η φόρτωση και αποθήκευση μηνυμάτων ηλεκτρονικού ταχυδρομείου σε διάφορες μορφές είναι απαραίτητη. Αυτή η ενότητα παρουσιάζει τη φόρτωση ενός αρχείου ηλεκτρονικού ταχυδρομείου και την αποθήκευσή του ως MSG.

**1. Φορτώστε το μήνυμα ηλεκτρονικού ταχυδρομείου:**
```csharp
using Aspose.Email;
using System.IO;

string documentDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

MailMessage loadedEmail = MailMessage.Load(documentDir + "/SampleEmail.eml");
```

**2. Αποθήκευση σε άλλη μορφή:**
```csharp
loadedEmail.Save(outputDir + "/ConvertedEmail.msg", SaveOptions.DefaultMsgUnicode);
// Η παράμετρος SaveOptions καθορίζει τη μορφή εξόδου.
```
