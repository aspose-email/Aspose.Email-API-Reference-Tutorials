### Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: την ανίχνευση της μορφής TNEF και τη μετατροπή μορφών email.

#### Εντοπισμός εάν ένα μήνυμα ηλεκτρονικού ταχυδρομείου είναι σε μορφή TNEF

Αυτή η λειτουργία σάς βοηθά να προσδιορίσετε εάν ένα δεδομένο `.eml` Το αρχείο είναι ενθυλακωμένο σε TNEF, μια ιδιόκτητη μορφή της Microsoft που χρησιμοποιείται για email με πλούσια μορφοποίηση.

**Φόρτωση του email**
```csharp
using System;
using System.IO;
using Aspose.Email;

// Ορίστε τη διαδρομή του καταλόγου εγγράφων σας.
string dataDir = "@YOUR_DOCUMENT_DIRECTORY/Message.eml";

// Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου από το αρχείο.
MailMessage mail = MailMessage.Load(new FileInfo(dataDir));
```

**Έλεγχος για μορφή TNEF**
```csharp
// Ελέγξτε αν η αρχική μορφή του email είναι TNEF.
bool isTnef = mail.IsTnef;

Console.WriteLine("The email is in TNEF format: " + isTnef);
```

- **Παράμετροι**: `IsTnef` Ελέγχει αν η αρχική μορφή του email ήταν TNEF. 
- **Επιστρεφόμενη τιμή**Επιστρέφει μια λογική τιμή που υποδεικνύει εάν το αρχείο είναι TNEF.

#### Αποθήκευση μηνύματος ηλεκτρονικού ταχυδρομείου σε διαφορετική μορφή

Αυτό το χαρακτηριστικό δείχνει τη μετατροπή ενός `.eml` αρχείο σε ένα `.msg` αρχείο, το οποίο μπορεί να είναι χρήσιμο για συμβατότητα με διαφορετικά προγράμματα-πελάτες ηλεκτρονικού ταχυδρομείου.

**Φόρτωση και μετατροπή**
```csharp
using Aspose.Email;

// Ορίστε τις διαδρομές για τους καταλόγους εισόδου και εξόδου.
string dataDir = "@YOUR_DOCUMENT_DIRECTORY/Message.eml";
string outputDir = "@YOUR_OUTPUT_DIRECTORY/SavedEmail.msg";

// Φορτώστε το μήνυμα ηλεκτρονικού ταχυδρομείου από ένα αρχείο σε μορφή .eml.
MailMessage mail = MailMessage.Load(new FileInfo(dataDir));

// Μετατρέψτε και αποθηκεύστε το φορτωμένο email σε μορφή .msg χρησιμοποιώντας την κλάση MapiMessage.
MapiMessage mapiMsg = MapiMessage.FromMailMessage(mail);
mapiMsg.Save(outputDir);
```

- **Παράμετροι**: `FromMailMessage()` μετατρέπει ένα `MailMessage` να `MapiMessage`.
- **Μέθοδος Αποθήκευσης**: Το `save()` Η μέθοδος γράφει το μετατρεπόμενο μήνυμα στην καθορισμένη διαδρομή.
