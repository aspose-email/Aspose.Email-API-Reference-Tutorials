## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στη διαδικασία υλοποίησης, χωρισμένη σε χαρακτηριστικά για λόγους σαφήνειας.

### Χαρακτηριστικό 1: Φόρτωση και μετατροπή VCF σε MailMessage

#### Επισκόπηση
Ξεκινάμε φορτώνοντας ένα αρχείο επαφής VCF και μετατρέποντάς το σε ένα `MailMessage` αντικείμενο χρησιμοποιώντας Aspose.Email. Αυτό μας επιτρέπει να χειριζόμαστε τα δεδομένα επικοινωνίας μέσα στις λειτουργίες email απρόσκοπτα.

##### Βήμα 1: Φόρτωση του αρχείου VCF
Αρχικά, ορίστε τον κατάλογο όπου αποθηκεύονται τα αρχεία VCF:
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
```
Φορτώστε το αρχείο VCF χρησιμοποιώντας `MapiContact.FromVCard` μέθοδος:
```csharp
// Φόρτωση του αρχείου επαφής VCF
MapiContact contact = MapiContact.FromVCard(documentDirectory + "/Contact.vcf");
```

##### Βήμα 2: Μετατροπή σε MailMessage
Μετατρέψτε το φορτωμένο VCF σε ένα `MailMessage` για περαιτέρω επεξεργασία. Χρησιμοποιούμε μια ροή μνήμης για να χειριστούμε αποτελεσματικά τη μετατροπή.
```csharp
// Μετατρέψτε το φορτωμένο VCF σε MailMessage
MemoryStream ms = new MemoryStream();
contact.Save(ms, ContactSaveFormat.Msg);
ms.Position = 0;
MapiMessage msg = MapiMessage.FromStream(ms);

MailConversionOptions conversionOptions = new MailConversionOptions();
MailMessage mailMessage = msg.ToMailMessage(conversionOptions);
```

### Δυνατότητα 2: Προετοιμασία και αποθήκευση ως MHTML με στοιχεία επικοινωνίας

#### Επισκόπηση
Στη συνέχεια, ετοιμάζουμε το `MailMessage` για να το αποδώσετε σε μορφή MHTML. Αυτό περιλαμβάνει στοιχεία επικοινωνίας για μια ολοκληρωμένη προβολή.

##### Βήμα 3: Ρύθμιση επιλογών αποθήκευσης
Προετοιμάστε τις απαραίτητες επιλογές για να αποθηκεύσετε το email ως αρχείο MHT:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

MhtSaveOptions mhtSaveOptions = new MhtSaveOptions();
mhtSaveOptions.CheckBodyContentEncoding = true;
mhtSaveOptions.PreserveOriginalBoundaries = true;

// Ορίστε επιλογές μορφοποίησης για να συμπεριλάβετε κεφαλίδες επαφών και πληροφορίες VCard
MhtFormatOptions formatOptions = MhtFormatOptions.WriteHeader | MhtFormatOptions.RenderVCardInfo;
mhtSaveOptions.RenderedContactFields = ContactFieldsSet.NameInfo | ContactFieldsSet.PersonalInfo |
                                       ContactFieldsSet.Telephones | ContactFieldsSet.Events;

mhtSaveOptions.MhtFormatOptions = formatOptions;
```

##### Βήμα 4: Αποθήκευση ως MHTML
Τέλος, αποθηκεύστε το `MailMessage` ως αρχείο MHTML με στοιχεία επικοινωνίας:
```csharp
// Αποθήκευση του MailMessage ως αρχείο MHT
mailMessage.Save(outputDirectory + "/ContactMhtml_out.mhtml", mhtSaveOptions);
```
