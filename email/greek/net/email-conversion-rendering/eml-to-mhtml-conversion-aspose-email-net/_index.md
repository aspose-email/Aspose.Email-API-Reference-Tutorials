## Οδηγός Εφαρμογής

### Φόρτωση και μετατροπή EML σε MHTML

Σε αυτήν την ενότητα, θα δούμε πώς να φορτώνουμε ένα αρχείο EML και να το μετατρέπουμε σε μορφή MHTML.

#### Βήμα 1: Ορισμός διαδρομών αρχείων

Αρχικά, καθορίστε πού βρίσκεται το αρχείο EML εισόδου και πού θέλετε να αποθηκεύσετε το αρχείο MHTML εξόδου:

```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

string emlFilePath = Path.Combine(documentDirectory, "Message.eml");
string mhtmlOutputPath = Path.Combine(outputDirectory, "AnEmail_out.mthml");
```

#### Βήμα 2: Φόρτωση του αρχείου EML

Χρήση `MailMessage.Load()` για να φορτώσετε το αρχείο EML σας σε ένα `MailMessage` αντικείμενο:

```csharp
// Η κλάση MailMessage χρησιμοποιείται για τον χειρισμό μηνυμάτων ηλεκτρονικού ταχυδρομείου.
MailMessage eml = MailMessage.Load(emlFilePath);
```

Αυτή η μέθοδος διαβάζει τα περιεχόμενα του αρχείου EML, συμπεριλαμβανομένων των κεφαλίδων και των συνημμένων.

#### Βήμα 3: Αποθήκευση ως MHTML

Τέλος, μετατρέψτε το `MailMessage` αντικείμενο σε ένα αρχείο MHTML χρησιμοποιώντας `Save()`:

```csharp
// Χρησιμοποιήστε την εντολή SaveOptions.DefaultMhtml για να καθορίσετε τη μορφή MHTML.
eml.Save(mhtmlOutputPath, SaveOptions.DefaultMhtml);
```

Αυτό το βήμα διασφαλίζει ότι το περιεχόμενο και τα συνημμένα του ηλεκτρονικού ταχυδρομείου διατηρούνται σε ένα μόνο αρχείο HTML.
