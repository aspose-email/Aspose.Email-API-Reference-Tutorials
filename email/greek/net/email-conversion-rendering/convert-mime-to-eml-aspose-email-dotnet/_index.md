Τώρα που έχετε ρυθμίσει το Aspose.Email στο έργο σας, ας εφαρμόσουμε τη λειτουργία μετατροπής MIME σε EML. Θα την αναλύσουμε σε εύκολα βήματα.

### Μετατροπή μηνύματος MIME σε EML (H2)

Αυτή η λειτουργία επιτρέπει τη μετατροπή ενός μηνύματος MIME σε αρχείο EML, το οποίο συχνά απαιτείται για εργασίες αρχειοθέτησης email ή μετεγκατάστασης.

#### Βήμα 1: Φόρτωση του μηνύματος MIME

Αρχικά, φορτώστε το μήνυμα MIME σας από ένα αρχείο:

```csharp
using Aspose.Email.Mime;
using System.IO;

// Ορισμός καταλόγων (αντικατάσταση με πραγματικές διαδρομές)
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY/";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY/";

// Φόρτωση του μηνύματος MIME
MailMessage msg = MailMessage.Load(documentDirectory + "Message2.msg");
```

**Εξήγηση**: Εδώ, χρησιμοποιούμε `MailMessage.Load()` για να διαβάσετε ένα μήνυμα από ένα αρχείο. Αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY/"` και `"YOUR_OUTPUT_DIRECTORY/"` με τις συγκεκριμένες διαδρομές σας.

#### Βήμα 2: Αποθήκευση του μηνύματος ως EML

Στη συνέχεια, αποθηκεύστε αυτό το φορτωμένο μήνυμα σε μορφή EML:

```csharp
// Μετατροπή και αποθήκευση του μηνύματος MIME ως αρχείο EML
msg.Save(outputDirectory + "ConvertMIMEMessageToEML_out.eml");
```

**Εξήγηση**: Το `Save()` Η μέθοδος γράφει το email στο δίσκο. Βεβαιωθείτε ότι ο κατάλογος εξόδου σας είναι εγγράψιμος.
