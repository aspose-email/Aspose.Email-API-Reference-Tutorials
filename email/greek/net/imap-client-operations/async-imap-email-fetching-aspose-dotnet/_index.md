## Οδηγός Εφαρμογής

Ας εξερευνήσουμε πώς να εφαρμόσουμε ασύγχρονη ανάκτηση email IMAP χρησιμοποιώντας το Aspose.Email για .NET.

### Βασική ασύγχρονη ανάκτηση email

Αυτή η ενότητα καλύπτει τη βασική μέθοδο ασύγχρονης λήψης μηνυμάτων ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας `BeginFetchMessage` και `EndFetchMessage`.

#### Βήμα 1: Αρχικοποίηση του ImapClient
```csharp
ImapClient client = new ImapClient();
client.Host = "domain.com";
client.Username = "username";
client.Password = "password";
```

#### Βήμα 2: Επιλέξτε τον φάκελο ηλεκτρονικού ταχυδρομείου
```csharp
client.SelectFolder("InBox");
```

#### Βήμα 3: Ξεκινήστε την ασύγχρονη λήψη μηνυμάτων
Ανάκτηση email χρησιμοποιώντας ασύγχρονες μεθόδους για την αποφυγή αποκλεισμού λειτουργιών.
```csharp
ImapMessageInfoCollection messages = client.ListMessages();
IAsyncResult res1 = client.BeginFetchMessage(messages[0].UniqueId);
IAsyncResult res2 = client.BeginFetchMessage(messages[1].UniqueId);

MailMessage msg1 = client.EndFetchMessage(res1);
MailMessage msg2 = client.EndFetchMessage(res2);
```

### Ασύγχρονη Ανάκτηση Ηλεκτρονικού Ταχυδρομείου με το ThreadPool

Η χρήση του .NET ThreadPool μπορεί να βελτιώσει την απόδοση διαχειριζόμενοι πολλαπλές λειτουργίες ανάκτησης ταυτόχρονα.

#### Βήμα 1: Αρχικοποίηση και ουρά εργασίας
```csharp
ImapClient client = new ImapClient();
client.Host = "domain.com";
client.Username = "username";
client.Password = "password";

List<MailMessage> messagesList = new List<MailMessage>();

ThreadPool.QueueUserWorkItem(delegate(object o)
{
    client.SelectFolder("folderName");
    ImapMessageInfoCollection messageInfos = client.ListMessages();

    foreach (ImapMessageInfo info in messageInfos)
    {
        messagesList.Add(client.FetchMessage(info.UniqueId));
    }
});
```

### Ασύγχρονη Ανάκτηση με Connection Scope και ThreadPool

Διασφαλίστε αποτελεσματική διαχείριση πόρων χρησιμοποιώντας εύρη σύνδεσης εντός της ομάδας νημάτων.

#### Βήμα 1: Υλοποίηση με χρήση πρότασης για διαχείριση σύνδεσης
```csharp
ImapClient client = new ImapClient();
client.Host = "domain.com";
client.Username = "username";
client.Password = "password";

List<MailMessage> messagesListWithScope = new List<MailMessage>();

ThreadPool.QueueUserWorkItem(delegate(object o)
{
    using (IDisposable connection = client.CreateConnection())
    {
        client.SelectFolder("FolderName");
        ImapMessageInfoCollection messageInfos = client.ListMessages();

        foreach (ImapMessageInfo info in messageInfos)
        {
            messagesListWithScope.Add(client.FetchMessage(info.UniqueId));
        }
    } // Η σύνδεση γίνεται εδώ
});
```
