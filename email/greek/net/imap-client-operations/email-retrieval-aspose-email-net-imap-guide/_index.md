### Χαρακτηριστικό: Ανάκτηση email που έφτασαν σήμερα
**Επισκόπηση:**
Ανακτήστε εύκολα τα email που έφτασαν σήμερα, χρησιμοποιώντας τις δυνατότητες ερωτημάτων του Aspose.Email.

#### Βήματα:
1. **Δημιουργήστε το ερώτημα για τα σημερινά email:**
```csharp
using Aspose.Email.Tools.Search;

MailQueryBuilder builder = new MailQueryBuilder();
builder.InternalDate.On(DateTime.Now);
```
2. **Εκτέλεση και ανάκτηση μηνυμάτων:**
```csharp
MailQuery query = builder.GetQuery();
ImapMessageInfoCollection messages = client.ListMessages(query);
Console.WriteLine($"Number of messages found: {messages.Count}");
```
### Δυνατότητα: Ανάκτηση μηνυμάτων ηλεκτρονικού ταχυδρομείου εντός εύρους ημερομηνιών
**Επισκόπηση:**
Αποκτήστε πρόσβαση σε email που έχετε λάβει εντός συγκεκριμένου εύρους ημερομηνιών, βελτιώνοντας τις δυνατότητες φιλτραρίσματος email σας.

#### Βήματα:
1. **Ορίστε το ερώτημα εύρους ημερομηνιών:**
```csharp
builder = new MailQueryBuilder();
builder.InternalDate.Before(DateTime.Now);
builder.InternalDate.Since(DateTime.Now.AddDays(-7));
```
2. **Εκτέλεση και ανάκτηση μηνυμάτων:**
```csharp
query = builder.GetQuery();
messages = client.ListMessages(query);
Console.WriteLine($"Number of messages found: {messages.Count}");
```
### Λειτουργία: Ανάκτηση μηνυμάτων ηλεκτρονικού ταχυδρομείου από συγκεκριμένο αποστολέα
**Επισκόπηση:**
Φιλτράρετε τα email που αποστέλλονται από έναν συγκεκριμένο αποστολέα για να βελτιστοποιήσετε τα εισερχόμενά σας.

#### Βήματα:
1. **Δημιουργήστε το ερώτημα για έναν συγκεκριμένο αποστολέα:**
```csharp
builder = new MailQueryBuilder();
builder.From.Contains("specific.sender@domain.com");
```
2. **Εκτέλεση και ανάκτηση μηνυμάτων:**
```csharp
query = builder.GetQuery();
messages = client.ListMessages(query);
Console.WriteLine($"Number of messages found: {messages.Count}");
```

**Επισκόπηση:**

**Επισκόπηση:**
Εστιάστε σε email που απευθύνονται σε έναν συγκεκριμένο παραλήπτη, ενισχύοντας την στοχευμένη επικοινωνία.

### Δυνατότητα: Ανάκτηση μηνυμάτων με προσαρμοσμένες σημαίες
**Επισκόπηση:**
Αξιοποιήστε προσαρμοσμένες σημαίες για να φιλτράρετε τα email με βάση συγκεκριμένα κριτήρια.

#### Βήματα:
1. **Ορίστε το ερώτημα που βασίζεται σε σημαία:**
```csharp

queryBuilder.HasFlags(ImapMessageFlags.Keyword("custom1"));
queryBuilder.HasNoFlags(ImapMessageFlags.Keyword("custom2"));
```
