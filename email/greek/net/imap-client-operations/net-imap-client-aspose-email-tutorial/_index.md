## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Αρχικοποίηση προγράμματος-πελάτη Imap και διαμόρφωση ασφαλείας

#### Επισκόπηση
Αυτή η ενότητα καλύπτει τον τρόπο ρύθμισης ενός προγράμματος-πελάτη IMAP χρησιμοποιώντας το Aspose.Email και τη διαμόρφωση των ρυθμίσεων ασφαλείας του για ασφαλή επικοινωνία.

**Βήμα 1: Αρχικοποίηση του προγράμματος-πελάτη IMAP**

```csharp
using Aspose.Email.Clients;
using Aspose.Email.Clients.Imap;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Διαδρομή κράτησης θέσης

// Δημιουργήστε μια νέα παρουσία του ImapClient με στοιχεία διακομιστή και διαπιστευτήρια.
ImapClient client = new ImapClient("imap.gmail.com", 993, "user@gmail.com", "password");
```

- **Παράμετροι**: 
  - Διεύθυνση διακομιστή: `"imap.gmail.com"` για το Gmail
  - Λιμάνι: `993` για συνδέσεις SSL
  - Όνομα χρήστη και κωδικός πρόσβασης: Τα διαπιστευτήριά σας μέσω email

**Βήμα 2: Ρύθμιση παραμέτρων επιλογών ασφαλείας**

```csharp
// Ορίστε τη λειτουργία ασφαλείας σε Αυτόματη για να επιτρέψετε την αυτόματη διαπραγμάτευση SSL/TLS.
client.SecurityOptions = SecurityOptions.Auto;
```

- **Γιατί**Εξασφαλίζει ασφαλή επικοινωνία ενεργοποιώντας το αυτόματο SSL/TLS.

### Λειτουργία 2: Ανάκτηση και αποθήκευση μηνυμάτων από διακομιστή IMAP

#### Επισκόπηση
Μάθετε πώς να ανακτάτε μηνύματα από τα εισερχόμενα ενός διακομιστή email και να τα αποθηκεύετε τοπικά ως αρχεία EML χρησιμοποιώντας το Aspose.Email για .NET.

**Βήμα 1: Ανάκτηση λίστας μηνυμάτων**

```csharp
try
{
    // Ανάκτηση της λίστας αντικειμένων πληροφοριών μηνύματος από τα ΕΙΣΕΡΧΟΜΕΝΑ.
    ImapMessageInfoCollection list = client.ListMessages();
    
    for (int i = 0; i < list.Count; i++)
    {
        string outputPath = "YOUR_OUTPUT_DIRECTORY" + list[i].UniqueId + ".eml";
        // Αποθηκεύστε κάθε μήνυμα χρησιμοποιώντας το μοναδικό του αναγνωριστικό ως όνομα αρχείου.
        client.SaveMessage(list[i].UniqueId, outputPath);
    }
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message); // Καταγράψτε ή εμφανίστε τυχόν σφάλματα που εντοπίστηκαν.
}
```

- **Παράμετροι**: 
  - `list[i].UniqueId`: Μοναδικό αναγνωριστικό της διεύθυνσης ηλεκτρονικού ταχυδρομείου που χρησιμοποιείται για την ονομασία αρχείων.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι τα διαπιστευτήρια διακομιστή είναι σωστά και ότι τα δικαιώματα επιτρέπουν την πρόσβαση στα INBOX.
- Επαληθεύστε ότι η σύνδεση δικτύου και οι ρυθμίσεις του τείχους προστασίας επιτρέπουν την κυκλοφορία IMAP στη θύρα 993.
