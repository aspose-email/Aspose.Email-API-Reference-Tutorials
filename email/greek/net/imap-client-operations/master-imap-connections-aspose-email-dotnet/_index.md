## Οδηγός Εφαρμογής

### Σύνδεση και σύνδεση στον διακομιστή IMAP

Η δημιουργία σύνδεσης με έναν διακομιστή IMAP με το Aspose.Email είναι απλή:

**Επισκόπηση:**
Αυτή η λειτουργία επιτρέπει ασφαλείς συνδέσεις με τον διακομιστή IMAP του παρόχου email σας, επιτρέποντάς σας να κάνετε έλεγχο ταυτότητας χρησιμοποιώντας τα διαπιστευτήριά σας.

**Βήματα Υλοποίησης:**

#### 1. Ρύθμιση λεπτομερειών σύνδεσης

Ρυθμίστε τον κεντρικό υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασής σας ως εξής:

```csharp
const string host = "your-host.com"; // Αντικατάσταση με τον πραγματικό κεντρικό υπολογιστή
const int port = 993; // Ασφαλής θύρα IMAP (IMAPS)
const string username = "user@host.com"; // Η διεύθυνση ηλεκτρονικού ταχυδρομείου σας
const string password = "password"; // Ο κωδικός πρόσβασης ηλεκτρονικού ταχυδρομείου σας
```

#### 2. Δημιουργήστε μια παρουσία του ImapClient

```csharp
ImapClient client = new ImapClient(host, port, username, password);
```
**Εξήγηση:**
Ο `ImapClient` δημιουργείται με λεπτομέρειες σύνδεσης για να διευκολύνει την επικοινωνία με τον διακομιστή.

#### 3. Συνδεθείτε στον διακομιστή IMAP

Χρησιμοποιήστε ένα μπλοκ try-catch για τον χειρισμό σφαλμάτων:

```csharp
try
{
    client.Connect(true);
}
catch (Exception ex)
{
    throw new Exception("Failed to connect and log into IMAP server: " + ex.Message);
}
```
**Γιατί αυτή η προσέγγιση;**
Το μπλοκ try-catch διασφαλίζει τον ομαλό χειρισμό των σφαλμάτων σύνδεσης, βοηθώντας στην αποσφαλμάτωση προβλημάτων όπως λανθασμένα διαπιστευτήρια ή προβλήματα δικτύου.

### Δημιουργία σύνθετων ερωτημάτων με συνθήκες AND

Η δημιουργία ερωτημάτων επιτρέπει την πραγματοποίηση πιο εξειδικευμένων αναζητήσεων μέσω email. Ας δημιουργήσουμε ένα ερώτημα χρησιμοποιώντας τη λογική συνθήκη AND:

#### Επισκόπηση

Αυτή η λειτουργία βοηθά στον περιορισμό των αποτελεσμάτων αναζήτησης συνδυάζοντας πολλαπλές συνθήκες που πρέπει να πληρούνται όλες.

**Βήματα Υλοποίησης:**

#### 1. Αρχικοποίηση του MailQueryBuilder

```csharp
MailQueryBuilder builder = new MailQueryBuilder();
```

#### 2. Ορισμός Συνθηκών Ερωτήματος

Συνδυάστε κριτήρια για πιο συγκεκριμένες αναζητήσεις:

```csharp
builder.From.Contains("SpecificHost.com");
builder.InternalDate.Before(DateTime.Now);
builder.InternalDate.Since(DateTime.Now.AddDays(-7));
```
**Εξήγηση:**
Το ερώτημα φιλτράρει τα μηνύματα ηλεκτρονικού ταχυδρομείου από έναν συγκεκριμένο τομέα που ελήφθησαν εντός της τελευταίας εβδομάδας.

#### 3. Ανάκτηση τελικού αντικειμένου ερωτήματος

```csharp
MailQuery query = builder.GetQuery();
```

### Συνδυασμός ερωτημάτων με συνθήκες OR

Συνδυάστε τις συνθήκες αναζήτησης χρησιμοποιώντας λογικό Ή για ευρύτερες αναζητήσεις:

**Επισκόπηση:**
Αυτή η λειτουργία παρέχει ευελιξία στην ανάκτηση μηνυμάτων ηλεκτρονικού ταχυδρομείου που πληρούν οποιοδήποτε από τα καθορισμένα κριτήρια.

#### Βήματα Υλοποίησης:

#### 1. Αρχικοποιήστε ξανά το MailQueryBuilder

```csharp
builder = new MailQueryBuilder(); // Επαναφορά builder
```

#### 2. Ορίστε Συνθήκες Ή

```csharp
builder.Or(
    builder.Subject.Contains("test"),
    builder.From.Contains("noreply@host.com")
);
```
**Εξήγηση:**
Αυτό το ερώτημα λαμβάνει μηνύματα ηλεκτρονικού ταχυδρομείου είτε με θέμα "test" είτε από έναν συγκεκριμένο αποστολέα.

#### 3. Ανάκτηση τελικού αντικειμένου ερωτήματος

```csharp
query = builder.GetQuery();
```
