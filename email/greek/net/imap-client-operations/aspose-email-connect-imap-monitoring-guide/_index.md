## Οδηγός Εφαρμογής

### Λειτουργία 1: Σύνδεση και σύνδεση στον διακομιστή IMAP

**Επισκόπηση:** Η σύνδεση σε έναν διακομιστή IMAP είναι το πρώτο βήμα στη διαχείριση των email μέσω προγραμματισμού. Εδώ, θα χρησιμοποιήσουμε `ImapClient` από το Aspose.Email για .NET.

#### Βήμα προς βήμα εφαρμογή:

**3.1 Αρχικοποίηση ImapClient**

```csharp
using Aspose.Email.Clients.Imap;

// Δημιουργήστε μια νέα παρουσία του ImapClient και συνδεθείτε στον διακομιστή.
ImapClient client = new ImapClient("imap.domain.com", "username", "password");
```

- **Παράμετροι:**
  - `"imap.domain.com"`: Αντικαταστήστε με τη διεύθυνση του διακομιστή IMAP σας.
  - `"username"`, `"password"`: Τα διαπιστευτήρια σύνδεσής σας.

**3.2 Σύνδεση με τον διακομιστή**

Βεβαιωθείτε ότι χειρίζεστε εξαιρέσεις κατά τη σύνδεση για ισχυρή διαχείριση σφαλμάτων.

### Λειτουργία 2: Έναρξη παρακολούθησης συμβάντων IMAP

**Επισκόπηση:** Η παρακολούθηση σε πραγματικό χρόνο συμβάντων email, όπως νέα ή διαγραμμένα μηνύματα, μπορεί να βελτιώσει την ανταπόκριση και τη λειτουργικότητα της εφαρμογής σας.

#### Βήμα προς βήμα εφαρμογή:

**3.3 Ρύθμιση παρακολούθησης συμβάντων**

```csharp
using System.Threading;
using Aspose.Email.Clients.Imap;

// Αρχικοποιήστε ένα συμβάν χειροκίνητης επαναφοράς για να χειριστείτε ασύγχρονες ειδοποιήσεις.
ManualResetEvent manualResetEvent = new ManualResetEvent(false);
ImapMonitoringEventArgs eventArgs = null;

// Έναρξη παρακολούθησης συμβάντων IMAP.
client.StartMonitoring(delegate(object sender, ImapMonitoringEventArgs e)
{
    eventArgs = e; // Καταγράψτε τα ορίσματα συμβάντων
    manualResetEvent.Set(); // Σήμα ότι έχει συμβεί ένα συμβάν
});

Thread.Sleep(2000); // Αφήστε λίγο χρόνο για να συμβούν τα γεγονότα
```

- **Διαμόρφωση κλειδιού:** Χρήση `StartMonitoring` μέθοδος με έναν πληρεξούσιο για τη διαχείριση ειδοποιήσεων.
  
**3.4 Χειρισμός ειδοποιήσεων**
Ο `manualResetEvent` βοηθά στον συγχρονισμό της διαδικασίας παρακολούθησης, σηματοδοτώντας όταν συμβαίνει ένα συμβάν.

### Δυνατότητα 3: Αποστολή email χρησιμοποιώντας το πρόγραμμα-πελάτη SMTP

**Επισκόπηση:** Η αποστολή email γίνεται απλή με το `SmtpClient` κλάση στο Aspose.Email για .NET.

#### Βήμα προς βήμα εφαρμογή:

**3.5 Αρχικοποίηση SmtpClient**

```csharp
using Aspose.Email.Clients.Smtp;
using Aspose.Email.Mime;

// Δημιουργήστε μια παρουσία του SmtpClient.
SmtpClient smtpClient = new SmtpClient("exchange.aspose.com", "username", "password");
```

- **Παράμετροι:**
  - `"exchange.aspose.com"`: Διεύθυνση διακομιστή SMTP.
  - `"username"`, `"password"`: Διαπιστευτήρια για την αποστολή email.

**3.6 Αποστολή email**

```csharp
// Δημιουργήστε και στείλτε ένα νέο μήνυμα ηλεκτρονικού ταχυδρομείου.
smtpClient.Send(new MailMessage("from@aspose.com", "to@aspose.com",
    "EMAILNET-34875 - " + Guid.NewGuid(), 
    "EMAILNET-34875 Support for IMAP idle command"));

manualResetEvent.WaitOne(10000); // Περιμένετε να διεκπεραιωθούν τα συμβάντα
```

### Λειτουργία 4: Διακοπή παρακολούθησης συμβάντων IMAP

**Επισκόπηση:** Η ασφαλής διακοπή της διαδικασίας παρακολούθησης διασφαλίζει ότι η εφαρμογή σας μπορεί να διαχειρίζεται αποτελεσματικά τους πόρους.

#### Βήμα προς βήμα εφαρμογή:

**3.7 Διακοπή παρακολούθησης**

```csharp
// Χρησιμοποιήστε τη μέθοδο StopMonitoring για να διακόψετε την ακρόαση συμβάντων.
client.StopMonitoring("Inbox");

smtpClient.Send(new MailMessage("from@aspose.com", "to@aspose.com",
    "EMAILNET-34875 - " + Guid.NewGuid(), 
    "EMAILNET-34875 Support for IMAP idle command"));

manualResetEvent.WaitOne(5000); // Βεβαιωθείτε ότι όλα τα συμβάντα διεκπεραιώνονται
```
