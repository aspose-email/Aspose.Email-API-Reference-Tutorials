## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε την υλοποίηση σε βασικά χαρακτηριστικά: δημιουργία και διαμόρφωση ενός `ImapClient`και μετονομασία φακέλων σε έναν διακομιστή IMAP. 

### Δημιουργία και ρύθμιση παραμέτρων του ImapClient
**Επισκόπηση**: Αυτή η λειτουργία δείχνει τη ρύθμιση ενός `ImapClient` παράδειγμα για να συνδεθείτε με ασφάλεια με την υπηρεσία παροχής email IMAP.

#### Βήμα 1: Αρχικοποίηση του ImapClient
```csharp
using Aspose.Email.Clients.Imap;

// Δημιουργήστε μια παρουσία της κλάσης ImapClient
ImapClient client = new ImapClient();
```

#### Βήμα 2: Ορισμός παραμέτρων σύνδεσης
Θα χρειαστεί να καθορίσετε τα στοιχεία του διακομιστή IMAP, συμπεριλαμβανομένου του κεντρικού υπολογιστή, της θύρας και των διαπιστευτηρίων.
```csharp
client.Host = "imap.gmail.com"; // Αντικαταστήστε με τη διεύθυνση του διακομιστή IMAP σας
client.Username = "your.username@gmail.com"; // Το όνομα χρήστη του ηλεκτρονικού σας ταχυδρομείου
client.Password = "your.password"; // Ο κωδικός πρόσβασης ηλεκτρονικού ταχυδρομείου σας
client.Port = 993; // Τυπική θύρα IMAP SSL
client.SecurityOptions = SecurityOptions.Auto; // Αυτόματη διαχείριση επιλογών ασφαλείας
```
**Επεξήγηση παραμέτρων**:
- **Πλήθος**: Η διεύθυνση του διακομιστή IMAP.
- **Όνομα χρήστη και κωδικός πρόσβασης**Διαπιστευτήρια για την πρόσβαση στο γραμματοκιβώτιό σας.
- **Λιμάνι**Συνήθως, το 993 χρησιμοποιείται για ασφαλείς συνδέσεις μέσω SSL/TLS.
- **Επιλογές ασφαλείας**: Ορισμός σε `Auto` για την αυτόματη διαχείριση των πρωτοκόλλων ασφαλείας.

### Μετονομασία φακέλων σε διακομιστή IMAP
**Επισκόπηση**Μάθετε πώς να αλλάζετε ονόματα φακέλων απευθείας από την εφαρμογή .NET χρησιμοποιώντας την κλάση ImapClient του Aspose.Email.

#### Βήμα 3: Μετονομασία φακέλου
Αυτή η λειτουργία αλλάζει το όνομα ενός υπάρχοντος φακέλου στο γραμματοκιβώτιό σας:
```csharp
try
{
    // Προσπάθεια μετονομασίας του φακέλου 'Aspose' σε 'Client'
    client.RenameFolder("Aspose", "Client"); 
}
catch (Exception ex)
{
    Console.WriteLine(Environment.NewLine + ex.Message); // Χειριστείτε τις εξαιρέσεις με χάρη
}
```
**Επεξήγηση παραμέτρων**:
- **Παλιό όνομα φακέλου**: Το τρέχον όνομα του φακέλου που θέλετε να μετονομάσετε.
- **Νέο όνομα φακέλου**: Το επιθυμητό νέο όνομα για τον φάκελό σας.

#### Βήμα 4: Απόρριψη πόρων
Να απελευθερώνετε πάντα πόρους μετά τη χρήση:
```csharp
client.Dispose();
```
