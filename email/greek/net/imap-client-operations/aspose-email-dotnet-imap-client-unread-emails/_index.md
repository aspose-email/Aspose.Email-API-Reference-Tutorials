Αξιοποιώντας τις ισχυρές λειτουργίες του Aspose.Email, θα βελτιστοποιήσετε τη διαδικασία διαχείρισης email σας και θα διασφαλίσετε ότι δεν θα χάσετε ποτέ σημαντικά μηνύματα.

### Βασική Αρχικοποίηση και Ρύθμιση

Για να αρχικοποιήσετε ένα πρόγραμμα-πελάτη IMAP, ξεκινήστε δημιουργώντας μια παρουσία του `ImapClient` από το Aspose.Email. Ακολουθεί μια βασική ρύθμιση:

```csharp
using Aspose.Email.Clients;
using Aspose.Email.Clients.Imap;

// Αρχικοποιήστε τον πελάτη IMAP με τα στοιχεία του διακομιστή.
ImapClient imapClient = new ImapClient();
imapClient.Host = "<HOST>";  // Αντικαταστήστε το <HOST> με τη διεύθυνση του διακομιστή IMAP σας
imapClient.Port = 993;       // Κοινή θύρα για συνδέσεις SSL
imapClient.Username = "<USERNAME>";  // Το όνομα χρήστη του ηλεκτρονικού σας ταχυδρομείου
imapClient.Password = "<PASSWORD>";   // Ο κωδικός πρόσβασης ηλεκτρονικού ταχυδρομείου σας

// Ενεργοποίηση κρυπτογράφησης TLS και έμμεσης ασφάλειας SSL.
imapClient.SupportedEncryption = EncryptionProtocols.Tls;
imapClient.SecurityOptions = SecurityOptions.SSLImplicit;
```

## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε την υλοποίηση σε λογικά βήματα για να διαμορφώσουμε αποτελεσματικά τον πελάτη IMAP.

### Αρχικοποίηση προγράμματος-πελάτη IMAP με το Aspose.Email .NET

#### Επισκόπηση
Η αρχικοποίηση ενός προγράμματος-πελάτη IMAP περιλαμβάνει τη ρύθμιση των απαραίτητων ρυθμίσεων, όπως στοιχεία κεντρικού υπολογιστή, πρωτόκολλα κρυπτογράφησης και διαπιστευτήρια. Αυτή η ρύθμιση επιτρέπει την ασφαλή επικοινωνία με τον διακομιστή email.

#### Βήματα διαμόρφωσης

1. **Ορισμός κεντρικού υπολογιστή και θύρας**
   - Ορίστε τη διεύθυνση και τον αριθμό θύρας του διακομιστή IMAP (συνήθως 993 για SSL).

2. **Ρύθμιση παραμέτρων διαπιστευτηρίων**
   - Δώστε έγκυρο όνομα χρήστη και κωδικό πρόσβασης για έλεγχο ταυτότητας με τον διακομιστή.

3. **Ενεργοποίηση κρυπτογράφησης**
   - Χρησιμοποιήστε πρωτόκολλα κρυπτογράφησης TLS για ασφαλή μετάδοση δεδομένων.
   - Ορίστε τις επιλογές ασφαλείας σε `SSLImplicit` για την επιβολή ασφαλών συνδέσεων.

### Ρύθμιση παραμέτρων του IMAP Query Builder για μη αναγνωσμένα μηνύματα

#### Επισκόπηση
Το ImapQueryBuilder χρησιμοποιείται για το φιλτράρισμα μη αναγνωσμένων μηνυμάτων ηλεκτρονικού ταχυδρομείου, διασφαλίζοντας ότι επεξεργάζεστε μόνο μηνύματα που δεν έχουν αναγνωστεί ακόμη.

#### Βήματα Υλοποίησης

1. **Δημιουργία μιας παρουσίας QueryBuilder**
   ```csharp
   using Aspose.Email.Tools.Search;

   ImapQueryBuilder imapQueryBuilder = new ImapQueryBuilder();
   ```

2. **Ορισμός κριτηρίων μη αναγνωσμένου μηνύματος**
   - Κριτήρια φιλτραρίσματος για τον εντοπισμό μη αναγνωσμένων μηνυμάτων:
     ```csharp
     imapQueryBuilder.HasNoFlags(ImapMessageFlags.IsRead);
     ```

3. **Δημιουργήστε το ερώτημα**
   ```csharp
   MailQuery query = imapQueryBuilder.GetQuery();
   ```

### Ορίστε τον πελάτη IMAP σε λειτουργία μόνο για ανάγνωση και επιλέξτε φάκελο

#### Επισκόπηση
Για να περιηγηθείτε με ασφάλεια στα email χωρίς να κάνετε αλλαγές, διαμορφώστε τον υπολογιστή-πελάτη σας σε λειτουργία μόνο για ανάγνωση και επιλέξτε τον επιθυμητό φάκελο για λειτουργίες.

#### Βήματα Υλοποίησης

1. **Ενεργοποίηση λειτουργίας μόνο για ανάγνωση**
   ```csharp
   imapClient.ReadOnly = true;
   ```

2. **Επιλογή φακέλου εισερχομένων**
   - Επιλέξτε τα "Εισερχόμενα" ως τον προεπιλεγμένο φάκελο για να λειτουργήσετε:
     ```csharp
     imapClient.SelectFolder("Inbox");
     ```

### Λίστα μη αναγνωσμένων μηνυμάτων στον επιλεγμένο φάκελο

#### Επισκόπηση
Αυτή η λειτουργία ανακτά και εμφανίζει σε λίστα όλα τα μη αναγνωσμένα μηνύματα από τον επιλεγμένο φάκελο χρησιμοποιώντας το κατασκευασμένο ερώτημα.

#### Βήματα Υλοποίησης

1. **Ανάκτηση μη αναγνωσμένων μηνυμάτων**
   - Χρήση `ListMessages` μέθοδος με το προηγουμένως ορισμένο ερώτημα:
     ```csharp
     ImapMessageInfoCollection messageInfoCol = imapClient.ListMessages(query);
     Console.WriteLine("Initial Unread Count: " + messageInfoCol.Count());
     ```

2. **Επιβεβαίωση συμπεριφοράς μόνο για ανάγνωση**
   - Επαναλάβετε την ανάκτηση μηνυμάτων για να διασφαλίσετε ότι η καταμέτρηση παραμένει αμετάβλητη σε λειτουργία μόνο για ανάγνωση:
     ```csharp
     if (messageInfoCol.Count() > 0)
     {
         imapClient.FetchMessage(messageInfoCol[0].SequenceNumber);
         
         messageInfoCol = imapClient.ListMessages(query);
         Console.WriteLine("Updated Unread Count: " + messageInfoCol.Count());
     }
     else
     {
         Console.WriteLine("No unread messages found");
     }
     ```
