## Οδηγός Εφαρμογής
### Ασύγχρονη λίστα email IMAP
Η λειτουργία που θα εφαρμόσουμε σάς επιτρέπει να παραθέτετε μηνύματα από έναν διακομιστή IMAP ασύγχρονα, κάτι που είναι ιδανικό για λειτουργίες χωρίς αποκλεισμό στην εφαρμογή σας.

#### Βήμα προς βήμα εφαρμογή
**1. Αρχικοποίηση του ImapClient**
Ξεκινήστε ρυθμίζοντας το `ImapClient` με τα στοιχεία του παρόχου email σας:

```csharp
// Δημιουργία και ρύθμιση παραμέτρων της παρουσίας ImapClient
ImapClient client = new ImapClient();
client.Host = "imap.gmail.com";
client.Username = "your.username@gmail.com";
client.Password = "your.password";
client.Port = 993;
client.SecurityOptions = SecurityOptions.Auto;
```

**2. Δημιουργήστε το ερώτημα αναζήτησης**
Χρήση `ImapQueryBuilder` για να δημιουργήσετε ένα ερώτημα που φιλτράρει τα μηνύματα ηλεκτρονικού ταχυδρομείου κατά θέμα:

```csharp
// Δημιουργήστε ένα ερώτημα αναζήτησης για μηνύματα ηλεκτρονικού ταχυδρομείου με θέμα «Θέμα» στη γραμμή θέματος
ImapQueryBuilder builder = new ImapQueryBuilder();
builder.Subject.Contains("Subject");
MailQuery query = builder.GetQuery();
```

**3. Ασύγχρονη λίστα μηνυμάτων**
Εκτελέστε την ασύγχρονη λειτουργία για να εμφανίσετε μηνύματα που ταιριάζουν με τα κριτήριά σας:

```csharp
try
{
    // Έναρξη ασύγχρονης καταχώρισης μηνυμάτων χρησιμοποιώντας το καθορισμένο ερώτημα
    IAsyncResult asyncResult = client.BeginListMessages(query);

    // Ολοκληρώστε τη λειτουργία και ανακτήστε τα αποτελέσματα
    ImapMessageInfoCollection messages = client.EndListMessages(asyncResult);
    
    foreach (var message in messages)
    {
        Console.WriteLine($"Subject: {message.Subject}");
    }
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    // Καθαρίστε τους πόρους
    if (client != null) client.Dispose();
}
```
