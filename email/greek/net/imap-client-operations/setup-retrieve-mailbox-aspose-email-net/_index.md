## Οδηγός Εφαρμογής

Θα αναλύσουμε αυτήν την υλοποίηση σε δύο κύρια χαρακτηριστικά: τη δημιουργία ενός `ExchangeClient` παρουσία και ανάκτηση πληροφοριών γραμματοκιβωτίου.

### Δυνατότητα 1: Δημιουργία μιας παρουσίας ExchangeClient

#### Επισκόπηση
Αυτή η ενότητα σας καθοδηγεί στην αρχικοποίηση του `ExchangeClient`, που χρησιμεύει ως πύλη για την αλληλεπίδραση με τις λειτουργίες του Exchange Server.

#### Βήμα προς βήμα εφαρμογή

**Αρχικοποίηση διαπιστευτηρίων:**
Ξεκινήστε ρυθμίζοντας τα διαπιστευτήρια σύνδεσής σας, συμπεριλαμβανομένης της διεύθυνσης URL του διακομιστή, του ονόματος χρήστη, του κωδικού πρόσβασης και του τομέα.

```csharp
using Aspose.Email.Clients.Exchange;

// Ορισμός παραμέτρων σύνδεσης για τον Exchange Server
string serverUrl = "https://Όνομα_υπολογιστή/ανταλλαγή/Όνομα_χρήστη";
string username = "Username";
string password = "password";
string domain = "domain";

// Δημιουργήστε μια παρουσία της κλάσης ExchangeClient
ExchangeClient client = new ExchangeClient(serverUrl, username, password, domain);
```

**Εξήγηση:**
- `serverUrl`: Η διεύθυνση URL προς τον διακομιστή Exchange. 
- `username`, `password`, και `domain`: Απαιτούνται διαπιστευτήρια για τον έλεγχο ταυτότητας.

### Δυνατότητα 2: Λήψη πληροφοριών γραμματοκιβωτίου από τον Exchange Server

#### Επισκόπηση
Μάθετε πώς να χρησιμοποιείτε το `ExchangeClient` μια παρουσία για την ανάκτηση πληροφοριών γραμματοκιβωτίου.

#### Βήμα προς βήμα εφαρμογή

**Ανάκτηση μεγέθους γραμματοκιβωτίου και λεπτομερών πληροφοριών:**
Χρησιμοποιήστε το `GetMailboxSize()` και `GetMailboxInfo()` μεθόδους για ολοκληρωμένες λεπτομέρειες γραμματοκιβωτίου.

```csharp
try
{
    // Λήψη του μεγέθους του γραμματοκιβωτίου σε byte
    long mailboxSize = client.GetMailboxSize();

    // Ανάκτηση λεπτομερών πληροφοριών γραμματοκιβωτίου
    ExchangeMailboxInfo mailboxInfo = client.GetMailboxInfo();
    
    // Παραδείγματα URI για επίδειξη (αντικατάσταση με πραγματικές διαδρομές)
    string inboxUri = mailboxInfo.InboxUri;
    string sentItemsUri = mailboxInfo.SentItemsUri;
    string draftsUri = mailboxInfo.DraftsUri;
}
catch (Exception ex)
{
    throw new Exception("An error occurred while retrieving mailbox information: " + ex.Message);
}
```

**Εξήγηση:**
- `GetMailboxSize()`: Ανακτά το τρέχον μέγεθος του γραμματοκιβωτίου σας σε byte.
- `GetMailboxInfo()`Παρέχει λεπτομερείς πληροφορίες, συμπεριλαμβανομένων των URI για διάφορους φακέλους όπως Εισερχόμενα, Απεσταλμένα και Πρόχειρα.
