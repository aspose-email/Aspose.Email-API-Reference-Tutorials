## Οδηγός Εφαρμογής
Θα καλύψουμε τρία βασικά χαρακτηριστικά: τη σύνδεση σε έναν διακομιστή IMAP, τη δημιουργία ενός φακέλου και τη μετακίνηση μηνυμάτων.

### Σύνδεση σε διακομιστή IMAP (H2)
#### Επισκόπηση
Η σύνδεση σε έναν διακομιστή IMAP είναι θεμελιώδης για τις εργασίες διαχείρισης email. Το Aspose.Email το απλοποιεί αυτό με το `ImapClient` τάξη.

#### Βήματα Υλοποίησης
##### Βήμα 1: Αρχικοποίηση του ImapClient
Δημιουργήστε μια νέα παρουσία του `ImapClient`, παρέχοντας τα στοιχεία του διακομιστή σας, τον αριθμό θύρας (συνήθως 993 για SSL), το όνομα χρήστη και τον κωδικό πρόσβασης:
```csharp
using (ImapClient client = new ImapClient("host.domain.com", 993, "username", "password"))
{
    Console.WriteLine("Connected to IMAP server successfully.");
}
```
**Εξήγηση**: Το `ImapClient` Ο κατασκευαστής δέχεται τη διεύθυνση κεντρικού υπολογιστή, τη θύρα, το όνομα χρήστη και τον κωδικό πρόσβασης. Τα τυλίγουμε σε ένα `using` δήλωση για την ορθή διάθεση των πόρων.

### Δημιουργία φακέλου σε λογαριασμό IMAP (H2)
#### Επισκόπηση
Η οργάνωση των email σε φακέλους είναι συνηθισμένη. Αυτή η λειτουργία ελέγχει την ύπαρξη φακέλου και τον δημιουργεί, εάν είναι απαραίτητο.

#### Βήματα Υλοποίησης
##### Βήμα 1: Έλεγχος ύπαρξης φακέλου
Χρησιμοποιήστε το `ExistFolder` μέθοδος για να επαληθεύσετε εάν ο επιθυμητός φάκελος υπάρχει στον διακομιστή:
```csharp
string folderName = "YOUR_DOCUMENT_DIRECTORY";

if (!client.ExistFolder(folderName))
{
    client.CreateFolder(folderName);
    Console.WriteLine($"Folder '{folderName}' created successfully.");
}
```
**Εξήγηση**: Αν `ExistFolder` επιστρέφει false, προχωράμε στη δημιουργία του φακέλου χρησιμοποιώντας `CreateFolder`.

### Μετακίνηση μηνύματος σε λογαριασμό IMAP (H2)
#### Επισκόπηση
Η μετακίνηση μηνυμάτων μεταξύ φακέλων μπορεί να βοηθήσει στη διαχείριση των ροών εργασίας email. Αυτή η λειτουργία δείχνει τη μετακίνηση ενός email με βάση το μοναδικό του αναγνωριστικό.

#### Βήματα Υλοποίησης
##### Βήμα 1: Προσθήκη και μετακίνηση μηνύματος
Αρχικά, επιλέξτε τα Εισερχόμενα για να εργαστείτε με μηνύματα. Στη συνέχεια, δημιουργήστε και προσθέστε ένα νέο μήνυμα πριν το μετακινήσετε σε άλλο φάκελο χρησιμοποιώντας το μοναδικό αναγνωριστικό του:
```csharp
string folderName = "YOUR_OUTPUT_DIRECTORY";

if (!client.ExistFolder(folderName))
    client.CreateFolder(folderName);

client.SelectFolder(ImapFolderInfo.InBox);
MailMessage message = new MailMessage(
    "from@domain.com",
    "to@domain.com",
    "Unique Message Subject - " + Guid.NewGuid(),
    "This is the body of the email.");

string uniqueId = client.AppendMessage(ImapFolderInfo.InBox, message);
client.MoveMessage(uniqueId, folderName);
Console.WriteLine($"Moved message with unique ID '{uniqueId}' to '{folderName}'.");
```
**Εξήγηση**Μετά την προσθήκη ενός νέου μηνύματος στα Εισερχόμενα, ανακτούμε το μοναδικό αναγνωριστικό του. Αυτό το αναγνωριστικό χρησιμοποιείται από `MoveMessage` για να το μετακινήσετε στον επιθυμητό φάκελο.
