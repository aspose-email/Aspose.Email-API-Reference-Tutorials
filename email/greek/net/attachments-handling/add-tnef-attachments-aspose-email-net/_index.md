## Οδηγός Εφαρμογής

Αυτή η ενότητα περιγράφει τον τρόπο προσθήκης συνημμένων TNEF χρησιμοποιώντας το Aspose.Email για .NET.

### Φόρτωση υπάρχοντος αρχείου email

Ξεκινήστε φορτώνοντας το αρχείο EML:

#### Βήμα 1: Αρχικοποίηση MailMessage
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
MailMessage mailMessage = MailMessage.Load(dataDir + "/tnefEml1.eml");
```
Ο `MailMessage` Η κλάση αντιπροσωπεύει ένα μήνυμα ηλεκτρονικού ταχυδρομείου. Η `Load` Η μέθοδος διαβάζει το αρχείο EML για περαιτέρω χειρισμό.

### Προσθήκη νέου συνημμένου

Για να προσθέσετε νέα συνημμένα διατηρώντας παράλληλα το TNEF, ακολουθήστε τα εξής βήματα:

#### Βήμα 2: Επισύναψη αρχείου
```csharp
mailMessage.Attachments.Add(new Attachment(File.OpenRead(dataDir + "/Untitled.jpg"), "Untitled.jpg", "image/jpg"));
```
Ο `Attachment` Η κλάση αντιπροσωπεύει το αρχείο που θέλετε να επισυνάψετε. Ανοίξτε την εικόνα σε λειτουργία ανάγνωσης και προσθέστε την στα συνημμένα του email.

### Ρύθμιση παραμέτρων επιλογών αποθήκευσης

Η διατήρηση των συνημμένων TNEF απαιτεί συγκεκριμένες επιλογές αποθήκευσης:

#### Βήμα 3: Ορισμός EmlSaveOptions
```csharp
EmlSaveOptions eo = new EmlSaveOptions(MailMessageSaveType.EmlFormat);
eo.FileCompatibilityMode = FileCompatibilityMode.PreserveTnefAttachments;
```
Ο `EmlSaveOptions` Η κλάση καθορίζει τον τρόπο αποθήκευσης του email. Η ρύθμιση `FileCompatibilityMode` να `PreserveTnefAttachments` διατηρεί όλα τα συνημμένα TNEF.

### Αποθήκευση του ενημερωμένου email

Τέλος, αποθηκεύστε το ενημερωμένο email σας με τα νέα συνημμένα:

#### Βήμα 4: Αποθήκευση αλλαγών
```csharp
mailMessage.Save(dataDir + "/test_out.eml", eo);
```
Αυτό το βήμα εγγράφει αλλαγές σε ένα νέο αρχείο EML, διατηρώντας όλα τα αρχικά συνημμένα μαζί με το νέο.
