## Οδηγός Εφαρμογής

### Αντικατάσταση ενσωματωμένου συνημμένου MSG

Αυτή η λειτουργία σάς επιτρέπει να αντικαταστήσετε ένα υπάρχον συνημμένο μέσα σε ένα μήνυμα MAPI. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Φόρτωση του αρχικού αρχείου MSG
Φορτώστε το αρχικό σας αρχείο MSG σε ένα `MapiMessage` αντικείμενο.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string fileName = dataDir + "/message3.msg"; // Αρχική διαδρομή αρχείου MSG

var message = MapiMessage.FromFile(fileName);
```

#### Βήμα 2: Προετοιμασία του εξαρτήματος για αντικατάσταση
Χρησιμοποιήστε ένα `MemoryStream` για να συγκρατήσετε προσωρινά το εξάρτημα που θέλετε να αντικαταστήσετε.

```csharp
using (var memoryStream = new MemoryStream())
{
    // Αποθηκεύστε το δεύτερο συνημμένο στο MemoryStream
    message.Attachments[2].Save(memoryStream);

    // Επαναφορά της θέσης της ροής στην αρχή
    memoryStream.Position = 0;
    
    // Φόρτωση ενός MapiMessage από το MemoryStream που περιέχει το συνημμένο
    var getData = MapiMessage.FromStream(memoryStream);
}
```

#### Βήμα 3: Αντικατάσταση του εξαρτήματος
Αντικαταστήστε ένα υπάρχον εξάρτημα με ένα που έχετε ετοιμάσει πρόσφατα.

```csharp
// Αντικαταστήστε το πρώτο συνημμένο με ένα νέο με το όνομα 'new 1'
message.Attachments.Replace(1, "new 1");
```

### Επεξήγηση Παραμέτρων και Μεθόδων
- **MapiMessage.FromFile**Φορτώνει ένα μήνυμα MAPI από ένα καθορισμένο αρχείο.
- **Ροή μνήμης**Χρησιμοποιείται για προσωρινή αποθήκευση κατά την επεξεργασία.
- **Συνημμένα[2].Αποθήκευση**Αποθηκεύει το συνημμένο στη ροή μνήμης.
- **μήνυμα.Συνημμένα.Αντικατάσταση**: Αντικαθιστά ένα υπάρχον συνημμένο με ένα νέο.

### Συμβουλές αντιμετώπισης προβλημάτων
- Χειρισμός εξαιρέσεων, ειδικά όταν πρόκειται για λειτουργίες εισόδου/εξόδου αρχείων.
- Επαληθεύστε ότι τα ευρετήρια συνημμένων υπάρχουν πριν από την πρόσβαση σε αυτά, για να αποτρέψετε σφάλματα χρόνου εκτέλεσης.
