## Οδηγός Εφαρμογής

### Εξαγωγή «Περιγραφής Περιεχομένου» από Συνημμένα Ηλεκτρονικού Ταχυδρομείου

Αυτή η ενότητα δείχνει πώς να ανακτήσετε μέσω προγραμματισμού την κεφαλίδα «Περιγραφή περιεχομένου».

#### Βήμα 1: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου
Φορτώστε το μήνυμα ηλεκτρονικού ταχυδρομείου σας χρησιμοποιώντας `MailMessage.Load()` παρέχοντας τη διαδρομή προς το αρχείο email:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
MailMessage message = MailMessage.Load(dataDir + "EmailWithAttandEmbedded.eml");
```
**Εξήγηση**: Αντικατάσταση `"YOUR_DOCUMENT_DIRECTORY"` με τον πραγματικό σας κατάλογο. Αυτό διασφαλίζει ότι το Aspose.Email διαβάζει και αναλύει το περιεχόμενο του email.

#### Βήμα 2: Ανάκτηση της «Περιγραφής Περιεχομένου»
Αποκτήστε πρόσβαση στην κεφαλίδα «Περιγραφή περιεχομένου» από το πρώτο συνημμένο:
```csharp
string description = message.Attachments[0].Headers["Content-Description"];
```
**Εξήγηση**Αυτή η γραμμή ανακτά την «Περιγραφή Περιεχομένου» για το πρώτο συνημμένο. Βεβαιωθείτε ότι το αρχείο email σας περιέχει συνημμένα με αυτήν τη συγκεκριμένη κεφαλίδα.
