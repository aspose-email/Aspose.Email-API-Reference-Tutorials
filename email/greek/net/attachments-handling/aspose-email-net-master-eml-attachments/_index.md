## Οδηγός Εφαρμογής
### Λειτουργία 1: Φόρτωση και εμφάνιση συνημμένων ηλεκτρονικού ταχυδρομείου
Αυτή η λειτουργία εστιάζει στη φόρτωση ενός αρχείου EML για πρόσβαση στα συνημμένα του. Δείτε πώς μπορείτε να την εφαρμόσετε:

#### Βήμα προς βήμα εφαρμογή
**Φόρτωση του αρχείου EML**
Ξεκινήστε δημιουργώντας ένα `MailMessage` και φορτώστε το αρχείο EML σας.

```csharp
using System;
using Aspose.Email.Mime;

// Αντικαταστήστε με την πραγματική διαδρομή καταλόγου εγγράφου σας
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";

// Δημιουργήστε μια παρουσία του MailMessage και φορτώστε ένα αρχείο email
MailMessage mailMsg = MailMessage.Load(dataDir + "/Test.eml", new EmlLoadOptions());
```

**Επανάληψη σε συνημμένα**
Περιστρέψτε κάθε συνημμένο στο `MailMessage` αντικείμενο.

```csharp
foreach (Attachment attachment in mailMsg.Attachments)
{
    // Αποθηκεύστε το συνημμένο στον καθορισμένο κατάλογο εξόδου
    attachment.Save("@YOUR_OUTPUT_DIRECTORY/" + attachment.Name);
}
```

**Εξήγηση**: 
- **Παράμετροι**: Η διαδρομή αρχείου και `EmlLoadOptions` καθορίστε τον τρόπο φόρτωσης του email.
- **Σκοπός**Αυτό το απόσπασμα κώδικα αποθηκεύει κάθε συνημμένο σε έναν καθορισμένο κατάλογο, επιτρέποντας εύκολη πρόσβαση αργότερα.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι το αρχείο EML υπάρχει στην καθορισμένη θέση.
- Βεβαιωθείτε ότι τα δικαιώματα του καταλόγου εξόδου σας έχουν οριστεί σωστά.

### Λειτουργία 2: Εξαγωγή και εμφάνιση ονομάτων συνημμένων
Στη συνέχεια, ας εξαγάγουμε και ας εμφανίσουμε τα ονόματα των συνημμένων από ένα αρχείο EML:

#### Βήμα προς βήμα εφαρμογή
**Πρόσβαση σε συνημμένα**
Χρησιμοποιήστε ένα `foreach` βρόχο για να αποκτήσετε πρόσβαση σε κάθε συνημμένο στο μήνυμα email που έχετε φορτώσει.

```csharp
using System;
using Aspose.Email.Mime;

// Αντικαταστήστε με την πραγματική διαδρομή καταλόγου εγγράφου σας
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";

MailMessage mailMsg = MailMessage.Load(dataDir + "/Test.eml", new EmlLoadOptions());

foreach (Attachment attachment in mailMsg.Attachments)
{
    // Εμφάνιση του ονόματος κάθε συνημμένου
    Console.WriteLine(attachment.Name);
}
```

**Εξήγηση**: 
- **Παράμετροι**: Όπως και πριν, αυτό χρησιμοποιεί επιλογές διαδρομής αρχείου και φόρτωσης.
- **Σκοπός**Επαναλαμβάνοντας τα συνημμένα, μπορείτε να εμφανίσετε τα ονόματά τους απευθείας στην κονσόλα σας, κάτι χρήσιμο για την καταγραφή ή τον εντοπισμό σφαλμάτων.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Εάν δεν εμφανίζονται ονόματα συνημμένων, ελέγξτε εάν το αρχείο EML περιέχει συνημμένα.
