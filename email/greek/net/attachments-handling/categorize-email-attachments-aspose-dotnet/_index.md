### Προσδιορισμός ενσωματωμένων και κανονικών συνημμένων
**Επισκόπηση:**
Ο πρωταρχικός στόχος είναι η διάκριση μεταξύ των ενσωματωμένων και των κανονικών συνημμένων σε ένα μήνυμα ηλεκτρονικού ταχυδρομείου, επιτρέποντας την καλύτερη οργάνωση και επεξεργασία του περιεχομένου του ηλεκτρονικού ταχυδρομείου.

#### Βήμα 1: Ορίστε τον κατάλογο εγγράφων σας
Ξεκινήστε καθορίζοντας τη διαδρομή όπου αποθηκεύονται τα email σας:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/RemoveAttachments.msg";
```
**Εξήγηση**: Αντικατάσταση `YOUR_DOCUMENT_DIRECTORY` με την πραγματική διαδρομή καταλόγου όπου βρίσκονται τα αρχεία email σας. Αυτή η ρύθμιση διασφαλίζει ότι ο κώδικας εντοπίζει και επεξεργάζεται σωστά το καθορισμένο αρχείο.

#### Βήμα 2: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου
Χρησιμοποιήστε το Aspose.Email για να φορτώσετε το μήνυμα από ένα αρχείο:
```csharp
var message = MapiMessage.FromFile(dataDir);
```
**Εξήγηση**: `MapiMessage.FromFile` Διαβάζει ένα email που είναι αποθηκευμένο σε μορφή MSG, προετοιμάζοντάς το για επεξεργασία συνημμένων.

#### Βήμα 3: Επανάληψη μέσω συνημμένων
Επαναλάβετε κάθε συνημμένο και προσδιορίστε τον τύπο του χρησιμοποιώντας την ακόλουθη λογική:
```csharp
var attachments = message.Attachments;

for (int i = 0; i < attachments.Count; i++)
{
    var attachment = attachments[i];
    
    if (IsInlineAttachment(attachment, message))
    {
        System.Console.WriteLine($"{attachment.LongFileName} is inline attachment");
    }
}
```
**Εξήγηση**: Το `IsInlineAttachment` Η μέθοδος ελέγχει εάν ένα συνημμένο πρέπει να κατηγοριοποιηθεί ως ενσωματωμένο με βάση τα συμφραζόμενα στο σώμα του email. Τα συνημμένα σε γραμμή είναι συνήθως εικόνες ή αρχεία CSS ενσωματωμένα σε email HTML.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Προβλήματα διαδρομής αρχείου**Βεβαιωθείτε ότι η διαδρομή του αρχείου σας έχει ρυθμιστεί σωστά και είναι προσβάσιμη.
- **Λανθασμένη ταξινόμηση τύπου συνημμένου**: Ελέγξτε ξανά το `IsInlineAttachment` λογική για να διασφαλιστεί ότι ευθυγραμμίζεται με τον τρόπο που ορίζονται οι ενσωματωμένοι πόροι στη μορφή email σας.
