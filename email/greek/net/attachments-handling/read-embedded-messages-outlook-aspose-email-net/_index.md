### Ανάγνωση ενσωματωμένου μηνύματος από συνημμένο

#### Επισκόπηση

Η εξαγωγή μηνυμάτων που είναι ενσωματωμένα σε συνημμένα μπορεί να είναι δύσκολη, αλλά με το Aspose.Email, είναι απλή. Αυτή η λειτουργία επιτρέπει στους προγραμματιστές να διαβάζουν και να επεξεργάζονται αυτά τα κρυφά μηνύματα αποτελεσματικά.

#### Βήματα Υλοποίησης

1. **Ρυθμίστε το περιβάλλον σας**
   
   Ορίστε τον κατάλογο στον οποίο βρίσκεται το έγγραφό σας:
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Βεβαιωθείτε ότι αυτό έχει ρυθμιστεί σωστά
   ```

2. **Φόρτωση του μηνύματος MAPI**

   Φόρτωση αρχείου μηνύματος χρησιμοποιώντας το Aspose.Email `MapiMessage` τάξη.
   
   ```csharp
   string fileName = dataDir + "/WithEmbeddedMsg.msg";
   var message = MapiMessage.FromFile(fileName);
   ```

3. **Έλεγχος για ενσωματωμένα μηνύματα**

   Επαληθεύστε εάν το πρώτο συνημμένο είναι ενσωματωμένο μήνυμα του Outlook:
   
   ```csharp
   if (message.Attachments[0].ObjectData.IsOutlookMessage)
   {
       // Προχωρήστε στην εξαγωγή του μηνύματος
   }
   ```

4. **Εξαγωγή και μετατροπή**

   Εξαγάγετε το ενσωματωμένο μήνυμα και μετατρέψτε το σε ένα `MapiMessage` αντικείμενο για περαιτέρω επεξεργασία.
   
   ```csharp
   var embeddedMessage = message.Attachments[0].ObjectData.ToMapiMessage();
   ```
