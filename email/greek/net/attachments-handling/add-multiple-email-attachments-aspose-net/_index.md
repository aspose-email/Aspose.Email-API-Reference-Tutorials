Τώρα που είστε έτοιμοι, ας προχωρήσουμε στην εφαρμογή της δυνατότητας προσθήκης πολλαπλών συνημμένων ηλεκτρονικού ταχυδρομείου.

### Προσθήκη πολλαπλών συνημμένων
**Επισκόπηση**
Αυτή η ενότητα εξηγεί πώς να προσθέσετε περισσότερα από ένα συνημμένα σε ένα email χρησιμοποιώντας το Aspose.Email για .NET. Αυτό είναι ιδιαίτερα χρήσιμο κατά την αυτοματοποίηση εργασιών email σε επιχειρηματικές εφαρμογές.

#### Βήμα 1: Ορισμός διαδρομών εγγράφων
Ξεκινήστε καθορίζοντας διαδρομές προς τα έγγραφά σας:

```csharp
string dataDir = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY");
```
**Εξήγηση:** Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` με τη διαδρομή όπου είναι αποθηκευμένα τα αρχεία σας. Αυτό διασφαλίζει ότι τα συνημμένα σας μπορούν να αναφερθούν με ακρίβεια κατά τον χρόνο εκτέλεσης.

#### Βήμα 2: Δημιουργήστε ένα αντικείμενο MailMessage
Δημιουργήστε ένα αντικείμενο μηνύματος ηλεκτρονικού ταχυδρομείου στο οποίο θα προσθέσετε συνημμένα:

```csharp
MailMessage message = new MailMessage();
message.From = "sender@example.com";
message.To = "recipient@example.com";
message.Subject = "Documents Attached";
```
**Εξήγηση:** Εδώ, ρυθμίζουμε τη βασική δομή του email μας με τα στοιχεία του αποστολέα και του παραλήπτη.

#### Βήμα 3: Προσθήκη συνημμένων
Επαναλάβετε τα αρχεία σας και επισυνάψτε τα:

```csharp
string[] fileNames = { "file1.pdf", "file2.docx" }; // Παραδείγματα ονομάτων αρχείων

foreach (var fileName in fileNames)
{
    var attachment = new Attachment(Path.Combine(dataDir, fileName));
    message.Attachments.Add(attachment);
}
```
**Εξήγηση:** Αυτός ο βρόχος επεξεργάζεται κάθε όνομα αρχείου στη λίστα σας, δημιουργώντας ένα `Attachment` αντικείμενο για κάθε ένα και προσθήκη του στο email.

#### Βήμα 4: Στείλτε το email
Τέλος, διαμορφώστε τον πελάτη SMTP και στείλτε το email:

```csharp
SmtpClient client = new SmtpClient("smtp.example.com", 587, "username", "password");
client.SecurityOptions = SecurityOptions.Auto;
client.Send(message);
```
**Εξήγηση:** Διαμορφώστε το `SmtpClient` με τα στοιχεία του διακομιστή σας. Προσαρμόστε τις ρυθμίσεις ασφαλείας όπως απαιτείται για τον πάροχο email σας.

### Συμβουλές αντιμετώπισης προβλημάτων
