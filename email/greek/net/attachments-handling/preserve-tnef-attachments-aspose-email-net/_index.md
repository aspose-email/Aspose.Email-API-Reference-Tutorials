## Οδηγός Εφαρμογής

### Διατήρηση συνημμένων TNEF
Αυτή η λειτουργία διασφαλίζει ότι όταν αποθηκεύετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου με συνημμένο TNEF, διατηρούνται τα αρχικά όρια και η ακεραιότητα των δεδομένων.

#### Βήμα 1: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου
Αρχικά, φορτώστε το μήνυμα ηλεκτρονικού ταχυδρομείου σας στο `MailMessage` αντικείμενο. Βεβαιωθείτε ότι το `LoadOptions` Η παράμετρος έχει οριστεί για τη διαχείριση συνημμένων TNEF.
```csharp
// Ορίστε τη διαδρομή καταλόγου που περιέχει τα αρχεία email σας
string dataDir = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY");

// Φόρτωση ενός υπάρχοντος αρχείου email με ενεργοποιημένες τις επιλογές TNEF
var loadOptions = new Aspose.Email.LoadOptions { PreserveTnefAttachments = true };
MailMessage message = MailMessage.Load(Path.Combine(dataDir, "example.msg"), loadOptions);
```
*Γιατί αυτό το βήμα;*Η φόρτωση του email με επιλογές TNEF διασφαλίζει ότι όλοι οι τύποι συνημμένων αναγνωρίζονται και διατηρούνται.

#### Βήμα 2: Αποθήκευση ως EML
Στη συνέχεια, αποθηκεύστε το φορτωμένο μήνυμά σας σε μορφή EML, διασφαλίζοντας ότι τα συνημμένα TNEF παραμένουν άθικτα.
```csharp
// Καθορίστε τη διαδρομή αρχείου εξόδου για το αρχείο EML
string emlFilePath = Path.Combine(dataDir, "output.eml");

// Αποθήκευση του MailMessage ως αρχείο EML με διατηρημένα συνημμένα
message.Save(emlFilePath, Aspose.Email.SaveOptions.DefaultEml);
```
*Τι συμβαίνει εδώ;*Η αποθήκευση σε μορφή EML χρησιμοποιώντας τις προεπιλεγμένες επιλογές διατηρεί όλες τις αρχικές ιδιότητες του email σας.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Διαφθορά Συνημμένου**: Βεβαιωθείτε `PreserveTnefAttachments` έχει οριστεί σε true κατά τη φόρτωση.
- **Σφάλματα "Δεν βρέθηκε αρχείο"**Ελέγξτε ξανά τις διαδρομές αρχείων και βεβαιωθείτε ότι είναι προσβάσιμες από την εφαρμογή.
