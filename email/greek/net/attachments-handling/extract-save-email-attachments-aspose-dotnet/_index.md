Αρχικά, φορτώστε το αρχείο email σας χρησιμοποιώντας `MailMessage.Load`:

```csharp

```

**Βήμα 2: Επανάληψη και αποθήκευση συνημμένων**

Στη συνέχεια, επαναλάβετε τα συνημμένα και αποθηκεύστε τα στον επιθυμητό κατάλογο:

```csharp

    // Αποθήκευση κάθε συνημμένου στον καθορισμένο κατάλογο

}
```
