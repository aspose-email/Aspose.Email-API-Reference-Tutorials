## Οδηγός Εφαρμογής

### Εξαγωγή ενσωματωμένων συνημμένων

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε ενσωματωμένα συνημμένα από ένα αρχείο MSG, αποθηκεύοντάς τα ως ξεχωριστά αρχεία στο δίσκο. Η διαδικασία περιλαμβάνει τη φόρτωση του αρχείου MSG, την επανάληψη των συνημμένων του και τον εντοπισμό των ενσωματωμένων.

#### Βήμα προς βήμα διαδικασία
**1. Φορτώστε το αρχείο MSG**
```csharp
MapiMessage message = MapiMessage.FromFile(dataDir + "/MSG file with RTF Formatting.msg");
```
- **Εξήγηση**Αυτή η γραμμή φορτώνει το αρχείο MSG σας σε ένα `MapiMessage` αντικείμενο, το οποίο αντιπροσωπεύει ένα μήνυμα ηλεκτρονικού ταχυδρομείου στο Aspose.Email.

**2. Επαναλάβετε μέσω συνημμένων**
```csharp
MapiAttachmentCollection attachments = message.Attachments;
foreach (MapiAttachment attachment in attachments)
{
    if(IsAttachmentInline(attachment))
    {
        SaveAttachment(attachment, new Guid().ToString());
    }
}
```
- **Εξήγηση**: Ανακτάτε όλα τα συνημμένα από το `message` και ελέγξτε το καθένα για να διαπιστώσετε εάν είναι ενσωματωμένο.

**3. Προσδιορίστε εάν ένα Συνημμένο είναι Ενσωματωμένο**
```csharp
static bool IsAttachmentInline(MapiAttachment attachment)
{
    foreach (MapiProperty property in attachment.ObjectData.Properties.Values)
    {
        if (property.Name == "\x0003ObjInfo")
        {
            ushort odtPersist1 = BitConverter.ToUInt16(property.Data, 0);
            return (odtPersist1 & (1 << (7 - 1))) == 0;
        }
    }
    return false;
}
```
- **Εξήγηση**Αυτή η μέθοδος ελέγχει συγκεκριμένες ιδιότητες του συνημμένου για να προσδιορίσει εάν είναι ενσωματωμένο. Εξετάζει μια δυαδική ιδιότητα και αξιολογεί τις σημαίες της.

**4. Αποθήκευση ενσωματωμένων συνημμένων**
```csharp
static void SaveAttachment(MapiAttachment attachment, string fileName)
{
    foreach (MapiProperty property in attachment.ObjectData.Properties.Values)
    {
        if (property.Name == "Package")
        {
            using (FileStream fs = new FileStream(fileName, FileMode.Create, FileAccess.Write))
            {
                fs.Write(property.Data, 0, property.Data.Length);
            }
        }
    }
}
```
- **Εξήγηση**Μόλις αναγνωριστεί ως ενσωματωμένο, το συνημμένο αποθηκεύεται στον δίσκο με ένα μοναδικό όνομα αρχείου.
