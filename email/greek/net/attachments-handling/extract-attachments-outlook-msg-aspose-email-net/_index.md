## Οδηγός Εφαρμογής

### Εξαγωγή συνημμένων από αρχείο MSG του Outlook
Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε συνημμένα από ένα αρχείο MSG του Outlook και να τα αποθηκεύσετε τοπικά.

#### Οδηγίες βήμα προς βήμα:
**1. Φορτώστε το αρχείο MSG**
Αρχικά, φορτώστε το αρχείο MSG χρησιμοποιώντας το `MapiMessage.FromFile()` μέθοδος.

```csharp
using Aspose.Email.Mapi;

// Ορίστε τη διαδρομή του καταλόγου εγγράφων σας.
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// Καθορίστε το όνομα του αρχείου MSG σας.
string fileName = "outputAttachments.msg";

// Φορτώστε το αρχείο MSG σε ένα αντικείμενο MapiMessage.
MapiMessage message = MapiMessage.FromFile(dataDir + "/" + fileName);
```

**2. Εξαγωγή και αποθήκευση συνημμένων**
Επαναλάβετε κάθε συνημμένο στο φορτωμένο αρχείο MSG, αποθηκεύοντάς το στον επιθυμητό κατάλογο εξόδου.

```csharp
// Καθορίστε τη διαδρομή του καταλόγου εξόδου σας.
string outputPath = "YOUR_OUTPUT_DIRECTORY";

foreach (MapiAttachment attachment in message.Attachments)
{
    // Αποθηκεύστε κάθε συνημμένο χρησιμοποιώντας το αρχικό του όνομα αρχείου.
    attachment.Save(outputPath + "/" + attachment.FileName);
}
```

**Εξήγηση:**
- `MapiMessage.FromFile()`Φορτώνει το αρχείο MSG σε ένα διαχειρίσιμο αντικείμενο.
- `message.Attachments`: Πρόσβαση στη συλλογή συνημμένων εντός του αρχείου MSG.
- `attachment.Save()`Αποθηκεύει κάθε συνημμένο στον καθορισμένο κατάλογο.

### Φόρτωση και επεξεργασία αρχείου MSG του Outlook
Η φόρτωση ενός αρχείου MSG είναι μόνο το πρώτο βήμα. Δείτε πώς μπορείτε να ξεκινήσετε αυτήν τη διαδικασία:

