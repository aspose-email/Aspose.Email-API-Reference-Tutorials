### Χαρακτηριστικό 1: Εντοπισμός μορφής αρχείου

#### Επισκόπηση

Η ανίχνευση της μορφής αρχείου ενός μηνύματος email είναι ζωτικής σημασίας για τη σωστή επεξεργασία του. Αυτή η λειτουργία σάς επιτρέπει να προσδιορίσετε μέσω προγραμματισμού εάν τα αρχεία email σας είναι `.msg`, `.eml`ή άλλες μορφές που υποστηρίζονται από το Aspose.Email.

#### Βήμα προς βήμα εφαρμογή

##### Βήμα 1: Χρήση `FileFormatUtil.DetectFileFormat`

Ορίστε τη διαδρομή αρχείου σε μια μεταβλητή:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY/message.msg";
```

Στη συνέχεια, χρησιμοποιήστε το `DetectFileFormat` μέθοδος για τον προσδιορισμό της μορφής:

```csharp
// Εντοπισμός της μορφής αρχείου του μηνύματος ηλεκτρονικού ταχυδρομείου
FileFormatInfo info = FileFormatUtil.DetectFileFormat(dataDir);
```

##### Εξήγηση
- **Παράμετροι**: Η διαδρομή του αρχείου email σας.
- **Επιστρεφόμενη τιμή**: Α `FileFormatInfo` αντικείμενο που περιέχει λεπτομέρειες σχετικά με την ανιχνευμένη μορφή.

#### Βήμα 2: Εμφάνιση εντοπιζόμενης μορφής (Προαιρετικό)

Για επαλήθευση, μπορείτε να εκτυπώσετε την ανιχνευμένη μορφή:

```csharp
// Έξοδος κονσόλας για επαλήθευση (σχολιάστηκε στην παραγωγή)
Console.WriteLine("The message format is: " + info.FileFormatType);
```
