## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα περιηγηθούμε στη διαδικασία σύνδεσης με έναν διακομιστή Exchange και στην προανάκτηση μεγεθών μηνυμάτων.

### Σύνδεση με Exchange Server
#### Επισκόπηση
Η σύνδεση με έναν διακομιστή Exchange περιλαμβάνει τη δημιουργία μιας παρουσίας του `IEWSClient` με τα διαπιστευτήριά σας. Αυτό σας επιτρέπει να αλληλεπιδράτε με τα γραμματοκιβώτια χρηστών στον διακομιστή.

#### Βήμα προς βήμα εφαρμογή
1. **Δημιουργήστε μια παρουσία του `IEWSClient`:**
   ```csharp
   // Αρχικοποίηση IEWSClient με στοιχεία διακομιστή και διαπιστευτήρια
   IEWSClient client = EWSClient.GetEWSClient("https://outlook.office365.com/ews/exchange.asmx", "testUser", "προεπιλογή", "τομέας");
   ```
2. **Ανάκτηση πληροφοριών μηνύματος:**
   Χρησιμοποιήστε το `ListMessages` μέθοδος για την ανάκτηση πληροφοριών μηνυμάτων από τα εισερχόμενα.
   ```csharp
   // Ανάκτηση μηνυμάτων από τα Εισερχόμενα
   ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);
   ```
3. **Βασικές λεπτομέρειες εμφάνισης:**
   Βρόχος μέσα από κάθε μία `ExchangeMessageInfo` στη συλλογή και εμφάνιση λεπτομερειών όπως θέμα, αποστολέας, παραλήπτης και μέγεθος.
   ```csharp
   foreach (ExchangeMessageInfo msgInfo in msgCollection)
   {
       Console.WriteLine("Subject: " + msgInfo.Subject);
       Console.WriteLine("From: " + msgInfo.From.ToString());
       Console.WriteLine("To: " + msgInfo.To.ToString());
       Console.WriteLine("Message Size: " + msgInfo.Size);
       Console.WriteLine("==================================");
   }
   ```

#### Εξήγηση
- **Παράμετροι**: Το `EWSClient.GetEWSClient` Η μέθοδος απαιτεί τη διεύθυνση URL του διακομιστή Exchange, το όνομα χρήστη, τον κωδικό πρόσβασης και τον τομέα.
- **Επιστρεφόμενες τιμές**: `ListMessages` Επιστρέφει μια συλλογή από αντικείμενα πληροφοριών μηνύματος.
