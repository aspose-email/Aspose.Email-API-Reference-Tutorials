## Οδηγός Εφαρμογής

### Δυνατότητα 1: Φόρτωση και ανάλυση αρχείου μηνύματος Outlook

Το πρώτο βήμα είναι να φορτώσετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου από ένα αρχείο. Αυτή η λειτουργία θα σας δείξει πώς να εξαγάγετε βασικές ιδιότητες όπως θέμα, όνομα αποστολέα, περιεχόμενο σώματος και συνημμένα.

#### Επισκόπηση
Αυτή η ενότητα σάς δείχνει πώς να χρησιμοποιήσετε το Aspose.Email για .NET για να διαβάσετε ένα αρχείο MSG ή EML του Outlook και να αποκτήσετε πρόσβαση στα βασικά του στοιχεία.

##### Βήμα 1: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου
Αρχικά, ορίστε τη διαδρομή όπου αποθηκεύονται τα αρχεία email σας. Στη συνέχεια, φορτώστε ένα μήνυμα χρησιμοποιώντας `MapiMessage.FromMailMessage`.

```csharp
using System;
using Aspose.Email.Mapi;

public class Feature1
{
    public static void Run()
    {
        string dataDir = @"YOUR_DOCUMENT_DIRECTORY/"; 
        MapiMessage msg = MapiMessage.FromMailMessage(dataDir + "Message.eml");

        // Εμφάνιση ιδιοτήτων email
        Console.WriteLine("Subject:" + msg.Subject);
        Console.WriteLine("From:" + msg.SenderName);
        Console.WriteLine("Body:" + msg.Body);
        Console.WriteLine("Attachment Count:" + msg.Attachments.Count);
    }
}
```

**Γιατί αυτό έχει σημασία:** Η φόρτωση του μηνύματος παρέχει πρόσβαση σε όλα τα στοιχεία του, επιτρέποντας λεπτομερή χειρισμό και εξαγωγή δεδομένων.

##### Βήμα 2: Εξαγωγή ιδιοτήτων ηλεκτρονικού ταχυδρομείου
Χρήση ιδιοτήτων του `MapiMessage` για την εξαγωγή λεπτομερειών όπως θέμα, όνομα αποστολέα και περιεχόμενο σώματος. Ο αριθμός των συνημμένων εμφανίζεται επίσης χρησιμοποιώντας `msg.Attachments.Count`.

### Χαρακτηριστικό 2: Επανάληψη μέσω συνημμένων

Μόλις φορτώσετε το μήνυμα ηλεκτρονικού ταχυδρομείου, η επανάληψη των συνημμένων του γίνεται απλή.

#### Επισκόπηση
Αυτό το μέρος εξηγεί πώς να επαναλάβετε την επανάληψη κάθε συνημμένου σε ένα αρχείο μηνύματος και να τα αποθηκεύσετε ξεχωριστά.

##### Βήμα 1: Αποθήκευση συνημμένων
Μπορείτε να επαναλάβετε `msg.Attachments` και χρησιμοποιήστε το `Save` μέθοδο για κάθε μία. Βεβαιωθείτε ότι έχετε ρυθμίσει έναν κατάλογο εξόδου για την αποθήκευση αυτών των αρχείων.

```csharp
using System;
using Aspose.Email.Mapi;

public class Feature2
{
    public static void Run(MapiMessage msg)
    {
        foreach (MapiAttachment attachment in msg.Attachments)
        {
            Console.WriteLine("Attachment:" + attachment.FileName);
            string outputPath = @"YOUR_OUTPUT_DIRECTORY/" + attachment.LongFileName;
            attachment.Save(outputPath);
        }
    }
}
```

**Γιατί αυτό έχει σημασία:** Η ξεχωριστή αποθήκευση συνημμένων σάς επιτρέπει να τα διαχειρίζεστε και να τα αποθηκεύετε όπως απαιτείται, κάτι που είναι ιδιαίτερα χρήσιμο σε εργασίες αυτοματισμού.
