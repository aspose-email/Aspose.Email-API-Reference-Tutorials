## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στη φόρτωση ενός μηνύματος ηλεκτρονικού ταχυδρομείου από ένα αρχείο και στον έλεγχο της κατάστασης επιστροφής του.

### Φόρτωση μηνύματος ηλεκτρονικού ταχυδρομείου

#### Βήμα 1: Ρύθμιση διαδρομής αρχείου

Ορίστε τη διαδρομή προς τα αρχεία email σας:

```csharp
string fileName = "YOUR_DOCUMENT_DIRECTORY\\failed1.msg";
```

#### Βήμα 2: Φόρτωση του email

Χρησιμοποιήστε το Aspose.Email για να φορτώσετε το μήνυμα από ένα αρχείο:

```csharp
MailMessage mail = MailMessage.Load(fileName);
```
Αυτό το βήμα διαβάζει το περιεχόμενο του email σας σε ένα `MailMessage` αντικείμενο για περαιτέρω επεξεργασία.

### Έλεγχος κατάστασης εγκατάλειψης

#### Βήμα 3: Ελέγξτε αν το email έχει ανακληθεί

Προσδιορίστε εάν το μήνυμα ηλεκτρονικού ταχυδρομείου έχει επιστρέψει:

```csharp
BounceResult result = mail.CheckBounced();
```
Ο `CheckBounced()` η μέθοδος επιστρέφει ένα `BounceResult` αντικείμενο με λεπτομέρειες αναπήδησης.

### Κατανόηση λεπτομερειών αναπήδησης

#### Βήμα 4: Πρόσβαση σε πληροφορίες αναπήδησης

Πρόσβαση σε διάφορες ιδιότητες του `BounceResult` για να κατανοήσετε γιατί ένα email επέστρεψε:

```csharp
bool isBounced = result.IsBounced;
string action = result.Action; // Προτεινόμενες ενέργειες (π.χ., επανάληψη)
MailAddress recipient = result.Recipient;
string reason = result.Reason; // Λόγος για αναπήδηση
string status = result.Status; // Κατάσταση εγκατάλειψης (π.χ. Επιτυχία, Αποτυχία)
// Πρόσβαση στις λεπτομέρειες του αρχικού μηνύματος, εάν είναι διαθέσιμες
string originalMessageToAddress1 = result.OriginalMessage.To[0].Address;
```
Κάθε ιδιότητα παρέχει πληροφορίες σχετικά με το συμβάν επιστροφής email, βοηθώντας σας να λάβετε τεκμηριωμένες αποφάσεις σχετικά με τον χειρισμό των email που έχουν επιστρέψει email.
