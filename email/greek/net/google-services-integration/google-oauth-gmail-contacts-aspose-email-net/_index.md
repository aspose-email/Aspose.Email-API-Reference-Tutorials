### Ανάκτηση διακριτικού πρόσβασης Google OAuth

#### Επισκόπηση
Αυτή η λειτουργία επιτρέπει στην εφαρμογή σας να πραγματοποιεί έλεγχο ταυτότητας με τους διακομιστές της Google χρησιμοποιώντας το OAuth 2.0, παρέχοντας ασφαλή πρόσβαση στα δεδομένα χρήστη.

#### Βήμα προς βήμα εφαρμογή

**1. Ορίστε τα διαπιστευτήρια χρήστη**
```csharp
GoogleTestUser user = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
string accessToken;
string refreshToken;
```

**2. Ανάκτηση Πρόσβασης και Ανανέωση Tokens**
Χρησιμοποιήστε το `GetAccessToken` μέθοδος για την απόκτηση διακριτικών.
```csharp
GoogleOAuthHelper.GetAccessToken(user, out accessToken, out refreshToken);
```
- **Παράμετροι**: Τα διαπιστευτήρια χρήστη σας (`GoogleTestUser`) και μεταβλητές για την αποθήκευση διακριτικών.
- **Επιστρεφόμενες τιμές**Το διακριτικό πρόσβασης και το διακριτικό ανανέωσης αποθηκεύονται σε αντίστοιχες μεταβλητές.

**Συμβουλή αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι το αναγνωριστικό πελάτη και το μυστικό σας έχουν διαμορφωθεί σωστά στο Google Cloud Console για να αποφύγετε σφάλματα ελέγχου ταυτότητας.

#### Επισκόπηση
Η ενημέρωση των στοιχείων μιας επαφής στο Gmail μπορεί εύκολα να διαχειριστεί με το Aspose.Email, βελτιώνοντας τη διαχείριση των δεδομένων χρήστη.

Δημιουργήστε μια παρουσία χρησιμοποιώντας το διακριτικό πρόσβασης.
```csharp
using (IGmailClient client = GmailClient.GetInstance(accessToken, user.EMail))
{
```

**2. Ανάκτηση και ενημέρωση επαφών**
Ανακτήστε επαφές, τροποποιήστε τα στοιχεία μίας και αποθηκεύστε τις αλλαγές στο Gmail.
```csharp
    Contact[] contacts = client.GetAllContacts();
```
