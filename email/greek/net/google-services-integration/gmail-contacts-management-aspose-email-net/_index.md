## Οδηγός Εφαρμογής

### Λειτουργία 1: Αρχικοποίηση του Βοηθού Google OAuth
#### Επισκόπηση
Για την αλληλεπίδραση με τις υπηρεσίες Google, απαιτείται έλεγχος ταυτότητας. Αυτή η λειτουργία επιδεικνύει την αρχικοποίηση και την ανάκτηση διακριτικών πρόσβασης χρησιμοποιώντας το `GoogleOAuthHelper` τάξη.

#### Βήματα Υλοποίησης
**Βήμα 1**: Ορισμός διαπιστευτηρίων χρήστη
Ξεκινήστε δημιουργώντας μια νέα παρουσία του `GoogleTestUser`, μεταβιβάζοντας τα διαπιστευτήριά σας:

```csharp
// Αρχικοποίηση του Βοηθού Google OAuth
using Aspose.Email.Clients.Google;
using System;

public static void InitializeGoogleOAuth()
{
    // Ορισμός διαπιστευτηρίων χρήστη
    GoogleTestUser User2 = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
    
    string accessToken;
    string refreshToken;
    // Λήψη διακριτικών πρόσβασης και ανανέωσης για έλεγχο ταυτότητας OAuth
    GoogleOAuthHelper.GetAccessToken(User2, out accessToken, out refreshToken);
}
```
**Εξήγηση**:  
- `GoogleTestUser`: Αντιπροσωπεύει τα διαπιστευτήρια χρήστη που απαιτούνται για τον έλεγχο ταυτότητας.
- `GetAccessToken`Ανακτά τα απαραίτητα διακριτικά πρόσβασης και ανανέωσης.

### Λειτουργία 2: Ανάκτηση όλων των επαφών Gmail
#### Επισκόπηση
Μόλις ολοκληρωθεί η επαλήθευση, μπορείτε να ανακτήσετε όλες τις επαφές σας από έναν λογαριασμό Gmail χρησιμοποιώντας το `IGmailClient`.

#### Βήματα Υλοποίησης
**Βήμα 1**: Δημιουργία στιγμιαίας λειτουργίας του προγράμματος-πελάτη Gmail
Χρησιμοποιήστε το διακριτικό πρόσβασης και το email χρήστη για να δημιουργήσετε μια παρουσία του `GmailClient`:

```csharp
// Ανάκτηση όλων των επαφών Gmail
using Aspose.Email.Clients.Google;
using Aspose.Email.PersonalInfo;
using System.Collections.Generic;

public static void GetAllGmailContacts(string accessToken, string userEmail)
{
    // Δημιουργήστε ένα στιγμιότυπο του προγράμματος-πελάτη Gmail με το διακριτικό πρόσβασης και το email χρήστη
    using (IGmailClient client = GmailClient.GetInstance(accessToken, userEmail))
    {
        // Ανάκτηση όλων των επαφών από τον λογαριασμό Gmail
        Contact[] contacts = client.GetAllContacts();
        
        int contactCount = contacts.Length;
        Console.WriteLine($"Total Contacts: {contactCount}");
    }
}
```
**Εξήγηση**:  
- `GmailClient.GetInstance`Δημιουργεί μια παρουσία πελάτη για πρόσβαση στις υπηρεσίες Gmail.
- `GetAllContacts`: Ανακτά όλες τις επαφές από τον καθορισμένο λογαριασμό Gmail.

### Λειτουργία 3: Διαγραφή συγκεκριμένης επαφής Gmail
#### Επισκόπηση
Για να διατηρήσετε τη λίστα επαφών σας, ίσως χρειαστεί να διαγράψετε συγκεκριμένες καταχωρίσεις. Αυτή η λειτουργία δείχνει τη διαγραφή μιας επαφής με βάση το Google ID της χρησιμοποιώντας `IGmailClient`.

#### Βήματα Υλοποίησης
**Βήμα 1**: Αναγνώριση και διαγραφή της επαφής
Ανακτήστε όλες τις επαφές για να βρείτε και να διαγράψετε την επιθυμητή:

```csharp
// Διαγραφή συγκεκριμένης επαφής Gmail
using Aspose.Email.Clients.Google;
using Aspose.Email.PersonalInfo;

public static void DeleteGmailContact(string accessToken, string userEmail, string contactGoogleId)
{
    // Δημιουργήστε ένα στιγμιότυπο του προγράμματος-πελάτη Gmail με το διακριτικό πρόσβασης και το email χρήστη
    using (IGmailClient client = GmailClient.GetInstance(accessToken, userEmail))
    {
        Contact[] contacts = client.GetAllContacts();
        
        Contact contactToDelete = Array.Find(contacts, c => c.Id.GoogleId == contactGoogleId);
        
        if (contactToDelete != null)
        {
            // Διαγράψτε την καθορισμένη επαφή χρησιμοποιώντας το αναγνωριστικό Google της
            client.DeleteContact(contactToDelete.Id.GoogleId);
        }
    }
}
```
**Εξήγηση**:  
- `Array.Find`: Αναζητά μια επαφή με βάση το Google ID της.
- `DeleteContact`Καταργεί την αναγνωρισμένη επαφή από τον λογαριασμό σας στο Gmail.
