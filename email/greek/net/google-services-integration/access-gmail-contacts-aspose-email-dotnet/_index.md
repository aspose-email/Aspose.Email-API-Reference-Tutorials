## Οδηγός Εφαρμογής
Αυτή η ενότητα καλύπτει την πρόσβαση σε όλες τις επαφές του Gmail και την ανάκτηση επαφών από συγκεκριμένες ομάδες.

### Πρόσβαση σε όλες τις επαφές σε έναν λογαριασμό Gmail
**Επισκόπηση:** Ανακτήστε όλες τις επαφές από τον λογαριασμό Gmail ενός χρήστη χρησιμοποιώντας το Aspose.Email για .NET.

#### Βήμα 1: Ρύθμιση ελέγχου ταυτότητας
Επαληθεύστε την ταυτότητά σας με την υπηρεσία OAuth της Google:
```csharp
string accessToken = "YOUR_ACCESS_TOKEN"; // Αντικαταστήστε με το πραγματικό σας διακριτικό πρόσβασης
string userEmail = "YOUR_EMAIL_ADDRESS"; // Αντικαταστήστε με τη διεύθυνση ηλεκτρονικού ταχυδρομείου του χρήστη

googleTestUser user2 = new googleTestUser("user", "email address", "password", "clientId", "client secret");
GmailOAuthHelper.GetAccessToken(user2, out accessToken, out _);
```

#### Βήμα 2: Πρόσβαση στις Επαφές
Δημιουργήστε μια παρουσία του `IGmailClient` και να ανακτήσετε όλες τις επαφές:
```csharp
using (IGmailClient client = GmailClient.GetInstance(accessToken, userEmail))
{
    Contact[] contacts = client.GetAllContacts();
    foreach (Contact contact in contacts)
    {
        Console.WriteLine(contact.DisplayName + ", " + contact.EmailAddresses[0]);
    }
}
```

**Εξήγηση:** Αρχικοποίηση του `IGmailClient` χρησιμοποιώντας το διακριτικό πρόσβασης και το email. Το `GetAllContacts()` Η μέθοδος ανακτά όλες τις διαθέσιμες επαφές.

### Ανάκτηση επαφών από μια συγκεκριμένη ομάδα
**Επισκόπηση:** Ανάκτηση επαφών εντός μιας συγκεκριμένης ομάδας στον λογαριασμό Gmail του χρήστη.

#### Βήμα 1: Ανάκτηση όλων των ομάδων
Αρχικά, λάβετε όλες τις ομάδες επαφών:
```csharp
using (IGmailClient client = GmailClient.GetInstance(accessToken, userEmail))
{
    ContactGroupCollection groups = client.GetAllGroups();
```

#### Βήμα 2: Προσδιορισμός και ανάκτηση επαφών ομάδας
Βρείτε την ομάδα με βάση τον τίτλο της και ανακτήστε τις επαφές:
```csharp
GoogleContactGroup group = null;
foreach (GoogleContactGroup g in groups)
{
    if (g.Title == "TestGroup")
    {
        group = g;
        break;
    }
}

if (group != null)
{
    Contact[] contacts2 = client.GetContactsFromGroup(group.Id);
    foreach (Contact con in contacts2)
    {
        Console.WriteLine(con.DisplayName + ", " + con.EmailAddresses[0].ToString());
    }
}
```

**Εξήγηση:** Αυτό το απόσπασμα αναζητά μια ομάδα με τίτλο "TestGroup" και ανακτά όλες τις επαφές εντός αυτής της ομάδας χρησιμοποιώντας `GetContactsFromGroup()`.
