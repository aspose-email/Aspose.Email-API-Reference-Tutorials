## Οδηγός Εφαρμογής
Τώρα που έχετε ρυθμίσει το Aspose.Email για .NET, ας εφαρμόσουμε τις δυνατότητές του.

### Επαλήθευση ταυτότητας με το Google API
**Επισκόπηση:** Ο έλεγχος ταυτότητας είναι το πρώτο βήμα για την πρόσβαση στα δεδομένα του Ημερολογίου Google. Χρησιμοποιώντας το Aspose.Email, αποκτήστε πρόσβαση και ανανεώστε αποτελεσματικά τα διακριτικά.

#### Βήμα προς βήμα εφαρμογή
1. **Δημιουργήστε έναν δοκιμαστικό χρήστη:**
   ```csharp
   GoogleTestUser user = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
   ```
2. **Λήψη διακριτικών πρόσβασης και ανανέωσης:**
   ```csharp
   string accessToken;
   string refreshToken;
   GoogleOAuthHelper.GetAccessToken(user, out accessToken, out refreshToken);
   ```

### Λίστα ημερολογίων και ανάκτηση ραντεβού
**Επισκόπηση:** Η καταχώριση ημερολογίων βοηθά στον προσδιορισμό του ημερολογίου με το οποίο θα εργαστείτε, ενώ η ανάκτηση ραντεβού επιτρέπει τη λεπτομερή διαχείριση.

#### Βήμα προς βήμα εφαρμογή
1. **Αρχικοποίηση προγράμματος-πελάτη Gmail:**
   ```csharp
   using (IGmailClient client = GmailClient.GetInstance(accessToken, user.EMail))
   {
       string sourceCalendarId = client.ListCalendars()[0].Id;
   }
   ```
2. **Ανάκτηση Ραντεβού από Ημερολόγιο:**
   ```csharp
   Appointment[] appointmentsBeforeMove = client.ListAppointments(sourceCalendarId);
   ```

### Μετακίνηση ραντεβού μεταξύ ημερολογίων
**Επισκόπηση:** Η μετακίνηση ραντεβού είναι απαραίτητη για την αναδιοργάνωση εργασιών σε διαφορετικά ημερολόγια.

#### Βήμα προς βήμα εφαρμογή
1. **Λάβετε το μοναδικό αναγνωριστικό ενός ραντεβού:**
   ```csharp
   string targetAppointmentUniqueId = client.ListAppointments(sourceCalendarId)[0].UniqueId;
   ```
2. **Μετακίνηση του ραντεβού:**
   ```csharp
   Appointment movedAppointment = client.MoveAppointment(sourceCalendarId, destinationCalendarId, targetAppointmentUniqueId);
   ```

### Διαγραφή ραντεβού από ημερολόγιο
**Επισκόπηση:** Η διαγραφή περιττών ραντεβού βοηθά στη διατήρηση ενός καθαρού και οργανωμένου ημερολογίου.

#### Βήμα προς βήμα εφαρμογή
1. **Διαγραφή του ραντεβού:**
   ```csharp
   client.DeleteAppointment(destinationCalendarId, movedAppointment.UniqueId);
   ```
2. **Επιβεβαίωση διαγραφής:**
   ```csharp
   Appointment[] appointmentsAfterDeletion = client.ListAppointments(destinationCalendarId);
   ```
