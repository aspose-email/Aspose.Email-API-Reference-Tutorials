## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε ξεχωριστά χαρακτηριστικά:

### Λειτουργία ανάκτησης διακριτικού πρόσβασης
#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει την απόκτηση ενός διακριτικού πρόσβασης και ενός διακριτικού ανανέωσης χρησιμοποιώντας το Aspose.Email για .NET, επιτρέποντας την ασφαλή πρόσβαση στην υπηρεσία Gmail.

**Βήμα 1: Αρχικοποίηση διαπιστευτηρίων χρήστη**
Ορίστε τα διαπιστευτήρια χρήστη, συμπεριλαμβανομένου του email, του αναγνωριστικού πελάτη και του μυστικού ονόματος πελάτη, τα οποία είναι κρίσιμα για τον έλεγχο ταυτότητας OAuth.
```csharp
GoogleTestUser User = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
```

**Βήμα 2: Ανάκτηση διακριτικών**
Χρησιμοποιήστε το `GetAccessToken` μέθοδος για την ανάκτηση διακριτικών πρόσβασης και ανανέωσης, κρίσιμη για αιτήματα με έλεγχο ταυτότητας.
```csharp
string accessToken;
string refreshToken;
GoogleOAuthHelper.GetAccessToken(User, out accessToken, out refreshToken);
```
- **Παράμετροι:** Διαπιστευτήρια χρήστη ενθυλακωμένα σε ένα `GoogleTestUser` αντικείμενο.
- **Επιστρεφόμενες τιμές:** Συμβολοσειρές διακριτικού πρόσβασης και διακριτικού ανανέωσης.

#### Συμβουλές αντιμετώπισης προβλημάτων
Βεβαιωθείτε ότι το αναγνωριστικό πελάτη και το μυστικό σας έχουν ρυθμιστεί σωστά στην Κονσόλα προγραμματιστή Google. Οι εσφαλμένες διαμορφώσεις μπορούν να οδηγήσουν σε αποτυχίες ελέγχου ταυτότητας.

### Διαγραφή συγκεκριμένης λειτουργίας ημερολογίου
#### Επισκόπηση
Αυτή η λειτουργία επιτρέπει την πρόσβαση σε έναν λογαριασμό Gmail χρησιμοποιώντας ένα διακριτικό πρόσβασης και τη διαγραφή ημερολογίων με βάση συγκεκριμένα προθέματα σύνοψης.

**Βήμα 1: Αρχικοποίηση του προγράμματος-πελάτη Gmail**
Δημιουργήστε ένα `GmailClient` παράδειγμα με το ανακτημένο διακριτικό πρόσβασης, απαραίτητο για κλήσεις API με έλεγχο ταυτότητας.
```csharp
using (IGmailClient client = GmailClient.GetInstance(accessToken, User.EMail))
```

**Βήμα 2: Ορισμός και διαγραφή ημερολογίων**
Ανάκτηση όλων των ημερολογίων και διαγραφή εκείνων των οποίων οι συνόψεις ταιριάζουν με ένα καθορισμένο πρόθεμα.
```csharp
string summaryPrefix = "Calendar summary - ";
ExtendedCalendar[] calendars = client.ListCalendars();
foreach (ExtendedCalendar calendar in calendars)
{
    if (calendar.Summary.StartsWith(summaryPrefix))
        client.DeleteCalendar(calendar.Id);
}
```
- **Παράμετροι:** Διακριτικό πρόσβασης για έλεγχο ταυτότητας και email χρήστη.
- **Βασικές διαμορφώσεις:** Πρόθεμα σύνοψης που χρησιμοποιείται για τον προσδιορισμό των ημερολογίων-στόχων.

#### Συμβουλές αντιμετώπισης προβλημάτων
Επαληθεύστε την εγκυρότητα του διακριτικού πρόσβασης πριν από την εκτέλεση λειτουργιών. Τα ληγμένα διακριτικά μπορούν να οδηγήσουν σε αποτυχημένα αιτήματα API.
