## Οδηγός Εφαρμογής

Θα καλύψουμε τρία βασικά χαρακτηριστικά: την απόκτηση διακριτικών OAuth, την εισαγωγή ημερολογίων και την ανάκτηση/ενημέρωση ημερολογίων.

### Λήψη διακριτικού πρόσβασης Google OAuth

#### Επισκόπηση
Επαληθεύστε έναν χρήστη χρησιμοποιώντας το σύστημα OAuth 2.0 της Google με το Aspose.Email για .NET.

**Βήμα προς βήμα εφαρμογή**

1. **Αρχικοποίηση διαπιστευτηρίων χρήστη**
   Δημιουργήστε μια παρουσία του `GoogleTestUser` με τα στοιχεία του πελάτη σας.
   ```csharp
   GoogleTestUser User2 = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
   ```

2. **Απόκτηση πρόσβασης και ανανέωση διακριτικών**
   Χρησιμοποιήστε τη μέθοδο helper για να λάβετε tokens:
   ```csharp
   string accessToken;
   string refreshToken;
   GoogleOAuthHelper.GetAccessToken(User2, out accessToken, out refreshToken);
   ```
   - `accessToken`: Χρησιμοποιείται για τον έλεγχο ταυτότητας αιτημάτων API.
   - `refreshToken`: Λαμβάνει ένα νέο διακριτικό πρόσβασης μόλις λήξει.

### Εισαγωγή Ημερολογίου στο Gmail

#### Επισκόπηση
Εισαγάγετε ένα νέο ημερολόγιο στον λογαριασμό σας στο Gmail χρησιμοποιώντας το Aspose.Email.

**Βήμα προς βήμα εφαρμογή**

1. **Έλεγχος ταυτότητας χρησιμοποιώντας το διακριτικό OAuth**
   Επαναχρησιμοποιήστε το διακριτικό πρόσβασης από το προηγούμενο βήμα.
   
2. **Δημιουργήστε μια παρουσία IGmailClient**
   ```csharp
   using (IGmailClient client = GmailClient.GetInstance(accessToken, User2.EMail))
   ```
   
3. **Ορισμός και εισαγωγή νέου ημερολογίου**
   Ορίστε ένα ημερολόγιο με μοναδικές λεπτομέρειες:
   ```csharp
   Aspose.Email.Clients.Google.Calendar calendar = new Aspose.Email.Clients.Google.Calendar(
       "summary - " + Guid.NewGuid().ToString(), null, null, "America/Los_Angeles");
   
   string id = client.CreateCalendar(calendar);
   ```

### Ανάκτηση και ενημέρωση ημερολογίου

#### Επισκόπηση
Μάθετε πώς να ανακτήσετε ένα υπάρχον Ημερολόγιο Google και να ενημερώσετε τις πληροφορίες του χρησιμοποιώντας το Aspose.Email.

**Βήμα προς βήμα εφαρμογή**

1. **Έλεγχος ταυτότητας χρησιμοποιώντας το διακριτικό OAuth**
   Επαναχρησιμοποιήστε το διακριτικό πρόσβασης από τα προηγούμενα βήματα.
   
2. **Λήψη του ημερολογίου με βάση το αναγνωριστικό**
   ```csharp
   string id = "existing_calendar_id";  // Αντικατάσταση με το πραγματικό αναγνωριστικό ημερολογίου
   Aspose.Email.Clients.Google.Calendar cal = client.FetchCalendar(id);
   ```

3. **Επαλήθευση και ενημέρωση λεπτομερειών ημερολογίου**
   Συγκρίνετε τις ανακτημένες λεπτομέρειες και ενημερώστε εάν είναι απαραίτητο:
   ```csharp
   if ((localCalendar.Summary == cal.Summary) && (localCalendar.TimeZone == cal.TimeZone)) {
       cal.Description = "Description - " + Guid.NewGuid().ToString();
       cal.Location = "Location - " + Guid.NewGuid().ToString();
       client.UpdateCalendar(cal);
   }
   ```
