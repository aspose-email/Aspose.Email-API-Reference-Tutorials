## Οδηγός Εφαρμογής

### Φόρτωση επαφών με καθορισμένη κωδικοποίηση (H2)
Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε την κωδικοποίηση κατά τη φόρτωση επαφών, διασφαλίζοντας τη συμβατότητα και την ορθότητα σε διαφορετικά συστήματα.

#### Βήμα προς βήμα εφαρμογή (H3)
1. **Ορισμός του καταλόγου εγγράφων**
   Καθορίστε πού βρίσκονται τα αρχεία VCF σας:
   ```csharp
   // Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων
   String dataDir = "YOUR_DOCUMENT_DIRECTORY";
   ```
2. **Καθορισμός συνόλου χαρακτήρων κωδικοποίησης**
   Επιλέξτε την κωδικοποίηση για την ανάγνωση του αρχείου, όπως UTF-8 για ευρεία συμβατότητα.
   ```java
   Charset charset = StandardCharsets.UTF_8;
   ```
3. **Φόρτωση της επαφής από αρχείο VCF**
   Χρήση `MapiContact.FromVCard` μέθοδος με παραμέτρους: διαδρομή αρχείου και κωδικοποίηση συνόλου χαρακτήρων.
   ```csharp
   MapiContact contactReadFromFile = MapiContact.FromVCard(dataDir + "/Contact.vcf", charset);
   ```
#### Παράμετροι Επεξήγηση
- **Διαδρομή αρχείου**: Τοποθεσία του αρχείου VCF σας.
- **Κωδικοποίηση συνόλου χαρακτήρων**: Διασφαλίζει τη σωστή επεξεργασία των ειδικών χαρακτήρων.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Επαληθεύστε ότι η διαδρομή προς το αρχείο VCF είναι σωστή και προσβάσιμη.
- Βεβαιωθείτε ότι το καθορισμένο σύνολο χαρακτήρων ταιριάζει με την πραγματική κωδικοποίηση του αρχείου VCF.
