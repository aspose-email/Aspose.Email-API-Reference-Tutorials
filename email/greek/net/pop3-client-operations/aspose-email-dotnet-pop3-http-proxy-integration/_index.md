## Οδηγός Εφαρμογής

### Πρόσβαση στο γραμματοκιβώτιο POP3 μέσω διακομιστή μεσολάβησης HTTP
Αυτή η λειτουργία επιτρέπει στην εφαρμογή σας να συνδεθεί σε έναν διακομιστή POP3 χρησιμοποιώντας έναν ενδιάμεσο διακομιστή μεσολάβησης HTTP, κάτι που είναι κρίσιμο σε περιβάλλοντα περιορισμένου δικτύου.

#### Δημιουργήστε μια παρουσία του HttpProxy
Ξεκινήστε δημιουργώντας ένα `HttpProxy` στιγμιότυπο με τις απαραίτητες λεπτομέρειες κεντρικού υπολογιστή και θύρας. Αυτό ρυθμίζει τη σύνδεσή σας μέσω του καθορισμένου διακομιστή μεσολάβησης:
```csharp
// Ορίστε τις ρυθμίσεις του proxy σας
HttpProxy proxy = new HttpProxy("18.222.124.59", 8080); // Αντικατάσταση με την πραγματική διεύθυνση και θύρα διακομιστή μεσολάβησης
```

#### Αρχικοποίηση προγράμματος-πελάτη POP3
Στήνω `Pop3Client` για να αλληλεπιδράσετε με το γραμματοκιβώτιο μέσω του διακομιστή μεσολάβησης HTTP:
```csharp
// Διαμορφώστε τις ρυθμίσεις του διακομιστή email σας
Pop3Client client = new Pop3Client("pop.example.com", "username", "password");

// Αντιστοίχιση της παρουσίας HttpProxy στον υπολογιστή-πελάτη
client.Proxy = proxy;
```
- **Παράμετροι**:
  - `"pop.example.com"`: Το όνομα κεντρικού υπολογιστή του διακομιστή POP3.
  - `"username"` και `"password"`Διαπιστευτήρια για την πρόσβαση στο γραμματοκιβώτιό σας.

#### Σύνδεση και λήψη μηνυμάτων ηλεκτρονικού ταχυδρομείου
Αφού ολοκληρωθεί η εγκατάσταση, συνδεθείτε στον διακομιστή και λάβετε τα email:
```csharp
try
{
    client.Connect(true); // Χρησιμοποιήστε SSL εάν απαιτείται από τον διακομιστή
    int messageCount = client.GetMessageCount();
    
    Console.WriteLine($"Total Messages: {messageCount}");

    for (int i = 1; i <= messageCount; i++)
    {
        MailMessage msg = client.FetchMessage(i);
        Console.WriteLine($"Subject: {msg.Subject}");
    }
}
catch (Exception ex)
{
    Console.WriteLine("Error: " + ex.Message);
}
fine
{
    client.Dispose();
}
```
- **Επιστρεφόμενες τιμές**:
  - `GetMessageCount()`: Ανακτά τον συνολικό αριθμό μηνυμάτων στα εισερχόμενα.
  - `FetchMessage(int)`: Ανακτά ένα συγκεκριμένο μήνυμα ηλεκτρονικού ταχυδρομείου με βάση τον δείκτη μηνυμάτων του.

#### Συμβουλές αντιμετώπισης προβλημάτων
Συνήθη προβλήματα περιλαμβάνουν σφάλματα συνδεσιμότητας δικτύου ή αποτυχίες ελέγχου ταυτότητας. Βεβαιωθείτε ότι οι ρυθμίσεις του διακομιστή μεσολάβησης είναι σωστές και ότι έχετε έγκυρα διαπιστευτήρια διακομιστή. Επίσης, επαληθεύστε εάν απαιτείται SSL/TLS από τον διακομιστή POP3 για ασφαλείς συνδέσεις.
