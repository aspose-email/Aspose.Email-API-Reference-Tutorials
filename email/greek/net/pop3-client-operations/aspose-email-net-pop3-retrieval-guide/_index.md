## Ρύθμιση του Aspose.Email για .NET
Έχοντας θέσει τις προϋποθέσεις, αρχικοποιήστε το πακέτο Aspose.Email. Δείτε πώς μπορείτε να ρυθμίσετε το έργο σας:
1. **Εγκατάσταση του Aspose.Email**Χρησιμοποιήστε μία από τις παραπάνω μεθόδους εγκατάστασης.
2. **Αρχικοποίηση Aspose.Email**Εισαγάγετε τους απαραίτητους χώρους ονομάτων και διαμορφώστε τον πελάτη POP3.

```csharp
using Aspose.Email.Clients.Pop3;
using System;

const string host = "your.pop3server.com";
const int port = 110; // Τυπική μη κρυπτογραφημένη θύρα
const string username = "user@host.com";
const string password = "password";

Pop3Client client = new Pop3Client(host, port, username, password);
```

**Γιατί αυτή η ρύθμιση;** Αυτή η αρχικοποίηση συνδέει την εφαρμογή σας με τον διακομιστή POP3 για λειτουργίες ανάκτησης email.

## Οδηγός Εφαρμογής
Χωρίστε κάθε λειτουργία σε διαχειρίσιμα βήματα χρησιμοποιώντας το Aspose.Email.

### Σύνδεση και σύνδεση σε διακομιστή POP3
Η σύνδεση είναι απλή με το Aspose.Email:
1. **Ρύθμιση παραμέτρων του προγράμματος-πελάτη**:
   ```csharp
   Pop3Client client = new Pop3Client(host, port, username, password);
   ```
2. **Χειρισμός εξαιρέσεων σύνδεσης**:
   ```csharp
   try {
       // Επιτυχής σύνδεση και σύνδεση
   } catch (Exception ex) {
       Console.WriteLine(ex.Message); // Εμφάνιση μηνύματος σφάλματος εάν η σύνδεση αποτύχει
   }
   ```

### Λάβετε τα email που έφτασαν σήμερα
Για να φιλτράρετε τα email που λάβατε σήμερα:
1. **Δημιουργήστε το ερώτημα**:
   ```csharp
   MailQueryBuilder builder = new MailQueryBuilder();
   builder.InternalDate.On(DateTime.Now);
   ```
2. **Εκτέλεση και ανάκτηση μηνυμάτων**:
   ```csharp
   MailQuery query = builder.GetQuery();
   Pop3MessageInfoCollection messages = client.ListMessages(query);
   Console.WriteLine("Today: " + messages.Count + ": message(s) found.");
   ```

### Λήψη email από τις τελευταίες 7 ημέρες
Για να ανακτήσετε τα email της προηγούμενης εβδομάδας:
1. **Ορισμός εύρους ημερομηνιών**:
   ```csharp
   builder.InternalDate.Before(DateTime.Now);
   builder.InternalDate.Since(DateTime.Now.AddDays(-7));
   ```
2. **Ανάκτηση και εμφάνιση μηνυμάτων**:
   ```csharp
   query = builder.GetQuery();
   messages = client.ListMessages(query);
   Console.WriteLine("Last 7 Days: " + messages.Count + ": message(s) found.");
   ```

### Λήψη email από συγκεκριμένο αποστολέα
Φιλτράρισμα email κατά διεύθυνση αποστολέα:
1. **Ορισμός κριτηρίων αποστολέα**:
   ```csharp
   builder.From.Contains("specific.sender@example.com");
   ```
2. **Ανάκτηση και εξαγωγή μηνυμάτων**:
   ```csharp
   query = builder.GetQuery();
   messages = client.ListMessages(query);
   Console.WriteLine("Specific Sender: " + messages.Count + ": message(s) found.");
   ```

### Λήψη email από συγκεκριμένο τομέα
Για να φιλτράρετε τα email από έναν συγκεκριμένο τομέα:
1. **Ρύθμιση παραμέτρων κριτηρίων τομέα**:
   ```csharp
   builder.From.Contains("specificdomain.com");
   ```
2. **Εκτέλεση και εμφάνιση αποτελεσμάτων**:
   ```csharp
   query = builder.GetQuery();
   messages = client.ListMessages(query);
   Console.WriteLine("Specific Domain: " + messages.Count + ": message(s) found.");
   ```

### Λήψη email που αποστέλλονται σε συγκεκριμένο παραλήπτη
Φιλτράρισμα email που αποστέλλονται σε συγκεκριμένο παραλήπτη:
1. **Ορισμός κριτηρίων παραλήπτη**:
   ```csharp
   builder.To.Contains("recipient@example.com");
   ```
2. **Ανάκτηση και Εξαγωγή Μηνυμάτων**:
   ```csharp
   query = builder.GetQuery();
   messages = client.ListMessages(query);
   Console.WriteLine("Specific Recipient: " + messages.Count + ": message(s) found.");
   ```
