## Οδηγός Εφαρμογής

Ας χωρίσουμε τη διαδικασία σε λογικά τμήματα με βάση τα βασικά χαρακτηριστικά.

### Δημιουργία και ρύθμιση παραμέτρων Pop3Client

**Επισκόπηση:**
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε μια παρουσία του `Pop3Client` και να διαμορφώσετε τις ρυθμίσεις σύνδεσής του.

#### Βήμα 1: Δημιουργία νέας παρουσίας

Ξεκινήστε δημιουργώντας μια νέα παρουσία του `Pop3Client` τάξη:

```csharp
Pop3Client client = new Pop3Client();
```

#### Βήμα 2: Διαμόρφωση ρυθμίσεων σύνδεσης

Ορίστε τις απαραίτητες παραμέτρους όπως κεντρικό υπολογιστή, όνομα χρήστη, κωδικό πρόσβασης, θύρα και επιλογές ασφαλείας:

```csharp
client.Host = "pop.gmail.com"; // Καθορίστε τη διεύθυνση του διακομιστή POP3.
client.Username = "your.username@gmail.com"; // Ορίστε το όνομα χρήστη του email σας.
client.Password = "your.password"; // Ορίστε τον κωδικό πρόσβασης του email σας.
client.Port = 995; // Χρησιμοποιήστε τη θύρα 995 για συνδέσεις SSL.
client.SecurityOptions = SecurityOptions.Auto; // Αυτόματος προσδιορισμός επιλογών ασφαλείας.
```

**Εξήγηση:**
- **Πλήθος:** Η διεύθυνση του διακομιστή POP3. Για το Gmail, χρησιμοποιήστε `pop.gmail.com`.
- **Όνομα χρήστη και κωδικός πρόσβασης:** Τα διαπιστευτήριά σας μέσω email.
- **Λιμάνι:** Το 995 χρησιμοποιείται συνήθως για ασφαλείς συνδέσεις με SSL/TLS.
- **Επιλογές ασφαλείας:** Στρώνομαι στη δουλειά `Auto` για να επιτρέψετε στον πελάτη να καθορίσει αυτόματα το πρωτόκολλο ασφαλείας.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι το τείχος προστασίας ή το antivirus δεν εμποδίζουν τη σύνδεση.
- Ελέγξτε ξανά τα διαπιστευτήριά σας και τις ρυθμίσεις του διακομιστή σας, εάν αντιμετωπίσετε σφάλματα ελέγχου ταυτότητας.

### Ανάκτηση μεγέθους γραμματοκιβωτίου, πληροφοριών και αριθμού μηνυμάτων

**Επισκόπηση:**
Αυτή η λειτουργία δείχνει πώς να ανακτήσετε το μέγεθος του γραμματοκιβωτίου, τις πληροφορίες και τον αριθμό μηνυμάτων χρησιμοποιώντας ένα `Pop3Client` παράδειγμα.

#### Βήμα 1: Ανάκτηση μεγέθους γραμματοκιβωτίου

Χρησιμοποιήστε το `GetMailboxSize()` μέθοδος:

```csharp
long nSize = client.GetMailboxSize();
```

#### Βήμα 2: Λήψη λεπτομερών πληροφοριών

Λήψη λεπτομερών πληροφοριών γραμματοκιβωτίου, συμπεριλαμβανομένου του αριθμού μηνυμάτων και του μεγέθους καταληψιμότητας:

```csharp
Pop3MailboxInfo info = client.GetMailboxInfo();
int nMessageCount = info.MessageCount;
long nOccupiedSize = info.OccupiedSize;
```

**Εξήγηση:**
- **Μέγεθος:** Συνολικό μέγεθος του γραμματοκιβωτίου σε byte.
- **nMessageCount:** Αριθμός μηνυμάτων στο γραμματοκιβώτιο.
- **Μέγεθος χωρίς Κατοίκηση:** Χώρος που καταλαμβάνεται από ηλεκτρονικά μηνύματα.
