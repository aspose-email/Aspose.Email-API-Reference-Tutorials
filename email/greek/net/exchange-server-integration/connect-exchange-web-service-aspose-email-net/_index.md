## Οδηγός Εφαρμογής

### Δυνατότητα 1: Σύνδεση με την υπηρεσία Exchange Web
Η σύνδεση με τον Ενσωματωμένο Ιστότοπο (EWS) είναι το πρώτο βήμα για την αλληλεπίδραση με το Microsoft Exchange. Δείτε πώς μπορείτε να το πετύχετε αυτό:

#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει τη δημιουργία σύνδεσης με έναν διακομιστή Exchange χρησιμοποιώντας το Aspose.Email για .NET, επιτρέποντας περαιτέρω λειτουργίες όπως ανάκτηση email και δημιουργία ερωτημάτων.

#### Βήμα προς βήμα εφαρμογή

##### 1. Ορισμός λεπτομερειών διακομιστή EWS
Ξεκινήστε καθορίζοντας το URI του διακομιστή, το όνομα χρήστη, τον κωδικό πρόσβασης και τον τομέα:
```csharp
const string mailboxUri = "https://outlook.office365.com/ews/exchange.asmx";
const string username = "username"; // Αντικαταστήστε με το όνομα χρήστη σας
const string password = "password"; // Αντικαταστήστε με τον κωδικό πρόσβασής σας
cost string domain = "domain";    // Αντικαταστήστε με τον τομέα σας
```

##### 2. Δημιουργία σύνδεσης με το EWS
Χρησιμοποιήστε το `EWSClient.GetEWSClient` μέθοδος σύνδεσης:
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, username, password, domain);
Console.WriteLine("Connected to Exchange Web Service.");
client.Dispose();
```
**Εξήγηση**Η σύνδεση δημιουργείται χρησιμοποιώντας τα διαπιστευτήριά σας και τα στοιχεία του διακομιστή σας. Βεβαιωθείτε ότι είναι σωστά για να αποφύγετε εξαιρέσεις.

##### 3. Χειρισμός εξαιρέσεων
Να τυλίγετε πάντα τη λογική σύνδεσής σας σε ένα μπλοκ try-catch:
```csharp
try {
    // Ο κωδικός σύνδεσης εδώ...
} catch (Exception ex) {
    Console.WriteLine("Error connecting to EWS: " + ex.Message);
}
```
**Συμβουλή αντιμετώπισης προβλημάτων**Συνήθη προβλήματα περιλαμβάνουν λανθασμένα διαπιστευτήρια ή URI διακομιστή. Ελέγξτε ξανά αυτές τις τιμές εάν αντιμετωπίσετε σφάλματα.

### Χαρακτηριστικό 2: Δημιουργία ερωτημάτων με το ExchangeQueryBuilder
Η δημιουργία ερωτημάτων επιτρέπει το φιλτράρισμα και την αναζήτηση μηνυμάτων με βάση συγκεκριμένα κριτήρια.

#### Επισκόπηση
Μάθετε πώς να χρησιμοποιείτε το `ExchangeQueryBuilder` τάξη για να δημιουργήσετε στοχευμένες αναζητήσεις μέσω email.

#### Βήμα προς βήμα εφαρμογή

##### 1. Αρχικοποίηση του ExchangeQueryBuilder
Ξεκινήστε δημιουργώντας μια παρουσία του `ExchangeQueryBuilder`:
```csharp
ExchangeQueryBuilder builder = new ExchangeQueryBuilder();
```

##### 2. Ορισμός κριτηρίων για το ερώτημα
Προσθέστε συνθήκες στο ερώτημά σας, όπως φιλτράρισμα κατά θέμα ή ημερομηνία:
```csharp
builder.Subject.Contains("Newsletter");
builder.InternalDate.On(DateTime.Now);
```
**Εξήγηση**Αυτή η ρύθμιση αναζητά μηνύματα ηλεκτρονικού ταχυδρομείου με θέμα "Ενημερωτικό δελτίο" και τα οποία ελήφθησαν σήμερα.

##### 3. Δημιουργία MailQuery
Μετατρέψτε τον κατασκευαστή σας σε ένα `MailQuery` αντιρρήσεις για την εκτέλεσή του:
```csharp
MailQuery query = builder.GetQuery();
Console.WriteLine("Query built for subject containing 'Newsletter' and emails received today.");
```

### Λειτουργία 3: Ανάκτηση μηνυμάτων χρησιμοποιώντας το ερώτημα EWS
Με την εδραίωση της σύνδεσης και τα ερωτήματα έτοιμα, μπορείτε πλέον να ανακτήσετε μηνύματα από το γραμματοκιβώτιό σας του Exchange.

#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει την ανάκτηση μηνυμάτων ηλεκτρονικού ταχυδρομείου με βάση προκαθορισμένα κριτήρια χρησιμοποιώντας το Aspose.Email για .NET.

#### Βήμα προς βήμα εφαρμογή

##### 1. Σύνδεση με EWS (Επαναχρησιμοποίηση διαπιστευτηρίων)
Επαναφέρετε το πρόγραμμα-πελάτη EWS, εάν χρειάζεται:
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, username, password, domain);
```

##### 2. Δημιουργία και εκτέλεση ερωτήματος
Χρησιμοποιήστε το δικό σας `ExchangeQueryBuilder` για να φιλτράρετε μηνύματα:
```csharp
ExchangeQueryBuilder builder = new ExchangeQueryBuilder();
builder.Subject.Contains("Newsletter");
builder.InternalDate.On(DateTime.Now);
MailQuery query = builder.GetQuery();
```

##### 3. Ανάκτηση μηνυμάτων
Ανάκτηση των φιλτραρισμένων μηνυμάτων ηλεκτρονικού ταχυδρομείου από τα εισερχόμενα:
```csharp
ExchangeMessageInfoCollection messages = client.ListMessages(client.MailboxInfo.InboxUri, query, false);
Console.WriteLine("Retrieved " + messages.Count + " message(s) from inbox.");
client.Dispose();
```
**Εξήγηση**: Αυτό ανακτά όλα τα email που ταιριάζουν με τα κριτήριά σας και εμφανίζει τον αριθμό τους.
