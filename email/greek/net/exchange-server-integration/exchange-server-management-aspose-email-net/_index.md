## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίηση σε δύο κύριες λειτουργίες: σύνδεση σε διακομιστή Exchange και διαχείριση συνομιλιών.

### Σύνδεση με Exchange Server

**Επισκόπηση**
Η σύνδεση σε έναν διακομιστή Exchange σάς επιτρέπει να έχετε πρόσβαση και να διαχειρίζεστε στοιχεία γραμματοκιβωτίου μέσω προγραμματισμού.

#### Βήμα 1: Ρύθμιση διαπιστευτηρίων
```csharp
using System.Net;
using Aspose.Email.Clients.Exchange.WebService;

var mailboxUri = "https://exchange/ews/exchange.asmx"; // Αντικαταστήστε με τη διεύθυνση URL του Exchange Server σας
var domain = "\"; // Αφήστε κενό εάν δεν ισχύει
var username = "username@ASE305.onmicrosoft.com"; // Η διεύθυνση ηλεκτρονικού ταχυδρομείου σας
var password = "password"; // Ο ασφαλής κωδικός πρόσβασής σας

NetworkCredential credentials = new NetworkCredential(username, password, domain);
```
**Εξήγηση**: 
Ο `NetworkCredential` Το αντικείμενο περιέχει τα στοιχεία σύνδεσής σας, τα οποία είναι απαραίτητα για τον έλεγχο ταυτότητας. Βεβαιωθείτε ότι το URI δείχνει στο τελικό σημείο EWS του διακομιστή Exchange.

#### Βήμα 2: Δημιουργία σύνδεσης
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials); // Σύνδεση με διακομιστή Exchange
```
**Εξήγηση**: 
Αυτή η γραμμή κώδικα ξεκινά μια σύνδεση με τον Exchange server χρησιμοποιώντας το παρεχόμενο URI και τα διαπιστευτήρια. Επιστρέφει ένα `IEWSClient` αντικείμενο που μπορείτε να χρησιμοποιήσετε για περαιτέρω λειτουργίες.

### Εύρεση και Μετακίνηση Συνομιλιών

**Επισκόπηση**
Μόλις συνδεθείτε, μπορείτε να βρείτε συνομιλίες στα εισερχόμενά σας και να τις μετακινήσετε με βάση συγκεκριμένα κριτήρια.

#### Βήμα 1: Ανάκτηση συνομιλιών στα εισερχόμενα
```csharp
var inboxUri = client.MailboxInfo.InboxUri;
ExchangeMessageInfoCollection messages = client.ListMessages(inboxUri);
```
**Εξήγηση**: 
Ο `ListMessages` Η μέθοδος ανακτά όλες τις συνομιλίες στα εισερχόμενά σας. Αυτή η συλλογή θα χρησιμοποιηθεί για το φιλτράρισμα και τη μετακίνηση συγκεκριμένων συνομιλιών.

#### Βήμα 2: Μετακίνηση συνομιλιών με βάση μια συνθήκη
```csharp
foreach (var messageInfo in messages)
{
    if (messageInfo.ConversationTopic.Contains("Specific Keyword"))
    {
        client.MoveItem(messageInfo.UniqueUri, "DestinationFolderId");
    }
}
```
**Εξήγηση**: 
Επαναλάβετε κάθε συζήτηση και εφαρμόστε τη συνθήκη σας. Εάν το θέμα μιας συζήτησης περιέχει την καθορισμένη λέξη-κλειδί, μετακινήστε την σε έναν καθορισμένο φάκελο.

### Συμβουλές αντιμετώπισης προβλημάτων

- **Προβλήματα σύνδεσης**Επαληθεύστε ότι το URI του γραμματοκιβωτίου είναι σωστό και προσβάσιμο.
- **Αποτυχίες ελέγχου ταυτότητας**: Ελέγξτε ξανά τα διαπιστευτήρια για ακρίβεια.
