## Οδηγός Εφαρμογής
Αυτή η ενότητα καλύπτει τη σύνδεση σε έναν Exchange Server και την ανάκτηση επαφών.

### Σύνδεση με τον Exchange Server
Η ασφαλής σύνδεση με τον Exchange Server σας είναι απαραίτητη. Δείτε πώς:

#### Βήμα 1: Ορισμός διαπιστευτηρίων
Ρυθμίστε τα διαπιστευτήριά σας για το WebDAV χρησιμοποιώντας `NetworkCredential` για ασφαλή έλεγχο ταυτότητας.

```csharp
using System.Net;

string mailboxURI = "http://ex2003/exchange/administrator"; // URI διακομιστή Exchange
string username = "administrator";
string password = "pwd";
string domain = "domain.local";

// Δημιουργήστε διαπιστευτήρια δικτύου για σύνδεση στον διακομιστή
NetworkCredential credential = new NetworkCredential(username, password, domain);
```

#### Βήμα 2: Αρχικοποίηση `ExchangeClient`
Χρήση `ExchangeClient` από το Aspose.Email για να δημιουργήσετε μια σύνδεση.

```csharp
using Aspose.Email.Clients.Exchange.Dav;

// Δημιουργήστε μια παρουσία του ExchangeClient με URI και διαπιστευτήρια
tExchangeClient client = new ExchangeClient(mailboxURI, credential);
```

### Ανάκτηση επαφών από τον Exchange Server
Μόλις συνδεθείτε, ανακτήστε τις επαφές που είναι αποθηκευμένες στον διακομιστή σας.

#### Βήμα 1: Ανάκτηση επαφών
Χρήση `GetContacts` μέθοδος για την ανάκτηση όλων των επαφών.

```csharp
using Aspose.Email.Mapi;

// Ανάκτηση επαφών από τον κατάλογο
Contact[] contacts = client.GetContacts(client.MailboxInfo.ContactsUri);
```

#### Βήμα 2: Εμφάνιση στοιχείων επικοινωνίας
Περιηγηθείτε σε κάθε επαφή και εμφανίστε σχετικές λεπτομέρειες, όπως ονόματα και διευθύνσεις ηλεκτρονικού ταχυδρομείου.

```csharp
foreach (MapiContact contact in contacts)
{
    Console.WriteLine("Name: " + contact.NameInfo.DisplayName + ", Email Address: " + contact.ElectronicAddresses.Email1);
}
```

### Συμβουλές αντιμετώπισης προβλημάτων
