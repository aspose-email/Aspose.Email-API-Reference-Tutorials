## Οδηγός Εφαρμογής

### Σύνδεση με Exchange Server
Αυτή η λειτουργία εστιάζει στη δημιουργία σύνδεσης με έναν διακομιστή Exchange χρησιμοποιώντας τη βιβλιοθήκη Aspose.Email .NET.

#### Βήμα 1: Ορισμός διαπιστευτηρίων
```csharp
using System.Net;

string mailboxURI = "https://ex2010/ews/exchange.asmx";
string username = "test.exchange";
string password = "pwd";
string domain = "ex2010.local";

NetworkCredential credential = new NetworkCredential(username, password, domain);
```
#### Βήμα 2: Δημιουργία σύνδεσης
```csharp
using Aspose.Email.Clients.Exchange.WebService;

IEWSClient client = EWSClient.GetEWSClient(mailboxURI, credential);
// Τώρα μπορείτε να αλληλεπιδράσετε με τον διακομιστή.
```

### Ανάκτηση κανόνων εισερχομένων από τον Exchange Server
Μόλις συνδεθείτε, η ανάκτηση των κανόνων εισερχομένων είναι απλή.

#### Βήμα 1: Ανάκτηση κανόνων εισερχομένων
```csharp
using Aspose.Email.Clients.Exchange;

InboxRule[] inboxRules = client.GetInboxRules();
```
#### Βήμα 2: Εμφάνιση λεπτομερειών κανόνα
Επαναλάβετε τους ανακτημένους κανόνες για να εμφανίσετε τις λεπτομέρειες τους:
```csharp
foreach (InboxRule inboxRule in inboxRules)
{
    Console.WriteLine("Display Name: " + inboxRule.DisplayName);

    if (inboxRule.Conditions.FromAddresses.Count > 0)
    {
        foreach (MailAddress fromAddress in inboxRule.Conditions.FromAddresses)
        {
            Console.WriteLine("From: " + fromAddress.DisplayName + " - " + fromAddress.Address);
        }
    }
}
```

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι το URI του διακομιστή Exchange είναι σωστό.
- Επαληθεύστε ότι τα διαπιστευτήρια δικτύου είναι ακριβή και ότι έχετε τα απαραίτητα δικαιώματα.
- Ελέγξτε για τυχόν προβλήματα με το τείχος προστασίας που ενδέχεται να εμποδίζουν τη σύνδεση.
