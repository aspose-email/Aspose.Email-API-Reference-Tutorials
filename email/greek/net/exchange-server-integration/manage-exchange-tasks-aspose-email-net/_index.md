## Οδηγός Εφαρμογής

### Αρχικοποίηση του προγράμματος-πελάτη Exchange
Δημιουργήστε μια σύνδεση με τον διακομιστή Exchange:

#### Επισκόπηση
Δημιουργία μιας παρουσίας του `ExchangeClient` επιτρέπει την αλληλεπίδραση με τον διακομιστή Exchange. Αυτό το βήμα περιλαμβάνει την παροχή των απαραίτητων διαπιστευτηρίων και των URL τελικού σημείου.

#### Βήματα
1. **Συμπερίληψη απαιτούμενων χώρων ονομάτων**:
   ```csharp
   using Aspose.Email.Clients.Exchange.WebService;
   ```
2. **Αρχικοποίηση του προγράμματος-πελάτη**:
   ```csharp
   IEWSClient client = EWSClient.GetEWSClient(
       "https://outlook.office365.com/ews/exchange.asmx", 
       "testUser", 
       "pwd", 
       "domain"
   );
   ```
   - `GetEWSClient`: Συνδέεται με τον διακομιστή Exchange χρησιμοποιώντας τα παρεχόμενα διαπιστευτήρια.
   - Παράμετροι:
     - URL τελικού σημείου: Η διεύθυνση τελικού σημείου των υπηρεσιών Exchange Web Services.
     - Όνομα χρήστη, Κωδικός πρόσβασης, Τομέας: Διαπιστευτήρια για έλεγχο ταυτότητας.

### Ανάκτηση εργασιών από τον Exchange Server

#### Επισκόπηση
Η ανάκτηση εργασιών επιτρέπει την ιεράρχηση προτεραιοτήτων και τη διαχείριση του φόρτου εργασίας.

#### Βήματα
1. **Πρόσβαση στο URI εργασίας**:
   ```csharp
   using Aspose.Email.Clients.Exchange.WebService;

   public static void ListExchangeTasks(IEWSClient client)
   {
       ExchangeMessageInfoCollection tasks = client.ListMessages(client.MailboxInfo.TasksUri);
   }
   ```
   - `ListMessages`: Ανακτά όλα τα μηνύματα που σχετίζονται με την εργασία από τον διακομιστή.

### Φιλτράρισμα και διαγραφή εργασιών με βάση το θέμα

#### Επισκόπηση
Το φιλτράρισμα και η διαγραφή συγκεκριμένων εργασιών διατηρεί έναν καθαρό χώρο εργασίας, διασφαλίζοντας ότι μόνο οι σχετικές εργασίες παραμένουν ενεργές.

#### Βήματα
1. **Επανάληψη σε συλλογή εργασιών**:
   ```csharp
   using Aspose.Email.Clients.Exchange.WebService;
   using Aspose.Email.Mime;

   public static void FilterAndDeleteTasks(IEWSClient client)
   {
       foreach (ExchangeMessageInfo info in client.ListMessages(client.MailboxInfo.TasksUri))
       {
           ExchangeTask task = client.FetchTask(info.UniqueUri);
           
           if (task.Subject.Equals("test"))
           {
               client.DeleteItem(task.UniqueUri, DeletionOptions.DeletePermanently);
           }
       }
   }
   ```
   - `FetchTask`Ανακτά λεπτομερείς πληροφορίες σχετικά με μια συγκεκριμένη εργασία χρησιμοποιώντας το μοναδικό URI της.
   - `DeleteItem`: Διαγράφει οριστικά την εργασία από τον διακομιστή.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Σφάλματα ελέγχου ταυτότητας**Επαληθεύστε τα διαπιστευτήρια και τη διεύθυνση URL του τελικού σημείου. Ελέγξτε για προβλήματα δικτύου που εμποδίζουν την πρόσβαση.
