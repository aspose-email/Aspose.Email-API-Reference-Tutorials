## Οδηγός Εφαρμογής

### Σύνδεση με Exchange Server χρησιμοποιώντας EWS
Η σύνδεση με τον διακομιστή είναι ζωτικής σημασίας για τη διαχείριση των email μέσω προγραμματισμού.

#### Βήματα:
**Βήμα 1: Δημιουργήστε μια παρουσία του προγράμματος-πελάτη EWS**
```csharp
using System;
using Aspose.Email.Clients.Exchange.WebService;

public static void ConnectToExchangeServer()
{
    // Δημιουργήστε μια παρουσία του προγράμματος-πελάτη EWS με διεύθυνση URL διακομιστή, όνομα χρήστη, κωδικό πρόσβασης και τομέα
    IEWSClient client = EWSClient.GetEWSClient(
        "https://outlook.office365.com/ews/exchange.asmx", 
        "testUser", 
        "pwd", 
        "domain");
}
```
**Εξήγηση:**
- `GetEWSClient` αρχικοποιεί μια σύνδεση χρησιμοποιώντας τα παρεχόμενα διαπιστευτήρια.

### Δημιουργία και προσθήκη νέου μηνύματος ηλεκτρονικού ταχυδρομείου
Μάθετε πώς να δημιουργείτε ένα μήνυμα ηλεκτρονικού ταχυδρομείου και να το προσθέτετε στον διακομιστή σας.

#### Βήματα:
**Βήμα 1: Δημιουργήστε ένα αντικείμενο MailMessage**
```csharp
using System;
using Aspose.Email.Mime;
using Aspose.Email.Clients.Exchange.WebService;

public static void CreateAndAppendEmail(IEWSClient client)
{
    // Δημιουργήστε ένα νέο αντικείμενο MailMessage
    MailMessage message = new MailMessage(
        "from@domain.com", 
        "to@domain.com", 
        "EMAILNET-34997 - " + Guid.NewGuid().ToString(), 
        "EMAILNET-34997 Exchange: Copy a message and get reference to the new Copy item");

    // Προσάρτηση του δημιουργημένου μηνύματος ηλεκτρονικού ταχυδρομείου στον διακομιστή
    string messageUri = client.AppendMessage(message);
}
```
**Εξήγηση:**
- `MailMessage` αντιπροσωπεύει ένα email με αποστολέα, παραλήπτη, θέμα και σώμα.
- `AppendMessage` αποθηκεύει το μήνυμα στον διακομιστή.

### Αντιγραφή μηνύματος ηλεκτρονικού ταχυδρομείου σε άλλο φάκελο
Οργανώστε αποτελεσματικά τα email σας αντιγράφοντάς τα μεταξύ φακέλων χρησιμοποιώντας το URI τους.

#### Βήματα:
**Βήμα 1: Χρησιμοποιήστε το IEWSClient για να αντιγράψετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου**
```csharp
using System;
using Aspose.Email.Clients.Exchange.WebService;

public static void CopyEmailMessage(IEWSClient client, string messageUri)
{
    // Αντιγραφή μηνύματος ηλεκτρονικού ταχυδρομείου στον φάκελο DeletedItems
    string newMessageUri = client.CopyItem(
        messageUri, 
        client.MailboxInfo.DeletedItemsUri);
}
```
**Εξήγηση:**
- `CopyItem` μετακινεί ένα μήνυμα από την τρέχουσα θέση του σε έναν άλλο φάκελο.
