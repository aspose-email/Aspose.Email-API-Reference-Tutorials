## Οδηγός Εφαρμογής

### Σύνδεση με τις υπηρεσίες Exchange Web Services
**Επισκόπηση:** Δημιουργήστε μια σύνδεση με έναν διακομιστή Exchange χρησιμοποιώντας το `EWSClient` τάξη από το Aspose.Email.

#### Βήμα 1: Δημιουργήστε μια παρουσία του IEWSClient
Δώστε τη διεύθυνση URL του διακομιστή σας, το όνομα χρήστη, τον κωδικό πρόσβασης και τον τομέα σας:
```csharp
using Aspose.Email.Clients.Exchange.WebService;

public void ConnectToExchangeWebService()
{
    // Αρχικοποίηση του προγράμματος-πελάτη EWS με διαπιστευτήρια
    IEWSClient client = EWSClient.GetEWSClient(
        "https://outlook.office365.com/ews/exchange.asmx",
        "testUser",
        "pwd",
        "domain"
    );
    
    // Ο «πελάτης» είναι πλέον έτοιμος να αλληλεπιδράσει με τον διακομιστή Exchange.
}
```
**Επεξήγηση παραμέτρων:**
- **URL διακομιστή**: Τελικό σημείο για τις υπηρεσίες Exchange Web Services. Επαληθεύστε την προσβασιμότητά του.
- **Όνομα χρήστη, Κωδικός πρόσβασης, Τομέας**: Διαπιστευτήρια για έλεγχο ταυτότητας στον διακομιστή Exchange.

### Ανάκτηση πληροφοριών γραμματοκιβωτίου
**Επισκόπηση:** Μόλις συνδεθείτε, ανακτήστε λεπτομέρειες γραμματοκιβωτίου, όπως μέγεθος και URI φακέλων.

#### Βήμα 1: Λήψη μεγέθους γραμματοκιβωτίου
Ανάκτηση του συνολικού μεγέθους του γραμματοκιβωτίου σε byte:
```csharp
long mailboxSize = client.GetMailboxSize();
```

#### Βήμα 2: Λήψη πληροφοριών γραμματοκιβωτίου
Ανάκτηση URI για εισερχόμενα, απεσταλμένα στοιχεία, πρόχειρα κ.λπ.:
```csharp
ExchangeMailboxInfo mailboxInfo = client.GetMailboxInfo();

string mailboxUri = mailboxInfo.MailboxUri;
string inboxUri = mailboxInfo.InboxUri;
string sentItemsUri = mailboxInfo.SentItemsUri;
string draftsUri = mailboxInfo.DraftsUri;

// Χρησιμοποιήστε αυτά τα URI για να αλληλεπιδράσετε με συγκεκριμένους φακέλους.
```
**Επιστρεφόμενες τιμές:**
- **Μέγεθος γραμματοκιβωτίου**: Μέγεθος του γραμματοκιβωτίου σε byte.
- **Πληροφορίες ExchangeMailbox**Περιέχει URI και πρόσθετες λεπτομέρειες σχετικά με το γραμματοκιβώτιο.

### Συμβουλές αντιμετώπισης προβλημάτων
- Επαληθεύστε ότι τα διαπιστευτήρια είναι σωστά και ότι έχετε τα απαραίτητα δικαιώματα.
- Ελέγξτε τη συνδεσιμότητα δικτύου με τη διεύθυνση URL του διακομιστή Exchange.
- Βεβαιωθείτε ότι καμία ρύθμιση τείχους προστασίας ή διακομιστή μεσολάβησης δεν εμποδίζει την πρόσβαση.
