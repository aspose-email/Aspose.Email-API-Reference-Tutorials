## Οδηγός Εφαρμογής

Τώρα που η ρύθμισή μας έχει ολοκληρωθεί, ας προχωρήσουμε στην εφαρμογή της δυνατότητας καταχώρισης μηνυμάτων από έναν διακομιστή Exchange.

### Σύνδεση σε διακομιστή Exchange

Για να καταχωρίσετε τα email, συνδεθείτε πρώτα στον Exchange server χρησιμοποιώντας το Aspose.Email. Θα χρειαστείτε το URI του server και τα διαπιστευτήριά σας για αυτό το βήμα.

**Βήμα 1: Δημιουργία σύνδεσης**

Δημιουργήστε μια νέα παρουσία του `ExchangeClient`:

```csharp
string exchangeUri = "https://Όνομα_υπολογιστή/ανταλλαγή/Όνομα_χρήστη"; // Το URI του Exchange Server σας
string username = "username"; // Όνομα χρήστη του Exchange Server σας
string password = "password"; // Ο κωδικός πρόσβασής σας στον Exchange Server

try
{
    var domain = new Domain(); // Πλαίσιο κράτησης θέσης για την κλάση τομέα, εάν χρειάζεται
    ExchangeClient client = new ExchangeClient(exchangeUri, username, password, domain);

    // Συνέχεια στη λίστα μηνυμάτων
}
catch (Exception ex)
{
    Console.Write("Error connecting: " + ex.Message);
}
```

Εδώ, `ExchangeClient` λαμβάνει το URI του διακομιστή και τα διαπιστευτήρια ως παραμέτρους, διευκολύνοντας μια ασφαλή σύνδεση.

### Λίστα μηνυμάτων από τα Εισερχόμενα

Με μια εδραιωμένη σύνδεση, μπορούμε πλέον να ανακτήσουμε τα email:

**Βήμα 2: Ανάκτηση μηνυμάτων**

Χρησιμοποιήστε τον πελάτη για να λάβετε μηνύματα από τα εισερχόμενά σας:

```csharp
try
{
    ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);

    // Εμφάνιση πληροφοριών μηνύματος
}
catch (Exception ex)
{
    Console.Write("Error retrieving messages: " + ex.Message);
}
```

`ListMessages` ανακτά όλα τα μηνύματα από το καθορισμένο URI γραμματοκιβωτίου, επιστρέφοντάς τα ως συλλογή.

### Εμφάνιση πληροφοριών μηνύματος

Αφού ανακτήσετε τα μηνύματα, μπορείτε να τα ανακτήσετε ξανά για να εμφανίσετε τις απαραίτητες λεπτομέρειες:

**Βήμα 3: Επανάληψη και εμφάνιση**

```csharp
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Sent Date: " + msgInfo.Date.ToString());
    Console.WriteLine("Read?: " + msgInfo.IsRead.ToString());
    Console.WriteLine("Message ID: " + msgInfo.MessageId);
    Console.WriteLine("Unique URI: " + msgInfo.UniqueUri);
}
```

Αυτός ο βρόχος επαναλαμβάνεται σε κάθε μήνυμα, εκτυπώνοντας βασικά χαρακτηριστικά όπως θέμα, αποστολέα, παραλήπτη και κατάσταση ανάγνωσης.
