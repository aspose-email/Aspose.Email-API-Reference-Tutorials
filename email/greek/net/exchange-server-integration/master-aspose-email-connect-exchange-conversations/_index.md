## Οδηγός Εφαρμογής

### Σύνδεση με Exchange Server χρησιμοποιώντας EWSClient

**Επισκόπηση**: Δημιουργία σύνδεσης με έναν διακομιστή Exchange χρησιμοποιώντας το Aspose.Email `EWSClient`.

#### Βήμα 1: Ρύθμιση διαπιστευτηρίων
Ρυθμίστε τις παραμέτρους των διαπιστευτηρίων δικτύου που χρησιμοποιούνται για τον έλεγχο ταυτότητας με τον διακομιστή Exchange:
```csharp
using System;
using System.Net;
using Aspose.Email.Clients.Exchange.WebService;

const string mailboxUri = "https://ανταλλαγή/ews/exchange.asmx";
const string domain = "";
const string username = "username@ASE305.onmicrosoft.com";
const string password = "password";

// Δημιουργία αντικειμένου NetworkCredential με διαπιστευτήρια χρήστη
NetworkCredential credentials = new NetworkCredential(username, password, domain);
```

#### Βήμα 2: Σύνδεση με τον Exchange Server
Χρησιμοποιώντας `EWSClient`, συνδεθείτε στο γραμματοκιβώτιό σας:
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
Console.WriteLine("Connected to Exchange Server");
```

### Εύρεση και διαγραφή συγκεκριμένων συνομιλιών

**Επισκόπηση**: Ανάκτηση συνομιλιών από τα εισερχόμενα και διαγραφή όσων αντιστοιχούν σε συγκεκριμένα κριτήρια.

#### Βήμα 1: Ανάκτηση όλων των στοιχείων συνομιλίας
Ανάκτηση όλων των στοιχείων συνομιλίας στον φάκελο Εισερχομένων:
```csharp
using Aspose.Email.Clients.Exchange.WebService;
using Aspose.Email.Mime;

// Ανάκτηση συνομιλιών από τα Εισερχόμενα
ExchangeConversation[] conversations = client.FindConversations(client.MailboxInfo.InboxUri);
```

#### Βήμα 2: Έλεγχος θέματος συνομιλίας και διαγραφή
Επαναλάβετε κάθε συζήτηση για να βρείτε εκείνες που πληρούν τα κριτήριά σας:
```csharp
foreach (ExchangeConversation conversation in conversations)
{
    // Ελέγξτε εάν το θέμα της συνομιλίας περιέχει μια συγκεκριμένη συμβολοσειρά
    if (conversation.ConversationTopic.Contains("test email"))
    {
        // Διαγραφή στοιχείων συνομιλίας με βάση την συνθήκη
        client.DeleteConversationItems(conversation.ConversationId);
    }
}
```
