## Οδηγός Εφαρμογής

### Δυνατότητα 1: Έλεγχος ταυτότητας και ρύθμιση προγράμματος-πελάτη Exchange

Επαληθεύστε τον έλεγχο ταυτότητας και δημιουργήστε μια παρουσία του `ExchangeClient` σε αυτήν την ενότητα.

**Επισκόπηση:**
Ο έλεγχος ταυτότητας με τον διακομιστή Exchange είναι απαραίτητος για την πρόσβαση στο ηλεκτρονικό ταχυδρομείο. Δείτε πώς μπορείτε να ρυθμίσετε έναν πελάτη χρησιμοποιώντας τα διαπιστευτήριά σας.

#### Βήμα 1: Δημιουργήστε το `ExchangeClient` Παράδειγμα
```csharp
using Aspose.Email.Clients.Exchange;

// Ορίστε τη διεύθυνση URL του διακομιστή σας, το όνομα χρήστη, τον κωδικό πρόσβασης και τον τομέα.
string url = "http://ex07sp1/ανταλλαγή/Διαχειριστής";
string username = "user";
string password = "pwd";
string domain = "domain";

// Αρχικοποιήστε το ExchangeClient με διαπιστευτήρια.
ExchangeClient client = new ExchangeClient(url, username, password, domain);
```

**Εξήγηση:**
- **διεύθυνση URL**: Η διεύθυνση URL του διακομιστή όπου φιλοξενείται ο Exchange Server σας.
- **όνομα χρήστη/κωδικός πρόσβασης/τομέας**: Απαιτούνται διαπιστευτήρια για τον έλεγχο ταυτότητας.

### Λειτουργία 2: Καταχώριση μηνυμάτων από τα Εισερχόμενα

Χρησιμοποιήστε το πιστοποιημένο `ExchangeClient` για να εμφανίσετε τα μηνύματα στα εισερχόμενα.

**Επισκόπηση:**
Η δημιουργία καταχωρίσεων email μέσω προγραμματισμού εξοικονομεί χρόνο και αυτοματοποιεί επαναλαμβανόμενες εργασίες. Δείτε πώς μπορείτε να ανακτήσετε αποτελεσματικά τα μηνύματα email.

#### Βήμα 2: Ανάκτηση μηνυμάτων ηλεκτρονικού ταχυδρομείου
```csharp
// Ας υποθέσουμε ότι ο «πελάτης» έχει ήδη δημιουργηθεί όπως φαίνεται προηγουμένως.
ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);
```

**Εξήγηση:**
- `ListMessages`Ανακτά όλα τα μηνύματα από το καθορισμένο URI γραμματοκιβωτίου (σε αυτήν την περίπτωση, τα εισερχόμενα).

### Λειτουργία 3: Εμφάνιση πληροφοριών μηνύματος

Περιηγηθείτε στα ανακτημένα μηνύματα και εμφανίστε τις βασικές τους πληροφορίες.

#### Βήμα 3: Εκτύπωση λεπτομερειών email
```csharp
using System;

// Κάντε επανάληψη σε κάθε μήνυμα στη συλλογή.
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Message Size: " + msgInfo.Size);
    Console.WriteLine("==================================");
}
```

**Εξήγηση:**
- **Πληροφορίες μηνύματος**: Αντιπροσωπεύει ένα μεμονωμένο email, παρέχοντας πρόσβαση σε ιδιότητες όπως `Subject`, `From`, και `Size`.
