## Οδηγός Εφαρμογής

### Σύνδεση με Exchange Server

**Επισκόπηση**Η δημιουργία σύνδεσης με τον διακομιστή Exchange είναι απαραίτητη για την πρόσβαση και τη διαχείριση των email μέσω προγραμματισμού.

#### Βήμα 1: Ορισμός διαπιστευτηρίων
Ορίστε τα διαπιστευτήρια δικτύου σας χρησιμοποιώντας `NetworkCredential`, το οποίο περιλαμβάνει το όνομα χρήστη, τον κωδικό πρόσβασης και τον τομέα σας. Αυτό διασφαλίζει ασφαλή έλεγχο ταυτότητας κατά τη σύνδεση στον διακομιστή.

```csharp
using System.Net;

const string mailboxUri = "https://ανταλλαγή/ews/exchange.asmx";
const string domain = "";
const string username = "username@ASE305.onmicrosoft.com";
const string password = "password";

NetworkCredential credentials = new NetworkCredential(username, password, domain);
```

#### Βήμα 2: Δημιουργία προγράμματος-πελάτη EWS
Χρήση `EWSClient.GetEWSClient` για να δημιουργήσετε μια παρουσία του `IEWSClient`, διευκολύνοντας τη σύνδεση με τον διακομιστή Exchange.

```csharp
// Δημιουργήστε μια παρουσία του IEWSClient για να συνδεθείτε στον διακομιστή Exchange.
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
```

### Εύρεση συνομιλιών στα Εισερχόμενα

**Επισκόπηση**Αφού δημιουργήσετε μια σύνδεση, μπορείτε να βρείτε και να επαναλάβετε τις συζητήσεις στα εισερχόμενά σας.

#### Βήμα 1: Ανάκτηση συνομιλιών
Χρήση `FindConversations` για να ανακτήσετε όλα τα στοιχεία συνομιλίας από τον φάκελο εισερχομένων. Αυτή η μέθοδος επιστρέφει έναν πίνακα από `ExchangeConversation` αντικείμενα.

```csharp
using Aspose.Email.Clients.Exchange.WebService;

// Υποθέτοντας ότι ο «πελάτης» είναι ήδη συνδεδεμένος.
ExchangeConversation[] conversations = client.FindConversations(client.MailboxInfo.InboxUri);
```

#### Βήμα 2: Εμφάνιση λεπτομερειών συνομιλίας
Επαναλάβετε κάθε συζήτηση και εμφανίστε σχετικές λεπτομέρειες, όπως το θέμα και την κατάσταση σημαίας. Αυτό μπορεί να βοηθήσει στο φιλτράρισμα ή την επεξεργασία συγκεκριμένων νημάτων.

```csharp
foreach (ExchangeConversation conversation in conversations)
{
    Console.WriteLine("Topic: " + conversation.ConversationTopic);
    Console.WriteLine("Flag Status: " + conversation.FlagStatus.ToString());
}
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Προβλήματα σύνδεσης**Βεβαιωθείτε ότι το URI του γραμματοκιβωτίου σας και τα διαπιστευτήρια είναι σωστά. Ελέγξτε για τυχόν περιορισμούς δικτύου ή ρυθμίσεις τείχους προστασίας που ενδέχεται να εμποδίζουν τη σύνδεση.
