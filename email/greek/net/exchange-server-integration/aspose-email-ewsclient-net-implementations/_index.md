## Οδηγός Εφαρμογής

### Αρχικοποίηση προγράμματος-πελάτη Exchange

Δημιουργήστε στιγμιότυπα του `EWSClient` κλάση χρησιμοποιώντας διαπιστευτήρια χρήστη:

**Αρχικοποίηση Πελατών:**

```csharp
using Aspose.Email.Clients.Exchange.WebService;

// Δημιουργία προγραμμάτων-πελατών EWS για δύο διαφορετικούς χρήστες
IEWSClient client1 = EWSClient.GetEWSClient("https://outlook.office365.com/ews/exchange.asmx", "testUser1", "προεπιλογή χρήστη", "τομέας");
IEWSClient client2 = EWSClient.GetEWSClient("https://outlook.office365.com/ews/exchange.asmx", "testUser2", "προβλήματα χρήστη", "τομέας");
```

### Διαγραφή και Προσθήκη Μηνύματος

Διαγραφή μηνυμάτων από έναν συγκεκριμένο φάκελο και προσθήκη νέων.

**Διαγραφή μηνυμάτων:**

```csharp
using Aspose.Email.Clients.Exchange;
using Aspose.Email.Mime;

string folder = "Drafts";

// Διαγραφή όλων των μηνυμάτων στον καθορισμένο φάκελο για τον client1
foreach (ExchangeMessageInfo messageInfo in client1.ListMessages(folder))
{
    client1.DeleteItem(messageInfo.UniqueUri, DeletionOptions.DeletePermanently);
}
```

**Προσθήκη μηνυμάτων:**

```csharp
string subj1 = string.Format("NETWORKNET_33354 {0} {1}", "User", "User1");
client1.AppendMessage(folder, new MailMessage("User1@exchange.conholdate.local", "To@aspsoe.com", subj1, ""));

// Επαναλάβετε για τον client2 με διαφορετικό θέμα και παραλήπτες
```

### Πλαστοπροσωπία και Καταχώριση Μηνυμάτων

Μίμηση χρήστη για την εμφάνιση μηνυμάτων.

**Μίμηση χρήστη:**

```csharp
client1.ImpersonateUser(ItemChoice.PrimarySmtpAddress, "User2@exchange.conholdate.local");
ExchangeMessageInfoCollection messInfoColl1 = client1.ListMessages(folder);

// Επαναφορά πλαστοπροσωπίας
client1.ResetImpersonation();
```
