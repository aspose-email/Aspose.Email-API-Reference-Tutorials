### Δυνατότητα 1: Σύνδεση με την υπηρεσία Exchange Web Service (H2)

#### Επισκόπηση
Η σύνδεση σε έναν διακομιστή Exchange χρησιμοποιώντας το Aspose.Email σάς επιτρέπει να αλληλεπιδράτε με το γραμματοκιβώτιό σας μέσω προγραμματισμού. Αυτή η λειτουργία εστιάζει στη δημιουργία σύνδεσης μέσω `EWSClient`.

**Βήμα 1**: Δημιουργήστε μια παρουσία του `EWSClient`.

```csharp
using System;
using Aspose.Email.Clients.Exchange.WebService;

public class ConnectToExchangeWebService
{
    public void Run()
    {
        // Αρχικοποίηση EWSClient με τη διεύθυνση URL του διακομιστή και τα διαπιστευτήρια
        IEWSClient client = EWSClient.GetEWSClient(
            "https://outlook.office365.com/ews/exchange.asmx",
            "testUser",  // Όνομα χρήστη
            "pwd",       // Σύνθημα
            "domain"     // Πεδίο ορισμού
        );
    }
}
```

**Εξήγηση**: Το `GetEWSClient` Η μέθοδος απαιτεί τη διεύθυνση URL του διακομιστή, τα διαπιστευτήρια χρήστη (όνομα χρήστη, κωδικό πρόσβασης και τομέα). Αυτή η ρύθμιση είναι κρίσιμη για τον έλεγχο ταυτότητας και την πρόσβαση στο γραμματοκιβώτιό σας.

### Δυνατότητα 2: Πρόσβαση σε προσαρμοσμένο φάκελο στο γραμματοκιβώτιο του Exchange (H2)

#### Επισκόπηση
Μόλις συνδεθείτε, ίσως χρειαστεί να αποκτήσετε πρόσβαση σε συγκεκριμένους φακέλους μέσα στο γραμματοκιβώτιό σας. Αυτή η λειτουργία δείχνει τον έλεγχο της ύπαρξης ενός προσαρμοσμένου φακέλου και την καταγραφή των μηνυμάτων του.

**Βήμα 1**: Ελέγξτε αν υπάρχει ο προσαρμοσμένος φάκελος.

```csharp
using Aspose.Email.Clients.Exchange;
using Aspose.Email.Clients.Exchange.WebService;

public class AccessCustomFolder
{
    public void Run(IEWSClient client)
    {
        // Λήψη πληροφοριών γραμματοκιβωτίου
        ExchangeMailboxInfo mailbox = client.GetMailboxInfo();
        ExchangeMessageInfoCollection messages = null;
        ExchangeFolderInfo subfolderInfo = new ExchangeFolderInfo();

        // Ελέγξτε την ύπαρξη του προσαρμοσμένου φακέλου
        client.FolderExists(mailbox.InboxUri, "TestInbox", out subfolderInfo);

        if (subfolderInfo != null)
        {
            // Λίστα μηνυμάτων στον φάκελο που βρέθηκε
            messages = client.ListMessages(subfolderInfo.Uri);
            foreach (ExchangeMessageInfo info in messages)
            {
                string strMessageURI = info.UniqueUri;
                MailMessage msg = client.FetchMessage(strMessageURI);
                Console.WriteLine("Subject: " + msg.Subject);
            }
        }
        else
        {
            Console.WriteLine("No folder with this name found.");
        }
    }
}
```

**Εξήγηση**: Το `FolderExists` Η μέθοδος ελέγχει την ύπαρξη ενός συγκεκριμένου φακέλου, επιστρέφοντας το URI του, εάν υπάρχει. Εάν ο φάκελος υπάρχει, `ListMessages` ανακτά όλα τα μηνύματα που περιέχει.
