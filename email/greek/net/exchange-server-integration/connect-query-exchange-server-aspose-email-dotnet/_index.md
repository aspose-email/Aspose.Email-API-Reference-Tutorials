## Οδηγός Εφαρμογής

### Σύνδεση με την υπηρεσία Exchange Web Service (EWS)
Αυτή η ενότητα δείχνει πώς να συνδεθείτε σε έναν διακομιστή Exchange χρησιμοποιώντας EWS με διαπιστευτήρια χρήστη. Θα χρησιμοποιήσουμε το Aspose.Email για .NET για να το πετύχουμε αυτό.

#### Επισκόπηση
Η σύνδεση με το EWS σάς επιτρέπει να αλληλεπιδράτε μέσω προγραμματισμού με τις υπηρεσίες email σας, επιτρέποντας εργασίες αυτοματοποίησης και ενοποίησης απευθείας από την εφαρμογή σας.

**Βήμα 1: Εισαγωγή απαραίτητων χώρων ονομάτων**

```csharp
using Aspose.Email.Clients.Exchange;
using Aspose.Email.Clients.Exchange.WebService;
```

**Βήμα 2: Ρύθμιση διαπιστευτηρίων**
Αντικαθιστώ `"mailboxUri"`, `"username"`, `"password"`, και `"domain"` με τις πραγματικές σου αξίες.

```csharp
const string mailboxUri = "https://outlook.office365.com/ews/exchange.asmx";
const string username = "your_username";
const string password = "your_password";
const string domain = "your_domain";
```

**Βήμα 3: Δημιουργήστε ένα πρόγραμμα-πελάτη EWS**
Αυτό το απόσπασμα δείχνει πώς να δημιουργήσετε και να απορρίψετε ένα `IEWSClient` παράδειγμα.

```csharp
try
{
    // Δημιουργήστε μια σύνδεση χρησιμοποιώντας τα καθορισμένα διαπιστευτήρια.
    IEWSClient client = EWSClient.GetEWSClient(mailboxUri, username, password, domain);
    
    // Χρησιμοποιήστε τον πελάτη για διάφορες λειτουργίες...

    // Να φροντίζετε πάντα να αποσυνδέεστε μετά την ολοκλήρωση των εργασιών σας.
    client.Dispose();
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message);  // Καταγράψτε τυχόν εξαιρέσεις που προκύπτουν
}
```

**Εξήγηση:**
- **Παράμετροι**: `mailboxUri`, `username`, `password`, και `domain` είναι απαραίτητα για την αυθεντικοποίηση.
- **Επιστρεφόμενες τιμές**: Ένα παράδειγμα του `IEWSClient` Επιστρέφεται η εντολή, την οποία μπορείτε να χρησιμοποιήσετε για να αλληλεπιδράσετε με τον EWS.

### Δημιουργία ερωτήματος αλληλογραφίας χρησιμοποιώντας το ExchangeQueryBuilder
Τώρα που συνδεθήκαμε με τον διακομιστή, ας δημιουργήσουμε ένα ερώτημα ηλεκτρονικού ταχυδρομείου. Θα επικεντρωθούμε στα μηνύματα ηλεκτρονικού ταχυδρομείου με θέμα "Ενημερωτικό δελτίο" που αποστέλλονται σήμερα.

#### Επισκόπηση
Χρησιμοποιώντας `ExchangeQueryBuilder`, μπορείτε εύκολα να δημιουργήσετε ερωτήματα για το φιλτράρισμα και την ανάκτηση συγκεκριμένων μηνυμάτων ηλεκτρονικού ταχυδρομείου από το γραμματοκιβώτιό σας.

**Βήμα 1: Εισαγωγή του χώρου ονομάτων αναζήτησης**

```csharp
using Aspose.Email.Tools.Search;
```

**Βήμα 2: Αρχικοποίηση του ExchangeQueryBuilder**
Το εργαλείο δημιουργίας χρησιμοποιείται για τη ρύθμιση κριτηρίων αναζήτησης για email.

```csharp
ExchangeQueryBuilder builder = new ExchangeQueryBuilder();

// Συμπεριλάβετε μόνο email με την ένδειξη «Ενημερωτικό δελτίο» στη γραμμή θέματος.
builder.Subject.Contains("Newsletter", true);

// Φιλτράρισμα email που αποστέλλονται την τρέχουσα ημέρα.
builder.InternalDate.On(DateTime.Now);
```

**Βήμα 3: Κατασκευή και χρήση του ερωτήματος**
Το κατασκευασμένο ερώτημα μπορεί να χρησιμοποιηθεί για την εμφάνιση μηνυμάτων που πληρούν τα κριτήριά σας.

```csharp
MailQuery query = builder.GetQuery();

// Το αντικείμενο `query` είναι τώρα έτοιμο για χρήση με τη μέθοδο ListMessages για την ανάκτηση email.
```
