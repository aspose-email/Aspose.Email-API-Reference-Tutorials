Αυτός ο οδηγός θα σας δείξει πώς να συνδεθείτε σε έναν διακομιστή Exchange και να παραθέσετε υποφακέλους σε ένα γραμματοκιβώτιο με ακρίβεια και ευκολία.

### Τι θα μάθετε:

## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στη σύνδεση με έναν διακομιστή Exchange και στην καταχώριση υποφακέλων χρησιμοποιώντας το Aspose.Email για .NET.

### Σύνδεση με Exchange Server χρησιμοποιώντας EWS

**Επισκόπηση:**
Η σύνδεση σε έναν διακομιστή Exchange μέσω EWS επιτρέπει στην εφαρμογή σας να αλληλεπιδρά απρόσκοπτα με δεδομένα email.

#### Βήμα 1: Καθορισμός διαπιστευτηρίων δικτύου
Ορίστε τα απαραίτητα διαπιστευτήρια για τον έλεγχο ταυτότητας με τον διακομιστή Exchange:
```csharp
string mailboxUri = "https://ανταλλαγή/ews/exchange.asmx";
string domain = @"";
string username = "username@ASE305.onmicrosoft.com";
string password = "password";

NetworkCredential credentials = new NetworkCredential(username, password, domain);
```
*Εξήγηση:* Αυτά τα διαπιστευτήρια ελέγχουν την ταυτότητα της εφαρμογής σας στον διακομιστή.

#### Βήμα 2: Αρχικοποίηση προγράμματος-πελάτη EWS
Δημιουργήστε μια παρουσία του `IEWSClient` για επικοινωνία με τον διακομιστή Exchange:
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
```
*Παράμετροι:* 
- `mailboxUri`: Το τελικό σημείο για τον διακομιστή Exchange.
- `credentials`: Τα στοιχεία ελέγχου ταυτότητας δικτύου σας.

### Λίστα υποφακέλων στον Exchange Server χρησιμοποιώντας τον EWS

**Επισκόπηση:**
Η δημιουργία καταλόγων σε υποφακέλους μέσα σε ένα γραμματοκιβώτιο ηλεκτρονικού ταχυδρομείου σάς βοηθά να οργανώνετε και να διαχειρίζεστε αποτελεσματικά τα μηνύματα ηλεκτρονικού ταχυδρομείου.

#### Βήμα 1: Ανάκτηση πληροφοριών ριζικού φακέλου
Ανακτήστε τον ριζικό φάκελο για να εμφανίσετε τους υποφακέλους του:
```csharp
using Aspose.Email.Clients.Exchange;

ExchangeFolderInfoCollection folderInfoCol = client.ListSubFolders(client.MailboxInfo.RootUri);
```
*Επιστρεφόμενη τιμή:* Μια συλλογή από `ExchangeFolderInfo` αντικείμενα που αντιπροσωπεύουν κάθε υποφάκελο.

#### Βήμα 2: Επεξεργασία κάθε υποφακέλου
Επαναλάβετε τη συλλογή για να αποκτήσετε πρόσβαση στις λεπτομέρειες κάθε φακέλου:
```csharp
foreach (ExchangeFolderInfo folderInfo in folderInfoCol)
{
    // Επεξεργασία κάθε φακέλου με βάση τον τύπο του
}
```
*Εξήγηση:* Αυτός ο βρόχος σάς επιτρέπει να χειρίζεστε φακέλους ξεχωριστά, προσαρμόζοντας τις ενέργειες ανάλογα με τις ανάγκες.

### Προσδιορισμός τύπων φακέλων χρησιμοποιώντας το EWS

**Επισκόπηση:**
Η κατανόηση των τύπων φακέλων βοηθά στην προσαρμογή συγκεκριμένων λειτουργιών για διαφορετικές κατηγορίες email.

#### Βήμα 1: Προσδιορίστε τον τύπο φακέλου
Χρησιμοποιήστε μια πρόταση εναλλαγής πεζών-κεφαλαίων για να αναγνωρίσετε και να επεξεργαστείτε κάθε τύπο φακέλου:
```csharp
foreach (ExchangeFolderInfo folderInfo in folderInfoCol)
{
    switch (folderInfo.FolderType)
    {
        case ExchangeFolderType.Appointment:
            // Χειρισμός φακέλου τύπου ραντεβού
            break;
        case ExchangeFolderType.Contact:
            // Χειρισμός φακέλου τύπου επαφής
            break;
        case ExchangeFolderType.Task:
            // Χειρισμός φακέλου τύπου εργασίας
            break;
        case ExchangeFolderType.Note:
            // Χειρισμός φακέλου μηνυμάτων ηλεκτρονικού ταχυδρομείου
            break;
        case ExchangeFolderType.StickyNote:
            // Χειρισμός φακέλου τύπου StickyNote
            break;
        case ExchangeFolderType.Journal:
            // Χειρισμός φακέλου τύπου ημερολογίου
            break;
        default:
            // Χειριστείτε άλλους τύπους εάν είναι απαραίτητο
            break;
    }
}
```
*Παράμετροι:* `folderInfo.FolderType` καθορίζει τη λογική για την επεξεργασία κάθε φακέλου.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι τα διαπιστευτήριά σας είναι σωστά για να αποφύγετε σφάλματα ελέγχου ταυτότητας.
- Επαληθεύστε τη διεύθυνση URL και τη διαθεσιμότητα του διακομιστή κατά τη σύνδεση μέσω EWS.
