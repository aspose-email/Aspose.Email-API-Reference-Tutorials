## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει σε κάθε βήμα που απαιτείται για την ανάκτηση επαφών από έναν διακομιστή Exchange.

### Βήμα 1: Δημιουργήστε μια παρουσία του IEWSClient

**Επισκόπηση**Δημιουργήστε μια ασφαλή σύνδεση με τον διακομιστή Exchange χρησιμοποιώντας το πρόγραμμα-πελάτη EWS του Aspose.Email.

```csharp
IEWSClient client = EWSClient.GetEWSClient(
    "https://outlook.office365.com/ews/exchange.asmx", 
    "testUser",    
    "pwd",         
    "domain"       
);
```

**Εξήγηση**: Αντικατάσταση `"testUser"`, `"pwd"`, και `"domain"` με τα πραγματικά διαπιστευτήρια του διακομιστή Exchange. Αυτό το βήμα ελέγχει τον έλεγχο ταυτότητας και αρχικοποιεί τη σύνδεση.

### Βήμα 2: Λίστα όλων των επαφών

**Επισκόπηση**: Ανάκτηση επαφών από το γραμματοκιβώτιό σας χρησιμοποιώντας τον Ενσωματωμένο Ιστότοπο (EWS).

```csharp
Contact[] contacts = client.GetContacts(client.MailboxInfo.ContactsUri);
```

**Εξήγηση**: `client.GetContacts` ανακτά όλες τις διαθέσιμες επαφές. Η μέθοδος επιστρέφει έναν πίνακα αντικειμένων επαφών, τα οποία στη συνέχεια μπορούν να υποβληθούν σε επεξεργασία.

### Βήμα 3: Εμφάνιση στοιχείων επικοινωνίας

**Επισκόπηση**: Επαναλάβετε την επεξεργασία των ανακτημένων επαφών για να εμφανίσετε τα ονόματα και τις διευθύνσεις email τους.

```csharp
foreach (MapiContact contact in contacts)
{
    Console.WriteLine(
        "Name: " + contact.NameInfo.DisplayName + ", Email Address: " +
        contact.ElectronicAddresses.Email1
    );
}
```

**Εξήγηση**: Επαναλάβετε κάθε `MapiContact` για πρόσβαση και εμφάνιση σημαντικών πληροφοριών όπως το όνομα και η διεύθυνση ηλεκτρονικού ταχυδρομείου.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι η διεύθυνση URL του διακομιστή, το όνομα χρήστη, ο κωδικός πρόσβασης και ο τομέας έχουν ρυθμιστεί σωστά.
- Ελέγξτε τη συνδεσιμότητα δικτύου εάν δεν είναι δυνατή η σύνδεση με τον διακομιστή Exchange.
- Επαληθεύστε ότι η έκδοση API σας είναι συμβατή με την έκδοση του .NET framework που διαθέτετε.
