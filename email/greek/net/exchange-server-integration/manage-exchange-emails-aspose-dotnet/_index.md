## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στη σύνδεση με έναν διακομιστή Exchange, στην καταχώριση μηνυμάτων στα εισερχόμενά σας και στην αποθήκευσή τους ως αρχεία MSG.

### Σύνδεση με Exchange Server χρησιμοποιώντας EWS

Η σύνδεση με τον διακομιστή Exchange είναι το πρώτο βήμα. Δείτε πώς μπορείτε να δημιουργήσετε μια σύνδεση χρησιμοποιώντας το Aspose.Email για .NET:

#### Αρχικοποίηση παραμέτρων σύνδεσης
```csharp
string ewsUrl = "https://outlook.office365.com/ews/exchange.asmx";
string username = "testUser";
string password = "pwd";
string domain = "domain";
```

#### Δημιουργία στιγμιότυπου EWSClient
Δημιουργήστε μια παρουσία του `EWSClient` τάξη παρέχοντας τα διαπιστευτήριά σας:
```csharp
IEWSClient client = EWSClient.GetEWSClient(ewsUrl, username, password, domain);
```
Ο `client` Το αντικείμενο είναι πλέον έτοιμο για αλληλεπίδραση με τον διακομιστή Exchange.

### Καταχώριση μηνυμάτων στα Εισερχόμενα χρησιμοποιώντας τον EWS

Μόλις συνδεθείτε, μπορείτε να εμφανίσετε μια λίστα με τα μηνύματα από τα εισερχόμενά σας. Δείτε πώς:

#### Ανάκτηση μηνυμάτων
Χρησιμοποιήστε το `ListMessages` μέθοδος για να λάβετε πληροφορίες σχετικά με μηνύματα στον φάκελο Εισερχόμενα:
```csharp
ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);
```

#### Επανάληψη μέσω μηνυμάτων
Επαναλάβετε κάθε μήνυμα για να το επεξεργαστείτε όπως απαιτείται:
```csharp
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    string strMessageURI = msgInfo.UniqueUri;
}
```
Αυτό το τμήμα κώδικα ανακτά το μοναδικό URI κάθε μηνύματος, το οποίο μπορεί να χρησιμοποιηθεί για περαιτέρω επεξεργασία.

### Ανάκτηση και αποθήκευση μηνυμάτων σε μορφή MSG

Ενδέχεται να χρειαστεί να αποθηκεύσετε μηνύματα από τον διακομιστή Exchange τοπικά. Δείτε πώς μπορείτε να ανακτήσετε μεμονωμένα μηνύματα ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας τα URI τους και να τα αποθηκεύσετε ως αρχεία MSG:

#### Αποθήκευση μηνυμάτων τοπικά
Επαναλάβετε μέσω του `msgCollection` που λάβατε νωρίτερα, ανακτήστε κάθε μήνυμα και αποθηκεύστε το:
```csharp
string outputDirectory = "/path/to/output/directory";
int count = 0;

foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    string strMessageURI = msgInfo.UniqueUri;
    MailMessage message = client.FetchMessage(strMessageURI);
    message.Save(outputDirectory + (count++) + "_out.msg", SaveOptions.DefaultMsgUnicode);
}
```
Αυτός ο κώδικας ανακτά κάθε email και το αποθηκεύει ως αρχείο MSG, χρησιμοποιώντας τον καθορισμένο κατάλογο.
