## Οδηγός Εφαρμογής

### Καταχώριση δημόσιων φακέλων

Ανάκτηση όλων των δημόσιων φακέλων στο γραμματοκιβώτιό σας του Exchange χρησιμοποιώντας `IEWSClient`:

#### Επισκόπηση
Αυτοματοποιήστε εργασίες και διαχειριστείτε αποτελεσματικά τα δεδομένα email, καταχωρώντας τους διαθέσιμους δημόσιους φακέλους.

#### Βήματα Υλοποίησης
##### Βήμα 1: Δημιουργία στιγμιότυπου προγράμματος-πελάτη EWS
Δημιουργήστε ένα υπόδειγμα `IEWSClient` αντικείμενο με έγκυρα διαπιστευτήρια:

```csharp
// Αντικαταστήστε με τα πραγματικά σας διαπιστευτήρια
string url = "https://outlook.office365.com/ews/exchange.asmx";
string username = "your-email@example.com";
string password = "your-password";

IEWSClient client = EWSClient.GetEWSClient(url, username, password);
```

##### Βήμα 2: Ανάκτηση δημόσιων φακέλων
Ανάκτηση όλων των δημόσιων φακέλων χρησιμοποιώντας το `ListPublicFolders` μέθοδος:

```csharp
// Ανάκτηση και επανάληψη σε κάθε δημόσιο φάκελο
ExchangeFolderInfoCollection publicFolders = client.ListPublicFolders(client.MailboxInfo.RootUri);

foreach (ExchangeFolderInfo folder in publicFolders)
{
    Console.WriteLine($"Folder: {folder.DisplayName}");
}
```

##### Επεξήγηση των αποσπασμάτων κώδικα
- **`IEWSClient.GetEWSClient`**: Δημιουργεί μια σύνδεση με τον διακομιστή Exchange.
  - *Παράμετροι*: URL, όνομα χρήστη, κωδικός πρόσβασης.
  - *Σκοπός*: Έλεγχος ταυτότητας και αρχικοποίηση πρόσβασης EWS.

- **`ListPublicFolders`**:
  - *Επιστροφές*Συλλογή δημόσιων φακέλων (`ExchangeFolderInfoCollection`).
  - *Χρήση*: Επαναλάβετε την περιήγηση σε κάθε φάκελο για ενέργειες ή ανάκτηση δεδομένων.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι τα διαπιστευτήρια είναι σωστά.
- Επαληθεύστε τη συνδεσιμότητα δικτύου με τη διεύθυνση URL του διακομιστή Exchange.
- Ελέγξτε τις ρυθμίσεις του τείχους προστασίας που ενδέχεται να μπλοκάρουν τα τελικά σημεία του EWS.
