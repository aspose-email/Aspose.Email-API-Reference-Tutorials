## Οδηγός Εφαρμογής
### Δημιουργία και προσθήκη μηνυμάτων στον Exchange Server
#### Επισκόπηση
Η αυτοματοποίηση της δημιουργίας email και η προσάρτησή τους στον Exchange server σας βελτιστοποιεί την επικοινωνία. Αυτή η ενότητα δείχνει πώς να χρησιμοποιήσετε το Aspose.Email για .NET για να το πετύχετε αυτό αποτελεσματικά.

#### Βήμα προς βήμα εφαρμογή:
**1. Συνδεθείτε στον Exchange Server:**
```csharp
var client = EWSClient.GetEWSClient("exchange.domain.com", "username", "password");
```
Αυτό το βήμα δημιουργεί μια σύνδεση με τον διακομιστή Exchange χρησιμοποιώντας τα παρεχόμενα διαπιστευτήρια.

**2. Δημιουργία και προσθήκη μηνυμάτων ηλεκτρονικού ταχυδρομείου:**
Δείτε πώς μπορείτε να δημιουργήσετε πολλά μηνύματα και να τα προσθέσετε στα εισερχόμενά σας:
```csharp
int messagesNum = 12;
for (int i = 0; i < messagesNum; i++) {
    var message = new MailMessage(
        "from@domain.com",
        "to@domain.com",
        $"EMAILNET-35157_1 - {Guid.NewGuid()}",
        "EMAILNET-35157 Move paging parameters to separate class");
    client.AppendMessage(client.MailboxInfo.InboxUri, message);
}
```
**Εξήγηση:**
- **Μήνυμα αλληλογραφίας:** Δημιουργήστε ένα νέο μήνυμα ηλεκτρονικού ταχυδρομείου με αποστολέα, παραλήπτη, θέμα και σώμα.
- **Προσθήκη μηνύματος:** Προσθέτει το μήνυμα που δημιουργήθηκε στα εισερχόμενά σας στον διακομιστή Exchange.

### Επαλήθευση μηνυμάτων στον Exchange Server
#### Επισκόπηση
Αφού προσθέσετε μηνύματα, είναι σημαντικό να επαληθεύσετε ότι έχουν αποθηκευτεί σωστά. Αυτή η ενότητα σας καθοδηγεί στην επαλήθευση του αριθμού των email σε ένα φάκελο εισερχομένων.

**1. Λίστα όλων των μηνυμάτων:**
```csharp
var client = EWSClient.GetEWSClient("exchange.domain.com", "username", "password");
var totalMessageInfoCol = client.ListMessages(client.MailboxInfo.InboxUri);
int messageCount = totalMessageInfoCol.Count;
```
**Εξήγηση:**
- **Λίστα Μηνυμάτων:** Ανακτά όλα τα μηνύματα από τα εισερχόμενα.
- **Κόμης:** Παρέχει τον συνολικό αριθμό μηνυμάτων, επιτρέποντάς σας να επαληθεύσετε την επιτυχή προσθήκη.

### Ανάκτηση μηνυμάτων χρησιμοποιώντας την υποστήριξη σελιδοποίησης
#### Επισκόπηση
Η αποτελεσματική ανάκτηση μηνυμάτων ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας τη σελιδοποίηση βοηθά στη διαχείριση μεγάλων συνόλων δεδομένων. Αυτή η ενότητα δείχνει πώς να εφαρμόσετε τη σελιδοποίηση κατά την ανάκτηση μηνυμάτων ηλεκτρονικού ταχυδρομείου από έναν διακομιστή Exchange.

**1. Ρύθμιση παραμέτρων σελιδοποίησης:**
```csharp
var client = EWSClient.GetEWSClient("exchange.domain.com", "username", "password");
int itemsPerPage = 5;
List<ExchangeMessagePageInfo> pages = new List<ExchangeMessagePageInfo>();
```
**2. Ανάκτηση μηνυμάτων σε σελίδες:**
```csharp
ExchangeMessagePageInfo pageInfo = client.ListMessagesByPage(client.MailboxInfo.InboxUri, itemsPerPage);
pages.Add(pageInfo);

while (!pageInfo.LastPage) {
    pageInfo = client.ListMessagesByPage(client.MailboxInfo.InboxUri, itemsPerPage, pageInfo.PageOffset + 1);
    pages.Add(pageInfo);
}

int retrievedItems = 0;
foreach (ExchangeMessagePageInfo pageCol in pages) {
    retrievedItems += pageCol.Items.Count;
}
```
**Εξήγηση:**
- **ΛίσταΜηνυμάτωνΑνάΣελίδα:** Ανακτά μηνύματα σε καθορισμένα μεγέθη σελίδας.
- **Τελευταία Σελίδα & Μετατόπιση Σελίδας:** Διαχειριστείτε τη σελιδοποίηση για την αποτελεσματική διαχείριση μεγάλου όγκου email.
