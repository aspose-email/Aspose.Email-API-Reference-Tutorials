## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίησή μας σε δύο κύρια χαρακτηριστικά: την αρχικοποίηση του προγράμματος-πελάτη Exchange και την ανάλυση των επαφών με βάση το όνομα.

### Δυνατότητα 1: Αρχικοποίηση προγράμματος-πελάτη Exchange

Αυτή η λειτουργία εστιάζει στη δημιουργία μιας παρουσίας του `IEWSClient` κλάση χρησιμοποιώντας τα διαπιστευτήριά σας, τα οποία είναι κρίσιμα για την ασφαλή σύνδεση στον διακομιστή Exchange.

#### Βήμα προς βήμα εφαρμογή

**Αρχικοποίηση της παρουσίας του IEWSClient**

```csharp
public static void InitializeExchangeClient()
{
    // Δημιουργήστε μια παρουσία του IEWSClient με καθορισμένα διαπιστευτήρια και διεύθυνση URL διακομιστή
    IEWSClient client = EWSClient.GetEWSClient(
        "https://outlook.office365.com/ews/exchange.asmx", 
        "testUser",   // Όνομα χρήστη
        "pwd",        // Σύνθημα
        "domain"      // Πεδίο ορισμού
    );
}
```
- **Επεξήγηση παραμέτρων:**
  - `"https://outlook.office365.com/ews/exchange.asmx"`Η διεύθυνση URL διακομιστή για τις υπηρεσίες Web του Exchange.
  - `"testUser"`: Το όνομα χρήστη σας στο Exchange.
  - `"pwd"`: Ο κωδικός πρόσβασής σας.
  - `"domain"`: Ο τομέας που σχετίζεται με τον λογαριασμό.

### Λειτουργία 2: Επίλυση επαφών με βάση το όνομα

Εξερευνήστε πώς να επιλύσετε και να καταχωρίσετε σε λίστα επαφές από έναν διακομιστή Exchange χρησιμοποιώντας ένα όνομα επαφής, το οποίο είναι χρήσιμο για τον γρήγορο εντοπισμό συγκεκριμένων ατόμων.

#### Βήμα προς βήμα εφαρμογή

**Επίλυση και καταχώριση επαφών**

```csharp
public static void ResolveContactsUsingContactName()
{
    try
    {
        // Αρχικοποίηση της παρουσίας IEWSClient
        IEWSClient client = EWSClient.GetEWSClient(
            "https://outlook.office365.com/ews/exchange.asmx", 
            "testUser",   // Όνομα χρήστη
            "pwd",        // Σύνθημα
            "domain"      // Πεδίο ορισμού
        );

        // Επίλυση επαφών χρησιμοποιώντας ένα συγκεκριμένο όνομα και ανάκτηση των φωτογραφιών τους
        Contact[] contacts = client.ResolveContacts(
            "Changed Name",  // Όνομα επαφής για αναζήτηση
            ExchangeListContactsOptions.FetchPhoto // Δυνατότητα λήψης και φωτογραφιών επαφών
        );

        // Επανάληψη στις επιλυμένες επαφές
        foreach (MapiContact contact in contacts)
        {
            // Εξαγωγή εμφανιζόμενου ονόματος και διεύθυνσης ηλεκτρονικού ταχυδρομείου επαφής
            Console.WriteLine("Name: " + contact.NameInfo.DisplayName + ", Email Address: " + contact.ElectronicAddresses.Email1);
        }
    }
    catch (Exception ex)
    {
        // Χειρισμός εξαιρέσεων εμφανίζοντας το μήνυμα σφάλματος
        Console.WriteLine(ex.Message);
    }
}
```
- **Επεξήγηση παραμέτρων:**
  - `"Changed Name"`: Το όνομα της επαφής που θέλετε να επιλύσετε.
  - `ExchangeListContactsOptions.FetchPhoto`: Μια επιλογή για συμπερίληψη φωτογραφιών στα αποτελέσματα.
