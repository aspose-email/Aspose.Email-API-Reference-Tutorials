## Οδηγός Εφαρμογής

### Δυνατότητα 1: Σύνδεση με Exchange Server

Η σύνδεση σε έναν διακομιστή Exchange απαιτεί διαπιστευτήρια ελέγχου ταυτότητας και το URI του διακομιστή.

#### Επισκόπηση:
Θα δημιουργήσουμε μια σύνδεση χρησιμοποιώντας το NetworkCredential για ασφαλή έλεγχο ταυτότητας και, στη συνέχεια, θα αρχικοποιήσουμε μια `ExchangeClient`.

#### Βήματα:

**Βήμα 1:** Εισαγάγετε τους απαραίτητους χώρους ονομάτων και ρυθμίστε τις παραμέτρους σύνδεσής σας.

```csharp
using System.Net;
using Aspose.Email.Clients.Exchange;

string mailboxURI = "https://Ex2003/exchange/administrator"; // URI διακομιστή Exchange
string username = "administrator"; // Όνομα χρήστη
string password = "pwd";           // Σύνθημα
domain = "domain.local";    // Πεδίο ορισμού

// Δημιουργήστε ένα αντικείμενο NetworkCredential με τα παρεχόμενα διαπιστευτήρια
NetworkCredential credential = new NetworkCredential(username, password, domain);
```

**Βήμα 2:** Αρχικοποίηση `ExchangeClient` και να ανακτήσετε πληροφορίες γραμματοκιβωτίου.

```csharp
// Αρχικοποίηση ExchangeClient με το URI γραμματοκιβωτίου και τα διαπιστευτήρια
ExchangeClient client = new ExchangeClient(mailboxURI, credential);

// Ανάκτηση και αποθήκευση πληροφοριών γραμματοκιβωτίου
ExchangeMailboxInfo mailboxInfo = client.GetMailboxInfo();
```

### Λειτουργία 2: Λίστα μηνυμάτων από τα Εισερχόμενα

Τώρα που είμαστε συνδεδεμένοι, ας εμφανίσουμε όλα τα μηνύματα στα εισερχόμενά σας.

#### Επισκόπηση:
Ανακτήστε μια συλλογή μηνυμάτων και φιλτράρετε τα με βάση συγκεκριμένα κριτήρια.

#### Βήματα:

**Βήμα 1:** Ανάκτηση των μηνυμάτων στον φάκελο εισερχομένων.

```csharp
// Ανάκτηση μιας συλλογής μηνυμάτων στον φάκελο Εισερχόμενα χρησιμοποιώντας το ExchangeClient
ExchangeMessageInfoCollection msgInfoColl = client.ListMessages(mailboxInfo.InboxUri);
```

**Βήμα 2:** Φιλτράρετε και επεξεργαστείτε συγκεκριμένα μηνύματα.

```csharp
foreach (ExchangeMessageInfo msgInfo in msgInfoColl)
{
    // Ελέγξτε εάν το θέμα του μηνύματος περιέχει την ένδειξη "επεξεργασία αυτού του μηνύματος"
    if (msgInfo.Subject != null && msgInfo.Subject.ToLower().Contains("process this message"))
    {
        // Μετακίνηση του μηνύματος στον φάκελο «Επεξεργασμένα»
        string processedFolderUri = client.MailboxInfo.RootUri + "/Processed/" + msgInfo.Subject;
        client.MoveItems(msgInfo.UniqueUri, processedFolderUri);
    }
}
```

### Λειτουργία 3: Μετακίνηση μηνύματος στον φάκελο "Επεξεργασμένο"

#### Επισκόπηση:
Αυτή η λειτουργία δείχνει πώς να μετακινήσετε ένα μήνυμα από έναν φάκελο σε έναν άλλο με βάση κριτήρια.

#### Βήματα:

**Βήμα 1:** Κατασκευάστε το URI προορισμού και χρησιμοποιήστε το `MoveItems` μέθοδος για τη μετακίνηση συγκεκριμένων μηνυμάτων.

```csharp
// Κατασκευάστε το URI του επεξεργασμένου φακέλου με το θέμα ως μέρος της διαδρομής του
string processedFolderUri = client.MailboxInfo.RootUri + "/Processed/" + msgInfo.Subject;

// Μετακίνηση του καθορισμένου μηνύματος
client.MoveItems(msgInfo.UniqueUri, processedFolderUri);
```
