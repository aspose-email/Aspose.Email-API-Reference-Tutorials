## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στη ρύθμιση ενός προγράμματος-πελάτη EWS και στην ενημέρωση των επαφών.

### Ρύθμιση προγράμματος-πελάτη EWS

**Επισκόπηση:** Δημιουργήστε μια σύνδεση με τον διακομιστή Microsoft Exchange χρησιμοποιώντας το Aspose.Email για .NET. Αυτό το βήμα είναι κρίσιμο, καθώς επιτρέπει στην εφαρμογή σας να επικοινωνεί με τον διακομιστή Exchange, επιτρέποντας περαιτέρω λειτουργίες όπως η καταχώριση ή η ενημέρωση επαφών.

#### Βήμα 1: Ορισμός διαπιστευτηρίων διακομιστή

```csharp
string mailboxUri = "https://ex2010/ews/exchange.asmx";
string username = "test.exchange";
string password = "pwd";
string domain = "ex2010.local";

NetworkCredential credentials = new NetworkCredential(username, password, domain);
```

**Γιατί αυτό το βήμα;** 
Αυτά τα διαπιστευτήρια ελέγχουν την ταυτότητα του πελάτη σας με τον διακομιστή Exchange, διασφαλίζοντας ότι μόνο εξουσιοδοτημένες εφαρμογές μπορούν να έχουν πρόσβαση και να τροποποιούν τα δεδομένα επαφών.

#### Βήμα 2: Λήψη στιγμιότυπου IEWSClient

```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
```

**Τι κάνει:** 
Αυτή η γραμμή αρχικοποιεί ένα `IEWSClient` παράδειγμα χρησιμοποιώντας το παρεχόμενο URI γραμματοκιβωτίου και τα διαπιστευτήρια. Αυτός ο υπολογιστής-πελάτης θα είναι η πύλη σας για την εκτέλεση λειτουργιών στον διακομιστή Exchange.

### Καταχώριση και Ενημέρωση Επαφών

**Επισκόπηση:** Μόλις συνδεθείτε, μπορείτε να εμφανίσετε μια λίστα με όλες τις επαφές που είναι αποθηκευμένες στον διακομιστή Exchange και να ενημερώσετε τις πληροφορίες τους όπως απαιτείται.

#### Βήμα 1: Λίστα όλων των επαφών

```csharp
Contact[] contacts = client.GetContacts(client.MailboxInfo.ContactsUri);
```

**Πώς λειτουργεί:** 
Αυτή η μέθοδος ανακτά έναν πίνακα από `Contact` αντικείμενα από το γραμματοκιβώτιό σας. Στη συνέχεια, μπορείτε να τα χρησιμοποιήσετε επανειλημμένα για να αποκτήσετε πρόσβαση ή να τροποποιήσετε τα στοιχεία επικοινωνίας.

#### Βήμα 2: Ενημέρωση στοιχείων επαφής

```csharp
// Πρόσβαση στην πρώτη επαφή (βεβαιωθείτε ότι υπάρχει τουλάχιστον μία)
Contact contactToUpdate = contacts[0];

// Τροποποίηση του εμφανιζόμενου ονόματος
contactToUpdate.DisplayName = "David Ch";

// Αποθήκευση αλλαγών πίσω στον διακομιστή
client.UpdateContact(contactToUpdate);
```

**Βασικά σημεία:**
- **Πρόσβαση στις Επαφές:** Μπορείτε να επιλέξετε οποιαδήποτε επαφή από τον πίνακα για ενημερώσεις.
- **Τροποποίηση δεδομένων:** Ενημερώστε τα απαραίτητα πεδία, όπως `DisplayName`.
- **Διατήρηση Αλλαγών:** Χρήση `UpdateContact` για να αποθηκεύσετε τις τροποποιήσεις σας στον διακομιστή Exchange.
