## Οδηγός Εφαρμογής

### Σύνδεση σε διακομιστή IMAP

#### Επισκόπηση
Αυτή η ενότητα καλύπτει τη ρύθμιση μιας σύνδεσης με τον διακομιστή IMAP χρησιμοποιώντας `ImapClient`.

#### Οδηγός βήμα προς βήμα

**1. Ρύθμιση παραμέτρων του ImapClient**
Δημιουργήστε και διαμορφώστε το `ImapClient` παράδειγμα με τα στοιχεία του διακομιστή σας:

```csharp
using Aspose.Email.Clients;

ImapClient client = new ImapClient();
client.Host = "imap.gmail.com"; // Καθορίστε τον κεντρικό υπολογιστή του διακομιστή IMAP
client.Username = "your.username@gmail.com"; // Ορίστε το όνομα χρήστη του email σας
client.Password = "your.password"; // Ορίστε τον κωδικό πρόσβασης email σας
client.Port = 993; // Τυπική θύρα για συνδέσεις SSL
client.SecurityOptions = SecurityOptions.Auto; // Αυτόματη επιλογή επιλογών ασφαλείας
```

**Εξήγηση:**
- `Host` Καθορίζει τη διεύθυνση του διακομιστή IMAP.
- `Username` και `Password` απαιτούνται για την επαλήθευση ταυτότητας.
- Λιμάνι `993` Χρησιμοποιείται για ασφαλείς συνδέσεις (SSL/TLS).
- `SecurityOptions.Auto` εξασφαλίζει βέλτιστες ρυθμίσεις ασφαλείας.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι το δίκτυό σας επιτρέπει συνδέσεις στη θύρα 993.
- Επαληθεύστε ότι τα διαπιστευτήρια του email σας είναι σωστά.

### Δημιουργία και προσθήκη νέου μηνύματος σε φάκελο IMAP

#### Επισκόπηση
Μάθετε πώς να δημιουργείτε ένα νέο μήνυμα ηλεκτρονικού ταχυδρομείου και να το προσθέτετε στον φάκελο εισερχομένων.

#### Οδηγός βήμα προς βήμα

**1. Δημιουργήστε το MailMessage**
Δημιουργήστε μια νέα παρουσία του `MailMessage`:

```csharp
using Aspose.Email.Clients.Imap;
using Aspose.Email.Mime;

// Δημιουργήστε μια παρουσία MailMessage για το νέο μήνυμα ηλεκτρονικού ταχυδρομείου
MailMessage msg = new MailMessage("user@domain1.com", "user@domain2.com", "subject", "message");
```

**Εξήγηση:**
- `MailMessage` αντιπροσωπεύει ένα μήνυμα ηλεκτρονικού ταχυδρομείου με λεπτομέρειες αποστολέα, παραλήπτη, θέμα και σώμα.

**2. Επιλέξτε Φάκελο**
Επιλέξτε τον φάκελο εισερχομένων:

```csharp
// Επιλέξτε τον φάκελο Εισερχόμενα στον διακομιστή IMAP
client.SelectFolder(ImapFolderInfo.InBox);
```

**3. Προσθήκη μηνύματος**
Προσθήκη του μηνύματος στον τρέχοντα φάκελο:

```csharp
try
{
    // Εγγραφή στις αλλαγές στον τρέχοντα φάκελο (προαιρετικό)
    client.SubscribeFolder(client.CurrentFolder.Name);

    // Προσθήκη του νεοδημιουργημένου μηνύματος στον επιλεγμένο φάκελο
    client.AppendMessage(client.CurrentFolder.Name, msg);
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```

**Εξήγηση:**
- `SelectFolder` ορίζει τον ενεργό φάκελο.
- `AppendMessage` προσθέτει το email σας στον επιλεγμένο φάκελο.
