## Οδηγός Εφαρμογής

### Δυνατότητα 1: Σύνδεση με Exchange Server
#### Επισκόπηση
Η σύνδεση σε έναν διακομιστή Exchange είναι απαραίτητη για την πρόσβαση στις λειτουργίες του email. Αυτή η λειτουργία δείχνει πώς να ρυθμίσετε μια ασφαλή σύνδεση χρησιμοποιώντας το Aspose.Email για .NET.

**Βήμα 1: Ρύθμιση παραμέτρων διαπιστευτηρίων**
Δημιουργώ `NetworkCredential` με τη διεύθυνση email, τον κωδικό πρόσβασης και τον τομέα σας (εάν υπάρχει).

```csharp
using System.Net;
using Aspose.Email.Clients.Exchange.WebService;

const string mailboxUri = "<HOST>"; // Αντικαταστήστε με το URI του διακομιστή Exchange
const string username = "<EMAIL ADDRESS>";
const string password = "<PASSWORD>";
NetworkCredential credentials = new NetworkCredential(username, password, "");
```

**Βήμα 2: Δημιουργία σύνδεσης**
Χρήση `EWSClient.GetEWSClient` για να δημιουργήσετε μια σύνδεση με τον διακομιστή Exchange.

```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
client.Dispose();
// Αυτός ο κώδικας συνδέεται με έναν διακομιστή Exchange χρησιμοποιώντας τα παρεχόμενα διαπιστευτήρια.
```

### Λειτουργία 2: Λίστα στοιχείων από τα εισερχόμενα κοινόχρηστου γραμματοκιβωτίου
#### Επισκόπηση
Η πρόσβαση σε email σε ένα κοινόχρηστο γραμματοκιβώτιο είναι ζωτικής σημασίας για τα συνεργατικά περιβάλλοντα. Δείτε πώς μπορείτε να παραθέσετε στοιχεία στα εισερχόμενα ενός κοινόχρηστου γραμματοκιβωτίου.

**Βήμα 1: Αρχικοποίηση προγράμματος-πελάτη**
Βεβαιωθείτε ότι το δικό σας `IEWSClient` Η παρουσία είναι συνδεδεμένη στον διακομιστή με τα κατάλληλα διαπιστευτήρια.

```csharp
const string sharedEmail = "<SHARED EMAIL ADDRESS>";
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
```

**Βήμα 2: Λίστα στοιχείων εισερχομένων**
Ανάκτηση λίστας στοιχείων στον φάκελο εισερχομένων χρησιμοποιώντας `ListItems`.

```csharp
string[] items = client.ListItems(sharedEmail, "Inbox");
client.Dispose();
// Αυτός ο κώδικας παραθέτει όλα τα στοιχεία στον φάκελο "Εισερχόμενα" ενός συγκεκριμένου κοινόχρηστου γραμματοκιβωτίου.
```

### Λειτουργία 3: Ανάκτηση και εμφάνιση θέματος email από κοινόχρηστο γραμματοκιβώτιο
#### Επισκόπηση
Η ανάκτηση των θεμάτων των email βοηθά στην γρήγορη αναγνώριση σημαντικών μηνυμάτων. Δείτε πώς μπορείτε να τα ανακτήσετε και να τα εμφανίσετε.

**Βήμα 1: Επανάληψη στοιχείων**
Κάντε επανάληψη σε κάθε στοιχείο που ανακτάται από τα εισερχόμενα του κοινόχρηστου γραμματοκιβωτίου.

```csharp
foreach (string item in items)
{
    MapiMessage msg = client.FetchItem(item);
    Console.WriteLine("Subject:" + msg.Subject);
}
client.Dispose();
// Αυτός ο κώδικας ανακτά κάθε email από τα εισερχόμενα του κοινόχρηστου γραμματοκιβωτίου και εκτυπώνει το θέμα του.
```
