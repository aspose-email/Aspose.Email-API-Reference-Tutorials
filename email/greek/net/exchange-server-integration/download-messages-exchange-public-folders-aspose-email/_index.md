Οι επαγγελματίες πληροφορικής συχνά αντιμετωπίζουν την πρόκληση της προγραμματιστικής πρόσβασης και λήψης μηνυμάτων από δημόσιους φακέλους στο Exchange. Ακολουθώντας αυτόν τον οδηγό, θα μάθετε πώς να:
- Επαλήθευση ταυτότητας και σύνδεση σε διακομιστή Exchange χρησιμοποιώντας EWS (Exchange Web Services)
- Λίστα όλων των δημόσιων φακέλων και των υποφακέλων τους
- Λήψη μηνυμάτων από αυτούς τους φακέλους στο τοπικό σας σύστημα

## Οδηγός Εφαρμογής
Αυτή η ενότητα θα σας καθοδηγήσει στη λήψη μηνυμάτων από τους δημόσιους φακέλους του Exchange χρησιμοποιώντας C#.

### Έλεγχος ταυτότητας και σύνδεση
#### Επισκόπηση
Αρχικά, πραγματοποιήστε έλεγχο ταυτότητας με τον διακομιστή Exchange για να αποκτήσετε πρόσβαση σε δημόσιους φακέλους.

##### Βήμα 1: Έλεγχος ταυτότητας χρησιμοποιώντας διαπιστευτήρια δικτύου
Ξεκινήστε δημιουργώντας ένα `NetworkCredential` αντικείμενο:
```csharp
NetworkCredential credential = new NetworkCredential("administrator", "pwd", "ex2013.local");
```
- **Παράμετροι**: Απαιτούνται όνομα χρήστη, κωδικός πρόσβασης και τομέας για τον έλεγχο ταυτότητας.

##### Βήμα 2: Λήψη μιας παρουσίας του προγράμματος-πελάτη EWS
Χρησιμοποιήστε τα διαπιστευτήριά σας για να συνδεθείτε στον διακομιστή Exchange:
```csharp
IEWSClient client = EWSClient.GetEWSClient("https://exchange/ews/exchange.asmx", πιστοποιητικό);
```
- **MailboxUri**: Αυτό είναι το τελικό σημείο URL για την υπηρεσία Exchange Web Service.

### Καταχώριση και λήψη μηνυμάτων
#### Επισκόπηση
Στη συνέχεια, δημιουργήστε μια λίστα με τους δημόσιους φακέλους και κατεβάστε μηνύματα από κάθε φάκελο.

##### Βήμα 3: Λίστα όλων των δημόσιων φακέλων
Ανάκτηση όλων των διαθέσιμων δημόσιων φακέλων:
```csharp
ExchangeFolderInfoCollection folders = client.ListPublicFolders();
```
Επαναλάβετε την περιήγησή σας σε αυτούς τους φακέλους για να αποκτήσετε πρόσβαση στο περιεχόμενό τους:
```csharp
foreach (ExchangeFolderInfo publicFolder in folders)
{
    Console.WriteLine("Name: " + publicFolder.DisplayName);
    Console.WriteLine("Subfolders count: " + publicFolder.ChildFolderCount);
    ListMessagesFromSubFolder(publicFolder, client);
}
```
##### Βήμα 4: Λήψη μηνυμάτων από κάθε φάκελο
Για κάθε φάκελο, ανακτήστε και αποθηκεύστε τα μηνύματα:
```csharp
private static void ListMessagesFromSubFolder(ExchangeFolderInfo publicFolder, IEWSClient client)
{
    Console.WriteLine("Folder Name: " + publicFolder.DisplayName);

    ExchangeMessageInfoCollection msgInfoCollection = client.ListMessagesFromPublicFolder(publicFolder);
    foreach (ExchangeMessageInfo messageInfo in msgInfoCollection)
    {
        MailMessage msg = client.FetchMessage(messageInfo.UniqueUri);
        
        // Αποθήκευση κάθε μηνύματος σε ένα αρχείο
        Console.WriteLine(msg.Subject);
        msg.Save("YOUR_DOCUMENT_DIRECTORY/" + msg.Subject + ".msg", SaveOptions.DefaultMsgUnicode);
    }
}
```
### Επεξεργασία Υποφακέλων με Αναδρομική Επεξεργασία
#### Επισκόπηση
Χειρισμός υποφακέλων αναδρομικά:
##### Βήμα 6: Αναδρομική λίστα μηνυμάτων από υποφακέλους
Εάν ένας φάκελος περιέχει υποφακέλους, επεξεργαστείτε τον καθένα από αυτούς:
```csharp
if (publicFolder.ChildFolderCount > 0)
{
    ExchangeFolderInfoCollection subfolders = client.ListSubFolders(publicFolder);
    foreach (ExchangeFolderInfo subfolder in subfolders)
    {
        ListMessagesFromSubFolder(subfolder, client);
    }
}
```
