## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία προγράμματος-πελάτη EWS και ανάκτηση πληροφοριών γραμματοκιβωτίου

#### Επισκόπηση
Δημιουργία μιας παρουσίας του `IEWSClient` σας επιτρέπει να συνδεθείτε στον διακομιστή Exchange χρησιμοποιώντας διαπιστευτήρια, τα οποία είναι κρίσιμα για λειτουργίες όπως η ανάκτηση πληροφοριών γραμματοκιβωτίου.

**Βήμα 1**: Ορίστε τα στοιχεία και τα διαπιστευτήρια του διακομιστή σας.
```csharp
string mailboxUri = "https://ews.domain.com/ews/Exchange.asmx";
string domain = @"";
string username = "username";
string password = "password";

NetworkCredential credential = new NetworkCredential(username, password, domain);
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credential);
```

**Βήμα 2**: Ανάκτηση πληροφοριών γραμματοκιβωτίου.
```csharp
ExchangeMailboxInfo mailboxInfo = client.GetMailboxInfo();
Console.WriteLine("Mailbox info retrieved successfully.");
```

Εδώ, `GetMailboxInfo()` Επιστρέφει ένα αντικείμενο που περιέχει λεπτομέρειες όπως η διεύθυνση ηλεκτρονικού ταχυδρομείου και τα όρια αποθήκευσης, απαραίτητα για την κατανόηση της τρέχουσας κατάστασης του γραμματοκιβωτίου σας.

### Δυνατότητα 2: Δημιουργία αντιγράφων ασφαλείας φακέλων Exchange σε PST

#### Επισκόπηση
Η δημιουργία αντιγράφων ασφαλείας συγκεκριμένων φακέλων από τον διακομιστή Exchange σε ένα αρχείο PST διασφαλίζει πλεονασμό δεδομένων και εύκολη αρχειοθέτηση.

**Βήμα 1**: Ορισμός διαδρομών καταλόγου.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

**Βήμα 2**: Ανακτήστε τον φάκελο για τον οποίο θέλετε να δημιουργήσετε αντίγραφο ασφαλείας.
```csharp
ExchangeFolderInfo info = client.GetFolderInfo(mailboxInfo.InboxUri);
ExchangeFolderInfoCollection fc = new ExchangeFolderInfoCollection();
fc.Add(info);
```

**Βήμα 3**: Εκτελέστε τη λειτουργία δημιουργίας αντιγράφων ασφαλείας.
```csharp
client.Backup(fc, dataDir + "Backup_out.pst");
```

Ο `Backup` Η μέθοδος αποθηκεύει τα περιεχόμενα του καθορισμένου φακέλου σας σε ένα αρχείο PST στην καθορισμένη τοποθεσία.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Προβλήματα ελέγχου ταυτότητας**Βεβαιωθείτε ότι το όνομα χρήστη και ο κωδικός πρόσβασης είναι σωστά. Επαληθεύστε τη σύνδεση δικτύου με τον διακομιστή Exchange.
- **Σφάλματα πρόσβασης σε φακέλους**Επιβεβαιώστε ότι έχετε τα απαραίτητα δικαιώματα για πρόσβαση και δημιουργία αντιγράφων ασφαλείας φακέλων στον διακομιστή Exchange.
- **Όρια μεγέθους αρχείου PST**Εάν αντιμετωπίζετε όρια μεγέθους, σκεφτείτε να χωρίσετε τα μεγάλα γραμματοκιβώτια σε μικρότερα τμήματα.
