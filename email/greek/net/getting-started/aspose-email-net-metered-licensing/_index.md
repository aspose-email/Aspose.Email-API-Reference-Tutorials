## Οδηγός Εφαρμογής

Τώρα που είστε έτοιμοι, ας εμβαθύνουμε στην υλοποίηση βασικών λειτουργιών χρησιμοποιώντας το Aspose.Email.

### Χαρακτηριστικό: Εφαρμογή άδειας χρήσης με μετρητή

Η λειτουργία αδειοδότησης με μετρητή είναι ζωτικής σημασίας για τον αποτελεσματικό έλεγχο και τη διαχείριση της χρήσης του API σας.

#### Βήμα 1: Ρύθμιση του κλειδιού μετρητή σας

Για να εφαρμόσετε μια άδεια χρήσης με ογκοχρέωση, χρησιμοποιήστε το `SetMeteredKey` μέθοδο μεταβιβάζοντας τα δημόσια και ιδιωτικά σας κλειδιά. Αυτό σας βοηθά να διαχειρίζεστε αποτελεσματικά τις κλήσεις API.

```csharp
using Aspose.Email;

// Αποκτήστε πρόσβαση στην ιδιότητα SetMeteredKey και μεταβιβάστε τα κλειδιά σας.
Aspose.Email.Metered metered = new Aspose.Email.Metered();
metered.SetMeteredKey("YOUR_PUBLIC_KEY", "YOUR_PRIVATE_KEY");
```

**Παράμετροι**: Αντικατάσταση `YOUR_PUBLIC_KEY` και `YOUR_PRIVATE_KEY` με πραγματικές τιμές από τον λογαριασμό σας Aspose.

### Χαρακτηριστικό: Φόρτωση εγγράφου ηλεκτρονικού ταχυδρομείου

Η φόρτωση ενός εγγράφου email είναι απλή, επιτρέποντάς σας να επεξεργάζεστε email που είναι αποθηκευμένα στο δίσκο.

#### Βήμα 2: Ορίστε τη διαδρομή και φορτώστε το έγγραφο

Ξεκινήστε καθορίζοντας τον κατάλογο όπου βρίσκονται τα αρχεία email σας. Στη συνέχεια, χρησιμοποιήστε `MailMessage.Load` για να διαβάσετε το αρχείο email.

```csharp
using Aspose.Email;

// Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας.
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Φόρτωση του εγγράφου email από τον δίσκο.
MailMessage eml = MailMessage.Load(dataDir + "Message.eml");
```

**Παράμετροι**: Αντικατάσταση `YOUR_DOCUMENT_DIRECTORY` με την πραγματική διαδρομή όπου αποθηκεύονται τα email σας.

### Λειτουργία: Ανάκτηση θέματος email

Αφού φορτώσετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου, ίσως θελήσετε να αποκτήσετε πρόσβαση σε συγκεκριμένες πληροφορίες, όπως στη γραμμή θέματος.

#### Βήμα 3: Πρόσβαση και εμφάνιση θέματος email

Ανακτήστε το θέμα του φορτωμένου email χρησιμοποιώντας το `Subject` ιδιοκτησία.

```csharp
using Aspose.Email;

// Ανακτήστε το θέμα του φορτωμένου μηνύματος ηλεκτρονικού ταχυδρομείου.
string subject = eml.Subject;
Console.WriteLine("Email Subject: " + subject);
```
