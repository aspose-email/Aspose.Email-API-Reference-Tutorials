## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε βήμα προς βήμα την εφαρμογή της λειτουργίας.

### Ορισμός κριτηρίων βάσει ημερομηνίας για τον διαχωρισμό αρχείων PST

**Επισκόπηση:**
Για να διαχωρίσετε ένα αρχείο PST με βάση τα κριτήρια ημερομηνίας, ορίστε συγκεκριμένα εύρη ημερομηνιών χρησιμοποιώντας εργαλεία δημιουργίας ερωτημάτων που παρέχονται από το Aspose.Email.

#### Βήμα 1: Ρύθμιση των καταλόγων σας
Καθορίστε καταλόγους για αρχεία εισόδου και εξόδου:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // Κατάλογος εισαγωγής
string outputDir = "YOUR_OUTPUT_DIRECTORY";   // Κατάλογος εξόδου
```

#### Βήμα 2: Δημιουργία ερωτημάτων κριτηρίων ημερομηνίας
Χρήση `PersonalStorageQueryBuilder` για να δημιουργήσετε ερωτήματα που καθορίζουν τα εύρη ημερομηνιών για τον διαχωρισμό.

**Ερώτημα 1:** Ηλεκτρονικά μηνύματα από 1 Απριλίου 2005 έως 6 Απριλίου 2005.
```csharp
PersonalStorageQueryBuilder pstQueryBuilder1 = new PersonalStorageQueryBuilder();
pstQueryBuilder1.SentDate.Since(new DateTime(2005, 04, 01)); // Ημερομηνία έναρξης
pstQueryBuilder1.SentDate.Before(new DateTime(2005, 04, 07));   // Ημερομηνία λήξης
```

**Ερώτημα 2:** Ηλεκτρονικά μηνύματα από 7 Απριλίου 2005 έως 12 Απριλίου 2005.
```csharp
PersonalStorageQueryBuilder pstQueryBuilder2 = new PersonalStorageQueryBuilder();
pstQueryBuilder2.SentDate.Since(new DateTime(2005, 04, 07)); // Ημερομηνία έναρξης
pstQueryBuilder2.SentDate.Before(new DateTime(2005, 04, 13));   // Ημερομηνία λήξης
```

Προσθέστε αυτά τα ερωτήματα σε μια λίστα:
```csharp
IList<MailQuery> criteria = new List<MailQuery>();
criteria.Add(pstQueryBuilder1.GetQuery());
criteria.Add(pstQueryBuilder2.GetQuery());
```

#### Βήμα 3: Καθαρισμός του καταλόγου εξόδου
Βεβαιωθείτε ότι ο κατάλογος εξόδου σας δεν έχει προηγούμενα αρχεία PST πριν ξεκινήσετε:
```csharp
if (Directory.GetFiles(outputDir + "pathToPst", "*.pst").Length > 0)
{
    string[] files = Directory.GetFiles(outputDir + "pathToPst");
    foreach (string file in files)
    {
        if(file.Contains(".pst"))
            File.Delete(file); // Διαγραφή υπαρχόντων αρχείων PST
    }
}
```

#### Βήμα 4: Διαχωρίστε το αρχικό αρχείο PST
Φορτώστε το αρχικό σας αρχείο PST και διαιρέστε το χρησιμοποιώντας καθορισμένα κριτήρια:
```csharp
using (PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir + "PersonalStorage_New.pst"))
{
    personalStorage.SplitInto(criteria, outputDir + "pathToPst");
}
```
**Εξήγηση:**
- `FromFile`: Φορτώνει το αρχικό PST.
- `SplitInto`: Διαχωρίζει το αρχείο με βάση τα κριτήριά σας και το αποθηκεύει στον καθορισμένο κατάλογο.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι οι διαδρομές έχουν οριστεί σωστά τόσο για τους καταλόγους εισόδου όσο και για τους καταλόγους εξόδου για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".
- Επαληθεύστε ότι έχετε δικαιώματα εγγραφής για τον κατάλογο εξόδου.
- Ελέγξτε ότι όλα τα εύρη ημερομηνιών είναι έγκυρα και δεν επικαλύπτονται, εκτός εάν αυτό προβλέπεται.
