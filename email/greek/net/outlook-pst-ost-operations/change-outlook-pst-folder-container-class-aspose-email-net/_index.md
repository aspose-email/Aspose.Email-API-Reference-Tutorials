1. **Ορισμός διαδρομών καταλόγου**

   ```csharp
   string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
   ```
2. **Άνοιγμα του αρχείου PST**
   Χρησιμοποιήστε το Aspose.Email `PersonalStorage` κλάση για να ανοίξετε το αρχείο PST σας:
   ```csharp
   string path = dataDir + "/PersonalStorage1.pst";

   using (PersonalStorage personalStorage = PersonalStorage.FromFile(path))
   {
       // Περαιτέρω επεμβάσεις θα πραγματοποιηθούν εδώ.
   }
   ```
3. **Πρόσβαση στον επιθυμητό φάκελο**
   Μεταβείτε σε έναν συγκεκριμένο φάκελο, όπως "Εισερχόμενα":
   ```csharp
   FolderInfo folder = personalStorage.RootFolder.GetSubFolder("Inbox");
   ```
4. **Αλλαγή κλάσης κοντέινερ**
   Αλλάξτε την κλάση κοντέινερ του φακέλου προορισμού σας σε "IPF.Note":
