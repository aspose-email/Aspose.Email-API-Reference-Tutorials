Αρχικά, ορίστε τον κατάλογο όπου βρίσκεται το αρχείο OST:

```csharp

```

#### Βήμα 2: Φόρτωση του αρχείου OST

Φορτώστε το αρχείο OST σε ένα `PersonalStorage` αντικείμενο. Βεβαιωθείτε ότι το 'SampleOstFile.ost' υπάρχει στον καθορισμένο κατάλογο.

```csharp
string path = dataDir + "/SampleOstFile.ost";

{
    // Συνέχεια με τη μετατροπή...
}
```

#### Βήμα 3: Μετατροπή και αποθήκευση ως PST

Τώρα, μετατρέψτε το αρχείο OST σε μορφή PST και αποθηκεύστε το στον επιθυμητό κατάλογο εξόδου:

```csharp

```
