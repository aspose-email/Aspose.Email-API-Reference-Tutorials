## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Φόρτωση αρχείου OLM

**Επισκόπηση:** Αυτή η λειτουργία δείχνει πώς να φορτώσετε ένα αρχείο OLM χρησιμοποιώντας το Aspose.Email, δημιουργώντας τη βάση για περαιτέρω λειτουργίες.

#### Βήματα:

**Ορισμός καταλόγου εγγράφων:**
Ξεκινήστε καθορίζοντας τη διαδρομή όπου είναι αποθηκευμένο το έγγραφό σας. Αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική διαδρομή καταλόγου στο σύστημά σας.
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string olmFilePath = dataDir + "/SampleOLM.olm"; // Καθορίστε το όνομα του αρχείου OLM
```
**Φόρτωση του αρχείου:**
Χρήση `OlmStorage` για να φορτώσετε το αρχείο OLM. Αυτό το αντικείμενο σάς επιτρέπει να αλληλεπιδράσετε με τον χώρο αποθήκευσης email.
```csharp
using (OlmStorage storage = new OlmStorage(olmFilePath))
{
    // Η αποθήκευση OLM έχει πλέον φορτωθεί και είναι έτοιμη για περαιτέρω λειτουργίες.
}
```

### Λειτουργία 2: Εξαγωγή μηνυμάτων από φάκελο

**Επισκόπηση:** Μάθετε πώς να εξάγετε και να εμφανίζετε μηνύματα που είναι αποθηκευμένα σε φακέλους μέσα σε ένα αρχείο OLM.

#### Βήματα:
**Επανάληψη σε φακέλους:**
Ελέγξτε κάθε φάκελο στην ιεραρχία και επεξεργαστείτε αν περιέχει μηνύματα.
```csharp
foreach (OlmFolder folder in storage.FolderHierarchy)
{
    if (folder.HasMessages)
    {
        // Επεξεργασία κάθε μηνύματος σε αυτόν τον φάκελο
        foreach (MapiMessage msg in storage.EnumerateMessages(folder))
        {
            Console.WriteLine("Subject: " + msg.Subject);
        }
    }
}
```

### Χαρακτηριστικό 3: Ανάγνωση υποφακέλων

**Επισκόπηση:** Αυτή η λειτουργία δείχνει πώς να πλοηγηθείτε και να διαβάσετε υποφακέλους μέσα σε ένα αρχείο OLM.

#### Βήματα:
**Πρόσβαση σε υποφακέλους:**
Περιηγηθείτε στους υποφακέλους κάθε φακέλου και εμφανίστε τα ονόματά τους.
```csharp
foreach (OlmFolder folder in storage.FolderHierarchy)
{
    if (folder.SubFolders.Count > 0)
    {
        foreach (OlmFolder sub_folder in folder.SubFolders)
        {
            Console.WriteLine("Subfolder: " + sub_folder.Name);
        }
    }
}
```
