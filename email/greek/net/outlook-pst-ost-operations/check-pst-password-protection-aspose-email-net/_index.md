### Ελέγξτε τη λειτουργία προστασίας με κωδικό πρόσβασης
Αυτή η λειτουργία βοηθά στον προσδιορισμό του εάν ένα αρχείο PST έχει οριστεί ως κωδικός πρόσβασης, διασφαλίζοντας τον ασφαλή χειρισμό δεδομένων.

#### Βήμα προς βήμα εφαρμογή

**1. Ανοίξτε το αρχείο PST**
Ανοίξτε το αρχείο PST από τον κατάλογό του:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (PersonalStorage pst = PersonalStorage.FromFile(dataDir + "/passwordprotectedPST.pst"))
{
    bool isProtected = IsPasswordProtected(pst);
}
```

**2. Προσδιορίστε την κατάσταση προστασίας με κωδικό πρόσβασης**
Ο `IsPasswordProtected` Η μέθοδος ελέγχει για την ύπαρξη μιας ιδιότητας κωδικού πρόσβασης:

```csharp
private static bool IsPasswordProtected(PersonalStorage pst)
{
    // Ελέγξτε εάν το αρχείο PST έχει ορίσει μια ιδιότητα κωδικού πρόσβασης
    if (pst.Store.Properties.ContainsKey(MapiPropertyTag.PR_PST_PASSWORD))
    {
        long passwordHash = pst.Store.Properties[MapiPropertyTag.PR_PST_PASSWORD].GetLong();
        return passwordHash != 0; // Εάν ο κατακερματισμός είναι μη μηδενικός, τότε είναι προστατευμένος
    }
    return false;
}
```

- **Παράμετροι:** Ο `PersonalStorage` ένα αντίγραφο που αντιπροσωπεύει το αρχείο PST.
- **Επιστρεφόμενη τιμή:** Μια λογική τιμή που υποδεικνύει εάν έχει οριστεί κωδικός πρόσβασης.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι η διαδρομή προς το αρχείο PST είναι σωστή.
- Βεβαιωθείτε ότι έχετε δικαιώματα ανάγνωσης για το αρχείο.
