## Οδηγός Εφαρμογής

### Έλεγχος εάν ένα αρχείο PST προστατεύεται με κωδικό πρόσβασης

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσδιορίσετε εάν ένα αρχείο PST απαιτεί κωδικό πρόσβασης, κάτι που είναι απαραίτητο για τους ελέγχους ασφαλείας και τη διαχείριση πρόσβασης.

**Βήμα 1: Φόρτωση του αρχείου PST**

Φορτώστε το αρχείο PST χρησιμοποιώντας το Aspose.Email `PersonalStorage` τάξη:

```csharp
using (PersonalStorage pst = PersonalStorage.FromFile(dataDir + "/passwordprotectedPST.pst"))
{
    bool isPasswordProtected = pst.Store.IsPasswordProtected;
}
```
- **Παράμετροι:** `dataDir` θα πρέπει να είναι η διαδρομή προς τον κατάλογο εγγράφων σας.
- **Επιστρεφόμενη τιμή:** Μια λογική τιμή που υποδεικνύει εάν το αρχείο PST προστατεύεται με κωδικό πρόσβασης.

**Βήμα 2: Εξαγωγή αποτελεσμάτων**

Καταγράψτε ή εμφανίστε αυτά τα αποτελέσματα όπως απαιτείται:

```csharp
Console.WriteLine($"The storage is password protected - {isPasswordProtected}");
```

### Επικύρωση κωδικού πρόσβασης για ένα αρχείο PST

#### Επισκόπηση
Αυτή η λειτουργία επικυρώνει εάν ένας παρεχόμενος κωδικός πρόσβασης ταιριάζει με αυτόν που έχει οριστεί στο αρχείο PST, διασφαλίζοντας ασφαλή πρόσβαση.

**Βήμα 1: Φόρτωση και επικύρωση του κωδικού πρόσβασης**

Φορτώστε το αρχείο PST και ελέγξτε αν ο κωδικός πρόσβασης είναι έγκυρος:

```csharp
using (PersonalStorage pst = PersonalStorage.FromFile(dataDir + "/passwordprotectedPST.pst"))
{
    string testPassword = "Password1";
    bool isValid = pst.Store.IsPasswordValid(testPassword);
}
```
- **Παράμετροι:** `testPassword` θα πρέπει να είναι ο κωδικός πρόσβασης που επαληθεύετε.
- **Επιστρεφόμενη τιμή:** Μια λογική τιμή που υποδεικνύει εάν ο παρεχόμενος κωδικός πρόσβασης είναι έγκυρος.

**Βήμα 2: Εξαγωγή αποτελεσμάτων**

Καταγραφή ή εμφάνιση εάν η επικύρωση κωδικού πρόσβασης ήταν επιτυχής:

```csharp
Console.WriteLine($"Password {testPassword} is valid - {isValid}");
```
