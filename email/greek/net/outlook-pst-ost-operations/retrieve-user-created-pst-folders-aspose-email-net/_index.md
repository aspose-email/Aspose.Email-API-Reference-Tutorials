# Πώς να ανακτήσετε φακέλους PST που δημιουργήθηκαν από χρήστη χρησιμοποιώντας το Aspose.Email για .NET

## Εισαγωγή

Η αποτελεσματική διαχείριση δεδομένων email είναι απαραίτητη όταν ασχολείστε με μεγάλα αρχεία PST στο Microsoft Outlook. Το φιλτράρισμα και η ανάκτηση φακέλων που δημιουργούνται από χρήστες, εξαιρώντας αυτούς που δημιουργούνται από το σύστημα, μπορεί να είναι δύσκολη χωρίς τα κατάλληλα εργαλεία. [Aspose.Email για .NET](https://reference.aspose.com/email/net/) παρέχει μια ισχυρή λύση για την απλοποίηση αυτής της διαδικασίας.

Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στη χρήση του Aspose.Email για .NET για την αναζήτηση και ανάκτηση μόνο φακέλων που έχουν δημιουργηθεί από χρήστες από ένα αρχείο PST. Παρακολουθώντας, θα μάθετε:
- Ρύθμιση του περιβάλλοντός σας με το Aspose.Email
- Χρησιμοποιώντας `PersonalStorageQueryBuilder` για να φιλτράρετε φακέλους που έχουν δημιουργηθεί από χρήστες
- Εφαρμογή αποτελεσματικών αποσπασμάτων κώδικα
- Βελτιστοποίηση απόδοσης κατά τον χειρισμό μεγάλων αρχείων PST

Ας εμβαθύνουμε και ας βελτιώσουμε τις δεξιότητές σας στη διαχείριση δεδομένων email!

## Οδηγός Εφαρμογής

### Υποβολή ερωτημάτων και ανάκτηση φακέλων που δημιουργήθηκαν από χρήστες
Αυτή η ενότητα εστιάζει στη ρύθμιση ενός ερωτήματος για το φιλτράρισμα και την ανάκτηση φακέλων που δημιουργούνται μόνο από χρήστες.

#### 1. Φορτώστε το αρχείο PST
Αρχικά, φορτώστε το αρχείο PST του Outlook χρησιμοποιώντας το `FromFile` μέθοδος:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (PersonalStorage pst = PersonalStorage.FromFile(dataDir + "Outlook.pst"))
{
    // Συνέχεια με την υποβολή ερωτημάτων σε φακέλους...
}
```

#### 2. Δημιουργήστε ένα Εργαλείο Δημιουργίας Ερωτημάτων
Χρησιμοποιήστε το `PersonalStorageQueryBuilder` για να ορίσετε τις συνθήκες του ερωτήματός σας:

```csharp
// Δημιουργήστε ένα εργαλείο δημιουργίας ερωτημάτων για το φιλτράρισμα φακέλων που δημιουργούνται από χρήστες
PersonalStorageQueryBuilder queryBuilder = new PersonalStorageQueryBuilder();
queryBuilder.OnlyFoldersCreatedByUser.Equals(true);
```

Αυτό το βήμα φιλτράρει τους φακέλους, διασφαλίζοντας ότι στα αποτελέσματα περιλαμβάνονται μόνο αυτοί που δημιουργούνται από χρήστες.

#### 3. Ανάκτηση και εμφάνιση φακέλων
Ανάκτηση υποφακέλων που ταιριάζουν με τα κριτήριά σας και εμφάνιση των ονομάτων τους:

```csharp
// Λήψη υποφακέλων που ταιριάζουν με το ερώτημα
FolderInfoCollection subfolders = pst.RootFolder.GetSubFolders(queryBuilder.GetQuery());

// Επαναλάβετε την εκτέλεση κάθε φακέλου για να εκτελέσετε λειτουργίες
foreach (FolderInfo folder in subfolders)
{
    Console.WriteLine(folder.DisplayName);
}
```

**Εξήγηση**: Εδώ, `GetSubFolders` ανακτά φακέλους με βάση τις συνθήκες σας. Στη συνέχεια, επαναλαμβάνουμε αυτούς τους φακέλους και εκτυπώνουμε τα εμφανιζόμενα ονόματά τους.
