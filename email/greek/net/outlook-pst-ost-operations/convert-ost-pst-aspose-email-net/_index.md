## Οδηγός Εφαρμογής

### Μετατροπή OST σε PST (H2)
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο OST του Outlook σε μορφή PST, το οποίο μπορεί να είναι χρήσιμο για μετεγκατάσταση δεδομένων και δημιουργία αντιγράφων ασφαλείας.

#### Βήμα 1: Ορισμός διαδρομών αρχείων (H3)
Καθορίστε τη διαδρομή του αρχείου OST προέλευσης και τη διαδρομή εξόδου του αρχείου PST προορισμού:

```csharp
string sourceFilePath = "@YOUR_DOCUMENT_DIRECTORY/PersonalStorageFile.ost";
string targetFilePath = "@YOUR_OUTPUT_DIRECTORY/test.pst";
```

#### Βήμα 2: Ανοίξτε το αρχείο OST (H3)
Χρησιμοποιήστε το `FromFile` μέθοδος για να ανοίξετε το αρχείο OST, διαβάζοντας και φορτώνοντας τα περιεχόμενά του:

```csharp
using (PersonalStorage personalStorage = PersonalStorage.FromFile(sourceFilePath))
{
    // Συνεχίστε με τη μετατροπή
}
```

#### Βήμα 3: Αποθήκευση ως αρχείο PST (H3)
Μόλις ανοίξετε το αρχείο OST, χρησιμοποιήστε το `SaveAs` μέθοδος για να το μετατρέψετε και να το αποθηκεύσετε σε μορφή PST. Η `FileFormat.Pst` Η παράμετρος καθορίζει την επιθυμητή μορφή εξόδου:

```csharp
personalStorage.SaveAs(targetFilePath, FileFormat.Pst);
```
