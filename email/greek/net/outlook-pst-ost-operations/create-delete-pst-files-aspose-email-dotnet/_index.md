## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίηση σε δύο κύρια χαρακτηριστικά: δημιουργία/διαγραφή αρχείων PST και προσθήκη υποφακέλων/αρχείων σε αυτά.

### Λειτουργία 1: Δημιουργία και διαγραφή αρχείου PST

**Επισκόπηση**Αυτή η λειτουργία σάς βοηθά να δημιουργήσετε ένα νέο αρχείο PST σε μορφή Unicode ή να διαγράψετε ένα υπάρχον αρχείο, εάν υπάρχει ήδη.

#### Βήμα προς βήμα εφαρμογή:

##### 1. Ορίστε τη διαδρομή καταλόγου
Ξεκινήστε ορίζοντας τον κατάλογο όπου θα αποθηκευτούν τα αρχεία PST.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string path = Path.Combine(dataDir, "Ps1_out.pst");
```

##### 2. Ελέγξτε για υπάρχον PST και διαγράψτε το, εάν είναι απαραίτητο
Βεβαιωθείτε ότι δεν δημιουργείτε διπλότυπα υπάρχοντα αρχεία ελέγχοντας πρώτα την παρουσία τους.
```csharp
if (File.Exists(path))
{
    File.Delete(path);
}
```

##### 3. Δημιουργήστε ένα νέο αρχείο PST
Δημιουργήστε το νέο αρχείο χρησιμοποιώντας τη μορφή Unicode για να διασφαλίσετε τη συμβατότητα με διάφορα προγράμματα-πελάτες ηλεκτρονικού ταχυδρομείου.
```csharp
using (PersonalStorage personalStorage = PersonalStorage.Create(Path.Combine(dataDir, "Ps1_out.pst"), FileFormatVersion.Unicode))
{
    // Η δημιουργία PST ολοκληρώθηκε εδώ.
}
```

### Λειτουργία 2: Προσθήκη υποφακέλου και αρχείων σε PST

**Επισκόπηση**Αφού δημιουργήσετε ένα αρχείο PST, μπορείτε να οργανώσετε τα περιεχόμενά του προσθέτοντας υποφακέλους και αρχεία.

#### Βήμα προς βήμα εφαρμογή:

##### 1. Βεβαιωθείτε ότι υπάρχει το αρχείο PST
Ελέγξτε αν υπάρχει το αρχείο PST σας. Εάν όχι, δημιουργήστε το.
```csharp
if (!File.Exists(path))
{
    using (PersonalStorage personalStorage = PersonalStorage.Create(path, FileFormatVersion.Unicode))
    {
        // Ο ριζικός φάκελος δημιουργείται αυτόματα εδώ.
    }
}
```

##### 2. Ανοίξτε το υπάρχον αρχείο PST
Φορτώστε το υπάρχον αρχείο για να προσθέσετε υποφακέλους και αρχεία.
```csharp
using (PersonalStorage personalStorage = PersonalStorage.FromFile(path))
{
    // Άνοιγμα του ριζικού φακέλου του αρχείου PST
```

##### 3. Προσθήκη υποφακέλου
Δημιουργήστε έναν νέο υποφάκελο με το όνομα "Αρχεία" κάτω από τον ριζικό φάκελο.
```csharp
FolderInfo folder = personalStorage.RootFolder.AddSubFolder("Files");
```

##### 4. Προσθήκη αρχείων στον υποφάκελο
Προσθέστε αρχεία στον νεοδημιουργημένο υποφάκελο, καθορίζοντας διαδρομές αρχείων και τυχόν απαραίτητα χαρακτηριστικά.
```csharp
folder.AddFile(Path.Combine(dataDir, "attachment_1.doc"), null);
```
