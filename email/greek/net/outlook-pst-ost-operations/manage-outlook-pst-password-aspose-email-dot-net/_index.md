## Οδηγός Εφαρμογής

### Αφαίρεση ιδιότητας κωδικού πρόσβασης από ένα αρχείο PST

Αυτή η λειτουργία σάς επιτρέπει να καταργήσετε την προστασία με κωδικό πρόσβασης, καθιστώντας το PST προσβάσιμο χωρίς έλεγχο ταυτότητας.

#### Βήμα 1: Φόρτωση του αρχείου PST
Φορτώστε το αρχείο PST χρησιμοποιώντας το Aspose.Email `PersonalStorage` τάξη.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/PersonalStorage1.pst";
PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
```

#### Βήμα 2: Έλεγχος και κατάργηση κωδικού πρόσβασης
Ελέγξτε αν το αρχείο PST έχει ορίσει μια ιδιότητα κωδικού πρόσβασης και, στη συνέχεια, καταργήστε την ορίζοντας έναν κενό κωδικό πρόσβασης.

```csharp
if (personalStorage.Store.Properties.ContainsKey(MapiPropertyTag.PR_PST_PASSWORD))
{
    MapiProperty property = new MapiProperty(MapiPropertyTag.PR_PST_PASSWORD, BitConverter.GetBytes((long)0));
    personalStorage.Store.SetProperty(property);
}
```

*Εξήγηση*: Το `MapiPropertyTag.PR_PST_PASSWORD` ελέγχει για κωδικό πρόσβασης. Εάν υπάρχει, αντικαθίσταται με μηδέν byte για την αποτελεσματική κατάργηση του κωδικού πρόσβασης.

#### Συμβουλή αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι έχετε δικαιώματα εγγραφής στον κατάλογο που περιέχει το αρχείο PST.
  
### Ανάγνωση ιδιοτήτων αρχείου PST

Αποκτήστε πρόσβαση και εμφανίστε βασικές ιδιότητες του αρχείου PST σας.

#### Βήμα 1: Φόρτωση του αρχείου PST
Ξεκινήστε φορτώνοντας το αρχείο PST, όπως ακριβώς κάνετε και με την κατάργηση ενός κωδικού πρόσβασης.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/PersonalStorage1.pst";
PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
```

#### Βήμα 2: Πρόσβαση και εμφάνιση ιδιοτήτων
Αποκτήστε πρόσβαση σε ιδιότητες όπως το εμφανιζόμενο όνομα, τον αριθμό στοιχείων και το μέγεθος και, στη συνέχεια, εκτυπώστε τις.

```csharp
Console.WriteLine("Display Name: " + personalStorage.DisplayName);
Console.WriteLine("Total Number of Items: " + personalStorage.RootFolder.ContentCount);
Console.WriteLine("Total Size in Bytes: " + personalStorage.RootFolder.SizeInBytes);
```

*Εξήγηση*: `DisplayName` παρέχει ένα όνομα αναγνώσιμο από τον άνθρωπο, ενώ `ContentCount` και `SizeInBytes` δώσουν πληροφορίες για το περιεχόμενο του αρχείου.
