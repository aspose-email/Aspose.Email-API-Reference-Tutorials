## Οδηγός Εφαρμογής

### Αποθήκευση αρχείου MSG του Outlook ως προτύπου (μορφή OFT)

**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να αποθηκεύσετε ένα αρχείο MSG του Outlook απευθείας ως πρότυπο, απλοποιώντας τις επαναλαμβανόμενες εργασίες δημιουργίας email.

#### Βήμα προς βήμα εφαρμογή:
1. **Δημιουργήστε το αντικείμενο MapiMessage**
   
   Δημιουργήστε ένα νέο `MapiMessage` παράδειγμα με τον επιθυμητό αποστολέα, παραλήπτη, θέμα και σώμα.
   
   ```csharp
   using (MapiMessage mapi = new MapiMessage("test@from.to", "test@to.to", "template subject", "Template body"))
   {
       string oftMapiFileName = System.IO.Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "mapiToOft.msg");
       
       mapi.SaveAsTemplate(oftMapiFileName);
   }
   ```

2. **Παράμετροι και Διαμόρφωση:**
   - `SaveAsTemplate` χρησιμοποιείται για την αποθήκευση του μηνύματος σε μορφή OFT, η οποία είναι απαραίτητη για τη δημιουργία προτύπου.
   - Βεβαιωθείτε ότι έχετε καθορίσει μια έγκυρη διαδρομή καταλόγου όπου θα αποθηκευτεί το αρχείο.

### Φόρτωση αρχείου MSG στο MapiMessage

**Επισκόπηση:**
Η φόρτωση υπαρχόντων αρχείων MSG στην εφαρμογή σας επιτρέπει τον προγραμματιστικό χειρισμό ή την ανάγνωση δεδομένων email.

#### Βήματα Υλοποίησης:
1. **Φόρτωση του αρχείου MSG**
   
   Χρήση `MapiMessage.FromFile` για να φορτώσετε ένα αρχείο MSG σε ένα `MapiMessage` αντικείμενο.
   
   ```csharp
   string msgFilePath = System.IO.Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.msg");
   MapiMessage loadedMsg = MapiMessage.FromFile(msgFilePath);
   ```

2. **Πρόσβαση στις Ιδιότητες Μηνύματος**
   
   Μόλις φορτωθεί, αποκτήστε πρόσβαση σε διάφορες ιδιότητες όπως το θέμα και το σώμα.
   
   ```csharp
   Console.WriteLine("Subject: " + loadedMsg.Subject);
   Console.WriteLine("Body: " + loadedMsg.Body);
   ```
