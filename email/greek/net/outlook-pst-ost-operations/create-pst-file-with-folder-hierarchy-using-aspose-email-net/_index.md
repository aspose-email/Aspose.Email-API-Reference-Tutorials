## Οδηγός Εφαρμογής

Ας δούμε πώς δημιουργούμε ένα αρχείο PST και προσθέτουμε φακέλους χρησιμοποιώντας συμβολοσειρές.

### Δημιουργία νέου αρχείου PST

#### Επισκόπηση

Η δημιουργία ενός νέου αρχείου PST είναι απλή με τη βιβλιοθήκη Aspose.Email. Αυτή η ενότητα σας καθοδηγεί στη ρύθμιση του αρχικού σας περιβάλλοντος για την αποθήκευση δεδομένων email.

**Βήμα 1: Ορισμός διαδρομών καταλόγου**

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY\\CreateFolderHierarchyUsingStringNotation.pst";
```

Αντικαθιστώ `YOUR_DOCUMENT_DIRECTORY` με την πραγματική διαδρομή όπου θέλετε να αποθηκεύσετε το αρχείο PST.

#### Βήμα 2: Δημιουργήστε ένα νέο αρχείο PST

Εδώ, χρησιμοποιούμε τη μορφή Unicode για καλύτερη συμβατότητα και αποτελεσματικότητα αποθήκευσης:

```csharp
PersonalStorage personalStorage = PersonalStorage.Create(dataDir, FileFormatVersion.Unicode);
```

### Προσθήκη ιεραρχίας φακέλων

#### Επισκόπηση

Η προσθήκη φακέλων εντός της δομής PST βοηθά στην αποτελεσματική οργάνωση των δεδομένων email. Αυτή η ενότητα σάς δείχνει πώς να προσθέσετε μια ιεραρχία ένθετων φακέλων.

**Βήμα 3: Προσθήκη ιεραρχίας υποφακέλων**

Για να δημιουργήσετε υποφακέλους στον ριζικό σας φάκελο:

```csharp
personalStorage.RootFolder.AddSubFolder("Inbox\\Folder1\\Folder2");
```

Αυτό το απόσπασμα κώδικα απεικονίζει την προσθήκη φακέλων ορίζοντας τη διαδρομή ως `Inbox\Folder1\Folder2`.
