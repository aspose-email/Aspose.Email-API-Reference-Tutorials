# Πώς να δημιουργήσετε και να διαχειριστείτε αρχεία PST του Outlook χρησιμοποιώντας το Aspose.Email για .NET

**Τι θα μάθετε:**

## Οδηγός Εφαρμογής

### Δημιουργία νέου αρχείου PST

**Επισκόπηση**Αυτή η ενότητα παρουσιάζει τον τρόπο δημιουργίας ενός νέου αρχείου Προσωπικού Πίνακα Αποθήκευσης (PST) του Outlook σε μορφή Unicode για συμβατότητα και αποτελεσματικότητα.

#### Βήματα:
1. **Ορίστε τη διαδρομή αρχείου:**
   ```csharp
   string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "new_pst_out.pst");
   ```
2. **Έλεγχος για υπάρχον αρχείο:**
   Βεβαιωθείτε ότι δεν υπάρχει κανένα υπάρχον αρχείο στον κατάλογο προορισμού σας για να αποφύγετε διενέξεις:
   ```csharp
   if (File.Exists(dataDir)) {
       File.Delete(dataDir);
   }
   ```
3. **Δημιουργήστε το PST:**
   Αρχικοποιήστε ένα νέο αρχείο PST χρησιμοποιώντας τη μορφή Unicode, η οποία υποστηρίζει ευρύτερα σύνολα χαρακτήρων.
   ```csharp
   PersonalStorage personalStorage = PersonalStorage.Create(dataDir, FileFormatVersion.Unicode);
   ```

### Προσθήκη φακέλου σε PST

**Επισκόπηση**Μάθετε πώς να προσθέτετε υποφακέλους όπως «Εισερχόμενα» σε ένα υπάρχον αρχείο PST για καλύτερη οργάνωση.

#### Βήματα:
1. **Φόρτωση του υπάρχοντος PST:**
   ```csharp
   if (File.Exists(dataDir)) {
       PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
   }
   ```
2. **Προσθήκη υποφακέλου:**
   Προσθέστε έναν νέο φάκελο, όπως "Εισερχόμενα", κάτω από τον ριζικό φάκελο.
   ```csharp
   personalStorage.RootFolder.AddSubFolder("Inbox");
   ```

### Προσθήκη μηνυμάτων σε φάκελο σε PST

**Επισκόπηση**Αυτή η ενότητα απεικονίζει την προσθήκη μηνυμάτων σε έναν υπάρχοντα φάκελο "Εισερχόμενα" μέσα στο αρχείο PST σας.

#### Βήματα:
1. **Φόρτωση του υπάρχοντος αρχείου PST και μηνύματος:**
   Βεβαιωθείτε ότι και τα δύο αρχεία είναι προσβάσιμα:
   ```csharp
   if (File.Exists(dataDir)) {
       PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
   }
   string msgFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "MapiMsgWithPoll.msg");
   ```
2. **Επιλέξτε τον φάκελο και προσθέστε μηνύματα:**
   Ανάκτηση φακέλου «Εισερχόμενα» και προσθήκη μηνύματος:
   ```csharp
   FolderInfo inboxFolder = personalStorage.RootFolder.GetSubFolder("Inbox");
   MapiMessage mapiMsg = MapiMessage.FromFile(msgFilePath);
   inboxFolder.AddMessage(mapiMsg);
   ```
