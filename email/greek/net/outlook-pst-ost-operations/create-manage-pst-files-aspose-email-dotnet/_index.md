## Οδηγός Εφαρμογής
Αυτή η ενότητα θα χωριστεί σε δύο κύριες λειτουργίες: τη δημιουργία ενός αρχείου PST και την προσθήκη υποφακέλων σε ένα υπάρχον αρχείο PST.

### Χαρακτηριστικό 1: Δημιουργία νέου αρχείου PST
Η δημιουργία ενός νέου αρχείου PST είναι απλή με το Aspose.Email για .NET. Δείτε πώς μπορείτε να το πετύχετε:

#### Επισκόπηση:
Θα μάθετε να δημιουργείτε ένα νέο αρχείο PST σε μορφή Unicode, το οποίο είναι απαραίτητο για την υποστήριξη διαφόρων συνόλων χαρακτήρων παγκοσμίως.

#### Βήματα Υλοποίησης:

**Βήμα 1:** Ορίστε τη διαδρομή όπου θα δημιουργηθεί το νέο σας PST.
```csharp
string path = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "CreateNewPSTFile_out.pst");
if (File.Exists(path))
{
    File.Delete(path);
}
```
*Εξήγηση:* Αυτό το τμήμα κώδικα ορίζει μια διαδρομή αρχείου και διαγράφει οποιοδήποτε υπάρχον αρχείο για την αποφυγή διενέξεων.

**Βήμα 2:** Δημιουργήστε το αρχείο PST.
```csharp
// Δημιουργήστε ένα νέο αρχείο PST με μορφή Unicode.
PersonalStorage personalStorage = PersonalStorage.Create(path, FileFormatVersion.Unicode);

// Προσθέστε έναν νέο φάκελο με το όνομα 'Εισερχόμενα' κάτω από τον ριζικό κατάλογο του PST.
personalStorage.RootFolder.AddSubFolder("Inbox");
```
*Εξήγηση:* Αυτός ο κώδικας δημιουργεί ένα νέο PST και προσθέτει έναν υποφάκελο "Εισερχόμενα".

### Λειτουργία 2: Προσθήκη υποφακέλων σε ένα υπάρχον αρχείο PST
Η προσθήκη υποφακέλων σε ένα υπάρχον αρχείο PST μπορεί να σας βοηθήσει να οργανώσετε αποτελεσματικά τα δεδομένα email σας.

#### Επισκόπηση:
Αυτή η λειτουργία σάς επιτρέπει να βελτιώσετε την οργάνωση των email προσθέτοντας φακέλους όπως "Απεσταλμένα".

#### Βήματα Υλοποίησης:

**Βήμα 1:** Ορίστε τη διαδρομή προς το υπάρχον PST σας.
```csharp
string path = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "ExistingPSTFile.pst");
if (!File.Exists(path))
{
    throw new FileNotFoundException($"The file at {path} was not found.");
}
```
*Εξήγηση:* Διασφαλίζει ότι το καθορισμένο αρχείο PST υπάρχει πριν προχωρήσει.

**Βήμα 2:** Φόρτωση και τροποποίηση του υπάρχοντος PST.
```csharp
// Φορτώστε το υπάρχον αρχείο PST.
PersonalStorage personalStorage = PersonalStorage.FromFile(path);

// Προσθέστε έναν υποφάκελο με το όνομα "Απεσταλμένα" στον ριζικό φάκελο του PST.
personalStorage.RootFolder.AddSubFolder("SentItems");
```
*Εξήγηση:* Αυτό το απόσπασμα φορτώνει ένα υπάρχον αρχείο PST και προσθέτει έναν υποφάκελο "Απεσταλμένα".
