## Οδηγός Εφαρμογής

### Συγχώνευση πολλαπλών αρχείων PST σε ένα μόνο αρχείο
Αυτή η λειτουργία σάς επιτρέπει να ενοποιήσετε πολλά αρχεία PST από έναν καθορισμένο κατάλογο σε ένα μόνο αρχείο.

#### Επισκόπηση
Εγγραφόμενοι σε συγκεκριμένα συμβάντα, μπορούμε να παρακολουθούμε τη διαδικασία συγχώνευσης και ακόμη και τον αριθμό των μηνυμάτων που μετακινούνται ανά φάκελο. Αυτό παρέχει διαφάνεια και έλεγχο κατά τη διάρκεια της λειτουργίας.

#### Βήματα Υλοποίησης

##### Βήμα 1: Ορισμός διαδρομών και αρχικοποίηση χώρου αποθήκευσης
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Ενημερώστε αυτό με τη διαδρομή καταλόγου σας
string dst = Path.Combine(dataDir, "Sub.pst");
int totalAdded = 0;

try
{
    using (PersonalStorage personalStorage = PersonalStorage.FromFile(dst))
    {
        // Εγγραφείτε σε εκδηλώσεις για να παρακολουθείτε τη διαδικασία
        personalStorage.StorageProcessed += PstMerge_OnStorageProcessed;
        personalStorage.ItemMoved += PstMerge_OnItemMoved;

        // Συγχώνευση όλων των αρχείων PST που βρίσκονται στον καθορισμένο κατάλογο
        personalStorage.MergeWith(Directory.GetFiles(Path.Combine(dataDir, "MergePST")));
    }
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message + "\nThis example will only work if you apply a valid Aspose Email License.");
}
```
- **Επεξήγηση παραμέτρων:**
  - `dataDir`: Κατάλογος όπου αποθηκεύονται τα αρχεία PST σας.
  - `dst`: Διαδρομή αρχείου προορισμού για το συγχωνευμένο PST.

##### Βήμα 2: Χειρισμός συμβάντων

**Χειριστής συμβάντων για επεξεργασία αποθήκευσης:**
Αυτό το συμβάν καταγράφει πότε γίνεται επεξεργασία κάθε χώρου αποθήκευσης.
```csharp
static void PstMerge_OnStorageProcessed(object sender, StorageProcessedEventArgs e)
{
    Console.WriteLine("*** The storage is merging: {0}", e.FileName);
}
```

**Χειριστής συμβάντων για μετακίνηση αντικειμένων:**
Παρακολουθεί τον αριθμό των μηνυμάτων που μετακινούνται ανά φάκελο και ενημερώνει ανάλογα.
```csharp
static void PstMerge_OnItemMoved(object sender, ItemMovedEventArgs e)
{
    static string currentFolder = null;
    static int messageCount = 0;

    if (currentFolder == null)
    {
        currentFolder = e.DestinationFolder.RetrieveFullPath();
    }

    string folderPath = e.DestinationFolder.RetrieveFullPath();

    if (currentFolder != folderPath)
    {
        Console.WriteLine("    Added {0} messages to \"{1}\"", messageCount, currentFolder);
        messageCount = 0;
        currentFolder = folderPath;
    }

    messageCount++;
    totalAdded++;
}
```
