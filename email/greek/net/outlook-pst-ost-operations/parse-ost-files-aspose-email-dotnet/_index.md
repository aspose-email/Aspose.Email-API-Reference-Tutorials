## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Ανάλυση αρχείων OST και ανάκτηση ονομάτων φακέλων

Αυτή η λειτουργία δείχνει πώς να ανοίξετε ένα αρχείο OST και να ανακτήσετε όλα τα ονόματα φακέλων μαζί με τα γονικά τους ονόματα χρησιμοποιώντας το Aspose.Email για .NET.

#### Επισκόπηση
Η ανάλυση ενός αρχείου OST σάς επιτρέπει να πλοηγηθείτε στη δομή του, προσδιορίζοντας το όνομα και την ιεραρχία κάθε φακέλου. Αυτό είναι κρίσιμο για την αποτελεσματική οργάνωση και πρόσβαση στα δεδομένα email.

##### Βήμα 1: Ορισμός διαδρομών καταλόγου
Ξεκινήστε καθορίζοντας τον κατάλογο όπου είναι αποθηκευμένα τα αρχεία OST:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string path = Path.Combine(dataDir, "PersonalStorage.pst");
```

##### Βήμα 2: Διαβάστε και ανοίξτε το αρχείο OST
Χρησιμοποιήστε έναν πίνακα byte για να διαβάσετε το αρχείο OST και να το ανοίξετε ως ροή:
```csharp
byte[] buffer = File.ReadAllBytes(path);
using (Stream s = File.OpenRead(path))
{
    PersonalStorage pst = PersonalStorage.FromStream(s);
    
    // Ανάκτηση ενός συγκεκριμένου φακέλου με βάση το αναγνωριστικό καταχώρισής του, εάν χρειάζεται
    FolderInfo target = pst.GetFolderById("AAAAAB9of1CGOidPhTb686WQY68igAAA");
    IList<string> folderData = new List<string>();
    
    // Περιηγηθείτε σε όλους τους φακέλους για να συγκεντρώσετε τα εμφανιζόμενα ονόματα και τα γονικά ονόματα
    WalkFolders(pst.RootFolder, "N/A", folderData);
}
```

##### Βήμα 3: Αναδρομική περιήγηση σε φακέλους
Ορίστε μια μέθοδο για την αναδρομική πλοήγηση στη δομή φακέλων:
```csharp
private static void WalkFolders(FolderInfo folder, string parentFolderName, IList<string> folderData)
{
    // Προσδιορίστε το εμφανιζόμενο όνομα ή χρησιμοποιήστε το 'ROOT' εάν είναι null ή κενό
    string displayName = (string.IsNullOrEmpty(folder.DisplayName)) ? "ROOT" : folder.DisplayName;
    
    // Μορφοποίηση και αποθήκευση των πληροφοριών του φακέλου ως συμβολοσειρά
    string folderNames = string.Format("DisplayName = {0}; Parent.DisplayName = {1}", displayName, parentFolderName);
    folderData.Add(folderNames);
    
    // Επεξεργασία υποφακέλων, εάν υπάρχουν
    if (!folder.HasSubFolders) return;
    
    FolderInfoCollection coll = folder.GetSubFolders(FolderKind.Search | FolderKind.Normal);
    foreach (FolderInfo subfolder in coll)
    {
        WalkFolders(subfolder, displayName, folderData);
    }
}
```
