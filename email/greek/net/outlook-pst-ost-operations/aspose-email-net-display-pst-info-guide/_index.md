## Οδηγός Εφαρμογής

### Εμφάνιση πληροφοριών αρχείου PST

Αυτή η ενότητα σας καθοδηγεί στη φόρτωση ενός αρχείου PST και στην εμφάνιση των λεπτομερειών του φακέλου του.

#### Φόρτωση του αρχείου PST

Καθορίστε τη διαδρομή προς το αρχείο PST και φορτώστε το χρησιμοποιώντας το `PersonalStorage.FromFile` μέθοδος:
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";
string dst = dataDir + "/PersonalStorage.pst";

// Φόρτωση του αρχείου PST
PersonalStorage personalStorage = PersonalStorage.FromFile(dst);
```

#### Λήψη όλων των υποφακέλων

Ανάκτηση όλων των υποφακέλων στον ριζικό φάκελο του αρχείου PST:
```csharp
FolderInfoCollection folderInfoCollection = personalStorage.RootFolder.GetSubFolders();
```

#### Επανάληψη και εμφάνιση πληροφοριών φακέλου

Επαναλάβετε την εκτέλεση κάθε φακέλου για να εμφανίσετε το όνομά του, τον συνολικό αριθμό στοιχείων και τον αριθμό των μη αναγνωσμένων στοιχείων:
```csharp
foreach (FolderInfo folderInfo in folderInfoCollection)
{
    Console.WriteLine("Folder: " + folderInfo.DisplayName);
    Console.WriteLine("Total items: " + folderInfo.ContentCount);
    Console.WriteLine("Total unread items: " + folderInfo.ContentUnreadCount);
    Console.WriteLine("-----------------------------------");
}
```

**Εξήγηση:**
- `folderInfo.DisplayName`: Ανακτά το όνομα του φακέλου.
- `folderInfo.ContentCount`: Παρέχει τον συνολικό αριθμό στοιχείων μέσα στον φάκελο.
- `folderInfo.ContentUnreadCount`: Δίνει τον αριθμό των μη αναγνωσμένων στοιχείων.

### Συμβουλές αντιμετώπισης προβλημάτων

- **Εξαίρεση "Δεν βρέθηκε αρχείο"**: Βεβαιωθείτε ότι το `dataDir` έχει ρυθμιστεί σωστά και δείχνει σε ένα υπάρχον αρχείο PST.
