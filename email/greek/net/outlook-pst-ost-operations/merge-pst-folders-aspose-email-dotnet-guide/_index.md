## Οδηγός Εφαρμογής

### Συγχώνευση φακέλων PST

Αυτή η λειτουργία δείχνει πώς να συγχωνεύσετε φακέλους από ένα αρχείο PST σε ένα άλλο χρησιμοποιώντας το Aspose.Email για .NET.

#### Βήμα προς βήμα διαδικασία

**1. Ορίστε τον κατάλογο εγγράφων σας**
Ορίστε τον κατάλογο εγγράφων σας όπου βρίσκονται τα αρχεία PST προέλευσης και προορισμού:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

**2. Ανοίξτε τα αρχεία PST προέλευσης και προορισμού**
Χρήση `PersonalStorage.FromFile` για να ανοίξετε και τα δύο αρχεία PST μέσα σε ένα `using` Δήλωση για την ορθή διαχείριση των πόρων:
```csharp
using (PersonalStorage destinationPst = PersonalStorage.FromFile(dataDir + "/destination.pst"))
using (PersonalStorage sourcePst = PersonalStorage.FromFile(dataDir + "/source.pst"))
{
    // Η υλοποίηση συνεχίζεται...
}
```

**3. Προσθέστε έναν νέο υποφάκελο στο PST προορισμού**
Δημιουργήστε έναν νέο φάκελο στο αρχείο PST προορισμού όπου θα συγχωνεύσετε τα περιεχόμενα από την πηγή:
```csharp
FolderInfo destinationFolder = destinationPst.RootFolder.AddSubFolder("MergedFolder");
```

**4. Ανάκτηση φακέλων από το αρχείο PST προέλευσης**
Πρόσβαση σε προκαθορισμένους φακέλους όπως `DeletedItems` για να επιδείξουν δυνατότητες συγχώνευσης:
```csharp
FolderInfo sourceFolder = sourcePst.GetPredefinedFolder(StandardIpmFolder.DeletedItems);
```

**5. Παρακολούθηση Μετακινημένων Αντικειμένων με Συνδρομή σε Εκδήλωση (Προαιρετικό)**
Για να παρακολουθείτε τα αντικείμενα που μετακινούνται, εγγραφείτε στο `ItemMoved` συμβάν:
```csharp
int totalAdded = 0;
destinationFolder.ItemMoved += (sender, e) => totalAdded++;
```

**6. Συγχώνευση φακέλου προέλευσης με τον φάκελο προορισμού**
Εκτελέστε τη λειτουργία συγχώνευσης:
```csharp
destinationFolder.MergeWith(sourceFolder);
```

### Χειρισμός συμβάντος μετακίνησης στοιχείου

Αυτή η προαιρετική λειτουργία παρακολουθεί και καταμετρά τα στοιχεία που μετακινούνται κατά τη διάρκεια της διαδικασίας συγχώνευσης.

#### Λεπτομέρειες Υλοποίησης

