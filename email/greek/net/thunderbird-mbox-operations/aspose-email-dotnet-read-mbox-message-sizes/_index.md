## Οδηγός Εφαρμογής

### Ανάγνωση μεγεθών μηνυμάτων MBOX
Αυτή η λειτουργία σάς επιτρέπει να διαβάσετε ένα αρχείο MBOX και να εξαγάγετε το μέγεθος κάθε μηνύματος ηλεκτρονικού ταχυδρομείου. 

#### Βήμα 1: Ρύθμιση διαδρομής αρχείου
Ξεκινήστε καθορίζοντας τη διαδρομή προς το αρχείο MBOX:

```csharp
string mboxFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "ExampleMbox.mbox");
```
**Γιατί;** Αυτή η διαδρομή υποδεικνύει πού αποθηκεύεται το αρχείο MBOX, το οποίο είναι κρίσιμο για την πρόσβαση στα email.

#### Βήμα 2: Ανοίξτε το αρχείο MBOX
Ανοίξτε το αρχείο MBOX χρησιμοποιώντας ένα `FileStream`:

```csharp
using (FileStream stream = new FileStream(mboxFilePath, FileMode.Open, FileAccess.Read))
{
    // Οι επόμενες λειτουργίες πηγαίνουν εδώ
}
```
**Γιατί;** Αυτό διασφαλίζει ότι το αρχείο είναι προσβάσιμο και μόνο για ανάγνωση, διατηρώντας την ακεραιότητα των δεδομένων.

#### Βήμα 3: Αρχικοποίηση του MboxrdStorageReader
Χρήση `MboxrdStorageReader` για να διαβάσετε μηνύματα:

```csharp
using (MboxrdStorageReader reader = new MboxrdStorageReader(stream, false))
{
    MailMessage msg;

    while ((msg = reader.ReadNextMessage()) != null)
    {
        long currentDataSize = reader.CurrentDataSize;
        Console.WriteLine($"Message Size: {currentDataSize} bytes");
        msg.Dispose();
    }
}
```
**Γιατί;** Αυτή η κλάση διευκολύνει την ανάγνωση κάθε μηνύματος διαδοχικά. Η απόρριψη των μηνυμάτων μετά την ανάγνωση είναι ζωτικής σημασίας για την αποτελεσματική διαχείριση της μνήμης.

### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι η διαδρομή του αρχείου MBOX είναι σωστή και προσβάσιμη.
- Ελέγξτε ότι το Aspose.Email έχει εγκατασταθεί σωστά στο έργο σας.
- Χειρισμός εξαιρέσεων για τον εντοπισμό σφαλμάτων κατά τη διάρκεια λειτουργιών αρχείων ή ανάλυσης μηνυμάτων.
