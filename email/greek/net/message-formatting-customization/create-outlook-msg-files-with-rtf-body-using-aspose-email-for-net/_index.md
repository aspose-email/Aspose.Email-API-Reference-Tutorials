## Οδηγός Εφαρμογής
### Δημιουργία αρχείων MSG με RTF Body
Αυτή η ενότητα εξηγεί πώς να μετατρέψετε ένα email που βασίζεται σε HTML σε μορφή MSG συμβατή με το Outlook χρησιμοποιώντας το Aspose.Email για .NET.
#### Βήμα 1: Ορισμός καταλόγων και διαδρομών αρχείων
Αρχικά, καθορίστε τους καταλόγους όπου θα αποθηκευτούν τα δεδομένα εισόδου και τα αρχεία εξόδου:
```csharp
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Outlook");
string outputFile = Path.Combine(dataDir, "CreatingMSGFilesWithRTFBody_out.msg");
```
#### Βήμα 2: Δημιουργία του μηνύματος ηλεκτρονικού ταχυδρομείου
Δημιουργήστε μια παρουσία του `MailMessage` και ορίστε τις ιδιότητές του όπως αποστολέα, παραλήπτη, θέμα και σώμα HTML:
```csharp
// Δημιουργήστε ένα νέο αντικείμενο MailMessage
MailMessage mailMsg = new MailMessage();

// Ορισμός βασικών ιδιοτήτων email
mailMsg.From = "from@domain.com";
mailMsg.To = "to@domain.com";
mailMsg.Subject = "subject";
mailMsg.HtmlBody = "<h3>rtf example</h3><p>creating an <b><u>outlook message (msg)</u></b> file using Aspose.Email.</p>";
```
#### Βήμα 3: Μετατροπή MailMessage σε MapiMessage
Για να μετατρέψετε το `MailMessage` σε μορφή συμβατή με αρχεία MSG του Outlook, χρησιμοποιήστε τον ακόλουθο κώδικα:
```csharp
// Μετατροπή του MailMessage σε αντικείμενο MapiMessage
MapiMessage outlookMsg = MapiMessage.FromMailMessage(mailMsg);
```
#### Βήμα 4: Αποθήκευση του αρχείου MSG
Τέλος, αποθηκεύστε το `MapiMessage` ως αρχείο MSG στον καθορισμένο κατάλογο:
```csharp
// Αποθήκευση του μηνύματος ως αρχείο .msg
outlookMsg.Save(outputFile);
```
