## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία ενσωμάτωσης εικόνων σε email χρησιμοποιώντας το Aspose.Email για .NET.

### Ορισμός διαδρομών αρχείων

Αρχικά, ορίστε τις διαδρομές αρχείων όπου θα αποθηκευτούν οι πόροι σας:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string dstEmail = dataDir + "/EmbeddedImage.msg";
```

### Δημιουργία μιας παρουσίας MailMessage

Ρυθμίστε τις βασικές ιδιότητες του email σας, συμπεριλαμβανομένων των αποστολέα, παραλήπτη και θέματος:

```csharp
MailMessage mail = new MailMessage();
mail.From = new MailAddress("test001@gmail.com");
mail.To.Add("test001@gmail.com");
mail.Subject = "This is an email";
```

### Δημιουργία του τμήματος απλού κειμένου

Δημιουργήστε μια προβολή απλού κειμένου του email σας για πελάτες που δεν υποστηρίζουν HTML:

```csharp
AlternateView plainView = AlternateView.CreateAlternateViewFromString(
    "This is my plain text content", null, "text/plain");
```

### Δημιουργία της προβολής HTML με ενσωματωμένη εικόνα

Δημιουργήστε μια έκδοση HTML του email σας και ενσωματώστε μια εικόνα χρησιμοποιώντας ένα Content ID (CID):

```csharp
string htmlContent = "Here is an embedded image.<img src='cid:barcode'>";
AlternateView htmlView = AlternateView.CreateAlternateViewFromString(
    htmlContent, null, "text/html");
```

### Ενσωμάτωση της εικόνας

Χρησιμοποιήστε το LinkedResource για να επισυνάψετε την εικόνα σας και να ορίσετε το ContentId της:

```csharp
LinkedResource barcode = new LinkedResource(dataDir + "/1.jpg", MediaTypeNames.Image.Jpeg)
{
    ContentId = "barcode"
};
mail.LinkedResources.Add(barcode);
```

Αυτό το βήμα είναι κρίσιμο, καθώς συσχετίζει την εικόνα με ένα συγκεκριμένο CID, επιτρέποντάς της να αναφέρεται στο περιεχόμενο HTML σας.

### Προσθήκη προβολών στο email

Επισυνάψτε και τις δύο προβολές (απλό κείμενο και HTML) στο μήνυμα ηλεκτρονικού ταχυδρομείου σας:

```csharp
mail.AlternateViews.Add(plainView);
mail.AlternateViews.Add(htmlView);
```

### Αποθήκευση του email

Τέλος, αποθηκεύστε το email σας με ενσωματωμένους πόρους σε μια καθορισμένη μορφή αρχείου:

```csharp
mail.Save(dataDir + "/EmbeddedImage_out.msg", SaveOptions.DefaultMsgUnicode);
```

Αυτό το βήμα διασφαλίζει ότι το email σας είναι έτοιμο για αποστολή ή περαιτέρω επεξεργασία.
