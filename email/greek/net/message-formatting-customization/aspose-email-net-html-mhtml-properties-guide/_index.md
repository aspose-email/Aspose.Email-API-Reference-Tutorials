## Οδηγός Εφαρμογής

Ας εξερευνήσουμε τις βασικές δυνατότητες που παρέχει το Aspose.Email για .NET.

### Ορισμός σώματος HTML σε μήνυμα ηλεκτρονικού ταχυδρομείου

**Επισκόπηση:** Η δημιουργία ενός πλούσιου σώματος HTML σάς επιτρέπει να δημιουργείτε οπτικά ελκυστικό περιεχόμενο email με μορφοποίηση, εικόνες και συνδέσμους.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργήστε ένα νέο αντικείμενο MailMessage**

```csharp
using Aspose.Email.Mime;

// Αρχικοποίηση του αντικειμένου μηνύματος ηλεκτρονικού ταχυδρομείου
MailMessage message = new MailMessage();
```

**2. Ορισμός περιεχομένου σώματος HTML**

```csharp
// Ορίστε το σώμα HTML
message.HtmlBody = "<html><body>This is the HTML body</body></html>";
```
- **Επεξήγηση παραμέτρων:** `HtmlBody` δέχεται μια συμβολοσειρά που αντιπροσωπεύει το περιεχόμενο HTML του email σας.

### Μετατροπή email σε μορφή MHTML

**Επισκόπηση:** Η μετατροπή των email σε μορφή MHTML επιτρέπει την αναπαράσταση ενός αρχείου που περιλαμβάνει όλους τους πόρους, διευκολύνοντας την αρχειοθέτηση και την προβολή.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργία και ρύθμιση παραμέτρων του MailMessage**

```csharp
using Aspose.Email.Storage.Mht;
using System.IO;

// Αρχικοποίηση του μηνύματος ηλεκτρονικού ταχυδρομείου με στοιχεία αποστολέα και παραλήπτη
MailMessage mailMsg = new MailMessage("from@example.com", "to@example.com");
mailMsg.Subject = "Email Subject";
mailMsg.Body = "This is the body of the email.";
```

**2. Μετατροπή σε MHTML**

```csharp
// Προετοιμασία ροής μνήμης για έξοδο
MemoryStream mhtmlStream = new MemoryStream();

// Αποθήκευση του μηνύματος σε μορφή MHTML
mailMsg.Save(mhtmlStream, SaveOptions.DefaultMhtml);
```
- **Διαμόρφωση κλειδιού:** `SaveOptions.DefaultMhtml` διασφαλίζει ότι όλοι οι πόροι περιλαμβάνονται στη μετατροπή.

### Φόρτωση και εμφάνιση ιδιοτήτων ηλεκτρονικού ταχυδρομείου

**Επισκόπηση:** Η φόρτωση ενός αρχείου email και η εμφάνιση των ιδιοτήτων του είναι χρήσιμη για σκοπούς εντοπισμού σφαλμάτων ή εξαγωγής δεδομένων.

#### Βήμα προς βήμα εφαρμογή

**1. Φόρτωση αρχείου email**

```csharp
using Aspose.Email;

// Φόρτωση του email από μια καθορισμένη διαδρομή
MailMessage loadedEmail = MailMessage.Load("YOUR_DOCUMENT_DIRECTORY\\example.eml");
```

**2. Εμφάνιση ιδιοτήτων ηλεκτρονικού ταχυδρομείου**

```csharp
// Εξαγωγή του θέματος και της διεύθυνσης του αποστολέα στην κονσόλα
Console.WriteLine(loadedEmail.Subject);
Console.WriteLine(loadedEmail.From.Address);
```
- **Επεξήγηση παραμέτρων:** `Load` διαβάζει ένα αρχείο email, ενώ ιδιότητες όπως `Subject` και `From` μπορεί να προσεγγιστεί άμεσα.
