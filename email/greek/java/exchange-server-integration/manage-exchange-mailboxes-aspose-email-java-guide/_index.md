## Οδηγός Εφαρμογής

### Σύνδεση με Exchange Server χρησιμοποιώντας EWS

Η σύνδεση σε έναν διακομιστή Exchange χρησιμοποιώντας το πρωτόκολλο EWS είναι απλή με το Aspose.Email για Java. Αυτή η λειτουργία σάς επιτρέπει να πραγματοποιήσετε έλεγχο ταυτότητας και να δημιουργήσετε μια συνεδρία.

#### Επισκόπηση
Χρησιμοποιώντας το `EWSClient.getEWSClient` μέθοδος, δημιουργήστε μια παρουσία του `IEWSClient`, το οποίο παρέχει πρόσβαση σε λειτουργίες γραμματοκιβωτίου.

#### Βήμα προς βήμα εφαρμογή

1. **Αρχικοποίηση σύνδεσης:**
   
   ```java
   import com.aspose.email.EWSClient;
   import com.aspose.email.IEWSClient;

   IEWSClient client = EWSClient.getEWSClient(
       "https://outlook.office365.com/exchangeews/exchange.asmx", 
       "testUser", 
       "pwd", 
       "domain"
   );
   ```
   - **Παράμετροι:**
     - URL του τελικού σημείου EWS του διακομιστή Exchange.
     - Όνομα χρήστη, κωδικός πρόσβασης και τομέας για έλεγχο ταυτότητας.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι ρυθμίσεις δικτύου σας επιτρέπουν συνδέσεις με την καθορισμένη διεύθυνση URL του διακομιστή Exchange.
- Επαληθεύστε ότι τα διαπιστευτήρια είναι σωστά και ότι έχετε τα κατάλληλα δικαιώματα.

### Ανάκτηση πληροφοριών γραμματοκιβωτίου

Η πρόσβαση στις λεπτομέρειες των γραμματοκιβωτίων μπορεί να είναι κρίσιμη για εφαρμογές που χρειάζονται πληροφορίες σχετικά με τα γραμματοκιβώτια των χρηστών.

#### Επισκόπηση
Ο `getMailboxInfo` Η μέθοδος ανακτά βασικές πληροφορίες όπως το URI Εισερχομένων, βοηθώντας σας να πλοηγηθείτε αποτελεσματικά στους φακέλους των γραμματοκιβωτίων.

#### Βήμα προς βήμα εφαρμογή

1. **Λήψη λεπτομερειών γραμματοκιβωτίου:**
   
   ```java
   import com.aspose.email.ExchangeMailboxInfo;

   ExchangeMailboxInfo mailboxInfo = client.getMailboxInfo();
   ```
   - Αυτή η κλήση επιστρέφει ένα `ExchangeMailboxInfo` αντικείμενο που περιέχει διάφορες ιδιότητες του γραμματοκιβωτίου του χρήστη.

### Λίστα μηνυμάτων στον φάκελο Εισερχομένων

Για να διαχειριστείτε ή να αναλύσετε μηνύματα ηλεκτρονικού ταχυδρομείου, ίσως χρειαστεί να καταχωρίσετε όλα τα μηνύματα σε έναν συγκεκριμένο φάκελο, όπως τα Εισερχόμενα.

#### Επισκόπηση
Ο `listMessages` Η μέθοδος ανακτά αντικείμενα πληροφοριών μηνύματος από καθορισμένα γραμματοκιβώτια ή φακέλους.

#### Βήμα προς βήμα εφαρμογή

1. **Λίστα μηνυμάτων εισερχομένων:**
   
   ```java
   import com.aspose.email.ExchangeMessageInfoCollection;

   ExchangeMessageInfoCollection msgInfoColl = client.listMessages(mailboxInfo.getInboxUri());

   for (com.aspose.email.ExchangeMessageInfo msgInfo : msgInfoColl) {
       // Επεξεργαστείτε κάθε μήνυμα όπως απαιτείται.
   }
   ```
   - **Παράμετροι:**
     - `getInboxUri()` Παρέχει το URI για την πρόσβαση στα μηνύματα στον φάκελο Εισερχόμενα.

### Διαγραφή συγκεκριμένων μηνυμάτων από τα Εισερχόμενα

Η αυτοματοποίηση της διαχείρισης email περιλαμβάνει τη διαγραφή μηνυμάτων με βάση συγκεκριμένα κριτήρια, όπως λέξεις-κλειδιά θέματος.

#### Επισκόπηση
Επαναλάβετε τα μηνύματα του γραμματοκιβωτίου και διαγράψτε αυτά που πληρούν συγκεκριμένες προϋποθέσεις χρησιμοποιώντας το `deleteItem` μέθοδος.

#### Βήμα προς βήμα εφαρμογή

1. **Διαγραφή στοχευμένων μηνυμάτων:**
   
   ```java
   import com.aspose.email.DeletionOptions;

   for (com.aspose.email.ExchangeMessageInfo msgInfo : msgInfoColl) {
       if (msgInfo.getSubject() != null && msgInfo.getSubject().contains("delete")) {
           client.deleteItem(msgInfo.getUniqueUri(), DeletionOptions.getDeletePermanently());
       }
   }
   ```
   - **Παράμετροι:**
     - `getUniqueUri()` ανακτά το μοναδικό αναγνωριστικό του μηνύματος.
     - Χρήση `DeletionOptions` για οριστική διαγραφή.
