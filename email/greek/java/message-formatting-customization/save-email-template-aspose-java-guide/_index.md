### Οδηγός Εφαρμογής

Ας εξερευνήσουμε πώς να αποθηκεύσουμε ένα email ως πρότυπο.

#### Δημιουργία και αποθήκευση προτύπου email

**Επισκόπηση:** Αυτή η ενότητα καλύπτει τη δημιουργία ενός `MailMessage` ένα παράδειγμα με λεπτομέρειες αποστολέα, παραλήπτη, θέματος και σώματος πριν το αποθηκεύσετε σε μορφή OFT.

**Βήμα 1: Δημιουργία MailMessage**

Ξεκινάμε αρχικοποιώντας το `MailMessage` αντικείμενο:

```java
import com.aspose.email.MailMessage;
import com.aspose.email.MsgSaveOptions;

// Αρχικοποίηση μιας νέας παρουσίας MailMessage
MailMessage eml = new MailMessage("test@from.to", "test@to.com");
eml.setSubject("Test Email Template");
eml.setBody("This is an example email body.");
```

**Βήμα 2: Αποθήκευση ως OFT**

Για να αποθηκεύσετε αυτό το μήνυμα σε μορφή OFT, χρησιμοποιήστε `MsgSaveOptions`:

```java
// Ορισμός επιλογών αποθήκευσης για τη μορφή OFT
MsgSaveOptions saveOptions = SaveOptions.getDefaultOft();

// Αποθήκευση του MailMessage σε μορφή OFT
eml.save("output.oft", saveOptions);
```

**Εξήγηση:** 
- **MailMessage**: Αυτή η κλάση ενσωματώνει ένα μήνυμα ηλεκτρονικού ταχυδρομείου, συμπεριλαμβανομένων λεπτομερειών όπως αποστολέας, παραλήπτης, θέμα και σώμα.
- **Επιλογές Αποθήκευσης Μηνυμάτων**Παρέχει επιλογές για την αποθήκευση μηνυμάτων σε διαφορετικές μορφές. Εδώ χρησιμοποιούμε `getDefaultOft()` για να καθορίσετε τη μορφή OFT.
