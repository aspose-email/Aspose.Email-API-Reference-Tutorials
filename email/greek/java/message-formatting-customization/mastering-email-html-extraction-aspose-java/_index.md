## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στην εξαγωγή κειμένου HTML από email χρησιμοποιώντας το Aspose.Email για Java. Θα επικεντρωθούμε σε δύο κύριες λειτουργίες: τη συμπερίληψη και την εξαίρεσή τους από τις διευθύνσεις URL.

### Εξαγωγή σώματος HTML με διευθύνσεις URL

#### Επισκόπηση

Σε αυτήν τη λειτουργία, εξάγουμε το περιεχόμενο HTML ενός email διατηρώντας παράλληλα τυχόν ενσωματωμένες διευθύνσεις URL. Αυτό είναι ιδιαίτερα χρήσιμο όταν οι σύνδεσμοι αποτελούν μέρος των αναγκών σας σε ανάλυση ή αναφορές.

#### Βήματα Υλοποίησης

1. **Φόρτωση email ως αντικειμένου MailMessage:**
   
   Υποθέτω `mail` έχει ήδη φορτωθεί ως `MailMessage` αντικείμενο.

2. **Εξαγωγή σώματος HTML που περιλαμβάνει διευθύνσεις URL:**

   Χρησιμοποιήστε το `getHtmlBodyText()` μέθοδος με `true` για να συμπεριλάβετε URL:

   ```java
   // Εξαγωγή κειμένου σώματος HTML, συμπεριλαμβανομένων των URL.
   String body_with_url = mail.getHtmlBodyText(true);
   ```

   - **Εξήγηση παραμέτρου:** 
     - Η λογική παράμετρος `true` σηματοδοτεί ότι οι διευθύνσεις URL θα πρέπει να διατηρηθούν στην έξοδο.

### Εξαγωγή σώματος HTML χωρίς διευθύνσεις URL

#### Επισκόπηση

Αυτή η λειτουργία εστιάζει στην εξαγωγή μόνο του κειμένου από το σώμα HTML ενός email, εξαιρουμένων τυχόν ενσωματωμένων URL. Αυτό είναι χρήσιμο για την ανάλυση κειμένου ή όταν οι σύνδεσμοι δεν είναι σχετικοί με τις ανάγκες σας.

#### Βήματα Υλοποίησης

1. **Εξαγωγή σώματος HTML εξαιρουμένων των διευθύνσεων URL:**

   Χρησιμοποιήστε το `getHtmlBodyText()` μέθοδος με `false`:

   ```java
   // Εξαγωγή κειμένου HTML χωρίς να συμπεριλάβετε URL.
   String body_without_url = mail.getHtmlBodyText(false);
   ```

   - **Εξήγηση παραμέτρου:** 
     - Η λογική παράμετρος `false` υποδεικνύει ότι οι διευθύνσεις URL πρέπει να παραλειφθούν από την έξοδο.
