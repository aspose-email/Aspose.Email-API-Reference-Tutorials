## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει την απόδοση υπερσυνδέσμων με και χωρίς `href` γνωρίσματα.

### Προσαρμοσμένη απόδοση υπερσυνδέσμων με Href

#### Επισκόπηση
Βελτιώστε την ασφάλεια και τη χρηστικότητα των συνδέσμων συμπεριλαμβάνοντας το `href` χαρακτηριστικό στο σώμα HTML ενός email. Αυτή η προσέγγιση διατηρεί την ακεραιότητα των υπερσυνδέσμων.

#### Βήματα Υλοποίησης

##### Βήμα 1: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου
Φορτώστε το μήνυμα ηλεκτρονικού ταχυδρομείου σας από ένα αρχείο:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String fileName = dataDir + "LinksSample.eml";
MailMessage msg = MailMessage.load(fileName);
```

##### Βήμα 2: Απόδοση υπερσυνδέσμων με Href
Υλοποιήστε ένα `HyperlinkRenderingCallback` για την επεξεργασία υπερσυνδέσμων και την συμπερίληψη του `href` ιδιότης:

```java
String htmlTextHref = msg.getHtmlBodyText(new HyperlinkRenderingCallback() {
    @Override
    public String invoke(String source) {
        return renderHyperlinkWithHref(source);
    }
});
```

##### Βήμα 3: Εξαγωγή και μορφοποίηση υπερσυνδέσμου
Δημιουργήστε μια μέθοδο για την εξαγωγή του `href` χαρακτηριστικό και μορφοποίησέ το:

```java
private static String renderHyperlinkWithHref(String source) {
    int start = source.indexOf("href=\"") + "href=\"".length();
    int end = source.indexOf(\"", start);
    String href = source.substring(start, end);

    start = source.indexOf(">") + 1;
    end = source.indexOf("<", start);
    String text = source.substring(start, end);

    return text + "<" + href + ">";
}
```
**Εξήγηση**: Αυτή η μέθοδος αναγνωρίζει και εξάγει το `href` από μια ετικέτα υπερσυνδέσμου. Κατασκευάζει μια μορφοποιημένη συμβολοσειρά τόσο με το κείμενο του συνδέσμου όσο και με τη διεύθυνση URL του.

### Προσαρμοσμένη απόδοση υπερσυνδέσμων χωρίς Href

#### Επισκόπηση
Εξαιρέστε το `href` χαρακτηριστικό για βελτίωση της ασφάλειας ή όταν απαιτείται μόνο η εμφάνιση κειμένου συνδέσμου.

#### Βήματα Υλοποίησης

##### Βήμα 1: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου
Φορτώστε το μήνυμα ηλεκτρονικού ταχυδρομείου σας:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String fileName = dataDir + "LinksSample.eml";
MailMessage msg = MailMessage.load(fileName);
```

##### Βήμα 2: Απόδοση υπερσυνδέσμων χωρίς Href
Χρησιμοποιήστε ένα `HyperlinkRenderingCallback` να αποκλείσει το `href` ιδιότης:

```java
String htmlTextHrefLess = msg.getHtmlBodyText(new HyperlinkRenderingCallback() {
    @Override
    public String invoke(String source) {
        return renderHyperlinkWithoutHref(source);
    }
});
```

##### Βήμα 3: Εξαγωγή και μορφοποίηση υπερσυνδέσμου
Υλοποιήστε τη μέθοδο για τη μορφοποίηση υπερσυνδέσμων χωρίς `href`:

```java
private static String renderHyperlinkWithoutHref(String source) {
    int start = source.indexOf(">") + 1;
    int end = source.indexOf("<", start);
    return source.substring(start, end);
}
```
**Εξήγηση**Αυτή η μέθοδος ανακτά μόνο το ορατό κείμενο ενός υπερσυνδέσμου εξαιρώντας το `href` ιδιότης.
