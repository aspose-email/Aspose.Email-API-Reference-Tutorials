- Κατάργηση συγκεκριμένων ή όλων των κατηγοριών από ένα μήνυμα ηλεκτρονικού ταχυδρομείου
- Ρύθμιση του Aspose.Email για Java στο περιβάλλον σας

Για να ενσωματώσετε το Aspose.Email στο έργο Java σας, μπορείτε να χρησιμοποιήσετε την ακόλουθη εξάρτηση Maven:

```xml

    <groupId>com.aspose</groupId>
    <artifactId>aspose-email</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

## Οδηγός Εφαρμογής
Θα εξερευνήσουμε κάθε λειτουργία βήμα προς βήμα: προσθήκη κατηγοριών, ανάκτηση τους, κατάργηση συγκεκριμένων και διαγραφή όλων των κατηγοριών από ένα μήνυμα του Outlook.
### Προσθήκη κατηγοριών σε ένα μήνυμα του Outlook
Η προσθήκη κατηγοριών βοηθά στην αποτελεσματική οργάνωση των email. Δείτε πώς μπορείτε να το κάνετε:
#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει την προσθήκη πολλαπλών κατηγοριών σε ένα μόνο μήνυμα ηλεκτρονικού ταχυδρομείου Outlook.
#### Βήματα
1. **Φόρτωση του email**
   
   ```java
   import com.aspose.email.MapiMessage;
   
   MapiMessage msg = MapiMessage.fromFile("YOUR_DOCUMENT_DIRECTORY/message.msg");
   ```
2. **Προσθήκη κατηγοριών**
   
   Χρήση `FollowUpManager.addCategory` για να αντιστοιχίσετε κατηγορίες.

   ```java
   import com.aspose.email.FollowUpManager;

   FollowUpManager.addCategory(msg, "Purple Category");
   FollowUpManager.addCategory(msg, "Red Category");
   ```
#### Εξήγηση
- Ο `MapiMessage.fromFile()` Η μέθοδος φορτώνει το μήνυμα του Outlook από μια καθορισμένη διαδρομή αρχείου.
- `FollowUpManager.addCategory()` προσθέτει το καθορισμένο όνομα κατηγορίας στο μήνυμα ηλεκτρονικού ταχυδρομείου.
### Ανάκτηση κατηγοριών από ένα μήνυμα του Outlook
Για να ανακτήσετε κατηγορίες που έχουν αντιστοιχιστεί σε ένα email:
#### Επισκόπηση
Αυτή η λειτουργία ανακτά όλες τις κατηγορίες που συνδέονται με ένα συγκεκριμένο μήνυμα ηλεκτρονικού ταχυδρομείου.
#### Βήματα
1. **Φόρτωση του email**
   
   ```java
   MapiMessage msg = MapiMessage.fromFile("YOUR_DOCUMENT_DIRECTORY/message.msg");
   ```
2. **Ανάκτηση κατηγοριών**
   
   ```java
   import com.aspose.email.system.collections.IList;

   IList categories = FollowUpManager.getCategories(msg);
   // Έξοδος: Αυτό θα σας δώσει τη λίστα κατηγοριών.
   ```
#### Εξήγηση
- `FollowUpManager.getCategories()` Επιστρέφει μια λίστα που περιέχει όλες τις κατηγορίες που είναι συνημμένες στο μήνυμα ηλεκτρονικού ταχυδρομείου.
### Κατάργηση συγκεκριμένης κατηγορίας από ένα μήνυμα του Outlook
Εάν χρειάζεται να καταργήσετε συγκεκριμένες κατηγορίες:
#### Επισκόπηση
Αυτή η λειτουργία καταργεί τις καθορισμένες κατηγορίες, συμβάλλοντας στη διατήρηση της συνάφειας και της σαφήνειας στην κατηγοριοποίηση των μηνυμάτων σας.
#### Βήματα
1. **Φόρτωση του email**
   
   ```java
   MapiMessage msg = MapiMessage.fromFile("YOUR_DOCUMENT_DIRECTORY/message.msg");
   ```
2. **Κατάργηση κατηγορίας**
   
   ```java
   FollowUpManager.removeCategory(msg, "Red Category");
   ```
#### Εξήγηση
- `FollowUpManager.removeCategory()` καταργεί την καθορισμένη κατηγορία από το email σας.
### Διαγραφή όλων των κατηγοριών από ένα μήνυμα του Outlook
Για να καταργήσετε όλες τις κατηγορίες ταυτόχρονα:
#### Επισκόπηση
Αυτή η λειτουργία διαγράφει κάθε κατηγορία που έχει αντιστοιχιστεί σε ένα μήνυμα για την πλήρη κατάργηση των ετικετών.
#### Βήματα
1. **Φόρτωση του email**
   
   ```java
   MapiMessage msg = MapiMessage.fromFile("YOUR_DOCUMENT_DIRECTORY/message.msg");
   ```
2. **Εκκαθάριση όλων των κατηγοριών**
   
   ```java
   FollowUpManager.clearCategories(msg);
   ```
#### Εξήγηση
- `FollowUpManager.clearCategories()` διαγράφει όλες τις κατηγορίες από το μήνυμα.
