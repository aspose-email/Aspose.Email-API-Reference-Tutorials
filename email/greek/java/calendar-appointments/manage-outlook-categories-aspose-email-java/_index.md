# Διαχείριση Κατηγοριών Outlook με το Aspose.Email για Java

## Εισαγωγή
Σε αυτό το tutorial θα μάθετε πώς να **διαχειρίζεστε κατηγορίες Outlook** με το Aspose.Email για Java. Η διαχείριση των κατηγοριών στα μηνύματα Outlook μπορεί να βελτιώσει σημαντικά την οργάνωση και την αποδοτικότητα ανάκτησης—ιδιαίτερα όταν χειρίζεστε μεγάλο όγκο email. Με το **Aspose.Email για Java**, μπορείτε εύκολα να προσθέσετε, να ανακτήσετε και να **αφαιρέσετε ετικέτες κατηγοριών Outlook** από τα μηνύματα Outlook προγραμματιστικά. Αυτός ο οδηγός καλύπτει επίσης πώς να **καθαρίσετε όλες τις κατηγορίες Outlook** όταν χρειάζεστε ένα καθαρό ξεκίνημα.

### Τι θα μάθετε
- Πώς να προσθέτετε κατηγορίες σε ένα μήνυμα Outlook  
- Πώς να ανακτάτε τη λίστα των εκχωρημένων κατηγοριών  
- Πώς να αφαιρείτε συγκεκριμένες ή όλες τις κατηγορίες από ένα email  
- Πώς να ρυθμίσετε το Aspose.Email για Java στο περιβάλλον σας  

Έτοιμοι να βελτιώσετε τη διαχείριση των email σας; Ας ξεκινήσουμε με τις προαπαιτούμενες ρυθμίσεις!

## Οδηγός Υλοποίησης
Θα εξετάσουμε κάθε λειτουργία βήμα‑βήμα: προσθήκη κατηγοριών, ανάκτηση, αφαίρεση συγκεκριμένων και εκκαθάριση όλων των κατηγοριών από ένα μήνυμα Outlook.

### Προσθήκη Κατηγοριών σε Μήνυμα Outlook
Η προσθήκη κατηγοριών βοηθά στην αποτελεσματική οργάνωση των email.

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να προσθέσετε πολλαπλές κατηγορίες σε ένα μόνο email Outlook.

#### Βήματα
1. **Φόρτωση του Email**

   ```java
   import com.aspose.email.MapiMessage;
   
   MapiMessage msg = MapiMessage.fromFile("YOUR_DOCUMENT_DIRECTORY/message.msg");
   ```

2. **Προσθήκη Κατηγοριών**

   Χρησιμοποιήστε `FollowUpManager.addCategory` για να εκχωρήσετε κατηγορίες.

   ```java
   import com.aspose.email.FollowUpManager;

   FollowUpManager.addCategory(msg, "Purple Category");
   FollowUpManager.addCategory(msg, "Red Category");
   ```

#### Επεξήγηση
- Η μέθοδος `MapiMessage.fromFile()` φορτώνει το μήνυμα Outlook από το καθορισμένο μονοπάτι αρχείου.  
- Η `FollowUpManager.addCategory()` προσθέτει το όνομα της καθορισμένης κατηγορίας στο email.

### Ανάκτηση Κατηγοριών από Μήνυμα Outlook
Για να ανακτήσετε τις κατηγορίες που έχουν εκχωρηθεί σε ένα email:

#### Επισκόπηση
Αυτή η λειτουργία φέρνει όλες τις κατηγορίες που συνδέονται με ένα συγκεκριμένο μήνυμα.

#### Βήματα
1. **Φόρτωση του Email**

   ```java
   MapiMessage msg = MapiMessage.fromFile("YOUR_DOCUMENT_DIRECTORY/message.msg");
   ```

2. **Ανάκτηση Κατηγοριών**

   ```java
   import com.aspose.email.system.collections.IList;

   IList categories = FollowUpManager.getCategories(msg);
   // Output: This will give you the list of categories.
   ```

#### Επεξήγηση
- Η `FollowUpManager.getCategories()` επιστρέφει μια λίστα που περιέχει όλες τις κατηγορίες που είναι συνδεδεμένες με το email.

### Αφαίρεση Συγκεκριμένης Κατηγορίας από Μήνυμα Outlook
Εάν χρειάζεται να **αφαιρέσετε ετικέτες κατηγοριών Outlook**, ακολουθήστε τα παρακάτω βήματα:

#### Επισκόπηση
Αυτή η λειτουργία αφαιρεί τις καθορισμένες κατηγορίες, βοηθώντας στη διατήρηση της σχετικότητας και της σαφήνειας στην οργάνωση των μηνυμάτων σας.

#### Βήματα
1. **Φόρτωση του Email**

   ```java
   MapiMessage msg = MapiMessage.fromFile("YOUR_DOCUMENT_DIRECTORY/message.msg");
   ```

2. **Αφαίρεση Κατηγορίας**

   ```java
   FollowUpManager.removeCategory(msg, "Red Category");
   ```

#### Επεξήγηση
- Η `FollowUpManager.removeCategory()` αφαιρεί την καθορισμένη κατηγορία από το email σας.

### Εκκαθάριση Όλων των Κατηγοριών από Μήνυμα Outlook
Όταν χρειάζεται να **καθαρίσετε όλες τις κατηγορίες Outlook**, χρησιμοποιήστε την παρακάτω μέθοδο:

#### Επισκόπηση
Αυτή η λειτουργία αφαιρεί κάθε κατηγορία που έχει εκχωρηθεί σε ένα μήνυμα, εξασφαλίζοντας πλήρη αφαίρεση των ετικετών.

#### Βήματα
1. **Φόρτωση του Email**

   ```java
   MapiMessage msg = MapiMessage.fromFile("YOUR_DOCUMENT_DIRECTORY/message.msg");
   ```

2. **Καθαρισμός Όλων των Κατηγοριών**

   ```java
   FollowUpManager.clearCategories(msg);
   ```

#### Επεξήγηση
- Η `FollowUpManager.clearCategories()` διαγράφει όλες τις κατηγορίες από το μήνυμα.
