### Λειτουργία 1: Δημιουργία και ρύθμιση παραμέτρων συμβάντος ημερολογίου

#### Επισκόπηση
Η δημιουργία μιας συνάντησης στο ημερολόγιο MAPI περιλαμβάνει τον ορισμό των ωρών έναρξης και λήξης, μαζί με λεπτομέρειες όπως η τοποθεσία, το θέμα και η περιγραφή.

##### Βήμα προς βήμα εφαρμογή

**Ορισμός ημερομηνιών έναρξης και λήξης**

Ξεκινήστε ορίζοντας τις ημερομηνίες έναρξης και λήξης του συμβάντος:

```java
import com.aspose.email.MapiCalendar;
import java.util.Calendar;
import java.util.Date;

public MapiCalendar createAppointment() {
    Calendar cal = Calendar.getInstance();
    
    // Ορισμός της ημερομηνίας έναρξης
    cal.set(Calendar.YEAR, 2023);
    cal.set(Calendar.MONTH, Calendar.OCTOBER);
    cal.set(Calendar.DAY_OF_MONTH, 1);
    Date startDate = cal.getTime();
    
    // Ορισμός της ημερομηνίας λήξης
    cal.set(Calendar.HOUR_OF_DAY, 10);
    Date endDate = cal.getTime();
    
    return new MapiCalendar("Conference Room", "Important Meeting",
        "Discuss project milestones and updates.", startDate, endDate);
}
```

**Εξήγηση**: Αυτό το απόσπασμα κώδικα δημιουργεί ένα `MapiCalendar` παράδειγμα με καθορισμένες ημερομηνίες έναρξης και λήξης. Οι παράμετροι περιλαμβάνουν την τοποθεσία, το θέμα και την περιγραφή του συμβάντος.

### Λειτουργία 2: Προσθήκη συμμετεχόντων στη σύσκεψη

#### Επισκόπηση
Η προσθήκη συμμετεχόντων είναι απαραίτητη για να διασφαλιστεί ότι όλοι λαμβάνουν ειδοποιήσεις και μπορούν να συμμετάσχουν στην εκδήλωση.

##### Βήμα προς βήμα εφαρμογή

**Αρχικοποίηση συλλογής παραληπτών**

Για να διαχειριστείτε τους συμμετέχοντες στη σύσκεψη, αρχικοποιήστε ένα `MapiRecipientCollection`:

```java
import com.aspose.email.MapiCalendar;
import com.aspose.email.MapiRecipientCollection;
import com.aspose.email.MapiRecipientType;
import java.util.Date;

public MapiCalendar createMeetingWithAttendees(Date startDate, Date endDate) {
    MapiRecipientCollection attendees = new MapiRecipientCollection();
    
    // Προσθήκη κύριων παραληπτών
    attendees.add("attendee1@example.com", "John Doe", MapiRecipientType.MAPI_TO);
    attendees.add("attendee2@example.com", "Jane Smith", MapiRecipientType.MAPI_TO);
    
    return new MapiCalendar(
        "Main Office Boardroom",
        "Team Meeting",
        "Discuss quarterly goals.",
        startDate,
        endDate,
        "organizer@example.com",
        attendees
    );
}
```

**Εξήγηση**Αυτός ο κώδικας δημιουργεί μια λίστα με τους κύριους παραλήπτες καθορίζοντας τις διευθύνσεις ηλεκτρονικού ταχυδρομείου και τα εμφανιζόμενα ονόματά τους, διασφαλίζοντας ότι θα ειδοποιηθούν για το συμβάν.

### Λειτουργία 3: Δημιουργία και αποθήκευση σε αρχείο PST

#### Επισκόπηση
Η αποθήκευση συμβάντων ημερολογίου σε ένα αρχείο PST επιτρέπει την εύκολη κοινή χρήση και ενσωμάτωση με άλλα συστήματα.

##### Βήμα προς βήμα εφαρμογή

**Δημιουργία PST και προσθήκη συμβάντων**

Δείτε πώς μπορείτε να δημιουργήσετε ένα αρχείο PST και να προσθέσετε τα συμβάντα σας:

```java
import com.aspose.email.FileFormatVersion;
import com.aspose.email.FolderInfo;
import com.aspose.email.PersonalStorage;
import com.aspose.email.StandardIpmFolder;

public void createPSTWithCalendarEvents() {
    String pstFilePath = "/path/to/output/MapiCalendarToPST_out.pst";
    
    PersonalStorage pst = PersonalStorage.create(pstFilePath, FileFormatVersion.Unicode);
    FolderInfo calendarFolder = pst.createPredefinedFolder("Calendar", StandardIpmFolder.Appointments);

    MapiCalendar appointment = createAppointment();
    calendarFolder.addMapiMessageItem(appointment);
    
    Date startDate = new Date(); // Χρησιμοποιήστε πραγματικές ημερομηνίες από την εκδήλωσή σας
    Date endDate = new Date();
    MapiCalendar meeting = createMeetingWithAttendees(startDate, endDate);
    calendarFolder.addMapiMessageItem(meeting);
}
```

**Εξήγηση**Αυτό το απόσπασμα δείχνει τη δημιουργία ενός αρχείου PST σε μορφή Unicode και την προσθήκη ενός ραντεβού και μιας σύσκεψης σε αυτό. Διευκολύνει την οργανωμένη αποθήκευση συμβάντων ημερολογίου.
