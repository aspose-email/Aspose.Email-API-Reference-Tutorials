### Δημιουργία μηνύματος απάντησης από ένα υπάρχον αρχείο MSG

#### Επισκόπηση

Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα email απάντησης χρησιμοποιώντας περιεχόμενο από ένα υπάρχον αρχείο MSG. Αυτό μπορεί να είναι ιδιαίτερα χρήσιμο κατά την αυτοματοποίηση απαντήσεων στην εξυπηρέτηση πελατών ή στις εσωτερικές επικοινωνίες.

#### Βήματα

**1. Φορτώστε το αρχικό μήνυμα**

Αρχικά, φορτώστε το αρχικό αρχείο MSG σε ένα `MapiMessage` αντικείμενο:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MapiMessage originalMsg = MapiMessage.fromFile(dataDir + "message1.msg");
```

**2. Αρχικοποιήστε το ReplyBuilder**

Ρυθμίστε το `ReplyMessageBuilder`, το οποίο σας επιτρέπει να διαμορφώσετε τον τρόπο με τον οποίο θα κατασκευαστεί η απάντηση.

```java
ReplyMessageBuilder builder = new ReplyMessageBuilder();
builder.setReplyAll(true); // Στείλτε την απάντηση σε όλους τους παραλήπτες του αρχικού μηνύματος.
builder.setAdditionMode(OriginalMessageAdditionMode.Textpart); // Προσθέστε το αρχικό περιεχόμενο του μηνύματος σε λειτουργία κειμένου.
```

**3. Ορισμός περιεχομένου απάντησης**

Καθορίστε το περιεχόμενο HTML για την απάντησή σας:

```java
builder.setResponseText(
    "<p><b>Dear Friend,</b></p>" +
    "I want to introduce my co-author and co-teacher." +
    "<p><a href=\"www.google.com\">This is a first link</a></p>" +
    "<p><a href=\"www.google.com\">This is a second link</a></p>"
);
```

**4. Δημιουργήστε και αποθηκεύστε το μήνυμα απάντησης**

Δημιουργήστε το μήνυμα απάντησης και αποθηκεύστε το στην επιθυμητή τοποθεσία:

```java
MapiMessage replyMsg = builder.buildResponse(originalMsg);
String outputDir = "YOUR_OUTPUT_DIRECTORY/";
replyMsg.save(outputDir + "reply_out.msg");
```

### Δημιουργία μηνύματος προώθησης από ένα υπάρχον αρχείο MSG

#### Επισκόπηση

Η προώθηση email είναι μια άλλη συνηθισμένη εργασία που μπορεί να αυτοματοποιηθεί χρησιμοποιώντας το Aspose.Email. Αυτή η λειτουργία σάς επιτρέπει να προωθήσετε το περιεχόμενο ενός υπάρχοντος email σε νέους παραλήπτες.

#### Βήματα

**1. Φορτώστε το αρχικό μήνυμα**

Όπως και με τη λειτουργία απάντησης, φορτώστε το αρχικό σας μήνυμα:

```java
MapiMessage originalMsg = MapiMessage.fromFile(dataDir + "message1.msg");
```

**2. Αρχικοποίηση του ForwardBuilder**

Ρυθμίστε το `ForwardMessageBuilder` και διαμορφώστε το όπως απαιτείται.

```java
ForwardMessageBuilder builder = new ForwardMessageBuilder();
builder.setAdditionMode(OriginalMessageAdditionMode.Textpart); // Συμπεριλάβετε το αρχικό περιεχόμενο του μηνύματος.
```

**3. Δημιουργήστε και αποθηκεύστε το μήνυμα προώθησης**

Δημιουργήστε το προωθημένο μήνυμα και αποθηκεύστε το:

```java
MapiMessage forwardMsg = builder.buildResponse(originalMsg);
forwardMsg.save(outputDir + "forward_out.msg");
```
