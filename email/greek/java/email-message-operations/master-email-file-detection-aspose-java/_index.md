### Ανίχνευση Μορφής Αρχείου Email
**Επισκόπηση**: Αυτή η δυνατότητα σας επιτρέπει να προσδιορίσετε τη μορφή ενός αρχείου email (π.χ. MSG, EML) χρησιμοποιώντας το `FileFormatUtil`.

#### Βήμα 1: Καθορισμός Καταλόγου Εγγράφων
Αρχικά, ορίστε τη διαδρομή όπου αποθηκεύονται τα αρχεία email σας. Αντικαταστήστε το `YOUR_DOCUMENT_DIRECTORY` με την πραγματική διαδρομή του καταλόγου σας:
```java
String dataDir = YOUR_DOCUMENT_DIRECTORY + "email/Message.msg";
```

**Εξήγηση**: Αυτό το βήμα ορίζει τη διαδρομή αρχείου για την ανίχνευση.

#### Βήμα 2: Ανίχνευση Μορφής Αρχείου
Χρησιμοποιήστε το `FileFormatUtil.detectFileFormat()` για να εντοπίσετε τη μορφή του email:
```java
FileFormatInfo info = FileFormatUtil.detectFileFormat(dataDir);
```

**Γιατί**: Αυτή η μέθοδος επιστρέφει ένα αντικείμενο `FileFormatInfo` που περιέχει λεπτομέρειες για τη μορφή του αρχείου, κρίσιμες για την επόμενη επεξεργασία.

#### Βήμα 3: Ανάκτηση και Εκτύπωση Τύπου Μορφής
Τέλος, εξάγετε και εμφανίστε τη μορφή του εντοπισμένου email:
```java
System.out.println("The message format is: " + info.getFileFormatType());
```

**Σκοπός**: Εκτυπώνοντας τον τύπο μορφής, επιβεβαιώνετε ότι η λογική ανίχνευσης αρχείων λειτουργεί σωστά.
