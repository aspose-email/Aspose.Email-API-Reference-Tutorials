## Ρύθμιση του Aspose.Email για Java
Για να χρησιμοποιήσετε το Aspose.Email για Java, προσθέστε το ως εξάρτηση στο έργο Maven σας:
**Εξάρτηση Maven**

## Οδηγός Εφαρμογής
### Ανάκτηση πληροφοριών σύνοψης μηνύματος χρησιμοποιώντας τον αριθμό ακολουθίας
Αυτή η λειτουργία σάς επιτρέπει να έχετε πρόσβαση στις λεπτομέρειες του μηνύματος καθορίζοντας έναν αριθμό ακολουθίας που έχει εκχωρηθεί από τον διακομιστή.
#### Αρχικοποίηση του Πελάτη σας
Δημιουργήστε μια παρουσία του `Pop3Client` και ορίστε τις επιλογές ασφαλείας:
```java
Pop3Client client = new Pop3Client("host.domain.com", 456, "username", "password");
client.setSecurityOptions(SecurityOptions.Auto);
```
#### Ανάκτηση πληροφοριών μηνύματος χρησιμοποιώντας τον αριθμό ακολουθίας
Ανάκτηση λεπτομερειών σύνοψης μηνύματος χρησιμοποιώντας τον αριθμό ακολουθίας:
```java
String seqNum = "sequence number of a message from server";
Pop3MessageInfo messageInfo = client.getMessageInfo(seqNum);

if (messageInfo != null) {
    String subject = messageInfo.getSubject();
    String date = messageInfo.getDate();

    System.out.println("Subject: " + subject);
    System.out.println("Date: " + date);
}
```
- **Παράμετροι:** Αριθμός ακολουθίας ως αναγνωριστικό συμβολοσειράς.
- **Επιστρεφόμενη τιμή:** `Pop3MessageInfo` αντικείμενο που περιέχει τα στοιχεία του email.
### Ανάκτηση πληροφοριών σύνοψης μηνύματος χρησιμοποιώντας μοναδικό αναγνωριστικό
Αυτή η λειτουργία είναι παρόμοια, αλλά χρησιμοποιεί ένα μοναδικό αναγνωριστικό αντί για αριθμούς ακολουθίας για την ανάκτηση περιλήψεων μηνυμάτων.
#### Ανάκτηση πληροφοριών μηνύματος χρησιμοποιώντας μοναδικό αναγνωριστικό
Πρόσβαση στις πληροφορίες μηνύματος μέσω του μοναδικού αναγνωριστικού του:
```java

if (messageInfo != null) {

    String date = messageInfo.getDate();

}
```
- **Παράμετροι:** Μοναδικό αναγνωριστικό ως αναγνωριστικό συμβολοσειράς.
- **Επιστρεφόμενη τιμή:** `Pop3MessageInfo` αντικείμενο που περιέχει τα στοιχεία του email.
