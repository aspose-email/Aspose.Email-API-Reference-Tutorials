### Ανάγνωση μηνυμάτων από την αποθήκευση NSF

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να διαβάζετε μηνύματα email που είναι αποθηκευμένα σε ένα αρχείο NSF χρησιμοποιώντας το Aspose.Email για Java. 

##### 1. Απαραίτητες εισαγωγές
Βεβαιωθείτε ότι περιλαμβάνονται οι ακόλουθες εισαγωγές:
```java

import com.aspose.email.NotesStorageFacility;
```

##### 2. Ορίστε τη διαδρομή προς το αρχείο NSF σας
Ορίστε μια διαδρομή καταλόγου όπου αποθηκεύονται τα αρχεία NSF σας.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/email/";
```

##### 3. Αρχικοποίηση του NotesStorageFacility με το καθορισμένο αρχείο NSF
Δημιουργήστε μια παρουσία του `NotesStorageFacility` και καθορίστε το αρχείο NSF από το οποίο θέλετε να κάνετε ανάγνωση.
```java
NotesStorageFacility nsf = new NotesStorageFacility(dataDir + "SampleNSF.nsf");
```

##### 4. Επαναλάβετε κάθε μήνυμα στον χώρο αποθήκευσης NSF
Χρησιμοποιήστε ένα μπλοκ try-catch για να χειριστείτε πιθανές εξαιρέσεις και να επαναλάβετε κάθε μήνυμα ηλεκτρονικού ταχυδρομείου:
```java
try {

        // Αποκτήστε πρόσβαση σε ιδιότητες όπως θέμα, αποστολέας, παραλήπτες εδώ
    }
} catch (Exception e) {
    e.printStackTrace();
}
```
