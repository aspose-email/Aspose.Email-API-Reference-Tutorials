## Οδηγός Εφαρμογής

### Φόρτωση μηνύματος ηλεκτρονικού ταχυδρομείου
Αυτή η λειτουργία σάς επιτρέπει να φορτώνετε μηνύματα email από αρχεία στην εφαρμογή Java σας, παρέχοντας πρόσβαση για τον προγραμματισμό τους.

#### Βήμα 1: Εισαγωγή απαιτούμενων κλάσεων
```java
import com.aspose.email.MailMessage;
```

#### Βήμα 2: Ρύθμιση διαδρομής καταλόγου
Ορίστε τη διαδρομή όπου αποθηκεύονται τα αρχεία email σας:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/email/";
```
**Σημείωμα**: Αντικατάσταση `"YOUR_DOCUMENT_DIRECTORY"` με τον πραγματικό κατάλογο που περιέχει τα αρχεία email σας.

#### Βήμα 3: Φόρτωση του μηνύματος ηλεκτρονικού ταχυδρομείου
Χρήση `MailMessage.load()` για να φορτώσετε ένα αρχείο email:
```java
MailMessage message = MailMessage.load(dataDir + "messageWithAtt.eml");
```
Εδώ, `"messageWithAtt.eml"` είναι το όνομα του αρχείου email που θέλετε να φορτώσετε. Προσαρμόστε αυτό το όνομα αρχείου όπως απαιτείται.

### Ανάλυση περιεχομένου ηλεκτρονικού ταχυδρομείου
Αφού φορτώσετε το email σας, αποκτήστε πρόσβαση στο περιεχόμενο και τα συνημμένα του:
- **Θέμα**: `message.getSubject()`
- **Σώμα**: `message.getBody()`
- **Συνημμένα**: Επανάληψη μέσω `message.getAttachments()` για τα συνημμένα αρχεία.

**Συμβουλή αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι τα αρχεία email σας είναι σε μορφή EML ή MSG και ελέγξτε τις διαδρομές καταλόγων εάν προκύψουν προβλήματα.
