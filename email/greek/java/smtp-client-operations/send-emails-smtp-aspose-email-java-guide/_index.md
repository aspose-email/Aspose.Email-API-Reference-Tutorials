## Οδηγός Εφαρμογής

### Δυνατότητα: Αποστολή email μέσω SMTP

Η αποστολή ενός email περιλαμβάνει τη διαμόρφωση του προγράμματος-πελάτη SMTP με τα κατάλληλα στοιχεία διακομιστή και διαπιστευτήρια. Ας αναλύσουμε αυτήν τη διαδικασία βήμα προς βήμα.

#### Ρύθμιση παραμέτρων του προγράμματος-πελάτη SMTP

**Επισκόπηση:** Θα στήσουμε `SmtpClient` για να συνδεθείτε σε έναν διακομιστή SMTP του Gmail για την αποστολή μηνυμάτων ηλεκτρονικού ταχυδρομείου.

1. **Εισαγωγή απαιτούμενων κλάσεων:**
   
   ```java
   import com.aspose.email.SecurityOptions;
   import com.aspose.email.SmtpClient;
   ```

2. **Αρχικοποίηση του SmtpClient:**

   Θα διαμορφώσουμε το `SmtpClient` με τα στοιχεία του διακομιστή SMTP σας:

   ```java
   SmtpClient client = new SmtpClient("smtp.gmail.com", 587, "your-email@gmail.com", "your-password");
   client.setSecurityOptions(SecurityOptions.Auto);
   ```

   - **Επεξήγηση παραμέτρων:**
     - `"smtp.gmail.com"` είναι ο διακομιστής SMTP για το Gmail.
     - `587` είναι η θύρα που χρησιμοποιείται για TLS/STARTTLS.
     - Αντικαθιστώ `"your-email@gmail.com"` και `"your-password"` με τα πραγματικά σας διαπιστευτήρια.

3. **Στείλτε ένα email:**

   Δείτε πώς μπορείτε να δημιουργήσετε και να στείλετε ένα απλό email:

   ```java
   import com.aspose.email.MailMessage;

   MailMessage message = new MailMessage();
   message.setSubject("Test Subject");
   message.setBody("This is the body of the test email.");
   message.getTo().addMailAddress(new MailAddress("recipient@example.com"));

   client.send(message);
   System.out.println("Email sent successfully!");
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Σφάλματα ελέγχου ταυτότητας:** Βεβαιωθείτε ότι ο λογαριασμός σας στο Gmail επιτρέπει την "Πρόσβαση σε λιγότερο ασφαλείς εφαρμογές" εάν χρησιμοποιείτε κωδικό πρόσβασης.
- **Προβλήματα σύνδεσης:** Επαληθεύστε τη διεύθυνση και τον αριθμό θύρας του διακομιστή SMTP.
