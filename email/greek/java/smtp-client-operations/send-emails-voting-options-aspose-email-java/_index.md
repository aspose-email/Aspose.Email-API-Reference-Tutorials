# Πώς να υλοποιήσετε το Aspose.Email για Java: Αποστολή email με επιλογές ψηφοφορίας

## Οδηγός Εφαρμογής

### Δημιουργία σύνδεσης προγράμματος-πελάτη EWS
Για να στείλετε email μέσω του Microsoft Exchange, συνδεθείτε στον διακομιστή Exchange Web Services (EWS).

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να δημιουργήσετε μια σύνδεση χρησιμοποιώντας το Aspose.Email με τα παρεχόμενα διαπιστευτήρια και τη διεύθυνση URL υπηρεσίας.

#### Βήματα Υλοποίησης
1. **Εισαγωγή απαραίτητων τάξεων**
   ```java
   import com.aspose.email.EWSClient;
   import com.aspose.email.IEWSClient;
   ```
2. **Δημιουργία σύνδεσης**
   ```java
   IEWSClient client = EWSClient.getEWSClient(
       "https://exchange.aspose.com/exchangeews/Exchange.asmx",
       "username",
       "password",
       "aspose.com"
   );
   ```
   - Αντικαθιστώ `"username"` και `"password"` με τα πραγματικά σας διαπιστευτήρια.
   - Η διεύθυνση URL δείχνει στο τελικό σημείο του EWS.

### Δημιουργία και ρύθμιση παραμέτρων MailMessage
Η δημιουργία ενός μηνύματος ηλεκτρονικού ταχυδρομείου είναι απλή με το Aspose.Email. Μπορείτε εύκολα να ορίσετε λεπτομέρειες για τον αποστολέα, τον παραλήπτη, το θέμα και το σώμα του μηνύματος.

#### Επισκόπηση
Αυτή η ενότητα καλύπτει την κατασκευή ενός `MailMessage` αντικείμενο με βασικά στοιχεία email.

#### Βήματα Υλοποίησης
1. **Εισαγωγή της κλάσης**
   ```java
   import com.aspose.email.MailMessage;
   ```
2. **Δημιουργία στιγμιότυπου MailMessage**
   ```java
   String address = "firstname.lastname@aspose.com";
   MailMessage message = new MailMessage(
       address,  // Αποστολέας
       address,  // Παραλήπτης
       "Flagged Message",  // Θέμα
       "Make it concise and descriptive. The description may appear in search engines' search results pages..."
   );
   ```

### Ρύθμιση παραμέτρων επιλογών ψηφοφορίας για το MailMessage
Βελτιώστε τα email σας προσθέτοντας επιλογές ψηφοφορίας για να ζητήσετε γρήγορα σχόλια από τους παραλήπτες.

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσθέσετε κουμπιά ψηφοφορίας στο `MailMessage`.

#### Βήματα Υλοποίησης
1. **Εισαγωγή Επιλογών Παρακολούθησης**
   ```java
   import com.aspose.email.FollowUpOptions;
   ```
2. **Ορισμός κουμπιών ψηφοφορίας**
   ```java
   FollowUpOptions options = new FollowUpOptions();
   options.setVotingButtons("Yes;No;Maybe;Exactly!");
   ```

### Αποστολή αλληλογραφίαςΜήνυμα με επιλογές ψηφοφορίας
Συνδυάστε όλες τις λειτουργίες για να στείλετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου εξοπλισμένο με κουμπιά ψηφοφορίας μέσω του EWS.

#### Επισκόπηση
Αυτό το τελευταίο βήμα αποστέλλει το διαμορφωμένο μήνυμα ηλεκτρονικού ταχυδρομείου σας χρησιμοποιώντας την καθιερωμένη σύνδεση EWS.

#### Βήματα Υλοποίησης
1. **Δημιουργία σύνδεσης προγράμματος-πελάτη EWS** (επαναλαμβάνεται για λόγους κατανόησης)
   ```java
   IEWSClient client = EWSClient.getEWSClient(
       "https://exchange.aspose.com/exchangeews/Exchange.asmx",
       "username",
       "password",
       "aspose.com"
   );
   ```
2. **Δημιουργία και ρύθμιση παραμέτρων MailMessage** (επαναλαμβάνεται για λόγους κατανόησης)
   ```java
   String address = "firstname.lastname@aspose.com";
   MailMessage message = new MailMessage(
       address,
       address,
       "Flagged Message",
       "Make it concise and descriptive..."
   );
   ```
3. **Ρύθμιση παραμέτρων επιλογών ψηφοφορίας**
   ```java
   FollowUpOptions options = new FollowUpOptions();
   options.setVotingButtons("Yes;No;Maybe;Exactly!");
   ```
4. **Στείλτε το email**
   ```java
   client.send(message, options);
   ```
