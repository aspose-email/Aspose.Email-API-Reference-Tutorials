## Οδηγός Εφαρμογής
Ας εξερευνήσουμε κάθε χαρακτηριστικό βήμα προς βήμα.
### Δυνατότητα 1: Δημιουργία και αποθήκευση αρχείου μηνύματος του Outlook
**Επισκόπηση:**
Αυτός ο οδηγός σάς βοηθά να δημιουργήσετε ένα αρχείο MSG του Outlook από την αρχή χρησιμοποιώντας το Aspose.Email για Java.
#### Βήμα 1: Ορίστε τον κατάλογο εξόδου
Ξεκινήστε καθορίζοντας πού θα αποθηκευτούν τα αρχεία εξόδου σας:
```java
String dataDir = "YOUR_OUTPUT_DIRECTORY/";
```
#### Βήμα 2: Δημιουργήστε μια παρουσία MailMessage
Δημιουργήστε και διαμορφώστε ένα `MailMessage` αντικείμενο, ορίζοντας βασικές ιδιότητες όπως αποστολέας, παραλήπτης, θέμα και σώμα.
```java
MailMessage mailMsg = new MailMessage();
mailMsg.setFrom(new MailAddress("from@domain.com"));
MailAddressCollection addressCol = new MailAddressCollection();
addressCol.addMailAddress(new MailAddress("to@domain.com"));
mailMsg.setTo(addressCol);
mailMsg.setSubject("Creating an Outlook Message File");
mailMsg.setBody("This message is created by Aspose.Email for Java");
```
#### Βήμα 3: Μετατροπή και αποθήκευση του μηνύματος
Μετατρέψτε το δικό σας `MailMessage` σε ένα `MapiMessage`και, στη συνέχεια, αποθηκεύστε το ως αρχείο MSG.
```java
MapiMessage outlookMsg = MapiMessage.fromMailMessage(mailMsg);
String strMsgFile = dataDir + "message_out.msg";
auto_messag\save(strMsgFile);
```
### Λειτουργία 2: Η σημαία συμπίεσης σώματος έχει οριστεί σε True
**Επισκόπηση:**
Αυτή η λειτουργία δείχνει πώς να μειώσετε το μέγεθος του αρχείου MSG ενεργοποιώντας τη συμπίεση σώματος RTF.
#### Βήμα 1: Φόρτωση υπάρχοντος MailMessage
Φόρτωση ενός υπάρχοντος μηνύματος από έναν καθορισμένο κατάλογο:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MailMessage msg = MailMessage.load(dataDir + "message.msg");
```
#### Βήμα 2: Ενεργοποίηση συμπίεσης σώματος
Ρύθμιση παραμέτρων `MapiConversionOptions` για να ενεργοποιήσετε τη συμπίεση.
```java
MapiConversionOptions options = new MapiConversionOptions();
options.setUseBodyCompression(true);
MapiMessage ae_mapi = MapiMessage.fromMailMessage(msg, options);
ae_mapi.dispose(); // Καθαρίστε τους πόρους μετά τη χρήση.
```
### Χαρακτηριστικό 3: Η σημαία συμπίεσης σώματος έχει οριστεί σε False
**Επισκόπηση:**
Για ταχύτερη δημιουργία μηνυμάτων όταν το μέγεθος του αρχείου δεν αποτελεί πρόβλημα, απενεργοποιήστε τη συμπίεση σώματος RTF.
#### Βήμα 1: Φόρτωση υπάρχοντος μηνύματος αλληλογραφίας (Παρόμοιο με το παραπάνω)
```java
MailMessage msg = MailMessage.load(dataDir + "message.msg");
```
#### Βήμα 2: Απενεργοποίηση συμπίεσης σώματος
Δημιουργώ `MapiConversionOptions` χωρίς ρύθμιση συμπίεσης:
```java
MapiConversionOptions options = new MapiConversionOptions();
MapiMessage ae_mapi = MapiMessage.fromMailMessage(msg, options);
ae_mapi.dispose(); // Απορρίψτε τους πόρους για την αποφυγή διαρροών.
```
### Χαρακτηριστικό 4: Μετατροπή MSG σε μήνυμα MIME
**Επισκόπηση:**
Μετατρέψτε ένα αρχείο MSG του Outlook σε μορφή MIME για συμβατότητα σε διαφορετικά προγράμματα-πελάτες email.
#### Βήμα 1: Δημιουργήστε μια νέα παρουσία MapiMessage
Προετοιμάστε το `MapiMessage` με τις απαραίτητες παραμέτρους:
```java
MapiMessage msg = new MapiMessage("sender@test.com", "recipient@test.com",
    "Subject of Message", "Body of Message");
```
**Σημείωμα:** Αντικαταστήστε τα placeholders με πραγματικά δεδομένα.
