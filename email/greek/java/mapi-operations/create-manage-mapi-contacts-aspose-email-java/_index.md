## Οδηγός Εφαρμογής

Θα εξερευνήσουμε βασικά χαρακτηριστικά της διαχείρισης επαφών MAPI:

### Δημιουργία και αποθήκευση επαφής MAPI

#### Επισκόπηση

Δημιουργήστε μια νέα επαφή MAPI, συμπληρώστε την με λεπτομέρειες όπως ονόματα, διευθύνσεις, email και φωτογραφίες και, στη συνέχεια, αποθηκεύστε την ως αρχείο MSG.

#### Βήματα Υλοποίησης

**1. Ορίστε τις βασικές πληροφορίες της επαφής**

Ορίστε βασικές πληροφορίες για την επαφή σας:

```java
MapiContact contact = new MapiContact("Sebastian Wright", "SebastianWright@dayrep.com");
```

**2. Ορισμός ονόματος και επαγγελματικών ιδιοτήτων**

Διαμορφώστε πρόσθετες λεπτομέρειες ονόματος και επαγγελματικές πληροφορίες:

```java
contact.setNameInfo(new MapiContactNamePropertySet("Bertha", "A.", "Buell"));
contact.setProfessionalInfo(new MapiContactProfessionalPropertySet("Awthentikz", "Social work assistant"));
```

**3. Προσθέστε προσωπικά στοιχεία, στοιχεία διεύθυνσης, ηλεκτρονικού ταχυδρομείου και τηλεφώνου**

Εξατομικεύστε περαιτέρω την επαφή με URL, διευθύνσεις, email και αριθμούς τηλεφώνου:

```java
contact.getPersonalInfo().setPersonalHomePage("B2BTies.com");
contact.getPhysicalAddresses().getWorkAddress().setAddress("Im Astenfeld 59 8580 EDELSCHROTT");
contact.getElectronicAddresses().setEmail1(new MapiContactElectronicAddress("Experwas", "SMTP", "BerthaABuell@armyspy.com"));
contact.setTelephones(new MapiContactTelephonePropertySet("06605045265"));
```

**4. Φόρτωση και Ορισμός Φωτογραφίας**

Φόρτωση αρχείου εικόνας ως φωτογραφίας της επαφής:

```java
File fi = new File(dataDir + "Desert.jpg");
byte[] fileContent = Files.readAllBytes(fi.toPath());
MapiContactPhoto photo = new MapiContactPhoto(fileContent, MapiContactPhotoImageFormat.Jpeg);
contact.setPhoto(photo);
```

**5. Αποθηκεύστε την επαφή ως αρχείο MSG**

Αποθηκεύστε την επαφή σας σε έναν καθορισμένο κατάλογο:

```java
contact.save("YOUR_OUTPUT_DIRECTORY/Contact_out.msg", ContactSaveFormat.Msg);
```

### Φόρτωση επαφής από το MSG

#### Επισκόπηση

Φόρτωση μιας επαφής MAPI από ένα υπάρχον αρχείο MSG.

#### Βήματα Υλοποίησης

**1. Φορτώστε το MapiMessage**

Φορτώστε το αρχείο μηνύματός σας ως `MapiMessage` αντικείμενο:

```java
MapiMessage msg = MapiMessage.fromFile(dataDir + "messageMapi.msg");
```

**2. Μετατροπή σε αντικείμενο επαφής MAPI**

Μετατρέψτε το σε ένα `MapiContact` για περαιτέρω χειρισμό ή εμφάνιση:

```java
MapiContact mapiContact = (MapiContact) msg.toMapiMessageItem();
```

### Φόρτωση επαφής από VCard

#### Επισκόπηση

Φόρτωση επαφής απευθείας από ένα αρχείο VCF.

#### Βήματα Υλοποίησης

**1. Φόρτωση της επαφής MAPI**

Φορτώστε την επαφή χρησιμοποιώντας την αναπαράστασή της VCard:

```java
MapiContact mapiContact = MapiContact.fromVCard(dataDir + "microsoft.vcf");
```

### Φόρτωση επαφής VCard με καθορισμένη κωδικοποίηση

#### Επισκόπηση

Μάθετε πώς να φορτώνετε μια επαφή από ένα αρχείο VCF χρησιμοποιώντας συγκεκριμένη κωδικοποίηση χαρακτήρων.

#### Βήματα Υλοποίησης

**1. Φόρτωση της επαφής χρησιμοποιώντας κωδικοποίηση UTF-8**

Καθορίστε την κωδικοποίηση για τη σωστή ερμηνεία των χαρακτήρων:

```java
MapiContact contactReadFromFile = MapiContact.fromVCard(dataDir + "microsoft.vcf", StandardCharsets.UTF_8);
```
