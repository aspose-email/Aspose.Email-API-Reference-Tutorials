## Οδηγός Εφαρμογής

### Σύνδεση σε διακομιστή IMAP (H2)
Η ασφαλής σύνδεση σε έναν διακομιστή IMAP είναι το πρώτο βήμα στη διαχείριση των email μέσω προγραμματισμού.

#### Βήμα 1: Ρύθμιση του ImapClient
Ξεκινήστε δημιουργώντας μια παρουσία του `ImapClient` και ρύθμιση των στοιχείων του διακομιστή σας:

```java
import com.aspose.email.ImapClient;
import com.aspose.email.SecurityOptions;
import com.aspose.email.EncryptionProtocols;

// Δημιουργήστε το πρόγραμμα-πελάτη IMAP και ορίστε τις παραμέτρους σύνδεσης
ImapClient imapClient = new ImapClient();
imapClient.setHost("<HOST>"); // Αντικαταστήστε με τη διεύθυνση κεντρικού υπολογιστή του διακομιστή σας
imapClient.setPort(993); // Χρησιμοποιήστε τη θύρα 993 για συνδέσεις SSL
imapClient.setUsername("<USERNAME>"); // Το όνομα χρήστη του ηλεκτρονικού σας ταχυδρομείου
imapClient.setPassword("<PASSWORD>"); // Ο κωδικός πρόσβασης ηλεκτρονικού ταχυδρομείου σας

// Ρύθμιση παραμέτρων επιλογών ασφαλείας και πρωτοκόλλου κρυπτογράφησης
imapClient.setSupportedEncryption(EncryptionProtocols.Tls);
imapClient.setSecurityOptions(SecurityOptions.SSLImplicit);
```

**Γιατί αυτό έχει σημασία**Η χρήση SSL/TLS διασφαλίζει την ασφαλή σύνδεσή σας, προστατεύοντας τα ευαίσθητα δεδομένα από υποκλοπές.

### Καταχώριση μηνυμάτων από διακομιστή IMAP (H2)
Μόλις συνδεθείτε, μπορείτε να εμφανίσετε μια λίστα με όλα τα μηνύματα στο γραμματοκιβώτιό σας για να ανακτήσετε τους αριθμούς ακολουθίας τους για περαιτέρω επεξεργασία.

#### Βήμα 1: Λίστα μηνυμάτων

```java
import com.aspose.email.ImapMessageInfoCollection;

// Ανάκτηση μιας συλλογής αντικειμένων πληροφοριών μηνύματος
ImapMessageInfoCollection messageInfoCol = imapClient.listMessages();
int listCount = messageInfoCol.size(); // Προσδιορίστε τον αριθμό των μηνυμάτων

List<Integer> sequenceNumberList = new ArrayList<>();
for (com.aspose.email.ImapMessageInfo messageInfo : messageInfoCol) {
    sequenceNumberList.add(messageInfo.getSequenceNumber()); // Συλλογή αριθμών ακολουθίας
}
```

**Διαμόρφωση κλειδιού**Προσαρμόστε τις ρυθμίσεις του διακομιστή σας όπως απαιτείται για να διασφαλίσετε τη συμβατότητα με τον πάροχο IMAP σας.

### Ανάκτηση μηνυμάτων με βάση τους αριθμούς ακολουθίας και τα μοναδικά αναγνωριστικά (H2)
Αφού καταχωρίσετε τα μηνύματα, μπορείτε να ανακτήσετε συγκεκριμένα μηνύματα ηλεκτρονικού ταχυδρομείου χρησιμοποιώντας τους αριθμούς ακολουθίας τους ή τα μοναδικά αναγνωριστικά τους για λεπτομερή επεξεργασία.

#### Βήμα 1: Ανάκτηση με Αριθμούς Ακολουθίας

```java
import java.util.List;
import com.aspose.email.MailMessage;

// Λήψη μηνυμάτων με βάση τους συλλεγμένους αριθμούς ακολουθίας
List<MailMessage> fetchedMessagesBySNumMC = (List<MailMessage>) imapClient.fetchMessagesBySequences(sequenceNumberList);
int fetchedCountBySequence = fetchedMessagesBySNumMC.size(); // Αριθμός ανακτημένων μηνυμάτων
```

#### Βήμα 2: Ανάκτηση με μοναδικά αναγνωριστικά

```java
import java.util.ArrayList;
import com.aspose.email.ImapMessageInfo;

// Συλλογή μοναδικών αναγνωριστικών από τη συλλογή πληροφοριών μηνυμάτων
List<String> uniqueIdList = new ArrayList<>();
for (com.aspose.email.ImapMessageInfo messageInfo : messageInfoCol) {
    uniqueIdList.add(messageInfo.getUniqueId()); // Συγκεντρώστε μοναδικά αναγνωριστικά για ανάκτηση
}

// Ανάκτηση μηνυμάτων χρησιμοποιώντας τα μοναδικά τους αναγνωριστικά
List<MailMessage> fetchedMessagesByUidMC = (List<MailMessage>) imapClient.fetchMessagesByUids(uniqueIdList);
int fetchedCountByUniqueIds = fetchedMessagesByUidMC.size(); // Αριθμός μοναδικά αναγνωρισμένων μηνυμάτων
```

**Συμβουλή αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι έχετε επαρκή δικαιώματα για τη λήψη μηνυμάτων ηλεκτρονικού ταχυδρομείου από τον διακομιστή και επαληθεύστε τη συνδεσιμότητα δικτύου εάν προκύψουν προβλήματα.
