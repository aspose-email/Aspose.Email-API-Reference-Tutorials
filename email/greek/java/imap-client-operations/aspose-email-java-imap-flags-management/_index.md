## Οδηγός Εφαρμογής
### Ρύθμιση σημαιών μηνυμάτων
**Επισκόπηση**Αυτή η λειτουργία σάς επιτρέπει να επισημάνετε συγκεκριμένα μηνύματα ηλεκτρονικού ταχυδρομείου ως αναγνωσμένα σε ένα γραμματοκιβώτιο IMAP χρησιμοποιώντας το Aspose.Email Java API.

#### Βήμα 1: Αρχικοποίηση του ImapClient
```java
import com.aspose.email.ImapClient;
import com.aspose.email.ImapFolderInfo;
import com.aspose.email.ImapMessageFlags;
import com.aspose.email.SecurityOptions;

// Δημιουργήστε μια παρουσία του ImapClient και ορίστε τις λεπτομέρειες του διακομιστή
ImapClient client = new ImapClient();
client.setHost("imap.gmail.com");
client.setPort(993);
client.setUsername("username");
client.setPassword("password");
client.setSecurityOptions(SecurityOptions.Auto);

// Επιλέξτε τον φάκελο εισερχομένων στον οποίο θα δουλέψετε
client.selectFolder(ImapFolderInfo.IN_BOX);
```
**Εξήγηση**Εδώ, αρχικοποιούμε `ImapClient` με τα στοιχεία του διακομιστή IMAP σας. Οι επιλογές ασφαλείας έχουν οριστεί σε αυτόματες για ασφαλείς συνδέσεις.

#### Βήμα 2: Επισήμανση μηνύματος ως αναγνωσμένου
```java
// Αλλάξτε την κατάσταση της σημαίας «Ανάγνωση» του μηνύματος ID 1 για να το επισημάνετε ως αναγνωσμένο
client.changeMessageFlags(1, ImapMessageFlags.isRead());
```
**Εξήγηση**Χρησιμοποιούμε `changeMessageFlags` με το αναγνωριστικό μηνύματος και το `isRead()` μέθοδος για να ορίσετε την κατάσταση του email.

### Αφαίρεση σημαιών μηνυμάτων
**Επισκόπηση**Αυτή η λειτουργία δείχνει πώς να επαναφέρετε ένα email σε μη αναγνωσμένο, αφαιρώντας τη σημαία «Ανάγνωση».

#### Βήμα 1: Αρχικοποίηση του ImapClient
(Επαναχρησιμοποιήστε τον κώδικα αρχικοποίησης προγράμματος-πελάτη από τον ορισμό σημαιών.)

#### Βήμα 2: Κατάργηση της σημαίας «Ανάγνωση»
```java
// Αφαίρεση της σημαίας «Ανάγνωση» από το αναγνωριστικό μηνύματος 1, επισημαίνοντάς το ως μη αναγνωσμένο
client.removeMessageFlags(1, ImapMessageFlags.isRead());
```
**Εξήγηση**: Παρόμοια με την τοποθέτηση της σημαίας, `removeMessageFlags` χρησιμοποιείται με το `isRead()` μέθοδος για την εκκαθάριση της κατάστασης ανάγνωσης.
