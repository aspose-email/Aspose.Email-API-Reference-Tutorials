## Οδηγός Εφαρμογής

Θα χωρίσουμε τη διαδικασία σε λογικά μέρη, καθένα από τα οποία θα εστιάζει σε ένα συγκεκριμένο χαρακτηριστικό.

### Φόρτωση αρχείου EML

#### Επισκόπηση
Η φόρτωση ενός αρχείου EML είναι απλή με το Aspose.Email για Java. Χρησιμοποιήστε το `MailMessage` κλάση για να φορτώσετε αποτελεσματικά τα δεδομένα email σας.

#### Βήματα:
**Βήμα 1: Εισαγωγή απαιτούμενων κλάσεων**
```java
import com.aspose.email.MailMessage;
```

**Βήμα 2: Φόρτωση αρχείου EML**
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MailMessage mailMessage = MailMessage.load(dataDir + "TestAppointment.eml");
```
*Εδώ, `dataDir` είναι ο κατάλογος όπου βρίσκεται το αρχείο EML σας.*

### Μετατροπή EML σε MSG με προσαρμοσμένες επιλογές

#### Επισκόπηση
Το Aspose.Email σάς επιτρέπει να μετατρέψετε ένα αρχείο EML σε μορφή MSG, εφαρμόζοντας παράλληλα προσαρμοσμένες επιλογές μετατροπής για καλύτερο έλεγχο της εξόδου.

**Βήμα 1: Εισαγωγή απαραίτητων κλάσεων**
```java
import com.aspose.email.MapiConversionOptions;
import com.aspose.email.OutlookMessageFormat;
import com.aspose.email.MapiMessage;
```

**Βήμα 2: Δημιουργία και ρύθμιση παραμέτρων επιλογών μετατροπής**
```java
MapiConversionOptions conversionOptions = new MapiConversionOptions();
conversionOptions.setFormat(OutlookMessageFormat.Unicode);
conversionOptions.setForcedRtfBodyForAppointment(false);
```
*Σύνθεση `ForcedRtfBodyForAppointment` Η τιμή "false" διασφαλίζει ότι η HTML προτιμάται έναντι της RTF, όταν είναι διαθέσιμη.*

**Βήμα 3: Μετατροπή MailMessage σε MapiMessage**
```java
MapiMessage mapiMessage = MapiMessage.fromMailMessage(mailMessage, conversionOptions);
```

### Έλεγχος και εκτύπωση τύπου σώματος αρχείου MSG

#### Επισκόπηση
Προσδιορίστε εάν ο τύπος σώματος του αρχείου MSG σας είναι HTML ή RTF. Αυτό το βήμα βοηθά στην κατανόηση του τρόπου με τον οποίο θα αποδίδεται το περιεχόμενο του email σας.

**Βήμα 1: Έλεγχος τύπου περιεχομένου σώματος**
```java
import com.aspose.email.BodyContentType;

if(mapiMessage.getBodyType() == BodyContentType.Html){
    System.out.println("The body type is HTML.");
} else if(mapiMessage.getBodyType() == BodyContentType.Rtf) {
    System.out.println("The body type is RTF.");
}
```

### Αποθήκευση αρχείου MSG στον κατάλογο εξόδου

#### Επισκόπηση
Τέλος, αποθηκεύστε το μήνυμα MAPI που έχει μετατραπεί ως αρχείο MSG στον επιθυμητό κατάλογο εξόδου.

**Βήμα 1: Ρύθμιση καταλόγου εξόδου**
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/";
```

**Βήμα 2: Αποθήκευση αρχείου MSG**
```java
try {
    mapiMessage.save(outputDir + "TestAppointment_out.msg");
} catch (IOException e) {
    e.printStackTrace();
}
```
*Βεβαιωθείτε ότι ο κατάλογος υπάρχει για να αποτρέψετε `IOException`.*
