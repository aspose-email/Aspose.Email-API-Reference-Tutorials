# Πώς να Διαβάσετε τις Κεφαλίδες και να Δημιουργήσετε Προσαρμοσμένες Κεφαλίδες Email με το Aspose.Email

## Εισαγωγή στις Κεφαλίδες Email

Σε αυτό το σεμινάριο θα ανακαλύψετε **πώς να διαβάζετε τις κεφαλίδες** πληροφορίες, θα μάθετε **πώς να προσθέτετε τιμές κεφαλίδας** και θα καταλάβετε γιατί οι προσαρμοσμένες κεφαλίδες email είναι απαραίτητες για σύγχρονα ροές μηνυμάτων. Οι κεφαλίδες email λειτουργούν όπως ένα ψηφιακό φάκελο, μεταφέροντας μεταδεδομένα όπως αποστολέας, παραλήπτης, διαδρομή δρομολόγησης και χρονικές σημάνσεις. Στο τέλος αυτού του οδηγού θα μπορείτε να **εξάγετε κεφαλίδες email**, να δημιουργήσετε τα δικά σας προσαρμοσμένα πεδία και να διαβάσετε την κεφαλίδα θέματος του email — όλα με το Aspose.Email for Java.

## Γρήγορες Απαντήσεις
- **Ποιος είναι ο κύριος τρόπος για να προσθέσετε μια προσαρμοσμένη κεφαλίδα;** Χρησιμοποιήστε τη συλλογή `Headers` σε ένα αντικείμενο `MailMessage`.  
- **Πώς μπορώ να διαβάσω την κεφαλίδα Subject μετά τη φόρτωση ενός email;** Καλέστε `message.getHeaders().get("Subject")`.  
- **Χρειάζομαι άδεια για τη χρήση των API κεφαλίδων;** Μια δοκιμαστική έκδοση λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγή.  
- **Υπάρχει κάποιο όριο στα ονόματα προσαρμοσμένων κεφαλίδων;** Ακολουθήστε τις συμβάσεις ονοματοδοσίας RFC 5322 (π.χ., ξεκινήστε με “X-”).  
- **Ποια έκδοση του Aspose.Email υποστηρίζει αυτές τις δυνατότητες;** Όλες οι πρόσφατες εκδόσεις (2024‑2026) περιλαμβάνουν πλήρη διαχείριση κεφαλίδων.

## Τι Είναι μια Κεφαλίδα και Γιατί Θα Θέλατε να τη Διαβάσετε;

Οι κεφαλίδες είναι γραμμές απλού κειμένου που τοποθετούνται στην κορυφή ενός μηνύματος email. Περιγράφουν ποιος έστειλε το μήνυμα, πότε εστάλη και πώς ταξίδεψε μέσω των διακομιστών αλληλογραφίας. Η δυνατότητα **ανάγνωσης των κεφαλίδων** σας επιτρέπει να:

* Διαγνώσετε προβλήματα παράδοσης εξετάζοντας την αλυσίδα `Received`.  
* Συσχετίσετε μηνύματα με εσωτερικά IDs εργασιών (`X-Job-ID`).  
* Υλοποιήσετε προσαρμοσμένη λογική δρομολόγησης χρησιμοποιώντας πεδία όπως `X-Priority`.

## Πώς να Προσθέσετε Προσαρμοσμένη Κεφαλίδα (Δημιουργία Προσαρμοσμένων Κεφαλίδων Email)

### Βήμα 1: Αρχικοποίηση ενός MailMessage

```java
MailMessage message = new MailMessage();
```

### Βήμα 2: Προσθήκη προσαρμοσμένης κεφαλίδας

```java
message.getHeaders().add("X-Custom-Header", "My Custom Value");
```

> **Συμβουλή:** Προσθέστε πρόθεμα `X-` στις προσαρμοσμένες κεφαλίδες για να συμμορφώνονται με το RFC 5322 και να αποφεύγετε συγκρούσεις με τα τυπικά πεδία.

### Βήμα 3: Αποστολή του email

```java
SmtpClient client = new SmtpClient("smtp.example.com");
client.send(message);
```

## Πώς να Διαβάσετε τις Κεφαλίδες Email (Διαβάστε την Κεφαλίδα Θέματος Email)

### Βήμα 1: Φορτώστε το email από αρχείο ή ροή

```java
MailMessage message = MailMessage.load("path/to/email.eml");
```

### Βήμα 2: Εξάγετε οποιαδήποτε κεφαλίδα χρειάζεστε

```java
String subject = message.getHeaders().get("Subject");
String sender = message.getHeaders().get("From");
```

> **Σημείωση:** Η συλλογή `Headers` επιστρέφει `null` εάν η ζητούμενη κεφαλίδα δεν υπάρχει, οπότε ελέγχετε πάντα για `null` πριν χρησιμοποιήσετε την τιμή.

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Λύση |
|----------|-------|------|
| Η κεφαλίδα δεν εμφανίζεται στο ληφθέν email | Ο διακομιστής SMTP αφαιρεί άγνωστες κεφαλίδες | Βεβαιωθείτε ότι ο διακομιστής επιτρέπει προσαρμοσμένες κεφαλίδες `X-` ή ρυθμίστε τον να τις διατηρεί. |
| `null` επιστράφηκε κατά την ανάγνωση μιας κεφαλίδας | Λάθος όνομα κεφαλίδας (διάκριση πεζών‑κεφαλαίων) | Χρησιμοποιήστε το ακριβές όνομα κεφαλίδας όπως αποθηκεύεται, π.χ., "Subject" όχι "subject". |
| Διπλές κεφαλίδες | Προσθήκη της ίδιας κεφαλίδας πολλές φορές | Χρησιμοποιήστε `addOrUpdate` (αν υπάρχει) ή αφαιρέστε την παλιά καταχώρηση πριν προσθέσετε μια νέα. |

## Συχνές Ερωτήσεις

**Ε: Πώς μπορώ να δω τις κεφαλίδες email σε δημοφιλείς πελάτες email;**  
Α: Οι περισσότεροι πελάτες επιτρέπουν την προβολή της ακατέργαστης πηγής — ψάξτε για επιλογές όπως “View Original”, “Show Headers” ή “View Source”.

**Ε: Είναι κρυπτογραφημένες οι κεφαλίδες email;**  
Α: Όχι. Οι κεφαλίδες είναι μεταδεδομένα απλού κειμένου και μεταδίδονται σε καθαρό κείμενο εκτός εάν ολόκληρο το μήνυμα είναι κρυπτογραφημένο (π.χ., S/MIME).

**Ε: Μπορώ να τροποποιήσω τις κεφαλίδες email μετά την αποστολή ενός email;**  
Α: Μόλις το μήνυμα είναι στο δίκτυο, οι κεφαλίδες είναι αμετάβλητες. Ορίστε όλες τις απαιτούμενες κεφαλίδες **πριν** καλέσετε `client.send(message)`.

**Ε: Ποιος είναι ο σκοπός της κεφαλίδας “Received”;**  
Α: Καταγράφει κάθε βήμα που διανύει το email, βοηθώντας τους διαχειριστές να εντοπίσουν προβλήματα παράδοσης και να παρακολουθήσουν τη διαδρομή.

**Ε: Πώς μπορώ να εξάγω κεφαλίδες email από μια μεγάλη δέσμη email;**  
Α: Χρησιμοποιήστε το `MailMessage.load` του Aspose.Email σε βρόχο ή αξιοποιήστε το `MailMessageCollection` για μαζική επεξεργασία.
