## Οδηγός Εφαρμογής

### Σύνδεση με διακομιστή POP3
**Επισκόπηση:** Η δημιουργία μιας σύνδεσης είναι το πρώτο βήμα για την πρόσβαση σε δεδομένα email από έναν διακομιστή.

#### Βήμα 1: Αρχικοποίηση και διαμόρφωση `Pop3Client`
Αρχικά, εισαγάγετε τις απαραίτητες κλάσεις:

```java
import com.aspose.email.Pop3Client;
```

Δημιουργήστε μια παρουσία του `Pop3Client` και διαμορφώστε το με τα στοιχεία του διακομιστή σας:

```java
// Δημιουργία αντικειμένου Pop3Client
Pop3Client client = new Pop3Client();

// Ορισμός λεπτομερειών διακομιστή
client.setHost("exchange.domain.com");
client.setUsername("username");
client.setPassword("password");
```

Αυτή η ρύθμιση σάς επιτρέπει να δημιουργήσετε μια σύνδεση χρησιμοποιώντας τα παρεχόμενα διαπιστευτήρια.

#### Βήμα 2: Ανάκτηση πληροφοριών γραμματοκιβωτίου
**Επισκόπηση:** Μόλις συνδεθείτε, ανακτήστε κρίσιμα στατιστικά στοιχεία γραμματοκιβωτίου, όπως τον αριθμό και το μέγεθος μηνυμάτων.

Πρώτον, εισαγωγή `Pop3MailboxInfo`:

```java
import com.aspose.email.Pop3MailboxInfo;
```

Χρησιμοποιήστε τον πελάτη για να λάβετε πληροφορίες σχετικά με το γραμματοκιβώτιό σας:

```java
// Λήψη πληροφοριών γραμματοκιβωτίου
Pop3MailboxInfo mailBoxInfo = client.getMailboxInfo();

// Ανάκτηση αριθμού μηνυμάτων και μεγέθους καταληψιμότητας
int messageCount = mailBoxInfo.getMessageCount();
long nOccupiedSize = mailBoxInfo.getOccupiedSize();
```

Αυτός ο κώδικας ανακτά τον αριθμό των μηνυμάτων και τον συνολικό χώρο που καταλαμβάνουν στον διακομιστή.
