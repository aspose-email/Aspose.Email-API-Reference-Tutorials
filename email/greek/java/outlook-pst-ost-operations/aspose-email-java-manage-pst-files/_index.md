## Οδηγός Εφαρμογής

Θα χωρίσουμε το σεμινάριο σε διαχειρίσιμες ενότητες με βάση συγκεκριμένα χαρακτηριστικά.

### Δημιουργία νέου αρχείου PST

Η δημιουργία ενός νέου αρχείου PST είναι απλή με το Aspose.Email. Αυτή η λειτουργία σάς επιτρέπει να ξεκινήσετε νέο χώρο αποθήκευσης για email και σχετικά δεδομένα.

#### Βήμα 1: Ρύθμιση διαδρομής καταλόγου
Καθορίστε πού θα αποθηκευτεί το νέο σας αρχείο PST:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/outlook/";
```

#### Βήμα 2: Δημιουργήστε το αρχείο PST

Χρησιμοποιήστε το `PersonalStorage.create()` μέθοδος για την αρχικοποίηση ενός νέου αρχείου PST. Η πρώτη παράμετρος είναι η διαδρομή και η δεύτερη καθορίζει την έκδοση μορφής (0 για Unicode).

```java
import com.aspose.email.PersonalStorage;

// Δημιουργήστε μια νέα παρουσία του PersonalStorage
PersonalStorage pst = PersonalStorage.create(dataDir + "newSample_out.pst", 0);
```

### Προσθήκη φακέλου στη ρίζα του PST

Η προσθήκη φακέλων βοηθά στην οργάνωση των email σας μέσα στο αρχείο PST. Αυτή η ενότητα δείχνει πώς να προσθέσετε έναν φάκελο σε επίπεδο ρίζας.

#### Βήμα 1: Φόρτωση του αρχείου PST
Υποθέτοντας ότι έχετε ένα υπάρχον ή ένα πρόσφατα δημιουργημένο αρχείο PST:

```java
PersonalStorage pst = PersonalStorage.fromFile(dataDir + "newSample_out.pst");
```

#### Βήμα 2: Προσθήκη νέου φακέλου
Δημιουργήστε και προσθέστε έναν φάκελο με το όνομα `myInbox` στο βασικό επίπεδο του PST.

```java
pst.getRootFolder().addSubFolder("myInbox");
```

### Προσθήκη μηνύματος σε συγκεκριμένο φάκελο στο PST

Η διαχείριση μηνυμάτων email είναι ζωτικής σημασίας. Δείτε πώς μπορείτε να προσθέσετε ένα υπάρχον αρχείο .msg στο PST σας:

#### Βήμα 1: Φόρτωση του αρχείου PST και MSG
Εξασφαλίζω `newSample_out.pst` υπάρχει με έτοιμη τη δομή φακέλων.

```java
PersonalStorage pst = PersonalStorage.fromFile(dataDir + "newSample_out.pst");
MapiMessage message = MapiMessage.fromFile(dataDir + "message.msg");
```

#### Βήμα 2: Προσθήκη του μηνύματος στον φάκελο
Εισαγάγετε το μήνυμα ηλεκτρονικού ταχυδρομείου σας στον καθορισμένο φάκελο.

```java
pst.getRootFolder().getSubFolder("myInbox").addMessage(message);
```
