## Οδηγός Εφαρμογής

Θα χωρίσουμε αυτόν τον οδηγό σε τρία κύρια χαρακτηριστικά για να σας βοηθήσουμε να διαχειριστείτε αποτελεσματικά τα αρχεία PST σας.

### Πρόσβαση και τροποποίηση δομής αρχείου PST

#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει την πρόσβαση σε ένα αρχείο PST, τον εντοπισμό συγκεκριμένων υποφακέλων και την αλλαγή της κλάσης κοντέινερ τους χρησιμοποιώντας το Aspose.Email. `FolderInfo` και `PersonalStorage` τάξεις.

#### Βήματα για την εφαρμογή
##### Φόρτωση του αρχείου PST
Ξεκινήστε φορτώνοντας το αρχείο PST. Αυτό αρχικοποιεί ένα `PersonalStorage` αντικείμενο.
```java
import com.aspose.email.PersonalStorage;
// Φόρτωση του αρχείου PST
PersonalStorage pst = PersonalStorage.fromFile("YOUR_DOCUMENT_DIRECTORY/SampleContacts.pst");
```
##### Πρόσβαση στον ριζικό φάκελο
Αποκτήστε πρόσβαση στον ριζικό φάκελο για να περιηγηθείτε στους υποφακέλους του.
```java
import com.aspose.email.FolderInfo;
// Λήψη του ριζικού φακέλου από το φορτωμένο PST
FolderInfo rootFolder = pst.getRootFolder();
```
##### Εντοπίστε τον υποφάκελο «Επαφές»
Χρησιμοποιήστε το `getSubFolder` μέθοδος για την εύρεση ενός συγκεκριμένου υποφακέλου με βάση το όνομα.
```java
// Αποκτήστε πρόσβαση στον υποφάκελο «Επαφές» μέσα στον ριζικό φάκελο
FolderInfo contactsFolder = rootFolder.getSubFolder("Contacts");
```
##### Αλλαγή κλάσης κοντέινερ
Τροποποιήστε την κλάση κοντέινερ του υποφακέλου προορισμού σας. Εδώ, την αλλάζουμε σε "IPF.Note".
```java
// Αλλαγή της κλάσης κοντέινερ του φακέλου «Επαφές»
contactsFolder.changeContainerClass("IPF.Note");
```
##### Απόρριψη Πόρων
Τέλος, βεβαιωθείτε ότι έχετε απορρίψει το `PersonalStorage` αντίρρηση για την απελευθέρωση πόρων.
```java
// Καθαρισμός με την απόρριψη του αντικειμένου PST
pst.dispose();
```
### Χρησιμοποιήστε τις κλάσεις FolderInfo και PersonalStorage του Aspose.Email

#### Επισκόπηση
Μάθετε πώς να αξιοποιείτε αυτές τις κλάσεις για χειρισμό φακέλων μέσα σε ένα αρχείο PST, συμπεριλαμβανομένης της πρόσβασης και της διαχείρισης υποφακέλων.

##### Οδηγός βήμα προς βήμα
1. **Φόρτωση του αρχείου PST**
   - Χρήση `PersonalStorage.fromFile` για να φορτώσετε το αρχείο σας.
2. **Λήψη του ριζικού φακέλου**
   - Ανακτήστε τη ρίζα χρησιμοποιώντας `getRootFolder`.
3. **Πρόσβαση σε συγκεκριμένο υποφάκελο**
   - Αποκτήστε πρόσβαση σε έναν συγκεκριμένο φάκελο, όπως "Επαφές", με `getSubFolder`.
4. **Απόρριψη πόρων**
   - Πάντα να καλείς `dispose` στο `PersonalStorage` αντικείμενο μετά τις επεμβάσεις.
