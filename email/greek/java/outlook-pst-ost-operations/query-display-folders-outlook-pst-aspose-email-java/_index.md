### Επισκόπηση χαρακτηριστικών

Αυτή η λειτουργία σάς επιτρέπει να φιλτράρετε και να εμφανίσετε σε λίστα μόνο τους φακέλους σε ένα αρχείο PST του Outlook που δημιουργήθηκαν από τον τρέχοντα χρήστη. Είναι ιδιαίτερα χρήσιμη για χρήστες που χρειάζονται να διαχειρίζονται τα δεδομένα email τους πιο αποτελεσματικά.

#### Βήμα 1: Φόρτωση του αρχείου PST

Αρχικά, φορτώστε το αρχείο PST χρησιμοποιώντας το Aspose.Email:

```java
// Ορίστε τον κατάλογο που περιέχει τα αρχεία PST σας
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/outlook/";

// Φόρτωση του αρχείου PST
PersonalStorage pst = PersonalStorage.fromFile(dataDir + "Outlook.pst");
```

#### Βήμα 2: Δημιουργήστε ένα Εργαλείο Δημιουργίας Ερωτημάτων

Ρυθμίστε ένα εργαλείο δημιουργίας ερωτημάτων για να φιλτράρετε τους φακέλους που δημιουργούνται από τον τρέχοντα χρήστη:

```java
// Αρχικοποίηση του εργαλείου δημιουργίας ερωτημάτων
PersonalStorageQueryBuilder queryBuilder = new PersonalStorageQueryBuilder();
queryBuilder.getOnlyFoldersCreatedByUser().equals(true);
```

#### Βήμα 3: Ανάκτηση και εμφάνιση φακέλων

Χρησιμοποιήστε το εργαλείο δημιουργίας ερωτημάτων για να ανακτήσετε υποφακέλους που ταιριάζουν με τα κριτήριά σας και, στη συνέχεια, επαναλάβετε την περιήγησή σας σε αυτούς για να εμφανίσετε τα ονόματα των φακέλων:

```java
// Λήψη φακέλων με βάση το ερώτημα
FolderInfoCollection subfolders = pst.getRootFolder().getSubFolders(queryBuilder.getQuery());

// Επανάληψη και εκτύπωση ονομάτων φακέλων
for (FolderInfo folder : subfolders) {
    System.out.println(folder.getDisplayName());
}
```

#### Βήμα 4: Απόρριψη πόρων

Βεβαιωθείτε ότι οι πόροι αποδεσμεύονται σωστά μετά τη χρήση:
