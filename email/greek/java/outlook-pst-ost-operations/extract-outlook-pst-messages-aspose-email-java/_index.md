### Χαρακτηριστικό 3: Αναδρομική εξαγωγή μηνυμάτων από φακέλους και υποφακέλους (H2)
Για να διασφαλίσετε την ολοκληρωμένη εξαγωγή, χρειάζεστε μια αναδρομική προσέγγιση για όλους τους φακέλους και τους υποφακέλους.

#### Επισκόπηση
Ο `extractMsgFiles` Η μέθοδος χειρίζεται αυτό επαναλαμβάνοντας τα μηνύματα και επεξεργαζόμενη αναδρομικά κάθε υποφάκελο.
```java
import com.aspose.email.MessageInfo;
import com.aspose.email.MapiMessage;

private static void extractMsgFiles(FolderInfo folderInfo, PersonalStorage pst, String strPSTFile) {
    // Δημιουργία καταλόγου για τα μηνύματα του τρέχοντος φακέλου
    String folderName = strPSTFile + "/" + folderInfo.getDisplayName();
    new File(folderName).mkdir();

    // Επεξεργασία όλων των μηνυμάτων στον τρέχοντα φάκελο
    MessageInfoCollection messageInfoCollection = folderInfo.getContents();
    for (int i = 0; i < messageInfoCollection.size(); i++) {
        MessageInfo messageInfo = (MessageInfo) messageInfoCollection.get_Item(i);
        MapiMessage message = pst.extractMessage(messageInfo);

        // Απολυμάνετε και αποθηκεύστε το μήνυμα
        String messageName = getRidOfIllegalFileNameCharacters(
            message.getSubject() == null || message.getSubject().isEmpty()
                ? messageInfo.getEntryIdString()
                : message.getSubject());
        message.save(folderName + "/" + messageName + ".msg");
    }

    // Επεξεργασία υποφακέλων αναδρομικά
    for (int i = 0; i < folderInfo.getSubFolders().size(); i++) {
        FolderInfo subfolderInfo = (FolderInfo) folderInfo.getSubFolders().get_Item(i);
        extractMsgFiles(subfolderInfo, pst, strPSTFile);
    }
}
```

### Λειτουργία 4: Απολύμανση ονομάτων αρχείων για την αποθήκευση μηνυμάτων (H2)
Είναι σημαντικό να καθαρίζετε τα ονόματα αρχείων για να αποφεύγετε τους μη επιτρεπόμενους χαρακτήρες που θα μπορούσαν να προκαλέσουν σφάλματα κατά τη διάρκεια των εργασιών αρχείων.

#### Επισκόπηση
Ο `getRidOfIllegalFileNameCharacters` Η μέθοδος αντικαθιστά τους προβληματικούς χαρακτήρες με ένα κενό και περιορίζει το μήκος των ονομάτων αρχείων:
```java
import java.io.File;

private static String getRidOfIllegalFileNameCharacters(String strName) {
    // Αντικατάσταση μη επιτρεπόμενων χαρακτήρων με κενό
    String strLegalName = strName.replace(":", " ").replace("\\", " ").replace("?", " ")
                                 .replace("/", " ").replace("|", " ").replace("*", " ")
                                 .replace("<", " ").replace(">", " ").replace("\t", " ").replace("\"", " ");

    // Περικοπή σε μέγιστο μήκος 100 χαρακτήρων
    if (strLegalName.length() >= 100) {
        strLegalName = strLegalName.substring(0, 100);
    }
    return strLegalName;
}
```
