# Αποθήκευση μηνυμάτων PST σε ροές και αρχεία χρησιμοποιώντας το Aspose.Email για Java

## Εισαγωγή

Σε αυτόν τον οδηγό, θα καλύψουμε:
- Εξαγωγή και αποθήκευση μηνυμάτων από ένα αρχείο PST
- Τεχνικές για την αποθήκευση email ως ροές ή μεμονωμένα αρχεία .msg
- Πρακτικές εφαρμογές σε σενάρια πραγματικού κόσμου

Είστε έτοιμοι να βελτιώσετε τις δεξιότητές σας στη διαχείριση email με το Aspose.Email Java; Ας ξεκινήσουμε εξετάζοντας τις προϋποθέσεις!

## Οδηγός Εφαρμογής

### Αποθήκευση μηνυμάτων από PST σε ροή χρησιμοποιώντας το MessageInfo

Αυτή η λειτουργία σάς επιτρέπει να αποθηκεύετε μηνύματα email απευθείας από ένα αρχείο PST σε μια ροή, χρησιμοποιώντας συγκεκριμένα ένα `ByteArrayOutputStream`.

#### Επισκόπηση

Αξιοποιώντας το `MessageInfo` κλάση, αποκτήστε πρόσβαση στις λεπτομέρειες του μηνύματος και επαναλάβετε τις διαδικασίες για να αποθηκεύσετε κάθε μήνυμα αποτελεσματικά.

#### Βήματα Υλοποίησης

1. **Φόρτωση του αρχείου PST**
   
   Ξεκινήστε φορτώνοντας το αρχείο PST:
   ```java
   PersonalStorage pst = PersonalStorage.fromFile("YOUR_DOCUMENT_DIRECTORY/PersonalStorage.pst");
   ```
   
2. **Πρόσβαση στον φάκελο Εισερχομένων**
   
   Πρόσβαση σε μηνύματα εντός του υποφακέλου 'myInbox':
   ```java
   FolderInfo inbox = pst.getRootFolder().getSubFolder("myInbox");
   ```
   
3. **Επανάληψη και αποθήκευση μηνυμάτων σε ροή**
   
   Χρησιμοποιήστε έναν βρόχο για να απαριθμήσετε μηνύματα, αποθηκεύοντας το καθένα σε ένα `ByteArrayOutputStream`:
   ```java
   for (Object obj : inbox.enumerateMessages()) {
       MessageInfo messageInfo = (MessageInfo) obj;
       pst.saveMessageToStream(messageInfo.getEntryIdString(), new ByteArrayOutputStream());
   }
   ```

### Αποθήκευση μηνυμάτων από PST σε αρχεία χρησιμοποιώντας το MessageInfo

Αυτή η λειτουργία περιλαμβάνει την αποθήκευση μηνυμάτων ως μεμονωμένα αρχεία .msg χρησιμοποιώντας το `FileOutputStream`.

#### Επισκόπηση

Δημιουργήστε ένα αρχείο για κάθε μήνυμα με το όνομα του θέματός του, διευκολύνοντας τη διαχείριση των αρχείων email.

#### Βήματα Υλοποίησης

1. **Φόρτωση του αρχείου PST**
   
   Παρόμοια με την προηγούμενη ενότητα:
   ```java
   PersonalStorage pst = PersonalStorage.fromFile("YOUR_DOCUMENT_DIRECTORY/PersonalStorage.pst");
   ```
   
2. **Πρόσβαση και επανάληψη μηνυμάτων**
   
   Πρόσβαση σε μηνύματα στα 'Εισερχόμενά μου' και προετοιμασία για έξοδο αρχείου:
   ```java
   FolderInfo inbox = pst.getRootFolder().getSubFolder("myInbox");

   for (Object obj : inbox.enumerateMessages()) {
       MessageInfo messageInfo = (MessageInfo) obj;
       File file = new File(messageInfo.getSubject() + ".msg");
       
       try (FileOutputStream fop = new FileOutputStream(file)) {
           pst.saveMessageToStream(messageInfo.getEntryIdString(), fop);
       } catch (FileNotFoundException e) {
           // Εξαίρεση χειρισμού
       }
   }
   ```

### Αποθήκευση μηνυμάτων από PST σε ροή χρησιμοποιώντας αναγνωριστικά εισόδου

Αυτή η προσέγγιση αποθηκεύει μηνύματα χρησιμοποιώντας το `enumerateMessagesEntryId()` μέθοδος.

#### Επισκόπηση

Επαναλάβετε τα ID εισαγωγής μηνυμάτων και αποθηκεύστε το καθένα ως ροή, επιτρέποντας την αποτελεσματική μαζική επεξεργασία.

#### Βήματα Υλοποίησης

1. **Φόρτωση του αρχείου PST**
   
   ```java
   PersonalStorage pst = PersonalStorage.fromFile("YOUR_DOCUMENT_DIRECTORY/PersonalStorage.pst");
   ```
   
2. **Πρόσβαση στα Εισερχόμενα και Επανάληψη με βάση το ID Καταχώρησης**
   
   Χρησιμοποιήστε αναγνωριστικά καταχωρίσεων για να αποθηκεύσετε μηνύματα:
   ```java
   FolderInfo inbox = pst.getRootFolder().getSubFolder("myInbox");

   for (Object obj : inbox.enumerateMessagesEntryId()) {
       String entryId = (String) obj;
       pst.saveMessageToStream(entryId, new ByteArrayOutputStream());
   }
   ```
