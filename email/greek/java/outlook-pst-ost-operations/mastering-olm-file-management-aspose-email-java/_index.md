## Οδηγός Εφαρμογής

### Φόρτωση χώρου αποθήκευσης OLM

#### Επισκόπηση

Το πρώτο βήμα είναι η φόρτωση ενός αρχείου αποθήκευσης OLM χρησιμοποιώντας το Aspose.Email αρχικοποιώντας το `OlmStorage` κλάση με τη διαδρομή του αρχείου σας.

#### Οδηγός βήμα προς βήμα

**1. Ορίστε τη διαδρομή αρχείου:**

Ξεκινήστε καθορίζοντας τον κατάλογο όπου βρίσκεται το αρχείο OLM:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "SampleOLM.olm";
```

**2. Δημιουργήστε μια παρουσία του `OlmStorage`:**

Φορτώστε το αρχείο OLM χρησιμοποιώντας τη διαδρομή του:

```java
OlmStorage storage = new OlmStorage(dataDir);
```

#### Εξήγηση
- **`dataDir`**: Η διαδρομή προς το αρχείο OLM σας, απαραίτητη για την πρόσβαση και τη φόρτωση δεδομένων.
- **`new OlmStorage(dataDir)`**: Δημιουργεί ένα `OlmStorage` αντικείμενο, κρίσιμο για την εκτέλεση λειτουργιών στο φορτωμένο αρχείο OLM.

### Ανάκτηση ιεραρχίας φακέλων

#### Επισκόπηση

Μόλις φορτωθεί ο χώρος αποθήκευσης OLM, ανακτήστε την ιεραρχία των φακέλων του για να κατανοήσετε τη δομή των αποθηκευμένων μηνυμάτων ηλεκτρονικού ταχυδρομείου.

#### Οδηγός βήμα προς βήμα

**1. Φόρτωση OlmStorage:**

Υποθέστε ότι `OlmStorage` έχει ήδη αρχικοποιηθεί όπως φαίνεται προηγουμένως:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/SampleOLM.olm";
OlmStorage storage = new OlmStorage(dataDir);
```

**2. Ιεραρχία ανάκτησης φακέλων:**

Χρησιμοποιήστε τη μέθοδο για να λάβετε μια λίστα φακέλων:

```java
double folders = storage.getFolderHierarchy();
```

**3. Εκτύπωση αριθμού μηνυμάτων για κάθε φάκελο:**

Επαναλάβετε τους φακέλους και εμφανίστε τον αριθμό των μηνυμάτων τους:

```java
for (OlmFolder folder : folders) {
    System.out.println("Message Count [" + folder.getName() + "]: " + folder.getMessageCount());
}
```

#### Εξήγηση
- **`getFolderHierarchy()`**Ανακτά όλους τους φακέλους εντός του χώρου αποθήκευσης OLM για περαιτέρω εξερεύνηση.
- **`folder.getMessageCount()`**: Παρέχει έναν αριθμό μηνυμάτων σε κάθε φάκελο, χρήσιμο για γρήγορες πληροφορίες.

### Συμβουλές αντιμετώπισης προβλημάτων
