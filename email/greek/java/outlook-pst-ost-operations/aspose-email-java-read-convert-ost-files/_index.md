## Οδηγός Εφαρμογής

### Ανάγνωση αρχείου OST

Η πρώτη λειτουργία που θα εξερευνήσουμε είναι η ανάγνωση ενός αρχείου OST για την εμφάνιση των ονομάτων των υποφακέλων του.

#### Επισκόπηση

Αυτή η λειτουργικότητα σάς επιτρέπει να φορτώσετε ένα αρχείο OST του Microsoft Outlook και να παραθέσετε όλα τα ονόματα υποφακέλων που περιέχονται σε αυτό. Αυτό μπορεί να είναι ιδιαίτερα χρήσιμο για εργασίες ελέγχου ή μετεγκατάστασης δεδομένων.

#### Βήματα για την εφαρμογή

**1. Φορτώστε το αρχείο OST**

Ξεκινήστε ορίζοντας τη διαδρομή προς το αρχείο OST και φορτώνοντάς το χρησιμοποιώντας το Aspose.Email:

```java
import com.aspose.email.FolderInfo;
import com.aspose.email.FolderInfoCollection;
import com.aspose.email.PersonalStorage;

public class ReadOSTFeature {
    public static void main(String[] args) {
        // Ορίστε τη διαδρομή προς το αρχείο OST
        String strPSTFile = "YOUR_DOCUMENT_DIRECTORY/Sample.ost";
        
        // Φόρτωση του αρχείου PST (OST) του Outlook
        PersonalStorage pst = PersonalStorage.fromFile(strPSTFile);
    }
}
```

**2. Ανάκτηση και εμφάνιση υποφακέλων**

Μόλις φορτωθεί, αποκτήστε πρόσβαση στους υποφακέλους του ριζικού φακέλου και περιηγηθείτε σε αυτούς για να εμφανίσετε κάθε όνομα:

```java
// Ανάκτηση υποφακέλων του ριζικού φακέλου
FolderInfoCollection folderInfoCollection = pst.getRootFolder().getSubFolders();

// Επαναλάβετε την εκτέλεση κάθε υποφακέλου και εμφανίστε το όνομά του
for (int i = 0; i < folderInfoCollection.size(); i++) {
    FolderInfo folderInfo = (FolderInfo) folderInfoCollection.get_Item(i);
    System.out.println(folderInfo.getDisplayName());
}
```

#### Διαμόρφωση κλειδιού
- Ο `fromFile` μέθοδος `PersonalStorage` είναι απαραίτητο για τη φόρτωση του αρχείου OST.
- Πρόσβαση σε υποφακέλους μέσω `getSubFolders()` σας επιτρέπει να αλληλεπιδράτε με κάθε φάκελο ξεχωριστά.

### Μετατροπή OST σε PST

Τώρα, ας δούμε πώς να μετατρέψουμε ένα αρχείο OST σε μορφή PST.

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε τα αρχεία OST σας σε πιο ευέλικτη μορφή PST για διαφορετικά προγράμματα-πελάτες email ή για σκοπούς δημιουργίας αντιγράφων ασφαλείας.

#### Βήματα για την εφαρμογή

**1. Ορισμός διαδρομών εισόδου και εξόδου**

Καθορίστε τις διαδρομές τόσο για το αρχείο OST εισόδου όσο και για το αρχείο PST εξόδου:

```java
import com.aspose.email.FileFormat;
import com.aspose.email.PersonalStorage;

public class ConvertOSTToPSTFeature {
    public static void main(String[] args) {
        // Ορίστε τη διαδρομή για τα αρχεία εισόδου και εξόδου
        String inputFilePath = "YOUR_DOCUMENT_DIRECTORY/input.ost";
        String outputFilePath = "YOUR_OUTPUT_DIRECTORY/output.pst";
        
        // Φόρτωση του αρχείου OST
        PersonalStorage ost = PersonalStorage.fromFile(inputFilePath);
    }
}
```

**2. Εκτελέστε μετατροπή**

Μετατρέψτε το φορτωμένο αρχείο OST σε μορφή PST χρησιμοποιώντας το `saveAs` μέθοδος:

```java
// Αποθηκεύστε το φορτωμένο OST ως αρχείο PST στον καθορισμένο κατάλογο
ost.saveAs(outputFilePath, FileFormat.Pst);
```

#### Διαμόρφωση κλειδιού
- Ο `FileFormat.Pst` Η παράμετρος καθορίζει την επιθυμητή μορφή εξόδου.
- Βεβαιωθείτε ότι οι κατάλογοί σας έχουν ρυθμιστεί σωστά για να αποφύγετε σφάλματα στη διαδρομή αρχείου.
