## Οδηγός Υλοποίησης (aspose email java tutorial)

### Χαρακτηριστικό 1: Φόρτωση Αρχείου PST

#### Βήμα 1: Ορίστε τη Διαδρομή του Καταλόγου Σας
Καθορίστε πού βρίσκεται το αρχείο PST και ορίστε τη διαδρομή.

```java
String pstFilePath = "YOUR_DOCUMENT_DIRECTORY/Sub.pst";
```

#### Βήμα 2: Φορτώστε το Αρχείο PST

```java
PersonalStorage pst = PersonalStorage.fromFile(pstFilePath);
```

### Χαρακτηριστικό 2: Εξαγωγή Συνημμένων από Emails

#### Βήμα 1: Πρόσβαση στον Υποφάκελο Inbox

```java
FolderInfo inboxFolder = pst.getRootFolder().getSubFolder("Inbox");
```

#### Βήμα 2: Επανάληψη Μέσα στα Emails και Εξαγωγή Συνημμένων

```java
for (String entryId : inboxFolder.enumerateMessagesEntryId()) {
    MapiAttachmentCollection attachments = pst.extractAttachments(entryId);
    
    if (!attachments.isEmpty()) {
        for (MapiAttachment attachment : attachments) {
            String outputFilePath = "YOUR_OUTPUT_DIRECTORY/" + attachment.getLongFileName();
            attachment.save(outputFilePath); // Save each attachment
        }
    }
}
```

### Κύριες Επιλογές Διαμόρφωσης

- **Φάκελος Εξόδου:** Επαληθεύστε ότι ο φάκελος υπάρχει και ότι η εφαρμογή έχει δικαιώματα εγγραφής.  
- **Διαχείριση Σφαλμάτων:** Τυλίξτε τη λογική σε μπλοκ `try‑catch` για να χειριστείτε ευγενικά σφάλματα I/O ή κατεστραμμένα στοιχεία PST.  

### Συμβουλές Επίλυσης Προβλημάτων (πώς να εξάγετε pst συνημμένα)

- **Αρχείο δεν βρέθηκε:** Ελέγξτε ξανά τη συμβολοσειρά `pstFilePath`; χρησιμοποιήστε απόλυτες διαδρομές για αξιοπιστία.  
- **Θέματα δικαιωμάτων:** Εκτελέστε το JVM με τα κατάλληλα δικαιώματα συστήματος αρχείων ή επιλέξτε φάκελο εντός του προσωπικού καταλόγου του χρήστη.  
- **Μεγάλα αρχεία PST:** Εξετάστε την επεξεργασία μηνυμάτων σε παρτίδες και καλέστε `System.gc()` μετά από κάθε παρτίδα για απελευθέρωση μνήμης.

- **Διαχείριση Μνήμης:** Για PST μεγαλύτερα από 1 GB, αυξήστε το heap του JVM (`-Xmx2g` ή περισσότερο).  
- **Εξαγωγή σε Παρτίδες:** Επεξεργαστείτε περιορισμένο αριθμό μηνυμάτων ανά επανάληψη για να διατηρήσετε τη χρήση μνήμης χαμηλή.

## Συχνές Ερωτήσεις

**Ε:** *Τι είναι ένα αρχείο PST;*  
**Α:** Ένα αρχείο PST (Personal Storage Table) είναι αρχείο δεδομένων του Outlook που αποθηκεύει email, επαφές, στοιχεία ημερολογίου και συνημμένα.

**Ε:** *Μπορώ να εξάγω συνημμένα και από αρχεία OST;*  
**Α:** Ναι, το Aspose.Email υποστηρίζει τόσο PST όσο και OST. Χρησιμοποιήστε το ίδιο API· απλώς δείξτε το `PersonalStorage.fromFile` στο αρχείο OST.

**Ε:** *Πώς διαχειρίζομαι κρυπτογραφημένα αρχεία PST;*  
**Α:** Παρέχετε τον κωδικό πρόσβασης κατά το άνοιγμα του καταστήματος: `PersonalStorage.fromFile(pstFilePath, "password")`. Ανατρέξτε στην τεκμηρίωση του Aspose για λεπτομέρειες κρυπτογράφησης.

**Ε:** *Μπορώ να φιλτράρω ποια email θα επεξεργαστώ;*  
**Α:** Σίγουρα. Πριν καλέσετε `extractAttachments`, μπορείτε να ελέγξετε κάθε `MapiMessage` για θέμα, αποστολέα ή ημερομηνία και να παραλείψετε τα ανεπιθύμητα στοιχεία.

**Ε:** *Χρειάζεται άδεια για ανάπτυξη;*  
**Α:** Μια προσωρινή άδεια αρκεί για δοκιμές. Για παραγωγική χρήση, αγοράστε πλήρη άδεια ώστε να αφαιρεθούν οι περιορισμοί αξιολόγησης.
