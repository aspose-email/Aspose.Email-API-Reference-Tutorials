## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει βήμα προς βήμα στην εξαγωγή συνημμένων από αρχεία PST.

### Χαρακτηριστικό 1: Φόρτωση αρχείου PST

Η φόρτωση ενός αρχείου PST είναι το πρώτο βήμα για την πρόσβαση στο περιεχόμενό του. Δείτε πώς μπορείτε να το κάνετε:

#### Βήμα 1: Ορίστε τη διαδρομή καταλόγου σας
Προσδιορίστε πού βρίσκεται το αρχείο PST και ορίστε τη διαδρομή ανάλογα.
```java
String pstFilePath = "YOUR_DOCUMENT_DIRECTORY/Sub.pst";
```

#### Βήμα 2: Φόρτωση του αρχείου PST

Χρησιμοποιήστε το Aspose.Email `PersonalStorage` κλάση για να φορτώσετε το αρχείο PST.
```java
PersonalStorage pst = PersonalStorage.fromFile(pstFilePath);
```

### Λειτουργία 2: Εξαγωγή συνημμένων από ηλεκτρονικά μηνύματα

Μόλις φορτώσετε το PST, η εξαγωγή συνημμένων είναι απλή. Δείτε πώς:

#### Βήμα 1: Πρόσβαση στον υποφάκελο «Εισερχόμενα»

Ξεκινήστε αποκτώντας πρόσβαση στον φάκελο Εισερχόμενα όπου αποθηκεύονται τα περισσότερα μηνύματα ηλεκτρονικού ταχυδρομείου.
```java
FolderInfo inboxFolder = pst.getRootFolder().getSubFolder("Inbox");
```

#### Βήμα 2: Επαναλάβετε την επεξεργασία μέσω email και εξαγάγετε συνημμένα

Περιηγηθείτε σε κάθε καταχώρηση email στον φάκελο για να εξαγάγετε συνημμένα.
```java
for (String entryId : inboxFolder.enumerateMessagesEntryId()) {
    MapiAttachmentCollection attachments = pst.extractAttachments(entryId);
    
    if (!attachments.isEmpty()) {
        for (MapiAttachment attachment : attachments) {
            String outputFilePath = "YOUR_OUTPUT_DIRECTORY/" + attachment.getLongFileName();
            attachment.save(outputFilePath); // Αποθήκευση κάθε συνημμένου
        }
    }
}
```

## Ενότητα Συχνών Ερωτήσεων

1. **Τι είναι ένα αρχείο PST;**  
   Ένα αρχείο PST (Personal Storage Table - Πίνακας Προσωπικής Αποθήκευσης) είναι μια μορφή αρχείου δεδομένων του Outlook που χρησιμοποιείται για την αποθήκευση αντιγράφων μηνυμάτων, συμβάντων ημερολογίου και άλλων στοιχείων.
2. **Μπορώ να εξαγάγω συνημμένα και από αρχεία OST;**  
   Το Aspose.Email υποστηρίζει τόσο τις μορφές PST όσο και OST. Ελέγξτε την τεκμηρίωση για συγκεκριμένες μεθόδους χειρισμού αρχείων OST.
3. **Τι πρέπει να κάνω εάν η εφαρμογή μου παρουσιάσει σφάλμα κατά την επεξεργασία ενός μεγάλου αρχείου PST;**  
   Εξετάστε το ενδεχόμενο αύξησης της κατανομής μνήμης ή επεξεργασίας του PST σε μικρότερα τμήματα.
4. **Υπάρχει τρόπος να εξαγάγω συνημμένα μόνο από συγκεκριμένα email;**  
   Ναι, μπορείτε να φιλτράρετε τα email κατά θέμα, αποστολέα ή ημερομηνία πριν από την εξαγωγή συνημμένων.
5. **Πώς μπορώ να χειριστώ κρυπτογραφημένα αρχεία PST;**  
   Θα χρειαστεί να δώσετε τον κωδικό πρόσβασης κατά τη φόρτωση ενός κρυπτογραφημένου αρχείου PST. Ανατρέξτε στην τεκμηρίωση του Aspose.Email για οδηγίες σχετικά με τον χειρισμό της κρυπτογράφησης.
