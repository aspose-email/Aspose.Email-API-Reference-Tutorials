## Ρύθμιση του Aspose.Email για Java

### Εγκατάσταση μέσω Maven

Συμπεριλάβετε το Aspose.Email στο έργο σας χρησιμοποιώντας το Maven προσθέτοντας την ακόλουθη εξάρτηση στο `pom.xml` αρχείο:


### Βασική Αρχικοποίηση

Ξεκινήστε να χρησιμοποιείτε το Aspose.Email για Java αρχικοποιώντας ένα `MailMessage` αντικείμενο:

```java
import com.aspose.email.MailAddress;
import com.aspose.email.MailMessage;

// Αρχικοποίηση του αντικειμένου MailMessage
MailMessage message = new MailMessage();
message.setFrom(new MailAddress("sender@sender.com"));
message.getTo().addMailAddress(new MailAddress("receiver@gmail.com"));
```

## Οδηγός Εφαρμογής

### Δημιουργία και αποστολή μηνύματος ηλεκτρονικού ταχυδρομείου

**Επισκόπηση:** Αυτή η ενότητα καλύπτει τη δημιουργία μιας βασικής δομής email με διευθύνσεις αποστολέα και παραλήπτη.

#### Αρχικοποίηση του `MailMessage` Αντικείμενο

```java
// Ορισμός διεύθυνσης «Από»
message.setFrom(new MailAddress("sender@sender.com"));

// Προσθήκη διεύθυνσης «Προς»
message.getTo().addMailAddress(new MailAddress("receiver@gmail.com"));
```

### Επισύναψη αρχείων σε μήνυμα ηλεκτρονικού ταχυδρομείου

**Επισκόπηση:** Μάθετε πώς να επισυνάπτετε διαφορετικούς τύπους αρχείων, όπως κείμενο, εικόνες και έγγραφα, στα email σας.

#### Ορισμός διαδρομών καταλόγου για συνημμένα

Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY/"` με την πραγματική διαδρομή όπου είναι αποθηκευμένα τα αρχεία σας:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
```

#### Προσθήκη συνημμένων

Κάθε συνημμένο προστίθεται χρησιμοποιώντας το `getAttachments()` μέθοδος `MailMessage`:

```java
// Προσθήκη αρχείου κειμένου
Attachment textAttachment = new Attachment(dataDir + "1.txt");
message.getAttachments().addItem(textAttachment);

// Προσθήκη αρχείου εικόνας (μορφή JPEG)
message.getAttachments().addItem(new Attachment(dataDir + "1.jpg"));

// Προσθήκη εγγράφου Word
message.getAttachments().addItem(new Attachment(dataDir + "1.doc"));

// Προσθήκη αρχείου RAR
message.getAttachments().addItem(new Attachment(dataDir + "1.rar"));

// Προσθήκη εγγράφου PDF
message.getAttachments().addItem(new Attachment(dataDir + "1.pdf"));
```

### Αποθήκευση μηνύματος ηλεκτρονικού ταχυδρομείου σε δίσκο

**Επισκόπηση:** Αυτό το μέρος δείχνει πώς να αποθηκεύσετε το μήνυμα ηλεκτρονικού ταχυδρομείου, με όλα τα συνημμένα, ως αρχείο MSG.

#### Ορισμός διαδρομής καταλόγου εξόδου

Αντικαθιστώ `"YOUR_OUTPUT_DIRECTORY/"` με την επιθυμητή διαδρομή εξόδου:

```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/";
```

#### Αποθήκευση του μηνύματος ηλεκτρονικού ταχυδρομείου

Χρησιμοποιήστε το `save()` μέθοδος για την εγγραφή του email στο δίσκο:

```java
message.save(outputDir + "AddAttachmentToANewEmailMessage_out.msg", SaveOptions.getDefaultMsg());
```
