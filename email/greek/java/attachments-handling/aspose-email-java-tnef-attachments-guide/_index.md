## Οδηγός Εφαρμογής
### Προσθήκη νέων συνημμένων στο κύριο μήνυμα που περιέχει TNEF
**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να προσθέσετε ένα νέο συνημμένο σε ένα υπάρχον μήνυμα EML που περιέχει δεδομένα TNEF.

#### Βήμα 1: Φόρτωση του υπάρχοντος μηνύματος ηλεκτρονικού ταχυδρομείου
Φορτώστε το μήνυμα ηλεκτρονικού ταχυδρομείου σας χρησιμοποιώντας `MailMessage.load()`, καθορίζοντας τη διαδρομή προς το αρχείο EML σας.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MailMessage eml = MailMessage.load(dataDir + "MainMessage.eml");
```

#### Βήμα 2: Προσθήκη του νέου συνημμένου
Ανοίξτε ένα νέο συνημμένο και προσθέστε το στη συλλογή συνημμένων του email. Εδώ, προσθέτουμε μια εικόνα ως παράδειγμα.

```java
try (FileInputStream fi = new FileInputStream(dataDir + "barcode.png")) {
    eml.getAttachments().addItem(new Attachment(fi, "barcode.png", "image/png"));
}
```

#### Βήμα 3: Αποθήκευση του τροποποιημένου μηνύματος ηλεκτρονικού ταχυδρομείου
Αφού προσθέσετε το συνημμένο, αποθηκεύστε το τροποποιημένο μήνυμα ηλεκτρονικού ταχυδρομείου σε ένα νέο αρχείο.

```java
eml.save(dataDir + "test_out.eml");
```
**Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι όλες οι διαδρομές αρχείων είναι σωστές και προσβάσιμες. Οι λανθασμένες διαδρομές θα οδηγήσουν σε `FileNotFoundException`.

### Δημιουργία TNEF EML από MSG
**Επισκόπηση:**
Μετατρέψτε ένα υπάρχον αρχείο MSG σε μορφή EML με δυνατότητα TNEF.

#### Βήμα 1: Φόρτωση του αρχείου MSG
Χρήση `MapiMessage.fromFile()` για να φορτώσετε το αρχείο MSG σας, προετοιμάζοντάς το για μετατροπή.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MapiMessage msg = MapiMessage.fromFile(dataDir + "Message.msg");
```

#### Βήμα 2: Ορισμός επιλογών μετατροπής
Καθορίστε ότι το μήνυμα θα πρέπει να μετατραπεί ως TNEF χρησιμοποιώντας `MailConversionOptions`.

```java
MailConversionOptions options = new MailConversionOptions();
options.setConvertAsTnef(true);
```

#### Βήμα 3: Μετατροπή και αποθήκευση
Εκτελέστε τη μετατροπή από MSG σε μορφή EML με δυνατότητα TNEF.

```java
MailMessage mail = msg.toMailMessage(options);
mail.save(dataDir + "converted_message.eml");
```

### Διατήρηση συνημμένων TNEF κατά τη φόρτωση αρχείων EML
**Επισκόπηση:**
Διατηρήστε τα συνημμένα TNEF κατά τη φόρτωση ενός υπάρχοντος αρχείου EML.

#### Βήμα 1: Ορισμός επιλογών φόρτωσης
Χρήση `MsgLoadOptions` για τη διατήρηση των συνημμένων TNEF κατά τη διαδικασία φόρτωσης.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MsgLoadOptions msgLoadOptions = new MsgLoadOptions();
msgLoadOptions.setPreserveTnefAttachments(true);
```

#### Βήμα 2: Φόρτωση αρχείου EML με επιλογές
Φορτώστε το αρχείο EML χρησιμοποιώντας τις καθορισμένες επιλογές για να διατηρήσετε τα συνημμένα TNEF.

```java
MailMessage eml = MailMessage.load(dataDir + "test.eml", msgLoadOptions);
```

### Εντοπισμός εάν ένα μήνυμα είναι TNEF
**Επισκόπηση:**
Ελέγξτε εάν ένα μήνυμα ηλεκτρονικού ταχυδρομείου περιέχει δεδομένα TNEF.

#### Βήμα 1: Φόρτωση του αρχείου EML
Φορτώστε το αρχείο email σας για να ελέγξετε για παρουσία TNEF.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MailMessage mail = MailMessage.load(dataDir + "test.eml");
```

#### Βήμα 2: Εντοπισμός παρουσίας TNEF
Χρήση `getOriginalIsTnef()` μέθοδος για να προσδιορίσετε εάν το μήνυμα περιέχει δεδομένα TNEF.

```java
boolean isTnef = mail.getOriginalIsTnef();
system.out.println("Is TNEF: " + isTnef);
```
