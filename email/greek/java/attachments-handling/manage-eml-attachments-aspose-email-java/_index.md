## Οδηγός Εφαρμογής

Ας εξερευνήσουμε κάθε χαρακτηριστικό βήμα προς βήμα.

### Φόρτωση αρχείου EML

#### Επισκόπηση
Μάθετε πώς να φορτώσετε ένα υπάρχον αρχείο EML χρησιμοποιώντας το Aspose.Email για Java. Θα χρησιμοποιήσουμε το `MailMessage.load` μέθοδος με καθορισμένη διαδρομή και `EmlLoadOptions`.

#### Απόσπασμα κώδικα

```java
import com.aspose.email.EmlLoadOptions;
import com.aspose.email.MailMessage;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
MailMessage msg = MailMessage.load(dataDir + "EmailWithAttachment.eml", new EmlLoadOptions());
```

**Εξήγηση**: 
- `dataDir` είναι το σημείο όπου βρίσκεται το αρχείο EML σας.
- `EmlLoadOptions` παρέχει πρόσθετες επιλογές για τη φόρτωση μηνυμάτων ηλεκτρονικού ταχυδρομείου.

### Αρχικοποίηση Συλλογής Συνημμένων

#### Επισκόπηση
Μόλις φορτώσετε το αρχείο EML, αρχικοποιήστε ένα `AttachmentCollection` για να αποκτήσετε πρόσβαση στα συνημμένα του.

#### Απόσπασμα κώδικα

```java
import com.aspose.email.AttachmentCollection;

AttachmentCollection attachments = msg.getAttachments();
```

**Εξήγηση**: 
- Ο `getAttachments()` Η μέθοδος ανακτά όλα τα συνημμένα από το μήνυμα ηλεκτρονικού ταχυδρομείου.

### Επανάληψη Συνημμένων και Ονομάτων Εμφάνισης

#### Επισκόπηση
Επαναλάβετε την περιήγηση σε κάθε συνημμένο στη συλλογή για να εμφανίσετε το όνομά του, κάτι που είναι χρήσιμο για την καταγραφή ή την επαλήθευση.

#### Απόσπασμα κώδικα

```java
import com.aspose.email.Attachment;

for (int index = 0; index < attachments.size(); index++) {
    Attachment attachment = (Attachment) attachments.get_Item(index);
    System.out.println(attachment.getName());
}
```

**Εξήγηση**: 
- Χρησιμοποιήστε έναν βρόχο για να αποκτήσετε πρόσβαση σε κάθε συνημμένο με βάση τον δείκτη του.
- Ο `getName()` Η μέθοδος ανακτά το όνομα του συνημμένου.

### Αποθήκευση συνημμένων σε δίσκο

#### Επισκόπηση
Μάθετε πώς να αποθηκεύετε συνημμένα από ένα μήνυμα ηλεκτρονικού ταχυδρομείου στον δίσκο, κάτι που είναι απαραίτητο για την επεξεργασία ή την αρχειοθέτηση.

#### Απόσπασμα κώδικα

```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";

for (int index = 0; index < attachments.size(); index++) {
    Attachment attachment = (Attachment) attachments.get_Item(index);
    attachment.save(outputDir + "/attachment_" + attachment.getName());
}
```

**Εξήγηση**: 
- `outputDir` είναι το σημείο όπου θέλετε να αποθηκεύσετε τα συνημμένα.
- Ο `save()` Η μέθοδος γράφει κάθε συνημμένο σε ένα αρχείο με ένα μοναδικό όνομα.
