## Οδηγός Εφαρμογής

### Εισαγωγή συνημμένου MSG σε συγκεκριμένη θέση

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να εισαγάγετε ένα νέο συνημμένο σε ένα υπάρχον αρχείο .MSG σε μια καθορισμένη θέση. Αυτό είναι ιδιαίτερα χρήσιμο όταν η σειρά των συνημμένων έχει σημασία για λόγους παρουσίασης ή συμμόρφωσης.

#### Οδηγίες βήμα προς βήμα

**1. Φορτώστε το υπάρχον αρχείο MSG**

Φορτώστε το αρχείο MSG που περιέχει ενσωματωμένα συνημμένα:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/outlook/";
MapiMessage msg = MapiMessage.fromFile(dataDir + "WithEmbeddedMsg.msg");
```

**2. Αποθήκευση συνημμένου για επίδειξη**

Πριν εισαγάγουμε ένα νέο συνημμένο, ας αποθηκεύσουμε το πρώτο για λόγους επίδειξης:

```java
msg.getAttachments().get_Item(0).save("YOUR_OUTPUT_DIRECTORY" + "/attachment_out.msg");
```

**3. Φόρτωση ενός άλλου αρχείου MSG**

Προετοιμάστε ένα άλλο αρχείο MSG που θέλετε να εισαγάγετε ως συνημμένο:

```java
MapiMessage emb = MapiMessage.fromStream(new FileInputStream(dataDir + "WithEmbeddedMsg.msg"));
```

**4. Εισαγωγή του νέου συνημμένου**

Εισαγάγετε αυτό το νέο αρχείο MSG στο ευρετήριο 1 στην υπάρχουσα συλλογή συνημμένων σας:

```java
msg.getAttachments().insert(1, "new 11", emb);
```

**5. Αποθηκεύστε το τροποποιημένο αρχείο MSG**

Τέλος, αποθηκεύστε το ενημερωμένο αρχείο MSG στον κατάλογο εξόδου σας:

```java
msg.save("YOUR_OUTPUT_DIRECTORY" + "/insertMSGAttachment_out.msg");
```

### Αντικατάσταση περιεχομένων ενσωματωμένου συνημμένου MSG

#### Επισκόπηση

Η αντικατάσταση ενός υπάρχοντος συνημμένου διασφαλίζει ότι μπορείτε να ενημερώσετε το περιεχόμενο ενός email χωρίς να αλλάξετε τη συνολική του δομή.

#### Οδηγίες βήμα προς βήμα

**1. Φορτώστε το αρχείο MSG με τα συνημμένα**

Ξεκινήστε φορτώνοντας το αρχείο MSG που περιέχει τα συνημμένα:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/outlook/";
MapiMessage msg = MapiMessage.fromFile(dataDir + "insertMSGAttachment_out.msg");
```

**2. Αποθήκευση υπάρχοντος συνημμένου**

Για επίδειξη, αποθηκεύστε ένα από τα υπάρχοντα συνημμένα:

```java
msg.getAttachments().get_Item(0).save("YOUR_OUTPUT_DIRECTORY" + "/attachment_out.msg");
```

**3. Φορτώστε ένα νέο αρχείο MSG για αντικατάσταση**

Φορτώστε ένα άλλο αρχείο MSG που θα αντικαταστήσει το τρέχον συνημμένο:

```java
MapiMessage emb = MapiMessage.fromStream(new FileInputStream(dataDir + "insertMSGAttachment_out.msg"));
```

**4. Επανατοποθετήστε το εξάρτημα**

Αντικαταστήστε το συνημμένο στο ευρετήριο 1 με αυτό το νέο:

```java
msg.getAttachments().replace(1, "new 1", emb);
```

**5. Αποθήκευση αλλαγών στο αρχείο MSG**

Αποθηκεύστε τις αλλαγές σας για να διατηρήσετε την ενημερωμένη δομή:

```java
msg.save("YOUR_OUTPUT_DIRECTORY" + "/replaceEmbeddedMSGAttachment_out.msg");
```
