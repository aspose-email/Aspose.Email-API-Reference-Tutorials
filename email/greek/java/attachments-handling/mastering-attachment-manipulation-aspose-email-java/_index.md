## Οδηγός Υλοποίησης

### Εισαγωγή Συνημμένου MSG σε Συγκεκριμένη Θέση

#### Επισκόπηση

Αυτή η λειτουργία σας επιτρέπει να **add attachment to msg** σε ακριβή θέση — χρήσιμο όταν η σειρά των συνημμένων έχει σημασία για τη συμμόρφωση ή την παρουσίαση.

#### Οδηγίες Βήμα‑Βήμα

**1. Φόρτωση του Υπάρχοντος Αρχείου MSG**  

Φορτώστε το αρχείο MSG που ήδη περιέχει ενσωματωμένα συνημμένα:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/outlook/";
MapiMessage msg = MapiMessage.fromFile(dataDir + "WithEmbeddedMsg.msg");
```

**2. Αποθήκευση Συνημμένου για Επίδειξη**  

Θα εξάγουμε το πρώτο συνημμένο ώστε να δείτε τι μετακινείται:

```java
msg.getAttachments().get_Item(0).save("YOUR_OUTPUT_DIRECTORY" + "/attachment_out.msg");
```

**3. Φόρτωση Άλλου Αρχείου MSG**  

Προετοιμάστε το αρχείο MSG που θέλετε να εισάγετε ως νέο συνημμένο:

```java
MapiMessage emb = MapiMessage.fromStream(new FileInputStream(dataDir + "WithEmbeddedMsg.msg"));
```

**4. Εισαγωγή του Νέου Συνημμένου**  

Εισάγετε το νέο αρχείο MSG στη θέση index 1 στη συλλογή των συνημμένων:

```java
msg.getAttachments().insert(1, "new 11", emb);
```

**5. Αποθήκευση του Τροποποιημένου Αρχείου MSG**  

Αποθηκεύστε τις αλλαγές σε ένα νέο αρχείο:

```java
msg.save("YOUR_OUTPUT_DIRECTORY" + "/insertMSGAttachment_out.msg");
```

### Αντικατάσταση Περιεχομένου Ενσωματωμένου Συνημμένου MSG

#### Επισκόπηση

Όταν το περιεχόμενο ενός συνημμένου email χρειάζεται ενημέρωση, μπορείτε να **how to replace attachment** χωρίς να αλλάξετε τη δομή του γύρω μηνύματος.

#### Οδηγίες Βήμα‑Βήμα

**1. Φόρτωση του Αρχείου MSG με Συνημμένα**  

Ανοίξτε το αρχείο MSG που ήδη περιέχει το συνημμένο που σκοπεύετε να αντικαταστήσετε:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/outlook/";
MapiMessage msg = MapiMessage.fromFile(dataDir + "insertMSGAttachment_out.msg");
```

**2. Αποθήκευση Υπάρχοντος Συνημμένου**  

Εξάγετε ένα από τα τρέχοντα συνημμένα για αναφορά:

```java
msg.getAttachments().get_Item(0).save("YOUR_OUTPUT_DIRECTORY" + "/attachment_out.msg");
```

**3. Φόρτωση Νέου Αρχείου MSG για Αντικατάσταση**  

Φορτώστε το αρχείο MSG που θα γίνει το νέο συνημμένο:

```java
MapiMessage emb = MapiMessage.fromStream(new FileInputStream(dataDir + "insertMSGAttachment_out.msg"));
```

**4. Αντικατάσταση του Συνημμένου**  

Αντικαταστήστε το παλιό συνημμένο στη θέση index 1 με το νέο:

```java
msg.getAttachments().replace(1, "new 1", emb);
```

**5. Αποθήκευση Αλλαγών στο Αρχείο MSG**  

Γράψτε το ενημερωμένο μήνυμα ξανά στο δίσκο:

```java
msg.save("YOUR_OUTPUT_DIRECTORY" + "/replaceEmbeddedMSGAttachment_out.msg");
```

## Συνηθισμένα Πιθανά Σφάλματα & Επίλυση Προβλημάτων

- **Invalid Index** – Η προσπάθεια εισαγωγής ή αντικατάστασης σε δείκτη που δεν υπάρχει προκαλεί `ArgumentOutOfRangeException`. Πάντα ελέγξτε πρώτα το `msg.getAttachments().size()`.
- **Stream Leaks** – Η παράλειψη κλεισίματος των αντικειμένων `FileInputStream` μπορεί να οδηγήσει σε εξάντληση των χειριστών αρχείων. Χρησιμοποιήστε try‑with‑resources για αυτόματο κλείσιμο.
- **License Not Set** – Η εκτέλεση χωρίς έγκυρη άδεια θα προσθέσει υδατογραφήματα αξιολόγησης στην έξοδο. Βεβαιωθείτε ότι το `license.setLicense(...)` καλείται πριν από οποιαδήποτε χρήση API.
