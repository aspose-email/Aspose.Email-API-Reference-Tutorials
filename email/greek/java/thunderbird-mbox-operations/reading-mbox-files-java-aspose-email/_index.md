## Οδηγός Εφαρμογής
### Ανάγνωση αρχείων MBOX με το Aspose.Email για Java
Ας αναλύσουμε τη διαδικασία ανάγνωσης μηνυμάτων από ένα αρχείο MBOX σε σαφή βήματα:

#### Βήμα 1: Ανοίξτε το αρχείο MBOX
Ξεκινήστε ανοίγοντας το αρχείο MBOX χρησιμοποιώντας `FileInputStream`Αυτό το βήμα περιλαμβάνει τον καθορισμό της διαδρομής προς το αρχείο MBOX.
```java
import java.io.FileInputStream;
import com.aspose.email.MailMessage;
import com.aspose.email.MboxrdStorageReader;

String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/ExampleMbox.mbox"; // Αντικατάσταση με την πραγματική διαδρομή

FileInputStream stream = new FileInputStream(dataDir);
```

#### Βήμα 2: Δημιουργήστε το MBOX Reader
Στη συνέχεια, δημιουργήστε ένα `MboxrdStorageReader` παράδειγμα για την ανάγνωση μηνυμάτων. Η παράμετρος `false` υποδεικνύει ότι δεν χρησιμοποιείται κωδικοποίηση Unicode.
```java
MboxrdStorageReader reader = new MboxrdStorageReader(stream, false);
```

#### Βήμα 3: Διαβάστε κάθε μήνυμα
Εκτελέστε επανάληψη σε κάθε μήνυμα στο αρχείο MBOX και επεξεργαστείτε το όπως απαιτείται. Εδώ, θα εκτυπώσουμε το μέγεθος κάθε μηνύματος email σε byte:
```java
MailMessage msg = null;

while ((msg = reader.readNextMessage()) != null) {
    long currentDataSize = reader.getCurrentDataSize(); // Λήψη του μεγέθους του τρέχοντος μηνύματος

    // Μπορείτε να καταγράψετε ή να εκτυπώσετε το μέγεθος των δεδομένων εδώ
    System.out.println("Email Size: " + currentDataSize + " bytes");
    
    msg.dispose(); // Απορρίψτε κάθε MailMessage για να ελευθερώσετε πόρους
}
```

### Επεξήγηση Παραμέτρων και Μεθόδων
- `FileInputStream`Ανοίγει μια σύνδεση με το αρχείο MBOX σας.
- `MboxrdStorageReader`Διευκολύνει την ανάγνωση μηνυμάτων από τη μορφή MBOX.
- `readNextMessage()`: Ανακτά το επόμενο μήνυμα στο αρχείο MBOX. Επιστρέφει την τιμή null όταν δεν υπάρχουν άλλα διαθέσιμα μηνύματα.
- `getCurrentDataSize()`: Παρέχει το μέγεθος του τρέχοντος μηνύματος ηλεκτρονικού ταχυδρομείου, χρήσιμο για ανάγκες καταγραφής ή επεξεργασίας.
