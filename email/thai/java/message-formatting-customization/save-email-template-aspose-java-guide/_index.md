### คู่มือการใช้งาน

มาสำรวจวิธีการบันทึกอีเมลเป็นเทมเพลตกัน

#### การสร้างและการบันทึกเทมเพลตอีเมล

**ภาพรวม:** ส่วนนี้จะครอบคลุมถึงการสร้าง `MailMessage` อินสแตนซ์ที่มีรายละเอียดผู้ส่ง ผู้รับ หัวเรื่อง และเนื้อหา ก่อนที่จะบันทึกในรูปแบบ OFT

**ขั้นตอนที่ 1: สร้าง MailMessage**

เราเริ่มต้นด้วยการเริ่มต้น `MailMessage` วัตถุ:

```java
import com.aspose.email.MailMessage;
import com.aspose.email.MsgSaveOptions;

// เริ่มต้นอินสแตนซ์ MailMessage ใหม่
MailMessage eml = new MailMessage("test@from.to", "test@to.com");
eml.setSubject("Test Email Template");
eml.setBody("This is an example email body.");
```

**ขั้นตอนที่ 2: บันทึกเป็น OFT**

หากต้องการบันทึกข้อความนี้ในรูปแบบ OFT ให้ใช้ `MsgSaveOptions`-

```java
// กำหนดตัวเลือกการบันทึกสำหรับรูปแบบ OFT
MsgSaveOptions saveOptions = SaveOptions.getDefaultOft();

// บันทึก MailMessage เป็นรูปแบบ OFT
eml.save("output.oft", saveOptions);
```

**คำอธิบาย:** 
- **เมลข้อความ**คลาสนี้จะรวมข้อความอีเมล์ รวมถึงรายละเอียด เช่น ผู้ส่ง ผู้รับ หัวเรื่อง และเนื้อหา
- **ตัวเลือกการบันทึกข้อความ**: ให้ตัวเลือกสำหรับการบันทึกข้อความในรูปแบบต่างๆ ที่นี่เราใช้ `getDefaultOft()` เพื่อระบุรูปแบบ OFT

### การประยุกต์ใช้งานจริง

การบันทึกอีเมลเป็นเทมเพลตนั้นมีประโยชน์ในหลายสถานการณ์:
1. **แคมเปญอีเมล์อัตโนมัติ:** สร้างอีเมลส่วนบุคคลได้อย่างรวดเร็วเพื่อวัตถุประสงค์ทางการตลาดโดยไม่ต้องกำหนดส่วนหัวและส่วนท้ายใหม่
2. **ระบบสนับสนุนลูกค้า:** สร้างมาตรฐานคำตอบในขณะที่ยังอนุญาตให้ปรับแต่งสำหรับการสอบถามที่เฉพาะเจาะจงได้
3. **การสื่อสารภายใน:** รักษาความสอดคล้องในการสื่อสารขององค์กรด้วยการใช้โครงสร้างอีเมลที่กำหนดไว้ล่วงหน้า
