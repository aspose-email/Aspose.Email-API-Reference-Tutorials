## คู่มือการใช้งาน
### การสร้างข้อความอีเมล์
การสร้างข้อความอีเมลเกี่ยวข้องกับการกำหนดเนื้อหาและรายละเอียดของผู้รับ นี่คือวิธีดำเนินการนี้โดยใช้ Aspose.Email สำหรับ Java:
#### ภาพรวม
หัวข้อนี้ครอบคลุมถึงการสร้างข้อความอีเมลโดยระบุผู้ส่ง ผู้รับ เนื้อหา HTML และการเข้ารหัส
##### ขั้นตอนที่ 1: ประกาศอินสแตนซ์ MailMessage ใหม่
เริ่มต้นด้วยการสร้างตัวอย่าง `MailMessage` คลาสซึ่งแสดงข้อความอีเมล์ของคุณ
```java
import com.aspose.email.MailAddress;
import com.aspose.email.MailMessage;

// ประกาศอินสแตนซ์ใหม่ของ MailMessage
MailMessage message = new MailMessage();
```
##### ขั้นตอนที่ 2: ตั้งค่าผู้ส่งและผู้รับ
กำหนดที่อยู่ผู้ส่งโดยใช้ `setFrom()` และใส่ที่อยู่ผู้รับด้วย `getTo()-add()`.
```java
// ตั้งค่าที่อยู่อีเมลของผู้ส่ง
message.setFrom(new MailAddress("sender@sender.com"));

// เพิ่มที่อยู่อีเมลผู้รับ
message.getTo().add("receiver@receiver.com");
```
##### ขั้นตอนที่ 3: กำหนดเนื้อหาและการเข้ารหัส HTML
ตั้งค่าเนื้อหา HTML ของข้อความของคุณโดยใช้ `setHtmlBody()` และระบุการเข้ารหัสเพื่อแสดงอักขระได้อย่างแม่นยำ
```java
// ตั้งค่าเนื้อหา HTML ของข้อความ
message.setHtmlBody("<html><body>This is the Html body</body></html>");

// ระบุการเข้ารหัสสำหรับเนื้อหาอีเมล
message.setBodyEncoding(java.nio.charset.Charset.forName("US-ASCII"));
```
### การกำหนดค่าไคลเอนต์ SMTP และการส่งอีเมล
การกำหนดค่าไคลเอนต์ SMTP ช่วยให้คุณสามารถส่งข้อความที่คุณสร้างขึ้นผ่านเครือข่ายได้
#### ภาพรวม
หัวข้อนี้สาธิตการกำหนดค่าการตั้งค่า SMTP เช่น โฮสต์ ชื่อผู้ใช้ รหัสผ่าน พอร์ต และการส่งอีเมล
##### ขั้นตอนที่ 1: สร้างอินสแตนซ์ของ SmtpClient
เริ่มต้นด้วยการสร้างตัวอย่าง `SmtpClient`ซึ่งรับผิดชอบในการส่งอีเมล์
```java
import com.aspose.email.SmtpClient;

// สร้างอินสแตนซ์ของ SmtpClient
SmtpClient client = new SmtpClient();
```
##### ขั้นตอนที่ 2: กำหนดค่าการตั้งค่า SMTP
ตั้งค่ารายละเอียดเซิร์ฟเวอร์ SMTP ของคุณรวมทั้งโฮสต์ ข้อมูลประจำตัว และพอร์ต
```java
// ตั้งค่าโฮสต์เซิร์ฟเวอร์ SMTP
client.setHost("smtp.server.com");

// ระบุชื่อผู้ใช้สำหรับการยืนยันตัวตน
client.setUsername("Username");

// ระบุรหัสผ่านสำหรับการยืนยันตัวตน
client.setPassword("Password");

// ตั้งค่าพอร์ตเซิร์ฟเวอร์ SMTP (ค่าเริ่มต้นคือ 25)
client.setPort(25);
```
##### ขั้นตอนที่ 3: ส่งข้อความอีเมล์
สุดท้ายใช้ `send()` วิธีการส่งข้อความอีเมล์ของคุณ
```java
try {
    // ส่งข้อความโดยใช้ไคลเอนต์ที่กำหนดค่าไว้
    client.send(message);
} catch (Exception e) {
    System.out.println("Error sending email: " + e.getMessage());
}
```
### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่ารายละเอียดเซิร์ฟเวอร์ SMTP ถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่าไฟร์วอลล์หรือการตั้งค่าเครือข่ายของคุณอนุญาตการเชื่อมต่อขาออกบนพอร์ตที่คุณระบุ
