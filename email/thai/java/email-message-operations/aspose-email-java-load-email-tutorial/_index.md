## คู่มือการใช้งาน

### โหลดข้อความอีเมลจากไฟล์

ฟีเจอร์นี้สาธิตการโหลดข้อความอีเมลที่เก็บไว้ใน `.msg` ไฟล์ นี่คือวิธีการใช้งาน:

#### ภาพรวมของคุณสมบัติ

การโหลดอีเมลเป็นสิ่งสำคัญสำหรับการประมวลผลหรือวิเคราะห์ข้อมูลอีเมล Aspose.Email นำเสนอวิธีการที่ตรงไปตรงมาเพื่อให้บรรลุเป้าหมายนี้โดยใช้โค้ดน้อยที่สุด

#### การดำเนินการแบบทีละขั้นตอน

##### 1. ระบุไดเรกทอรีเอกสารของคุณ

กำหนดเส้นทางที่คุณ `.msg` ไฟล์จะถูกเก็บไว้:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

แทนที่ `YOUR_DOCUMENT_DIRECTORY` พร้อมด้วยเส้นทางไดเร็กทอรีจริงที่มีไฟล์อีเมลของคุณ

##### 2. โหลดข้อความจากไฟล์ .msg

ใช้ `MailMessage.load()` วิธีการอ่านไฟล์อีเมล์ลงในแอปพลิเคชันของคุณ:

```java
// สร้างอินสแตนซ์ของ MsgLoadOptions หากคุณต้องการตัวเลือกการโหลดที่เฉพาะเจาะจง
MsgLoadOptions loadOptions = new MsgLoadOptions();

// โหลดข้อความโดยใช้เส้นทางและตัวเลือกการโหลดเสริม
MailMessage originalMsg = MailMessage.load(dataDir + "Message.msg", loadOptions);
```

**คำอธิบาย**: เดอะ `load()` วิธีการอ่านไฟล์อีเมลและส่งคืน `MailMessage` วัตถุซึ่งคุณสามารถจัดการหรือดึงข้อมูลออกมาได้ ปรับแต่งพฤติกรรมการโหลดด้วย `MsgLoadOptions`-

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีของคุณถูกต้องเพื่อหลีกเลี่ยง `FileNotFoundException`-
- ตรวจสอบว่า `.msg` ไฟล์ไม่เสียหาย
