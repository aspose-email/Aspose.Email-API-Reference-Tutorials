### คู่มือการใช้งาน
มาแบ่งคุณสมบัติแต่ละอย่างออกเป็นขั้นตอนโดยละเอียด

#### การเชื่อมต่อกับ Exchange Web Service (EWS)
**ภาพรวม**การสร้างการเชื่อมต่อกับ EWS ถือเป็นขั้นตอนแรกในการใช้ประโยชน์จากความสามารถของ Aspose.Email สำหรับ Java ซึ่งช่วยให้คุณสามารถดำเนินการต่างๆ กับกล่องจดหมายของคุณได้ เช่น การอ่านข้อความหรือการส่งอีเมล

##### ขั้นตอนที่ 1: สร้างอินสแตนซ์ของ IEWSClient
```java
import com.aspose.email.EWSClient;
import com.aspose.email.IEWSClient;

public class ConnectToExchange {
    public static void main(String[] args) {
        // สร้างอินสแตนซ์เพื่อเชื่อมต่อกับเซิร์ฟเวอร์ EWS
        // พารามิเตอร์: URL, ชื่อผู้ใช้, รหัสผ่าน
        IEWSClient client = EWSClient.getEWSClient("exchange.domain.com/exchangeews/Exchange.asmx", "username", "password", "");
    }
}
```

- **พารามิเตอร์**-
  - `URL`: จุดสิ้นสุดสำหรับบริการ Exchange ของคุณ
  - `username` และ `password`: ข้อมูลประจำตัวสำหรับการยืนยันตัวตน

#### การเพิ่มส่วนหัวที่กำหนดเองในคำขอ EWS
**ภาพรวม**:ส่วนหัวแบบกำหนดเองอาจมีความสำคัญต่อการเพิ่มข้อมูลเมตาหรือการควบคุมด้านต่างๆ ของคำขอที่คุณส่งไปยังเซิร์ฟเวอร์ EWS ซึ่งจะช่วยเพิ่มการทำงาน เช่น การยึดข้อความ

##### ขั้นตอนที่ 2: เพิ่มส่วนหัวที่กำหนดเอง
```java
import com.aspose.email.EWSClient;
import com.aspose.email.IEWSClient;

public class AddCustomHeaders {
    public static void main(String[] args) {
        // เชื่อมต่อกับเซิร์ฟเวอร์ EWS
        IEWSClient client = EWSClient.getEWSClient("exchange.domain.com/exchangeews/Exchange.asmx", "username", "password", "");
        
        // เพิ่มส่วนหัวแบบกำหนดเองเพื่อการจัดการคำขอที่ได้รับการปรับปรุง
        client.addHeader("X-AnchorMailbox", "username@domain.com");
    }
}
```

- **การกำหนดค่าคีย์**: เดอะ `X-AnchorMailbox` ส่วนหัวช่วยในสถานการณ์ที่จำเป็นต้องรักษาสถานะของกล่องจดหมายไว้ระหว่างการดำเนินการ

#### การแสดงรายการข้อความในกล่องจดหมาย EWS
**ภาพรวม**หากต้องการโต้ตอบกับอีเมลของคุณ ขั้นแรกคุณต้องเข้าถึงและแสดงรายการข้อความในกล่องจดหมาย การดำเนินการนี้มีความสำคัญพื้นฐานสำหรับงานการประมวลผลหรือการจัดการที่ตามมา

##### ขั้นตอนที่ 3: ดึงข้อมูลและรายการข้อความ
```java
import com.aspose.email.EWSClient;
import com.aspose.email.ExchangeMessageInfoCollection;
import com.aspose.email.IEWSClient;

public class ListInboxMessages {
    public static void main(String[] args) {
        // สร้างการเชื่อมต่อกับเซิร์ฟเวอร์ EWS
        IEWSClient client = EWSClient.getEWSClient("exchange.domain.com/exchangeews/Exchange.asmx", "username", "password", "");
        
        // รับ URI กล่องจดหมายจากข้อมูลกล่องจดหมาย
        String inboxUri = client.getMailboxInfo().getInboxUri();
        
        // รายการข้อความทั้งหมดในกล่องจดหมาย
        ExchangeMessageInfoCollection messageInfoCol = client.listMessages(inboxUri);
    }
}
```

- **การกำหนดค่าคีย์**: เดอะ `listMessages` วิธีการดึงคอลเลกชันของวัตถุข้อมูลข้อความจากโฟลเดอร์ URI ที่ระบุ
