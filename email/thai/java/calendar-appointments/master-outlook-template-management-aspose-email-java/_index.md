# แปลง oft เป็น msg – การจัดการเทมเพลต Outlook อย่างเชี่ยวชาญด้วย Aspose.Email สำหรับ Java

ในคู่มือฉบับครอบคลุมนี้คุณจะได้ค้นพบ **วิธีแปลง OFT เป็น MSG** ปรับปรุงคุณสมบัติเครื่องมือเทมเพลต Outlook และบันทึกไฟล์ MSG ของเทมเพลต Outlook — ทั้งหมดนี้ด้วยไลบรารี Aspose.Email ที่ทรงพลังสำหรับ Java ไม่ว่าคุณจะสร้างแคมเปญอีเมลอัตโนมัติหรือสร้างคำเชิญประชุม การเชี่ยวชาญกระบวนการ **convert oft to msg** จะช่วยประหยัดเวลาและลดข้อผิดพลาดจากการทำมือ

## คำตอบด่วน
- **“convert oft to msg” หมายถึงอะไร?** มันแปลง Outlook Template (OFT) ให้เป็น Outlook Message (MSG) ที่กำหนดค่าเต็มรูปแบบแล้ว  
- **ไลบรารีใดทำการแปลง?** Aspose.Email สำหรับ Java  
- **ต้องมีลิขสิทธิ์หรือไม่?** เวอร์ชันทดลองใช้ได้สำหรับการทดสอบ; ลิขสิทธิ์เต็มจะเปิดฟีเจอร์ทั้งหมด  
- **สามารถใช้ Maven สำหรับจัดการ dependencies ได้หรือไม่?** ใช่, เพิ่ม artifact ของ Aspose.Email ใน Maven  
- **ต้องใช้ Java 16 หรือไม่?** แนะนำให้ใช้, แต่ JDK รุ่นหลังก็รองรับเช่นกัน  

## “convert oft to msg” คืออะไร?
*การดำเนินการ “convert oft to msg” จะเปลี่ยนไฟล์ Outlook Template (OFT) ให้เป็นไฟล์ Outlook Message (MSG) มาตรฐาน โดยคงรูปแบบ, ไฟล์แนบ, และเมตาดาต้าไว้ หลังจากแปลงคุณจะได้อีเมลที่พร้อมส่งซึ่งสามารถแก้ไข, ปรับให้เป็นส่วนบุคคล, และส่งผ่านเซิร์ฟเวอร์หรือไคลเอนต์ใด ๆ ที่รองรับรูปแบบ MSG*  

## ทำไมต้องใช้ Aspose.Email สำหรับ Java เพื่ออัตโนมัติกระบวนการอีเมล Outlook ด้วย Java?
Aspose.Email รองรับ **รูปแบบเข้าและออกกว่า 50 รูปแบบ** — รวมถึง OFT, MSG, EML, และ MHTML — และสามารถประมวลผลไฟล์ขนาด **ถึง 2 GB** โดยไม่ต้องโหลดเอกสารทั้งหมดเข้าสู่หน่วยความจำ API แบบ pure‑Java ของมันทำให้ไม่จำเป็นต้องติดตั้ง Outlook หรือ Microsoft Office บนเซิร์ฟเวอร์, ให้การอัตโนมัติอีเมลที่เชื่อถือได้และความเร็วสูง

## คู่มือการดำเนินการ

### วิธีแปลง OFT เป็น MSG ด้วย Aspose.Email สำหรับ Java?

ส่วนนี้อธิบายกระบวนการครบวงจรในการเปลี่ยนเทมเพลต Outlook ให้เป็น Outlook Message ที่กำหนดค่าเต็มรูปแบบ ขั้นแรกโหลดไฟล์ OFT, จากนั้นปรับข้อมูลฟิลด์เช่นผู้ส่ง, ผู้รับ, และเนื้อหาตัวข้อความ, และสุดท้ายบันทึกผลลัพธ์เป็นไฟล์ MSG วิธีการนี้เบา, ใช้เพียงไม่กี่บรรทัดโค้ด, และสามารถนำไปใช้ในงานแบตช์หรือเว็บเซอร์วิสสำหรับการประมวลผลปริมาณมาก

#### โหลดและอัปเดตไฟล์เทมเพลต Outlook

##### ภาพรวม

เรียนรู้การจัดการเนื้อหาของไฟล์ OFT (Outlook Template) และแปลงเป็นข้อความอีเมล MSG ที่กำหนดค่าเต็มรูปแบบ

##### ขั้นตอนการทำงาน

**1. โหลดเทมเพลต Outlook**

`MailMessage` เป็นคลาสหลักของ Aspose.Email ที่ใช้แทนข้อความอีเมลในหน่วยความจำ มีคุณสมบัติสำหรับหัวเรื่อง, เนื้อหา, ผู้รับ, และไฟล์แนบ

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MailMessage message = MailMessage.load(dataDir + "sample.oft");
```

**2. ตั้งค่าผู้ส่งและผู้รับ**

`MailMessage` ให้คุณตั้งค่า `from`, `to`, `cc`, และ `bcc` ได้โดยตรง, ทำให้ MSG สุดท้ายสะท้อนข้อมูลการส่งที่ถูกต้อง

```java
message.setSender(new MailAddress("john@abc.com", "John"));
message.getTo().addMailAddress(new MailAddress("william@xzy.com", "William"));
```

**3. อัปเดตเนื้อหา HTML Body**

คุณสามารถกำหนดสตริง HTML ให้กับ `mailMessage.setHtmlBody()` เพื่อปรับเทมเพลตด้วยข้อมูลไดนามิก เช่น ชื่อ, วันที่, หรือลิงก์ประชุม

```java
String htmlBody = message.getHtmlBody();
htmlBody = htmlBody.replace("DisplayName", "<b>William</b>");
htmlBody = htmlBody.replace("MeetingPlace", "<u>Hall 1, Convention Center, New York, USA</u>");
htmlBody = htmlBody.replace("MeetingTime", "<u>Monday, June 28, 2010</u>");
message.setHtmlBody(htmlBody);
```

**4. บันทึกเป็นไฟล์ MSG**

การเรียก `mailMessage.save("output.msg", SaveOptions.getDefaultMsg())` จะเขียนข้อความที่เตรียมพร้อมเต็มรูปแบบลงดิสก์ในรูปแบบ MSG, เสร็จสิ้นการดำเนินการ **convert oft to msg**

```java
MapiMessage mapimessage = MapiMessage.fromMailMessage(message);
mapimessage.setMessageFlags(MapiMessageFlags.MSGFLAG_UNSENT);
mapimessage.save(dataDir + "Invitation.msg");
```

### วิธีสร้างเทมเพลต Outlook ใหม่ (OFT) ด้วย Aspose.Email?

การสร้างเทมเพลต Outlook ใหม่จากศูนย์ช่วยให้คุณกำหนดเค้าโครงมาตรฐานที่สามารถนำกลับมาใช้ซ้ำได้ในแคมเปญหรือการแจ้งเตือนต่าง ๆ คุณเริ่มด้วยการสร้าง `MapiMessage`, ตั้งค่าคุณสมบัติต่าง ๆ (หัวเรื่อง, เนื้อหา, ไฟล์แนบ), แล้วบันทึกเป็นไฟล์ OFT เทมเพลตนี้สามารถโหลด, ปรับแต่ง, และแปลงเป็น MSG ได้ตามต้องการ

**1. สร้างข้อความอีเมลใหม่**

`MapiMessage` เป็นการแสดงระดับต่ำของ Aspose.Email สำหรับข้อความ Outlook, ให้การควบคุมเต็มรูปแบบของคุณสมบัติ MAPI ที่จำเป็นสำหรับไฟล์ OFT

```java
MapiMessage mapi = new MapiMessage("test@from.to", "test@to.to", "template subject", "Template body");
```

**2. บันทึกเป็นไฟล์เทมเพลต**

บันทึกอินสแตนซ์ `MapiMessage` เป็นไฟล์ OFT เพื่อใช้ในอนาคต

```java
try {
    mapi.saveAsTemplate(dataDir + "mapiToOft.oft");
} finally {
    if (mapi != null) ((IDisposable)mapi).dispose();
}
```

สถานการณ์จริงที่ฟีเจอร์เหล่านี้โดดเด่น:

3. **การแจกจ่ายเอกสาร** – เก็บประกาศที่ใช้บ่อยเป็นเทมเพลต OFT และสร้างไฟล์ MSG ตามความต้องการ
