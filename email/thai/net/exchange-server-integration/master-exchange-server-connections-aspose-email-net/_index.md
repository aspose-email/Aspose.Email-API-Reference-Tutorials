## คู่มือการใช้งาน

ส่วนนี้แบ่งออกเป็นขั้นตอนเชิงตรรกะสำหรับแต่ละฟีเจอร์ที่เรากำลังใช้งาน: การเชื่อมต่อกับเซิร์ฟเวอร์ Exchange และการลบการกำหนดค่าของผู้ใช้

### เชื่อมต่อกับเซิร์ฟเวอร์ Exchange
การเชื่อมต่อกับเซิร์ฟเวอร์ Exchange โดยใช้ไคลเอนต์ EWS ของ Aspose.Email ช่วยลดความยุ่งยากในการจัดการอีเมลด้วยโปรแกรม โดยทำได้ดังนี้:

#### ขั้นตอนที่ 1: กำหนดข้อมูลประจำตัว
สร้างข้อมูลประจำตัวเครือข่ายด้วยชื่อผู้ใช้ รหัสผ่าน และโดเมนของคุณ
```csharp
using System.Net;
using Aspose.Email.Clients.Exchange.WebService;

const string mailboxUri = "https://การแลกเปลี่ยน/ews/exchange.asmx";
const string domain = @"\\";
const string username = "username@ASE305.onmicrosoft.com";
const string password = "password";

// สร้างข้อมูลประจำตัวเครือข่าย networkCredential credentials = new NetworkCredential(ชื่อผู้ใช้, รหัสผ่าน, โดเมน);
```

#### ขั้นตอนที่ 2: รับไคลเอนต์ EWS
ใช้ URI ของกล่องจดหมายและข้อมูลรับรองเพื่อสร้างการเชื่อมต่อ
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
```
การ `GetEWSClient` วิธีการนี้มีความสำคัญเนื่องจากจะรวมข้อมูลที่จำเป็นทั้งหมดสำหรับการเชื่อมต่อกับเซิร์ฟเวอร์ Exchange

#### ตัวเลือกการกำหนดค่าคีย์
- **URI ของกล่องจดหมาย**URL ปลายทางของ EWS ของเซิร์ฟเวอร์ Exchange ของคุณ
- **ข้อมูลประจำตัว**:ให้แน่ใจว่าถูกต้องและได้รับอนุญาตที่เหมาะสม

### ลบการกำหนดค่าผู้ใช้
ต่อไปเราจะมาดูวิธีการลบการกำหนดค่าผู้ใช้จากกล่องจดหมายโดยใช้ไคลเอนต์ Aspose.Email

#### ขั้นตอนที่ 1: กำหนดชื่อการกำหนดค่าผู้ใช้
ระบุการกำหนดค่าที่คุณต้องการลบโดยใช้ตัวระบุเฉพาะภายในกล่องจดหมาย
```csharp
using Aspose.Email.Clients.Exchange.WebService;
using Aspose.Email.Mime;

UserConfigurationName userConfigName = new UserConfigurationName("inbox.config\\");
```

#### ขั้นตอนที่ 2: ลบการกำหนดค่า
การใช้ประโยชน์จาก `DeleteUserConfiguration` วิธีการลบการกำหนดค่าที่ระบุ
```csharp
client.DeleteUserConfiguration(userConfigName);
```
ขั้นตอนนี้จะลบการตั้งค่าแบบกำหนดเองใดๆ ที่เกี่ยวข้องกับ "inbox.config" จากกล่องจดหมาย Exchange ของผู้ใช้
