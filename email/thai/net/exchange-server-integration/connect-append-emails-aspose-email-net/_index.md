## คู่มือการใช้งาน

### การเชื่อมต่อกับเซิร์ฟเวอร์ IMAP

#### ภาพรวม
หัวข้อนี้ครอบคลุมการตั้งค่าการเชื่อมต่อกับเซิร์ฟเวอร์ IMAP ของคุณโดยใช้ `ImapClient`-

#### คำแนะนำทีละขั้นตอน

**1. กำหนดค่า ImapClient**
สร้างและกำหนดค่า `ImapClient` อินสแตนซ์พร้อมรายละเอียดเซิร์ฟเวอร์ของคุณ:

```csharp
using Aspose.Email.Clients;

ImapClient client = new ImapClient();
client.Host = "imap.gmail.com"; // ระบุโฮสต์เซิร์ฟเวอร์ IMAP
client.Username = "your.username@gmail.com"; // ตั้งชื่อผู้ใช้อีเมล์ของคุณ
client.Password = "your.password"; // ตั้งรหัสผ่านอีเมล์ของคุณ
client.Port = 993; // พอร์ตมาตรฐานสำหรับการเชื่อมต่อ SSL
client.SecurityOptions = SecurityOptions.Auto; // เลือกตัวเลือกความปลอดภัยโดยอัตโนมัติ
```

**คำอธิบาย:**
- `Host` ระบุที่อยู่เซิร์ฟเวอร์ IMAP
- `Username` และ `Password` จำเป็นสำหรับการยืนยันตัวตน
- ท่าเรือ `993` ใช้สำหรับการเชื่อมต่อที่ปลอดภัย (SSL/TLS)
- `SecurityOptions.Auto` ช่วยให้มั่นใจถึงการตั้งค่าความปลอดภัยที่เหมาะสมที่สุด

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเครือข่ายของคุณอนุญาตการเชื่อมต่อกับพอร์ต 993
- ตรวจสอบว่าข้อมูลรับรองอีเมล์ของคุณถูกต้อง

### การสร้างและการผนวกข้อความใหม่ไปยังโฟลเดอร์ IMAP

#### ภาพรวม
เรียนรู้วิธีการสร้างข้อความอีเมลใหม่และผนวกเข้าในโฟลเดอร์กล่องจดหมาย

#### คำแนะนำทีละขั้นตอน

**1. สร้าง MailMessage**
สร้างอินสแตนซ์ใหม่ของ `MailMessage`-

```csharp
using Aspose.Email.Clients.Imap;
using Aspose.Email.Mime;

// สร้างอินสแตนซ์ MailMessage สำหรับข้อความอีเมลใหม่
MailMessage msg = new MailMessage("user@domain1.com", "user@domain2.com", "subject", "message");
```

**คำอธิบาย:**
- `MailMessage` หมายถึงอีเมลที่มีรายละเอียดผู้ส่ง ผู้รับ หัวเรื่อง และเนื้อหา

**2. เลือกโฟลเดอร์**
เลือกโฟลเดอร์กล่องจดหมาย:

```csharp
// เลือกโฟลเดอร์กล่องจดหมายบนเซิร์ฟเวอร์ IMAP
client.SelectFolder(ImapFolderInfo.InBox);
```

**3. ผนวกข้อความ**
ผนวกข้อความไปยังโฟลเดอร์ปัจจุบัน:

```csharp
try
{
    // สมัครรับการเปลี่ยนแปลงในโฟลเดอร์ปัจจุบัน (ทางเลือก)
    client.SubscribeFolder(client.CurrentFolder.Name);

    // ผนวกข้อความที่สร้างใหม่ไปยังโฟลเดอร์ที่เลือก
    client.AppendMessage(client.CurrentFolder.Name, msg);
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```

**คำอธิบาย:**
- `SelectFolder` ตั้งค่าโฟลเดอร์ที่ใช้งานอยู่
- `AppendMessage` เพิ่มอีเมล์ของคุณไปยังโฟลเดอร์ที่เลือก
