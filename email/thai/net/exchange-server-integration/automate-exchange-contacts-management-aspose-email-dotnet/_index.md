## คู่มือการใช้งาน
เราจะแบ่งการใช้งานออกเป็นสองคุณสมบัติหลัก: การลบข้อมูลติดต่อและการจัดการข้อมูลติดต่อ

### ลบรายชื่อติดต่อจาก Exchange Server โดยใช้ EWS
ฟีเจอร์นี้สาธิตวิธีการเชื่อมต่อกับเซิร์ฟเวอร์ Exchange โดยใช้ Aspose.Email สำหรับ .NET และลบรายชื่อติดต่อเฉพาะ

#### ภาพรวม
การทำให้การลบรายชื่อติดต่อโดยอัตโนมัติช่วยประหยัดเวลาได้มาก โดยเฉพาะเมื่อต้องจัดการกับชุดข้อมูลขนาดใหญ่หรือภารกิจการบำรุงรักษาตามปกติ ด้วยการเชื่อมต่อกับเซิร์ฟเวอร์ Exchange ผ่าน EWS คุณสามารถลบรายชื่อติดต่อที่ไม่จำเป็นออกโดยการเขียนโปรแกรมตามเกณฑ์ เช่น ชื่อ

#### ขั้นตอนการลบข้อมูลติดต่อ
**ขั้นตอนที่ 1: ดึงข้อมูลติดต่อ**
ขั้นแรก ดึงข้อมูลติดต่อทั้งหมดจากเซิร์ฟเวอร์ Exchange ของคุณ:

```csharp
Contact[] contacts = client.GetContacts(client.MailboxInfo.ContactsUri);
```

**ขั้นตอนที่ 2: ระบุและลบข้อมูลติดต่อเฉพาะ**
ทำซ้ำผ่านรายชื่อที่ดึงมาเพื่อค้นหาและลบรายชื่อที่ต้องการ ที่นี่ เรากำลังมองหา "จอห์น เท็ดดี้"

```csharp
string strContactToDelete = "John Teddy";

foreach (Contact contact in contacts)
{
    if (contact.DisplayName.Equals(strContactToDelete))
        client.DeleteItem(contact.Id.EWSId, DeletionOptions.DeletePermanently);
}
```

**ขั้นตอนที่ 3: กำจัดลูกค้า**
ควรแน่ใจเสมอว่าได้ปล่อยทรัพยากรโดยการกำจัดไคลเอนต์ EWS:

```csharp
client.Dispose();
```

#### เคล็ดลับการแก้ไขปัญหา
- **ปัญหาการเชื่อมต่อ**: ตรวจสอบให้แน่ใจว่า URL เซิร์ฟเวอร์และข้อมูลรับรองของคุณถูกต้อง
- **ข้อผิดพลาดเกี่ยวกับการอนุญาต**: ตรวจสอบว่าผู้ใช้มีสิทธิ์เพียงพอในการลบข้อมูลติดต่อ

### จัดการรายชื่อติดต่อ Exchange โดยใช้ EWS
การจัดการผู้ติดต่อเกี่ยวข้องกับการดึงข้อมูลผู้ติดต่อจาก Exchange Server เพื่อวัตถุประสงค์ต่างๆ เช่น การแสดงผลหรือการประมวลผลเพิ่มเติม

#### ภาพรวม
การดึงข้อมูลติดต่อช่วยให้คุณจัดการ อัปเดต หรือวิเคราะห์ข้อมูลติดต่อได้อย่างมีประสิทธิภาพ กระบวนการนี้มีความสำคัญอย่างยิ่งในการทำให้สมุดที่อยู่ของคุณเป็นปัจจุบันและเพื่อให้แน่ใจว่าช่องทางการสื่อสารยังคงชัดเจน

#### ขั้นตอนการกู้คืนข้อมูลติดต่อ
**ขั้นตอนที่ 1: ดึงข้อมูลติดต่อ**
คล้ายกับคุณสมบัติการลบ เริ่มต้นด้วยการดึงข้อมูลรายชื่อติดต่อทั้งหมดที่มีอยู่:

```csharp
Contact[] contacts = client.GetContacts(client.MailboxInfo.ContactsUri);
```

**ขั้นตอนที่ 2: ดำเนินการเรียกคืนข้อมูลติดต่อ**
ดำเนินการตามที่ต้องการกับผู้ติดต่อแต่ละราย นี่คือตัวอย่างการพิมพ์รายละเอียดผู้ติดต่อเพื่อตรวจสอบ (แม้ว่าเราจะข้ามขั้นตอนนี้ในโค้ดของเรา):

```csharp
foreach (Contact contact in contacts)
{
    // ตัวอย่างการดำเนินการ: พิมพ์รายละเอียดการติดต่อ
    // Console.WriteLine(contact.DisplayName);
}
```

**ขั้นตอนที่ 3: กำจัดลูกค้า**
เช่นเคย อย่าลืมปล่อยทรัพยากร:

```csharp
client.Dispose();
```

#### เคล็ดลับการแก้ไขปัญหา
- **ความสอดคล้องของข้อมูล**: ตรวจสอบให้แน่ใจว่าข้อมูลเซิร์ฟเวอร์ Exchange ของคุณได้รับการซิงโครไนซ์แล้ว
- **ปัญหาคอขวดด้านประสิทธิภาพ**:หากต้องจัดการกับผู้ติดต่อจำนวนมาก ควรพิจารณาเพิ่มประสิทธิภาพการสอบถามของคุณ
