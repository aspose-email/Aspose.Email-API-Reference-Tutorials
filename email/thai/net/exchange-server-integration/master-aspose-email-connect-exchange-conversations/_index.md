## คู่มือการใช้งาน

### เชื่อมต่อกับ Exchange Server โดยใช้ EWSClient

**ภาพรวม**:สร้างการเชื่อมต่อกับเซิร์ฟเวอร์ Exchange โดยใช้ Aspose.Email `EWSClient`-

#### ขั้นตอนที่ 1: ตั้งค่าข้อมูลประจำตัว
กำหนดค่าข้อมูลประจำตัวเครือข่ายที่ใช้สำหรับการตรวจสอบสิทธิ์กับเซิร์ฟเวอร์ Exchange:
```csharp
using System;
using System.Net;
using Aspose.Email.Clients.Exchange.WebService;

const string mailboxUri = "https://แลกเปลี่ยน/ews/exchange.asmx";
const string domain = "";
const string username = "username@ASE305.onmicrosoft.com";
const string password = "password";

// การสร้างวัตถุ NetworkCredential ด้วยข้อมูลประจำตัวผู้ใช้
NetworkCredential credentials = new NetworkCredential(username, password, domain);
```

#### ขั้นตอนที่ 2: เชื่อมต่อกับเซิร์ฟเวอร์ Exchange
โดยใช้ `EWSClient`, เชื่อมต่อกับกล่องจดหมายของคุณ:
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
Console.WriteLine("Connected to Exchange Server");
```

### ค้นหาและลบการสนทนาที่เฉพาะเจาะจง

**ภาพรวม**:ดึงบทสนทนาจากกล่องจดหมาย และลบบทสนทนาที่ตรงตามเกณฑ์เฉพาะ

#### ขั้นตอนที่ 1: ดึงรายการสนทนาทั้งหมด
ดึงรายการสนทนาทั้งหมดในโฟลเดอร์กล่องจดหมายของคุณ:
```csharp
using Aspose.Email.Clients.Exchange.WebService;
using Aspose.Email.Mime;

// การดึงบทสนทนาจากกล่องจดหมาย
ExchangeConversation[] conversations = client.FindConversations(client.MailboxInfo.InboxUri);
```

#### ขั้นตอนที่ 2: ตรวจสอบหัวข้อการสนทนาและลบ
ทำซ้ำในแต่ละบทสนทนาเพื่อค้นหาบทสนทนาที่ตรงตามเกณฑ์ของคุณ:
```csharp
foreach (ExchangeConversation conversation in conversations)
{
    // ตรวจสอบว่าหัวข้อสนทนามีสตริงเฉพาะหรือไม่
    if (conversation.ConversationTopic.Contains("test email"))
    {
        // การลบรายการสนทนาตามเงื่อนไข
        client.DeleteConversationItems(conversation.ConversationId);
    }
}
```

### เคล็ดลับการแก้ไขปัญหา

- **ปัญหาการเชื่อมต่อ**: ตรวจสอบให้แน่ใจว่าข้อมูลประจำตัวและ URL ของเซิร์ฟเวอร์ Exchange ของคุณถูกต้อง
- **สิทธิ์การเข้าถึง**:ตรวจสอบว่าผู้ใช้มีสิทธิ์เพียงพอในการเข้าถึงและแก้ไขการสนทนาในกล่องจดหมาย
