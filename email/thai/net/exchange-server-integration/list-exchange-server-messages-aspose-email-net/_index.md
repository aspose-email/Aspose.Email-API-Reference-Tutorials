## คู่มือการใช้งาน

ตอนนี้การตั้งค่าของเราเสร็จเรียบร้อยแล้ว มาดูการใช้งานคุณลักษณะการแสดงรายการข้อความจากเซิร์ฟเวอร์ Exchange กัน

### เชื่อมต่อกับเซิร์ฟเวอร์ Exchange

หากต้องการแสดงรายการอีเมล ให้เชื่อมต่อกับเซิร์ฟเวอร์ Exchange ของคุณโดยใช้ Aspose.Email ก่อน คุณจะต้องใช้ URI ของเซิร์ฟเวอร์และข้อมูลรับรองสำหรับขั้นตอนนี้

**ขั้นตอนที่ 1: สร้างการเชื่อมต่อ**

สร้างอินสแตนซ์ใหม่ของ `ExchangeClient`-

```csharp
string exchangeUri = "https://ชื่อเครื่อง/exchange/ชื่อผู้ใช้"; // URI ของเซิร์ฟเวอร์ Exchange ของคุณ
string username = "username"; // ชื่อผู้ใช้ Exchange Server ของคุณ
string password = "password"; // รหัสผ่านเซิร์ฟเวอร์ Exchange ของคุณ

try
{
    var domain = new Domain(); // ตัวแทนสำหรับคลาสโดเมนหากจำเป็น
    ExchangeClient client = new ExchangeClient(exchangeUri, username, password, domain);

    // ดำเนินการรายการข้อความ
}
catch (Exception ex)
{
    Console.Write("Error connecting: " + ex.Message);
}
```

ที่นี่, `ExchangeClient` ใช้ URI ของเซิร์ฟเวอร์และข้อมูลรับรองเป็นพารามิเตอร์ ช่วยให้เชื่อมต่อได้อย่างปลอดภัย

### รายการข้อความจากกล่องจดหมาย

เมื่อเชื่อมต่อเรียบร้อยแล้ว เราสามารถดึงอีเมลได้:

**ขั้นตอนที่ 2: ดึงข้อความ**

ใช้ไคลเอนต์เพื่อดึงข้อความจากกล่องจดหมายของคุณ:

```csharp
try
{
    ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);

    // แสดงข้อมูลข้อความ
}
catch (Exception ex)
{
    Console.Write("Error retrieving messages: " + ex.Message);
}
```

`ListMessages` ดึงข้อความทั้งหมดจาก URI ของกล่องจดหมายที่ระบุ โดยส่งคืนเป็นคอลเลกชัน

### แสดงข้อมูลข้อความ

หลังจากดึงข้อความแล้ว คุณสามารถทำซ้ำข้อความเหล่านั้นเพื่อแสดงรายละเอียดที่จำเป็นได้:

**ขั้นตอนที่ 3: ทำซ้ำและแสดงผล**

```csharp
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Sent Date: " + msgInfo.Date.ToString());
    Console.WriteLine("Read?: " + msgInfo.IsRead.ToString());
    Console.WriteLine("Message ID: " + msgInfo.MessageId);
    Console.WriteLine("Unique URI: " + msgInfo.UniqueUri);
}
```

ลูปนี้จะวนซ้ำผ่านแต่ละข้อความ โดยพิมพ์คุณลักษณะหลักๆ เช่น หัวเรื่อง ผู้ส่ง ผู้รับ และสถานะการอ่าน
