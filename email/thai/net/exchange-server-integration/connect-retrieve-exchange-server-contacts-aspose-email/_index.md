## คู่มือการใช้งาน
หัวข้อนี้ครอบคลุมการเชื่อมต่อกับ Exchange Server และการดึงข้อมูลติดต่อ

### การเชื่อมต่อกับเซิร์ฟเวอร์ Exchange
การเชื่อมต่อกับ Exchange Server ของคุณอย่างปลอดภัยถือเป็นสิ่งสำคัญ ดังต่อไปนี้:

#### ขั้นตอนที่ 1: กำหนดข้อมูลประจำตัว
ตั้งค่าข้อมูลประจำตัว WebDAV ของคุณโดยใช้ `NetworkCredential` เพื่อการยืนยันตัวตนที่ปลอดภัย

```csharp
using System.Net;

string mailboxURI = "http://ex2003/exchange/administrator"; // URI ของเซิร์ฟเวอร์ Exchange
string username = "administrator";
string password = "pwd";
string domain = "domain.local";

// สร้างข้อมูลประจำตัวเครือข่ายสำหรับการเชื่อมต่อกับเซิร์ฟเวอร์
NetworkCredential credential = new NetworkCredential(username, password, domain);
```

#### ขั้นตอนที่ 2: เริ่มต้นใช้งาน `ExchangeClient`
ใช้ `ExchangeClient` จาก Aspose.Email เพื่อสร้างการเชื่อมต่อ

```csharp
using Aspose.Email.Clients.Exchange.Dav;

// สร้างอินสแตนซ์ของ ExchangeClient พร้อมด้วย URI และข้อมูลประจำตัว
tExchangeClient client = new ExchangeClient(mailboxURI, credential);
```

### การดึงข้อมูลติดต่อจาก Exchange Server
เมื่อเชื่อมต่อแล้วให้ดึงข้อมูลติดต่อที่จัดเก็บไว้บนเซิร์ฟเวอร์ของคุณ

#### ขั้นตอนที่ 1: ดึงข้อมูลติดต่อ
ใช้ `GetContacts` วิธีการดึงข้อมูลรายชื่อติดต่อทั้งหมด

```csharp
using Aspose.Email.Mapi;

// ดึงข้อมูลติดต่อจากไดเร็กทอรี
Contact[] contacts = client.GetContacts(client.MailboxInfo.ContactsUri);
```

#### ขั้นตอนที่ 2: แสดงข้อมูลการติดต่อ
วนซ้ำผ่านรายชื่อติดต่อแต่ละรายและแสดงรายละเอียดที่เกี่ยวข้อง เช่น ชื่อและที่อยู่อีเมล

```csharp
foreach (MapiContact contact in contacts)
{
    Console.WriteLine("Name: " + contact.NameInfo.DisplayName + ", Email Address: " + contact.ElectronicAddresses.Email1);
}
```

### เคล็ดลับการแก้ไขปัญหา
- **ข้อผิดพลาดในการเชื่อมต่อ**: ตรวจสอบว่า URI ของเซิร์ฟเวอร์ ชื่อผู้ใช้ รหัสผ่าน และโดเมนของคุณถูกต้อง
- **ปัญหาการอนุญาต**: ตรวจสอบให้แน่ใจว่าบัญชีผู้ใช้มีสิทธิ์ในการเข้าถึงรายชื่อติดต่อบน Exchange Server
