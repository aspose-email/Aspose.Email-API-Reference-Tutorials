## คู่มือการใช้งาน

### คุณสมบัติ 1: การตรวจสอบและการตั้งค่าไคลเอนต์ Exchange

ตรวจสอบสิทธิ์และสร้างอินสแตนซ์ของ `ExchangeClient` ในส่วนนี้

**ภาพรวม:**
การตรวจสอบสิทธิ์กับเซิร์ฟเวอร์ Exchange เป็นสิ่งสำคัญสำหรับการเข้าถึงอีเมล ต่อไปนี้เป็นวิธีตั้งค่าไคลเอนต์โดยใช้ข้อมูลประจำตัวของคุณ

#### ขั้นตอนที่ 1: สร้าง `ExchangeClient` ตัวอย่าง
```csharp
using Aspose.Email.Clients.Exchange;

// กำหนด URL เซิร์ฟเวอร์ ชื่อผู้ใช้ รหัสผ่าน และโดเมนของคุณ
string url = "http://ex07sp1/การแลกเปลี่ยน/ผู้ดูแลระบบ";
string username = "user";
string password = "pwd";
string domain = "domain";

// เริ่มต้น ExchangeClient ด้วยข้อมูลประจำตัว
ExchangeClient client = new ExchangeClient(url, username, password, domain);
```

**คำอธิบาย:**
- **ที่อยู่**: URL ของเซิร์ฟเวอร์ที่โฮสต์ Exchange Server ของคุณ
- **ชื่อผู้ใช้/รหัสผ่าน/โดเมน**: ข้อมูลประจำตัวที่จำเป็นสำหรับการยืนยันตัวตน

### คุณสมบัติ 2: การแสดงรายการข้อความจากกล่องจดหมาย

ใช้สิ่งที่ได้รับการรับรอง `ExchangeClient` เพื่อแสดงรายการข้อความในกล่องจดหมาย

**ภาพรวม:**
การแสดงรายการอีเมลด้วยโปรแกรมช่วยประหยัดเวลาและทำให้กระบวนการที่ซ้ำซากกลายเป็นระบบอัตโนมัติ ต่อไปนี้เป็นวิธีการดึงข้อความอีเมลอย่างมีประสิทธิภาพ

#### ขั้นตอนที่ 2: ดึงอีเมล
```csharp
// ถือว่าได้สร้าง 'ไคลเอนต์' ไว้แล้วตามที่แสดงไว้ก่อนหน้านี้
ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);
```

**คำอธิบาย:**
- `ListMessages`: ดึงข้อความทั้งหมดจาก URI ของกล่องจดหมายที่ระบุ (ในกรณีนี้คือกล่องจดหมายเข้า)

### คุณสมบัติที่ 3: การแสดงข้อมูลข้อความ

วนซ้ำข้อความที่ดึงมาและแสดงข้อมูลพื้นฐานของข้อความเหล่านั้น

#### ขั้นตอนที่ 3: พิมพ์รายละเอียดอีเมล์
```csharp
using System;

// วนซ้ำผ่านแต่ละข้อความในคอลเลคชัน
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Message Size: " + msgInfo.Size);
    Console.WriteLine("==================================");
}
```

**คำอธิบาย:**
- **ข้อความข้อมูล**: แสดงถึงอีเมลส่วนบุคคล ซึ่งให้การเข้าถึงคุณสมบัติ เช่น `Subject`- `From`, และ `Size`-
