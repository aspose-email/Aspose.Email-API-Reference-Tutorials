## คู่มือการใช้งาน

ตอนนี้เรามาดูการส่งอีเมลเป็น TNEF โดยใช้ Aspose.Email สำหรับ .NET กัน

### การโหลดข้อความอีเมลด้วยตัวเลือก TNEF ที่เก็บรักษาไว้

#### ขั้นตอนที่ 1: โหลดไฟล์อีเมล์ของคุณ
เริ่มต้นด้วยการโหลดข้อความอีเมลของคุณจาก `.eml` ไฟล์สำหรับเก็บรักษาไฟล์แนบ TNEF:
```csharp
using System;
using Aspose.Email.Mime;
using Aspose.Email.Clients.Smtp;

var emlFileName = @"YOUR_DOCUMENT_DIRECTORY\Message.eml"; // เส้นทางไปยังไฟล์ EML

// โหลดอีเมลพร้อมตัวเลือกเพื่อรักษาไฟล์แนบ TNEF
MailMessage eml1 = MailMessage.Load(emlFileName, new EmlLoadOptions() { PreserveTnefAttachments = true });
```

**คำอธิบาย**: เดอะ `EmlLoadOptions` คลาสได้รับการกำหนดค่าให้รักษาข้อมูลเฉพาะ TNEF ให้คงอยู่ในระหว่างการโหลด

### การตั้งค่าข้อมูลอีเมล์

#### ขั้นตอนที่ 2: กำหนดค่าผู้ส่งและผู้รับ
ตั้งค่าช่องผู้ส่งและผู้รับอีเมล์:
```csharp
eml1.From = "somename@gmail.com";
eml1.To.Clear();
eml1.To.Add(new MailAddress("first.last@test.com"));
```

**คำอธิบาย**:แทนที่อีเมลตัวแทนด้วยที่อยู่จริง

#### ขั้นตอนที่ 3: กำหนดหัวเรื่องและวันที่
เพิ่มบรรทัดหัวเรื่องและตั้งวันที่ปัจจุบันให้กับอีเมลของคุณ:
```csharp
eml1.Subject = "With PreserveTnef flag during loading";
eml1.Date = DateTime.Now;
```

### การกำหนดค่าไคลเอนต์ SMTP สำหรับ TNEF

#### ขั้นตอนที่ 4: ตั้งค่า SMTP
กำหนดค่าไคลเอนต์ SMTP เพื่อส่งอีเมลโดยใช้รูปแบบ TNEF:
```csharp
SmtpClient client = new SmtpClient("smtp.gmail.com", 587, "somename", "password");
client.SecurityOptions = SecurityOptions.Auto; // ปรับการตั้งค่าความปลอดภัยโดยอัตโนมัติ
client.UseTnef = true;
```

**คำอธิบาย**: เดอะ `UseTnef` คุณสมบัตินี้มีความสำคัญต่อการส่งอีเมลในรูปแบบ TNEF ตรวจสอบให้แน่ใจว่าเซิร์ฟเวอร์ SMTP ของคุณรองรับคุณสมบัตินี้

### การส่งอีเมล

#### ขั้นตอนที่ 5: ส่งอีเมลของคุณ
สุดท้ายให้ส่งอีเมลและจัดการข้อยกเว้นที่อาจเกิดขึ้น:
```csharp
try
{
    client.Send(eml1);
}
catch (Exception ex)
{
    Console.Write(ex.Message); // บันทึกหรือจัดการข้อยกเว้นตามความเหมาะสม
}
```

**คำอธิบาย**การห่อกระบวนการส่งในบล็อก try-catch จะช่วยจัดการข้อผิดพลาดได้อย่างสวยงาม
