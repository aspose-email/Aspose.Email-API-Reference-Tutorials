## คู่มือการใช้งาน

ตอนนี้คุณได้ตั้งค่าทุกอย่างเรียบร้อยแล้ว มาดูขั้นตอนการใช้งานกันเลย ในส่วนนี้จะแนะนำคุณเกี่ยวกับการสร้างและส่งอีเมลโดยใช้ Aspose.Email สำหรับ .NET

### การส่งข้อความอีเมล์ด้วย Aspose.Email สำหรับ .NET

#### ภาพรวม
คุณสมบัตินี้ช่วยให้ผู้พัฒนาสามารถส่งอีเมลผ่าน Exchange Web Services โดยใช้ข้อมูลประจำตัวเครือข่ายอย่างปลอดภัย

#### ขั้นตอนที่ 1: กำหนดค่าข้อมูลประจำตัวเครือข่าย

ขั้นแรกให้สร้าง `NetworkCredential` วัตถุ ซึ่งจะทำให้มั่นใจได้ว่าชื่อผู้ใช้และรหัสผ่านของคุณจะถูกส่งต่ออย่างปลอดภัยเมื่อเชื่อมต่อกับเซิร์ฟเวอร์อีเมล:

```csharp
string mailboxUri = "https://แลกเปลี่ยน.domain.com/ews/Exchange.asmx";
string domain = @"";
string username = "username";
string password = "password";

// สร้างข้อมูลประจำตัว
NetworkCredential credential = new NetworkCredential(username, password, domain);
```

#### ขั้นตอนที่ 2: เชื่อมต่อกับไคลเอนต์อีเมล

ขั้นตอนต่อไป เชื่อมต่อกับเซิร์ฟเวอร์อีเมลของคุณโดยใช้ `IEWSClient`ซึ่งมีวิธีการในการส่งอีเมล์:

```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credential);
```

#### ขั้นตอนที่ 3: สร้างและส่งข้อความอีเมล

สร้างใหม่ `MailMessage` วัตถุที่ระบุรายละเอียดผู้ส่งและผู้รับ จากนั้นส่งอีเมลโดยใช้ `IEWSClient.Send` วิธี:

```csharp
// สร้างข้อความอีเมล์
MailMessage message = new MailMessage("user@domain.com", "recipient@domain.com")
{
    Subject = "Test Email",
    Body = "This is a test email sent using Aspose.Email for .NET."
};

// ส่งอีเมล์
client.Send(message);
```
**พารามิเตอร์:**
- `mailboxUri`: URI ของเซิร์ฟเวอร์ Exchange ของคุณ
- `credential`:ข้อมูลประจำตัวเครือข่ายสำหรับการยืนยันตัวตน

#### เคล็ดลับการแก้ไขปัญหา

- **ข้อผิดพลาดในการรับรองความถูกต้อง:** ตรวจสอบให้แน่ใจว่าชื่อผู้ใช้และรหัสผ่านของคุณถูกต้องและได้รับอนุญาตที่จำเป็น
- **ปัญหาเครือข่าย:** ตรวจสอบว่าการตั้งค่าเครือข่ายของคุณอนุญาตการเชื่อมต่อกับเซิร์ฟเวอร์เมลที่ระบุ
