## คู่มือการใช้งาน

### ตั้งค่าไคลเอนต์ SMTP พร้อมการเชื่อมโยง IP เฉพาะ

หัวข้อนี้สาธิตวิธีการผูกจุดสิ้นสุดเฉพาะภายในสำหรับไคลเอนต์ SMTP โดยใช้ Aspose.Email

#### ภาพรวม
การผูกไคลเอนต์ SMTP กับที่อยู่ IP เฉพาะช่วยให้แอปพลิเคชันของคุณโต้ตอบกับเซิร์ฟเวอร์อีเมลได้ในลักษณะที่ควบคุมได้ เพิ่มความปลอดภัยและรับรองความสอดคล้องกับนโยบายเครือข่าย

#### การดำเนินการแบบทีละขั้นตอน

##### กำหนดค่าไคลเอนต์ SMTP
เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `SmtpClient` คลาส ตั้งค่ารายละเอียดเซิร์ฟเวอร์ รวมถึงข้อมูลประจำตัวและตัวเลือกความปลอดภัย:

```csharp
// สร้างวัตถุไคลเอนต์ SMTP
SmtpClient client = new SmtpClient("smtp.gmail.com", 587);

// ตั้งค่าข้อมูลประจำตัวลูกค้า
client.Username = "your-email@gmail.com";
client.Password = "your-password";

// กำหนดค่าการตั้งค่า SSL
client.SecurityOptions = SecurityOptions.Auto;
```

##### ผูกเข้ากับที่อยู่ IP เฉพาะ
หากต้องการผูกไคลเอนต์ SMTP กับจุดสิ้นสุดเฉพาะในพื้นที่ ให้ใช้ `IPEndPoint` และตั้งค่าผ่านฟังก์ชั่นการโทรกลับ:

```csharp
// กำหนดจุดสิ้นสุดในพื้นที่ด้วย IP และพอร์ตเฉพาะ
IPAddress localIP = IPAddress.Parse("192.168.1.5");
int localPort = 1025;

// ผูกจุดสิ้นสุด
client.LocalNetworkSettings = new SmtpClient.LocalNetworkSettings()
{
    LocalEndpoint = new IPEndPoint(localIP, localPort)
};

// ฟังก์ชั่นการโทรกลับเพื่อจัดการการผูกมัด
client.BeforeSend += (sender, e) =>
{
    Console.WriteLine("Binding to specific IP: " + client.LocalNetworkSettings.LocalEndpoint);
};
```

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่า IP และพอร์ตที่ระบุนั้นพร้อมใช้งานบนเครือข่ายของคุณ
- ตรวจสอบข้อมูลประจำตัวและการตั้งค่าเซิร์ฟเวอร์ SMTP หากมีปัญหาการเชื่อมต่อเกิดขึ้น
