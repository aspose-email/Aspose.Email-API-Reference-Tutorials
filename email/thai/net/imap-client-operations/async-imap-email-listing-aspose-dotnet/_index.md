## คู่มือการใช้งาน
### รายชื่ออีเมล IMAP แบบอะซิงโครนัส
คุณลักษณะที่เราจะนำมาใช้ช่วยให้คุณสามารถแสดงรายการข้อความจากเซิร์ฟเวอร์ IMAP ได้อย่างอะซิงโครนัส ซึ่งเหมาะอย่างยิ่งสำหรับการดำเนินการที่ไม่เกิดการบล็อกในแอปพลิเคชันของคุณ

#### การดำเนินการแบบทีละขั้นตอน
**1. เริ่มต้น ImapClient**
เริ่มต้นโดยการตั้งค่า `ImapClient` พร้อมรายละเอียดผู้ให้บริการอีเมล์ของคุณ:

```csharp
// สร้างและกำหนดค่าอินสแตนซ์ ImapClient
ImapClient client = new ImapClient();
client.Host = "imap.gmail.com";
client.Username = "your.username@gmail.com";
client.Password = "your.password";
client.Port = 993;
client.SecurityOptions = SecurityOptions.Auto;
```

**2. สร้างแบบสอบถามการค้นหา**
ใช้ `ImapQueryBuilder` เพื่อสร้างแบบสอบถามที่กรองอีเมลตามหัวเรื่อง:

```csharp
// สร้างคำค้นหาสำหรับอีเมลที่มี 'หัวเรื่อง' ในบรรทัดหัวเรื่อง
ImapQueryBuilder builder = new ImapQueryBuilder();
builder.Subject.Contains("Subject");
MailQuery query = builder.GetQuery();
```

**3. แสดงรายการข้อความแบบอะซิงโครนัส**
ดำเนินการแบบอะซิงโครนัสเพื่อแสดงรายการข้อความที่ตรงตามเกณฑ์ของคุณ:

```csharp
try
{
    // เริ่มแสดงรายการข้อความแบบอะซิงโครนัสโดยใช้แบบสอบถามที่ระบุ
    IAsyncResult asyncResult = client.BeginListMessages(query);

    // ดำเนินการให้เสร็จสิ้นและรับผลลัพธ์
    ImapMessageInfoCollection messages = client.EndListMessages(asyncResult);
    
    foreach (var message in messages)
    {
        Console.WriteLine($"Subject: {message.Subject}");
    }
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    // ทำความสะอาดทรัพยากร
    if (client != null) client.Dispose();
}
```
