## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเชื่อมต่อกับเซิร์ฟเวอร์ Exchange และการแสดงรายการข้อความในกล่องจดหมาย

### การเชื่อมต่อกับบริการเว็บ Exchange
#### ภาพรวม
การเชื่อมต่อกับ Microsoft Exchange Server ช่วยให้แอปพลิเคชันสามารถโต้ตอบกับบริการอีเมลผ่านโปรแกรมได้ คุณลักษณะนี้ใช้ `IEWSClient` อินเทอร์เฟซที่จัดทำโดย Aspose.Email

**ขั้นตอนที่ 1: สร้างอินสแตนซ์ของ `ExchangeWebServiceClient`**
```csharp
// เริ่มต้นไคลเอนต์ด้วยข้อมูลประจำตัวเซิร์ฟเวอร์ Exchange ของคุณ
IEWSClient client = EWSClient.GetEWSClient("https://outlook.office365.com/ews/exchange.asmx", "ชื่อผู้ใช้", "รหัสผ่าน");
```
- **คำอธิบายพารามิเตอร์**: แทนที่ `"UserName"` และ `"Password"` ด้วยข้อมูลประจำตัว Exchange ที่แท้จริง ตรวจสอบให้แน่ใจว่า URL ตรงกับการกำหนดค่าเซิร์ฟเวอร์ของคุณ

**ขั้นตอนที่ 2: พยายามเชื่อมต่อ**
```csharp
try
{
    Console.WriteLine("Connected to the Exchange server successfully.");
}
catch (Exception ex)
{
    Console.Write(ex.Message);
}
```
- **วัตถุประสงค์**:โค้ดนี้พยายามเชื่อมต่อและพิมพ์ข้อความแจ้งความสำเร็จหรือข้อยกเว้นใดๆ ที่พบ เพื่อช่วยในการแก้ไขปัญหา

### รายการข้อความจากกล่องจดหมาย
#### ภาพรวม
เมื่อเชื่อมต่อแล้ว คุณสามารถแสดงรายการข้อความในกล่องจดหมายของคุณได้ `ListMessages` วิธีการดึงข้อมูลข้อความ

**ขั้นตอนที่ 1: รายการข้อความ**
```csharp
// โดยถือว่ามีการกำหนดค่าเริ่มต้นเป็น 'client' ดังแสดงด้านบน
ExchangeMessageInfoCollection msgCollection = client.ListMessages(client.MailboxInfo.InboxUri);
```
- **คำอธิบาย**: ดึงข้อความทั้งหมดจาก URI กล่องจดหมายโดยใช้ `ListMessages`-

**ขั้นตอนที่ 2: แสดงรายละเอียดข้อความ**
```csharp
foreach (ExchangeMessageInfo msgInfo in msgCollection)
{
    Console.WriteLine("Subject: " + msgInfo.Subject);
    Console.WriteLine("From: " + msgInfo.From.ToString());
    Console.WriteLine("To: " + msgInfo.To.ToString());
    Console.WriteLine("Message ID: " + msgInfo.MessageId);
    Console.WriteLine("Unique URI: " + msgInfo.UniqueUri);
}
```
- **วัตถุประสงค์**:ทำซ้ำแต่ละข้อความโดยแสดงรายละเอียดสำคัญเช่นหัวเรื่องและผู้ส่ง
