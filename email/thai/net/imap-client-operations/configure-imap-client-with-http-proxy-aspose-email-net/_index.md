## คู่มือการใช้งาน

### การกำหนดค่าไคลเอนต์ IMAP ด้วยพร็อกซี HTTP

#### ภาพรวม
คุณสมบัตินี้ช่วยให้สามารถตั้งค่าพร็อกซี HTTP เพื่อเข้าถึงอีเมลผ่านทางโปรโตคอล IMAP ซึ่งจำเป็นเมื่อนโยบายเครือข่ายกำหนดให้การรับส่งข้อมูลทั้งหมดต้องผ่านเซิร์ฟเวอร์เฉพาะ

**ขั้นตอนที่ 1: สร้างอินสแตนซ์ HttpProxy**
```csharp
// เริ่มต้น HttpProxy ด้วยที่อยู่โฮสต์และหมายเลขพอร์ต
HttpProxy proxy = new HttpProxy("18.222.124.59", 8080);
```
- **พารามิเตอร์**: IP หรือ ชื่อโฮสต์ (`"18.222.124.59"`) และหมายเลขพอร์ต (`8080`-

**ขั้นตอนที่ 2: เริ่มต้น ImapClient**
```csharp
using (ImapClient client = new ImapClient("imap.domain.com", "username", "password"))
{
    // กำหนดพร็อกซีให้กับคุณสมบัติพร็อกซีของไคลเอนต์
    client.Proxy = proxy;

    // เลือกโฟลเดอร์กล่องจดหมาย
    client.SelectFolder("Inbox");
}
```
- **วัตถุประสงค์**- `ImapClient` เชื่อมต่อคุณกับเซิร์ฟเวอร์อีเมลของคุณ การใช้พร็อกซีช่วยให้แน่ใจว่าคำขอทั้งหมดได้รับการกำหนดเส้นทางอย่างถูกต้อง

**เคล็ดลับการแก้ไขปัญหา**: ตรวจสอบให้แน่ใจว่าการตั้งค่าพร็อกซีสอดคล้องกับการตั้งค่าที่ผู้ดูแลระบบเครือข่ายของคุณให้ไว้เพื่อการเชื่อมต่อที่ประสบความสำเร็จ

### การเริ่มต้นไคลเอนต์ IMAP ขั้นพื้นฐานและการเลือกโฟลเดอร์

#### ภาพรวม
สำหรับสภาพแวดล้อมที่ไม่มีพร็อกซี HTTP คุณลักษณะนี้อนุญาตให้มีการเริ่มต้นไคลเอนต์ IMAP ขั้นพื้นฐานเพื่อเข้าถึงโฟลเดอร์อีเมล เช่น กล่องจดหมายเข้า โดยตรง

**ขั้นตอนที่ 1: เริ่มต้น ImapClient**
```csharp
using (ImapClient client = new ImapClient("imap.domain.com", "username", "password"))
{
    // เลือกโฟลเดอร์ที่คุณต้องการทำงานด้วย
    client.SelectFolder("Inbox");
}
```
- **คำอธิบาย**ขั้นตอนนี้จะเชื่อมต่อกับเซิร์ฟเวอร์อีเมลของคุณโดยไม่ต้องใช้พร็อกซี ตรวจสอบให้แน่ใจว่าใช้ข้อมูลประจำตัวที่ถูกต้อง

## การประยุกต์ใช้งานจริง
1. **การจัดการอีเมล์ขององค์กร**:เข้าถึงและจัดการอีเมลอย่างมีประสิทธิภาพพร้อมปฏิบัติตามนโยบายเครือข่ายของบริษัท
2. **การเข้าถึงระยะไกลที่ปลอดภัย**:ใช้พร็อกซี HTTP เพื่อเข้าถึงกล่องจดหมายขององค์กรอย่างปลอดภัยจากเครือข่ายภายนอก
3. **ระบบอัตโนมัติอีเมล์**:ทำให้การประมวลผลอีเมลเป็นอัตโนมัติ พร้อมทั้งให้สอดคล้องกับมาตรการรักษาความปลอดภัยเครือข่าย
4. **การทดสอบการพัฒนา**:ทดสอบแอปพลิเคชันที่เกี่ยวข้องกับ IMAP ในสภาพแวดล้อมที่จำลองการเข้าถึงอินเทอร์เน็ตที่จำกัด
