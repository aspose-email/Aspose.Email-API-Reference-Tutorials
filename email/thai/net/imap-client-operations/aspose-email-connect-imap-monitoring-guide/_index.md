## คู่มือการใช้งาน

### คุณสมบัติ 1: เชื่อมต่อและเข้าสู่ระบบเซิร์ฟเวอร์ IMAP

**ภาพรวม:** การเชื่อมต่อกับเซิร์ฟเวอร์ IMAP เป็นขั้นตอนแรกในการจัดการอีเมลด้วยโปรแกรม ในที่นี้เราจะใช้ `ImapClient` จาก Aspose.Email สำหรับ .NET

#### การดำเนินการทีละขั้นตอน:

**3.1 เริ่มต้น ImapClient**

```csharp
using Aspose.Email.Clients.Imap;

// สร้างอินสแตนซ์ใหม่ของ ImapClient และเชื่อมต่อกับเซิร์ฟเวอร์
ImapClient client = new ImapClient("imap.domain.com", "username", "password");
```

- **พารามิเตอร์:**
  - `"imap.domain.com"`:แทนที่ด้วยที่อยู่เซิร์ฟเวอร์ IMAP ของคุณ
  - `"username"`- `"password"`: ข้อมูลการเข้าสู่ระบบของคุณ

**3.2 เชื่อมต่อกับเซิร์ฟเวอร์**

ตรวจสอบให้แน่ใจว่าคุณจัดการข้อยกเว้นในระหว่างการเชื่อมต่อเพื่อการจัดการข้อผิดพลาดที่มีประสิทธิภาพ

### คุณสมบัติ 2: เริ่มตรวจสอบเหตุการณ์ IMAP

**ภาพรวม:** การตรวจสอบเหตุการณ์อีเมลแบบเรียลไทม์ เช่น ข้อความใหม่หรือข้อความที่ถูกลบ สามารถปรับปรุงการตอบสนองและการทำงานของแอปพลิเคชันของคุณได้

#### การดำเนินการทีละขั้นตอน:

**3.3 ตั้งค่าการตรวจสอบเหตุการณ์**

```csharp
using System.Threading;
using Aspose.Email.Clients.Imap;

// เริ่มเหตุการณ์การรีเซ็ตด้วยตนเองเพื่อจัดการการแจ้งเตือนแบบอะซิงโครนัส
ManualResetEvent manualResetEvent = new ManualResetEvent(false);
ImapMonitoringEventArgs eventArgs = null;

// เริ่มตรวจสอบเหตุการณ์ IMAP
client.StartMonitoring(delegate(object sender, ImapMonitoringEventArgs e)
{
    eventArgs = e; // จับข้อโต้แย้งเหตุการณ์
    manualResetEvent.Set(); // สัญญาณว่าเหตุการณ์ได้เกิดขึ้น
});

Thread.Sleep(2000); // เผื่อเวลาให้เหตุการณ์ต่างๆ เกิดขึ้นบ้าง
```

- **การกำหนดค่าคีย์:** ใช้ `StartMonitoring` วิธีการที่มีผู้มอบหมายในการจัดการการแจ้งเตือน
  
**3.4 การจัดการการแจ้งเตือน**
การ `manualResetEvent` ช่วยซิงโครไนซ์กระบวนการตรวจสอบโดยส่งสัญญาณเมื่อเหตุการณ์เกิดขึ้น

### คุณสมบัติที่ 3: ส่งอีเมลโดยใช้ไคลเอนต์ SMTP

**ภาพรวม:** การส่งอีเมลทำได้ง่ายด้วย `SmtpClient` คลาสใน Aspose.Email สำหรับ .NET

#### การดำเนินการทีละขั้นตอน:

**3.5 เริ่มต้น SmtpClient**

```csharp
using Aspose.Email.Clients.Smtp;
using Aspose.Email.Mime;

// สร้างอินสแตนซ์ของ SmtpClient
SmtpClient smtpClient = new SmtpClient("exchange.aspose.com", "username", "password");
```

- **พารามิเตอร์:**
  - `"exchange.aspose.com"`: ที่อยู่เซิร์ฟเวอร์ SMTP
  - `"username"`- `"password"`: ข้อมูลประจำตัวในการส่งอีเมล์

**3.6 ส่งอีเมล์**

```csharp
// สร้างและส่งข้อความอีเมล์ใหม่
smtpClient.Send(new MailMessage("from@aspose.com", "to@aspose.com",
    "EMAILNET-34875 - " + Guid.NewGuid(), 
    "EMAILNET-34875 Support for IMAP idle command"));

manualResetEvent.WaitOne(10000); // รอให้มีการดำเนินการเหตุการณ์
```

### คุณสมบัติ 4: หยุดการตรวจสอบเหตุการณ์ IMAP

**ภาพรวม:** การหยุดกระบวนการตรวจสอบอย่างปลอดภัยจะช่วยให้แอปพลิเคชันของคุณสามารถจัดการทรัพยากรได้อย่างมีประสิทธิภาพ

#### การดำเนินการทีละขั้นตอน:

**3.7 หยุดการตรวจสอบ**

```csharp
// ใช้การ StopMonitoring เพื่อหยุดการฟังเหตุการณ์
client.StopMonitoring("Inbox");

smtpClient.Send(new MailMessage("from@aspose.com", "to@aspose.com",
    "EMAILNET-34875 - " + Guid.NewGuid(), 
    "EMAILNET-34875 Support for IMAP idle command"));

manualResetEvent.WaitOne(5000); // ให้แน่ใจว่ามีการจัดการเหตุการณ์ทั้งหมด
```
