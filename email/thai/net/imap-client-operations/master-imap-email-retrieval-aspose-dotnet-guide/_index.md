## คู่มือการใช้งาน

### เริ่มต้น ImapClient
การเริ่มต้นของ `ImapClient` เป็นสิ่งสำคัญสำหรับการตั้งค่าการเชื่อมต่อที่ปลอดภัยกับเซิร์ฟเวอร์ IMAP ของคุณ คุณสามารถกำหนดค่าได้ดังนี้:

#### การตั้งค่าโฮสต์และพอร์ต
ระบุโฮสต์เซิร์ฟเวอร์ IMAP และหมายเลขพอร์ต:
- **เจ้าภาพ**: ใช้ชื่อโดเมนหรือที่อยู่ IP ของผู้ให้บริการอีเมล์ของคุณ
- **ท่าเรือ**โดยทั่วไป 993 จะใช้สำหรับการเชื่อมต่อ SSL
```csharp
imapClient.Host = "<HOST>";
imapClient.Port = 993;
```

#### รายละเอียดการรับรองความถูกต้อง
ระบุชื่อผู้ใช้และรหัสผ่านของคุณสำหรับการตรวจสอบสิทธิ์ ซึ่งจะอนุญาตให้เข้าถึงบัญชีอีเมลของคุณได้:
```csharp
imapClient.Username = "<USERNAME>";
imapClient.Password = "<PASSWORD>";
```

#### โปรโตคอลการเข้ารหัส
รับประกันการสื่อสารที่ปลอดภัยโดยการตั้งค่าโปรโตคอลการเข้ารหัสที่รองรับ:
```csharp
imapClient.SupportedEncryption = EncryptionProtocols.Tls;
```

### รายการข้อความจากเซิร์ฟเวอร์ IMAP
เมื่อเชื่อมต่อแล้ว คุณสามารถแสดงรายการข้อความทั้งหมดที่มีอยู่ในกล่องจดหมายของคุณได้:

#### ดึงข้อมูลการรวบรวมข้อความ
ใช้ `ListMessages` เพื่อรับการรวบรวมข้อมูลข้อความ:
```csharp
ImapMessageInfoCollection messageInfoCol = imapClient.ListMessages();
int[] sequenceNumberAr = messageInfoCol.Select(mi => mi.SequenceNumber).ToArray();
string[] uniqueIdAr = messageInfoCol.Select(mi => mi.UniqueId).ToArray();
```

### ดึงข้อความตามหมายเลขลำดับ
หากต้องการดึงอีเมลเฉพาะ คุณสามารถใช้หมายเลขลำดับได้:

#### การดึงข้อมูลโดยใช้หมายเลขลำดับ
ส่งหมายเลขลำดับที่ต้องการไปยัง `FetchMessages`-
```csharp
IList<MailMessage> fetchedMessages = imapClient.FetchMessages(sequenceNumbers);
```

### ดึงข้อความโดยใช้ ID เฉพาะ
อีกวิธีหนึ่งคือการดึงข้อความโดยใช้ ID เฉพาะ:

#### ดึงข้อมูลอีเมล์โดยใช้รหัสเฉพาะ
ใช้ตัวระบุเฉพาะที่ได้รับมาก่อนหน้านี้เพื่อดึงอีเมล:

2. **โซลูชันการสำรองข้อมูล**:นำระบบสำรองอีเมลด้วยการดึงข้อมูลผ่านโปรแกรมโดยใช้ IMAP
