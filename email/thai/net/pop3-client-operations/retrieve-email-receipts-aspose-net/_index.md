## คู่มือการใช้งาน

ในหัวข้อนี้ เราจะอธิบายวิธีดึงข้อมูลใบตอบรับการอ่านและการจัดส่ง

### การดึงข้อมูลใบเสร็จ

#### ภาพรวม:
คุณสมบัตินี้ช่วยให้คุณแยกและวิเคราะห์ว่าอีเมลที่คุณส่งถูกเปิดหรือส่งสำเร็จหรือไม่

#### ขั้นตอนที่ 1: โหลดข้อความอีเมล์
เริ่มต้นด้วยการโหลด `.msg` ไฟล์ที่ประกอบด้วยข้อความอีเมล นี่คือจุดเริ่มต้นในการค้นหาข้อมูลใบเสร็จ

**โค้ดตัวอย่าง:**
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
MapiMessage msg = MapiMessage.FromFile(dataDir + "\TestMessage.msg");
```

#### ขั้นตอนที่ 2: ทำซ้ำผู้รับ
สำหรับผู้รับแต่ละราย ตรวจสอบสถานะการอ่านและการจัดส่งใบเสร็จ

**การเข้าถึงข้อมูลผู้รับ:**
```csharp
foreach (MapiRecipient recipient in msg.Recipients)
{
    Console.WriteLine(string.Format("Recipient: {0}", recipient.DisplayName));
    
    // ตรวจสอบข้อมูลใบเสร็จ
    if (recipient.MessageStatus == MapiMessageStatus.Read && recipient.ReceiptType == MapiRecipientReceiptType.Read)
    {
        Console.WriteLine("Read Receipt Received.");
    }

    if (recipient.MessageStatus == MapiMessageStatus.Delivered)
    {
        Console.WriteLine("Delivery Receipt Received.");
    }
}
```

**คำอธิบาย:**
- **MapiMessage จากไฟล์**: โหลดข้อความจากไฟล์ที่ระบุ
- **ข้อความผู้รับ**: ให้สิทธิ์ในการเข้าถึงรายละเอียดของผู้รับแต่ละราย
- **MessageStatus และ ReceiptType**:ใช้ในการตรวจสอบสถานะการรับสินค้า

### เคล็ดลับการแก้ไขปัญหา:
- ให้แน่ใจว่าคุณ `.msg` ไฟล์ได้รับการจัดรูปแบบอย่างถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่า Aspose.Email ได้รับการติดตั้งและอ้างอิงอย่างถูกต้องในโครงการของคุณ
