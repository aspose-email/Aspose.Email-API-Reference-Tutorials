## คู่มือการใช้งาน
เราจะกล่าวถึงคุณสมบัติหลักสองประการ: การเชื่อมต่อกับเซิร์ฟเวอร์ POP3 และการค้นหาอีเมลด้วยตัวกรองที่แยกแยะตัวพิมพ์เล็ก/ใหญ่

### คุณสมบัติ 1: เชื่อมต่อและเข้าสู่ระบบ POP3

#### ภาพรวม:
การเชื่อมต่อกับเซิร์ฟเวอร์ POP3 เป็นขั้นตอนแรกในการจัดการอีเมลของคุณผ่านโปรแกรม Aspose.Email สำหรับ .NET ช่วยลดความยุ่งยากของกระบวนการนี้ ช่วยให้คุณสามารถรวมฟังก์ชันอีเมลเข้ากับแอปพลิเคชันของคุณได้อย่างง่ายดาย

**ขั้นตอนที่ 1: กำหนดพารามิเตอร์การเชื่อมต่อ**
สร้างคลาสที่รวมรายละเอียดการเชื่อมต่อและการเริ่มต้นระบบ `Pop3Client`-

```csharp
using Aspose.Email.Clients.Pop3;

namespace Pop3ConnectionExample
{
    public class ConnectAndLoginPOP3
    {
        public void Run()
        {
            // กำหนดพารามิเตอร์การเชื่อมต่อ
            const string host = "your.pop3.host.com";     // ระบุโฮสต์เซิร์ฟเวอร์ POP3
            const int port = 110;                         // หมายเลขพอร์ต POP3 เริ่มต้น
            const string username = "user@host.com";      // ที่อยู่อีเมล์ของคุณ
            const string password = "password";           // รหัสผ่านสำหรับบัญชีอีเมล์ของคุณ

            // สร้างอินสแตนซ์ Pop3Client ด้วยพารามิเตอร์ที่กำหนด
            Pop3Client client = new Pop3Client(host, port, username, password);
            
            // ทางเลือก: ตรวจสอบสถานะการเชื่อมต่อ
            if (client.Connected)
            {
                Console.WriteLine("Connected to POP3 server successfully.");
            }
        }
    }
}
```

**ตัวเลือกการกำหนดค่าคีย์:**
- **เจ้าภาพ**: ที่อยู่เซิร์ฟเวอร์ POP3 ของคุณ
- **ท่าเรือ**โดยทั่วไปคือ 110 สำหรับการเชื่อมต่อที่ไม่ปลอดภัยหรือ 995 สำหรับการเชื่อมต่อที่ปลอดภัย
- **ชื่อผู้ใช้และรหัสผ่าน**: ข้อมูลประจำตัวสำหรับการพิสูจน์ตัวตนกับเซิร์ฟเวอร์

#### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าการตั้งค่าไฟร์วอลล์อนุญาตการเชื่อมต่อกับพอร์ตที่ระบุ
- ตรวจสอบว่าข้อมูลประจำตัวและรายละเอียดเซิร์ฟเวอร์ถูกต้อง
- ใช้บล็อก try-catch เพื่อจัดการข้อยกเว้นอย่างเหมาะสม

### คุณสมบัติที่ 2: ใช้ตัวกรองตามตัวพิมพ์ใหญ่-เล็กเพื่อค้นหาอีเมล

#### ภาพรวม:
การค้นหาอีเมลตามเกณฑ์เฉพาะนั้นมีความสำคัญสำหรับแอปพลิเคชันต่างๆ มากมาย Aspose.Email ช่วยให้คุณสามารถใช้ตัวกรองที่คำนึงถึงตัวพิมพ์เล็ก/ใหญ่ได้ ซึ่งจะช่วยเพิ่มความแม่นยำในการค้นหา

**ขั้นตอนที่ 1: เชื่อมต่อและพิสูจน์ตัวตน**
ทำให้มั่นใจ `Pop3Client` ได้รับการตั้งค่าเรียบร้อยแล้วตามที่แสดงในฟีเจอร์ 1

```csharp
using Aspose.Email.Tools.Search;

namespace EmailSearchExample
{
    public class CaseSensitiveEmailSearch
    {
        public void Run()
        {
            // ถือว่า Pop3Client เชื่อมต่อและได้รับการตรวจสอบแล้ว
            Pop3Client client = new Pop3Client("your.pop3.host.com", 110, "user@host.com", "password");

            try
            {
                // สร้างอินสแตนซ์ MailQueryBuilder
                MailQueryBuilder builder1 = new MailQueryBuilder();

                // เพิ่มตัวกรองแบบแยกแยะตัวพิมพ์ใหญ่-เล็กสำหรับอีเมลจาก 'tesT'
                builder1.From.Contains("tesT", true);

                // รับแบบสอบถามตามการกำหนดค่าของผู้สร้าง
                MailQuery query1 = builder1.GetQuery();
                
                // รายการข้อความที่ตรงกับเกณฑ์การค้นหา
                Pop3MessageInfoCollection messageInfoCol1 = client.ListMessages(query1);
                
                Console.WriteLine($"Found {messageInfoCol1.Count} emails matching the criteria.");
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error searching emails: " + ex.Message);
            }
        }
    }
}
```

**ตัวเลือกการกำหนดค่าคีย์:**
- **ความไวต่อตัวพิมพ์ใหญ่และเล็ก**: ตั้งค่าเป็น `true` เพื่อให้ตรงกับกรณีที่แน่นอน
- **เครื่องมือสร้างแบบสอบถาม**:อำนวยความสะดวกให้การสร้างแบบสอบถามที่ซับซ้อนได้อย่างง่ายดาย

#### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบการเชื่อมต่อเครือข่ายเมื่อเข้าถึงเซิร์ฟเวอร์
- จัดการข้อยกเว้นเพื่อหลีกเลี่ยงการหยุดทำงานของแอพพลิเคชั่นในระหว่างการดำเนินการค้นหาอีเมล

## การประยุกต์ใช้งานจริง
ต่อไปนี้เป็นกรณีการใช้งานจริงบางส่วนที่สามารถนำคุณลักษณะเหล่านี้ไปใช้:

2. **โซลูชันการเก็บถาวรอีเมล์**:เชื่อมต่อและดาวน์โหลดอีเมลเฉพาะเพื่อวัตถุประสงค์ในการเก็บถาวร เพื่อให้แน่ใจว่าข้อมูลที่ละเอียดอ่อนได้รับการระบุอย่างถูกต้อง

4. **การวิเคราะห์การตลาด**ติดตามประสิทธิผลของแคมเปญส่งเสริมการขายโดยค้นหาอีเมลที่มีคำสำคัญหรือวลีที่เจาะจง
5. **การบูรณาการกับ CRM**ปรับปรุงระบบ CRM ด้วยการดึงและประมวลผลการสื่อสารกับลูกค้าผ่านทาง POP3
