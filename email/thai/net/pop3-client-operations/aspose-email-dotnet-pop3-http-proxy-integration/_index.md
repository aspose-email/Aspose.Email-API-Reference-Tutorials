### การเข้าถึงกล่องจดหมาย POP3 ผ่านทางพร็อกซี HTTP
คุณสมบัตินี้ช่วยให้แอปพลิเคชันของคุณเชื่อมต่อกับเซิร์ฟเวอร์ POP3 โดยใช้พร็อกซี HTTP ตัวกลาง ซึ่งมีความสำคัญในสภาพแวดล้อมเครือข่ายที่มีข้อจำกัด

#### สร้างอินสแตนซ์ของ HttpProxy
เริ่มต้นด้วยการสร้าง `HttpProxy` อินสแตนซ์ที่มีรายละเอียดโฮสต์และพอร์ตที่จำเป็น ซึ่งจะกำหนดค่าการเชื่อมต่อของคุณผ่านพร็อกซีที่ระบุ:
```csharp
// กำหนดค่าพร็อกซีของคุณ
HttpProxy proxy = new HttpProxy("18.222.124.59", 8080); // แทนที่ด้วยที่อยู่พร็อกซีและพอร์ตจริง
```

#### เริ่มต้นไคลเอนต์ POP3
ตั้งค่า `Pop3Client` ในการโต้ตอบกับกล่องจดหมายผ่านพร็อกซี HTTP:
```csharp
// กำหนดค่าการตั้งค่าเซิร์ฟเวอร์อีเมลของคุณ
Pop3Client client = new Pop3Client("pop.example.com", "username", "password");

// กำหนดอินสแตนซ์ HttpProxy ให้กับไคลเอนต์
client.Proxy = proxy;
```
- **พารามิเตอร์**-
  - `"pop.example.com"`:ชื่อโฮสต์ของเซิร์ฟเวอร์ POP3
  - `"username"` และ `"password"`ข้อมูลประจำตัวสำหรับการเข้าถึงกล่องจดหมายของคุณ

#### การเชื่อมต่อและการดึงอีเมล
เมื่อการตั้งค่าเสร็จสมบูรณ์แล้ว ให้เชื่อมต่อกับเซิร์ฟเวอร์และรับอีเมล:
```csharp
try
{
    client.Connect(true); // ใช้ SSL หากเซิร์ฟเวอร์ต้องการ
    int messageCount = client.GetMessageCount();
    
    Console.WriteLine($"Total Messages: {messageCount}");

    for (int i = 1; i <= messageCount; i++)
    {
        MailMessage msg = client.FetchMessage(i);
        Console.WriteLine($"Subject: {msg.Subject}");
    }
}
catch (Exception ex)
{
    Console.WriteLine("Error: " + ex.Message);
}
fine
{
    client.Dispose();
}
```
- **ค่าส่งคืน**-
  - `GetMessageCount()`: ดึงจำนวนข้อความทั้งหมดในกล่องจดหมาย
  - `FetchMessage(int)`:ดึงอีเมลที่เฉพาะเจาะจงโดยดัชนีข้อความ

#### เคล็ดลับการแก้ไขปัญหา
ปัญหาทั่วไป ได้แก่ ข้อผิดพลาดในการเชื่อมต่อเครือข่ายหรือความล้มเหลวในการรับรองความถูกต้อง ตรวจสอบให้แน่ใจว่าการตั้งค่าพร็อกซีของคุณถูกต้องและคุณมีข้อมูลประจำตัวเซิร์ฟเวอร์ที่ถูกต้อง นอกจากนี้ ให้ตรวจสอบว่าเซิร์ฟเวอร์ POP3 จำเป็นต้องใช้ SSL/TLS สำหรับการเชื่อมต่อที่ปลอดภัยหรือไม่
