## คู่มือการใช้งาน
ดึงข้อมูลสรุปอีเมลโดยใช้ไคลเอนต์ POP3 ของ Aspose.Email

### ดึงข้อมูลสรุปข้อความโดยใช้รหัสเฉพาะ
#### ภาพรวม
ดึงข้อมูลสำคัญ เช่น หัวเรื่องและวันที่โดยไม่ต้องดาวน์โหลดข้อความทั้งหมด เหมาะสำหรับการสแกนอีเมลอย่างรวดเร็ว

#### ขั้นตอน
**ขั้นตอนที่ 1: กำหนดรายละเอียดเซิร์ฟเวอร์**
ระบุรายละเอียดเซิร์ฟเวอร์ POP3 ของคุณ:
```csharp
string host = "host.domain.com"; // แทนที่ด้วยโดเมนโฮสต์จริง
int port = 456; // หมายเลขพอร์ตที่ถูกต้อง
string username = "username"; // ชื่อผู้ใช้จริง
string password = "password"; // รหัสผ่านจริง
```

**ขั้นตอนที่ 2: สร้างอินสแตนซ์ Pop3Client**
การเริ่มต้น `Pop3Client` และกำหนดค่าตัวเลือกความปลอดภัย:
```csharp
Pop3Client client = new Pop3Client(host, port, username, password);
client.SecurityOptions = SecurityOptions.Auto;
```

**ขั้นตอนที่ 3: กำหนดรหัสข้อความที่ไม่ซ้ำกัน**
ระบุข้อความโดยใช้ ID เฉพาะ:
```csharp
string uniqueId = "unique id of a message from server"; // รหัสประจำตัวที่ไม่ซ้ำใครจริง
```

**ขั้นตอนที่ 4: ดึงข้อมูลสรุป**
รับรายละเอียดสรุปด้วย `GetMessageInfo` วิธี:
```csharp
Pop3MessageInfo messageInfo = client.GetMessageInfo(uniqueId);
```

**ขั้นตอนที่ 5: ส่งออกรายละเอียดข้อความ**
ตรวจสอบและพิมพ์ข้อมูลที่เรียกค้น:
```csharp
if (messageInfo != null)
{
    Console.WriteLine(messageInfo.Subject); // หัวข้อข้อความ
    Console.WriteLine(messageInfo.Date);    // วันที่ข้อความ
}
```
#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบข้อมูลประจำตัวเซิร์ฟเวอร์ POP3
- ตรวจสอบให้แน่ใจว่ามีรหัสเฉพาะของข้อความอยู่ในกล่องจดหมาย
