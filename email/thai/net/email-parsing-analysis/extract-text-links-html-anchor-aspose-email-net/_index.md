# วิธีการแยกข้อความและลิงก์จากแท็กสมอ HTML โดยใช้ Aspose.Email สำหรับ .NET

## คู่มือการใช้งาน
ให้เราแบ่งการใช้งานออกเป็นสองคุณสมบัติหลัก: การแยกไฮเปอร์ลิงก์ที่มีแอตทริบิวต์ href และการดึงข้อความธรรมดาจากแท็กสมอ

### คุณสมบัติ 1: เรนเดอร์ไฮเปอร์ลิงก์ด้วย Href
คุณลักษณะนี้ช่วยให้คุณแยกทั้ง URL และข้อความที่เกี่ยวข้องจากแท็กยึด HTML ได้

#### ภาพรวม
คุณจะวิเคราะห์สตริง HTML เพื่อดึงข้อมูลอ้างอิงไฮเปอร์ลิงก์ (`href`) และแสดงข้อความภายใน `<a>` แท็ก

#### การดำเนินการแบบทีละขั้นตอน

**ขั้นตอนที่ 1:** ระบุตัวตน `href` ตำแหน่งของแอตทริบิวต์

```csharp
string source = "<a href='https://example.com'>ตัวอย่าง</a>";
int startHref = source.IndexOf("href=\"") + "href=\"".Length;
```

*ทำไม* ขั้นตอนนี้จะระบุตำแหน่งของไฮเปอร์ลิงก์ภายในแท็กเพื่อให้สามารถแยกข้อมูลได้อย่างแม่นยำ

**ขั้นตอนที่ 2:** กำหนดจุดสิ้นสุดของ `href` คุณลักษณะ.

```csharp
int endHref = source.IndexOf("\"", startHref);
string href = source.Substring(startHref, endHref - startHref);
```

*ทำไม* มันช่วยแยก URL โดยการทำเครื่องหมายจุดสิ้นสุดไว้ภายในแท็ก

**ขั้นตอนที่ 3:** การแยกข้อความระหว่าง `<a>` แท็ก

```csharp
int startText = source.IndexOf(">") + 1;
int endText = source.IndexOf("<", startText);
string text = source.Substring(startText, endText - startText);
```

*ทำไม* นี่จะจับข้อความลิงก์ที่มองเห็นได้เพื่อการแสดงผลหรือการใช้งานในแอปพลิเคชันของคุณ

**ขั้นตอนที่ 4:** รวมข้อความและ href เพื่อสร้างผลลัพธ์

```csharp
string link = $"{text} <{href}>";
Console.WriteLine(link); // ผลลัพธ์: ตัวอย่าง <https://example.com>
```

### คุณสมบัติที่ 2: เรนเดอร์ไฮเปอร์ลิงก์โดยไม่มี Href
คุณลักษณะนี้มุ่งเน้นที่การดึงเฉพาะข้อความที่มองเห็นได้จากแท็กสมอ โดยไม่สนใจ URL

#### ภาพรวม
มีประโยชน์เมื่อคุณต้องการเพียงข้อความที่แสดงสำหรับอินเทอร์เฟซผู้ใช้หรือการประมวลผลเพิ่มเติม

#### การดำเนินการแบบทีละขั้นตอน

**แยกข้อความเท่านั้น**

```csharp
int startNoHref = source.IndexOf(">") + 1;
int endNoHref = source.IndexOf("<", startNoHref);
string plainText = source.Substring(startNoHref, endNoHref - startNoHref);
Console.WriteLine(plainText); // ผลลัพธ์: ตัวอย่าง
```

*ทำไม* วิธีนี้จะดึงข้อความอย่างมีประสิทธิภาพโดยไม่ต้องประมวลผล URL

3. **โครงการขูดข้อมูล:** ดึงข้อมูลและวิเคราะห์ไฮเปอร์ลิงก์จากหน้าเว็บ
