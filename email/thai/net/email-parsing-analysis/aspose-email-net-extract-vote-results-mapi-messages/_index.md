## คู่มือการใช้งาน

มาแบ่งการใช้งานออกเป็นขั้นตอนที่จัดการได้เพื่อให้แน่ใจว่าคุณสามารถอ่านผลการโหวตจากข้อความ MAPI ได้อย่างมีประสิทธิภาพ

### การอ่านข้อมูลผลการโหวต

ฟีเจอร์นี้จะแสดงวิธีการดึงข้อมูลการโหวต เช่น คำตอบและเวลาตอบกลับจากผู้รับอีเมล ต่อไปนี้คือรายละเอียดทีละขั้นตอน:

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอกสาร
เริ่มต้นด้วยการระบุเส้นทางที่ไฟล์ข้อความของคุณตั้งอยู่
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/AddVotingButtonToExistingMessage.msg";
```

#### ขั้นตอนที่ 2: โหลดข้อความ MAPI
โหลดข้อความ MAPI จากไฟล์โดยใช้ Aspose.Email `MapiMessage` ระดับ.
```csharp
MapiMessage msg = MapiMessage.FromFile(dataDir);
```

#### ขั้นตอนที่ 3: ทำซ้ำผ่านผู้รับ
วนซ้ำผ่านผู้รับแต่ละรายเพื่อเข้าถึงคำตอบการโหวตและเวลาตอบกลับของพวกเขา
```csharp
foreach (MapiRecipient recipient in msg.Recipients)
{
    // ดึงข้อมูลชื่อที่แสดงของผู้รับ
    string displayName = recipient.DisplayName;

    // แยกคำตอบการโหวตโดยใช้คุณสมบัติ PR_RECIPIENT_AUTORESPONSE_PROP_RESPONSE
    string response = recipient.Properties[MapiPropertyTag.PR_RECIPIENT_AUTORESPONSE_PROP_RESPONSE].GetString();

    // รับเวลาตอบสนองด้วยคุณสมบัติ PR_RECIPIENT_TRACKSTATUS_TIME
    DateTime responseTime = recipient.Properties[MapiPropertyTag.PR_RECIPIENT_TRACKSTATUS_TIME].GetDateTime();
}
```

#### คำอธิบายรหัส
- **ชื่อที่แสดง**- `recipient.DisplayName` ให้ตัวระบุที่สามารถอ่านได้สำหรับผู้รับแต่ละราย
- **คุณสมบัติการตอบสนอง**:ใช้คุณสมบัติ PR_RECIPIENT_AUTORESPONSE_PROP_RESPONSE เพื่อเข้าถึงคำตอบการโหวต
- **เวลาตอบสนอง**:PR_RECIPIENT_TRACKSTATUS_TIME จะบันทึกเวลาที่บันทึกการตอบสนองแต่ละครั้ง ซึ่งมีประโยชน์สำหรับการติดตามการมีส่วนร่วม

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ข้อความของคุณถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่า Aspose.Email ได้รับการติดตั้งและอ้างอิงอย่างถูกต้องในโครงการของคุณ
- หากไม่มีคุณสมบัติ ให้ตรวจสอบว่าไคลเอนต์อีเมลที่ใช้รองรับคุณสมบัติ MAPI เหล่านี้หรือไม่

2. **การรวบรวมคำติชม**:ใช้อีเมล์อัตโนมัติเพื่อรวบรวมความคิดเห็นเกี่ยวกับผลิตภัณฑ์หรือบริการอย่างมีประสิทธิภาพ
3. **การวางแผนกิจกรรม**ติดตามการตอบรับ RSVP สำหรับกิจกรรมโดยตรงผ่านการโต้ตอบทางอีเมล
