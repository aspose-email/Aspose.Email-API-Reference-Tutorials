## คู่มือการใช้งาน
ในส่วนนี้ เราจะแนะนำคุณเกี่ยวกับการอ่านข้อความจากไฟล์ NSF โดยใช้ Aspose.Email สำหรับ .NET

### การเข้าถึงข้อความ NSF
**ภาพรวม:**
อ่านและระบุข้อความที่เก็บไว้ในไฟล์ NSF ได้อย่างง่ายดาย `NotesStorageFacility` คลาสนี้ให้การเข้าถึงไฟล์เหล่านี้ ซึ่งทำให้สามารถทำซ้ำในแต่ละข้อความได้

#### ขั้นตอนที่ 1: สร้างอินสแตนซ์ NotesStorageFacility
สร้างอินสแตนซ์ของ `NotesStorageFacility` ระดับ:

```csharp
using Aspose.Email.Storage.Nsf;

string dataDir = @"C:\path\to\your\nsf\files"; // อัพเดตเส้นทางนี้
using (NotesStorageFacility nsf = new NotesStorageFacility(dataDir + "\SampleNSF.nsf"))
{
    // เราจะแจกแจงข้อความไว้ที่นี่
}
```

#### ขั้นตอนที่ 2: ระบุข้อความ
ใช้ `EnumerateMessages` วิธีการวนซ้ำผ่านแต่ละข้อความ:

```csharp
foreach (MailMessage eml in nsf.EnumerateMessages())
{
    System.Console.WriteLine(eml.Subject);
}
```

**คำอธิบาย:**
- **พารามิเตอร์**: ผู้สร้างของ `NotesStorageFacility` นำเส้นทางไปยังไฟล์ NSF
- **วิธีการ วัตถุประสงค์**- `EnumerateMessages()` ดึงข้อความแต่ละข้อความเป็น `MailMessage`โดยให้สามารถเข้าถึงคุณสมบัติต่างๆ ได้ เช่น เรื่องของ...
