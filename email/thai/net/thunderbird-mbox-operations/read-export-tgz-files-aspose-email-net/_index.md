## คู่มือการใช้งาน

### คุณสมบัติ 1: การอ่านไฟล์ TGZ และการส่งออกเนื้อหา

ฟีเจอร์นี้สาธิตวิธีการเปิดไฟล์ TGZ โดยใช้ไลบรารี Aspose.Email และส่งออกเนื้อหาของไฟล์นั้น

#### การดำเนินการทีละขั้นตอน:

**ขั้นตอนที่ 1: เริ่มต้นเส้นทาง**
ตั้งค่าเส้นทางสำหรับไดเร็กทอรีอินพุต (ข้อมูล) และเอาท์พุต แทนที่ตัวแทนด้วยชื่อไดเร็กทอรีจริง
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

**ขั้นตอนที่ 2: เปิดไฟล์ TGZ**
การเริ่มต้น `TgzReader` เพื่ออ่านจากไฟล์ TGZ ที่ระบุ

```csharp
try
{
    using (TgzReader reader = new TgzReader(dataDir + "/ZimbraSample.tgz"))
    {
        // ดำเนินการส่งออกเนื้อหาในขั้นตอนที่ 3
```

**ขั้นตอนที่ 3: ส่งออกเนื้อหา**
ส่งออกเนื้อหาของไฟล์ TGZ ไปยังไดเร็กทอรีเอาต์พุตที่กำหนด

```csharp
reader.ExportTo(outputDir);
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message);  // จัดการข้อยกเว้นอย่างสง่างาม
}
```

#### คำอธิบาย:
- `TgzReader` อ่านและแตกไฟล์เก็บถาวร TGZ
- การ `ExportTo` วิธีการคัดลอกไฟล์ไปยังตำแหน่งใหม่ ช่วยให้ดึงข้อมูลได้ง่าย

### คุณลักษณะที่ 2: การตั้งค่าเส้นทางสำหรับไดเร็กทอรีข้อมูลและเอาต์พุต

ตรวจสอบให้แน่ใจว่าไดเร็กทอรีของคุณพร้อมสำหรับการดำเนินการไฟล์โดยทำตามขั้นตอนเหล่านี้:

**ขั้นตอนที่ 1: ตรวจสอบไดเรกทอรีเอกสาร**
ตรวจสอบว่าไดเร็กทอรีเอกสารมีอยู่หรือไม่ หากจำเป็น ให้สร้างขึ้นมาใหม่
```csharp
string documentDirectory = "YOUR_DOCUMENT_DIRECTORY";

if (!Directory.Exists(documentDirectory))
{
    Directory.CreateDirectory(documentDirectory);
}
```

**ขั้นตอนที่ 2: ตรวจสอบไดเรกทอรีผลลัพธ์**
ในทำนองเดียวกัน ให้แน่ใจว่ามีการตั้งค่าไดเร็กทอรีเอาต์พุตแล้ว
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";

if (!Directory.Exists(outputDirectory))
{
    Directory.CreateDirectory(outputDirectory);
}
```

#### คำอธิบาย:
- ใช้ `Directory.Exists` เพื่อตรวจสอบการมีอยู่ของไดเร็กทอรี
- `Directory.CreateDirectory` จัดการการสร้างหากจำเป็น

### เคล็ดลับการแก้ไขปัญหา

1. **ไม่พบไฟล์**:ตรวจสอบให้แน่ใจว่าเส้นทางถูกระบุอย่างถูกต้องและมีไดเร็กทอรีอยู่
2. **ปัญหาการอนุญาต**:เรียกใช้แอปพลิเคชันของคุณด้วยสิทธิ์ที่เหมาะสม โดยเฉพาะการสร้างไดเร็กทอรีหรือการเข้าถึงไฟล์
