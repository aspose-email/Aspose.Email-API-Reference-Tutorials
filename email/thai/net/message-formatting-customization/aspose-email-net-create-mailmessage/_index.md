## คู่มือการใช้งาน

ตอนนี้เรามาดูวิธีการสร้างและกำหนดค่ากัน `MailMessage` ทีละขั้นตอน:

### การสร้างอินสแตนซ์ของ MailMessage

เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `MailMessage`อ็อบเจ็กต์นี้ช่วยให้คุณสามารถกำหนดและจัดการเนื้อหาอีเมลผ่านโปรแกรมได้

```csharp
using Aspose.Email.Mime;

// สร้างอินสแตนซ์ใหม่ของ MailMessage
MailMessage message = new MailMessage("Sender <sender@from.com>", "Recipient <recipient@to.com>");
```

#### คำอธิบาย:
- **`new MailMessage()`**: เริ่มต้นข้อความเมลด้วยผู้ส่งและผู้รับหลัก
- **`"Sender <sender@from.com>"`**: กำหนดแหล่งที่มาของอีเมล

### การกำหนดค่าที่อยู่ 'ถึง'

เพิ่มผู้รับหลายรายลงในของคุณ `MailMessage`สิ่งนี้จำเป็นสำหรับการส่งอีเมลถึงหลายๆ คนในครั้งเดียว

```csharp
// เพิ่มที่อยู่ 'ถึง' หลายรายการในอีเมล
message.To.Add("receiver1@receiver.com");
message.To.Add("receiver2@receiver.com");
message.To.Add("receiver3@receiver.com");
message.To.Add("receiver4@receiver.com");
```

#### คำอธิบาย:
- **`message.To.Add()`**:ผนวกที่อยู่ผู้รับแต่ละรายลงในรายชื่อที่อยู่ 'ถึง'

### การเพิ่มที่อยู่ CC (สำเนาคาร์บอน)

ผู้รับ CC จะได้รับสำเนาอีเมลของคุณและจะปรากฏให้ผู้รับคนอื่นๆ ทราบ ซึ่งมีประโยชน์ในการแจ้งข้อมูลให้บุคคลที่เกี่ยวข้องทราบ

```csharp
// เพิ่มที่อยู่ 'CC' (สำเนาคาร์บอน)
message.CC.Add("CC1@receiver.com");
message.CC.Add("CC2@receiver.com");
```

#### คำอธิบาย:
- **`message.CC.Add()`**: เพิ่มที่อยู่อีเมลลงในรายชื่อผู้รับ CC

### การเพิ่มที่อยู่ BCC (สำเนาคาร์บอนแบบซ่อน)

BCC ช่วยให้คุณส่งอีเมลโดยไม่เปิดเผยที่อยู่ผู้รับทั้งหมด และรักษาความเป็นส่วนตัวของผู้ติดต่อบางราย

```csharp
// เพิ่มที่อยู่ 'BCC' (สำเนาคาร์บอนแบบซ่อน)
message.Bcc.Add("Bcc1@receiver.com");
message.Bcc.Add("Bcc2@receiver.com");
```

#### คำอธิบาย:
- **`message.Bcc.Add()`**: เพิ่มที่อยู่อีเมลลงในรายการ BCC

### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าที่อยู่อีเมลทั้งหมดถูกต้อง
- ตรวจสอบการติดตั้งไลบรารีอีกครั้งหากเกิดข้อผิดพลาดระหว่างการตั้งค่า
