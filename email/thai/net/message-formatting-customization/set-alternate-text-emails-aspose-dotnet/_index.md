## คู่มือการใช้งาน

ต่อไปเรามาดูการตั้งค่าข้อความทางเลือกสำหรับข้อความอีเมล์กัน

### สร้างอินสแตนซ์ MailMessage
เริ่มต้นด้วยการประกาศ `MailMessage` วัตถุ:

```csharp
// ประกาศอินสแตนซ์ MailMessage
MailMessage message = new MailMessage();
```

ขั้นตอนนี้จะเริ่มต้นวัตถุอีเมลของคุณซึ่งคุณจะเพิ่มเนื้อหาและการกำหนดค่า

### ตั้งค่าข้อความทางเลือกด้วย AlternateView
มุมมองทางเลือกช่วยให้สามารถแสดงอีเมลฉบับเดียวกันได้หลายรูปแบบ วิธีสร้างมุมมองทางเลือกมีดังนี้

```csharp
// สร้าง AlternateView โดยมีเนื้อหาที่ระบุเป็นสตริง
AlternateView alternate = AlternateView.CreateAlternateViewFromString("This is the alternate text.");
```

การ `CreateAlternateViewFromString` วิธีนี้ใช้สตริงข้อความธรรมดา ทำให้มั่นใจได้ว่าหากอีเมลของคุณไม่สามารถแสดง HTML หรือไฟล์แนบได้อย่างถูกต้อง ข้อความนี้จะถูกแสดงแทน

### เพิ่ม AlternateView ให้กับ MailMessage
สุดท้ายให้เพิ่มมุมมองทางเลือกให้กับข้อความของคุณ:

```csharp
// เพิ่ม AlternateView ที่สร้างขึ้นลงในคอลเล็กชัน AlternateViews ของ MailMessage
message.AlternateViews.Add(alternate);
```

ขั้นตอนนี้จะช่วยให้แน่ใจว่าอีเมลของคุณสามารถกลับมาใช้ข้อความนี้ได้เมื่อจำเป็น
