## คู่มือการใช้งาน
เรามาดูขั้นตอนการสร้างและบันทึกงาน Outlook ทีละขั้นตอนกัน

### การสร้างวัตถุ MapiTask
เอ `MapiTask` วัตถุแสดงถึงงาน Outlook เริ่มต้นด้วยการเริ่มต้นด้วยคุณสมบัติที่จำเป็น:

#### ขั้นตอนที่ 1: กำหนดงาน
```csharp
MapiTask task = new MapiTask(
    “สิ่งที่ต้องทำ”, 
    "Just click and type to add new task", 
    DateTime.Now, 
    DateTime.Now.AddDays(3)
);
```
- **"To Do"** เป็นวิชาที่.
- คำอธิบายให้ข้อมูลเพิ่มเติม
- วันที่เริ่มต้นและวันครบกำหนดจะถูกตั้งเป็นวันที่ปัจจุบันและสามวันจากนี้

#### ขั้นตอนที่ 2: กำหนดความก้าวหน้าและความพยายาม
```csharp
task.PercentComplete = 20;
task.EstimatedEffort = 2000; // ภายในไม่กี่นาที
```
- กำหนดเปอร์เซ็นต์ความเสร็จสมบูรณ์ของงาน
- กำหนดเวลาประมาณการความพยายามเป็นนาทีเพื่อติดตามเวลาที่ใช้ไป

#### ขั้นตอนที่ 3: ประวัติงานและการอัปเดต
```csharp
task.History = MapiTaskHistory.Assigned;
task.LastUpdate = DateTime.Now;
```
- บันทึกเมื่องานได้รับมอบหมายหรืออัปเดตครั้งล่าสุดเพื่อการติดตามที่ดีขึ้น

### การกำหนดค่าความเป็นเจ้าของและบริษัท
กำหนดรายละเอียดความเป็นเจ้าของและบริษัทที่เกี่ยวข้อง:

```csharp
task.Users.Owner = "Darius";
task.Users.LastAssigner = "Harkness";
task.Users.LastDelegate = "Harkness";
task.Users.Ownership = MapiTaskOwnership.AssignersCopy;

task.Companies = new string[] { "company1", "company2", "company3" };
```

### การจัดหมวดหมู่และการเพิ่มข้อมูลเมตา
ใช้หมวดหมู่เพื่อจัดระเบียบ:

```csharp
task.Categories = new string[] { "category1", "category2", "category3" };
task.Mileage = "Some test mileage";
task.Billing = "Test billing information";
task.Users.Delegator = "Test Delegator";
```

### การสรุปคุณสมบัติของงาน
ตั้งค่าความไวและสถานะ:

```csharp
task.Sensitivity = MapiSensitivity.Personal;
task.Status = MapiTaskStatus.Complete;

// ปรับประมาณการความพยายามหากจำเป็น
task.EstimatedEffort = 5; // ภายในไม่กี่นาที
```

### การบันทึกงานเป็นไฟล์ MSG
สุดท้ายให้บันทึกงานของคุณ:

```csharp
string outputPath = "@YOUR_OUTPUT_DIRECTORY/MapiTask.msg";
task.Save(outputPath, TaskSaveFormat.Msg);
```

ให้แน่ใจว่าได้เปลี่ยน `@YOUR_OUTPUT_DIRECTORY` พร้อมด้วยเส้นทางไดเร็กทอรีจริงที่คุณต้องการบันทึกไฟล์
