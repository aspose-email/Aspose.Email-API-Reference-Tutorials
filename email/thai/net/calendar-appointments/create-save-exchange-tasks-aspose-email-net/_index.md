## คู่มือการใช้งาน

### การสร้างงานการแลกเปลี่ยน

การสร้างงาน Exchange เกี่ยวข้องกับการตั้งค่าคุณสมบัติเพื่อกำหนดขอบเขตและสถานะ มาแบ่งขั้นตอนที่เกี่ยวข้องออกเป็นดังนี้:

#### 1. เริ่มต้นวัตถุ ExchangeTask

เริ่มต้นด้วยการสร้างอินสแตนซ์ใหม่ของ `ExchangeTask` คลาส วัตถุนี้จะแสดงงานของคุณในเซิร์ฟเวอร์ Exchange

```csharp
using Aspose.Email.Clients.Exchange.WebService;

// สร้างอินสแตนซ์ใหม่ของคลาส ExchangeTask
ExchangeTask task = new ExchangeTask();
```

#### 2. กำหนดหัวข้องาน

กำหนดตัวระบุที่ไม่ซ้ำกันให้กับแต่ละงานโดยใช้ `Guid.NewGuid()`. ซึ่งจะช่วยในการแยกแยะงานเมื่อจัดการรายการหลายรายการ

```csharp
// ตั้งค่าหัวเรื่องด้วยตัวระบุที่ไม่ซ้ำกัน
string taskSubject = "TASK-ID - " + Guid.NewGuid();
task.Subject = taskSubject;
```

#### 3. กำหนดสถานะงาน

กำหนดสถานะของงานของคุณเพื่อระบุสถานะปัจจุบัน สำหรับตัวอย่างนี้ เราจะทำเครื่องหมายว่า `InProgress`-

```csharp
// ทำเครื่องหมายงานว่ากำลังดำเนินการอยู่
	task.Status = ExchangeTaskStatus.InProgress;
```

#### 4. กำหนดวันที่เริ่มต้นและครบกำหนด

กำหนดเวลาที่งานเริ่มต้นและวันที่ครบกำหนดที่จะต้องทำให้เสร็จ

```csharp
// ตั้งวันที่เริ่มต้นเป็นเวลาปัจจุบัน
	task.StartDate = DateTime.Now;

// กำหนดวันครบกำหนดภายในสามวันจากนี้
	task.DueDate = task.StartDate.AddDays(3);
```

#### 5. บันทึกงานเป็นไฟล์ข้อความ

สุดท้าย ให้บันทึกงานที่คุณกำหนดค่าไว้เป็น `.msg` ไฟล์บนดิสก์

```csharp
// บันทึกงานลงในไดเร็กทอรีที่ระบุ
	task.Save(@"YOUR_DOCUMENT_DIRECTORY\Message.msg");
```

### เคล็ดลับการแก้ไขปัญหา

- ให้แน่ใจว่า `YOUR_DOCUMENT_DIRECTORY` จะถูกแทนที่ด้วยเส้นทางจริงที่คุณมีสิทธิ์ในการเขียน
- หากคุณพบปัญหาใดๆ โปรดตรวจสอบการตั้งค่าใบอนุญาต Aspose.Email ของคุณและตรวจสอบการอัปเดต
