## คู่มือการใช้งาน

### เชื่อมต่อกับ Exchange Web Service (EWS)

การเชื่อมต่อกับ EWS เป็นขั้นตอนแรกในการจัดการปฏิทินด้วยโปรแกรม ช่วยให้คุณสามารถเข้าถึงและจัดการข้อมูลปฏิทินโดยใช้ Aspose.Email

#### ภาพรวม
ฟีเจอร์นี้สาธิตวิธีการสร้างการเชื่อมต่อกับเซิร์ฟเวอร์ Exchange ผ่านจุดสิ้นสุดของบริการเว็บ

#### ขั้นตอน:

##### 1. สร้างอินสแตนซ์ของ `IEWSClient`
คุณจะต้องมีข้อมูลประจำตัวและ URL ของบริการสำหรับขั้นตอนนี้
```csharp
using (IEWSClient client = EWSClient.GetEWSClient(
    "https://อีเมล: outlook.office365.com/ews/exchange.asmx
    "testUser",
    "pwd",
    "domain"))
{
    // สร้างการเชื่อมต่อสำเร็จแล้ว
}
```

- **พารามิเตอร์**-
  - `"https://outlook.office365.com/ews/exchange.asmx"`: URL ของบริการเว็บ Exchange
  - `"testUser"`- `"pwd"`- `"domain"`: ข้อมูลประจำตัวสำหรับการยืนยันตัวตน

##### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าข้อมูลประจำตัวของคุณมีสิทธิ์เพียงพอในการเข้าถึง EWS
- ตรวจสอบว่า URL บริการถูกต้องและสามารถเข้าถึงได้จากเครือข่ายของคุณ

### การอนุญาตการเข้าถึงปฏิทินของผู้แทน

เมื่อเชื่อมต่อแล้ว คุณสามารถมอบสิทธิ์การเข้าถึงปฏิทินให้กับผู้ใช้รายอื่นได้ คุณลักษณะนี้จะช่วยจัดการว่าใครสามารถดูหรือแก้ไขกิจกรรมในปฏิทินที่ต้องการได้

#### ภาพรวม
หัวข้อนี้จะแสดงวิธีการตั้งค่าผู้ใช้ผู้รับมอบสิทธิ์พร้อมกับสิทธิ์การเข้าถึงโฟลเดอร์ปฏิทินที่เจาะจง

#### ขั้นตอน:

##### 1. ตั้งค่าผู้ใช้มอบหมาย
```csharp
ExchangeDelegateUser delegateUser = new ExchangeDelegateUser("sharingfrom@domain.com", ExchangeDelegateFolderPermissionLevel.NotSpecified);
delegateUser.FolderPermissions.CalendarFolderPermissionLevel = ExchangeDelegateFolderPermissionLevel.Reviewer;
```

- **พารามิเตอร์**-
  - `"sharingfrom@domain.com"`:ที่อยู่อีเมลของผู้ใช้ที่ต้องการมอบสิทธิ์ให้
  - `ExchangeDelegateFolderPermissionLevel.Reviewer`: กำหนดระดับสิทธิ์ในการเข้าถึงปฏิทิน

##### 2. การเข้าถึงการมอบหมาย
```csharp
client.DelegateAccess(delegateUser, "sharingfrom@domain.com");
```

### สร้างและส่งคำเชิญการแชร์ปฏิทิน

การสร้างคำเชิญให้แชร์ปฏิทินเป็นสิ่งสำคัญสำหรับการกำหนดตารางเวลาร่วมกัน คุณลักษณะนี้จะทำให้กระบวนการเชิญผู้ใช้เข้าร่วมกิจกรรมในปฏิทินของคุณเป็นไปโดยอัตโนมัติ

#### ภาพรวม
เรียนรู้วิธีการสร้างและส่งคำเชิญการแชร์ปฏิทินโดยใช้ Aspose.Email

#### ขั้นตอน:

##### 1. เชื่อมต่อกับ EWS
สร้างการเชื่อมต่อใหม่อีกครั้งตามที่แสดงไว้ในส่วนก่อนหน้า

##### 2. สร้างคำเชิญแบ่งปันปฏิทิน
```csharp
MapiMessage mapiMessage = client.CreateCalendarSharingInvitationMessage("sharingfrom@domain.com");
```

- **พารามิเตอร์**-
  - `"sharingfrom@domain.com"`: ที่อยู่อีเมลของผู้ได้รับคำเชิญ

##### 3. แปลงและส่งข้อความ
```csharp
MailConversionOptions options = new MailConversionOptions { แปลงเป็นTnef = true };
var mail = mapiMessage.ToMailMessage(options);
client.Send(mail);
```

- **ConvertAsTnef**:รับประกันความเข้ากันได้กับไคลเอนต์อีเมลที่ต้องการรูปแบบ TNEF

3. **การวางแผนกิจกรรม**ปรับปรุงคำเชิญเข้าร่วมกิจกรรมโดยการส่งคำเชิญผ่านปฏิทินไปยังผู้เข้าร่วมโดยอัตโนมัติ
