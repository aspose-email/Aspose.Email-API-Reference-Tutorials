# วิธีการสร้าง MapiTask ที่เกิดขึ้นซ้ำทุกวันโดยใช้ Aspose.Email สำหรับ .NET | คำแนะนำทีละขั้นตอน

## การแนะนำ

การจัดการงานที่เกิดขึ้นซ้ำๆ ในแต่ละวันอย่างมีประสิทธิภาพถือเป็นสิ่งสำคัญสำหรับการรักษาประสิทธิภาพการทำงาน ด้วย Aspose.Email สำหรับ .NET คุณสามารถสร้างและกำหนดค่างาน Outlook ได้อย่างราบรื่นด้วยการเขียนโปรแกรม คู่มือนี้จะแนะนำคุณตลอดขั้นตอนการสร้าง `MapiTask`การตั้งค่าคุณสมบัติ และเพิ่มรูปแบบการเกิดซ้ำรายวันโดยใช้คุณสมบัติอันแข็งแกร่งของ Aspose.Email

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่าสภาพแวดล้อมของคุณด้วย Aspose.Email สำหรับ .NET
- การสร้างและการกำหนดค่า `MapiTask` โดยมีแอตทริบิวต์เช่น ชื่อ เนื้อหา วันเริ่มต้น วันครบกำหนด และรัฐ
- การเพิ่มรูปแบบการเกิดซ้ำรายวันให้กับงาน
- การบันทึกงานที่กำหนดค่าไว้เป็นไฟล์ข้อความ Outlook

มาเริ่มด้วยการครอบคลุมข้อกำหนดเบื้องต้นกันก่อน

## คู่มือการใช้งาน

### สร้างและกำหนดค่า MapiTask
การสร้าง `MapiTask` เกี่ยวข้องกับการตั้งค่าคุณลักษณะที่จำเป็น เช่น ชื่อ เนื้อหา วันเริ่มต้น วันครบกำหนด และรัฐ

#### การสร้างงาน
```csharp
using Aspose.Email.Mapi;

DateTime StartDate = new DateTime(2015, 7, 16);
DateTime DueDate = new DateTime(2015, 7, 16);

// สร้างอินสแตนซ์ MapiTask ใหม่
task = new MapiTask("This is test task", "Sample Body", StartDate, DueDate);

// ตั้งค่าสถานะของงานเป็น NotAssigned
task.State = MapiTaskState.NotAssigned;
```
**คำอธิบาย**: ที่นี่เราจะสร้างตัวอย่าง `MapiTask` โดยมีชื่อ เนื้อหา วันที่เริ่มต้น และวันที่ครบกำหนด นอกจากนี้เรายังกำหนดสถานะเริ่มต้นด้วย

### กำหนดรูปแบบการเกิดซ้ำรายวันสำหรับ MapiTask
เพิ่มรูปแบบการเกิดซ้ำรายวันเพื่อให้แน่ใจว่างานจะเกิดซ้ำไปเรื่อยๆ

#### การตั้งค่ารูปแบบการเกิดซ้ำ
```csharp
var record = new MapiCalendarDailyRecurrencePattern
{
    PatternType = MapiCalendarRecurrencePatternType.Day,
    Period = 1, // งานจะเกิดขึ้นซ้ำทุกวัน
    EndType = MapiCalendarRecurrenceEndType.NeverEnd, // การเกิดขึ้นซ้ำไม่สิ้นสุด
};

// กำหนดรูปแบบการเกิดซ้ำให้กับงาน
task.Recurrence = record;
```
**คำอธิบาย**:ส่วนย่อยนี้จะกำหนดรูปแบบการเกิดซ้ำรายวันซึ่งจะไม่มีวันสิ้นสุด `PatternType` ถูกตั้งเป็น `Day`, และ `Period` ระบุช่วงเวลาเป็นวันระหว่างเหตุการณ์ที่เกิดขึ้น

### บันทึก MapiTask ลงในไฟล์
สุดท้าย ให้บันทึกงานที่กำหนดค่าของคุณเป็นไฟล์ข้อความ Outlook

#### การบันทึกงาน
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // แทนที่ด้วยเส้นทางไดเร็กทอรีเอกสารของคุณ

// บันทึก MapiTask ลงในไฟล์ .msg
task.Save(dataDir + "/SetDailyNeverEndRecurrence_out.msg", TaskSaveFormat.Msg);
```
**คำอธิบาย**:รหัสนี้จะบันทึกงานของคุณลงใน `.msg` ไฟล์ที่สามารถเปิดใน Outlook ได้
