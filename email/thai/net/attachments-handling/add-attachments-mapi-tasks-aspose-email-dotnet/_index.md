# วิธีการเพิ่มไฟล์แนบในงาน MAPI โดยใช้ Aspose.Email สำหรับ .NET

## การแนะนำ

การจัดการงานอีเมลที่มีไฟล์แนบสามารถเพิ่มประสิทธิภาพการทำงานได้อย่างมาก คู่มือนี้สาธิตวิธีการเพิ่มไฟล์แนบโดยตรงภายในงาน MAPI โดยใช้ Aspose.Email สำหรับ .NET ซึ่งเป็นไลบรารีที่ครอบคลุมซึ่งออกแบบมาเพื่อการจัดการอีเมลและงานต่างๆ ได้อย่างง่ายดาย

### สิ่งที่คุณจะได้เรียนรู้:
- การรวมสิ่งที่แนบมาในงาน MAPI ด้วย Aspose.Email

## คู่มือการใช้งาน

### ภาพรวมของการเพิ่มสิ่งที่แนบมาในงาน MAPI

ฟีเจอร์นี้ช่วยให้สามารถแนบไฟล์โดยตรงกับงานที่สร้างโดยใช้โปรโตคอล MAPI ซึ่งเป็นประโยชน์สำหรับระบบการจัดการงานที่ต้องการเอกสารประกอบหรือไฟล์ที่เกี่ยวข้องแนบมาโดยตรง

#### ขั้นตอนที่ 1: สร้างและกำหนดค่างานของคุณ
เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `MapiTask`วัตถุนี้แสดงถึงงานอีเมลของคุณ

```csharp
using System;
using Aspose.Email.Mapi;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// สร้างงาน MAPI ใหม่พร้อมรายละเอียดที่ระบุ
MapiTask testTask = new MapiTask("Task with attachment", "This is the description of your task.", DateTime.Now, DateTime.Now.AddDays(1));
```
*หมายเหตุ: เปลี่ยน `YOUR_DOCUMENT_DIRECTORY` และ `YOUR_OUTPUT_DIRECTORY` ด้วยเส้นทางจริงบนระบบของคุณ*

#### ขั้นตอนที่ 2: เพิ่มสิ่งที่แนบมาในงานของคุณ
หากต้องการเพิ่มสิ่งที่แนบมา ให้ใช้ `MapiAttachment` คลาส ระบุเส้นทางไฟล์และชื่อสำหรับสิ่งที่แนบมา

```csharp
// สร้างไฟล์แนบ MAPI
string filePath = System.IO.Path.Combine(dataDir, "sample.pdf");
MapiAttachment attachment = new MapiAttachment("sample.pdf", System.IO.File.ReadAllBytes(filePath));

// เพิ่มสิ่งที่แนบมากับงานของคุณ
testTask.Attachments.Add(attachment);
```
*คำอธิบาย: เราอ่านไบต์ไฟล์จาก `filePath` และสร้างใหม่ `MapiAttachment`ซึ่งจะถูกเพิ่มเข้าในไฟล์แนบของงาน*

#### ขั้นตอนที่ 3: บันทึกงานของคุณ
สุดท้าย ให้บันทึกงาน MAPI ของคุณด้วยไฟล์แนบในไดเร็กทอรีเอาต์พุต

```csharp
// กำหนดเส้นทางการบันทึก
string outputPath = System.IO.Path.Combine(outputDir, "TaskWithAttachment.msg");

// บันทึกงานเป็นไฟล์ .msg
testTask.Save(outputPath);
```

### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าไดเร็กทอรี `dataDir` และ `outputDir` มีอยู่ก่อนที่จะรันโค้ดของคุณ
- ตรวจสอบข้อยกเว้นที่เกี่ยวข้องกับเส้นทางไฟล์หรือการอนุญาต
