## คู่มือการใช้งาน

### ดึงข้อมูลสิ่งที่แนบมาจากไฟล์ MSG ของ Outlook
คุณลักษณะนี้ช่วยให้คุณสามารถแยกสิ่งที่แนบมาจากไฟล์ MSG ของ Outlook และบันทึกไว้ในเครื่องได้

#### คำแนะนำทีละขั้นตอน:
**1. โหลดไฟล์ MSG**
ขั้นแรกให้โหลดไฟล์ MSG โดยใช้ `MapiMessage.FromFile()` วิธี.

```csharp
using Aspose.Email.Mapi;

// ตั้งค่าเส้นทางไดเร็กทอรีเอกสารของคุณ
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// ระบุชื่อไฟล์ MSG ของคุณ
string fileName = "outputAttachments.msg";

// โหลดไฟล์ MSG ลงในวัตถุ MapiMessage
MapiMessage message = MapiMessage.FromFile(dataDir + "/" + fileName);
```

**2. แตกไฟล์และบันทึกไฟล์แนบ**
ทำซ้ำผ่านสิ่งที่แนบมาแต่ละรายการในไฟล์ MSG ที่โหลดแล้ว บันทึกลงในไดเร็กทอรีเอาต์พุตที่คุณต้องการ

```csharp
// ระบุเส้นทางไดเร็กทอรีเอาท์พุตของคุณ
string outputPath = "YOUR_OUTPUT_DIRECTORY";

foreach (MapiAttachment attachment in message.Attachments)
{
    // บันทึกสิ่งที่แนบมาแต่ละรายการโดยใช้ชื่อไฟล์ดั้งเดิม
    attachment.Save(outputPath + "/" + attachment.FileName);
}
```

**คำอธิบาย:**
- `MapiMessage.FromFile()`:โหลดไฟล์ MSG ลงในอ็อบเจ็กต์ที่สามารถจัดการได้
- `message.Attachments`: เข้าถึงการรวบรวมสิ่งที่แนบมาภายในไฟล์ MSG
- `attachment.Save()`: บันทึกสิ่งที่แนบมาแต่ละรายการไปยังไดเร็กทอรีที่คุณระบุ

### โหลดและประมวลผลไฟล์ MSG ของ Outlook
การโหลดไฟล์ MSG เป็นเพียงขั้นตอนแรกเท่านั้น ต่อไปนี้เป็นวิธีเริ่มต้นกระบวนการนี้:

```csharp
using Aspose.Email.Mapi;

// ตั้งค่าเส้นทางสำหรับไดเร็กทอรีข้อมูลและไดเร็กทอรีเอาต์พุตตามที่แสดงไว้ก่อนหน้านี้
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string fileName = "outputAttachments.msg";

// โหลด MSG ลงในวัตถุ MapiMessage ตามที่สาธิตไว้ก่อนหน้านี้
MapiMessage message = MapiMessage.FromFile(dataDir + "/" + fileName);

// ขณะนี้คุณสามารถประมวลผลสิ่งที่แนบมาหรือส่วนอื่น ๆ ของอีเมลได้
```

## การประยุกต์ใช้งานจริง
การแยกและบันทึกสิ่งที่แนบมาจากไฟล์ MSG มีประโยชน์หลายประการ:
- **การเก็บข้อมูลถาวร**: ทำการเก็บถาวรแบบอัตโนมัติเพื่อวัตถุประสงค์ด้านการปฏิบัติตามกฎระเบียบ
- **เวิร์กโฟลว์การประมวลผลอีเมล**:บูรณาการเข้ากับระบบที่ต้องการการจัดการเนื้อหาอีเมลอัตโนมัติ
- **เครื่องมือย้ายเนื้อหา**:ใช้งานในเครื่องมือที่ออกแบบมาเพื่อย้ายอีเมลระหว่างแพลตฟอร์มที่แตกต่างกัน
