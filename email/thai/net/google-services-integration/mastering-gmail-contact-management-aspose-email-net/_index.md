### คุณสมบัติ 1: การตรวจสอบสิทธิ์ OAuth

การตรวจสอบสิทธิ์ OAuth เป็นสิ่งสำคัญสำหรับการเข้าถึงรายชื่อติดต่อ Gmail อย่างปลอดภัย คุณสามารถตั้งค่าได้ดังนี้:

#### ภาพรวม
ฟีเจอร์นี้สาธิตการรับโทเค็นการเข้าถึงและการรีเฟรชโทเค็นซึ่งจำเป็นสำหรับการโต้ตอบกับ Gmail ผ่านทาง Aspose.Email

**การดำเนินการแบบทีละขั้นตอน**

1. **กำหนดข้อมูลประจำตัวผู้ใช้**
   ```csharp
   GoogleTestUser user = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
   ```

2. **รับการเข้าถึงและรีเฟรชโทเค็น**
   ```csharp
   string accessToken;
   string refreshToken;
   GoogleOAuthHelper.GetAccessToken(user, out accessToken, out refreshToken);
   ```

ขั้นตอนนี้จะช่วยให้มั่นใจถึงการเข้าถึงที่ปลอดภัยโดยการแลกเปลี่ยนข้อมูลประจำตัวลูกค้าเป็นโทเค็น

### คุณสมบัติที่ 2: การสร้างรายชื่อผู้ติดต่อ Gmail

การสร้างรายละเอียดการติดต่อที่ครอบคลุมใน Gmail สามารถทำได้โดยอัตโนมัติด้วย Aspose.Email

#### ภาพรวม
เรียนรู้วิธีป้อนข้อมูลในฟิลด์ต่างๆ เช่น ที่อยู่ หมายเลขโทรศัพท์ และกิจกรรม เมื่อสร้างรายชื่อติดต่อ Gmail ใหม่

**การดำเนินการแบบทีละขั้นตอน**

1. **เริ่มต้นการติดต่อใหม่**
   ```csharp
   Contact contact = new Contact();
   ```

2. **กรอกข้อมูลพื้นฐาน**
   ```csharp
   contact.GivenName = "GivenName";
   contact.Surname = "Surname";
   ```

3. **เพิ่มที่อยู่และเบอร์โทรศัพท์**
   ```csharp
   PostalAddress address = new PostalAddress {
       Address = "Address",
       City = "City"
   };
   contact.PhysicalAddresses.Add(address);

   PhoneNumber pnWork = new PhoneNumber { Number = "1234567890", Category = PhoneNumberCategory.Work };
   contact.PhoneNumbers.Add(pnWork);
   ```

4. **เพิ่มรายละเอียดเพิ่มเติม**
   ```csharp
   contact.Events.Birthday = DateTime.Now.AddYears(-30);
   contact.EmailAddresses.Add(new EmailAddress { Address = "email@gmail.com" });
   ```

### การใช้ IGmailClient เพื่อสร้างข้อมูลติดต่อ

#### ภาพรวม
เรียนรู้วิธีใช้อินเทอร์เฟซ IGmailClient เพื่อสร้างรายชื่อติดต่อใน Gmail ด้วยโปรแกรม

**การดำเนินการแบบทีละขั้นตอน**

1. **เริ่มต้น IGmailClient**
   ```csharp
   IGmailClient client = GmailClient.GetInstance(accessToken, user.EMail);
   ```

2. **สร้างการติดต่อ**
   ```csharp
   string contactUri = client.CreateContact(contact);
   ```

กระบวนการนี้ช่วยให้สร้างรายชื่อติดต่อได้โดยอัตโนมัติและเป็นจำนวนมาก ช่วยเพิ่มประสิทธิภาพ
