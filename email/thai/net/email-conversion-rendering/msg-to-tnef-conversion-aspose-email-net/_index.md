## คู่มือการใช้งาน

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการดำเนินการแปลง MSG เป็น TNEF และการจัดการรูปแบบต่างๆ

### แปลง MSG เป็น TNEF

#### ภาพรวม
การแปลงไฟล์ Outlook MSG เป็น TNEF จะรวมเนื้อหาที่หลากหลายไว้ด้วยกัน ทำให้มั่นใจได้ถึงความเที่ยงตรงในระหว่างการขนส่งข้ามระบบ

#### ขั้นตอนการแปลง
**1. โหลดไฟล์ MSG:**
```csharp
using System.IO;
using Aspose.Email.Mapi;

string documentDir = "YOUR_DOCUMENT_DIRECTORY";
MapiMessage msg = MapiMessage.FromFile(documentDir + "/Message.msg");
```

**2. ตั้งค่าตัวเลือกการแปลง:**
```csharp
MailConversionOptions options = new MailConversionOptions { ConvertAsTnef = true };
```

**3. แปลงและประมวลผล:**
```csharp
using Aspose.Email;

MailMessage mail = msg.ToMailMessage(options);
// การประมวลผลเพิ่มเติมหรือการบันทึก 'เมล' สามารถเกิดขึ้นได้ที่นี่
```

### โหลดและบันทึกไฟล์อีเมล์
#### ภาพรวม
การโหลดและบันทึกอีเมลในรูปแบบต่างๆ ถือเป็นสิ่งสำคัญ ในส่วนนี้จะสาธิตการโหลดไฟล์อีเมลและบันทึกเป็น MSG

**1. โหลดข้อความอีเมล์:**
```csharp
using Aspose.Email;
using System.IO;

string documentDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

MailMessage loadedEmail = MailMessage.Load(documentDir + "/SampleEmail.eml");
```

**2. บันทึกในรูปแบบอื่น:**
```csharp
loadedEmail.Save(outputDir + "/ConvertedEmail.msg", SaveOptions.DefaultMsgUnicode);
// พารามิเตอร์ SaveOptions ระบุรูปแบบเอาต์พุต
```
