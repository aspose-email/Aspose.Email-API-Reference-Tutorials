### คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเปลี่ยนแบบอักษรในระหว่างการแปลง EML เป็น MHT

#### ขั้นตอนที่ 1: ตั้งค่าเส้นทางไดเรกทอรี
กำหนดเส้นทางสำหรับไฟล์อินพุตและเอาต์พุตของคุณ:

```csharp
string dataDir = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "Data");
string inputFile = Path.Combine(dataDir, "input.eml");
string outputFile = Path.Combine(dataDir, "output.mht");
```

#### ขั้นตอนที่ 2: โหลดไฟล์ EML
โหลดไฟล์ EML ของคุณลงใน `MailMessage` วัตถุ:

```csharp
var message = MailMessage.Load(inputFile);
```

การโหลดอีเมลทำให้คุณสามารถจัดการเนื้อหาได้ก่อนการแปลง

#### ขั้นตอนที่ 3: ปรับแต่งรูปแบบแบบอักษร
ปรับแต่งเนื้อหา HTML ของอีเมลโดยการเปลี่ยนรูปแบบอักษร:

```csharp
message.HtmlBody = message.HtmlBody.Replace("font-family", "font-family: 'Your Custom Font', sans-serif;");
```

โค้ดตัวอย่างนี้แทนที่อินสแตนซ์ของ `font-family` ตามสไตล์ที่คุณต้องการ

#### ขั้นตอนที่ 4: แปลงเป็น MHT
บันทึกอีเมลที่แก้ไขเป็นไฟล์ MHT:

```csharp
var mhtSaveOptions = new MhtmlSaveOptions();
message.Save(outputFile, mhtSaveOptions);
```

การ `MhtmlSaveOptions` คลาสนี้อนุญาตให้มีการกำหนดค่าเพิ่มเติมได้หากจำเป็น
