## คู่มือการใช้งาน

ตอนนี้คุณได้ตั้งค่า Aspose.Email ในโครงการของคุณแล้ว เรามาเริ่มใช้งานฟีเจอร์การแปลง MIME เป็น EML กันเลย เราจะแบ่งฟีเจอร์นี้ออกเป็นขั้นตอนที่ทำตามได้ง่าย

### แปลงข้อความ MIME เป็น EML (H2)

คุณลักษณะนี้ช่วยให้สามารถแปลงข้อความ MIME เป็นไฟล์ EML ซึ่งมักจำเป็นสำหรับการเก็บถาวรอีเมลหรือการย้ายข้อมูล

#### ขั้นตอนที่ 1: โหลดข้อความ MIME

ขั้นแรก โหลดข้อความ MIME ของคุณจากไฟล์:

```csharp
using Aspose.Email.Mime;
using System.IO;

// กำหนดไดเรกทอรี (แทนที่ด้วยเส้นทางจริง)
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY/";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY/";

// โหลดข้อความ MIME
MailMessage msg = MailMessage.Load(documentDirectory + "Message2.msg");
```

**คำอธิบาย**: ที่นี่เราใช้ `MailMessage.Load()` เพื่ออ่านข้อความจากไฟล์ แทนที่ `"YOUR_DOCUMENT_DIRECTORY/"` และ `"YOUR_OUTPUT_DIRECTORY/"` ด้วยเส้นทางที่เฉพาะเจาะจงของคุณ

#### ขั้นตอนที่ 2: บันทึกข้อความเป็น EML

ขั้นตอนต่อไป ให้บันทึกข้อความที่โหลดนี้ในรูปแบบ EML:

```csharp
// แปลงและบันทึกข้อความ MIME เป็นไฟล์ EML
msg.Save(outputDirectory + "ConvertMIMEMessageToEML_out.eml");
```

**คำอธิบาย**: เดอะ `Save()` วิธีการเขียนอีเมลลงในดิสก์ ตรวจสอบให้แน่ใจว่าไดเร็กทอรีเอาต์พุตของคุณสามารถเขียนได้

### เคล็ดลับการแก้ไขปัญหา
- **ข้อยกเว้นไม่พบไฟล์**ตรวจสอบเส้นทางไฟล์ของคุณอีกครั้ง
- **ปัญหาการอนุญาต**: ตรวจสอบสิทธิ์การเขียนสำหรับไดเร็กทอรีเอาท์พุต
