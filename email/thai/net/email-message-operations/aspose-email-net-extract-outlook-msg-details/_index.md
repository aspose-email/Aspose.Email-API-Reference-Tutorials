## คู่มือการใช้งาน

### กำลังโหลดไฟล์ MSG
#### ภาพรวม
เราจะเริ่มต้นด้วยการโหลดไฟล์ MSG และแยกคุณสมบัติพื้นฐาน เช่น หัวเรื่อง ที่อยู่อีเมลของผู้ส่ง เนื้อหา ผู้รับ และไฟล์แนบ

#### การดำเนินการแบบทีละขั้นตอน
**1. โหลดไฟล์ MSG**
สร้าง `MapiMessage` อินสแตนซ์จากไฟล์ MSG ของคุณ:
```csharp
using System;
using Aspose.Email.Mapi;

// ระบุเส้นทางไปยังไฟล์ MSG ของคุณ
string dataDir = @"C:\Path\To\Your\File\message.msg";

// สร้างอินสแตนซ์ของ MapiMessage จากไฟล์
MapiMessage msg = MapiMessage.FromFile(dataDir);
```
**2. ดึงข้อมูลหัวเรื่องและผู้ส่ง**
ดึงข้อมูลหัวเรื่องและที่อยู่อีเมลของผู้ส่ง:
```csharp
// รับเรื่อง
Console.WriteLine("Subject: " + msg.Subject);

// รับจากที่อยู่
Console.WriteLine("From: " + msg.SenderEmailAddress);
```
**3. ดึงข้อมูลเนื้อหาอีเมล**
แสดงเนื้อหาอีเมล์:
```csharp
// รับร่างกาย
Console.WriteLine("Body: " + msg.Body);
```
**4. ดึงข้อมูลผู้รับ**
วนรอบผู้รับแต่ละรายและพิมพ์ที่อยู่อีเมลของพวกเขา:
```csharp
// รับข้อมูลผู้รับ
foreach (MapiRecipient recipient in msg.Recipients)
{
    Console.WriteLine("Recipient: " + recipient.EmailAddress);
}
```
**5. รายการสิ่งที่แนบมา**
ระบุรายการสิ่งที่แนบมาทั้งหมดและแสดงชื่อของสิ่งที่แนบมา:
```csharp
// รับไฟล์แนบ
foreach (MapiAttachment att in msg.Attachments)
{
    Console.WriteLine("Attachment Name: " + att.FileName);
    Console.WriteLine("Attachment Display Name: " + att.DisplayName);
}
```

## การประยุกต์ใช้งานจริง
ฟังก์ชันนี้สามารถใช้งานได้ในสถานการณ์ต่างๆ เช่น:
1. **ระบบจัดเก็บอีเมล์ถาวร**:ทำให้การดึงข้อมูลรายละเอียดอีเมล์ออกมาเป็นอัตโนมัติเพื่อวัตถุประสงค์ในการเก็บถาวร
2. **เครื่องมือสนับสนุนลูกค้า**:รวมเข้ากับระบบที่ต้องการวิเคราะห์อีเมลลูกค้าสำหรับตั๋วการสนับสนุน
3. **การตลาดแบบอัตโนมัติ**:แยกและวิเคราะห์เนื้อหาอีเมลเพื่อปรับแต่งกลยุทธ์ทางการตลาด
