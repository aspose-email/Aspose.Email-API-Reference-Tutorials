## คู่มือการใช้งาน

ในส่วนนี้เราจะแบ่งกระบวนการออกเป็นสองคุณลักษณะหลัก: การโหลดไฟล์ EML และบันทึกกลับไปยังดิสก์

### คุณสมบัติ 1: โหลดไฟล์ EML

#### ภาพรวม
ฟีเจอร์นี้สาธิตวิธีการโหลดไฟล์ EML ที่มีอยู่โดยใช้ Aspose.Email สำหรับ .NET เหมาะอย่างยิ่งสำหรับแอปพลิเคชันที่ต้องอ่านเนื้อหาอีเมลด้วยโปรแกรม

##### คำแนะนำทีละขั้นตอน

**ขั้นตอนที่ 1**: ตั้งค่าไดเรกทอรี

กำหนดเส้นทางที่ไฟล์ EML ของคุณอยู่:

```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";
```

**ขั้นตอนที่ 2**:โหลดไฟล์ EML

ใช้ `MailMessage.Load` เพื่ออ่านไฟล์ EML วิธีนี้จะแยกวิเคราะห์อีเมลและให้ข้อมูล `MailMessage` คัดค้านให้ดำเนินการต่อไป

```csharp
using Aspose.Email.Mime;

// โหลดไฟล์ EML ที่มีอยู่
MailMessage mailMessage = MailMessage.Load(dataDir + "/Attachments.eml");
```

**คำอธิบาย**- 
- การ `Load` ฟังก์ชันอ่านไฟล์ EML ที่คุณระบุและแปลงเป็น `MailMessage` วัตถุที่ช่วยให้คุณสามารถจัดการข้อมูลอีเมล์ภายในแอปพลิเคชันของคุณได้

#### ภาพรวม

##### คำแนะนำทีละขั้นตอน
