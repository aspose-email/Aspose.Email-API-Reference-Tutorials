## คู่มือการใช้งาน

### การแยกไฟล์ PST ออกเป็นส่วนๆ
หัวข้อนี้จะอธิบายวิธีการแบ่งไฟล์ PST ขนาดใหญ่ให้เป็นส่วนเล็กๆ โดยใช้ Aspose.Email สำหรับ .NET

#### ภาพรวมของคุณสมบัติ
การ `SplitInto` วิธีการนี้จะแบ่งไฟล์ PST หนึ่งไฟล์ออกเป็นส่วนย่อยๆ โดยแต่ละส่วนจะมีขนาดที่กำหนดไว้ ซึ่งมีประโยชน์เมื่อต้องจัดการกับไฟล์ PST ที่มีขนาดใหญ่เกินไปและจัดการได้ยาก

#### ขั้นตอนการดำเนินการ

##### 1. ตั้งค่าเส้นทางและไดเรกทอรี
ระบุไดเร็กทอรีสำหรับไฟล์ PST ต้นทางของคุณและปลายทางสำหรับชิ้นส่วนที่แยก
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY\Sub.pst";
String dstSplit = dataDir + "Chunks\\";
```

##### 2. ล้างไฟล์ที่มีอยู่ในโฟลเดอร์ปลายทาง
หลีกเลี่ยงความขัดแย้งโดยการลบไฟล์ใดๆ ที่มีอยู่ในโฟลเดอร์ปลายทางของคุณ:
```csharp
foreach (string file__1 in Directory.GetFiles(dstSplit))
{
    File.Delete(file__1);
}
```

##### 3. โหลดไฟล์ PST และแยก
โหลดไฟล์ PST ของคุณและแบ่งเป็นส่วนๆ ตามขนาดที่กำหนด เช่น 5MB
```csharp
using (PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir))
{
    // ตัวเลือก: สมัครรับข่าวสารกิจกรรมเพื่อติดตามความคืบหน้า
    personalStorage.StorageProcessed += PstSplit_OnStorageProcessed;
    personalStorage.ItemMoved += PstSplit_OnItemMoved;

    // แบ่งไฟล์ PST ออกเป็นชิ้นขนาด 5MB
    personalStorage.SplitInto(5000000, dataDir + "\\Chunks\\");
}
```

##### คำอธิบายพารามิเตอร์และวิธีการ
- **`FromFile(dataDir)`**:โหลด PST จากเส้นทางที่ระบุ
- **`SplitInto(5000000, destinationPath)`**: แบ่งไฟล์ออกเป็นส่วนๆ โดยแต่ละส่วนจะมีขนาดไม่เกิน 5MB พารามิเตอร์แรกคือขนาดชิ้นส่วนเป็นไบต์

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าคุณมีสิทธิ์เพียงพอสำหรับการอ่านและเขียนไฟล์
- ตรวจสอบว่าเส้นทางที่ระบุมีอยู่และสามารถเข้าถึงได้
- ตรวจสอบหน้าเอกสารของ Aspose เพื่อดูการอัปเดตหรือหมายเหตุเกี่ยวกับความเข้ากันได้หากมีปัญหาเกิดขึ้นกับ `SplitInto` วิธี.

## การประยุกต์ใช้งานจริง

2. **การโยกย้ายข้อมูล**:เมื่อย้ายอีเมลระหว่างระบบ การแยกไฟล์ PST จะช่วยหลีกเลี่ยงปัญหาข้อจำกัดของขนาด
