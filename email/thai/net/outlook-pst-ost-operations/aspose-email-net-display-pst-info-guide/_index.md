## คู่มือการใช้งาน

### แสดงข้อมูลของไฟล์ PST

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการโหลดไฟล์ PST และการแสดงรายละเอียดโฟลเดอร์

#### โหลดไฟล์ PST

ระบุเส้นทางไปยังไฟล์ PST ของคุณและโหลดโดยใช้ `PersonalStorage.FromFile` วิธี:
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";
string dst = dataDir + "/PersonalStorage.pst";

// โหลดไฟล์ PST
PersonalStorage personalStorage = PersonalStorage.FromFile(dst);
```

#### รับโฟลเดอร์ย่อยทั้งหมด

ดึงโฟลเดอร์ย่อยทั้งหมดในโฟลเดอร์รากของไฟล์ PST:
```csharp
FolderInfoCollection folderInfoCollection = personalStorage.RootFolder.GetSubFolders();
```

#### ทำซ้ำและแสดงข้อมูลโฟลเดอร์

ทำซ้ำในแต่ละโฟลเดอร์เพื่อแสดงชื่อ จำนวนรายการทั้งหมด และจำนวนรายการที่ยังไม่ได้อ่าน:
```csharp
foreach (FolderInfo folderInfo in folderInfoCollection)
{
    Console.WriteLine("Folder: " + folderInfo.DisplayName);
    Console.WriteLine("Total items: " + folderInfo.ContentCount);
    Console.WriteLine("Total unread items: " + folderInfo.ContentUnreadCount);
    Console.WriteLine("-----------------------------------");
}
```

**คำอธิบาย:**
- `folderInfo.DisplayName`: ดึงชื่อโฟลเดอร์
- `folderInfo.ContentCount`: ให้จำนวนรายการทั้งหมดภายในโฟลเดอร์
- `folderInfo.ContentUnreadCount`: แสดงจำนวนรายการที่ยังไม่ได้อ่าน

### เคล็ดลับการแก้ไขปัญหา

- **ข้อยกเว้นไม่พบไฟล์**: ให้แน่ใจว่าคุณ `dataDir` ได้รับการตั้งค่าอย่างถูกต้องและชี้ไปยังไฟล์ PST ที่มีอยู่
- **ปัญหาการอนุญาต**: ตรวจสอบให้แน่ใจว่าแอปพลิเคชันของคุณมีสิทธิ์การอ่านสำหรับไดเร็กทอรีที่ระบุ

ฟังก์ชันนี้สามารถนำไปประยุกต์ใช้ในสถานการณ์ต่างๆ ได้ เช่น:

2. **เครื่องมือย้ายข้อมูล:** ประเมินและจัดระเบียบข้อมูลอย่างรวดเร็ว ก่อนที่จะย้ายไปยังระบบใหม่
3. **การตรวจสอบการปฏิบัติตาม:** ตรวจสอบข้อความที่ยังไม่ได้อ่านหรือประเภทเนื้อหาเฉพาะเพื่อจุดประสงค์ในการปฏิบัติตามกฎระเบียบ

## ส่วนคำถามที่พบบ่อย

1. **ไฟล์ PST คืออะไร?** 
   ไฟล์ PST (Personal Storage Table) ถูกใช้โดย Microsoft Outlook เพื่อจัดเก็บอีเมล รายชื่อติดต่อ และรายการอื่นๆ ไว้ในเครื่องคอมพิวเตอร์ของคุณ

2. **ฉันจะติดตั้ง Aspose.Email สำหรับ .NET ได้อย่างไร?**
   ใช้ .NET CLI หรือตัวจัดการแพ็คเกจตามที่แสดงไว้ก่อนหน้านี้ในคู่มือนี้

3. **ฉันสามารถเข้าถึงโฟลเดอร์ย่อยภายในไฟล์ PST โดยใช้ Aspose.Email ได้หรือไม่**
   ใช่ คุณสามารถดึงข้อมูลและโต้ตอบกับโฟลเดอร์ย่อยทั้งหมดภายในไฟล์ PST ได้โดยใช้ `GetSubFolders()` วิธี.

4. **ข้อมูลประเภทใดที่สามารถดึงออกมาจากโฟลเดอร์ PST ได้บ้าง?**
   คุณสามารถดึงข้อมูลรายละเอียดต่างๆ เช่น ชื่อโฟลเดอร์ จำนวนรายการทั้งหมด และจำนวนรายการที่ยังไม่ได้อ่าน

5. **มีข้อจำกัดใด ๆ ในการเข้าถึงไฟล์ PST ขนาดใหญ่หรือไม่**
   ไฟล์ PST ขนาดใหญ่จำเป็นต้องมีการจัดการหน่วยความจำที่มีประสิทธิภาพเพื่อป้องกันปัญหาด้านประสิทธิภาพการทำงาน
