## คู่มือการใช้งาน
ใช้งานฟีเจอร์ในการแยกไฟล์แนบจากไฟล์ PST ด้วยขั้นตอนเหล่านี้:

### ฟีเจอร์การแตกไฟล์แนบ
คุณสมบัตินี้จะช่วยแยกไฟล์แนบที่ไม่ใช่ .msg ออกจากไฟล์ PST โดยอัตโนมัติ

#### ขั้นตอนที่ 1: เปิดไฟล์ PST
เปิดไฟล์ PST ของคุณโดยใช้ `PersonalStorage` ระดับ:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // ตั้งค่าเส้นทางไดเรกทอรีเอกสารของคุณที่นี่

using (PersonalStorage personalstorage = PersonalStorage.FromFile(dataDir + "/Outlook.pst"))
{
    // รหัสสำหรับทำงานกับไฟล์ PST ที่เปิดอยู่
}
```

#### ขั้นตอนที่ 2: เข้าถึงโฟลเดอร์ 'กล่องจดหมาย'
เข้าถึงโฟลเดอร์เฉพาะที่มีอีเมลของคุณ:
```csharp
FolderInfo folder = personalstorage.RootFolder.GetSubFolder("Inbox");
```

#### ขั้นตอนที่ 3: ทำซ้ำผ่านข้อความ
ทำซ้ำผ่านแต่ละข้อความเพื่อแยกสิ่งที่แนบมา:
```csharp
foreach (var messageInfo in folder.EnumerateMessagesEntryId())
{
    MapiAttachmentCollection attachments = personalstorage.ExtractAttachments(messageInfo);
    
    if (attachments.Count != 0)
    {
        foreach (var attachment in attachments)
        {
            // ตรวจสอบชื่อไฟล์ที่ถูกต้องและข้ามไฟล์ .msg
            if (!string.IsNullOrEmpty(attachment.LongFileName) && !attachment.LongFileName.Contains(".msg"))
            {
                // บันทึกสิ่งที่แนบมาที่นี่
            }
        }
    }
}
```

### ตัวเลือกการกำหนดค่าคีย์
- **การข้ามไฟล์ .msg**: ตรวจสอบให้แน่ใจว่าได้ตรวจสอบและข้ามไฟล์แนบข้อความ Microsoft Outlook (.msg)
  
### เคล็ดลับการแก้ไขปัญหา
- **ปัญหาเส้นทางไฟล์**: ตรวจสอบว่าของคุณ `dataDir` เส้นทางถูกต้องและสามารถเข้าถึงได้
- **ข้อผิดพลาดเกี่ยวกับการอนุญาต**: ตรวจสอบให้แน่ใจว่าคุณมีสิทธิ์การอ่านไฟล์ PST

## การประยุกต์ใช้งานจริง
การแยกไฟล์แนบ PST อาจเป็นประโยชน์ในสถานการณ์เช่น:
1. **การโยกย้ายข้อมูล**:การย้ายข้อมูลอีเมล์ไปยังระบบใหม่โดยยังคงรักษาไฟล์แนบไว้
2. **การจัดเก็บถาวร**:การจัดระเบียบอีเมลสำคัญและไฟล์แนบ
3. **การปฏิบัติตามกฎหมาย**:การเก็บรักษาเอกสารที่จำเป็นจากอีเมลตามมาตรฐานทางกฎหมาย

การบูรณาการกับระบบต่างๆ เช่น ซอฟต์แวร์ CRM หรือระบบการจัดการเอกสารสามารถเพิ่มประโยชน์ได้

## การพิจารณาประสิทธิภาพ
เพิ่มประสิทธิภาพการทำงานเมื่อแยกไฟล์แนบ PST โดย:
- **การประมวลผลแบบแบตช์**:จัดการการใช้หน่วยความจำอย่างมีประสิทธิภาพผ่านการดำเนินการแบบแบตช์
- **การประมวลผลแบบขนาน**:ใช้การประมวลผลแบบขนานเพื่อเร่งความเร็วในการสกัดข้อมูล
