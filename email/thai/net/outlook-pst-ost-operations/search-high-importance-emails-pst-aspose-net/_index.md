## คู่มือการใช้งาน
### ค้นหาข้อความที่มีความสำคัญสูงใน PST
#### ภาพรวม
สำรวจวิธีค้นหาข้อความที่ทำเครื่องหมายว่ามีความสำคัญสูงภายในไฟล์ PST ของ Outlook โดยใช้ Aspose.Email ฟีเจอร์นี้มีประโยชน์สำหรับการจัดลำดับความสำคัญของอีเมลอย่างรวดเร็ว

##### ขั้นตอนที่ 1: โหลดไฟล์ PST
ขั้นแรก โหลดไฟล์ PST ที่คุณต้องการแยกอีเมลที่มีความสำคัญสูง:

```csharp
using Aspose.Email.Storage.Pst;

string dataDir = \@"YOUR_DOCUMENT_DIRECTORY";
PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir + "Outlook.pst");
```

##### ขั้นตอนที่ 2: เข้าถึงโฟลเดอร์กล่องจดหมาย
เข้าถึงโฟลเดอร์เฉพาะที่เก็บข้อความของคุณ ที่นี่เราจะเน้นที่กล่องจดหมาย:

```csharp
FolderInfo inboxFolder = personalStorage.RootFolder.GetSubFolder("Inbox");
```

##### ขั้นตอนที่ 3: สร้างแบบสอบถามสำหรับข้อความที่มีความสำคัญสูง
ใช้ประโยชน์ `PersonalStorageQueryBuilder` เพื่อสร้างแบบสอบถามที่กรองอีเมลตามระดับความสำคัญ:

```csharp
PersonalStorageQueryBuilder builder = new PersonalStorageQueryBuilder();
builder.Importance.Equals((int)MapiImportance.High);
MessageInfoCollection highImportanceMessages = inboxFolder.GetContents(builder.GetQuery());
```

ที่นี่เราตั้งค่าตัวกรองความสำคัญเป็น `High`โดยจะดึงเฉพาะข้อความที่ถือว่าสำคัญเท่านั้น

##### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ PST ถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่าโฟลเดอร์กล่องจดหมายมีอยู่ในโครงสร้าง PST ของคุณ
- ตรวจสอบปัญหาที่อาจเกิดขึ้นกับสิทธิ์หรือสิทธิ์ในการเข้าถึง
