## คู่มือการใช้งาน
หัวข้อนี้ครอบคลุมถึงการสร้างงาน MAPI ที่มีคุณสมบัติเฉพาะ และการตั้งค่าการเกิดซ้ำรายปี
### สร้างและบันทึก MapiTask
#### ภาพรวม
การสร้างงานเกี่ยวข้องกับการกำหนดคุณสมบัติ เช่น หัวเรื่อง เนื้อหา วันเริ่มต้น วันครบกำหนด และสถานะ เราจะบันทึกเป็นไฟล์ `.msg` ไฟล์มาตรฐานสำหรับงาน Outlook
#### ขั้นตอนการดำเนินการ
**1. ตั้งค่าไดเรกทอรี**
กำหนดเส้นทางไปยังเอกสารของคุณและไดเร็กทอรีเอาท์พุต:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```
**2. กำหนดค่าโซนเวลา**
ปรับวันที่ตามเขตเวลาท้องถิ่น:
```csharp
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan timeSpan = localZone.GetUtcOffset(DateTime.Now);
DateTime StartDate = new DateTime(2023, 1, 1).Add(timeSpan);
DateTime DueDate = new DateTime(2023, 12, 31).Add(timeSpan);
```
**3. สร้าง MapiTask**
สร้างตัวอย่าง `MapiTask` โดยมีคุณสมบัติตามที่กำหนด:
```csharp
MapiTask task = new MapiTask("Yearly Review Task", "Annual review of project milestones.", StartDate, DueDate);
task.State = MapiTaskState.NotStarted;
```
**4. บันทึกงานเป็นไฟล์ .msg**
บันทึกงานที่สร้างขึ้นไปยังไดเร็กทอรีเอาท์พุต:
```csharp
string filePath = outputDir + "/YearlyReviewTask_out.msg";
task.Save(filePath, TaskSaveFormat.Msg);
```
### ตั้งค่าการเกิดซ้ำรายปีสำหรับ MapiTask
#### ภาพรวม
รูปแบบการเกิดซ้ำมีความสำคัญสำหรับงานที่ทำซ้ำในระยะเวลาหนึ่ง เราจะกำหนดรูปแบบการเกิดซ้ำรายปีไว้ที่นี่
#### ขั้นตอนการดำเนินการ
**1. กำหนดรูปแบบการเกิดซ้ำ**
สร้าง `MapiCalendarYearlyRecurrencePattern`-
```csharp
MapiCalendarYearlyRecurrencePattern rec = new MapiCalendarYearlyRecurrencePattern
{
    DayOfMonth = 15,
    MonthOfYear = MapiMonth.January, // เริ่มต้นในเดือนมกราคม
    Type = MapiCalendarRecurrenceType.Month,
    EndType = MapiCalendarRecurrenceEndType.NeverEnding,
};
```
**2. กำหนดการเกิดซ้ำให้กับงาน**
กำหนดรูปแบบการเกิดซ้ำให้กับงาน:
```csharp
MapiTask task = new MapiTask("Yearly Review Task", "Annual review of project milestones.", DateTime.Now, DateTime.Now.AddDays(1));
task.Recurrence = rec;
```
**3. บันทึกงานที่เกิดขึ้นซ้ำ**
บันทึกงานที่เกิดขึ้นซ้ำในลักษณะเดียวกับงานที่ไม่เกิดขึ้นซ้ำ:
```csharp
string filePath = outputDir + "/YearlyReviewTask_out.msg";
task.Save(filePath, TaskSaveFormat.Msg);
```
### เคล็ดลับการแก้ไขปัญหา
- รับรองเส้นทางใน `dataDir` และ `outputDir` ถูกต้องครับ.
- ตรวจสอบว่า Aspose.Email ได้รับอนุญาตถูกต้องเพื่อหลีกเลี่ยงข้อจำกัด
- ตรวจสอบการตั้งค่าโซนเวลาหากงานปรากฏพร้อมวันที่ไม่ถูกต้อง
