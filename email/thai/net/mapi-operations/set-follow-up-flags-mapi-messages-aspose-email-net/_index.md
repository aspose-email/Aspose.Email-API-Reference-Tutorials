## คู่มือการใช้งาน

มาดูขั้นตอนที่จำเป็นในการตั้งค่าแฟล็กติดตามข้อความ MAPI กัน

### ขั้นตอนที่ 1: เริ่มต้นใช้งาน MailMessage
เริ่มต้นด้วยการสร้าง `MailMessage` วัตถุ ทำหน้าที่เป็นข้อความอีเมลพื้นฐานของคุณซึ่งประกอบด้วยรายละเอียดผู้ส่ง ผู้รับ และเนื้อหา

```csharp
using Aspose.Email.Mapi;
using Aspose.Email.Mime;

// เริ่มต้น MailMessage ด้วยผู้ส่ง ผู้รับ และเนื้อหา
MailMessage mailMsg = new MailMessage();
mailMsg.Sender = "sender@example.com"; // ตั้งค่าที่อยู่ผู้ส่งอีเมล์
mailMsg.To = "recipient@example.com"; // ตั้งค่าที่อยู่อีเมลของผู้รับ
mailMsg.Body = "This message will test if follow-up options can be added to a new MAPI message.";
```

### ขั้นตอนที่ 2: แปลง MailMessage เป็น MapiMessage
เพื่อใช้ประโยชน์จากคุณสมบัติขั้นสูง เช่น การตั้งค่าการติดตามผล แปลงของคุณ `MailMessage` เข้าไปใน `MapiMessage`-

```csharp
// แปลง MailMessage เป็น MapiMessage เพื่อการใช้งานเพิ่มเติมด้วยฟีเจอร์การติดตาม
MapiMessage mapi = MapiMessage.FromMailMessage(mailMsg);
```

### ขั้นตอนที่ 3: กำหนดวันที่ติดตามผล
กำหนดวันที่เกี่ยวข้องกับงานติดตามของคุณ ได้แก่ วันเริ่มต้น วันเตือนความจำ และวันครบกำหนด

```csharp
// กำหนดวันที่เริ่มต้น วันเตือนความจำ และวันครบกำหนดสำหรับตัวเลือกการติดตาม
DateTime dtStartDate = new DateTime(2023, 10, 1, 9, 0, 0); // วันที่เริ่มต้นของรายการการดำเนินการ
DateTime dtReminderDate = new DateTime(2023, 10, 2, 9, 0, 0); // เตือนความจำก่อนวันครบกำหนด
DateTime dtDueDate = dtReminderDate.AddDays(7); // กำหนดส่งงานติดตามผล
```

### ขั้นตอนที่ 4: สร้างตัวเลือกติดตามผล
สร้าง `FollowUpOptions` วัตถุที่มีพารามิเตอร์ที่ระบุเช่นเรื่องและวันที่

```csharp
// สร้าง FollowUpOptions ที่มีหัวเรื่อง วันเริ่มต้น วันครบกำหนด และวันที่เตือนความจำ
FollowUpOptions options = new FollowUpOptions("Project Update", dtStartDate, dtDueDate, dtReminderDate);
```

### ขั้นตอนที่ 5: ใช้ตัวเลือกติดตามผล
ใช้ `FollowUpManager` เพื่อใช้ตัวเลือกเหล่านี้กับข้อความของคุณ

```csharp
// ใช้ตัวเลือกการติดตามกับ MapiMessage โดยใช้ FollowUpManager
FollowUpManager.SetOptions(mapi, options);
```

### ขั้นตอนที่ 6: บันทึกข้อความ
สุดท้าย ให้บันทึกข้อความที่แก้ไขของคุณโดยใช้การตั้งค่าแฟล็กการติดตามผล

```csharp
// บันทึกข้อความที่แก้ไขพร้อมแฟล็กติดตามไปยังไดเร็กทอรีที่ระบุ
mapi.Save(@"YOUR_OUTPUT_DIRECTORY\SetFollowUpFlag_out.msg");
```
