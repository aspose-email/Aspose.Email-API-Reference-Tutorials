# สร้างและบันทึก MAPI Notes ลงในไฟล์ PST โดยใช้ Aspose.Email สำหรับ .NET: คู่มือที่ครอบคลุม

## การแนะนำ

คุณกำลังมองหาวิธีจัดการบันทึกดิจิทัลของคุณอย่างมีประสิทธิภาพโดยการสร้างและบันทึกลงในไฟล์ PST โดยใช้ C# หรือไม่ คู่มือฉบับสมบูรณ์นี้จะแสดงวิธีการใช้ Aspose.Email สำหรับ .NET เพื่อสร้างบันทึก MAPI ตั้งค่าคุณสมบัติ และบันทึกลงในไฟล์ PST ใหม่ ไม่ว่าคุณจะเป็นนักพัฒนาที่มีประสบการณ์หรือเพิ่งเริ่มต้นการเขียนโปรแกรมอีเมล บทช่วยสอนนี้จะแนะนำคุณในทุกขั้นตอน

## คู่มือการใช้งาน
หัวข้อนี้ครอบคลุมขั้นตอนการสร้างและการบันทึกบันทึก MAPI ลงในไฟล์ PST ทีละขั้นตอน

### สร้างและลบไฟล์ PST ที่มีอยู่
เริ่มต้นด้วยการตั้งค่าไดเรกทอรีเอกสารของคุณและจัดการไฟล์ที่มีอยู่:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // ตั้งค่านี้ให้เป็นเส้นทางจริงของคุณ
if (File.Exists(dataDir + "/SampleNote_out.pst"))
{
    File.Delete(dataDir + "/SampleNote_out.pst"); // ลบออกหากมีอยู่เพื่อเริ่มต้นใหม่
}
```

### สร้างไฟล์ PST ใหม่และโฟลเดอร์ที่กำหนดไว้ล่วงหน้า
สร้างไฟล์ PST ใหม่ในรูปแบบ Unicode โดยมีโฟลเดอร์ "บันทึก" ที่กำหนดไว้ล่วงหน้า:
```csharp
using (PersonalStorage pst = PersonalStorage.Create(dataDir + "/SampleNote_out.pst", FileFormatVersion.Unicode))
{
    FolderInfo notesFolder = pst.CreatePredefinedFolder("Notes", StandardIpmFolder.Notes);
```

### โหลดและแปลง MSG เป็น MAPI Note
โหลดไฟล์ MSG ที่มีอยู่และแปลงเป็น `MapiMessage`-
```csharp
MapiMessage message = MapiMessage.FromFile(dataDir + "/Note.msg"); // ตรวจสอบว่าคุณมีไฟล์ MSG นี้
```

### สร้างและปรับแต่งบันทึก
#### หมายเหตุ #1: หมายเหตุสีเหลือง
ตั้งค่าคุณสมบัติเช่น หัวเรื่อง, ข้อความเนื้อหา และสีสำหรับบันทึกแรก
```csharp
// สร้างโน้ต #1 ด้วยสีเหลือง
MapiNote note1 = (MapiNote)message.ToMapiMessageItem();
note1.Subject = "Yellow color note";
note1.Body = "This is a yellow color note";
```

#### หมายเหตุ #2: โน้ตสีชมพู
ปรับแต่งโน้ตที่สองด้วยคุณสมบัติที่แตกต่างกัน
```csharp
// สร้างโน้ต #2 ด้วยสีชมพู
MapiNote note2 = (MapiNote)message.ToMapiMessageItem();
note2.Subject = "Pink color note";
note2.Body = "This is a pink color note";
note2.Color = NoteColor.Pink;
```

#### หมายเหตุ #3: หมายเหตุสีน้ำเงินพร้อมขนาด
เพิ่มมิติให้กับโน้ตที่สามเพื่อการปรับแต่งเพิ่มเติม
```csharp
// สร้างหมายเหตุ #3 ด้วยสีน้ำเงินและขนาดที่เฉพาะเจาะจง
MapiNote note3 = (MapiNote)message.ToMapiMessageItem();
note3.Subject = "Blue color note";
note3.Body = "This is a blue color note";
note3.Color = NoteColor.Blue;
note3.Height = 500; // ความสูงที่กำหนดเอง
note3.Width = 500; // ความกว้างที่กำหนดเอง
```

### บันทึกหมายเหตุลงในไฟล์ PST
เพิ่มบันทึกทั้งหมดที่สร้างไปยังโฟลเดอร์ 'บันทึก' ในไฟล์ PST ใหม่ของคุณ:
```csharp
// เพิ่มหมายเหตุลงในโฟลเดอร์
notesFolder.AddMapiMessageItem(note1);
notesFolder.AddMapiMessageItem(note2);
notesFolder.AddMapiMessageItem(note3);
}
```
