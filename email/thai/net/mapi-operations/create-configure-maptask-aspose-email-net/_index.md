## คู่มือการใช้งาน

### การสร้างและการกำหนดค่า MapiTask

**ภาพรวม:** คุณสมบัตินี้ช่วยให้คุณสร้างงานที่มีคุณสมบัติโดยละเอียด และตั้งค่ารูปแบบการเกิดซ้ำสำหรับการเตือนเป็นระยะๆ

#### ขั้นตอนที่ 1: สร้าง MapiTask ใหม่
เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `MapiTask`-
```csharp
using Aspose.Email.Mapi;

// เริ่มต้นงานใหม่ด้วยชื่อ เนื้อหา วันที่เริ่มต้น และวันที่ครบกำหนด
MapiTask task = new MapiTask("This is test task", "Sample Body", new DateTime(2015, 7, 1), new DateTime(2015, 7, 1));
task.State = MapiTaskState.NotAssigned;
```
**คำอธิบาย:** ที่นี่, `MapiTask` เริ่มต้นด้วยชื่อเรื่องและเนื้อหา โดยกำหนดวันเริ่มต้นและวันครบกำหนดเดิมเป็นวันที่ 1 กรกฎาคม 2558

#### ขั้นตอนที่ 2: กำหนดรูปแบบการเกิดซ้ำรายปี
ถัดไป กำหนดค่าให้งานเกิดขึ้นซ้ำทุกปี:
```csharp
// กำหนดรูปแบบการเกิดซ้ำรายปีเริ่มต้นในวันที่ 15 และเกิดซ้ำทุก ๆ 12 เดือนเป็นเวลา 3 ครั้ง
var rec = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15,
    Period = 12,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    OccurrenceCount = 3,
};

// ตรวจสอบให้แน่ใจว่าจำนวนการเกิดขึ้นมีอย่างน้อย 1 เพื่อหลีกเลี่ยงการกำหนดค่าที่ไม่ถูกต้อง
if (rec.OccurrenceCount == 0)
{
    rec.OccurrenceCount = 1;
}
task.Recurrence = rec;
```
**คำอธิบาย:** บล็อคนี้จะตั้งค่าการเกิดซ้ำรายปีเริ่มตั้งแต่ 15 กรกฎาคม และเกิดขึ้นทุกปีเป็นเวลาสามครั้ง

### การปรับโซนเวลา

**ภาพรวม:** ปรับวันที่งานตามเขตเวลาท้องถิ่นเพื่อให้แน่ใจว่าการกำหนดตารางเวลาในแต่ละภูมิภาคมีความถูกต้องแม่นยำ

#### ขั้นตอนที่ 3: รับการชดเชยโซนเวลาท้องถิ่น
ปรับ `DateTime` วัตถุที่ใช้โซนเวลาท้องถิ่นปัจจุบัน:
```csharp
using System;

// ดึงข้อมูลโซนเวลาปัจจุบันและค่าชดเชย UTC
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);

// ปรับวันที่โดยการเพิ่มค่าชดเชยเขตเวลาท้องถิ่น
DateTime StartDate = new DateTime(2015, 7, 1).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 1).Add(ts);
DateTime endByDate = new DateTime(2020, 12, 31).Add(ts);
```
**คำอธิบาย:** โค้ดนี้จะปรับวันที่เริ่มต้นและครบกำหนดงานเพื่อให้สะท้อนถึงโซนเวลาท้องถิ่น ซึ่งจำเป็นสำหรับแอปพลิเคชันที่ใช้งานในสถานที่ทางภูมิศาสตร์ที่แตกต่างกัน

## การประยุกต์ใช้งานจริง
- **การจัดการโครงการ:** ทำให้งานที่เกิดขึ้นซ้ำเป็นอัตโนมัติสำหรับเหตุการณ์สำคัญของโครงการ
- **ผลผลิตส่วนบุคคล:** ตั้งค่าการแจ้งเตือนสำหรับเป้าหมายส่วนตัวหรือกำหนดเวลาโดยใช้รูปแบบรายปี
- **การจัดตารางการดำเนินธุรกิจ:** บูรณาการกับแอปปฏิทินเพื่อกำหนดตารางการประชุมอัตโนมัติทุกปี

ความเป็นไปได้ในการบูรณาการได้แก่การเชื่อมโยงงานเหล่านี้เข้ากับระบบ CRM และปรับปรุงการแจ้งเตือนอีเมลอัตโนมัติตามการเปลี่ยนแปลงสถานะของงาน
