# วิธีการสร้างรายการบันทึก MAPI พร้อมไฟล์แนบใน C# โดยใช้ Aspose.Email .NET

## การแนะนำ

การจัดการอีเมลอย่างมีประสิทธิภาพถือเป็นสิ่งสำคัญสำหรับธุรกิจและผู้เชี่ยวชาญ การสร้างรายการบันทึกประจำวันที่มีโครงสร้างที่สามารถแชร์ได้บนแพลตฟอร์มต่างๆ พร้อมแนบไฟล์ถือเป็นความท้าทายที่ไม่เหมือนใคร บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการนำ MAPI Journals มาใช้โดยใช้ Aspose.Email .NET เพื่อจัดการงานเหล่านี้อย่างมีประสิทธิภาพ

เราจะครอบคลุม:
- การตั้งค่าเส้นทางไฟล์สำหรับไดเร็กทอรีอินพุตและเอาท์พุต
- การสร้างวัตถุ MapiJournal ด้วยคุณสมบัติที่จำเป็น
- การเพิ่มสิ่งที่แนบมาในรายการ MapiJournal ของคุณ
- บันทึกวารสารเป็นไฟล์ MSG

## คู่มือการใช้งาน

### การตั้งค่าเส้นทางไฟล์
**ภาพรวม**การกำหนดเส้นทางไฟล์จะช่วยให้แอปพลิเคชันของคุณทราบว่าจะอ่านไฟล์อินพุตและบันทึกเอาต์พุตที่ใด
```csharp
using System;

string documentDirectory = @"C:\\Your\\Document\\Directory"; // แทนที่ด้วยเส้นทางจริง
string outputDirectory = @"C:\\Your\\Output\\Directory";      // แทนที่ด้วยเส้นทางจริง
```
### การสร้างวัตถุ MapiJournal
**ภาพรวม**:วัตถุ MapiJournal เป็นศูนย์กลางในการสร้างรายการวารสารที่มีโครงสร้าง
```csharp
using Aspose.Email.Mapi;

MapiJournal journal = new MapiJournal(
    subject: "testJournal",
    body: "This is a test journal",
    categories: "Phone call",
    importance: MapiMessageImportance.Normal);

// ตั้งค่าเวลาเริ่มต้นและสิ้นสุดสำหรับรายการบันทึกรายวัน
journal.StartTime = DateTime.Now;
journal.EndTime = journal.StartTime.AddHours(1);
// เพิ่มบริษัทที่มีส่วนเกี่ยวข้องในรายการสมุดรายวันนี้
timeJournal.Companies = new string[] { "company 1", "company 2", "company 3" };
```
### การเพิ่มสิ่งที่แนบมากับ MapiJournal
**ภาพรวม**:ปรับปรุงรายการวารสารของคุณโดยแนบไฟล์ที่เกี่ยวข้อง
```csharp
using System.IO;

string[] attachFileNames = new string[] {
    documentDirectory + "\\Desert.jpg",
    documentDirectory + "\\download.png"
};

foreach (string attach in attachFileNames)
{
    // แนบไฟล์ไปที่ MapiJournal โดยใช้ไบต์ไฟล์
    journal.Attachments.Add(attach, File.ReadAllBytes(attach));
}
```
### การบันทึกวัตถุ MapiJournal
**ภาพรวม**สุดท้าย ให้บันทึกวัตถุ MapiJournal ที่คุณกำหนดค่าเป็นไฟล์ MSG
```csharp
journal.Save(outputDirectory + "\\AddAttachmentsToMapiJournal_out.msg");
```
## การประยุกต์ใช้งานจริง
1. **บันทึกการสนับสนุนลูกค้า**:สร้างบันทึกการโทรสนับสนุนแบบอัตโนมัติพร้อมไฟล์แนบเพื่อใช้ในการอ้างอิงในอนาคต
2. **สรุปการประชุม**:จัดทำและแจกจ่ายวารสารการประชุม รวมถึงเอกสารวาระการประชุม
3. **การจัดการโครงการ**:ติดตามความคืบหน้าของโครงการผ่านรายการบันทึกที่แนบมากับรายงาน
4. **แคมเปญการตลาด**:บันทึกกลยุทธ์และผลลัพธ์ของแคมเปญในรูปแบบวารสารรวมศูนย์
