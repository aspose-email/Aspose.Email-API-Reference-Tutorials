## คู่มือการใช้งาน

### การสร้างและการบันทึกงาน MAPI
เริ่มต้นด้วยการสร้างงาน MAPI ง่ายๆ และบันทึกเป็นไฟล์ MSG ฟังก์ชันนี้จะช่วยให้สร้างงานโดยอัตโนมัติ เพื่อให้แน่ใจว่ามีความสอดคล้องและมีประสิทธิภาพ

**ขั้นตอนที่ 1: กำหนดคุณสมบัติของงาน**
เริ่มต้นด้วยการกำหนดวันที่เริ่มต้นและครบกำหนดสำหรับงานของคุณ:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);
DateTime StartDate = new DateTime(2015, 7, 1).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 1).Add(ts);
```

**ขั้นตอนที่ 2: สร้างงาน MAPI**
เริ่มต้น `MapiTask` ด้วยคุณสมบัติที่คุณกำหนด:
```csharp
MapiTask task = new MapiTask("This is test task", "Sample Body", StartDate, DueDate);
task.State = MapiTaskState.NotAssigned;
```
ในข้อความสั้นๆ นี้:
- “นี่คืองานทดสอบ” และ “เนื้อหาตัวอย่าง” คือหัวเรื่องและเนื้อหาของงาน
- `StartDate` และ `DueDate` ระบุเวลาเริ่มต้นและสิ้นสุดงาน

**ขั้นตอนที่ 3: บันทึกงาน**
บันทึกงานของคุณในรูปแบบ MSG:
```csharp
task.Save(dataDir + "Monthly_out.msg", TaskSaveFormat.Msg);
```

### การกำหนดค่ารูปแบบการเกิดซ้ำรายเดือนสำหรับงาน MAPI
ขั้นตอนต่อไปคือการตั้งค่ารูปแบบการเกิดซ้ำรายเดือนบนวัตถุงาน MAPI ที่มีอยู่ ซึ่งเหมาะอย่างยิ่งสำหรับงานที่ต้องเกิดขึ้นซ้ำเป็นระยะๆ

**ขั้นตอนที่ 1: กำหนดรูปแบบการเกิดซ้ำ**
สร้าง `MapiCalendarMonthlyRecurrencePattern`-
```csharp
var rec = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15,
    Period = 12,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    OccurrenceCount = 3,
    WeekStartDay = DayOfWeek.Monday
};
```
การกำหนดค่านี้จะตั้งค่าให้การทำงานเกิดขึ้นซ้ำในวันที่ 15 ของทุกเดือน โดยทำซ้ำสามครั้งในช่วงเวลา 12 เดือน

**ขั้นตอนที่ 2: ใช้การเกิดซ้ำกับงาน**
กำหนดรูปแบบการเกิดซ้ำให้กับคุณ `MapiTask`-
```csharp
MapiTask taskWithRecurrence = new MapiTask("This is test task", "Sample Body", DateTime.Now, DateTime.Now.AddDays(30));
taskWithRecurrence.Recurrence = rec;
```

**ขั้นตอนที่ 3: บันทึกงานด้วยการเกิดซ้ำ**
บันทึกงานประจำของคุณเป็นไฟล์ MSG:
```csharp
string dataDirOutput = "YOUR_OUTPUT_DIRECTORY";
taskWithRecurrence.Save(dataDirOutput + "Monthly_out_with_recurrence.msg", TaskSaveFormat.Msg);
```

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่ารูปแบบวันที่และเวลาทั้งหมดได้รับการตั้งค่าอย่างถูกต้องเพื่อหลีกเลี่ยงข้อผิดพลาด
- ตรวจสอบว่าเส้นทางไดเร็กทอรีมีอยู่ก่อนบันทึกไฟล์

## การประยุกต์ใช้งานจริง
1. **การจัดการโครงการ:** กำหนดงานอัตโนมัติสำหรับเหตุการณ์สำคัญที่เกิดขึ้นซ้ำของโครงการ
2. **รอบการเรียกเก็บเงิน:** กำหนดเวลาการเรียกเก็บเงินรายเดือนโดยไม่ต้องดำเนินการด้วยตนเอง
3. **ตารางการบำรุงรักษา:** ตั้งค่าการแจ้งเตือนการบำรุงรักษาสำหรับการอัปเดตอุปกรณ์หรือซอฟต์แวร์
4. **การวางแผนกิจกรรม:** จัดการงานการวางแผนกิจกรรมที่เกิดขึ้นซ้ำทุกปีหรือทุก ๆ สองปี

ความเป็นไปได้ในการบูรณาการได้แก่การซิงค์กับแอปพลิเคชันปฏิทิน เช่น Microsoft Outlook หรือ Google Calendar ซึ่งจะช่วยปรับปรุงเวิร์กโฟลว์การจัดการงาน
