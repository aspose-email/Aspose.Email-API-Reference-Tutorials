# Extrahování příloh e‑mailů pomocí Aspose.Email pro Java

V tomto hubu objevíte vše, co potřebujete k **extrahování příloh e‑mailů** z nejběžnějších formátů pošty pomocí Aspose.Email pro Java. Ať už vytváříte službu zpracování pošty, archivujete data Outlooku, nebo jen potřebujete vytáhnout soubory z MSG, EML nebo PST zpráv, tyto průvodce krok za krokem vám ukážou, jak to udělat rychle a spolehlivě.

## Rychlé odpovědi
- **Jaký je nejjednodušší způsob, jak extrahovat přílohy z PST souboru?** Použijte `PersonalStorage` k otevření PST a iterujte přes objekty `Message`, voláním `Message.getAttachments()`.  
- **Mohu extrahovat vložené (embedded) obrázky jako samostatné soubory?** Ano – zacházejte s nimi jako s běžnými přílohami; Aspose.Email je zpřístupňuje prostřednictvím stejného API.  
- **Potřebuji licenci pro spuštění příkladů?** Dočasná licence funguje pro vývoj; pro produkci je vyžadována plná licence.  
- **Jaké formáty jsou podporovány pro extrahování příloh?** MSG, EML, EMLX, MHTML a PST soubory jsou plně podporovány.  
- **Existuje způsob, jak automaticky uložit extrahované soubory?** Samozřejmě – zavolejte `Attachment.save(filePath)` uvnitř smyčky k zápisu každé přílohy na disk.

## Co znamená „extrahování příloh e‑mailů“?
Extrahování příloh e‑mailů znamená programově načíst e‑mailovou zprávu (nebo soubor poštovní schránky) a vytáhnout všechny soubory, které byly připojeny, jako jsou dokumenty, obrázky nebo vložené objekty, aby mohly být uloženy, zpracovány nebo přeposlány jinam.

## Proč použít Aspose.Email pro Java k extrahování příloh e‑mailů?
- **Kompletní podpora formátů** – Pracuje s MSG, EML, PST a dalšími, aniž by bylo nutné mít nainstalovaný Outlook.  
- **Žádná COM interop** – Čisté Java API, ideální pro multiplatformní servery.  
- **Vysoký výkon** – Zpracování založené na streamech vám umožní efektivně pracovat s velkými poštovními schránkami.  
- **Bohaté zacházení s přílohami** – Podporuje běžné, vložené a TNEF‑kódované přílohy přímo z krabice.

## Předpoklady
- Java 8 nebo vyšší.  
- Knihovna Aspose.Email pro Java (stáhněte z oficiálního webu).  
- Dočasná nebo plná licence Aspose pro produkční použití.

## Často kladené otázky

**Q: Jak extrahuji přílohy e‑mailů z jedné MSG souboru?**  
A: Načtěte soubor pomocí `MailMessage.load("file.msg")` a zavolejte `mailMessage.getAttachments()`; poté iterujte a uložte každou přílohu.

**Q: Mohu extrahovat přílohy z šifrovaných nebo chráněných heslem PST souborů?**  
A: Ano. Zadejte heslo při otevírání instance `PersonalStorage`: `PersonalStorage.fromFile("file.pst", password)`.

**Q: Jaký je rozdíl mezi běžnými a vloženými (inline) přílohami?**  
A: Běžné přílohy jsou samostatné soubory, zatímco vložené přílohy jsou zabudovány v těle e‑mailu (často obrázky). Aspose.Email zachází s oběma jako s objekty `Attachment`, což vám umožňuje s nimi pracovat jednotně.

**Q: Existuje limit velikosti příloh, které mohu extrahovat?**  
A: Knihovna streamuje data, takže jste omezeni pouze dostupnou pamětí a úložným prostorem, nikoli velikostí přílohy.

**Q: Musím po uložení příloh ručně zavírat streamy?**  
A: Při použití `Attachment.save()` knihovna automaticky spravuje uvolnění streamu, ale pokud otevřete vlastní streamy, nezapomeňte je zavřít, aby nedošlo k únikům.
