## Často kladené otázky
**Q: Mohu použít Aspose.Email na cloudové platformě jako AWS nebo Azure?**  
A: Rozhodně. Knihovna funguje na jakémkoli Java runtime, včetně cloudových prostředí.

**Q: Co když můj poskytovatel SMTP vyžaduje autentizaci OAuth2?**  
A: Aspose.Email podporuje získání OAuth2 tokenu; token můžete předat `SmtpClient` pro autentizaci.

**Q: Jak mohu otestovat svou konfiguraci lokálně, aniž bych posílal skutečné e‑maily?**  
A: Použijte lokální nástroj pro testování SMTP, jako je MailHog nebo Papercut; nastavte hostitele a port tak, aby ukazovaly na tento nástroj.

**Q: Existuje způsob, jak zaznamenat surovou SMTP konverzaci pro ladění?**  
A: Ano—povolte `SmtpClient.setEnableSsl(true)` a nastavte `SmtpClient.setLogEnabled(true)`, aby se zachytily podrobné logy.

**Q: Podporuje Aspose.Email odesílání příloh větších než 25 MB?**  
A: Samotná knihovna neklade žádné omezení velikosti; nicméně musíte respektovat limity svého poskytovatele SMTP.
