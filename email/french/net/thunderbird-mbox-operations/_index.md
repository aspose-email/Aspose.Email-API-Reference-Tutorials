# Tutoriels sur les opérations de fichiers Thunderbird et MBOX pour Aspose.Email Java

Nos tutoriels Thunderbird et MBOX fournissent des conseils complets pour gérer les formats d'e-mail open source avec Aspose.Email en Java. Ces guides détaillés expliquent comment accéder aux banques de courrier Thunderbird, traiter les fichiers MBOX, extraire les messages des archives et utiliser les formats d'e-mail open source par programmation. Chaque tutoriel inclut des exemples de code Java fonctionnels pour les opérations courantes, vous aidant à créer des applications s'intégrant parfaitement à Thunderbird et traitant les collections de fichiers MBOX standard tout en préservant l'intégrité du contenu des e-mails.

## Tutoriels disponibles

### [Comment lire le nombre total de messages d'un fichier MBOX avec Aspose.Email pour .NET](./aspose-email-net-read-mbox-messages-count/)
Apprenez à compter efficacement le nombre total d'e-mails dans un fichier MBOX avec Aspose.Email pour .NET. Idéal pour la migration de données et la validation des sauvegardes.

Apprenez à lire et exporter efficacement des fichiers TGZ avec Aspose.Email pour .NET grâce à ce guide étape par étape. Idéal pour gérer les archives d'e-mails et les formats compressés.

### [Comment lire et enregistrer des e-mails Thunderbird au format EML/MSG avec Aspose.Email .NET](./aspose-email-net-thunderbird-eml-msg-conversion/)
Découvrez comment extraire efficacement les e-mails d'un fichier Thunderbird MBOX à l'aide d'Aspose.Email pour .NET, en les convertissant aux formats EML ou MSG.
