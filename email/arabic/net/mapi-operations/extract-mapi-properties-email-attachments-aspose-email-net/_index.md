## دليل التنفيذ

### استخراج خاصية MAPI المسماة من مرفق البريد الإلكتروني

توضح هذه الميزة كيفية قراءة الخصائص المخصصة المضمنة في المرفقات باستخدام Aspose.Email لـ .NET.

#### تحميل وتحويل رسالة البريد الإلكتروني

ابدأ بتحميل رسالة البريد الإلكتروني الخاصة بك:

```csharp
// قم بتحديد المسار الذي سيتم تخزين ملفات البريد الإلكتروني الخاص بك فيه
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// تحميل البريد الإلكتروني من ملف
MailMessage mail = MailMessage.Load(dataDir + "outputAttachments.msg");

// تحويل MailMessage إلى MapiMessage للوصول إلى الممتلكات
MapiMessage mapi = MapiMessage.FromMailMessage(mail);
```

#### تكرار واستخراج الخاصية

بعد ذلك، قم بالتكرار خلال الخصائص المسماة للمرفق الأول:

```csharp
foreach (MapiNamedProperty namedProperty in mapi.Attachments[0].NamedProperties.Values)
{
    // تحقق مما إذا كان اسم الخاصية يتطابق مع "CustomAttGuid"
    if (string.Compare(namedProperty.NameId, "CustomAttGuid", StringComparison.OrdinalIgnoreCase) == 0)
    {
        // إرجاع التمثيل النصي للخاصية المسماة
        Console.WriteLine("Extracted Property: " + namedProperty.GetString());
        break;
    }
}
```

- **حدود**: `MailMessage.Load()` يتطلب مسار الملف. 
- **قيم الإرجاع**:الطريقة `GetString()` إرجاع قيمة الخاصية المسماة كسلسلة.

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن رسالة البريد الإلكتروني تحتوي على مرفقات تحتوي على خصائص مسماة.
- تأكد من كتابة "CustomAttGuid" بشكل صحيح واستخدام مقارنة غير حساسة لحالة الأحرف.
