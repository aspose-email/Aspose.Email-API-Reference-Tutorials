## دليل التنفيذ

يرشدك هذا القسم خلال عملية إنشاء ملف MSG الخاص بـ Outlook وحفظه باستخدام Aspose.Email لـ .NET.

### إنشاء رسالة بريد إلكتروني جديدة

ابدأ بإنشاء مثيل لـ `MailMessage` الفئة، التي تسمح لك بتعيين خصائص مثل المرسل والمستقبل والموضوع ومحتوى النص.

#### الخطوة 1: تحديد الدلائل
حدد المكان الذي سيتم فيه تخزين المستندات وملفات الإخراج الخاصة بك:
```csharp
private static string dataDir = @"YOUR_DOCUMENT_DIRECTORY/";
private static string dst = dataDir + "message.msg";
```

#### الخطوة 2: كتابة رسالة البريد الإلكتروني
إنشاء `MailMessage` المثيل وتعيين خصائصه:
```csharp
// قم بإنشاء مثيل لفئة MailMessage لإنشاء رسالة بريد إلكتروني جديدة.
MailMessage mailMsg = new MailMessage();

// قم بتعيين حقل "من" باستخدام عنوان البريد الإلكتروني للمرسل.
mailMsg.From = "from@domain.com";

// أضف المستلمين في حقل "إلى" في الرسالة.
mailMsg.To.Add("to@domain.com");

// قم بتحديد سطر الموضوع لرسالة البريد الإلكتروني.
mailMsg.Subject = "creating an outlook message file";

// تعيين محتوى نص رسالة البريد الإلكتروني.
mailMsg.Body = "This message is created by Aspose.Email";
```
هنا، قمنا بتعيين الحقول الأساسية مثل `From`، `To`، `Subject`، و `Body` لتأليف رسالتنا.

### تحويل ملف MSG وحفظه
بعد ذلك، قم بتحويل `MailMessage` إلى `MapiMessage` كائن للحفظ بتنسيق MSG.

#### الخطوة 3: التحويل والحفظ
تحويل `MailMessage` ل `MapiMessage`ثم احفظه:
```csharp
// تحويل MailMessage إلى MapiMessage، وهو أمر مطلوب لحفظه بصيغة .msg.
MapiMessage outlookMsg = MapiMessage.FromMailMessage(mailMsg);

// احفظ الرسالة المحولة في ملف MSG في مسار الوجهة المحدد.
outlookMsg.Save(dst);
```
هذه الخطوة حاسمة لأن `MapiMessage` يدعم تنسيق MSG بشكل أصلي.
