## دليل التنفيذ

### تحميل رسائل MAPI وتحويلها إلى التقويم

#### ملخص
ترتكز هذه الميزة على قراءة ملف رسالة MAPI وتحويله إلى `MapiCalendar` كائن لتسهيل التعامل مع أحداث التقويم برمجيًا.

#### التنفيذ خطوة بخطوة
**1. تحديد مسار الملف**
قم بإعداد المسار الذي سيتم تخزين ملف رسالة MAPI الخاص بك فيه:
```csharp
using Aspose.Email.Mapi;

namespace EmailProcessing
{
    public class LoadAndConvertMapiMessage
    {
        private static string dataDir = "YOUR_DOCUMENT_DIRECTORY";

        public void Process()
        {
            // تحديد المسار الكامل لملف رسالة MAPI
            string fileName = dataDir + "/Test Meeting.msg";
```
**2. قم بتحميل رسالة MAPI**
يستخدم `MapiMessage.FromFile()` لتحميل رسالتك في `MapiMessage` هدف:
```csharp
// قم بتحميل MapiMessage من الملف المحدد
MapiMessage message = MapiMessage.FromFile(fileName);
```
**3. التحويل إلى MapiCalendar**
تحويل الرسالة المحملة إلى `MapiCalendar` كائن للتلاعب بسهولة بخصائص التقويم:
```csharp
// تحويل الرسالة المحملة وإرسالها إلى كائن MapiCalendar
MapiCalendar calendar = (MapiCalendar)message.ToMapiMessageItem();
        }
    }
}
```
### إنشاء وتعيين مناطق زمنية للتقويم

#### ملخص
تتيح لك هذه الميزة إنشاء `MapiCalendarTimeZone` استخدام المنطقة الزمنية للنظام المحلي الخاص بك وتعيينها لأحداث التقويم للحصول على توقيت دقيق للحدث.

#### التنفيذ خطوة بخطوة
**1. إنشاء MapiCalendarTimeZone**
إنشاء مثيل جديد `MapiCalendarTimeZone` الكائن مع المنطقة الزمنية للنظام الحالي:
```csharp
using Aspose.Email.Mapi;
using System;

namespace EmailProcessing
{
    public class CreateAndAssignCalendarTimeZones
    {
        private MapiCalendar calendar; 

        public void ConfigureTimeZone()
        {
            // إنشاء MapiCalendarTimeZone جديد باستخدام معلومات المنطقة الزمنية للنظام المحلي
            MapiCalendarTimeZone timeZone = new MapiCalendarTimeZone(TimeZoneInfo.Local);
```
**2. تعيين إلى بداية ونهاية التقويم**
قم بتعيين كائن المنطقة الزمنية هذا لكل من أوقات البداية والنهاية لحدث التقويم الخاص بك:
```csharp
// تعيين تاريخ بداية ونهاية التقويم/المناطق الزمنية
calendar.StartDateTimeZone = timeZone;
calendar.EndDateTimeZone = timeZone;
        }
    }
}
```
