### تحميل جهات الاتصال من ملف MSG

#### ملخص
توضح هذه الميزة كيفية استخراج تفاصيل جهة الاتصال المخزنة في ملف MSG الخاص بـ Outlook عن طريق تحويلها إلى كائن MapiContact.

#### خطوات التنفيذ
**الخطوة 1: تحديد المسار إلى دليل المستندات الخاص بك**
قبل تحميل أي ملفات، حدد مكان وجود ملفات MSG الخاصة بك:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
يستبدل `"YOUR_DOCUMENT_DIRECTORY"` مع المسار الفعلي لملفاتك.

**الخطوة 2: تحميل ملف MSG إلى كائن MapiMessage**
لقراءة ملف MSG، استخدم `MapiMessage.FromFile` تتضمن هذه الخطوة تحديد مسار الملف وتحميله:
```csharp
// قم بتحميل ملف MSG إلى كائن MapiMessage.
MapiMessage msg = MapiMessage.FromFile(dataDir + "/Contact.msg");
```
**الخطوة 3: تحويل MapiMessage إلى MapiContact**
بمجرد تحميل ملف MSG، قم بتحويله إلى `MapiContact` كائن لتسهيل الوصول إلى تفاصيل الاتصال:
```csharp
// تحويل MapiMessage إلى كائن MapiContact.
MapiContact mapiContact = (MapiContact)msg.ToMapiMessageItem();
```
