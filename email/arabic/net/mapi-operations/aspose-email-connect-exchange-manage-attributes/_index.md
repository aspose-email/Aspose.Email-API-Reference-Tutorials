## دليل التنفيذ

### الاتصال بخادم Exchange
تتيح لك هذه الميزة الاتصال بخادم Exchange باستخدام EWS (خدمات الويب Exchange).

#### الخطوة 1: إعداد بيانات اعتماد الشبكة
قم بتحديد بيانات اعتماد الشبكة المطلوبة للاتصال.
```csharp
string mailboxURI = "https://ex2010/ews/exchange.asmx";
string username = "test.exchange";
string password = "pwd";
string domain = "ex2010.local";

NetworkCredential credential = new NetworkCredential(username, password, domain);
```

#### الخطوة 2: إنشاء اتصال باستخدام EWSClient
استخدم بيانات الاعتماد للاتصال بخادم Exchange الخاص بك.
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxURI, credential);
```

### العمل مع السمات الموسعة للرسائل
توضح هذه الميزة كيفية إدارة الخصائص المخصصة في رسائل البريد الإلكتروني المخزنة على خادم Exchange.

#### الخطوة 1: إنشاء واصف خاصية مخصص
قم بتحديد موصوف الخاصية لسماتك المخصصة:
```csharp
using Aspose.Email.Mapi;

PidNamePropertyDescriptor pd = new PidNamePropertyDescriptor(
    "MyTestProp",
    PropertyDataType.String,
    KnownPropertySets.PublicStrings);

string value = "MyTestPropValue";
```

#### الخطوة 2: إنشاء رسالة مخصصة وتعيينها
إنشاء رسالة بريد إلكتروني باستخدام خصائص مخصصة:
```csharp
MapiMessage message = new MapiMessage(
    "from@domain.com",
    "to@domain.com",
    "EMAILNET-38844 - " + Guid.NewGuid().ToString(),
    "EMAILNET-38844 EWS: Support for create, retrieve and update Extended Attributes for Emails");

message.SetProperty(pd, value);
```

#### الخطوة 3: إضافة الرسالة إلى Exchange Server
إرسال رسالتك المخصصة إلى الخادم:
```csharp
string uri = client.AppendMessage(message);
```

#### الخطوة 4: استرداد الخاصية المخصصة
جلب الرسالة باستخدام موصوف الخاصية واسترداد السمة المخصصة لها:
```csharp
MapiMessage mapiMessage = client.FetchItem(uri, new PropertyDescriptor[] { pd });
string fetchedValue = mapiMessage.NamedProperties[pd].GetString();
```

## قسم الأسئلة الشائعة
**1. كيف يمكنني التعامل مع خصائص مخصصة متعددة؟**
يمكنك تعريف متعددة `PidNamePropertyDescriptor` الحالات وإدارتها بشكل فردي داخل رسالة.

**2. ماذا لو كانت بيانات اعتماد الشبكة الخاصة بي لا تعمل؟**
تأكد من أن اسم المستخدم وكلمة المرور والنطاق يتطابقون مع تلك التي تم تكوينها على خادم Exchange الخاص بك.

**3. هل يمكنني استخدام هذا مع خوادم البريد الإلكتروني الأخرى بالإضافة إلى Exchange؟**

**4. كيف يمكنني التأكد من أن خصائصي المخصصة فريدة؟**
استخدم أسماء مميزة وضعها في المكان المناسب `KnownPropertySets`.
