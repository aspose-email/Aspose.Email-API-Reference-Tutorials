## دليل التنفيذ

في هذا القسم، سنستعرض كيفية إنشاء جهات اتصال MAPI وحفظها باستخدام Aspose.Email لـ .NET.

### الميزة: إنشاء جهة اتصال MAPI وتعبئتها

#### ملخص

توضح هذه الميزة كيفية إنشاء مثيل لـ `MapiContact` واملأه بتفاصيل الاتصال الأساسية مثل الاسم والمهنة والعناوين والبريد الإلكتروني وأرقام الهواتف والفئات والمزيد.

#### التنفيذ خطوة بخطوة

##### تهيئة دليل الإخراج

أولاً، قم بتحديد المكان الذي ستحفظ فيه ملفاتك:

```csharp
string dataDir = "YOUR_OUTPUT_DIRECTORY";
```

##### إنشاء كائن MapiContact جديد

ابدأ بتهيئة ملف جديد `MapiContact` هدف:

```csharp
MapiContact contact = new MapiContact();
```

##### تعيين المعلومات الأساسية

املأ التفاصيل الأساسية مثل الاسم والمهنة:

```csharp
contact.NameInfo = new MapiContactNamePropertySet("Bertha", "A.", "Buell");
contact.ProfessionalInfo = new MapiContactProfessionalPropertySet("Awthentikz", "Social work assistant");
```

##### إضافة معلومات الاتصال

قم بتضمين معلومات اتصال إضافية مثل العناوين الفعلية ورسائل البريد الإلكتروني وأرقام الهواتف:

```csharp
// العنوان الفعلي للعمل
contact.PhysicalAddresses.WorkAddress.Address = "Im Astenfeld 59 8580 EDELSCHROTT";

// بريد إلكتروني
contact.ElectronicAddresses.Email1 = new MapiContactElectronicAddress("Experwas", "SMTP", "BerthaABuell@armyspy.com");

// رقم الهاتف
contact.Telephones = new MapiContactTelephonePropertySet("06605045265");
```

##### إضافة تفاصيل إضافية

يمكنك أيضًا إضافة معلومات متنوعة وحقول محددة من قبل المستخدم:

```csharp
// معلومات متنوعة
contact.Mileage = "Some test mileage";
contact.Billing = "Test billing information";

// الحقول المحددة من قبل المستخدم
contact.OtherFields.Journal = true;
contact.OtherFields.Private = true;
contact.OtherFields.ReminderTime = new DateTime(2014, 1, 1, 0, 0, 55);
contact.OtherFields.UserField1 = "ContactUserField1";
```

##### تحميل صورة

قم بتحميل صورة إلى حقل جهة الاتصال للصورة:

```csharp
using (FileStream fs = File.OpenRead("YOUR_DOCUMENT_DIRECTORY/Desert.jpg"))
{
    byte[] buffer = new byte[fs.Length];
    fs.Read(buffer, 0, buffer.Length);
    contact.Photo = new MapiContactPhoto(buffer, MapiContactPhotoImageFormat.Jpeg);
}
```

### الميزة: حفظ جهة اتصال MAPI بتنسيقات مختلفة

#### ملخص

بمجرد ملء بياناتك `MapiContact` إذا كنت ترغب في حفظ كائن بالمعلومات المطلوبة، فقد حان الوقت لحفظه بتنسيقات مختلفة مثل MSG وVCF.

#### التنفيذ خطوة بخطوة

##### حفظ بتنسيق MSG

```csharp
contact.Save(dataDir + "/MapiContact_out.msg", ContactSaveFormat.Msg);
```

##### حفظ بتنسيق VCF

```csharp
contact.Save(dataDir + "/MapiContact_out.vcf", ContactSaveFormat.VCard);
```
