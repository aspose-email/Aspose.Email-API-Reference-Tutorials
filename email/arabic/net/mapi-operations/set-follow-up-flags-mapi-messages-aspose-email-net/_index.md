## دليل التنفيذ

دعونا نقوم بتفصيل الخطوات اللازمة لتعيين علامات المتابعة على رسائل MAPI.

### الخطوة 1: تهيئة MailMessage
ابدأ بإنشاء `MailMessage` هذا هو الكائن. يعمل هذا كرسالة بريد إلكتروني أساسية تحتوي على تفاصيل المرسل والمستلم والنص.

```csharp
using Aspose.Email.Mapi;
using Aspose.Email.Mime;

// قم بتهيئة MailMessage بالمرسل والمستقبل والنص.
MailMessage mailMsg = new MailMessage();
mailMsg.Sender = "sender@example.com"; // تعيين عنوان مرسل البريد الإلكتروني.
mailMsg.To = "recipient@example.com"; // تعيين عنوان البريد الإلكتروني للمستلم.
mailMsg.Body = "This message will test if follow-up options can be added to a new MAPI message.";
```

### الخطوة 2: تحويل MailMessage إلى MapiMessage
للاستفادة من الميزات المتقدمة مثل إعداد عمليات المتابعة، قم بتحويل `MailMessage` إلى `MapiMessage`.

```csharp
// قم بتحويل MailMessage إلى MapiMessage لمزيد من المعالجة باستخدام ميزات المتابعة.
MapiMessage mapi = MapiMessage.FromMailMessage(mailMsg);
```

### الخطوة 3: تحديد تواريخ المتابعة
قم بتحديد التواريخ ذات الصلة بمهمة المتابعة الخاصة بك، بما في ذلك تاريخ البدء وتاريخ التذكير وتاريخ الاستحقاق.

```csharp
// قم بتحديد تاريخ البدء وتاريخ التذكير وتاريخ الاستحقاق لخيارات المتابعة.
DateTime dtStartDate = new DateTime(2023, 10, 1, 9, 0, 0); // تاريخ بدء تنفيذ بند العمل.
DateTime dtReminderDate = new DateTime(2023, 10, 2, 9, 0, 0); // تنبيه تذكيري قبل تاريخ الاستحقاق.
DateTime dtDueDate = dtReminderDate.AddDays(7); // تاريخ استحقاق المهمة المتابعة.
```

### الخطوة 4: إنشاء خيارات المتابعة
إنشاء `FollowUpOptions` كائن بمعلمات محددة مثل الموضوع والتاريخ.

```csharp
// قم بإنشاء FollowUpOptions مع موضوع وتاريخ البدء وتاريخ الاستحقاق وتاريخ التذكير.
FollowUpOptions options = new FollowUpOptions("Project Update", dtStartDate, dtDueDate, dtReminderDate);
```

### الخطوة 5: تطبيق خيارات المتابعة
استخدم `FollowUpManager` لتطبيق هذه الخيارات على رسالتك.

```csharp
// قم بتطبيق خيارات المتابعة على MapiMessage باستخدام FollowUpManager.
FollowUpManager.SetOptions(mapi, options);
```

### الخطوة 6: حفظ الرسالة
وأخيرًا، احفظ رسالتك المعدلة مع تطبيق علامات المتابعة عليها.

```csharp
// احفظ الرسالة المعدلة مع علامات المتابعة في الدليل المحدد.
mapi.Save(@"YOUR_OUTPUT_DIRECTORY\SetFollowUpFlag_out.msg");
```
