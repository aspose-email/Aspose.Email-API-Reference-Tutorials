## دليل التنفيذ

### الميزة 1: تحميل رسائل MAPI وحفظها مع المرفقات

توضح هذه الميزة كيفية تحميل رسالة MAPI من ملف وحفظها مع المرفقات باستخدام Aspose.Email لـ .NET.

#### ملخص
الغرض من هذه الميزة هو إدارة رسائل MAPI عن طريق تحميلها في تطبيقك، وحفظها عند الحاجة، والتأكد من سلامة جميع المرفقات.

#### الخطوة 1: تحميل رسالة MAPI من الملف
```csharp
using Aspose.Email.Mapi;
using System;

class FeatureLoadAndSaveMAPI
{
    public static void Run()
    {
        // قم بتحديد مسار الدليل الذي يوجد به ملف الإدخال
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // قم بتحديث هذا باستخدام دليل المستندات الفعلي الخاص بك

        // تحميل رسالة MAPI من الملف.
        MapiMessage msg = MapiMessage.FromFile(dataDir + "/MsgWithAtt.msg");
        
        Console.WriteLine("MAPI message loaded successfully.");
    }
}
```
**توضيح:** يقوم هذا المقطع بتحميل رسالة MAPI من دليل محدد. تأكد `dataDir` تم ضبطه بشكل صحيح على بيئتك.

#### الخطوة 2: حفظ رسالة MAPI المحملة مع المرفقات
```csharp
public static void Run()
{
    // قم بتحديد مسار الدليل الذي يوجد به ملف الإدخال
    string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // قم بتحديث هذا باستخدام دليل المستندات الفعلي الخاص بك

    // تحميل رسالة MAPI من الملف.
    MapiMessage msg = MapiMessage.FromFile(dataDir + "/MsgWithAtt.msg");

    // احفظ رسالة MAPI المحملة في ملف آخر مع المرفقات.
    string outputFilePath = dataDir + "/AttachmentsToDestroy_out.msg";
    msg.Save(outputFilePath);

    Console.WriteLine("MAPI message saved successfully.");
}
```
**توضيح:** هنا، نحفظ الرسالة المُحمَّلة في ملف جديد. هذا يضمن حفظ جميع المرفقات أثناء عملية الحفظ.

### الميزة 2: تدمير المرفقات في رسالة MAPI

تُظهر هذه الميزة كيفية إزالة كافة المرفقات من ملف رسالة MAPI المحدد بشكل فعال.

#### ملخص
يمكن أن يساعد إزالة المرفقات غير الضرورية في تبسيط إدارة البريد الإلكتروني لديك وتقليل متطلبات التخزين.

#### الخطوة 1: تحديد الملف مع المرفقات
```csharp
using Aspose.Email.Mapi;
using System;

class FeatureDestroyAttachments
{
    public static void Run()
    {
        // قم بتحديد مسار الدليل حيث يوجد ملف الإخراج
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // قم بتحديث هذا باستخدام دليل المستندات الفعلي الخاص بك

        // حدد ملف رسالة MAPI الذي تريد تدمير المرفقات منه.
        string filePath = dataDir + "/AttachmentsToDestroy_out.msg";
        
        Console.WriteLine("File specified for attachment removal.");
    }
}
```
**توضيح:** تؤدي هذه الخطوة إلى إعداد المسار إلى ملفك المستهدف، مما يضمن عملك مع الملف الصحيح.

#### الخطوة 2: إزالة جميع المرفقات من الملف
```csharp
public static void Run()
{
    // قم بتحديد مسار الدليل حيث يوجد ملف الإخراج
    string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // قم بتحديث هذا باستخدام دليل المستندات الفعلي الخاص بك

    // حدد ملف رسالة MAPI الذي تريد تدمير المرفقات منه.
    string filePath = dataDir + "/AttachmentsToDestroy_out.msg";
    
    // اتصل بالطريقة الثابتة لإزالة كافة المرفقات من الملف المحدد.
    MapiMessage.DestroyAttachments(filePath);

    Console.WriteLine("All attachments removed successfully.");
}
```
**توضيح:** ال `MapiMessage.DestroyAttachments` تقوم هذه الطريقة بمسح جميع المرفقات بكفاءة، مما يضمن أن تكون رسالتك نظيفة ومبسطة.
