## دليل التنفيذ

### إنشاء وتكوين MapiTask

**ملخص:** تتيح لك هذه الميزة إنشاء مهام بخصائص مفصلة وإعداد أنماط التكرار للتذكيرات الدورية.

#### الخطوة 1: إنشاء MapiTask جديد
ابدأ بإنشاء مثيل لـ `MapiTask`:
```csharp
using Aspose.Email.Mapi;

// قم بإنشاء مهمة جديدة بالعنوان والنص وتواريخ البدء والاستحقاق
MapiTask task = new MapiTask("This is test task", "Sample Body", new DateTime(2015, 7, 1), new DateTime(2015, 7, 1));
task.State = MapiTaskState.NotAssigned;
```
**توضيح:** هنا، `MapiTask` تم تهيئة هذا النموذج بعنوان ونص. تاريخ البدء والاستحقاق مُحدد مبدئيًا في ١ يوليو ٢٠١٥.

#### الخطوة 2: تعيين نمط التكرار السنوي
بعد ذلك، قم بتكوين المهمة لتتكرر سنويًا:
```csharp
// تحديد نمط تكرار سنوي يبدأ في اليوم 15، ويتكرر كل 12 شهرًا لمدة 3 مرات
var rec = new MapiCalendarMonthlyRecurrencePattern
{
    Day = 15,
    Period = 12,
    PatternType = MapiCalendarRecurrencePatternType.Month,
    EndType = MapiCalendarRecurrenceEndType.EndAfterNOccurrences,
    OccurrenceCount = 3,
};

// تأكد من أن عدد مرات الظهور يجب أن يكون 1 على الأقل لتجنب التكوين غير الصحيح
if (rec.OccurrenceCount == 0)
{
    rec.OccurrenceCount = 1;
}
task.Recurrence = rec;
```
**توضيح:** يقوم هذا المربع بإعداد تكرار سنوي يبدأ في 15 يوليو، ويحدث كل عام لثلاث تكرارات.

### تعديل المنطقة الزمنية

**ملخص:** قم بضبط تواريخ المهام وفقًا للإزاحة الزمنية المحلية لضمان الجدولة الدقيقة عبر المناطق المختلفة.

#### الخطوة 3: الحصول على إزاحة المنطقة الزمنية المحلية
يُعدِّل `DateTime` الأشياء التي تستخدم المنطقة الزمنية المحلية الحالية:
```csharp
using System;

// استرداد المنطقة الزمنية الحالية وإزاحتها بتوقيت UTC
TimeZone localZone = TimeZone.CurrentTimeZone;
TimeSpan ts = localZone.GetUtcOffset(DateTime.Now);

// ضبط التواريخ عن طريق إضافة إزاحة المنطقة الزمنية المحلية
DateTime StartDate = new DateTime(2015, 7, 1).Add(ts);
DateTime DueDate = new DateTime(2015, 7, 1).Add(ts);
DateTime endByDate = new DateTime(2020, 12, 31).Add(ts);
```
**توضيح:** يقوم هذا الكود بضبط تاريخ بدء المهمة وتاريخ استحقاقها لتعكس المنطقة الزمنية المحلية، وهو أمر ضروري للتطبيقات المستخدمة عبر مواقع جغرافية مختلفة.
