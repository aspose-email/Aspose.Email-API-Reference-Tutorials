## دليل التنفيذ

### الميزة: إنشاء رسالة MAPI وحفظها باستخدام الاستطلاع

تتيح لك هذه الميزة إنشاء رسالة بريد إلكتروني باستخدام Aspose.Email لـ .NET، وتكوينها باستخدام خيارات الاستطلاع، وحفظها كملف.

#### ملخص
ستتعلم كيفية:
- إنشاء رسالة MAPI أساسية.
- إعداد أزرار التصويت داخل البريد الإلكتروني.
- احفظ الرسالة التي تم تكوينها في الموقع المطلوب.

#### خطوات التنفيذ

##### الخطوة 1: تحديد دليل الإخراج
ابدأ بتحديد المكان الذي تريد حفظ ملف الإخراج فيه. استبدل `"YOUR_OUTPUT_DIRECTORY"` مع المسار الفعلي على جهازك.
```csharp
string dataDir = "YOUR_OUTPUT_DIRECTORY";
```

##### الخطوة 2: إنشاء رسالة اختبار MAPI
قم بإنشاء الهيكل الأولي للرسالة باستخدام تفاصيل المرسل والمستقبل والموضوع والنص المحددة مسبقًا.
```csharp
private static MapiMessage CreateTestMessage(bool draft)
{
    MapiMessage msg = new MapiMessage(
        "from@test.com",
        "to@test.com",
        "Flagged Message",
        "Make it nice and short, but descriptive. The description may appear in search engines' search results pages..."
    );

    if (!draft)
    {
        msg.SetMessageFlags(msg.Flags ^ MapiMessageFlags.MSGFLAG_UNSENT);
    }

    return msg;
}
```
*توضيح*:هذه الطريقة تقوم ببناء `MapiMessage` كائن يحتوي على تفاصيل البريد الإلكتروني ويحدد الرسالة بشكل اختياري على أنها مرسلة.

##### الخطوة 3: إعداد خيارات الاستطلاع
قم بتكوين الاستطلاع بتحديد أزرار التصويت. هنا، نستخدم "نعم"، "لا"، "ربما"، و"بالضبط!" كخيارات.
```csharp
FollowUpOptions options = new FollowUpOptions();
options.VotingButtons = "Yes;No;Maybe;Exactly!";
```

##### الخطوة 4: تطبيق خيارات المتابعة على الرسالة
قم بربط تكوين الاستطلاع الخاص بك بالرسالة باستخدام `FollowUpManager`.
```csharp
FollowUpManager.SetOptions(msg, options);
```

##### الخطوة 5: حفظ رسالة MAPI في ملف
أخيرًا، احفظ الرسالة التي تم تكوينها في ملف في الدليل المحدد.
```csharp
msg.Save(dataDir + "/MapiMsgWithPoll.msg");
```

**نصائح استكشاف الأخطاء وإصلاحها**تأكد من ضبط جميع المسارات بشكل صحيح ومنحها الأذونات اللازمة. إذا واجهت مشاكل في حفظ الملفات، فتأكد من وجود الدليل أو أنشئه برمجيًا.
