### تشفير الرسالة

يضمن تشفير رسائل البريد الإلكتروني سرية رسائلك أثناء نقلها. إليك كيفية تحقيق ذلك باستخدام Aspose.Email:

#### الخطوة 1: إعداد البيئة الخاصة بك

أولاً، تأكد من أن شهادتك العامة جاهزة لأغراض التشفير. ستحتاج إلى المسار إلى `.cer` ملف.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string publicCertFile = dataDir + "MartinCertificate.cer";
X509Certificate2 publicCert = new X509Certificate2(publicCertFile);
```

#### الخطوة 2: إنشاء رسالة وتشفيرها

بعد ذلك، قم بإنشاء رسالة البريد الإلكتروني الخاصة بك واستخدم الشهادة لتشفيرها.

```csharp
using Aspose.Email.Mime;
using System.Security.Cryptography.X509Certificates;

MailMessage msg = new MailMessage("sender@example.com", "recipient@example.com");
msg.Subject = "Encrypted Email";
msg.Body = "This is an encrypted message.";

// تشفير الرسالة باستخدام الشهادة العامة
msg.Encrypt(publicCert);
```

في هذا المثال:
- ال `Encrypt` تستخدم الطريقة مثيل X509Certificate2 لتشفير محتوى البريد الإلكتروني.
- يتم تعيين الموضوع والنص قبل التشفير، مما يضمن أن الأطراف المصرح لها فقط هي التي يمكنها فك تشفيره.

#### نصائح استكشاف الأخطاء وإصلاحها
- **مشكلة شائعة:** إذا واجهت خطأً يتعلق بتحميل الشهادة، فتأكد من أن `.cer` مسار الملف صحيح.
