## دليل التنفيذ
يقوم هذا القسم بتقسيم العملية إلى خطوات قابلة للإدارة.

### تحميل وتفعيل الشهادات
#### ملخص
يُعد تحميل شهادات X.509 أمرًا بالغ الأهمية لتوقيع رسائل البريد الإلكتروني رقميًا. سنستخدم `Aspose.Email` لتحميل الشهادات العامة والخاصة من الملفات.

##### الخطوة 1: تحميل الشهادة العامة
الشهادة العامة، عادة في `.cer` يمكن تحميل التنسيق على النحو التالي:
```csharp
using System.Security.Cryptography.X509Certificates;

string publicCertFile = @"YOUR_DOCUMENT_DIRECTORY\MartinCertificate.cer";
X509Certificate2 publicCert = new X509Certificate2(publicCertFile);
```
*توضيح*:يقوم هذا المقطع بتحميل الشهادة من مسار ملف محدد. `X509Certificate2` يتم استخدام الفئة للتعامل مع الشهادة.

##### الخطوة 2: تحميل الشهادة الخاصة بكلمة المرور
يتطلب تحميل الشهادة الخاصة تحديد كلمة المرور الخاصة بها:
```csharp
string privateCertFile = @"YOUR_DOCUMENT_DIRECTORY\MartinCertificate.pfx";
X509Certificate2 privateCert = new X509Certificate2(privateCertFile, "password");
```
*توضيح*:يجب تحميل ملف PFX الذي يحتوي على المفتاح الخاص بكلمة مرور لأسباب أمنية.

### إنشاء رسالة بريد إلكتروني وتوقيعها
#### ملخص
بعد أن أصبحت شهاداتك جاهزة، فلنبدأ في إنشاء رسالة بريد إلكتروني وتوقيعها باستخدام Aspose.Email.

##### الخطوة 1: إنشاء `MailMessage`
أولاً، قم بإنشاء `MailMessage` هدف:
```csharp
using Aspose.Email.Mime;

MailMessage msg = new MailMessage("userfrom@gmail.com", "userto@domain.com");
msg.Subject = "Secure Communication";
msg.Body = "This is a digitally signed email.";
```
*توضيح*:هنا، قمنا بإعداد المرسل والمستقبل والموضوع ونص البريد الإلكتروني الخاص بنا.

##### الخطوة 2: إرفاق التوقيع الرقمي
لإرفاق التوقيع الرقمي:
```csharp
msg.Attachments.Add(new Attachment(privateCert));
```
*توضيح*نستخدم الشهادة الخاصة لتوقيع الرسالة. تضمن هذه الخطوة التحقق من سلامة الرسالة ومصدرها من قِبل المستلمين.
