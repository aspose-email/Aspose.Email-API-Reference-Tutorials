## دليل التنفيذ

دعونا نستكشف الميزات الرئيسية التي يوفرها Aspose.Email لـ .NET.

### تعيين نص HTML في رسالة البريد الإلكتروني

**ملخص:** يتيح لك إنشاء نص HTML غني إنشاء محتوى بريد إلكتروني جذاب بصريًا مع التنسيق والصور والروابط.

#### التنفيذ خطوة بخطوة

**1. إنشاء كائن MailMessage جديد**

```csharp
using Aspose.Email.Mime;

// تهيئة كائن رسالة البريد
MailMessage message = new MailMessage();
```

**2. تعيين محتوى نص HTML**

```csharp
// تعريف نص HTML
message.HtmlBody = "<html><body>This is the HTML body</body></html>";
```
- **المعلمات موضحة:** `HtmlBody` يأخذ سلسلة تمثل محتوى HTML الخاص ببريدك الإلكتروني.

### تحويل البريد الإلكتروني إلى تنسيق MHTML

**ملخص:** يتيح تحويل رسائل البريد الإلكتروني إلى تنسيق MHTML إمكانية تقديم ملفات فردية تتضمن جميع الموارد، مما يسهل أغراض الأرشفة والعرض.

#### التنفيذ خطوة بخطوة

**1. إنشاء وتكوين MailMessage**

```csharp
using Aspose.Email.Storage.Mht;
using System.IO;

// تهيئة رسالة البريد الإلكتروني بتفاصيل المرسل والمستلم
MailMessage mailMsg = new MailMessage("from@example.com", "to@example.com");
mailMsg.Subject = "Email Subject";
mailMsg.Body = "This is the body of the email.";
```

**2. التحويل إلى MHTML**

```csharp
// إعداد تدفق الذاكرة للإخراج
MemoryStream mhtmlStream = new MemoryStream();

// حفظ الرسالة بتنسيق MHTML
mailMsg.Save(mhtmlStream, SaveOptions.DefaultMhtml);
```
- **تكوين المفتاح:** `SaveOptions.DefaultMhtml` يضمن تضمين كافة الموارد في التحويل.

### تحميل وعرض خصائص البريد الإلكتروني

**ملخص:** يعد تحميل ملف بريد إلكتروني وعرض خصائصه مفيدًا لأغراض تصحيح الأخطاء أو استخراج البيانات.

#### التنفيذ خطوة بخطوة

**1. تحميل ملف البريد الإلكتروني**

```csharp
using Aspose.Email;

// تحميل البريد الإلكتروني من مسار محدد
MailMessage loadedEmail = MailMessage.Load("YOUR_DOCUMENT_DIRECTORY\\example.eml");
```

**2. عرض خصائص البريد الإلكتروني**

```csharp
// إخراج عنوان الموضوع والمرسل إلى وحدة التحكم
Console.WriteLine(loadedEmail.Subject);
Console.WriteLine(loadedEmail.From.Address);
```
- **المعلمات موضحة:** `Load` يقرأ ملف البريد الإلكتروني، بينما خصائص مثل `Subject` و `From` يمكن الوصول إليها مباشرة.
