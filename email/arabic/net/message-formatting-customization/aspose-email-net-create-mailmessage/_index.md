## دليل التنفيذ

الآن دعنا نوضح كيفية إنشاء وتكوين `MailMessage` خطوة بخطوة:

### إنشاء مثيل لـ MailMessage

ابدأ بإنشاء مثيل لـ `MailMessage`يسمح لك هذا الكائن بتحديد محتوى البريد الإلكتروني ومعالجته برمجيًا.

```csharp
using Aspose.Email.Mime;

// إنشاء مثيل جديد لـ MailMessage
MailMessage message = new MailMessage("Sender <sender@from.com>", "Recipient <recipient@to.com>");
```

#### توضيح:
- **`new MailMessage()`**:يتم تهيئة رسالة بريد إلكتروني تحتوي على المرسل والمستلم الأساسي.
- **`"Sender <sender@from.com>"`**:يحدد أصل البريد الإلكتروني.

### تكوين عناوين "إلى"

أضف عدة مستلمين إلى `MailMessage`يعد هذا أمرًا ضروريًا لإرسال رسائل البريد الإلكتروني إلى عدة أشخاص في وقت واحد.

```csharp
// إضافة عناوين "إلى" متعددة إلى البريد الإلكتروني
message.To.Add("receiver1@receiver.com");
message.To.Add("receiver2@receiver.com");
message.To.Add("receiver3@receiver.com");
message.To.Add("receiver4@receiver.com");
```

#### توضيح:
- **`message.To.Add()`**:يضيف عنوان كل مستلم إلى قائمة عناوين "إلى".

### إضافة عناوين CC (نسخة كربونية)

يتلقى مستلمو النسخة الكربونية نسخة من بريدك الإلكتروني، وتكون مرئية لجميع المستلمين الآخرين. هذا مفيد لإبقاء الأطراف المعنية على اطلاع.

```csharp
// إضافة عناوين "CC" (نسخة كربونية)
message.CC.Add("CC1@receiver.com");
message.CC.Add("CC2@receiver.com");
```

#### توضيح:
- **`message.CC.Add()`**:يضيف عنوان البريد الإلكتروني إلى قائمة مستلمي النسخة الكربونية.

### إضافة عناوين BCC (نسخة كربونية عمياء)

تتيح لك ميزة BCC إرسال رسائل البريد الإلكتروني دون الكشف عن عناوين جميع المستلمين، مما يحافظ على الخصوصية لبعض جهات الاتصال.

```csharp
// إضافة عناوين "نسخة كربونية عمياء"
message.Bcc.Add("Bcc1@receiver.com");
message.Bcc.Add("Bcc2@receiver.com");
```

#### توضيح:
- **`message.Bcc.Add()`**:يضيف عنوان البريد الإلكتروني إلى قائمة BCC.
