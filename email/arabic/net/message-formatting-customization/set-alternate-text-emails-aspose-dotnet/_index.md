## دليل التنفيذ

الآن، دعنا ننفذ إعداد النص البديل لرسالة البريد الإلكتروني.

### إنشاء مثيل MailMessage
ابدأ بالإعلان عن `MailMessage` هدف:

```csharp
// إعلان عن مثيل MailMessage.
MailMessage message = new MailMessage();
```

تعمل هذه الخطوة على تهيئة كائن البريد الإلكتروني الخاص بك حيث ستضيف المحتوى والتكوينات.

### تعيين نص بديل باستخدام AlternateView
يتيح لك العرض البديل عرض رسائل بريد إلكتروني مختلفة. إليك كيفية إنشاء عرض بديل:

```csharp
// إنشاء AlternateView بالمحتوى المحدد كسلسلة.
AlternateView alternate = AlternateView.CreateAlternateViewFromString("This is the alternate text.");
```

ال `CreateAlternateViewFromString` تأخذ الطريقة سلسلة نصية عادية، مما يضمن أنه إذا لم يتمكن بريدك الإلكتروني من عرض HTML أو المرفقات بشكل صحيح، فسيتم عرض هذا النص بدلاً من ذلك.

### إضافة AlternateView إلى MailMessage
وأخيرًا، أضف العرض البديل إلى رسالتك:

```csharp
// قم بإضافة AlternateView الذي تم إنشاؤه إلى مجموعة AlternateViews الخاصة بـ MailMessage.
message.AlternateViews.Add(alternate);
```

تضمن هذه الخطوة إمكانية الرجوع إلى هذا النص في بريدك الإلكتروني عند الحاجة إليه.
