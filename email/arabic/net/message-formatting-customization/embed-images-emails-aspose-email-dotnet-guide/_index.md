## دليل التنفيذ

دعونا نقوم بتقسيم عملية تضمين الصور في رسائل البريد الإلكتروني باستخدام Aspose.Email لـ .NET.

### تحديد مسارات الملفات

أولاً، قم بتحديد مسارات الملفات التي سيتم حفظ مواردك فيها:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string dstEmail = dataDir + "/EmbeddedImage.msg";
```

### إنشاء مثيل MailMessage

قم بإعداد الخصائص الأساسية لبريدك الإلكتروني بما في ذلك المرسل والمستلم والموضوع:

```csharp
MailMessage mail = new MailMessage();
mail.From = new MailAddress("test001@gmail.com");
mail.To.Add("test001@gmail.com");
mail.Subject = "This is an email";
```

### إنشاء جزء النص العادي

إنشاء عرض نص عادي لبريدك الإلكتروني للعملاء الذين لا يدعمون HTML:

```csharp
AlternateView plainView = AlternateView.CreateAlternateViewFromString(
    "This is my plain text content", null, "text/plain");
```

### إنشاء عرض HTML مع الصورة المضمنة

قم بإنشاء نسخة HTML من بريدك الإلكتروني وقم بتضمين صورة باستخدام معرف المحتوى (CID):

```csharp
string htmlContent = "Here is an embedded image.<img src='cid:barcode'>";
AlternateView htmlView = AlternateView.CreateAlternateViewFromString(
    htmlContent, null, "text/html");
```

### تضمين الصورة

استخدم LinkedResource لإرفاق صورتك وتعيين ContentId الخاص بها:

```csharp
LinkedResource barcode = new LinkedResource(dataDir + "/1.jpg", MediaTypeNames.Image.Jpeg)
{
    ContentId = "barcode"
};
mail.LinkedResources.Add(barcode);
```

تعتبر هذه الخطوة بالغة الأهمية لأنها تربط الصورة بمعرف CID محدد، مما يسمح بالإشارة إليها في محتوى HTML الخاص بك.

### إضافة وجهات نظر إلى البريد الإلكتروني

قم بإرفاق كلا العرضين (النص العادي وHTML) برسالة البريد الإلكتروني الخاصة بك:

```csharp
mail.AlternateViews.Add(plainView);
mail.AlternateViews.Add(htmlView);
```

### حفظ البريد الإلكتروني

أخيرًا، احفظ بريدك الإلكتروني بالموارد المضمنة بتنسيق ملف محدد:

```csharp
mail.Save(dataDir + "/EmbeddedImage_out.msg", SaveOptions.DefaultMsgUnicode);
```

تضمن هذه الخطوة أن يكون بريدك الإلكتروني جاهزًا للإرسال أو المعالجة الإضافية.
