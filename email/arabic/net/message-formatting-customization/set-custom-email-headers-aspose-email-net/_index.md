## دليل التنفيذ

### إعداد رؤوس البريد الإلكتروني

#### ملخص
يتيح لك تخصيص عناوين البريد الإلكتروني توفير بيانات تعريفية إضافية والتحكم في كيفية معالجة رسائل البريد الإلكتروني. سيرشدك هذا القسم إلى كيفية إعداد عناوين قياسية مختلفة مثل `ReplyTo`، `From`، `CC`، `BCC`، وكذلك المخصصة منها مثل `X-Mailer`.

##### إضافة عناوين البريد الإلكتروني
أولاً، دعنا نحدد من هو المرسل إليه البريد الإلكتروني، ومن هو المرسل إليه، والمستلمين الآخرين.

```csharp
// إنشاء مثيل لفئة MailMessage
MailMessage mailMessage = new MailMessage();

// حدد حقول البريد الإلكتروني: الرد إلى، من، إلى، نسخة كربونية، ونسخة مخفية
mailMessage.ReplyToList.Add("reply@reply.com");
mailMessage.From = "sender@sender.com";
mailMessage.To.Add("receiver1@receiver.com");
mailMessage.CC.Add("receiver2@receiver.com");
mailMessage.Bcc.Add("receiver3@receiver.com");
```

##### تعيين خصائص إضافية

بعد ذلك، قم بتكوين خصائص البريد الإلكتروني الأساسية الأخرى.

```csharp
// تعيين خصائص إضافية مثل التاريخ والموضوع وXMailer والرؤوس المخصصة
mailMessage.Subject = "test mail";
mailMessage.Date = new DateTime(2006, 3, 6);
mailMessage.XMailer = "Aspose.Email";

// إضافة رأس مخصص
mailMessage.Headers.Add("secret-header", "my secret value");
```

**توضيح**: 
- `ReplyToList` يسمح بتعيين عنوان البريد الإلكتروني للرد.
- ال `From`، `To`، `CC`، و `Bcc` الحقول واضحة، وتحدد عناوين البريد الإلكتروني الخاصة بكل منها.
- يمكن إضافة رؤوس مخصصة باستخدام `mailMessage.Headers.Add()`.

### حفظ رسائل البريد الإلكتروني

بعد تهيئة بريدك الإلكتروني، قد ترغب في حفظه على القرص لأغراض الأرشفة أو الاختبار. إليك الطريقة:

```csharp
// تحديد الدلائل للإدخال/الإخراج
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// حفظ رسالة البريد في ملف
mailMessage.Save($"{outputDir}/EmailOutput.eml");
```

**توضيح**: 
- `Save()` يتم استخدام الطريقة لكتابة رسالة البريد الإلكتروني إلى مسار محدد بتنسيق EML.
