### الميزة 1: الاتصال وتسجيل الدخول إلى POP3

#### ملخص:
يُعد الاتصال بخادم POP3 الخطوة الأولى لإدارة بريدك الإلكتروني برمجيًا. يُبسط Aspose.Email لـ .NET هذه العملية، مما يسمح لك بدمج وظائف البريد الإلكتروني في تطبيقاتك بسهولة.

**الخطوة 1: تحديد معلمات الاتصال**
إنشاء فئة تغلف تفاصيل الاتصال وتقوم بتهيئتها `Pop3Client`.

```csharp
using Aspose.Email.Clients.Pop3;

namespace Pop3ConnectionExample
{
    public class ConnectAndLoginPOP3
    {
        public void Run()
        {
            // تحديد معلمات الاتصال
            const string host = "your.pop3.host.com";     // تحديد مضيف خادم POP3
            const int port = 110;                         // رقم منفذ POP3 الافتراضي
            const string username = "user@host.com";      // عنوان بريدك  الإلكتروني
            const string password = "password";           // كلمة المرور لحساب البريد الإلكتروني الخاص بك

            // إنشاء مثيل Pop3Client باستخدام معلمات محددة
            Pop3Client client = new Pop3Client(host, port, username, password);
            
            // اختياريًا: التحقق من حالة الاتصال
            if (client.Connected)
            {
                Console.WriteLine("Connected to POP3 server successfully.");
            }
        }
    }
}
```

**خيارات تكوين المفاتيح:**
- **يستضيف**:عنوان خادم POP3 الخاص بك.
- **ميناء**:عادةً 110 للاتصالات غير الآمنة أو 995 للاتصالات الآمنة.
- **اسم المستخدم وكلمة المرور**:بيانات الاعتماد للمصادقة مع الخادم.

#### نصائح استكشاف الأخطاء وإصلاحها:
- تأكد من أن إعدادات جدار الحماية تسمح بالاتصال بالمنفذ المحدد.
- تأكد من صحة بيانات الاعتماد وتفاصيل الخادم المقدمة.
- استخدم كتل try-catch للتعامل مع الاستثناءات بسلاسة.

### الميزة 2: تطبيق المرشحات الحساسة لحالة الأحرف على رسائل البريد الإلكتروني للبحث

#### ملخص:
يُعدّ البحث في رسائل البريد الإلكتروني بناءً على معايير محددة أمرًا بالغ الأهمية للعديد من التطبيقات. يتيح لك Aspose.Email تطبيق مرشحات حساسة لحالة الأحرف، مما يُحسّن دقة البحث.

**الخطوة 1: الاتصال والمصادقة**
يضمن `Pop3Client` تم إعداده بالفعل كما هو موضح في الميزة 1.

```csharp
using Aspose.Email.Tools.Search;

namespace EmailSearchExample
{
    public class CaseSensitiveEmailSearch
    {
        public void Run()
        {
            // افترض أن Pop3Client متصل ومصادق عليه
            Pop3Client client = new Pop3Client("your.pop3.host.com", 110, "user@host.com", "password");

            try
            {
                // إنشاء مثيل MailQueryBuilder
                MailQueryBuilder builder1 = new MailQueryBuilder();

                // إضافة مرشح حساس لحالة الأحرف لرسائل البريد الإلكتروني من 'tesT'
                builder1.From.Contains("tesT", true);

                // احصل على الاستعلام بناءً على تكوين المنشئ
                MailQuery query1 = builder1.GetQuery();
                
                // قائمة الرسائل التي تتطابق مع معايير الاستعلام
                Pop3MessageInfoCollection messageInfoCol1 = client.ListMessages(query1);
                
                Console.WriteLine($"Found {messageInfoCol1.Count} emails matching the criteria.");
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error searching emails: " + ex.Message);
            }
        }
    }
}
```

**خيارات تكوين المفاتيح:**
- **حساسية الحالة**: تم الضبط على `true` للحصول على تطابقات دقيقة للحالة.
- **منشئ الاستعلامات**:يسهل إنشاء الاستعلامات المعقدة بسهولة.

#### نصائح استكشاف الأخطاء وإصلاحها:
- تأكد من اتصال الشبكة عند الوصول إلى الخادم.
- تعامل مع الاستثناءات لتجنب تعطل التطبيق أثناء عمليات البحث عن البريد الإلكتروني.
