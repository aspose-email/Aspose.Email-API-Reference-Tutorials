### حذف رسائل البريد الإلكتروني حسب الفهرس
**ملخص**:بمجرد الاتصال، قم باسترداد العدد الإجمالي للرسائل وحذف كل رسالة حسب فهرسها.

**الخطوة 2: استرداد عدد الرسائل**
```csharp

```
- **غاية**:يؤدي هذا إلى إرجاع عدد صحيح يمثل عدد رسائل البريد الإلكتروني الموجودة، والذي سنستخدمه للتكرار وحذف كل منها.

**الخطوة 3: حذف الرسائل حسب الفهرس**
```csharp

{
    // قم بالتكرار على جميع الرسائل وحذفها باستخدام رقم الفهرس الخاص بها

        client.DeleteMessage(i);
    }
}
catch (Exception ex)
{
    // التعامل مع أي استثناءات قد تحدث أثناء عملية الحذف

}
```
- **توضيح**:تتكرر الحلقة عبر كل بريد إلكتروني حسب فهرسه. `DeleteMessage(int)` يحذف البريد الإلكتروني في موضع محدد.
- **نصائح لاستكشاف الأخطاء وإصلاحها**:تأكد من صحة بيانات الاعتماد الخاصة بك وأن لديك الأذونات اللازمة لحذف رسائل البريد الإلكتروني.
