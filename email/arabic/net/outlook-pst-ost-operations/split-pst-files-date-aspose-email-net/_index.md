## دليل التنفيذ

في هذا القسم، سنشرح كيفية تنفيذ الميزة خطوة بخطوة.

### تحديد معايير تعتمد على التاريخ لتقسيم ملفات PST

**ملخص:**
لتقسيم ملف PST استنادًا إلى معايير التاريخ، قم بتحديد نطاقات تاريخ محددة باستخدام منشئي الاستعلامات المقدمة بواسطة Aspose.Email.

#### الخطوة 1: إعداد الدلائل الخاصة بك
حدد الدلائل لملفات الإدخال والإخراج:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // دليل الإدخال
string outputDir = "YOUR_OUTPUT_DIRECTORY";   // دليل الإخراج
```

#### الخطوة 2: إنشاء استعلامات معايير التاريخ
يستخدم `PersonalStorageQueryBuilder` لإنشاء استعلامات تحدد نطاقات التاريخ للتقسيم.

**الاستعلام 1:** رسائل البريد الإلكتروني من 1 أبريل 2005 إلى 6 أبريل 2005.
```csharp
PersonalStorageQueryBuilder pstQueryBuilder1 = new PersonalStorageQueryBuilder();
pstQueryBuilder1.SentDate.Since(new DateTime(2005, 04, 01)); // تاريخ البدء
pstQueryBuilder1.SentDate.Before(new DateTime(2005, 04, 07));   // تاريخ الانتهاء
```

**الاستعلام 2:** رسائل البريد الإلكتروني من 7 أبريل 2005 إلى 12 أبريل 2005.
```csharp
PersonalStorageQueryBuilder pstQueryBuilder2 = new PersonalStorageQueryBuilder();
pstQueryBuilder2.SentDate.Since(new DateTime(2005, 04, 07)); // تاريخ البدء
pstQueryBuilder2.SentDate.Before(new DateTime(2005, 04, 13));   // تاريخ الانتهاء
```

أضف هذه الاستعلامات إلى القائمة:
```csharp
IList<MailQuery> criteria = new List<MailQuery>();
criteria.Add(pstQueryBuilder1.GetQuery());
criteria.Add(pstQueryBuilder2.GetQuery());
```

#### الخطوة 3: تنظيف دليل الإخراج
تأكد من أن دليل الإخراج لديك خالٍ من ملفات PST السابقة قبل البدء:
```csharp
if (Directory.GetFiles(outputDir + "pathToPst", "*.pst").Length > 0)
{
    string[] files = Directory.GetFiles(outputDir + "pathToPst");
    foreach (string file in files)
    {
        if(file.Contains(".pst"))
            File.Delete(file); // حذف ملفات PST الموجودة
    }
}
```

#### الخطوة 4: تقسيم ملف PST الأصلي
قم بتحميل ملف PST الأصلي الخاص بك وقم بتقسيمه باستخدام المعايير المحددة:
```csharp
using (PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir + "PersonalStorage_New.pst"))
{
    personalStorage.SplitInto(criteria, outputDir + "pathToPst");
}
```
**توضيح:**
- `FromFile`:يقوم بتحميل ملف PST الأصلي.
- `SplitInto`:تقسيم الملف وفقًا لمعاييرك وحفظه في الدليل المحدد.
