#### الخطوة 1: تحديد مسارات الملفات (H3)
حدد مسار ملف OST المصدر ومسار إخراج ملف PST المستهدف:

```csharp
string sourceFilePath = "@YOUR_DOCUMENT_DIRECTORY/PersonalStorageFile.ost";
string targetFilePath = "@YOUR_OUTPUT_DIRECTORY/test.pst";
```

#### الخطوة 2: افتح ملف OST (H3)
استخدم `FromFile` الطريقة لفتح ملف OST وقراءة محتوياته وتحميلها:

```csharp
using (PersonalStorage personalStorage = PersonalStorage.FromFile(sourceFilePath))
{
    // المضي قدما في التحويل
}
```

#### الخطوة 3: الحفظ كملف PST (H3)
بمجرد فتح ملف OST، استخدم `SaveAs` طريقة لتحويله وحفظه بتنسيق PST. `FileFormat.Pst` تحدد المعلمة تنسيق الإخراج المطلوب:

```csharp
personalStorage.SaveAs(targetFilePath, FileFormat.Pst);
```
