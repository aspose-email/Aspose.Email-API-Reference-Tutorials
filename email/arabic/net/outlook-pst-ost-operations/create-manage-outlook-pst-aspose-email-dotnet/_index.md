## دليل التنفيذ

### إنشاء ملف PST جديد

**ملخص**:يوضح هذا القسم كيفية إنشاء ملف جدول تخزين شخصي (PST) جديد في Outlook بتنسيق Unicode لتحقيق التوافق والكفاءة.

#### خطوات:
1. **تحديد مسار الملف:**
   ```csharp
   string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "new_pst_out.pst");
   ```
2. **التحقق من وجود ملف موجود:**
   تأكد من عدم وجود أي ملف موجود في دليل الهدف الخاص بك لتجنب التعارضات:
   ```csharp
   if (File.Exists(dataDir)) {
       File.Delete(dataDir);
   }
   ```
3. **إنشاء PST:**
   قم بإنشاء ملف PST جديد باستخدام تنسيق Unicode، الذي يدعم مجموعات أحرف أوسع.
   ```csharp
   PersonalStorage personalStorage = PersonalStorage.Create(dataDir, FileFormatVersion.Unicode);
   ```

### إضافة مجلد إلى PST

**ملخص**:تعرف على كيفية إضافة مجلدات فرعية مثل "البريد الوارد" إلى ملف PST موجود لتحسين التنظيم.

#### خطوات:
1. **تحميل ملف PST الموجود:**
   ```csharp
   if (File.Exists(dataDir)) {
       PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
   }
   ```
2. **إضافة مجلد فرعي:**
   أضف مجلدًا جديدًا، مثل "البريد الوارد"، أسفل الجذر.
   ```csharp
   personalStorage.RootFolder.AddSubFolder("Inbox");
   ```

### إضافة الرسائل إلى مجلد في PST

**ملخص**:يوضح هذا القسم كيفية إضافة الرسائل إلى مجلد "البريد الوارد" الموجود ضمن ملف PST الخاص بك.

#### خطوات:
1. **قم بتحميل ملف PST والرسالة الموجودين:**
   تأكد من إمكانية الوصول إلى كلا الملفين:
   ```csharp
   if (File.Exists(dataDir)) {
       PersonalStorage personalStorage = PersonalStorage.FromFile(dataDir);
   }
   string msgFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "MapiMsgWithPoll.msg");
   ```
2. **حدد المجلد وأضف الرسائل:**
   استرداد مجلد "البريد الوارد" وإضافة رسالة:
   ```csharp
   FolderInfo inboxFolder = personalStorage.RootFolder.GetSubFolder("Inbox");
   MapiMessage mapiMsg = MapiMessage.FromFile(msgFilePath);
   inboxFolder.AddMessage(mapiMsg);
   ```
