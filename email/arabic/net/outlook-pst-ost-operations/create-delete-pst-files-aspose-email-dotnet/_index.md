## دليل التنفيذ

سنقوم بتقسيم التنفيذ إلى ميزتين رئيسيتين: إنشاء/حذف ملفات PST وإضافة مجلدات/ملفات فرعية إليها.

### الميزة 1: إنشاء ملف PST وحذفه

**ملخص**تساعدك هذه الميزة على إنشاء ملف PST جديد بتنسيق Unicode أو حذف ملف موجود إذا كان موجودًا بالفعل.

#### التنفيذ خطوة بخطوة:

##### 1. تحديد مسار الدليل
ابدأ بتعيين الدليل الذي سيتم تخزين ملفات PST فيه.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string path = Path.Combine(dataDir, "Ps1_out.pst");
```

##### 2. تحقق من وجود ملفات PST الموجودة واحذفها إذا لزم الأمر
تأكد من عدم تكرار الملفات الموجودة عن طريق التحقق من وجودها أولاً.
```csharp
if (File.Exists(path))
{
    File.Delete(path);
}
```

##### 3. إنشاء ملف PST جديد
قم بإنشاء الملف الجديد باستخدام تنسيق Unicode لضمان التوافق مع عملاء البريد الإلكتروني المختلفين.
```csharp
using (PersonalStorage personalStorage = PersonalStorage.Create(Path.Combine(dataDir, "Ps1_out.pst"), FileFormatVersion.Unicode))
{
    // تم الانتهاء من إنشاء ملف PST هنا.
}
```

### الميزة 2: إضافة مجلد فرعي وملفات إلى PST

**ملخص**:بعد إنشاء ملف PST، يمكنك تنظيم محتوياته عن طريق إضافة مجلدات فرعية وملفات.

#### التنفيذ خطوة بخطوة:

##### 1. تأكد من وجود ملف PST
تحقق مما إذا كان ملف PST موجودًا؛ إذا لم يكن موجودًا، فقم بإنشائه.
```csharp
if (!File.Exists(path))
{
    using (PersonalStorage personalStorage = PersonalStorage.Create(path, FileFormatVersion.Unicode))
    {
        // يتم إنشاء المجلد الجذر تلقائيًا هنا.
    }
}
```

##### 2. افتح ملف PST الموجود
قم بتحميل الملف الموجود لإضافة المجلدات الفرعية والملفات.
```csharp
using (PersonalStorage personalStorage = PersonalStorage.FromFile(path))
{
    // افتح المجلد الجذر لملف PST
```

##### 3. إضافة مجلد فرعي
قم بإنشاء مجلد فرعي جديد باسم "الملفات" أسفل المجلد الجذر.
```csharp
FolderInfo folder = personalStorage.RootFolder.AddSubFolder("Files");
```

##### 4. إضافة الملفات إلى المجلد الفرعي
أضف الملفات إلى المجلد الفرعي الذي تم إنشاؤه حديثًا، مع تحديد مسارات الملفات وأي سمات ضرورية.
```csharp
folder.AddFile(Path.Combine(dataDir, "attachment_1.doc"), null);
```
