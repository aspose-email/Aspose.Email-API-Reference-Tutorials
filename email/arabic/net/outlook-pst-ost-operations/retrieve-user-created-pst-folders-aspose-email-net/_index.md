## دليل التنفيذ

### الاستعلام عن المجلدات التي أنشأها المستخدم واسترجاعها
يركز هذا القسم على إعداد استعلام لتصفية واسترداد المجلدات التي أنشأها المستخدمون فقط.

#### 1. قم بتحميل ملف PST
أولاً، قم بتحميل ملف Outlook PST الخاص بك باستخدام `FromFile` طريقة:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (PersonalStorage pst = PersonalStorage.FromFile(dataDir + "Outlook.pst"))
{
    // متابعة الاستعلام عن المجلدات...
}
```

#### 2. إنشاء منشئ الاستعلامات
استخدم `PersonalStorageQueryBuilder` لتحديد شروط الاستعلام الخاصة بك:

```csharp
// إنشاء منشئ استعلامات لتصفية المجلدات التي أنشأها المستخدم
PersonalStorageQueryBuilder queryBuilder = new PersonalStorageQueryBuilder();
queryBuilder.OnlyFoldersCreatedByUser.Equals(true);
```

تقوم هذه الخطوة بتصفية المجلدات، مما يضمن تضمين المجلدات التي أنشأها المستخدمون فقط في النتائج.

#### 3. استرداد المجلدات وعرضها
جلب المجلدات الفرعية التي تطابق معاييرك وعرض أسمائها:

```csharp
// الحصول على المجلدات الفرعية المطابقة للاستعلام
FolderInfoCollection subfolders = pst.RootFolder.GetSubFolders(queryBuilder.GetQuery());

// قم بالتكرار خلال كل مجلد لإجراء العمليات
foreach (FolderInfo folder in subfolders)
{
    Console.WriteLine(folder.DisplayName);
}
```

**توضيح**: هنا، `GetSubFolders` يسترجع المجلدات بناءً على شروطك. ثم نكرر هذه المجلدات ونطبع أسماء عرضها.
