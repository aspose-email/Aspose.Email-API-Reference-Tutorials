## دليل التنفيذ

### قراءة ملفات OLM باستخدام Aspose.Email لـ .NET

#### ملخص
يوضح هذا القسم كيفية الوصول إلى بنية مجلد ملف OLM باستخدام `OlmStorage` فصل.

##### الخطوة 1: تهيئة OlmStorage
للبدء، قم بتهيئة `OlmStorage` باستخدام مسار ملف OLM. سيؤدي هذا إلى تحميل الملف إلى الذاكرة وتجهيزه للوصول.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/SampleOLM.olm";
OlmStorage storage = new OlmStorage(dataDir);
```

##### الخطوة 2: الوصول إلى التسلسل الهرمي للمجلدات
استخدام `storage.FolderHierarchy`يمكنك الوصول إلى بنية المجلد بالكامل الموجودة في ملف OLM. تُرجع هذه الخاصية قائمة من `OlmFolder` الكائنات التي تمثل كل مجلد من المستوى الأعلى.

```csharp
List<OlmFolder> folders = storage.FolderHierarchy;
```

##### الخطوة 3: طباعة مسارات المجلد
تنفيذ طريقة متكررة لاجتياز وطباعة كافة مسارات المجلد، بما في ذلك المجلدات الفرعية:

```csharp
public static void PrintPath(OlmStorage storage, List<OlmFolder> folders)
{
    foreach (OlmFolder folder in folders)
    {
        Console.WriteLine(folder.Path); // إخراج مسار المجلد الحالي.
        
        if (folder.SubFolders.Count > 0)
        {
            PrintPath(storage, folder.SubFolders); // طباعة المجلدات الفرعية بشكل متكرر.
        }
    }
}
```
