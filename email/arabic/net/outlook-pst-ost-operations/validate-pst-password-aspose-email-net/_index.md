## دليل التنفيذ

### التحقق مما إذا كان ملف PST محميًا بكلمة مرور

#### ملخص
تتيح لك هذه الميزة تحديد ما إذا كان ملف PST يتطلب كلمة مرور، وهو أمر ضروري لعمليات تدقيق الأمان وإدارة الوصول.

**الخطوة 1: تحميل ملف PST**

قم بتحميل ملف PST الخاص بك باستخدام Aspose.Email `PersonalStorage` فصل:

```csharp
using (PersonalStorage pst = PersonalStorage.FromFile(dataDir + "/passwordprotectedPST.pst"))
{
    bool isPasswordProtected = pst.Store.IsPasswordProtected;
}
```
- **حدود:** `dataDir` يجب أن يكون المسار إلى دليل المستند الخاص بك.
- **قيمة الإرجاع:** قيمة منطقية تشير إلى ما إذا كان ملف PST محميًا بكلمة مرور.

**الخطوة 2: نتائج الإخراج**

قم بتسجيل هذه النتائج أو عرضها حسب الحاجة:

```csharp
Console.WriteLine($"The storage is password protected - {isPasswordProtected}");
```

### التحقق من صحة كلمة المرور لملف PST

#### ملخص
تعمل هذه الميزة على التحقق مما إذا كانت كلمة المرور المقدمة تتطابق مع كلمة المرور الموجودة في ملف PST، مما يضمن الوصول الآمن.

**الخطوة 1: تحميل كلمة المرور والتحقق منها**

قم بتحميل ملف PST وتحقق من صحة كلمة المرور:

```csharp
using (PersonalStorage pst = PersonalStorage.FromFile(dataDir + "/passwordprotectedPST.pst"))
{
    string testPassword = "Password1";
    bool isValid = pst.Store.IsPasswordValid(testPassword);
}
```
- **حدود:** `testPassword` يجب أن تكون كلمة المرور التي تقوم بالتحقق منها.
- **قيمة الإرجاع:** قيمة منطقية تشير إلى ما إذا كانت كلمة المرور المقدمة صالحة.

**الخطوة 2: نتائج الإخراج**

سجل أو اعرض ما إذا كانت عملية التحقق من صحة كلمة المرور ناجحة:

```csharp
Console.WriteLine($"Password {testPassword} is valid - {isValid}");
```
