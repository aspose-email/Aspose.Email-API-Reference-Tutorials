## دليل التنفيذ

### عرض معلومات ملف PST

يرشدك هذا القسم خلال عملية تحميل ملف PST وعرض تفاصيل المجلد الخاص به.

#### تحميل ملف PST

حدد المسار إلى ملف PST الخاص بك وقم بتحميله باستخدام `PersonalStorage.FromFile` طريقة:
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";
string dst = dataDir + "/PersonalStorage.pst";

// تحميل ملف PST
PersonalStorage personalStorage = PersonalStorage.FromFile(dst);
```

#### الحصول على جميع المجلدات الفرعية

استرداد كافة المجلدات الفرعية في المجلد الجذر لملف PST:
```csharp
FolderInfoCollection folderInfoCollection = personalStorage.RootFolder.GetSubFolders();
```

#### تكرار وعرض معلومات المجلد

قم بالتكرار على كل مجلد لعرض اسمه وإجمالي عدد العناصر وعدد العناصر غير المقروءة:
```csharp
foreach (FolderInfo folderInfo in folderInfoCollection)
{
    Console.WriteLine("Folder: " + folderInfo.DisplayName);
    Console.WriteLine("Total items: " + folderInfo.ContentCount);
    Console.WriteLine("Total unread items: " + folderInfo.ContentUnreadCount);
    Console.WriteLine("-----------------------------------");
}
```

**توضيح:**
- `folderInfo.DisplayName`:استرجاع اسم المجلد.
- `folderInfo.ContentCount`:يوفر العدد الإجمالي للعناصر الموجودة داخل المجلد.
- `folderInfo.ContentUnreadCount`:يعطي عدد العناصر غير المقروءة.

### نصائح استكشاف الأخطاء وإصلاحها

- **استثناء عدم العثور على الملف**:تأكد من `dataDir` تم تعيينه بشكل صحيح ويشير إلى ملف PST موجود.
