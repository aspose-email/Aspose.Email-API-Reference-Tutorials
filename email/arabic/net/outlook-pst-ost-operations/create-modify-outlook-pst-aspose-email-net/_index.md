### إنشاء ملف Outlook PST باستخدام Aspose.Email لـ .NET
#### ملخص
إنشاء ملف PST جديد ضروري لبدء بيانات جديدة أو أرشفتها. يرشدك هذا القسم إلى كيفية إنشاء ملف PST بسيط لبرنامج Outlook باستخدام Aspose.Email لـ .NET.

#### الخطوة 1: تحديد مسار الدليل الخاص بك
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY"; 
string dst = Path.Combine(dataDir, "PersonalStorage.pst");
```
**توضيح**:حدد مكان حفظ ملف PST الجديد. تأكد من وجود الدليل أو معالجة إنشاء المسار في الكود الخاص بك.

#### الخطوة 2: التحقق من الملف الموجود وحذفه
```csharp
if (File.Exists(dst))
    File.Delete(dst);
```
**لماذا**:يضمن لك هذا البدء بملف جديد، وتجنب التعارضات مع أي بيانات موجودة.

#### الخطوة 3: إنشاء ملف PST جديد
```csharp
PersonalStorage pst = PersonalStorage.Create(dst, FileFormatVersion.Unicode);
```
**حدود**: 
- `dst`:مسار الوجهة لملف PST الجديد.
- `FileFormatVersion.Unicode`:يضمن التوافق ويدعم أحرف Unicode.

### إضافة مجلد فرعي إلى ملف PST موجود
#### ملخص
يُعد تنظيم ملف PST بمجلدات فرعية مثل "البريد الوارد" أمرًا بالغ الأهمية لإدارة بريدك الإلكتروني بكفاءة. يوضح هذا القسم كيفية إضافة مجلد فرعي برمجيًا.

#### الخطوة 1: فتح ملف PST موجود
```csharp
string dst = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "PersonalStorage.pst");
PersonalStorage pst = PersonalStorage.FromFile(dst);
```
**توضيح**: قم بالوصول إلى ملف PST الذي أنشأته أو لديك بالفعل. تأكد من إمكانية الوصول إليه وسلامته.

#### الخطوة 2: إضافة مجلد فرعي باسم "البريد الوارد"
```csharp
pst.RootFolder.AddSubFolder("Inbox");
```
**غاية**:ينشئ مجلدًا فرعيًا جديدًا أسفل جذر ملف PST الخاص بك، مما يساعد في تنظيم رسائل البريد الإلكتروني في فئات مثل "البريد الوارد".
