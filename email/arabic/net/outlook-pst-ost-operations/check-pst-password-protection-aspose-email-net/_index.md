### التحقق من ميزة حماية كلمة المرور
تساعد هذه الميزة في تحديد ما إذا كان ملف PST يحتوي على كلمة مرور معينة، مما يضمن التعامل الآمن مع البيانات.

#### التنفيذ خطوة بخطوة

**1. افتح ملف PST**
افتح ملف PST من الدليل الخاص به:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (PersonalStorage pst = PersonalStorage.FromFile(dataDir + "/passwordprotectedPST.pst"))
{
    bool isProtected = IsPasswordProtected(pst);
}
```

**2. تحديد حالة حماية كلمة المرور**
ال `IsPasswordProtected` تتحقق الطريقة من وجود خاصية كلمة المرور:

```csharp
private static bool IsPasswordProtected(PersonalStorage pst)
{
    // تحقق مما إذا كان ملف PST يحتوي على خاصية كلمة مرور محددة
    if (pst.Store.Properties.ContainsKey(MapiPropertyTag.PR_PST_PASSWORD))
    {
        long passwordHash = pst.Store.Properties[MapiPropertyTag.PR_PST_PASSWORD].GetLong();
        return passwordHash != 0; // إذا كانت التجزئة غير صفرية، فهي محمية
    }
    return false;
}
```

- **حدود:** ال `PersonalStorage` مثال يمثل ملف PST.
- **قيمة الإرجاع:** قيمة منطقية تشير إلى ما إذا تم تعيين كلمة مرور أم لا.

**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من أن المسار إلى ملف PST الخاص بك صحيح.
- تأكد من أن لديك أذونات القراءة للملف.
