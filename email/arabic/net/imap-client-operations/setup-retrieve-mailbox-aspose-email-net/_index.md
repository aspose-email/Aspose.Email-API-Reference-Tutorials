## دليل التنفيذ

سنقوم بتقسيم هذا التنفيذ إلى ميزتين رئيسيتين: إنشاء `ExchangeClient` المثيل واسترداد معلومات صندوق البريد.

### الميزة 1: إنشاء مثيل ExchangeClient

#### ملخص
يرشدك هذا القسم خلال عملية تهيئة `ExchangeClient`، بمثابة البوابة الخاصة بك للتفاعل مع وظائف خادم Exchange.

#### التنفيذ خطوة بخطوة

**تهيئة بيانات الاعتماد:**
ابدأ بإعداد بيانات اعتماد الاتصال الخاصة بك، بما في ذلك عنوان URL الخاص بالخادم، واسم المستخدم، وكلمة المرور، والنطاق.

```csharp
using Aspose.Email.Clients.Exchange;

// تحديد معلمات الاتصال لخادم Exchange
string serverUrl = "https://اسم الجهاز/التبادل/اسم المستخدم";
string username = "Username";
string password = "password";
string domain = "domain";

// إنشاء مثيل لفئة ExchangeClient
ExchangeClient client = new ExchangeClient(serverUrl, username, password, domain);
```

**توضيح:**
- `serverUrl`:عنوان URL لخادم Exchange الخاص بك. 
- `username`، `password`، و `domain`:البيانات المطلوبة للمصادقة.

### الميزة 2: الحصول على معلومات صندوق البريد من Exchange Server

#### ملخص
تعلم كيفية استخدام `ExchangeClient` مثال لاسترداد معلومات صندوق البريد.

#### التنفيذ خطوة بخطوة

**استرداد حجم صندوق البريد والمعلومات التفصيلية:**
استخدم `GetMailboxSize()` و `GetMailboxInfo()` طرق للحصول على تفاصيل شاملة لصندوق البريد.

```csharp
try
{
    // احصل على حجم صندوق البريد بالبايت
    long mailboxSize = client.GetMailboxSize();

    // استرجاع معلومات مفصلة عن صندوق البريد
    ExchangeMailboxInfo mailboxInfo = client.GetMailboxInfo();
    
    // أمثلة على عناوين URI للتوضيح (استبدالها بالمسارات الفعلية)
    string inboxUri = mailboxInfo.InboxUri;
    string sentItemsUri = mailboxInfo.SentItemsUri;
    string draftsUri = mailboxInfo.DraftsUri;
}
catch (Exception ex)
{
    throw new Exception("An error occurred while retrieving mailbox information: " + ex.Message);
}
```

**توضيح:**
- `GetMailboxSize()`:يستعيد الحجم الحالي لصندوق البريد الخاص بك بالبايت.
- `GetMailboxInfo()`:يوفر معلومات مفصلة، بما في ذلك عناوين URI لمجلدات مختلفة مثل البريد الوارد، والعناصر المرسلة، والمسودات.
