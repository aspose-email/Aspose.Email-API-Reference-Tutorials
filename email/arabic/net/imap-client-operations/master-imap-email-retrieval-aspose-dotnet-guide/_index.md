### التهيئة والإعداد الأساسي
لبدء استخدام Aspose.Email، ستحتاج أولًا إلى تهيئة ImapClient. إليك كيفية إعداده باستخدام إعدادات الاتصال الآمن:
```csharp
using Aspose.Email.Clients;
using Aspose.Email.Clients.Imap;

public static void InitializeImapClient()
{
    ImapClient imapClient = new ImapClient();
    imapClient.Host = "<HOST>";
    imapClient.Port = 993; // منفذ مشترك لاتصالات SSL
    imapClient.Username = "<USERNAME>";

    imapClient.SupportedEncryption = EncryptionProtocols.Tls;
    imapClient.SecurityOptions = SecurityOptions.SSLImplicit;
}
```

## دليل التنفيذ

### تهيئة ImapClient
تهيئة `ImapClient` ضروري لإعداد اتصال آمن بخادم IMAP. إليك كيفية تكوينه:

#### إعداد المضيف والمنفذ
حدد مضيف خادم IMAP ورقم المنفذ:
- **يستضيف**:استخدم اسم المجال أو عنوان IP الخاص بمزود البريد الإلكتروني الخاص بك.
- **ميناء**:عادةً ما يتم استخدام 993 لاتصالات SSL.
```csharp
imapClient.Host = "<HOST>";
imapClient.Port = 993;
```

#### تفاصيل المصادقة
أدخل اسم المستخدم وكلمة المرور للمصادقة. هذا يسمح لك بالوصول إلى حساب بريدك الإلكتروني.
```csharp
imapClient.Username = "<USERNAME>";
imapClient.Password = "<PASSWORD>";
```

#### بروتوكول التشفير
تأكد من تأمين الاتصال عن طريق ضبط بروتوكول التشفير المدعوم:
```csharp
imapClient.SupportedEncryption = EncryptionProtocols.Tls;
```

### قائمة الرسائل من خادم IMAP
بمجرد الاتصال، يمكنك إدراج جميع الرسائل المتوفرة في صندوق الوارد الخاص بك:

#### استرداد مجموعة الرسائل
يستخدم `ListMessages` للحصول على مجموعة من معلومات الرسالة:
```csharp
ImapMessageInfoCollection messageInfoCol = imapClient.ListMessages();
int[] sequenceNumberAr = messageInfoCol.Select(mi => mi.SequenceNumber).ToArray();
string[] uniqueIdAr = messageInfoCol.Select(mi => mi.UniqueId).ToArray();
```

### جلب الرسائل حسب رقم التسلسل
لجلب رسائل بريد إلكتروني محددة، يمكنك استخدام أرقام التسلسل الخاصة بها:

#### جلب البيانات باستخدام أرقام التسلسل
مرر أرقام التسلسل المطلوبة إلى `FetchMessages`:
```csharp
IList<MailMessage> fetchedMessages = imapClient.FetchMessages(sequenceNumbers);
```

بدلاً من ذلك، قم بجلب الرسائل باستخدام معرفات فريدة:

#### استرداد رسائل البريد الإلكتروني عن طريق معرف فريد
