### الاتصال بخادم IMAP

**ملخص**:توضح هذه الميزة كيفية إنشاء اتصال باستخدام Aspose.Email للتفاعل مع صندوق الوارد الخاص بالبريد الإلكتروني.

#### الخطوة 1: إعداد معلمات الاتصال
```csharp
using Aspose.Email.Clients.Imap;

const string host = "your_imap_host"; // استبدله بمضيف خادم IMAP الخاص بك
const int port = 143; // منفذ IMAP القياسي
const string username = "user@host.com"; // عنوان بريدك  الإلكتروني
const string password = "password"; // كلمة مرور بريدك الإلكتروني

ImapClient client = new ImapClient(host, port, username, password);
```

#### الخطوة 2: حدد مجلد البريد الوارد
```csharp
try
{
    client.SelectFolder("Inbox");
}
catch (Exception ex)
{
    Console.WriteLine(ex.Message);
}
finally
{
    client.Dispose(); // التخلص بشكل صحيح من العميل لتحرير الموارد
}
```
**توضيح**:يحدد هذا المقطع مجلد "البريد الوارد"، مما يسمح بإجراء عمليات إضافية مثل قراءة رسائل البريد الإلكتروني أو تصفيتها. `try-catch-finally` تضمن الكتلة أن يتم التعامل مع الاستثناءات بشكل سليم ويتم إصدار الموارد بشكل صحيح.
