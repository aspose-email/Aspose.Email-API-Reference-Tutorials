## دليل التنفيذ
### إعداد وكيل HTTP (H2)
#### ملخص
يوضح هذا القسم كيفية تكوين وكيل HTTP لاتصالات SMTP الخاصة بك.
##### الخطوة 1: إنشاء مثيل HttpProxy (H3)
إنشاء مثيل جديد من `HttpProxy` مع تفاصيل الوكيل الخاصة بك. تتضمن هذه الخطوة تحديد عنوان الوكيل ورقم المنفذ:
```csharp
// إنشاء مثيل لـ HttpProxy مع العنوان والمنفذ.
HttpProxy proxy = new HttpProxy("18.222.124.59", 8080);
```
**لماذا؟** يعمل الوكيل كوسيط، مما يسمح لتطبيقك بالتواصل عبر SMTP مع الالتزام بقيود الشبكة.
### تكوين عميل SMTP (H2)
#### ملخص
بعد ذلك، سنقوم بتكوين Aspose.Email `SmtpClient` استخدام بيانات الاعتماد ودمجها مع وكيل HTTP الذي تم إعداده مسبقًا.
##### الخطوة 1: تهيئة SmtpClient (H3)
ابدأ بتهيئة عميل SMTP الخاص بك باستخدام تفاصيل الخادم الضرورية:
```csharp
// استبدال العناصر النائبة بالقيم الفعلية.
SmtpClient client = new SmtpClient("YOUR_SMTP_SERVER", 587, "username", "password");
```
**لماذا؟** يؤدي هذا إلى إنشاء الأساس لإرسال رسائل البريد الإلكتروني عبر خادم SMTP محدد.
##### الخطوة 2: تعيين الوكيل (H3)
بمجرد التهيئة، قم بتعيين `HttpProxy` مثال إلى عميل SMTP:
```csharp
// تعيين الوكيل للعميل.
client.Proxy = proxy;
```
**لماذا؟** من خلال تعيين الوكيل، يمكنك التأكد من توجيه جميع طلبات SMTP الصادرة من خلاله.
### إرسال بريد إلكتروني (H2)
#### ملخص
أخيرًا، دعنا نرسل بريدًا إلكترونيًا باستخدام عميل SMTP الذي قمنا بتكوينه مع وكيل.
##### الخطوة 1: إنشاء مثيل MailMessage (H3)
إنشاء جديد `MailMessage` مثال لتحديد المرسل والمستلم والموضوع ونص رسالتك الإلكترونية:
```csharp
// إنشاء رسالة البريد.
MailMessage mailMessage = new MailMessage(
    "from@domain.com",
    "to@domain.com",
    "NETWORKNET-34226 - " + Guid.NewGuid().ToString(),
    "This is a test email sent through an HTTP proxy."
);
```
**لماذا؟** `MailMessage` يتضمن كل المعلومات اللازمة لإرسال بريد إلكتروني.
##### الخطوة 2: إرسال البريد الإلكتروني (H3)
استخدم عميل SMTP لإرسال رسالتك:
```csharp
// إرسال البريد الإلكتروني.
client.Send(mailMessage);
```
**لماذا؟** يستخدم هذا الإجراء كلاً من خادم SMTP وإعدادات الوكيل لتسليم بريدك الإلكتروني بنجاح.
