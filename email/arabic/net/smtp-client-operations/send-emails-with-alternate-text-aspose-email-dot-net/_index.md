## دليل التنفيذ

### إرسال بريد إلكتروني مع ميزة النص البديل

#### ملخص
تتيح لك هذه الميزة إرسال رسائل البريد الإلكتروني بمحتوى HTML وبدائل النص العادي باستخدام Aspose.Email لـ .NET، مما يضمن التوافق عبر جميع عملاء البريد الإلكتروني والأجهزة.

#### التنفيذ خطوة بخطوة

**1. تهيئة رسالة البريد**

ابدأ بإنشاء مثيل لـ `MailMessage` قم بإعداد المرسل والمستقبل ونص الرسالة:

```csharp
using System;
using Aspose.Email.Mime;
using Aspose.Email.Clients.Smtp;

class SendEmailWithAlternateTextFeature
{
    public static void Execute()
    {
        // إنشاء مثيل MailMessage جديد
        MailMessage message = new MailMessage();
        
        // تعيين عنوان "من" وعنوان البريد الإلكتروني للمستلم
        message.From = "sender@sender.com";
        message.To.Add("receiver@receiver.com");

        // إضافة نص عادي
        message.Body = "This is Plain Text Body";

        // إضافة عرض بديل HTML للعملاء الذين يدعمونه
        AlternateView alternateView = AlternateView.CreateAlternateViewFromString(
            "<html><body>This is the <b>HTML</b> version of the email.</body></html>",
            null,
            MediaTypeNames.Text.Html);
        message.AlternateViews.Add(alternateView);
    }
}
```

**2. تكوين عميل SMTP**

قم بإعداد `SmtpClient` مع تفاصيل الخادم لإرسال البريد الإلكتروني:

```csharp
// إنشاء وتكوين مثيل لـ SmtpClient
SmtpClient client = new SmtpClient();
client.Host = "smtp.server.com";  // استبدله بخادم مضيف SMTP الخاص بك
client.Username = "Username";     // اسم المستخدم للمصادقة الخاص بك
client.Password = "Password";     // كلمة مرور المصادقة الخاصة بك
client.Port = 25;                 // عادةً، المنفذ الافتراضي هو 25

try
{
    // إرسال رسالة البريد الإلكتروني باستخدام طريقة SmtpClient.Send
    client.Send(message);
}
catch (Exception ex)
{
    // التعامل مع الاستثناءات أثناء الإرسال
    System.Diagnostics.Trace.WriteLine(ex.ToString());
}
```

### تكوين عميل البريد الإلكتروني لإرسال رسائل البريد الإلكتروني

#### ملخص
يوضح هذا القسم كيفية تكوين عميل SMTP لإرسال رسائل البريد الإلكتروني.

**1. تهيئة وتعيين تفاصيل الخادم**

إنشاء جديد `SmtpClient` قم بإعداد تفاصيل الخادم الضرورية:

```csharp
using Aspose.Email.Clients.Smtp;

class EmailClientConfigurationFeature
{
    public static void Execute()
    {
        SmtpClient client = new SmtpClient();
        client.Host = "smtp.server.com";  // عنوان خادم مضيف SMTP
        client.Username = "Username";     // اسم المستخدم للمصادقة مع الخادم
        client.Password = "Password";     // كلمة المرور للمصادقة مع الخادم
        client.Port = 25;                 // رقم المنفذ الذي يستخدمه خادم SMTP (الافتراضي هو 25 عادةً)
    }
}
```
