```csharp

// إضافة عناوين متعددة إلى
```

### تكوين عميل SMTP

تتضمن هذه الخطوة إعداد `SmtpClient` إرسال رسائل البريد الإلكتروني عبر خادم محدد.

#### تهيئة وتكوين SmtpClient
```csharp
// إنشاء عميل SMTP وتكوينه
SmtpClient client = new SmtpClient("smtp.gmail.com", 587, "your.email@gmail.com", "your.password");
client.SecurityOptions = SecurityOptions.Auto; // يتم تحديد خيارات الأمان تلقائيًا استنادًا إلى إمكانيات الخادم.
```
