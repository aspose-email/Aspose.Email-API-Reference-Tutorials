## دليل التنفيذ

### الاتصال وتسجيل الدخول إلى خادم IMAP

#### ملخص
الاتصال بخادم IMAP هو الخطوة الأولى لإدارة رسائل البريد الإلكتروني برمجيًا. سنستخدم Aspose.Email `ImapClient` صف لهذا الغرض.

**الخطوة 1: تحديد بيانات الاعتماد**
ابدأ بتحديد بيانات اعتماد خادم IMAP الخاص بك:
```csharp
const string host = "your-imap-host";
const int port = 143; // منفذ IMAP الافتراضي
const string username = "user@host.com";
const string password = "password";
```

**الخطوة 2: إنشاء ImapClient واستخدامه**
إنشاء مثيل لـ `ImapClient` الفئة التي تستخدم هذه البيانات:
```csharp
using (ImapClient client = new ImapClient(host, port, username, password))
{
    Console.WriteLine("Connected and logged in to IMAP server.");
}
```

**نصائح لاستكشاف الأخطاء وإصلاحها**تأكد من أن شبكتك تسمح بالاتصالات عبر منفذ IMAP المحدد. إذا واجهت مشاكل في المصادقة، فتحقق جيدًا من بيانات اعتمادك.

### حدد مجلد IMAP

#### ملخص
بمجرد الاتصال، يعد تحديد مجلد مثل Inbox ضروريًا لإجراء العمليات داخله.

**الخطوة 1: الاتصال بالخادم**
إعادة استخدامنا `ImapClient`، قم بالاتصال كما هو موضح سابقًا:
```csharp
using (ImapClient client = new ImapClient("your-imap-host", 143, "user@host.com", "password"))
{
    // حدد مجلد البريد الوارد
    client.SelectFolder(ImapFolderInfo.InBox);
    Console.WriteLine("Inbox folder selected.");
}
```

### إنشاء وتنفيذ استعلام بحث IMAP

#### ملخص
يُعد البحث عن رسائل بريد إلكتروني محددة مهمة شائعة. سنوضح كيفية إنشاء استعلام بحث IMAP وتنفيذه.

**الخطوة 1: إنشاء ImapQueryBuilder**
استخدم `ImapQueryBuilder` لتحديد معايير البحث الخاصة بك:
```csharp
ImapQueryBuilder builder = new ImapQueryBuilder();
builder.Subject.Contains("Newsletter"); // تصفية حسب سطر الموضوع
builder.InternalDate.On(DateTime.Now);  // رسائل البريد الإلكتروني التي وردت اليوم
```

**الخطوة 2: تنفيذ استعلام البحث**
استخدم الاستعلام لاسترداد الرسائل:
```csharp
MailQuery query = builder.GetQuery();
ImapMessageInfoCollection messages = client.ListMessages(query);
Console.WriteLine($"Found {messages.Count} message(s) in Inbox.");
```

## قسم الأسئلة الشائعة
1. **ما هو رقم المنفذ الافتراضي لـ IMAP؟**
المنفذ الافتراضي هو 143، ولكن الاتصالات الآمنة تستخدم عادةً المنفذ 993.
2. **كيف أتعامل مع SSL/TLS مع Aspose.Email؟**
قم بتكوين `ImapClient` لتفعيل SSL حسب الحاجة: `client.SecurityOptions = SecurityOptions.Auto;`
3. **هل يمكنني البحث عن رسائل البريد الإلكتروني الأقدم من اليوم؟**
نعم، اضبط `InternalDate.On` الطريقة أو استخدام نطاقات التاريخ في `ImapQueryBuilder`.
4. **ماذا لو كان خادم IMAP الخاص بي يتطلب المصادقة عبر OAuth2؟**
يدعم Aspose.Email بروتوكول OAuth2. نفّذ الخطوات اللازمة للمصادقة باستخدام رموز OAuth.
5. **كيف أتعامل مع كميات كبيرة من رسائل البريد الإلكتروني بكفاءة؟**
استخدم التجميع وتحسين استعلاماتك لمعالجة رسائل البريد الإلكتروني في أجزاء قابلة للإدارة.
