### قراءة حالات المستلمين
#### ملخص
تتيح لك هذه الميزة الوصول إلى حالة تتبع كل مستلم وطباعتها في رسالة MAPI. وهي مفيدة لإدارة الردود على طلبات الاجتماعات بكفاءة.
##### الخطوة 1: تحميل رسالة MAPI
ابدأ بتحميل ملف طلب الاجتماع الخاص بك إلى `MapiMessage` هدف:
```csharp
// تأكد من أن هذا المسار يشير إلى ملف .msg الفعلي الخاص بك.
string filePath = @"YOUR_DOCUMENT_DIRECTORY\Test Meeting.msg";
MapiMessage message = MapiMessage.FromFile(filePath);
```
##### الخطوة 2: التكرار على المستلمين
قم بالتنقل عبر كل مستلم في الملف المحمل `MapiMessage` وطباعة حالة التتبع الخاصة بهم:
```csharp
foreach (MapiRecipient recipient in message.Recipients)
{
    // اطبع حالة التتبع لكل مستلم.
    Console.WriteLine(recipient.RecipientTrackStatus);
}
```
**توضيح**: ال `RecipientTrackStatus` توفر الخاصية رؤى حول ما إذا كان المستلم قد قبل أو رفض أو لم يستجب لطلب الاجتماع الخاص بك.
