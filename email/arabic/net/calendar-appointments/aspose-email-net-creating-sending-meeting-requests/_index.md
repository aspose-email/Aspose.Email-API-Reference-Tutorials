### إنشاء طلبات الاجتماعات وإرسالها عبر البريد الإلكتروني

#### ملخص
يتضمن إنشاء طلب اجتماع إعداد رسالة بريد إلكتروني تتضمن تفاصيل الموعد باستخدام Aspose.Email. تُؤتمت هذه الميزة عملية إرفاق دعوات التقويم برسائل البريد الإلكتروني.

#### التنفيذ خطوة بخطوة:

##### 1. إعداد MailMessage

ابدأ بإنشاء `MailMessage` مثال، والذي سيكون بمثابة حاوية البريد الإلكتروني الخاص بك:

```csharp
MailMessage msg = new MailMessage();
msg.From = "newcustomeronnet@gmail.com";
msg.To = "person1@domain.com, person2@domain.com, person3@domain.com, asposetest123@gmail.com";
```

##### 2. إنشاء موعد

إنشاء `Appointment` مثال مع التفاصيل اللازمة:

```csharp
Appointment app = new Appointment(
    "Room 112",
    new DateTime(2015, 7, 17, 13, 0, 0),
    new DateTime(2015, 7, 17, 14, 0, 0),
    msg.From,
    msg.To);
```

##### 3. تكوين تفاصيل الاجتماع

تعيين ملخص ووصف للاجتماع:

```csharp
app.Summary = "Release Meeting";
app.Description = "Discuss the next release";
```

##### 4. إرفاق الموعد بالبريد الإلكتروني

أضف الموعد كعرض بديل في رسالة البريد الإلكتروني الخاصة بك:

```csharp

```

### تكوين عميل SMTP لإرسال رسائل البريد الإلكتروني

#### ملخص
لإرسال رسائل البريد الإلكتروني، قم بتكوين `SmtpClient` مع تفاصيل خادم SMTP وبيانات الاعتماد الخاصة بك.

#### التنفيذ خطوة بخطوة:

##### 1. تكوين SmtpClient

إنشاء طريقة لإرجاع تكوين `SmtpClient`:

```csharp
private static SmtpClient GetSmtpClient()
{
    SmtpClient client = new SmtpClient(
        "smtp.gmail.com", 587,
        "your.email@gmail.com",
        "your.password");
    
    client.SecurityOptions = SecurityOptions.Auto;
    return client;
}
```

##### 2. أرسل البريد الإلكتروني

استخدم `try-catch` كتلة للتعامل مع الاستثناءات المحتملة عند الإرسال:

```csharp
SmtpClient client = GetSmtpClient();
try
{
    client.Send(msg);
}
catch (Exception ex)
{
    Console.WriteLine(ex.ToString());
}
```
