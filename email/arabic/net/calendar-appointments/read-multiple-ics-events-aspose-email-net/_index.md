#### الخطوة 2: تكرار الأحداث وإضافتها إلى القائمة

كرر كل حدث في `.ics` الملف باستخدام حلقة، وإضافته إلى القائمة الخاصة بك:

```csharp
// قم بالتنقل عبر كل حدث في ملف ICS وأضفه إلى القائمة
do {
    var currentEvent = reader.NextEvent();
    if (currentEvent != null)
        appointments.Add(currentEvent);
} while (reader.NextEvent() != null);
```

**توضيح**: ال `NextEvent()` تتكرر الطريقة عبر الأحداث، وتضمن الحلقة التقاط جميع المواعيد بكفاءة.
