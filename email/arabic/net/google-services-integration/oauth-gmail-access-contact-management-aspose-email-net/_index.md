# دمج الوصول إلى Gmail عبر OAuth وإدارة جهات الاتصال مع Aspose.Email لـ .NET

## مقدمة

هل تبحث عن دمج مصادقة حساب جوجل وإدارة جهات الاتصال بسلاسة في تطبيق .NET الخاص بك؟ أنت في المكان المناسب! في هذا البرنامج التعليمي الشامل، سنرشدك خلال استخدام Aspose.Email لـ .NET لاسترداد رموز OAuth وإدارة جهات اتصال Gmail. سواء كنت تُنشئ عميل بريد إلكتروني مخصصًا أو تُؤتمت سير عمل البريد الإلكتروني، فإن إتقان هذه الوظائف سيكون مفيدًا للغاية.

**ما سوف تتعلمه:**
- كيفية استرداد رمز وصول OAuth ورمز التحديث باستخدام Aspose.Email لـ .NET.
- كيفية تهيئة عميل Gmail باستخدام الرمز المميز الذي قمت باسترداده.
- تقنيات جلب جهات الاتصال وحفظها من حساب Gmail بتنسيقات MSG وVCF.

لنبدأ بإعداد المتطلبات الأساسية!

- فهم أساسي لبرمجة C#.
- المعرفة بواجهات برمجة تطبيقات Google ومفاهيم OAuth 2.0.

## دليل التنفيذ

الآن بعد أن تم إعداد كل شيء، دعنا نتعمق في تنفيذ ميزاتنا خطوة بخطوة. 

### استرداد رمز وصول OAuth

#### ملخص
يتيح لك استرداد رمز الوصول ورمز التحديث التواصل الآمن مع خدمات Google باستخدام بيانات اعتماد التطبيق لديك.

**الخطوة 1: إنشاء بيانات اعتماد المستخدم**
```csharp
GoogleTestUser user = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
string accessToken;
string refreshToken;
```
- **شرح المعلمات**:استبدال العناصر النائبة بتفاصيل المستخدم الفعلية وبيانات اعتماد عميل OAuth.
  
**الخطوة 2: استرداد رموز الوصول والتحديث**
```csharp
GoogleOAuthHelper.GetAccessToken(user, out accessToken, out refreshToken);
```
- **الطريقة والغرض**:تقوم هذه الطريقة بمصادقة المستخدم وإرجاع الرموز اللازمة لمكالمات API اللاحقة.

### تهيئة عميل Gmail

#### ملخص
مع رمز الوصول الخاص بك في متناول اليد، قم ببدء تشغيل `GmailClient` لإجراء عمليات مختلفة على حسابات Gmail.

**الخطوة 3: تهيئة IGmailClient**
```csharp
using (IGmailClient client = GmailClient.GetInstance(accessToken, user.EMail))
{
    // يمكنك الآن استخدام كائن العميل للعمليات الإضافية.
}
```
- **تكوين المفتاح**:استخدم رمز الوصول المسترد والبريد الإلكتروني لإنشاء العميل.

### جلب جهات الاتصال وحفظها من Gmail

#### ملخص
قم بالوصول إلى جهات الاتصال وحفظها بالتنسيقات المطلوبة باستخدام إمكانيات Aspose.Email.

**الخطوة 4: جلب جميع جهات الاتصال**
```csharp
Contact[] contacts = client.GetAllContacts();
```
- **توضيح**:استرجاع جميع جهات الاتصال المتوفرة ضمن حساب Google المعتمد.

**الخطوة 5: حفظ جهة اتصال محددة بتنسيق MSG وVCF**
```csharp
string dataDir = "@YOUR_DOCUMENT_DIRECTORY";
string outputDir = "@YOUR_OUTPUT_DIRECTORY";

// افترض أن جهة الاتصال [0] هي جهة الاتصال المطلوبة
Contact contact = contacts[0];

contact.Save(outputDir + "/contact_out.msg", ContactSaveFormat.Msg);
contact.Save(outputDir + "/contact_out.vcf", ContactSaveFormat.VCard);
```
- **حدود**:تحديد الدلائل لقراءة الملفات وحفظها.
- **شرح التنسيقات**:MSG هو تنسيق Microsoft Outlook، في حين أن VCF (vCard) مدعوم على نطاق واسع.

- تأكد من صحة بيانات اعتماد OAuth.
- تأكد من إعادة التحقق من مسارات الدليل لعمليات القراءة/الكتابة.
