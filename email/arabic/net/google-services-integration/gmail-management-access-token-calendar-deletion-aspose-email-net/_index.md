## دليل التنفيذ

دعونا نقسم التنفيذ إلى ميزات مميزة:

### ميزة استرداد رمز الوصول
#### ملخص
توضح هذه الميزة الحصول على رمز الوصول ورمز التحديث باستخدام Aspose.Email لـ .NET، مما يتيح الوصول الآمن إلى خدمة Gmail.

**الخطوة 1: تهيئة بيانات اعتماد المستخدم**
قم بتحديد بيانات اعتماد المستخدم بما في ذلك البريد الإلكتروني ومعرف العميل وسر العميل، وهي بيانات مهمة لمصادقة OAuth.
```csharp
GoogleTestUser User = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
```

**الخطوة 2: استرداد الرموز**
استخدم `GetAccessToken` طريقة لجلب رموز الوصول والتحديث، وهي ضرورية للطلبات التي تم التحقق من صحتها.
```csharp
string accessToken;
string refreshToken;
GoogleOAuthHelper.GetAccessToken(User, out accessToken, out refreshToken);
```
- **حدود:** بيانات اعتماد المستخدم مغلفة في `GoogleTestUser` هدف.
- **قيم العودة:** سلاسل رمز الوصول ورمز التحديث.

### حذف ميزة تقويم محددة
#### ملخص
تتيح لك هذه الميزة الوصول إلى حساب Gmail باستخدام رمز الوصول وحذف التقويمات استنادًا إلى بادئات ملخص محددة.

**الخطوة 1: تهيئة عميل Gmail**
إنشاء `GmailClient` مثيل مع رمز الوصول المسترد، وهو ضروري لمكالمات واجهة برمجة التطبيقات المصادق عليها.
```csharp
using (IGmailClient client = GmailClient.GetInstance(accessToken, User.EMail))
```

**الخطوة 2: تحديد التقويمات وحذفها**
جلب كافة التقويمات وحذف تلك التي تتطابق ملخصاتها مع بادئة محددة.
```csharp
string summaryPrefix = "Calendar summary - ";
ExtendedCalendar[] calendars = client.ListCalendars();
foreach (ExtendedCalendar calendar in calendars)
{
    if (calendar.Summary.StartsWith(summaryPrefix))
        client.DeleteCalendar(calendar.Id);
}
```
- **حدود:** رمز الوصول للمصادقة والبريد الإلكتروني للمستخدم.
- **التكوينات الرئيسية:** بادئة الملخص المستخدمة لتحديد التقويمات المستهدفة.
