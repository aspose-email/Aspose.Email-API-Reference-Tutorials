## دليل التنفيذ
الآن بعد أن قمت بإعداد Aspose.Email لـ .NET، فلنبدأ في تنفيذ ميزاته.

### المصادقة باستخدام Google API
**ملخص:** المصادقة هي الخطوة الأولى للوصول إلى بيانات تقويم جوجل. باستخدام Aspose.Email، احصل على رموز الوصول والتحديث بكفاءة.

#### التنفيذ خطوة بخطوة
1. **إنشاء مستخدم اختبار:**
   ```csharp
   GoogleTestUser user = new GoogleTestUser("user", "email address", "password", "clientId", "client secret");
   ```
2. **احصل على رموز الوصول والتحديث:**
   ```csharp
   string accessToken;
   string refreshToken;
   GoogleOAuthHelper.GetAccessToken(user, out accessToken, out refreshToken);
   ```

### قائمة التقويمات واسترجاع المواعيد
**ملخص:** تساعد قائمة التقويمات على تحديد التقويم الذي يجب العمل معه، بينما يسمح استرداد المواعيد بالإدارة التفصيلية.

#### التنفيذ خطوة بخطوة
1. **تهيئة عميل Gmail:**
   ```csharp
   using (IGmailClient client = GmailClient.GetInstance(accessToken, user.EMail))
   {
       string sourceCalendarId = client.ListCalendars()[0].Id;
   }
   ```
2. **استرجاع المواعيد من التقويم:**
   ```csharp
   Appointment[] appointmentsBeforeMove = client.ListAppointments(sourceCalendarId);
   ```

### نقل موعد بين التقويمات
**ملخص:** يعد نقل المواعيد أمرًا ضروريًا لإعادة تنظيم المهام عبر التقويمات المختلفة.

#### التنفيذ خطوة بخطوة
1. **احصل على معرف فريد للموعد:**
   ```csharp
   string targetAppointmentUniqueId = client.ListAppointments(sourceCalendarId)[0].UniqueId;
   ```
2. **نقل الموعد:**
   ```csharp
   Appointment movedAppointment = client.MoveAppointment(sourceCalendarId, destinationCalendarId, targetAppointmentUniqueId);
   ```

### حذف موعد من التقويم
**ملخص:** يساعد حذف المواعيد غير الضرورية في الحفاظ على تقويم نظيف ومنظم.

#### التنفيذ خطوة بخطوة
1. **حذف الموعد:**
   ```csharp
   client.DeleteAppointment(destinationCalendarId, movedAppointment.UniqueId);
   ```
2. **تأكيد الحذف:**
   ```csharp
   Appointment[] appointmentsAfterDeletion = client.ListAppointments(destinationCalendarId);
   ```
