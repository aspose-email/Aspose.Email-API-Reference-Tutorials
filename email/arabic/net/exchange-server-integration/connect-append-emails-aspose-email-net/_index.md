## دليل التنفيذ

### الاتصال بخادم IMAP

#### ملخص
يغطي هذا القسم إعداد اتصال بخادم IMAP الخاص بك باستخدام `ImapClient`.

#### دليل خطوة بخطوة

**1. تكوين ImapClient**
إنشاء وتكوين `ImapClient` مثال مع تفاصيل الخادم الخاص بك:

```csharp
using Aspose.Email.Clients;

ImapClient client = new ImapClient();
client.Host = "imap.gmail.com"; // تحديد مضيف خادم IMAP
client.Username = "your.username@gmail.com"; // قم بتعيين اسم المستخدم للبريد الإلكتروني الخاص بك
client.Password = "your.password"; // تعيين كلمة مرور البريد الإلكتروني الخاص بك
client.Port = 993; // المنفذ القياسي لاتصالات SSL
client.SecurityOptions = SecurityOptions.Auto; // تحديد خيارات الأمان تلقائيًا
```

**توضيح:**
- `Host` يحدد عنوان خادم IMAP.
- `Username` و `Password` مطلوبة للمصادقة.
- ميناء `993` يتم استخدامه للاتصالات الآمنة (SSL/TLS).
- `SecurityOptions.Auto` يضمن إعدادات الأمان المثالية.

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن الشبكة الخاصة بك تسمح بالاتصال بالمنفذ 993.
- تأكد من صحة بيانات اعتماد البريد الإلكتروني الخاص بك.

### إنشاء رسالة جديدة وإضافتها إلى مجلد IMAP

#### ملخص
تعرف على كيفية إنشاء رسالة بريد إلكتروني جديدة وإضافتها إلى مجلد البريد الوارد.

#### دليل خطوة بخطوة

**1. إنشاء رسالة بريدية**
إنشاء مثيل جديد من `MailMessage`:

```csharp
using Aspose.Email.Clients.Imap;
using Aspose.Email.Mime;

// إنشاء مثيل MailMessage لرسالة البريد الإلكتروني الجديدة
MailMessage msg = new MailMessage("user@domain1.com", "user@domain2.com", "subject", "message");
```

**توضيح:**
- `MailMessage` يمثل بريدًا إلكترونيًا يحتوي على تفاصيل المرسل والمستلم والموضوع والنص.

**2. حدد المجلد**
حدد مجلد البريد الوارد:

```csharp
// حدد مجلد البريد الوارد على خادم IMAP
client.SelectFolder(ImapFolderInfo.InBox);
```

**3. إضافة رسالة**
إضافة الرسالة إلى المجلد الحالي:

```csharp
try
{
    // الاشتراك في التغييرات في المجلد الحالي (اختياري)
    client.SubscribeFolder(client.CurrentFolder.Name);

    // إضافة الرسالة التي تم إنشاؤها حديثًا إلى المجلد المحدد
    client.AppendMessage(client.CurrentFolder.Name, msg);
}
catch (Exception ex)
{
    Console.Write(Environment.NewLine + ex.Message);
}
finally
{
    client.Dispose();
}
```

**توضيح:**
- `SelectFolder` تعيين المجلد النشط.
- `AppendMessage` يضيف بريدك الإلكتروني إلى المجلد المختار.
