## دليل التنفيذ

سيرشدك هذا القسم خلال عملية الاتصال بخادم Exchange وإدراج المجلدات الفرعية باستخدام Aspose.Email لـ .NET.

### الاتصال بخادم Exchange باستخدام EWS

**ملخص:**
يتيح الاتصال بخادم Exchange عبر EWS لتطبيقك التفاعل مع بيانات البريد الإلكتروني بسلاسة.

#### الخطوة 1: إنشاء بيانات اعتماد الشبكة
إعداد بيانات الاعتماد المطلوبة للمصادقة مع خادم Exchange:
```csharp
string mailboxUri = "https://التبادل/ews/exchange.asmx";
string domain = @"";
string username = "username@ASE305.onmicrosoft.com";
string password = "password";

NetworkCredential credentials = new NetworkCredential(username, password, domain);
```
*توضيح:* تقوم هذه بيانات الاعتماد بمصادقة تطبيقك على الخادم.

#### الخطوة 2: تهيئة عميل EWS
إنشاء مثيل لـ `IEWSClient` للتواصل مع خادم Exchange:
```csharp
IEWSClient client = EWSClient.GetEWSClient(mailboxUri, credentials);
```
*حدود:* 
- `mailboxUri`:نقطة النهاية لخادم Exchange الخاص بك.
- `credentials`:تفاصيل مصادقة الشبكة الخاصة بك.

### إدراج المجلدات الفرعية في Exchange Server باستخدام EWS

**ملخص:**
تساعدك قائمة المجلدات الفرعية داخل صندوق بريد البريد الإلكتروني على تنظيم رسائل البريد الإلكتروني وإدارتها بشكل فعال.

#### الخطوة 1: استرداد معلومات المجلد الجذر
جلب المجلد الجذر لإدراج مجلداته الفرعية:
```csharp
using Aspose.Email.Clients.Exchange;

ExchangeFolderInfoCollection folderInfoCol = client.ListSubFolders(client.MailboxInfo.RootUri);
```
*قيمة الإرجاع:* مجموعة من `ExchangeFolderInfo` الكائنات التي تمثل كل مجلد فرعي.

#### الخطوة 2: معالجة كل مجلد فرعي
قم بالتكرار خلال المجموعة للوصول إلى تفاصيل كل مجلد:
```csharp
foreach (ExchangeFolderInfo folderInfo in folderInfoCol)
{
    // معالجة كل مجلد على أساس نوعه
}
```
*توضيح:* تتيح لك هذه الحلقة التعامل مع المجلدات بشكل فردي، وتخصيص الإجراءات حسب الحاجة.

### تحديد أنواع المجلدات باستخدام EWS

**ملخص:**
يساعد فهم أنواع المجلدات على تخصيص وظائف محددة لفئات البريد الإلكتروني المختلفة.

#### الخطوة 1: تحديد نوع المجلد
استخدم عبارة switch-case لتحديد كل نوع من أنواع المجلدات ومعالجتها:
```csharp
foreach (ExchangeFolderInfo folderInfo in folderInfoCol)
{
    switch (folderInfo.FolderType)
    {
        case ExchangeFolderType.Appointment:
            // التعامل مع مجلد نوع الموعد
            break;
        case ExchangeFolderType.Contact:
            // التعامل مع مجلد نوع جهة الاتصال
            break;
        case ExchangeFolderType.Task:
            // التعامل مع مجلد نوع المهمة
            break;
        case ExchangeFolderType.Note:
            // التعامل مع مجلد رسائل البريد الإلكتروني
            break;
        case ExchangeFolderType.StickyNote:
            // التعامل مع مجلد نوع StickyNote
            break;
        case ExchangeFolderType.Journal:
            // التعامل مع مجلد نوع المجلة
            break;
        default:
            // التعامل مع أنواع أخرى إذا لزم الأمر
            break;
    }
}
```
*حدود:* `folderInfo.FolderType` يحدد المنطق لمعالجة كل مجلد.

**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من صحة بيانات الاعتماد الخاصة بك لتجنب أخطاء المصادقة.
- التحقق من عنوان URL الخاص بالخادم ومدى توفره عند الاتصال عبر EWS.
