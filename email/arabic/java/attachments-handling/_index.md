# استخراج مرفقات البريد الإلكتروني باستخدام Aspose.Email للـ Java

في هذه المحور ستكتشف كل ما تحتاجه **لاستخراج مرفقات البريد الإلكتروني** من أكثر تنسيقات البريد شيوعًا باستخدام Aspose.Email للـ Java. سواء كنت تبني خدمة معالجة البريد، أو تقوم بأرشفة بيانات Outlook، أو ببساطة تحتاج إلى استخراج الملفات من رسائل MSG أو EML أو PST، فإن هذه الأدلة خطوة بخطوة تُظهر لك كيفية القيام بذلك بسرعة وبشكل موثوق.

## إجابات سريعة
- **ما هي أسهل طريقة لاستخراج المرفقات من ملف PST؟** استخدم `PersonalStorage` لفتح ملف PST وتكرار كائنات `Message`، مع استدعاء `Message.getAttachments()`.  
- **هل يمكنني استخراج الصور المضمنة (inline) كملفات منفصلة؟** نعم – تعامل معها كمرفقات عادية؛ Aspose.Email يتيحها عبر نفس الـ API.  
- **هل أحتاج إلى ترخيص لتشغيل الأمثلة؟** الترخيص المؤقت يعمل للتطوير؛ الترخيص الكامل مطلوب للإنتاج.  
- **ما هي التنسيقات المدعومة لاستخراج المرفقات؟** ملفات MSG و EML و EMLX و MHTML و PST مدعومة بالكامل.  
- **هل هناك طريقة لحفظ الملفات المستخرجة تلقائيًا؟** بالطبع – استدعِ `Attachment.save(filePath)` داخل حلقة لكتابة كل مرفق إلى القرص.

## ما هو “استخراج مرفقات البريد الإلكتروني”؟
يعني استخراج مرفقات البريد الإلكتروني قراءة رسالة بريد إلكتروني (أو ملف صندوق بريد) برمجيًا واستخراج أي ملفات مرفقة، مثل المستات أو الصور أو الكائنات المضمنة، بحيث يمكن حفظها أو معالجتها أو إعادة توجيهها إلى مكان آخر.

## لماذا تستخدم Aspose.Email للـ Java لاستخراج مرفقات البريد الإلكتروني؟
- **تغطية شاملة للتنسيقات** – يعمل مع MSG و EML و PST وغيرها دون الحاجة إلى تثبيت Outlook.  
- **بدون تفاعل COM** – API نقي للـ Java، مثالي للخوادم متعددة المنصات.  
- **أداء عالي** – المعالجة القائمة على التدفق تتيح لك التعامل مع صناديق بريد كبيرة بكفاءة.  
- **معالجة غنية للمرفقات** – يدعم المرفقات العادية، والمضمنة، ومرفقات TNEF المشفرة مباشرةً.

## المتطلبات المسبقة
- Java 8 أو أعلى.  
- مكتبة Aspose.Email للـ Java (قم بتنزيلها من الموقع الرسمي).  
- ترخيص Aspose مؤقت أو كامل للاستخدام في الإنتاج.

## الأسئلة المتكررة

**س: كيف يمكنني استخراج مرفقات البريد الإلكتروني من ملف MSG واحد؟**  
ج: حمّل الملف باستخدام `MailMessage.load("file.msg")` واستدعِ `mailMessage.getAttachments()`؛ ثم قم بالتكرار وحفظ كل مرفق.

**س: هل يمكنني استخراج المرفقات من ملفات PST المشفرة أو المحمية بكلمة مرور؟**  
ج: نعم. قدّم كلمة المرور عند فتح كائن `PersonalStorage`: `PersonalStorage.fromFile("file.pst", password)`.

**س: ما الفرق بين المرفقات العادية والمرفقات المضمنة (inline)؟**  
ج: المرفقات العادية هي ملفات منفصلة، بينما المرفقات المضمنة تكون داخل جسم البريد (غالبًا صور). Aspose.Email يتعامل مع كلاهما ككائنات `Attachment`، مما يسمح بمعالجتها بشكل موحد.

**س: هل هناك حد لحجم المرفقات التي يمكنني استخراجها؟**  
ج: المكتبة تستخدم التدفق، لذا الحد يعتمد فقط على الذاكرة المتاحة ومساحة القرص، وليس على حجم المرفق.

**س: هل يجب علي إغلاق التدفقات يدويًا بعد حفظ المرفقات؟**  
ج: عند استخدام `Attachment.save()`، تتولى المكتبة إغلاق التدفق تلقائيًا، ولكن إذا فتحت تدفقات مخصصة، تذكّر إغلاقها لتجنب التسريبات.
