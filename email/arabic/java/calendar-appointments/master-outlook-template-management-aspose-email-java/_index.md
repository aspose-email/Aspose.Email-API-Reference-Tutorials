# تحويل OFT إلى MSG – إتقان إدارة قوالب Outlook باستخدام Aspose.Email للغة Java

في هذا الدليل الشامل ستكتشف **كيفية تحويل OFT إلى MSG**، وتحديث خصائص قالب Outlook، وحفظ ملفات MSG للقوالب—كل ذلك باستخدام مكتبة Aspose.Email القوية للغة Java. سواءً كنت تبني حملات بريد إلكتروني آلية أو تولد دعوات اجتماعات، فإن إتقان سير عمل **تحويل OFT إلى MSG** سيوفر لك الوقت ويقلل الأخطاء اليدوية.

## إجابات سريعة
- **ماذا يعني “تحويل OFT إلى MSG”؟** يحول قالب Outlook (OFT) إلى رسالة Outlook مكتملة الإعداد (MSG).  
- **أي مكتبة تتولى التحويل؟** Aspose.Email للغة Java.  
- **هل أحتاج إلى ترخيص؟** النسخة التجريبية تعمل للاختبار؛ الترخيص الكامل يفتح جميع الميزات.  
- **هل يمكنني استخدام Maven لإدارة الاعتمادات؟** نعم، أضف حزمة Aspose.Email إلى Maven.  
- **هل Java 16 مطلوب؟** يوصى به، لكن الإصدارات الأحدث من JDK مدعومة أيضًا.

## ما هو “تحويل OFT إلى MSG”؟
*عملية “تحويل OFT إلى MSG” تغير ملف قالب Outlook (OFT) إلى ملف رسالة Outlook قياسي (MSG)، مع الحفاظ على التنسيق والمرفقات والبيانات الوصفية. من خلال التحويل، تتحول القالب القابل لإعادة الاستخدام إلى بريد جاهز للإرسال يمكن تعديله برمجيًا، وتخصيصه، وإرساله عبر أي خادم بريد أو عميل يدعم صيغة MSG.*  

## لماذا نستخدم Aspose.Email للغة Java لأتمتة سير عمل البريد الإلكتروني في Outlook؟
يدعم Aspose.Email **أكثر من 50 صيغة إدخال وإخراج**—بما في ذلك OFT و MSG و EML و MHTML—ويمكنه معالجة ملفات تصل إلى **2 GB** دون تحميل المستند بالكامل في الذاكرة. API النقي للغة Java يلغي الحاجة إلى تثبيت Outlook أو Microsoft Office على الخادم، مما يوفر أتمتة بريد موثوقة وعالية الإنتاجية.

## المتطلبات المسبقة

قبل البدء، تأكد من وجود ما يلي:

- **مكتبة Aspose.Email للغة Java**: الإصدار 25.4 أو أحدث (المكتبة تدعم JDK 16+).  
- **مجموعة تطوير Java (JDK)**: يوصى بـ JDK 16 أو أعلى للحصول على أفضل أداء.

## دليل التنفيذ

### كيف تحوّل OFT إلى MSG باستخدام Aspose.Email للغة Java؟

تشرح هذه الفقرة العملية الكاملة لتحويل قالب Outlook إلى رسالة Outlook مكتملة الإعداد. أولاً، تقوم بتحميل ملف OFT، ثم تخصيص الحقول مثل المرسل، المستلم، ومحتوى النص، وأخيرًا تحفظ النتيجة كملف MSG. النهج خفيف الوزن، يتطلب بضع أسطر من الشيفرة فقط، ويمكن دمجه في وظائف دفعية أو خدمات ويب لمعالجة حجم كبير.

#### تحميل وتحديث ملف قالب Outlook

##### نظرة عامة

تعلم كيفية تعديل محتوى ملف OFT (قالب Outlook) وتحويله إلى رسالة MSG مكتملة الإعداد.

##### خطوات التنفيذ

**1. تحميل قالب Outlook**

`MailMessage` هي الفئة الأساسية في Aspose.Email لتمثيل رسالة بريد إلكتروني في الذاكرة. توفر خصائص للموضوع، النص، المستلمين، والمرفقات.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
MailMessage message = MailMessage.load(dataDir + "sample.oft");
```

**2. تعيين تفاصيل المرسل والمستلم**

`MailMessage` تتيح لك تعيين حقول `from`، `to`، `cc`، و `bcc` مباشرة، مما يضمن أن الـ MSG النهائي يعكس معلومات التوجيه الصحيحة.

```java
message.setSender(new MailAddress("john@abc.com", "John"));
message.getTo().addMailAddress(new MailAddress("william@xzy.com", "William"));
```

**3. تحديث محتوى النص HTML**

يمكنك إسناد سلسلة HTML إلى `mailMessage.setHtmlBody()` لتخصيص القالب ببيانات ديناميكية مثل الأسماء، التواريخ، أو روابط الاجتماعات.

```java
String htmlBody = message.getHtmlBody();
htmlBody = htmlBody.replace("DisplayName", "<b>William</b>");
htmlBody = htmlBody.replace("MeetingPlace", "<u>Hall 1, Convention Center, New York, USA</u>");
htmlBody = htmlBody.replace("MeetingTime", "<u>Monday, June 28, 2010</u>");
message.setHtmlBody(htmlBody);
```

**4. حفظ كملف MSG**

استدعاء `mailMessage.save("output.msg", SaveOptions.getDefaultMsg())` يكتب الرسالة المُجهزة بالكامل إلى القرص بصيغة MSG، مُكملًا عملية **تحويل OFT إلى MSG**.

```java
MapiMessage mapimessage = MapiMessage.fromMailMessage(message);
mapimessage.setMessageFlags(MapiMessageFlags.MSGFLAG_UNSENT);
mapimessage.save(dataDir + "Invitation.msg");
```

### كيف تنشئ قالب Outlook جديد (OFT) باستخدام Aspose.Email؟

إنشاء قالب Outlook جديد من الصفر يتيح لك تعريف تخطيط قياسي يمكن إعادة استخدامه عبر الحملات أو الإشعارات. تبدأ بإنشاء كائن `MapiMessage`، تضبط خصائصه (الموضوع، النص، المرفقات)، ثم تحفظه كملف OFT. يمكن لاحقًا تحميل هذا القالب، تخصيصه، وتحويله إلى MSG حسب الحاجة.

**1. إنشاء رسالة بريد إلكتروني جديدة**

`MapiMessage` هو تمثيل منخفض المستوى في Aspose.Email لرسالة Outlook، ويوفر تحكمًا كاملاً في خصائص MAPI المطلوبة لملفات OFT.

```java
MapiMessage mapi = new MapiMessage("test@from.to", "test@to.to", "template subject", "Template body");
```

**2. حفظ كملف قالب**

احفظ كائن `MapiMessage` كملف OFT لاستخدامه في المستقبل.

```java
try {
    mapi.saveAsTemplate(dataDir + "mapiToOft.oft");
} finally {
    if (mapi != null) ((IDisposable)mapi).dispose();
}
```

2. **دعوات اجتماعات** – ملء قوائم الحضور وتفاصيل الاجتماع ديناميكيًا، ثم تحويله إلى MSG لتسليم Outlook.  
3. **توزيع مستندات** – تخزين الإشعارات المتكررة كقوالب OFT وإنشاء ملفات MSG عند الطلب.

- **تحسين استهلاك الموارد** – بثّ محتويات HTML الكبيرة أو المرفقات بدلاً من تحميلها بالكامل في الذاكرة.  
- **إدارة الذاكرة** – حرّر كائنات `MailMessage` و `MapiMessage` فور الانتهاء لتفريغ الموارد الأصلية.  
- **المعالجة الدفعية** – عالج مجموعات القوالب على دفعات (مثلاً 100 ملف لكل دفعة) للحفاظ على حجم الذاكرة في JVM تحت السيطرة.
