## دليل التنفيذ

### الميزة 1: الاتصال بخادم Exchange

#### ملخص
يُعد الاتصال بخادم Exchange الخطوة الأولى لإدارة رسائل البريد الإلكتروني برمجيًا. سيرشدك هذا القسم إلى كيفية إنشاء اتصال آمن باستخدام Aspose.Email لـ Java.

##### دليل خطوة بخطوة

**تحديد بيانات الاعتماد**

ابدأ بتحديد عنوان URI الخاص بصندوق البريد الإلكتروني وبيانات اعتماد المستخدم:

```java
String mailboxUri = "<HOST>";  // عنوان مضيف خادم Exchange
String domain = "";
String username = "<USERNAME>";  // اسم المستخدم الخاص بك في Exchange
String password = "<PASSWORD>";  // كلمة مرور Exchange الخاصة بك
```

**إنشاء كائن NetworkCredential**

استخدم بيانات الاعتماد المحددة لإنشاء `NetworkCredential` هدف:

```java
NetworkCredential credentials = new NetworkCredential(username, password, domain);
```

**إنشاء اتصال**

أخيرًا، قم بإنشاء اتصال بخادم Exchange الخاص بك باستخدام `EWSClient`:

```java
IEWSClient client = EWSClient.getEWSClient(mailboxUri, credentials);
System.out.println("Connected successfully.");
```

#### نصائح استكشاف الأخطاء وإصلاحها

- **بيانات اعتماد غير صالحة**:تأكد من اسم المستخدم وكلمة المرور الخاصة بك.
- **مشاكل الشبكة**:تأكد من أن اتصال الشبكة لديك مستقر وأن عنوان الخادم صحيح.

### الميزة 2: قائمة الرسائل من البريد الوارد

#### ملخص
بمجرد الاتصال بخادم Exchange، يمكنك استرداد الرسائل المخزنة في صندوق الوارد. تتيح لك هذه الميزة الوصول إلى بيانات البريد الإلكتروني برمجيًا.

##### دليل خطوة بخطوة

**استرداد رسائل البريد الوارد**

بافتراض وجود اتصال، قم بإدراج جميع الرسائل في صندوق الوارد:

```java
ExchangeMessageInfoCollection msgCollection = client.listMessages(client.getMailboxInfo().getInboxUri());
for (ExchangeMessageInfo info : msgCollection) {
    System.out.println("Subject: " + info.getSubject());
}
```

#### توضيح

- **`listMessages()`**:تعمل هذه الطريقة على استرجاع رسائل البريد الإلكتروني من عنوان URI الخاص بصندوق البريد الإلكتروني المحدد.
- **`ExchangeMessageInfoCollection`**:مجموعة تحتوي على معلومات حول كل بريد إلكتروني.

### الميزة 3: أرشفة رسائل البريد الوارد

#### ملخص
أرشفة الرسائل تُساعدك على إدارة صندوق الوارد بنقل رسائل البريد الإلكتروني إلى مجلد الأرشيف. تعرّف على كيفية أتمتة هذه المهمة باستخدام Aspose.Email لجافا.

##### دليل خطوة بخطوة

**أرشفة كل رسالة**

قم بالتكرار خلال مجموعة الرسائل وأرشفة كل واحدة منها:

```java
for (ExchangeMessageInfo msgInfo : msgCollection) {
    client.archiveItem(client.getMailboxInfo().getInboxUri(), msgInfo.getUniqueUri());
}
System.out.println("All messages archived.");
```

#### توضيح

- **`archiveItem()`**:نقل البريد الإلكتروني إلى مجلد الأرشيف باستخدام عنوان URI الفريد الخاص به.
