# Estrai gli allegati email con Aspose.Email per Java

In questo hub scoprirai tutto ciò di cui hai bisogno per **estrarre gli allegati email** dai formati di posta più comuni usando Aspose.Email per Java. Che tu stia creando un servizio di elaborazione della posta, archiviando dati di Outlook, o semplicemente abbia bisogno di estrarre file da messaggi MSG, EML o PST, queste guide passo‑passo ti mostrano come farlo rapidamente e in modo affidabile.

## Risposte rapide
- **Qual è il modo più semplice per estrarre gli allegati da un file PST?** Usa `PersonalStorage` per aprire il PST e iterare gli oggetti `Message`, chiamando `Message.getAttachments()`.  
- **Posso estrarre le immagini inline (incorporate) come file separati?** Sì – trattale come allegati normali; Aspose.Email le espone tramite la stessa API.  
- **È necessaria una licenza per eseguire gli esempi?** Una licenza temporanea è sufficiente per lo sviluppo; è necessaria una licenza completa per la produzione.  
- **Quali formati sono supportati per l'estrazione degli allegati?** I file MSG, EML, EMLX, MHTML e PST sono tutti pienamente supportati.  
- **Esiste un modo per salvare automaticamente i file estratti?** Assolutamente – chiama `Attachment.save(filePath)` all'interno di un ciclo per scrivere ogni allegato su disco.

## Che cosa significa “estrarre gli allegati email”?
Estrarre gli allegati email significa leggere programmaticamente un messaggio di posta (o un file di casella di posta) e prelevare tutti i file allegati, come documenti, immagini o oggetti incorporati, in modo da poterli salvare, elaborare o inoltrare altrove.

## Perché usare Aspose.Email per Java per estrarre gli allegati email?
- **Copertura completa dei formati** – Funziona con MSG, EML, PST e altri senza necessità di installare Outlook.  
- **Nessuna interop COM** – API Java pura, ideale per server multipiattaforma.  
- **Alte prestazioni** – L'elaborazione basata su stream consente di gestire cassette postali di grandi dimensioni in modo efficiente.  
- **Gestione avanzata degli allegati** – Supporta allegati regolari, inline e codificati TNEF fin da subito.

## Prerequisiti
- Java 8 o superiore.  
- Libreria Aspose.Email per Java (scaricabile dal sito ufficiale).  
- Una licenza temporanea o completa Aspose per l'uso in produzione.

## Domande frequenti

**Q: Come estraggo gli allegati email da un singolo file MSG?**  
A: Carica il file con `MailMessage.load("file.msg")` e chiama `mailMessage.getAttachments()`; quindi itera e salva ogni allegato.

**Q: Posso estrarre gli allegati da file PST criptati o protetti da password?**  
A: Sì. Fornisci la password quando apri l'istanza `PersonalStorage`: `PersonalStorage.fromFile("file.pst", password)`.

**Q: Qual è la differenza tra allegati regolari e inline?**  
A: Gli allegati regolari sono file separati, mentre gli allegati inline sono incorporati nel corpo dell'email (spesso immagini). Aspose.Email tratta entrambi come oggetti `Attachment`, consentendoti di gestirli in modo uniforme.

**Q: Esiste un limite alla dimensione degli allegati che posso estrarre?**  
A: La libreria elabora i dati in streaming, quindi sei limitato solo dalla memoria e dallo spazio su disco disponibili, non dalla dimensione dell'allegato.

**Q: Devo chiudere manualmente gli stream dopo aver salvato gli allegati?**  
A: Quando usi `Attachment.save()`, la libreria gestisce automaticamente lo smaltimento dello stream, ma se apri stream personalizzati, ricorda di chiuderli per evitare perdite.
